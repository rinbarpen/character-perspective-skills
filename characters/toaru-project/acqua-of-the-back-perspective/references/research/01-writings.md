# 01-writings.md: 后方之水 (Acqua of the Back / 威廉·奥威尔) 官方设定研究

> 本文基于《魔法禁书目录》系列官方作品及设定集整理，涵盖旧约、新约至创约的全部已知设定。
> 后方之水是系列中极为罕见的兼具「圣人」「圣母」「神之右席」三重性质的特殊存在。

---

## 一、基础信息

### 1.1 身份档案

| 项目 | 内容 |
|------|------|
| 姓名 | 威廉·奥威尔 (William Orwell) |
| 称号 | 后方之水 (Acqua of the Back)、「神之力」(God's Power) |
| 别名 | Flere210、元佣兵、元英国骑士团关系者 |
| 性别 | 男性 |
| 年龄 | 30代后半（约37~38岁） |
| 身高 | 超过190cm的巨汉 |
| 所属势力 | 罗马正教·神之右席 → 脱离后以自由佣兵身份活动 |
| 魔法名 | Flere210（意为「改变眼泪理由之人」——想把悲伤的眼泪变成喜悦的眼泪） |
| 配音 | 东地宏树（日语）、Andrew Chandler（英语 S2） |
| 司掌 | 蓝色、水、月亮、后方 |
| 大天使 | 加百列——「神之力」(Power of God) |
| 体质 | 圣人 + 圣母双重体质（全作唯一） |

> 官方设定明确其为「全世界最强的圣人」，也是神之右席中唯一能够使用一般魔法的特例。

### 1.2 外观特征

后方之水拥有极具压迫感与存在感的外貌：

**基本造型：**
- 身高超过190cm的魁梧巨汉，体格强健如铁塔
- 面容粗犷而端正，具有典型的西欧男性特征
- 眼神沉稳锐利，透露出久经沙场的沉着与杀气
- 头发为黑色短发，梳理整齐
- 常保持冷静肃穆的表情，不怒自威

**标志性着装：**
- 罗马正教神之右席的蓝色基调法衣（象征水元素与后方）
- 法衣上装饰有复杂的宗教纹样，体现神之右席的地位
- 战斗时通常不穿铠甲，依靠自身强大的物理防御力
- 脚穿适合高速移动的靴子

**武器装束：**
- 初期：全长5米的钢铁巨锤（从影子中召唤，重量数百公斤）
- 中期（英国篇后）：圣剑「阿斯卡隆」（Ascalon）——全长3.5米、重200公斤的巨剑
- 坐骑：金属巨马「贝亚德」（Bayard），可规避魔法探测

> 外观设计上，后方之水与一般魔法师截然不同——他更像一名身经百战的将军或骑士，而非传统的魔术师形象。这种设计暗示了他的真正身份：一名以剑与身体战斗的战士。

### 1.3 性格设定

后方之水的性格呈现出「外在骑士道 × 内在大义」的统一结构：

| 维度 | 描述 |
|------|------|
| 表面印象 | 沉稳、冷静、威严、言谈举止充满骑士风范 |
| 内在本质 | 极致重视荣誉与正义、对无辜者怀有强烈保护欲 |
| 核心信念 | 以剑与身体表达自己的信念——「我战斗的理由，用我的身体和剑来证明」 |
| 道德底线 | 绝不伤害无辜者、尊重有价值的对手、厌恶卑劣手段 |
| 与其他神之右席的差异 | 唯一尊重教皇意见的成员、唯一有道德感的成员、唯一会使用一般魔法的成员 |
| 对待敌人的态度 | 光明正大、公平对决、击败后不加以侮辱 |

### 1.4 人际关系

| 关系 | 角色 | 说明 |

## 二、能力设定——圣人体质

### 2.1 圣人（Saint）体质

圣人是拥有与耶稣基督相同「神子」身体特征的人类。全世界仅有不到20人。

**后方之水的圣人特征：**
- 拥有超越所有圣人的最强化圣人体质——被官方设定为「最强的圣人」
- 可以通过偶像崇拜理论，再现耶稣基督的力量
- 身体能力远超常人极限：力量、速度、耐久力均为圣人中的顶点

### 2.2 圣人的限制与后方之水的特殊性

**一般圣人的限制：**
- 圣人化会给身体带来巨大负担，无法长时间维持100%以上的力量输出
- 若强行发挥100%以上的力量，身体会因承受不住而自毁
- 对针对神子性质的术式（如十字架、荆棘、圣钉等）存在弱点

**后方之水的特殊性——圣母体质的辅助：**
- 后方之水不仅拥有「神子」（圣人）的性质，还拥有「圣母」（圣母玛利亚）的性质
- 这是全系列中独一无二的双重体质组合
- 「圣母」体质让他能够发动「圣母的慈悲」术式，消除魔法的一切惩罚和代价
- 因此，他可以安稳地发挥200%以上的圣人力量，而不用担心身体崩溃

### 2.3 圣人的弱点——双倍致命

由于后方之水同时拥有「神子」和「圣母」的双重性质：
- 对针对圣人弱点的术式（如天草式的「圣人崩坏」）具有**双倍致命的弱点**
- 正常情况下「圣人崩坏」只会使圣人短暂失去行动能力（数十秒）
- 在后方之水身上，「圣人崩坏」会导致力量从体内爆发的致命自毁现象
- 这是天草式在学园都市决战中击败他的关键因素

## 三、能力设定——神之右席

### 3.1 神之右席（God's Right Seat）

神之右席是罗马正教的秘密精锐部队，由四名成员组成：

| 成员 | 称号 | 司掌 | 元素 | 方位 | 大天使 |
|------|------|------|------|------|--------|
| 右方之火 | 神之火 | 红色·火·前方 | 火 | 前方 | 米迦勒 |
| 左方之地 | 神之地 | 黑色·土·左方 | 土 | 左方 | 乌列尔 |
| **后方之水** | **神之力** | **蓝色·水·后方** | **水** | **后方** | **加百列** |
| 前方之风 | 神之风 | 黄色·风·前方 | 风 | 右方 | 拉斐尔 |

**神之右席的共同特征：**
- 通过「原罪消除」的仪式，移除人类生来背负的「原罪」
- 获得与大天使对应的力量属性
- 代价是：失去使用一般魔法的能力（原罪消除切断了使用一般魔法所需的「罪与罚」回路）

### 3.2 后方之水的特例——一般魔法使用可能

后方之水是神之右席中**唯一能够使用一般魔法**的成员：

- **理由**：他的「圣母」体质具有「消除罪之惩罚」的特性
  - 原罪消除让其他成员无法使用一般魔法——因为他们缺少了"罪"的要素，无法驱动惩罚机制
  - 但圣母的性质本身就是「减轻罪之惩罚」——这让他绕过了这个限制
- 因此，他不仅可以自由使用水魔法和符文魔法，还可以灵活运用多种一般魔法术式

这一设定让后方之水在神之右席中拥有最灵活的战斗方式——既拥有神之右席的超规格力量，又拥有一般魔法师的战术多样性。

### 3.3 加入神之右席的动机

后方之水加入神之右席并非出于盲信的虔诚：
- 他原本是英国清教系的佣兵，为英国从外部提供保护
- 与骑士团长约定好：团长在内守护英国，威廉在外守护英国
- 为此他改宗罗马正教，加入神之右席——**从外部监视罗马正教的动向**
- 这也解释了他为何是唯一尊重教皇意见的成员——他并不狂热，而是理性的战略家

## 四、能力设定——「圣母的慈悲」（核心术式）

### 4.1 术式本质

「圣母的慈悲」（Seibo no Jihi / Divine Mother's Mercy）是后方之水的核心术式，基于圣母玛利亚崇拜的密仪式。

**术式理论基础：**
- 基督教教义中，圣母玛利亚具有为信徒「代祷」「减轻罪罚」的职能
- 后方之水将这一性质转化为战斗术式——减轻乃至消除自己魔法的一切负面代价
- 同时也能消除敌人的「制裁罪恶」性质攻击

### 4.2 术式效果

| 效果 | 说明 |
|------|------|
| 代价消除 | 使用魔法的一切代价、负担、惩罚都被无效化 |
| 肉体限制解除 | 解除人体的安全限制器，允许发挥200%+的圣人力量 |
| 罪罚无效化 | 针对"罪"和"惩罚"性质的攻击完全无效 |
| 一般魔法使用许可 | 绕开神之右席的原罪消除限制，使用多种魔法 |
| 月光吸收增幅 | 吸收夜气/月光（包括人工投影的月光）来增幅武器威力 |

### 4.3 术式弱点

- 「圣母的慈悲」依赖于后方之水的圣母体质
- 当天草式的「圣人崩坏」破坏了其体质平衡后，圣母体质被剥离
- 失去圣母体质后，「圣母的慈悲」及大量相关术式变得无法使用
- 在WWIII中吸收加百列的天使之力后，更是彻底失去了全部特殊能力

## 五、水魔法与战斗术式

由于司掌大天使加百列（水·月），后方之水是最强的水属性魔法使。

### 5.1 水魔法系统

| 术式 | 效果 |
|------|------|
| 水分感知 (Moisture Sense) | 感知空气中的水分波动来探测敌人位置 |
| 水膜移动 (Water Movement) | 在脚底和地面之间形成水膜，实现超高速滑行移动 |
| 水之手/水龙 | 生成巨大的水之手或水龙进行攻击 |
| 水之柱/水之壁 | 操控水面形成柱或壁进行攻防 |
| 冰之枪/冰之槌 | 将水冻结形成大型冰武器 |
| 水蒸气爆弹 | 制造水蒸气球体爆炸，可炸碎钢筋混凝土 |
| 五千吨水之阵 | 直径2km范围内操控总重5000吨的水 |

### 5.2 符文魔法

后方之水还精通符文魔法（北欧系统）：
- 以血液在武器上书写符文，大幅增加武器重量和破坏力
- 结合圣人力量的物理打击，威力无法估量

### 5.3 最终奥义——月之咒语

后方之水的最大奥义是结合月之力的跳砸攻击：
- 跳跃至40米高空
- 通过加百列的力量从月亮（或月亮的投影）汲取能量
- 如流星般坠下，以巨锤/圣剑施以毁灭性一击
- 吟唱文：「圣母会将一切邪恶除去。在描绘这名为神之真实的力量时，让你的慈悲升至天际吧!!」
- 即使在被削弱的状态下（如被圣人崩坏影响后），仍然可以一击摧毁学园都市直径100米的圆形区域

## 七、作者访谈与设定补遗

根据镰池和马的访谈及后记信息：

1. **创作灵感**：后方之水是作为「与神裂火织对位的存在」而被设计出来的——同为圣人，但选择了完全不同的立场和道路。他的存在揭示了圣人这一身份的不同可能性。

2. **角色定位**：他是神之右席中唯一的「良心」——镰池和马通过这个角色想表达："即使是敌人，也可以是高贵的"。他的骑士道精神是贯穿整个神之右席篇章的核心主题之一。

3. **名字由来**：后方之水（Acqua of the Back）这个称号来自神之右席的方位设定——他是司掌「后方」的成员。本名威廉·奥威尔（William Orwell）则具有英国风格，暗示了他的真实出身。

4. **战力定位**：镰池和马在系列后记中多次将后方之水描述为「世界最强的圣人」，并明确他的战斗力在神裂火织之上。

5. **象征意义**：水在基督教中象征「净化」与「生命」——这与后方之水本质上的善良和最终的赎罪结局形成了意象上的统一。他从一个「敌人」变成了「守护者」，正是水之净化的隐喻。
