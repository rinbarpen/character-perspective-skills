# 06-timeline.md: 后方之水 (Acqua of the Back / 威廉·奥威尔) 完整时间线

> 本文按时间顺序梳理后方之水在《魔法禁书目录》系列中的完整经历，涵盖其背景故事、主要登场剧情、战斗记录以及后续发展。

---

## 一、时间线总览

| 时期 | 时间节点 | 主要事件 | 力量状态 |
|------|----------|----------|----------|
| 佣兵时代 | 故事开始前多年 | 以自由佣兵身份活动，保护英国 | 圣人体质（未觉醒圣母） |

## 二、佣兵时代（故事开始前——具体时间不明）

### 2.1 早年生涯

| 时间 | 事件 |
|------|------|
| 出生 | 出生于英国，具体出生地和出生时间不明。推测年龄在30代后半（约37-38岁）。 |
| 早期经历 | 作为圣人觉醒，拥有超越常人的身体能力。接受英国清教的训练和教育，成为一名魔法师。 |
| 佣兵出道 | 决定以自由佣兵的身份活跃于欧洲魔法战场。因战斗能力极高，迅速在佣兵界建立了名声。 |

### 2.2 佣兵时代的重要事迹

| 时间 | 事件 |
|------|------|
| 佣兵时代 | 在战场上结识骑士团长，两人成为好友和酒友，建立了深厚的友谊和相互尊重。 |
| 佣兵时代 | 多次以暗中的行动保护英国利益，虽然不被正式承认，但英国王室知晓他的贡献。 |
| 佣兵时代 | 接受英国清教·必要之恶教会的委托，执行各种危险任务。 |
| 佣兵时代 | 在与普通人的接触中，他还教导战乱地区的平民农业、医疗和生存技能——被称为「贤者」。 |

### 2.3 薇莉安公主救出事件

**时间：加入神之右席前**

- 英国第三皇女薇莉安被「西班牙占星术师结社」绑架
- 英国骑士团因政治内斗而无法出动救援
- 威廉独自深入敌阵，成功救出薇莉安
- 这一事件奠定了两人之间的深厚羁绊

### 2.4 拒绝骑士爵位

**时间：薇莉安救出事件后**

- 英国王室为表彰其功绩，授予他骑士爵位和贵族身份
- 王室甚至为他设计了纹章，保存在白金汉宫
- 但他拒绝了正式的骑士授勋，理由是对体制的不信任
- 与骑士团长达成约定：团长在内守护英国，威廉在外守护英国

### 2.5 佣兵时代总结

此时的威廉·奥威尔已经是一名成熟的战士——拥有丰富的实战经验、高超的战斗技巧和坚定的个人信念。他虽然不是正式骑士，但已经以骑士道精神生活了多年。

## 三、神之右席时期（故事开始前——旧约16卷）

### 3.1 转宗与加入

**时间：旧约故事开始前数年**

- 威廉·奥威尔决定改宗罗马正教
- 以他的圣人身份和战斗力，被邀请加入神之右席
- 经过特殊仪式：被消除了原罪，获得了神之右席的力量加成
- 同时，他的圣母体质被激活，获得了「圣母的慈悲」术式
- 获得称号「后方之水」——司掌水、蓝色、月、后方、大天使加百列

### 3.2 在神之右席中的活动

**时间：加入后至学园都市入侵前**

- 与前方之风、左方之地、右方之火三位同僚共事
- 发现左方之地进行人体实验，果断将其处决
- 对神之右席的终极计划和右方之火的野心保持警惕
- 在可能的范围内阻止神之右席的过度行为
- 同时继续暗中关注英国的动向

### 3.3 处决左方之地

**时间：旧约16卷直前**

- 发现左方之地在平民区进行人体实验，利用观光客（包括儿童）测试术式
- 未经审判或报告，直接以私刑处决左方之地
- 用教会的石柱一击将其腰斩——左方之地甚至来不及使用术式

> 这是后方之水在成为敌人前最重要的高光时刻，明确了他的道德观：正义高于组织忠诚。

## 四、学园都市入侵篇（旧约第16卷）

### 4.1 入侵阶段

| 时间 | 事件 |
|------|------|
| 10月中旬·日 | 后方之水独自攻入学园都市，目标：消灭上条当麻的幻想杀手 |
| 同日 | 遭遇天草式十字凄教的阻击——以压倒性力量击溃天草式全员 |
| 同日 | 追赶上条当麻和五和，宣告自己的目的——「我要的只是你的右手」 |
| 同日 | 被五和的「圣人崩坏」击中，暂时撤退前给予天草式「一天宽限」 |

### 4.2 决战阶段

| 时间 | 事件 |
|------|------|
| 约定的次日 | 后方之水如约返回战场 |
| 决战开始 | 天草式再次发动「圣人崩坏」——但后方之水用「圣母的慈悲」将其无效化 |
| 圣人对决 | 神裂火织赶到，两人展开「最强圣人vs最强圣人」的对决 |
| 激战过程 | 后方之水以200%的圣人力量全面压制神裂，水魔法与七天七刀激烈碰撞 |
| 战局转折 | 上条当麻用幻想杀手消除了后方之水的水魔法和月光术式 |
| 致命一击 | 五和趁机发动第二发「圣人崩坏」——因为后方之水的双重体质，圣人崩坏造成致命伤害 |
| 最终结局 | 后方之水被击落入人工湖。他在千钧一发之际构筑起分散力量的管道，避免了自爆，但**失去了圣母体质** |

### 4.3 战后

- 被英国清教俘虏，囚禁于伦敦塔
- 被剥离了「圣母的慈悲」术式——从200%力量的圣人跌落为普通圣人水平
- 在狱中等待命运的下一步

## 五、英国万圣夜政变篇（旧约第17-18卷）

### 5.1 政变背景

| 时间 | 事件 |
|------|------|
| 10月下旬 | 英国第二皇女凯莉莎发动政变，以卡提纳正统的力量控制英国骑士团 |
| 政变期间 | 骑士团长被凯莉莎操控，奉命处决第三皇女薇莉安 |
| 政变期间 | 必要之恶教会的战力（神裂、史提尔等）被卷入混战 |

### 5.2 后方之水的介入

| 时间 | 事件 |
|------|------|
| 政变高潮 | 后方之水从伦敦塔逃出（或被人放出） |
| 突入刑场 | 骑着金属巨马贝亚德闯入处刑现场，在骑士团长的斧头落下前救下薇莉安 |
| 第一幕 | 「抱歉，我来晚了，公主殿下。让一位淑女等待，可不是骑士该做的事。」 |
| 护送公主 | 将薇莉安送上贝亚德，指令战马带她前往安全地点 |

### 5.3 后方之水 vs 骑士团长

**地点：多佛尔海峡附近**

| 阶段 | 内容 |
|------|------|
| 开场 | 两位老友在战场上对峙——后方之水手持圣剑阿斯卡隆，骑士团长手持赫伦汀 |
| 初期 | 骑士团长使用「模式」和「归零」术式限制后方之水——初期后方之水处于下风 |
| 转折 | 后方之水利用阿斯卡隆的隐藏机关（剑柄内藏的钢丝剑）奇袭骑士团长 |
| 白热化 | 双方展开激烈的剑术对决——威廉虽然失去了圣母体质，但战斗技巧仍在 |
| 终结 | 后方之水用阿斯卡隆剑柄的金属栓一记重击，击昏骑士团长 |
| 胜利后 | 他没有杀死骑士团长——因为对方仍然是他认可的朋友 |

### 5.4 政变结局

- 后方之水成功保护了薇莉安公主
- 与上条当麻等人联手稳定了英国局势
- 政变平息后，他恢复了自由身
- 不再被视为囚犯或敌人——以威廉·奥威尔的身份重新开始

## 六、第三次世界大战篇（旧约第20-22卷）

### 6.1 战争背景

| 时间 | 事件 |
|------|------|
| 11月 | 右方之火发动第三次世界大战，试图利用大天使加百列的力量实现他的「计划」 |
| 战况 | 以俄罗斯为舞台，全球魔法势力卷入混战 |
| 天使降临 | 右方之火利用「天使坠落」事件的残余影响，召唤并操控大天使加百列 |

### 6.2 后方之水的参战

| 时间 | 事件 |
|------|------|
| 战争初期 | 后方之水以威廉·奥威尔的身份出现在战场上——不再是神之右席的成员 |
| 对上条当麻 | 与上条重逢，表明立场已经改变——「立场这种事情，在这个局面下已经没有意义了」 |
| 与神裂等人合作 | 与神裂火织、骑士团长等人一起，协同应对世界大战的危机 |

### 6.3 决定性行动——吸收加百列的天使之力

| 阶段 | 内容 |
|------|------|
| 决意 | 后方之水判断加百列的暴走必须被阻止——即使代价是自己的生命 |
| 术式发动 | 利用自己与加百列（水·月）的亲和性，发动吸收术式 |
| 过程 | 强行吸收了大天使加百列约50%的天使之力（Telesma） |
| 代价 | 体内的天使之力烧毁了他的全部魔力回路 |
| 结果 | 永久失去了：圣人体质、神之右席加成、「圣母的慈悲」术式、水魔法能力 |

### 6.4 濒死与再生

| 时间 | 事件 |
|------|------|
| 吸收后 | 后方之水濒临死亡，倒在战场上 |
| 被救 | 滨面仕上和滝壶理后发现他，鼓励他不要放弃 |
| 意志再起 | 想起对薇莉安公主的承诺——「我还有应该守护的东西。还有没完成的约定。」 |
| 最终 | 成功存活，但已是一个「普通人」 |

### 6.5 世界大战结局

- 右方之火的计划被阻止
- 加百列被上条当麻的幻想杀手送回天界
- 后方之水以失去全部力量为代价，完成了最后的战斗

## 七、失去力量后的活动

### 7.1 力量残留情况

| 项目 | 状态 |
|------|------|
| 圣人体质 | 完全丧失 |
| 神之右席加成 | 完全丧失 |
| 圣母的慈悲 | 完全丧失 |
| 水魔法 | 可以使用的术式大幅减少，威力大减 |
| 阿斯卡隆 | 基本无法挥动（原重200kg，失去圣人力量后无法使用） |
| 战斗经验 | 仍然保留——他仍是优秀的战术家和格斗家 |
| 符文魔法 | 部分保留——但威力远不如前 |

### 7.2 新约时期的活动

**新约第4卷之后：**

- 不再使用「后方之水」的称号，以本名「威廉·奥威尔」活动
- 与神裂火织、骑士团长、凯莉莎组成「屠魔四刃」（Demon Slayer Four Blades）
- 共同对抗欧提努斯（魔法之神）带来的危机
- 虽然战斗力大幅下降，但作为战术支援和经验提供者仍然发挥作用

**新约后期：**

- 在少数场景中短暂登场，主要作为曾经的重要角色的延续
- 与薇莉安公主的感情线偶尔被提及
- 与骑士团长的友谊继续维持

### 7.3 创约时期

**创约时期：**

- 登场次数进一步减少
- 在需要过去的战斗经验和知识的场景中偶尔出现
- 作为已退居二线的老练战士，为新一代提供支持

## 九、关键魔法/能力变化时序

```
【佣兵时代】                    【神之右席时期】            【战后时期】
  圣人体质（基础）     →    圣人体质 + 神之右席         圣人体质丧失
  一般魔法知识                + 圣母的慈悲              神之右席丧失
  战斗经验                    + 水魔法强化              圣母体质丧失
                              + 阿斯卡隆               仅残留魔法知识与经验
                                                     
力量量级：
  一般圣人级          →    最强圣人级（200%）    →    普通人级（仅经验）
```

> 后方之水是《魔法禁书目录》全系列中唯一一个「力量持续下降」的主要战斗角色。他不像其他角色那样不断获得新的力量，而是在每一次重要转折中失去一部分力量——但这些失去都是有意义的，都是他自己选择的结果。
