# 御坂美琴 完整时间线

> 调研来源：魔禁系列正传（旧约/新约/创约）、超电磁炮漫画及动画、某科学的一方通行、魔禁 Wiki（toaru.fandom.com）、中文维基及贴吧考据、日文原作（电击文库）

---

## 一、概要

**御坂美琴（Misaka Mikoto）** 是《魔法禁书目录》系列及其衍生的《某科学的超电磁炮》双线主角。本时间线覆盖其从《超电磁炮》开篇至《创约》的最新历程，涵盖旧约（OT，Old Testament，全22卷）、新约（NT，New Testament，全22卷R）、创约（GT，Genesis Testament，连载中）三条主线及《超炮》漫画及动画的时间节点。

> **注**：魔禁系列时间轴存在一定模糊性（作者镰池和马采用"事件序"而非严格时间点），以下标注的卷号/集数为确凿的作品出处。各事件在同一卷内的前后顺序是可靠的。

## 二、《超电磁炮》篇——学园都市日常与暗流（旧约时间线前期）

| 时间序 | 作品出处 | 事件 | 详细说明 |
|--------|---------|------|----------|
| 约 OT 之前数年 | 超炮漫画第1卷 / 动画S1 Ep1-2 | **与白井黑子相遇** | 美琴转入常盘台中学，与分配到同一宿舍的白井黑子相识。黑子从此以"姐姐大人"称呼，成为室友兼搭档。 |
| 约 OT 之前 | 超炮漫画第1卷 / 动画S1 Ep1-2 | **Level 5 日常** | 美琴作为学园都市仅七人的 Level 5 超能力者（第三位），编号「超电磁炮（Railgun）」。日常活动包含违反校规追赶自动贩卖机、用硬币发射超电磁炮、与风纪委员的黑子斗智斗勇。 |
| 约 OT 之前 | 超炮漫画第2卷 / 动画S1 Ep6-7 | **"幻想御手"（Level Upper）事件** | 春上衿衣等人的能力提升事件。美琴调查"Level Upper"幕后时遭遇**布束砥信**（在研究所内，此时美琴不知其与妹妹们的关联）。事件最终被佐天泪子持 Level Upper 昏迷触发，美琴救出佐天。事件真相为木山春生试图通过"幻想御手"收集运算模式以唤醒昏迷学生——美琴在此事件中初识警策看护，但不知其深层身份。 |
| 约 OT 之前 | 超炮漫画第3-4卷 / 动画S1 Ep11-14 | **"Tree Diagram"残骸事件** | 树形图设计者的残骸被用于"学习装置"事件。美琴追踪残骸走私，与**布束砥信**再次相遇。布束暗示美琴"不要再调查下去"。在事件末尾，美琴意识到"Tree Diagram 残骸"可能被用于某项与她有关的计划。此事件中**食蜂操祈**初登场（超炮时间线），二人因"遥控器"事件产生首次交集。 |
| 约 OT 之前 | 超炮漫画第5-7卷 / 动画S2（超炮S）Ep1-16 | **绝对能力者计划 / 妹妹篇（美琴视角）** | 美琴发现自己约两万个"妹妹"（克隆人）被用于一方通行的 Level 6 进化实验。她试图独自摧毁实验，袭击各研究机构并破坏情报系统。一方通行找到她并进行战斗——美琴惨败。上条当麻出现，为保护美琴而以"幻想杀手"击败一方通行，摧毁实验系统。美琴在事后向上条坦白"想叫他停止却又在心底松了口气"，也是第一次在他面前流泪。此事件是美琴对上条当麻感情的转折点——从略带对抗意识到深深依赖与信任（即"欠你一份人情"→"喜欢"）。 |
| 约 OT3 之后 | 超炮漫画第8卷 / 超炮S Ep17-24 | **妹妹们的后续** | 实验停止后，约一万个妹妹被送往世界各地接受医疗支援。美琴多方奔走为她们争取权益，并开始思考自己作为"原型"的责任。**最后之作**（Last Order，编号20001）被带到学园都市，美琴与她的关系成为后续重要的羁绊线。 |
| 约 OT5 前后 | 超炮漫画第9-10卷 / 动画S3（超炮T）Ep1-15 | **大霸星祭 / 食蜂操祈篇** | 大霸星祭期间，食蜂操祈与美琴因过去"遥控器实验"（桔梗天乃）事件产生的恩怨发酵。警策看护与食蜂联手绑架御坂美琴（假），引发一系列事件。美琴与食蜂的关系在此达到第一个高潮——二人最终达成不完整和解，食蜂归还日记。此篇章揭开了美琴作为"超能力者"背后的孤独与被各方觊觎的处境。 |
| 约 OT6 之后 | 超炮漫画第11-12卷 | **"天赋梦路"（Dream Ranker）事件** | 美琴参加精神感应型游戏测试，遭遇"分身"事件的超自然现象。揭示了学园都市以"能力"为目标的深层研究。同时，**御坂美琴的深层心理创伤**——对妹妹们的愧疚和对上条当麻"每次都被他救"的不甘——被具象化为"Dolly"（多莉：初代克隆体，编号0号）。美琴与多莉的短暂相处是她心理成长的重要一步。某科学的一方通行中更新了御坂妹妹的故事线，最终所有妹妹重获新生。 |
| 约 NT 初期 | 超炮漫画第13-17卷（连载中） | **"Jailbreaker"（越狱者）/ 战斗机器人事件** | 美琴追寻流浪的御坂妹妹们（编号10032等），遭遇前Skill-Out成员和新兴敌手。出现能与美琴对抗的战斗机器人体系——"轰炸者"（Scavenger）团队登场。美琴开始正面面对自己作为 Level 5 所承担的期望与责任，不再一味单打独斗，而是学会整合伙伴的力量。 |

## 三、《魔禁》旧约（OT）篇——与上条当麻的交集与成长

### 3.1 旧约早期邂逅（OT1-OT3）

| 卷号 | 事件 | 详细说明 |
|------|------|----------|
| **OT1** | **初次相遇** | 美琴在学园都市的某座铁桥上与**上条当麻**初次相遇。著名场景：美琴一边说着"我啊，有个大逆不道的习惯"一边破坏自动贩卖机；上条吐槽后，美琴对他发出超电磁炮——但被"幻想杀手"抹消。美琴因此将上条列为"必须打败的对手"，此后频繁找他打架（实则被上条"奇特的接近方式"吸引而不自知）。 |
| **OT2** | **日常追逐** | 美琴多次以"决斗"之名找上条（实战为单方面被幻想杀手抵消），二人关系逐渐从对抗向熟人/暧昧方向滑移。上条称其为"棒棒糖女"或"放电女"，美琴回应以"笨蛋"和踢击。 |
| **OT3** | **绝对能力者 / 妹妹事件** | 上条伤势刚恢复，美琴独走执行破坏计划。上条追赶上条被一方通行击败的现场，用幻想杀手打破实验系统。这是《魔禁》主线中美琴角色弧光最重要的转折点：<br>1. 她从"什么问题都要自己扛"转向允许他人介入<br>2. 她开始对上条产生深刻感情（虽然本人极力否认）<br>3. 美琴承认自己的软弱："其实我……好害怕……好想叫住他（一方通行）啊……却没有一个理由……" |

### 3.2 旧约中期——盟友化（OT5-OT13）

| 卷号 | 事件 | 详细说明 |
|------|------|----------|
| **OT5** | **学艺都市 / 残骸事件** | 受上条邀请（实则被迫卷入）前往学艺都市。美琴与上条并肩对付恋查（研究员）、以及要摧毁学艺都市的第三方。首次以"战友"身份合作而非单方面被救。体现了美琴战斗能力的价值，但也暴露出其面对魔法侧敌人时的知识短板。 |
| **OT6** | **天草式事件** | 美琴出场较少，但此卷中上条与**神裂火织**的关系被进一步展开，上条在魔法侧的存在感加深——这是美琴第一次意识到"上条当麻的生活中，有她完全不了解的另一个世界"。 |
| **OT8** | **奥索拉（Orsola）事件** | 美琴在学园都市外短暂出场。已开始习惯上条的"到处乱跑"式生活节奏。二人的日常互动（\n如一起吃饭、见面互怼）逐渐成为固定模式。 |
| **OT13** | **亚得里亚海女王篇尾声** | 美琴在学园都市收到上条从意大利寄回的伴手礼。这是二人关系已进入"虽不常见面但互相关心"的稳定阶段的佐证。 |

### 3.3 旧约后期——英雄的觉醒（OT15-OT22）

| 卷号 | 事件 | 详细说明 |
|------|------|----------|
| **OT15** | **后方之水（Acqua of the Back）入侵** | 后方之水通过学艺都市的线索找到美琴并与其交战。美琴首次正面面对圣人级别的对手——尽管全力奋战，仍被秒杀。这是她深刻意识到自己能力的"天花板"的时刻——在魔法侧真正的强者面前，Level 5 超能力者并不如她想象中强大。当后方之水殴打黑子时，美琴第一次感到真正的无力。上条拦下后方之水，救下二人。 |
| **OT16** | **不列颠万圣节篇 / 伦敦决战** | 美琴主动前往伦敦支援上条："那个笨蛋又在拼命了，这次轮到我帮他。"<br>1. 与**右方之火（Fiamma of the Right）** 短暂交手<br>2. 参与不列颠三派内战，与神裂火织、土御门元春等魔法侧人士共战<br>3. 验证了对上条的感情（虽然嘴上不承认，但行为上已完全是个愿意为所爱之人赴汤蹈火的少女）<br>此卷设定几乎等于美琴对上条心理地位的一次"正式宣告"：她不再被动等待救援，而是主动走向他的战场。 |
| **OT19-20** | **第三次世界大战篇 / 对法之敌** | **关键节点——美琴获得「AAA」（Anti-Art Attachment 反魔法武装）**<br><br>1. 学园都市统括理事会为控制美琴，将原本为"对付上条当麻"而开发的**反魔法外装**（AAA）交给美琴——一个装甲型自律战斗工具<br>2. 美琴接受AAA后，战斗力大幅跃升：能以超电磁炮原理释放远超此前水平的攻击，对天使级目标造成有效伤害<br>3. 美琴奔赴第三次世界大战前线（俄罗斯），目标是找到被右方之火带走的上条当麻。这一动机清晰表明：她已经不只是个"被卷入的学园都市超能力者"，而是个主动选择的战斗者<br>4. 与**右方之火**最终战间接相关：上条击穿右方之火后最终回到美琴身边。二人有简短的温暖重逢——上条失忆后（旧约此后的大事件）美琴不得不重新面对"他可能不记得我们的过去了"这一残酷事实。 |
| **OT21-22** | **失忆后的初遇** | 美琴对上条失忆事件后发展出一套心照不宣的默契：她不说，他也不追问。日常回到"一起吃饭、互怼"的模式，但美琴此时已经知道——自己爱上的这个男人，有比他失忆更深的痛苦。

> **旧约总结**：美琴的旧约线是一部"从只关心自己的日常→闯入非日常→主动拥抱非日常"的成长史。她在 OT3 学会让别人帮忙，在 OT15 学会正面面对无力感，在 OT16 学会主动跨入他的世界，在 OT19 获得足够强大的能力去守护重要的人。

## 四、《魔禁》新约（NT）篇——成熟与蜕变

### 4.1 新约早期——坚守与疏离（NT1-NT7）

| 卷号 | 事件 | 详细说明 |
|------|------|----------|
| **NT1** | **妹妹们的复活 / 欧提努斯登场** | 魔禁新约开篇。世界恢复和平后，美琴与上条一起将最后一批妹妹们送出国。日常关系继续。美琴此时已基本接受"上条身边总是怪事不断"的生活节奏。新威胁**欧提努斯**（Othinus）的伏笔开始出现。 |
| **NT3** | **对麦野沉利（Mugino Shizuri）** | "道具"（Item）组织的麦野沉利与美琴正面交战。麦野作为第四位的 Level 5，攻击力与美琴不相上下。美琴在战斗中展现出更强的应变能力和战术意识——不再像旧约那样只依赖"蛮力"对决。此战体现了美琴战斗风格的成熟。 |
| **NT4** | **对索尔（Thor）** | 科学侧与魔法侧冲突持续。美琴与象征"新世代魔法师"的索尔短暂交手。索尔对美琴的评价很有代表性："她很强，但她只是在守护自己那一小片世界——这和真正的英雄还差得远。"此评价精准影射美琴当时的心理局限。 |
| **NT5-6** | **对欧雷尔斯（Ollerus）势力** | 美琴参与对欧雷尔斯阵营的战斗（虽然她的功率输出对上位魔法师仍显不足）。此时美琴面对魔法侧已有相当经验，不再畏惧"未知"，但仍缺与"魔神"级别战斗的战略能力。 |
| **NT7** | **对上条赝品「恋查改·Rensa」** | 学园都市制造了一个完全复制上条当麻幻想抹消能力的机械强化体"恋查改·Rensa"。美琴面对这个"上条当麻的复制品"——她无法像对真正上条那样信任对方也手下留情，被迫全力战斗。这是一场深刻的心理折磨：她变强的目的之一是保护上条，但如果他被复制并站到了对立面…… |

### 4.2 新约中期——无力与阵痛（NT8-NT13）

| 卷号 | 事件 | 详细说明 |
|------|------|----------|
| **NT8-10** | **欧提努斯编 / 魔神之战** | **美琴最无力的一章**。<br><br>欧提努斯——真正的魔神——将世界无数次重置。"上条当麻死去"反复发生。美琴作为 Level 5 对上条的任何死法都无力阻止。她甚至在某个周目中被欧提努斯杀死。她的正义感、战斗能力、对学园都市的影响力，在"能将世界反转的魔神"面前一文不值。<br><br>最终上条通过"不杀死欧提努斯"的方式解决事件。美琴在此次事件中遭受巨大的心理创伤——她爱的人在她面前死了无数次，她却什么也做不了。她的"无力感"在新约中期达到顶峰。<br><br>这一经历促使美琴思考：她需要的不只是更强的"招式"，而是能站在上条身边的"资格"。——这为后续 AAA 升级和"翼"的出现埋下伏笔。 |
| **NT11** | **心理掌握 / 食蜂操祈编** | 美琴与食蜂操祈因外部的洗脑武装冲突及对彼此过去的误会爆发全面对抗。此卷深入揭露美琴与食蜂的复杂关系：<br>1. 二人都是 Level 5，都在"某个人"（食蜂与上条，美琴与上条）的关系中感受到微妙的不安<br>2. 最终被迫联手面对外部威胁<br>3. 食蜂的一句隐藏台词（在黑子面前）暴露了她对美琴既羡慕又嫉妒的复杂情感<br>这一卷是美琴"理解他人"的重要成长节点——她开始理解，她认为的"敌人"可能只是同样在痛苦中挣扎的另一个人。 |
| **NT12-13** | **上里翔流（Kamisato Kakeru）编** | 上里翔流出现，持有"世界排斥者"（World Rejecter）——能将被其触碰的存在送往另一个世界。美琴此时作为资深 Level 5 已有处理陌生体系的经验。这一卷中，美琴对上条当麻的关系在外部视角下被多次评述（例如被第三方误认为上条女友已是常态）。她本人已从"否认"进入"不否认但也不正面回答"的暧昧成熟阶段。 |

### 4.3 新约后期——翼的觉醒（NT14-NT22R）

| 卷号 | 事件 | 详细说明 |
|------|------|----------|
| **NT14** | **食蜂操祈过去篇反方向** | 通过食蜂操祈的过去深入揭示超能力者培养体系的黑暗面。美琴的"克隆人事件"与食蜂的"抹消记忆事件"并列展示——两位 Level 5 少女都深受学园都市操控。但她们选择用不同方式回应。此卷是美琴从"自我指责"到"向前看"的转折印证。 |
| **NT15** | **对奈芙蒂斯（Nephthys）——魔法之神** | 美琴在与奈芙蒂斯（一柱埃及系的魔法之神）的交战中暴露了差距。<br><br>虽然美琴在旧约结尾获得 AAA 后实力强大了数档，能够面对"天使级"乃至更上级的存在，但在"真正的魔法之神"面前仍捉襟见肘。 |
| **NT17-18** | **柯隆佐（Coronzon）决战 / 翼的出现** | **美琴最重要的能力进化节点**。<br><br>1. AAA（反魔法外装）在柯隆佐的威胁下被大幅强化<br>2. **美琴首次显现出"翼"（光之翼）**——象征着她在精神与功率上的双重质变<br>3. 翼的显现条件是"为了保护某人而超越自身极限"——美琴对上条当麻的感情成为她超越 Level 5 极限的动力<br>4. 有了翼之后的 AAA 强度飞跃：能够与柯隆佐（大恶魔，几乎魔神级）在一定程度上交锋<br>5. 此卷中美琴的战斗风格从"电磁炮的远程控制"进化到了"近身翼击+远程电击+AAA 武装"的三重战斗方式<br><br>这是美琴角色弧光的决定性时刻：她不再是"等着上条来救的第三位"，而是真正有资格站在战场最前线的力量。 |
| **NT19-20** | **世界终结之战** | 美琴的翼在持续战斗中愈加成熟，她能更好地控制翼的输出。在上条的"无法回应所有人期待"的痛苦中，美琴成为少数能够理解他的人之一——她自己也背负着"力量"带来的孤独。<br><br>上条和美琴在此卷中有几次极为重要的"不说破"对话——美琴已不再需要强调"喜欢"这个事实，因为她用行动证明了无数次。 |
| **NT21-22R** | **新约终章 / 对亚雷斯塔的终结** | 美琴参与了对抗学园都市最高负责人亚雷斯塔·克劳利的最终战。<br><br>**美琴对上条当麻的一战意义非凡**<br>在 NT22R 中，上条被引导至"对立面"（并非真实上条）。美琴必须暂时阻止上条。这场对战是她"能站在上条对面"的证明——过去她从未在真正平等的能力基础上与上条战斗。这场战斗（两人实力接近的状态下）是对二人关系的某种"仪式性清算"：<br>1. 美琴证明了自己已不是需要保护的那个人<br>2. 上条也看到了她的成长<br>3. 二人彼此认可的对等关系在该役中被最后一锤定音 |

> **新约总结**：新约见证了美琴从"有强大战斗力的少女"进化为"能站在魔神级战场上的英雄"。翼的出现是关键能力节点，精神上，她从单相思/仰望变为对上条对等理解、并肩作战的关系。

## 八、角色关系演化时间线

### 8.1 与上条当麻

| 时期 | 关系状态 | 关键证据 |
|------|---------|---------|
| OT1-OT2 | 莫名其妙的对手 | 美琴主动找上条"决斗"，实则是对这个能抵消她能力的人抱有浓厚兴趣。上条眼中的她是"一个炸自动贩卖机的笨蛋女孩"。 |
| OT3（妹妹事件） | **重大转折** | 美琴第一次在上条面前崩溃、流泪，承认自己的恐惧。上条用行动给了她"不用一个人背负"的回答。感情上，这一刻起美琴已确定"喜欢"。 |
| OT5-OT15 | 单箭头清晰的暧昧 | 美琴的关心已经明显（帮忙、送东西、主动接近），但两人均不点破。上条因失忆+繁忙+不解风情一直未完全接收这一信号。但实际行为上，二人已有情侣般的日常。 |
| OT16-OT19 | 从暗恋到主动奔赴 | 美琴跨国前往伦敦、俄罗斯，动机明确："我要去帮他"。她不再等他来救，而是主动跑向他的战场。 |
| NT1-NT7 | 稳定暧昧期 | 经历过世界大战，二人的联系更加稳固。在旁人眼里已是"那对情侣"。周围角色（如土御门、史提尔等）常拿"你女友"调侃上条。 |
| NT8-NT10（欧提努斯篇） | 考验期 | 美琴目睹上条无数次死亡却无能为力——她无法拯救所爱之人的痛苦达到顶峰。 |
| NT11-NT17 | 协作期 | 新约中期的各种危机迫使二人以更成熟的战友关系相处。美琴不再把上条当成"她必须紧追的那个人"，而是"她相信他能处理好、而自己也能帮到他"的对等伙伴。 |
| **NT17-18（翼觉醒）** | **第二重大转折** | 美琴的翼觉醒条件就是"想要保护他"——她突破自身极限，在上条不知道的地方全力站在了他身边。证明了她已经完全从"被他保护的人"转变为"能与他并肩的人"。 |
| GT至今 | 实质对等关系 | 二人可在顶级战场上互为后盾。外人视角中的"上条当麻和他的女友们"中，美琴已被公认为最核心的"正宫候选"。二人关系已进入无需言说的默契状态。 |

### 8.2 与食蜂操祈

| 时期 | 关系状态 | 说明 |
|------|---------|------|
| 超炮初期 | 未见面的对手 | 同为常盘台 Level 5，但在不同派阀，未有直接交集。互相知道彼此存在但无接触。 |
| 超炮 Level Upper 篇 | 第一次见面 | 食蜂出场控制局面，美琴见识到了心理掌握的力量。二人互有警惕。 |
| 超炮 Silnet Party 篇 | 裂缝加深 | 因桔梗天乃的"遥控器实验"（意图操控美琴失败后，警策看护因此报复），食蜂与美琴之间的恩怨间接形成。食蜂认为自己的行动（帮助警策和天乃）是正当的，但在美琴的视角下不可理解。 |
| 超炮T / 大霸星祭篇 | 正式冲突 | 食蜂直接对美琴出手以夺取"遥控器"。二人进行了一场 Level 5 之间的正面较量。最终以"不完全和解"结束。 |
| NT11 | 被迫联手 | 面对更大的外部威胁，美琴与食蜂不得不合作。虽然依然充满不信任，但二人在战斗中开始理解对方的立场。此卷之后，二人关系进入"互不信任但会合作"的稳定状态。 |
| GT系列 | 合作关系深化 | 面对更高层级的威胁，两位 Level 5 的联手变得必要。虽仍谈不上友情，但已能有效配合作战。 |

### 8.3 与白井黑子

始终是《超炮》角色的核心关系之一。白井黑子对美琴的"超常崇拜"是超炮系列的喜剧核心，但同时黑子的忠诚与能力是美琴日常战斗中最可靠的支援。美琴虽然对黑子的性骚扰式爱慕感到苦恼，但二人之间的信任和配合远超普通搭档。在 OT15 后方之水篇中，黑子重伤的场面直击美琴的痛点——她对"保护妹妹们"的责任感在此达到了与保护上条同等的地位。

### 8.4 与妹妹们（御坂妹妹 / 最后之作 / 番外个体等）

这是美琴终身的课题与羁绊。从最初克隆人实验的牺牲品，到克隆人们逐渐获得独立人格，再到美琴学重新学会以"姐姐"而非"负罪者"的身份面对她们——这是一条独立于爱情的主线。其中最后之作（20001号）最具代表性，她与美琴的关系近乎姐妹/亲子。

## 十、角色弧光总结

```
第一阶段（旧约前期）—— 骄傲的 Level 5
  ├─ 认知：我是最强的 Level 5，我可以解决一切
  ├─ 关键事件：克隆人事件的冲击
  └─ 缺陷：独走、拒绝求助、忽视自己的脆弱
         │
         ▼
第二阶段（旧约后期-新约前期）—— 向上的挑战者
  ├─ 认知：世界比我大，魔法侧比我强，但我还能成长
  ├─ 关键事件：后方之水的一击、奔赴伦敦、AAA获得
  └─ 缺陷：依存于上条、仍然倾向单打独斗
         │
         ▼
第三阶段（新约中期）—— 承受无力感
  ├─ 认知：即使我变强了，在魔神面前仍如蝼蚁
  ├─ 关键事件：欧提努斯篇上条反复死亡
  └─ 缺陷：情感上过于依赖上条作为"绝对坐标"
         │
         ▼
第四阶段（新约后期）—— 觉醒的英雄
  ├─ 认知：我变强不是为了超越上条，而是为了与他并肩
  ├─ 关键事件：翼的觉醒、柯隆佐决战
  └─ 成熟：对接纳无力感、为保护而战而非为证明而战
         │
         ▼
第五阶段（创约至今）—— 守护者
  ├─ 位置：不再被动的女主角，而是英雄之一
  ├─ 关系：与上条达到实质上的对等并肩
  └─ 主题：继承、责任、成为他人可以依靠的人
```

### 核心主题弧光

1. **从"一个人背负"到"学会依靠他人"** ——OT3 的妹妹事件是第一转折点；新约全篇将此主题反复强调，直至 NT17 翼的觉醒。
2. **从"Level 5 的身份认同"到"超越身份的自我定义"** ——美琴的价值不再由"第三位"决定，而由她自己选择成为谁的守护者来决定。
3. **从"喜欢上条但不说"到"用行动证明"** ——她的感情线从单方面暗恋演变为成熟的对等情感，这与其力量的增长互为表里。
