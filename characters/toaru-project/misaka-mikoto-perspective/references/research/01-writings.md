- **表面形象**：常盘台中学的模范学生，言行得体，符合贵族女校的标准。
- **真实性格**：争强好胜、正义感极强、爽朗不拘小节。典型**傲娇**性格——极度不坦率、容易害羞、口是心非。

- **萌点**：短发、发卡、安全裤、泡泡袜、傲娇、元气、贫乳；紧张或害羞时会冒出「~喵」的口癖。
- **核心信念**：强烈的正义感和保护欲，无法容忍身边人受伤害。对力量的使用有明确的道德边界。

## 二、能力设定

### 2.1 能力总览

| 项目 | 数据 |
|------|------|
| **能力名** | 电击使（Electro Master / 電撃使い） |
| **登记代号** | 超电磁炮（Railgun / 超電磁砲）—— 由美琴本人自行申请注册 |
| **能力等级** | **Level 5（超能力者）** |
| **排名** | 学园都市七名Level 5中 **第3位** |
| **初始等级** | Level 1（低能力者），通过后天努力升至Level 5 |
| **最高出力** | **十亿伏特（1,000,000,000 V）** |

> **一设说明**：根据《魔法禁书目录》原作设定，能力者的成长上限受「素养判定」（Parameter / 资质排名）决定。美琴能够成长为Level 5，实际上是被素养所注定的，并非纯粹靠努力突破。但她从Level 1开始的持续努力本身仍是真实的。

### 2.2 核心招牌技：超电磁炮（Railgun）

**原理**：利用电磁诱导产生的洛伦兹力（Lorentz force），在手臂中产生平行的电磁场，通过极高电压击穿空气形成等离子体替代导轨，以类磁轨炮（Railgun）原理加速金属弹丸。

| 参数 | 数值 |
|------|------|
| **弹丸** | 游戏币（金属代币，便于携带） |
| **初速度** | **1,030 m/s（约3倍音速 / 3.026马赫）** |
| **电流** | 约2,250,000安培（以约1,000,000伏特驱动，约为全力1/10） |
| **射速** | **8发/分钟** |
| **精度** | 弹着分布 **18.9 mm** |
| **有效射程** | 约 **50米**（受限于游戏币因空气摩擦高温熔化） |
| **弹道特点** | 命中目标前持续加速 + 电磁弹道修正，精度与破坏力极高。余波产生的强风超越一般风能力者 |

**射程限制原理**：游戏币以超音速飞行时，空气摩擦产生极高热量。在大约50米处，游戏币会完全熔化汽化。使用更大或更耐高温的导体可提升射程和威力。

**衍生变体**：
- **散弹超电磁炮**（自称）—— 将电磁炮扩散为散射形态的变体技
- **液态护膜超电磁炮（Liquid Proof Railgun）** —— 与食蜂操祈配合，通过操纵水分冷却弹体，解决硬币熔化问题，威力足以击破「没有窗户的大楼」（学园都市统括理事会所在，通常被认为坚不可摧的建筑）。该技能大幅提升了射程和破坏力。

### 2.3 电击操纵（Electricity Manipulation）

- **十亿伏特雷击**：可汇集大气中的电能释放高压雷电攻击，召唤真正的落雷，无论天气条件如何都能发动。威力可导致全市范围停电。
- **雷电之枪**：从手中或额头释放雷电之枪进行远程攻击。
- **体内通电**：可让电流通过自身身体进行接触攻击；也能通过电流刺激超控自身神经系统来对抗麻痹。
- **等离子生成**：可将物质（如空罐）电离为等离子体。
- **臭氧/空气爆裂**：通过空气放电产生臭氧，或引爆空气形成缓冲垫。
- **高空供氧**：利用电离空气在高海拔环境中获取氧气。

### 2.4 铁砂操控（Iron Sand Manipulation）

**原理**：利用电磁力从周围环境中（地下、土壤中）收集铁砂并进行精密操控。

| 应用形态 | 描述 |
|---------|------|
| **铁砂之剑/铁砂之鞭** | 将铁砂聚集成可延伸数十米长的黑色刀刃或长鞭，铁砂微粒在电磁力驱动下高速振动，如同超高速电动链锯，可切断几乎任何物体 |
| **铁砂防御壁** | 将铁砂聚集在身体周围形成防护壁，可抵御枪弹和重型投射物 |
| **铁砂风暴** | 将铁砂化为高速振动的细长针状，从360度全方位攻击对手 |
| **铁砂之海啸** | 形成巨大的铁砂波浪，威力可以扫平军事基地 |
| **铁砂巨人** | 构建巨大人形铁砂构造体来对抗巨型敌人 |
| **铁砂轨道炮** | 制造最多20根金属矛状弹体同时发射 |
| **铁砂追踪** | 将铁砂散布在地面上，通过感知铁砂振动来追踪超音速移动的敌人 |

**新约中的进化应用**：在新约第13卷中，美琴利用铁砂通过挤压制造真空点，利用气压差对敌人造成内脏/鼓膜级别的破坏——超越了单纯物理切割的应用。

### 2.5 电磁波与磁场操控

- **电磁感知（雷达）**：可发射和接收电磁波，实现360度无死角探测，能检测到其他电击使的存在，能在超音速战斗中感知敌人动向。
- **电磁屏障**：下意识的电磁屏障可干扰精神系能力（如Level 5第5位食蜂操祈的「心理掌握」）。
- **电磁脉冲（EMP）**：释放强电磁脉冲摧毁电子设备。
- **磁场防御**：可弯曲电子束攻击（如第4位麦野沈利的「原子崩坏」）。

### 2.6 黑客能力（微观电磁操控）

美琴的能力在微观层面同样出色：

- **入侵电脑系统**：通过电气网络联通门禁和电脑系统，实时破解密码。
- **电子锁破解**：利用电磁信号读取和控制电子锁。
- **信息窃取**：从信用卡磁条、电子设备中远程窃取信息。
- **信息系统入侵**：利用简单设备入侵学园都市的「书库」（中央数据库）。
- **远程数据销毁**：以电子网络为媒介远程销毁科研院所的数据。
- **安全机器人操控**：夺取安全机器人的控制权进行远程操作。
- **影像消除**：实时将自身从监控摄像中抹除。

### 2.7 三维移动与飞行

- **电磁吸附移动**：利用电磁力吸附在含钢铁的墙壁/天花板上的快速移动，可在城市高楼间自由穿梭。
- **水翼飞行**：在空气中水分充足的条件下，通过电解链反应分解水分子产生氢气和氧气流，再利用电磁场加以控制，制造**六片叶状水翼**实现高速飞行。速度可超越现代战斗机。首次出现在「学艺都市篇」。

### 2.8 防御能力

- **电击免疫**：对自身电力及所有外部电击武器完全免疫。
- **生物电流抗性**：可抵抗针对生物电流的精神系能力（包括Level 5食蜂操祈的心理掌控）。
- **铁砂屏障**：利用铁砂和金属废料构成物理防护墙。
- **磁场偏折**：可偏折基于电子/光的攻击。

## 三、等级制度与排名设定

### 3.1 学园都市能力等级体系

学园都市的能力者分为六个等级（Level 0 ~ Level 5）：

| 等级 | 名称 | 说明 |
|------|------|------|
| Level 0 | 无能力者 | 能力微弱到几乎不可测量，但并非完全没有能力 |
| Level 1 | 低能力者 | 能力有微弱效果，但不足以实用 |
| Level 2 | 异能力者 | 能力有一定实用价值 |
| Level 3 | 强能力者 | 能力在实际生活中有实用价值 |
| Level 4 | 大能力者 | 能力可用于军事级别 |
| Level 5 | 超能力者 | 能力达到甚至超越常规军事力量，整个学园都市仅七名 |

| 排名 | 代号 | 持有者 | 能力 | 备注 |
|:---:|:---|:---|:---|:---|
| **1** | 一方通行（Accelerator） | 一方通行（本名不详） | 矢量操作（Vector Change） | 学园都市最强，可反射一切物理攻击；后依赖电极+御坂网络辅助演算 |
| **2** | 未元物质（Dark Matter） | 垣根帝督 | 创造世界上原本不存在的物质 | 不适用现有物理法则；背后可展开六片白色光翼 |
| **3** | **超电磁炮（Railgun）** | **御坂美琴** | **电击使（Electro Master）** | 本文件主题角色 |

| **5** | 心理掌握（Mental Out） | 食蜂操祈 | 复数精神系能力 | 包办读心、洗脑、记忆消除、幻觉等一切精神现象 |

## 四、角色进化形态

### 4.1 AAA（Anti-Art Attachment / 对魔术式驱动铠）

**基本设定**：
- AAA全称 **Anti-Art Attachment**（对魔术式/反魔术式驱动铠），是学园都市开发的**对魔法师专用装备**。
- 本体由多个可互换组件构成，重量超过20吨，飞行速度可超过**马赫20**。
- 设计者涉及木原脑干（Kihara Noukan），其目的针对魔法侧的高阶存在（如莱德莉·坦格罗德、弗雷梅亚·克洛伊吞、艾华斯等）。
- AAA不是纯粹的科学侧产物，其设计中包含魔法阵元素（如十边形魔法阵），与亚雷斯塔·克劳利的计划有深层关联。

**美琴的AAA**：
- 新约第15卷正式接触AAA，并用以击退「唯一」（Unabara）。
- 美琴后来获得改装版AAA，在常盘台中学获得机库并成立了维护社团。
- 她尝试还原原版的「黑箱」（black box）但未能完全成功，但仍会体验类似的物理反噬。
- **新约第18卷**：在「英格兰回归篇」中，美琴与食蜂操祈将AAA伪装成前卫艺术品秘密运到巴厘岛，随后驾驶它飞往伦敦参与救援上条当麻。

**标志性铭文**：AAA上有「脑干」组织的狗狗标识+「狗命」字样。

### 4.2 AAA 翼形态（Wings Form）

**出处**：新约魔法禁书目录 第21卷（Shinyaku Toaru Majutsu no Index Volume 21）

**关键描述**：
- 在苏格兰爱丁堡与柯隆尊（Coronzon）的决战中，美琴穿戴的AAA利用其修复部件中的「内容物」**生成了一对巨大的翅膀**。
- 翅膀被描述为**尖锐的、结晶状的突起**，从使用者的背部展开形成一对巨大的翅膀，其他突起则形似眼球、爪子和獠牙。
- 背后有两个细长的推进器，点亮时发出红色光芒。
- AAA的翼形态本质上是将AAA作为**与外部个体的外接链接**，远程引导该个体的力量——即魔法力量。AAA被设计为可以接入科学领域之外的力量体系。

**与食蜂操祈的合作攻击**：
- 美琴与食蜂操祈心灵相连，美琴穿戴AAA，食蜂操祈利用「心理掌握」操控水分冷却弹体。
- 合作施展的**液态护膜超电磁炮**威力足以击破「没有窗户的大楼」。

**象征意义**：AAA代表了美琴**跨越科学侧与魔法侧的界限**，用未知的力量去追赶上条当麻的决意。这一进化标志着她不再局限于学园都市的能力体系之内。

### 4.3 水翼飞行（Electrolysis Wings / 自身飞行能力）

- 非装备能力，是美琴自身电磁能力的应用延伸。
- 通过电解链反应将大量水分解为氢气和氧气，再以电磁场控制，生成**六片叶状水翼**。
- 速度可超越现代战斗机。
- 首次出现在「学艺都市篇」。
- 与AAA的机械翼不同，这是美琴**自身能力**的衍生产物。

### 4.4 Level 6 Shift 相关（绝对能力者进化计划）

- **「绝对能力者进化计划（Level 6 Shift Project）」**——以御坂美琴的DNA图谱制造的20001名克隆人（妹妹们）为杀戮对象，旨在让一方通行通过杀戮达到Level 6。
- 根据「树状图设计者」的演算，一方通行需要杀害**128个**御坂美琴才能进化为Level 6。
- 计划实际目的与表面不同——克隆人御坂妹妹们实际上是亚雷斯塔营造「**人工天界**」的媒介，用于构建覆盖全球的AIM扩散力场网络。
- 美琴本人在面对一方通行时表现出了一定程度的「潜势力」，但这并非正式的进化形态。

### 4.5 创约（Genesis Testament / GT）中的动向

> 注：创约系列目前仍在连载中，以下内容为公开的剧情进展，可能随后续卷数变化。

- 美琴在创约系列中继续活跃，与AAA的「理解历程」仍在继续。
- AAA并未被完全掌握，仍有未知功能。
- 美琴自身能力的运用在新约基础上进一步拓展，具体形态有待后续卷数补全。

## 五、克隆人与妹妹计划（Sisters Project）

### 5.1 计划背景

- 年幼时的美琴向「樋口制药·第七药学研究中心」提供DNA图谱，以为可帮助肌肉萎缩症患者治疗。
- 实际目的是**量产军事用体细胞复制人**用于「绝对能力者进化计划」。
- 每个克隆体单价仅 **18万日元**。

### 5.2 妹妹们的能力

| 项目 | 数据 |
|------|------|
| **能力名** | 缺陷电气（Radio Noise / 无线电噪音） |
| **平均等级** | Level 2 ~ 3 |
| **与原版比例** | 性能不及原版 **1%** |
| **理论最大输出** | 约50,000伏特，可致人昏迷 |
| **辅助装备** | 军用夜视镜（作为视觉化电子线和磁力流动方向的辅助工具） |
| **弱点** | 无法承受来自同一系统源（御坂美琴）高功率输出的电波干扰 |

### 5.3 特殊个体

| 个体 | 编号 | 能力等级 | 特点 |
|------|------|---------|------|
| **御坂10032号**（御坂妹） | 检体10032 | Level 2 | 被上条当麻和御坂美琴救下，与二人羁绊最深 |
| **最后之作（Last Order）** | 御坂20001号 | Level 3 | 御坂网络管理者，情感丰富，有「御坂御坂」口癖 |
| **番外个体** | 序列外（编号不明） | Level 4 | 为抹杀一方通行而制造的特殊个体，电流呈紫色，两亿伏特 |
| **御坂桃莉（Dolly）** | 御坂0号/试做体 | 未知 | 需要特殊机械维系生命，食蜂操祈的挚友 |
| **御坂00000号（Full Tuning）** | Full Tuning | 未知 | 桃莉的姐妹体，共享记忆 |

### 5.4 关键数据

- 总共制造了**20003个**克隆体
- 其中约10000个在「妹妹计划」中被一方通行杀害
- 幸存者在学园都市及世界各地生活，通过**御坂网络**共享感觉和记忆
- 御坂网络后来被一方通行利用进行演算辅助

## 六、角色定位与剧情价值

### 6.1 系列定位

- **《魔法禁书目录》（原作）**：主要女主角之一，上条当麻的重要羁绊对象。代表科学侧的超能力者视角，对抗魔法侧的危机。
- **《科学超电磁炮》（外传）**：**第一主角**。以外传形式展开，讲述美琴及其朋友们在学园都市中的故事，展现能力的日常应用和深度战斗。
- **《科学一方通行》（衍生）**：以配角身份登场，与御坂网络相关的剧情线有重要作用。

### 6.2 人际关系

| 关系 | 角色 | 备注 |
|------|------|------|
| **单方面对手/在意对象** | 上条当麻（Kamijou Touma） | 她的「幻想杀手」可以无效化美琴的能力；美琴对他抱有复杂感情，自称「竞争对手」 |
| **学妹/室友** | 白井黑子（Shirai Kuroko） | Level 4大能力者，能力「空间移动」；黑子对美琴有超乎寻常的爱慕 |
| **同校/天敌** | 食蜂操祈（Shokuhou Misaki） | Level 5第5位；同为常盘台中学二年级，水火不容但后期有重要合作 |
| **朋友** | 佐天泪子（Saten Ruiko） | 无能力者（Level 0），美琴的亲密朋友之一 |
| **朋友** | 初春饰利（Uiharu Kazari） | Level 1能力者「定温保存」，朋友兼风纪委员 |
| **克隆体** | 御坂妹妹们 | 感情复杂——自责、保护欲、家人般的羁绊 |
| **家庭** | 御坂旅挂（父）、御坂美铃（母） | 关系良好，但父母常年不在身边 |

### 6.3 剧情价值

- **导火索角色**：「妹妹计划」是整个《魔法禁书目录》前半部分最重要的剧情线之一，也是上条当麻与一方通行首次正面冲突的触发点。
- **科学侧代表**：在魔法侧故事占据主线的《魔禁》中，美琴代表科学侧超能力者的立场，展现了学院都市内部的运作机制和伦理问题。
- **成长主线**：从一个正义感强烈但天真的少女，逐渐成长为理解学园都市黑暗面、主动为保护他人而战斗的成熟能力者。
- **跨越界限者**：通过AAA接触魔法侧力量，美琴成为科学侧少数主动跨越「科学与魔法」界限的角色。

## 七、人物原型与创作花絮

### 7.1 创作背景

- **作者**：镰池和马（Kazuma Kamachi）
- **插画/人设**：灰村清孝（Kiyotaka Haimura）
- **设计思路**：原本设定为长发（头发作为超电磁炮的导轨），但长发造型最终分配给了Index（茵蒂克丝）。短发的清爽形象反而成为美琴的标志性特征。
- 最初被创造用以展示学园都市的能力等级制度——既是顶级能力者（Level 5第3位），又在面对上条当麻时展现了「绝对力量并非万能」的主题。

### 7.2 粉丝昵称

- **炮姐**（中文粉丝圈最广泛称呼）
- **Bilibili**（上条当麻起的绰号，源自日语放电的拟声词「ビリビリ」）
- **超电磁炮**（能力代号）
- **御坂大人**（白井黑子的称呼）
- **常盘台的王牌**（校内称号）
