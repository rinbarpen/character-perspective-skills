## 2. 能力设定详解

### 2.1 幻想杀手 (Imagine Breaker / 幻想殺し)

#### 2.1.1 基本定义

| 项目 | 内容 |
|------|------|
| **位置** | 右手手腕到指尖（范围极小） |
| **性质** | 既非超能力也非魔法，无法用学园都市System Scan检测 |
| **等级判定** | 因无法检测被归为Level 0（无能力者） |
| **一设定位** | **「世界的基准点」「世界的修复点」** |
| **标志性台词** | 「幻想を、ぶち壊す！」 |

> **核心一设**：幻想杀手不是学园都市开发的能力，也无法在任何已知魔道书中找到记载。（出处：旧约1卷）

#### 2.1.2 能力效果

| 效果类别 | 说明 | 出处 |
|----------|------|------|
| **异能消除** | 接触即可无条件消除一切超自然力量（超能力、魔法、奇迹、祝福、诅咒等） | 旧约1卷 |
| **无差别发动** | 不受宿主意识控制，善恶不分——连回复术式、防御术式、念话能力也一并消除 | 旧约2卷 |
| **对地脉有效** | 可消除地脉/龙脉上的异常，但消除后周围会自然补充 | 新约卷 |
| **对魔神有效** | 可消除魔神级存在对世界的扭曲（如欧提努斯的相位操作） | 新约9-10卷 |
| **修复世界** | 本质不是「消灭」而是「还原」——将扭曲的世界恢复原状 | 新约卷 |

#### 2.1.3 本质揭露（按小说进展逐步揭示）

##### 第一阶段：「不明能力」（旧约1-16卷）
- 仅知其可消除异能，无法被学园都市仪器检测
- 被亚雷斯塔密切关注，列为DEFCON 3级危险
- 右方之火（神之右席）暗示此能力未发挥真正性能

##### 第二阶段：「世界的修复点」（旧约17-22卷）
- 欧雷尔斯（Ollerus）提出：幻想杀手是**「所有魔法师的希望与畏惧集结而成的东西」**
  - **希望**：无论世界被扭曲到何种地步，都能恢复原状
  - **恐惧**：所有因魔法产生的扭曲都可被消除
- 亚雷斯塔指出幻想杀手与**「神净」**（神浄）有关

##### 第三阶段：「魔神愿望的集合体」（新约9-13卷）
- 僧正（Magic God）明确指出：上条当麻本身比幻想杀手更加特殊
  - 当麻是**综观世界的计分者**（世界を俯瞰する採点者）
  - 幻杀手是**被他的本质吸引而来的附属品**
- 欧提努斯确认：幻想杀手是魔法师们为应对魔神失控而准备的**「保险丝」**

##### 第四阶段：「超越十字教框架」（旧约22卷）
- 亚雷斯塔直言：仅以**「十字教的程度」**无法解释神净讨魔的存在
  - 暗示当麻的本质位格超越了十字教神学体系
  - 与三大宗教的最高秘密并列，但超越之

##### 第五阶段：「创造/选择」（创约卷）
- 在创约阶段，当麻的右手能力被进一步揭示为**「治愈世界的万灵药」**
- 龙王模式确认：当麻的本质属于**「幻想侧」**，既非科学侧也非魔法侧

#### 2.1.4 历史渊源——布莱斯路之战（Blythe Road）

| 时间 | 事件 | 说明 |
|------|------|------|
| 约1900年前 | 以某位圣者的右手为材料，制成终极驱魔灵装「幻想杀手」 | 一设 |
| 1900年 | **布莱斯路之战**（Battle of Blythe Road）：黄金黎明内部分裂 | 一设，结合历史真实事件 |
| 战后 | 灵装被折断/摧毁，"消失"在历史中 | 一设 |
| 现代 | 力量抵达上条当麻处，成为其右手 | 一设 |

> **关键一设**：在1900年的布莱斯路之战中，黄金黎明组织在36 Blythe Road的Isis-Urania神殿保管着由圣者右手制成的箭形灵装。亚雷斯塔·克劳利利用此事件盗取该灵装，后灵装在战斗中被毁，其力量辗转最终寄宿于当麻右手中。
>
> **真实历史对应**：1900年4月，亚雷斯塔·克劳利确实闯入布来斯路36号的黄金黎明神殿，与诗人叶芝（W.B. Yeats）等成员对峙——史称「布莱斯路之战」。镰池和马将此事件融入了世界观。

### 2.2 关于「等级」的定义矛盾

| 来源 | 说法 | 可信度 |
|------|------|--------|
| 学园都市官方 | Level 0（无能力者） | 一设（但检测手段无法识别） |
| 亚雷斯塔 | 既非超能力者也非魔法师——「拥有独特能力的普通人」 | 一设（最高权威） |
| 部分读者/二设 | 「最强的无能力者」 | 民间说法 |
| 实际表现 | 可击败Level 5第一位一方通行 | 一设事实 |

> **一设明确**：幻想杀手不散发AIM扩散力场（与所有超能力者和原石不同），因此从科学侧角度确实判定为无能力。这一定位使当麻在政治层面上能自由介入魔法侧事务而不引发学园都市的舆论问题。——出处：亚雷斯塔视角相关章节

## 3. 隐藏力量：右手中的多重存在

> **核心概念**：上条当麻的右手并非只有幻想杀手一种力量。目前已知存在**三层结构**——「幻想杀手」「里幻（龙）」「神净讨魔」——三者层层包裹，彼此关联但性质不同。

### 3.1 三层结构总览

| 层级 | 名称 | 性质 | 表现 | 被压制者 |
|------|------|------|------|----------|
| **表层** | 幻想杀手（Imagine Breaker） | 异能消除/恢复基准点 | 右手常态能力 | — |
| **中层** | 里幻/龙之力（Invisible Thing） | 无色漩涡/多头龙/无形之物 | 右手被切断时显现 | 被幻想杀手压制 |
| **核心** | 神净讨魔（Kamijou no Ken） | 当麻的灵魂本质/世界免疫系统 | 人格化出现（新约22R） | 被幻想杀手压制 |

### 3.2 龙王之颚 / 龙首（Dragon Strike / 竜王の顎）

首次出现于旧约2卷（动画第1季对应章节），当右臂被切断时从断口显现。

#### 3.2.1 已知出现记录

| 作品卷号 | 触发条件 | 表现形态 | 备注 |
|----------|----------|----------|------|
| 旧约2卷 | 奥雷欧斯·伊扎德使用金色大衍术切断右臂 | **巨大透明龙王之首**（单头） | 具有精神干涉/记忆破坏能力 |
| 旧约SP/超炮外传 | 御坂美琴失控力量削断右手 | **八具龙首** | 共同啃噬接近Level 6的力量 |
| 新约22R | 右手被夺 | **无色漩涡/人格化神净讨魔** | 身穿晚礼服，与当麻对峙 |
| 创约9卷 | 当麻骗美琴用超电磁炮射断右臂 | **全力全开龙王模式**（17龙合一） | 本卷限定模式 |

#### 3.2.2 八龙属性（超炮外传中共出现八具龙首，各具属性）

| 编号 | 名称 | 属性/能力 |
|------|------|-----------|
| 1 | 最初之龙（First Dragon） | 擅长解咒与精神攻击，被咬到导致记忆破坏 |
| 2 | 盲龙（Blind Dragon） | 黑暗属性，强烈精神干涉，令对手陷入恐怖与混乱 |
| 3 | 单眼龙（Cyclops Dragon） | 眼镜蛇外形，水属性，拥有浓缩毒液的毒牙 |
| 4 | 四眼龙（Four-Eyed Dragon） | 混淆梦境与现实，催眠能力，歌声可击碎金属 |
| 5 | 骸炎龙（Skeletal Flame Dragon） | 火属性不死之龙，可燃烧目标生命力 |
| 6 | 冰晶龙（Ice Crystal Dragon） | 吐息冰气，**硬度堪比行星级别** |
| 7 | 枪头龙（Spearhead Dragon） | 浑身金光，放出雷击与镭射光 |
| 8 | 天使龙（Angel Dragon） | 头顶天使羽毛的神圣之龙 |

> **设定矛盾点**：关于龙的数量存在不同记载。
> - 旧约2卷仅出现**1头**龙首（龙王之颚）
> - 超炮外传/旧约SP中出现**8头**（八龙）
> - 部分资料提及有**17条龙**的设定（创约9相关讨论）
> - 推测：数量可能随当麻的成长/解封程度递增，非连续性错误

### 3.3 无形之物（Invisible Thing）

当幻想杀手被强行从右手剥离时，断口处出现的不明力量。

| 特征 | 描述 |
|------|------|
| **外观** | 不可视的能量漩涡/压缩力量团 |
| **初次** | 右方之火夺走右手后，从当麻断肩处显现 |
| **效果** | 弹开右方之火的「神圣之右」（Holy Right）攻击 |
| **恐怖性** | 右方之火（拥有行星级力量）感到**真实恐惧** |
| **后续** | 当麻捏碎该力量并长出新的右手，幻想杀手恢复 |
| **暗示** | **只有上条当麻这名少年身上的右手，才存在真正的力量** |

### 3.4 神净讨魔（Kamijou no Ken / 神浄討魔）

#### 3.4.1 基本定义

| 项目 | 内容 |
|------|------|
| **发音** | Kamijou Touma（与"上条当麻"同音） |
| **首次正式提及** | 旧约22卷，亚雷斯塔之口 |
| **本质** | 当麻的**灵魂本身**，是世界的计分者/治愈世界的万灵药 |
| **定位** | 超越十字教框架的存在 |
| **表现** | 新约22R人格化、创约9龙王模式 |

#### 3.4.2 核心一设引用

> **僧正（新约13卷）**：
> 「全世界魔法师的愿望（幻想杀手）不过是受到上条当麻的灵魂本质（神净讨魔）吸引而来的外物。」

> **亚雷斯塔（旧约22卷）**：
> 「仅以十字教的程度，无法解释神净讨魔的存在。」

> **创约9卷旁白**：
> 「神净讨魔是治愈世界的万灵药……平时沉睡，一旦出现『侵蚀世界的疾病』就会现身。」

#### 3.4.3 神净的两个侧面

| 侧面 | 描述 |
|------|------|
| **「神净」**（净化） | 温柔地治愈世界不好的部分 |
| **「讨魔」**（讨伐） | 冷酷地切除/消灭世界不好的部分 |

> 意象关联：**「龙对财宝进行守护，横断整个善恶二元论」**——神净具有超越善恶二元对立的存在论地位。出处：相关章节旁白。

#### 3.4.4 人格化神净讨魔（新约22R）

| 特征 | 描述 |
|------|------|
| **外貌** | 与当麻相同相貌，但穿晚礼服 |
| **眼部特征** | 眼梢嘴角有艳粉/翠绿色光芒 |
| **记忆** | 拥有当麻全部记忆（包括失忆前部分） |
| **性格** | 冷酷务实，认为「只要所有人都不质疑乖乖顺从就能解决一切」 |
| **结局** | 被当麻本体亲手击败并回归体内 |

#### 3.4.5 全力全开龙王模式（创约9卷）

| 项目 | 内容 |
|------|------|
| **触发** | 当麻骗美琴用超电磁炮射断右臂 |
| **本质** | 释放神净讨魔的力量，17龙融合为一 |
| **性质** | **本卷限定**模式（作者后记注明） |
| **意义** | 当麻放弃与对方「建立联系」的可能性，拒绝沟通后第一次获得的力量 |
| **代价** | 逐渐向「幻想侧」坠落，人性流失，身体被龙侵蚀 |

**能力细目**：

| 能力 | 表现 |
|------|------|
| 空间弯曲 | 踏一步即可让世界变形、空间沉没；用力踩可引发重力崩坏 |
| 龙首咬合 | 超高速物理攻击，秒杀级 |
| 空间吞噬流弹 | 不被当麻"允许"的攻击自动被空间吞没，无流弹伤及无辜 |
| 光之龙息 | 连100米以上空间一并切断，被切断的建筑会像史莱姆般自动复原 |
| 精神AOE | 龙首不规则开闭发出声音，令人心生恐惧无法动弹 |
| 变色龙隐身 | 融入环境 |
| 伤害转移 | 头被打爆后可转移伤害至龙尾并复原（一次保命） |
| 气象操控 | 蝙蝠翼、空气炮、冻结、紫电 |
| 电子龙闪 | 吞噬磁电管后释放3500°C闪光 |
| 耶梦加得级止血 | 以世界之蛇的重量强行止血 |
| **世界服从** | 无机物会主动避让——**世界本身服从于当麻** |

### 3.5 幻想杀手 vs 理想放逐（World Rejecter / 理想送り）

| 项目 | 幻想杀手 (Imagine Breaker) | 理想放逐 (World Rejecter) |
|------|---------------------------|---------------------------|
| **持有者** | 上条当麻 | 上里翔流（后被木原唯一夺走，已夺回） |
| **本质** | 修复、复原、留恋这个世界 | 抛弃、放弃、离开这个世界 |
| **来源** | 全体魔法师愿望的结晶（留在原有世界） | 魔神对上条失望后产生的愿望（前往新世界） |
| **效果** | 消除异能，修复世界扭曲 | 放逐到同一时间轴的剩余领域（新天地） |
| **触发** | 右手接触 | 手臂影子决定范围 |
| **关系** | 原本是**同一力量的分裂** | 后者是前者的"反面" |

> **一设背景**：当麻成为欧提努斯一人的理解者后，魔神们对现世失望，开始寻找幻想杀手的替代品。这一「放弃现有世界」的愿望汇聚到上里翔流的右手，形成了理想放逐。

## 4. 角色定位与主题

### 4.1 叙事定位

| 维度 | 定位 | 说明 |
|------|------|------|
| **故事核心** | 主人公 | 全系列（旧约、新约、创约）的主角 |
| **能力定位** | 异能之力的终结者 | 不是最强的能力者，而是所有异常现象的抹消者 |
| **阵营归属** | **幻想侧** | 既非科学侧也非魔法侧（创约确认） |
| **政治身份** | 学园都市高中生 | 名义上属于科学侧，便于两方介入 |
| **亚雷斯塔评价** | 影响其计划**97-98%**的关键变量 | DEFCON 3级 |

### 4.2 主题关键词

| 主题 | 表现 |
|------|------|
| **拯救与牺牲** | 从旧约到创约，持续面对「救一人还是救世界」的选择 |
| **失去与失忆** | 旧约1卷即失去全部记忆，后续不断面对失去的自我 |
| **伪善与真善** | 自嘲为伪善者，但行为证明本质善良 |
| **理解与沟通** | 拒绝用暴力解决问题，坚持与敌人对话（如欧提努斯） |
| **普通与非凡** | 最「普通」的高中生却背负拯救世界的命运 |

### 4.3 失忆设定

| 项目 | 内容 |
|------|------|
| **发生时间** | 旧约1卷结尾 |
| **原因** | 为保护茵蒂克丝，消除其记忆 |
| **程度** | **全部记忆消失**（仅保留右手能力的使用方法等身体记忆） |
| **知情者** | 主治医生「冥土追魂」、偶然得知的御坂美琴 |
| **意义** | 失忆前的当麻和失忆后的当麻本质上是不同人格（新约22R中人格化神净持有失忆前记忆可佐证） |

> **深度分析**：失忆设定并非单纯的剧情工具。它使当麻在角色弧光中始终处于「寻找自我」和「重新定义自己」的状态。新约22R中人格化神净讨魔拥有失忆前记忆且性格不同，暗示失忆前的当麻可能与现在的当麻存在明显差异。

## 5. 与其他角色的能力对比

### 5.1 能力类型对比表

| 角色 | 能力类型 | 效果范围 | 攻击手段 | 对幻杀相性 |
|------|----------|----------|----------|------------|
| **上条当麻** | 异能消除（幻想杀手） | 仅右手接触 | 近战拳击+右拳 | — |
| **一方通行** | 矢量操作（Level 5 No.1） | 全身被动/范围主动 | 矢量操控万物 | 幻杀可穿透反射，但物理二次现象无法消除 |
| **御坂美琴** | 电磁操纵（Level 5 No.3） | 周身电磁场 | 超电磁炮、铁砂、雷电 | 直接攻击可消除，但物理投掷物无效 |
| **茵蒂克丝** | 禁书目录（魔道书10万3000本） | 强制咏唱/约翰之笔 | 龙王之杀（龙杀） | 龙杀过快时幻杀来不及完全消除 |
| **欧提努斯** | 魔神（相位操作） | 宇宙级 | 世界扭曲、Gungnir | 幻杀可抵消但持续承受有极限 |
| **右方之火** | 神圣之右（神之右席） | 行星级 | 神之右击 | 未完全觉醒时被当麻击败 |
| **上里翔流** | 理想放逐 | 以手臂影子范围 | 放逐异能 | 与幻杀同源对立 |

### 5.2 对上条当麻vs一方通行的深度分析

两人共计**三次正面交手，一方通行全部战败**：

| 交手 | 卷号 | 战况 | 胜负因素 |
|------|------|------|----------|
| 第1次 | 旧约3卷 | 当麻在先被美琴大招消耗后，两拳击倒一方通行 | 幻杀穿透反射+意志力 |
| 第2次 | 旧约20卷 | 一方通行开启黑翼，当麻利用前兆感知+战术打晕 | 战术意识+不屈 |
| 第3次 | 新约13卷 | 一方通行白翼全开，仍被当麻击败 | 作者设定意图明确（此战有放水争议） |

**综合结论**：

| 维度 | 优势方 |
|------|--------|
| **纯粹破坏力** | 一方通行（宇宙结构级） |
| **一对一实战** | 上条当麻（三次全胜） |
| **能力上限** | 一方通行（射程、泛用性、成长性更优） |
| **意志/精神** | 上条当麻（承受5000亿次死亡轮回仍坚持本心） |
| **救赎能力** | 上条当麻（核心能力是"救赎"而非"破坏"） |

> **一句话总结**：一方通行是「最强的能力者」，上条当麻是「异能之力的终结者」——一个是破坏力的极致，一个是所有异常现象的抹消者。两者擅长的领域本质不同。

## 6. 能力限制与弱点

### 6.1 幻想杀手自身局限

| 限制 | 详细说明 | 出处 |
|------|----------|------|
| **范围极小** | 仅右手手腕至指尖，身体其他部位被击中会受伤 | 旧约1卷 |
| **无法消除二次物理现象** | 火球可以消除，但火球炸起的石块/水泥无法消除 | 旧约1卷（对史提尔之战关键） |
| **无法消除过量能量** | 能量过于庞大时无法瞬间完全消除 | 新约9卷（对欧提努斯） |
| **无法应对无媒介攻击** | 看不见摸不着的攻击无法预先打消 | 旧约卷 |
| **可被再生术式克制** | 如史提尔的「猎杀魔女之王」不破坏核心符文就会无限再生 | 旧约卷 |
| **自然现象无法消除** | 地脉、生命力、灵魂等自然均匀现象无法消除 | 新约卷 |

### 6.2 对大型攻击的应对极限

以下攻击曾突破或压制幻想杀手的消除速度：

| 攻击 | 出自 | 情况 |
|------|------|------|
| 茵蒂克丝「龙王之杀」 | 旧约卷 | 速度过快，消除不及，对拼至手腕流血 |
| 史提尔「猎杀魔女之王」 | 旧约卷 | 无限再生，需破坏核心符文 |
| 一方通行「黑翼」 | 旧约20卷 | 体积过大无法瞬间消除，只能抓住并扭转方向 |
| 欧提努斯「弩箭」 | 新约9卷 | 每次消除产生反作用力反弹自身 |
| 柯隆佐「魔剑· flaming sword」 | 新约22R | 魔法仪式级攻击，消除窗口极短 |
| 卡特娜「原版次元切断」 | 旧约相关卷 | 需在1.25秒内消除，否则成为已完成的"正常"现象 |

### 6.3 时间窗口设定——卡特娜原版的特殊机制

> **一设机制**：卡特娜·奥瑞吉纳尔（Curtana Original）的次元切断攻击具有特殊的时间机制——在发动后的**1.25秒内**，该现象属于"异能之力"，可用幻想杀手消除；超过1.25秒后，次元切断成为已完成的"正常物理现象"，无法再被消除。
>
> 这一设定暗示了幻想杀手的本质机制：它消除的是**「不自然的异常」**而非已经完成并固化为"自然"的结果。

### 6.4 身体层面的弱点

| 弱点 | 说明 |
|------|------|
| **普通人身体素质** | 没有超人体质，被刀刺枪击都会受伤 |
| **耐力有限** | 自己评价：能打1对1，勉强2对1，3对1必须逃 |
| **手臂反作用力** | 消除大型异能时右臂承受反作用力冲击 |
| **无法群战** | 面对多个方向的同时攻击极为不利（仅一只右手有效） |

### 6.5 不幸体质的真相

| 项目 | 说明 |
|------|------|
| **现象** | 极端霉运——钱包丢失、卷入麻烦、喝水呛到等 |
| **原因** | 幻想杀手被动无意识消除「神明的庇佑」「命运的红线」等微弱异能之力 |
| **本质** | 普通人享有的好运也是某种"超自然恩惠"，被右手一并消除 |
| **谣言** | 「待在上条身边的人也会跟着不幸」——不完全准确，但接近事实 |

## 7. 关键设定矛盾与演进

### 7.1 能力本质认知的演进（一设中的逐步揭示）

这不是矛盾，而是作者有意设置的**逐层解密**：

| 阶段 | 作品阶段 | 认知水平 | 代表发言 |
|------|----------|----------|----------|
| 初期 | 旧约1-12卷 | 简单的"消除异能" | 「碰到的异能都会消失」 |
| 中期 | 旧约13-22卷 | 开始暗示深层本质 | 「世界的基准点」（欧雷尔斯） |
| 后期 | 旧约22卷 | 神净概念引入 | 「十字教程度无法解释」（亚雷斯塔） |
| 新约 | 新约1-22R | 多层揭示 | 魔神愿望集合体、神净人格化 |
| 创约 | 创约1-9+ | 全面释放 | 全力全开龙王模式、幻想侧归属 |

### 7.2 龙的数量矛盾

| 出处 | 数量 | 说明 |
|------|------|------|
| 旧约2卷 | 1头 | 龙王之颚（透明龙首） |
| 旧约SP/超炮外传 | 8头 | 八龙各具属性 |
| 部分设定讨论 | 17条 | 创约9融合为1 |
| 创约9卷 | 融合重置 | 17龙合为单体的龙王模式 |

> **分析**：龙的数量可能随当麻的精神成长和解封程度递增，也可能存在不同的"龙群"层级。目前无官方明确解释，属于设定中的模糊地带。

### 7.3 里幻（龙）vs 神净讨魔的关系争议

| 观点 | 证据 | 问题 |
|------|------|------|
| **里幻=神净** | 创约9中从断口出现的透明团块=龙=神净，三者一致 | 新约22R中神净"捏爆"黑色漩涡（=里幻），暗示两者不同 |
| **里幻≠神净** | 神净在创约9中"捏爆"了里幻，明显是两个不同存在 | 创约9后半又统一表现为龙形态 |
| **三层嵌套说（暂定主流）** | 幻杀表层→里幻龙中层→神净核心，一层包裹一层 | 缺乏官方直接确认 |

> **现状**：截至创约9卷，镰池和马仍未完全揭示三层结构的精确关系。这是作品的核心悬念之一。

### 7.4 幻想杀手的历史矛盾

| 设定 | 出处 | 说明 |
|------|------|------|
| 「1900年布莱斯路之战中被毁的灵装」 | 旧约相关卷 | 暗示是唯一的历史形态 |
| 「已经以无数形态存在过」 | 新约相关卷 | 曾以人物、物品、建筑、场所等多重形态出现 |
| 「前代持有者用幻杀庇护整个黄金结社」 | 新约卷 | 暗示可以主动运用（当麻只会被动消除） |

> **分析**：上述说法并非矛盾，而是层次不同——灵装形态只是其历史形态之一。前代持有者对幻想杀手的运用方式（主动庇护/范围覆盖）远高于当麻的水平，暗示当麻尚未掌握能力的全部用法。

### 7.5 与原石（Gemstone）定义的关系矛盾

| 来源 | 说法 |
|------|------|
| 欧雷尔斯 | 「没有证据显示上条是原石能力者（因不散发AIM扩散力场）」 |
| 部分读者推测 | 可能是一种特殊的原石 |
| 一设结论 | 被明确排除在"原石"定义之外 |

### 7.6 右方之火对当麻的认知

| 引用 | 含义 |
|------|------|
| 认为幻想杀手与「神圣之右」类似 | 暗示两股右手力量同源或对应 |
| 认为当麻「不会使用本来的力量」 | 确认当麻还未能发挥幻想杀手全部性能 |
| 夺走右手后恐惧无形之物 | 确认真正的力量在当麻而非断手上 |

> **一设确认**：右方之火在夺走当麻右手并吸收幻想杀手后，发现力量并未达到预期效果。从断臂处显现的"无形之物"令他恐惧——这暗示**真正的力量始终在上条当麻这名少年身上，而非仅仅在右手中**。

## 补充信息

### 前兆感知（Precognition / 前兆の感知）

虽然并非官方定义的能力，但当麻在无数次战斗中培养出了一种**本能的战斗感知**：

| 项目 | 内容 |
|------|------|
| **别名** | 心眼（Mind's Eye）、预兆感知 |
| **机制** | 读取对手不经意的微小动作（眨眼、肌肉抽动、呼吸节奏等） |
| **其他应用** | 检测AIM扩散力场和灵脉（Telesma流）的微妙变化 |
| **命名者** | 一方通行、蕾薇妮雅·柏德蔚 |
| **可被克制** | 对手完全抑制所有无意识动作时无效（如Rensa曾做到） |

> 这不是超能力，而是纯粹的经验积累产物。

### 圣日耳曼共生（创约中）

当麻曾被圣日耳曼（St. Germain）病毒感染，但最终达成共生状态：
- 允许当麻与圣日耳曼切换身体控制权
- 通过左手获得**碳元素操纵魔法**能力
- 能力被幻想杀手的消除场削弱
- 圣日耳曼最终为保护当麻而自我牺牲
