# 01-writings.md: 绢旗最爱 (Kinuhata Saiai) 官方设定研究

> 本文基于《魔法禁书目录》系列、《科学超电磁炮》系列及《とある暗部の少女共棲》（Toaru Anbu no ITEM）官方作品及设定集整理，涵盖旧约、新约至创约的全部已知设定。

---

## 一、基础信息

### 1.1 身份档案

| 项目 | 内容 |
|------|------|
| 姓名 | 绢旗最爱（Kinuhata Saiai） |
| 日文名 | 絹旗 最愛（きぬはた さいあい） |
| 称号 | ITEM（道具）防御核心、"黑暗的五月计划"优等生 |
| 性别 | 女性 |
| 身高 | 约160cm（在ITEM成员中最矮） |
| 年龄 | 12岁（系列进行中） |
| 所属势力 | 学园都市暗部组织 ITEM（道具） |
| 能力等级 | 大能力者（Level 4） |
| 能力名称 | 氮气装甲（Offense Armor / 窒素装甲） |
| 声优 | 赤崎千夏（日语）/ Apphia Yu（英语） |
| 血型 | B型 |
| 星座 | 巨蟹座 |
| 兴趣 | 观看B级电影/C级电影、收集电影票根 |
| 其他特征 | 语癖：「超」（chō）的口头禅 |

> 官方设定称其为「暗部组织ITEM的正式成员」，也是学园都市黑暗五月计划的幸存者与成功个体。

### 1.2 外观特征

绢旗最爱拥有辨识度高的视觉设计，在系列角色中呈现出「年幼外表 × 暗部杀手」的反差形象：

**基本造型：**
- 及肩棕色鲍伯头（bob cut），在系列后期头发略微剪短
- 身材娇小——在ITEM四人中个子最矮
- 拥有与年龄不符的身材比例，但**极其在意自己缺乏胸部曲线**
- 纤瘦的美腿是她刻意展示的"武器"

**标志性着装：**
- **旧约时期**：短毛绒针织连衣裙（裙摆极短，特意露出大腿）
- **《超电磁炮》早期**：橙色连帽衫 + 热裤，更偏中性外观
- **后期造型**：皮夹克内搭短袖T恤 + 紧身短裤
- 条纹长袜
- **常备物品**：罐装液氮（挂在腰间或随身携带）、电影票根收集册

**官方衣着说明：**
- 作者镰池和马在设计上强调绢旗的「反差萌」——外表看起来是个乖巧的小学生，实际上是杀人如麻的暗部杀手
- 她的服装设计在系列中经历了一个从"酷美少女"（kuudere）到"喜剧+性感"形象的转变

> 外观在旧约第15卷初登场时以「短裙+长靴」为主，到新约及《超电磁炮》时期逐渐增加了更多个性化元素。

### 1.3 性格设定

绢旗最爱的性格呈现出「表面冷静 × 内在孩子气」的独特反差：

| 维度 | 描述 |
|------|------|
| 表面印象 | 冷静、沉着、说话带口头禅「超」的12岁少女 |
| 内在本质 | 极度重视友情、依赖同伴、害怕孤独的"依赖型"人格 |
| 核心矛盾 | 暗部杀手的职业要求 vs 天真孩子的兴趣爱好 |
| 社交定位 | ITEM中最冷静理性的成员，不像芙兰达那样疯癫，也不像麦野那样情绪不稳 |
| 对上条侧态度 | 基本中立，通过与滨面的关系间接接触 |

**性格细节：**

- **不服输**：虽然外表温和，但骨子里有着强烈的竞争心和不服输的精神
- **依赖朋友**：滨面仕上形容她是"超依赖朋友的类型"。当听说泷壶理后因过度使用「体晶」倒下时，担心得要命
- **怕孤独**：本质上害怕独自一人，对团队有极强的归属需求
- **爱开玩笑**：喜欢捉弄滨面，特别是用性暗示话题让他害羞
- **职业素养**：任务中冷静实际——目标服从就不杀，反抗就无情打击。对暗部间的厮杀没有心理障碍，但避免牵连无辜
- **善良的一面**：在《某少女的办公室》事件中，她不惜冒着任务失败的风险放走了被俘的Judgment成员

### 1.4 人际关系

## 二、能力设定

### 2.1 氮气装甲（Offense Armor / 窒素装甲）

绢旗最爱的能力是操纵大气中的**氮气**，在身体周围形成一层数厘米厚的高压压缩氮气装甲。

#### 能力核心机制

| 项目 | 内容 |
|------|------|
| 等级 | Level 4（大能力者） |
| 类型 | 自动防御型 + 物理强化型 |
| 操控对象 | 大气中的氮气分子 |
| 有效范围 | 距手掌仅数厘米——极其近距离 |
| 发动方式 | 自动发动（类似一方通行的"反射"），与被攻击意识无关 |

#### 防御面

- **全方位自动防御**：氮气装甲覆盖全身360度，在受到攻击时自动启动
- **防弹性能**：可以轻松抵御手枪子弹、来复枪子弹
- **重火力防御**：学园都市特制狙击枪直击也只能将她击退数步，不会造成实际伤害
- **连续打击弱点**：如以足够威力持续攻击同一点，氮气装甲仍可能被突破

> 绢旗本人的原话：「我的氮气装甲可不会输给区區的枪械哦。」

#### 攻击面

- **超人力气**：通过在手部包裹压缩氮气，可以单手举起汽车
- **空手弹弹**：能够以手掌直接弹开来复枪子弹
- **拳力惊人**：近身战中拳头的威力足以匹敌加特林机枪
- **彗星冲击**：将氮气包裹全身进行突击，如同彗星一般冲撞目标

#### 射程局限性

氮气的操控范围极其有限——距手掌数厘米已经是极限。因此她的攻击动作看起来像是"用手去拿/举东西"，实际上是通过手掌附近的压缩氮气进行物理操作。

#### 弱点与对策

| 弱点 | 说明 | 对策 |
|------|------|------|
| 氮气依赖 | 能力依赖于大气中的可用氮气 | 随身携带罐装液氮作为"补充弹匣" |
| 真空环境 | 如果敌人制造局部真空去除氮气，能力失效 | 释放罐装液氮到空气中瞬间补给 |
| 火焰消耗 | 大规模火焰会消耗空气中的氮气 | 先用液氮罐补充后再进入战斗 |

> 绢旗的对策非常实用主义——她永远随身携带小型液氮罐，在需要时释放到空气中立刻产生大量氮气供给能力使用。

### 2.2 能力来源：「黑暗的五月计划」（Dark May Project）

绢旗最爱的能力并非天生如此——它是一起残酷实验的产物。

**计划概要：**
- 学园都市为复制一方通行的能力而进行的秘密实验
- 将一方通行演算模式中的"防御性思考模式"移植到受试者的大脑中
- 目的是改变受试者的"个人现实"（Personal Reality），使其能力向矢量操控方向优化

**绢旗在计划中的定位：**
- 她原本是"Child Error"（无监护人儿童），被用作实验体
- 称其为计划中的**「优等生」**——她的能力成功发展为自动防御机制，且人格相对保持稳定
- 相比之下，另一幸存者誉望万化变得极不稳定且有严重暴力倾向
- 原能力推测为更广泛的大气控制类，经过计划改造后突变为目前的氮气装甲

> 据垣根帝督的评价：「绢旗是出自『暗黑五月计划』的遗留品，那个计划的目的是用类似的能力操作模式来复现一方通行的能力。」

**待ITEM奉命销毁该计划的研究资料时：**
- 他们发现大量被冷冻保存的实验体
- 绢旗是唯一被选中招募的——因为ITEM需要一个近战特化型战力来补充泷壶理后的追踪能力

### 2.3 装备

绢旗除了能力之外，还携带多种实弹武器：

| 装备 | 说明 |
|------|------|
| 手枪 | 熟练的枪手，精准度很高 |
| 对战车用飞弹 | 小型金属棒状（约30cm），拉绳发射，有效射程500米，可摧毁建筑物 |
| 罐装液氮 | 随身携带，用于紧急补充氮气 |
| 烟雾弹 | 用于掩护和战术撤退 |

> 关于她的对战车飞弹，粉丝常吐槽「平时到底藏在哪里的」。

## 三、组织背景

### 3.1 ITEM（道具）

ITEM是学园都市的暗部组织之一，与SCHOOL（学校）、MEMBER（人员）、BLOCK（区块）并列为学园都市主要暗部组织。

**组织构成（核心期）：**
| 成员 | 代号/能力 | 定位 |
|------|-----------|------|
| 麦野沉利 | Meltdowner（Level 5） | 首领、主力火力 |
| 绢旗最爱 | Offense Armor（Level 4） | 防御核心、近战特化 |
| 泷壶理后 | AIM Stalker（Level 4） | 追踪、辅助 |
| 芙兰达·赛维伦 | Explosives（无能力） | 炸弹专家、后勤 |

**组织历史：**
- **成立期**：在毁灭黑暗五月计划资料后被组建，绢旗被招募
- **活跃期**：执行学园都市上层指派的各类暗部任务——暗杀、情报搜集、破坏、镇压
- **芙兰达之死**：麦野在愤怒中杀死芙兰达（旧约第15卷），组织内部出现裂痕
- **重组期**：麦野被滨面击败后，绢旗短暂成为ITEM首领（旧约第19卷）
- **新约时期**：麦野回归，ITEM在滨面的安排下重新组队，成员包括麦野、绢旗、泷壶、滨面

### 3.2 暗部组织的生态

学园都市存在多个相互竞争的暗部组织。ITEM并非最强的，但以其成员能力的互补性著称：

- **绢旗的定位**：在团队中担任防御者和近战护卫
- **与其他组织的关系**：与SCHOOL（特别是垣根帝督）发生过正面冲突
- **任务类型**：以学园都市上层的"脏活"为主，处理无法通过正规渠道解决的问题

## 四、兴趣与爱好

### 4.1 B级/C级电影狂热

绢旗最爱的最大爱好是观看电影——但并非好莱坞大片，而是B级片（低成本商业片）和C级片（质量极低、常因太差而有趣的影片）。

**口味偏好：**
- 喜欢那些"本想挑战好莱坞却因为太认真反而变成C级"的作品
- 作品中提及的片名包括「Wildman N's Zombie Escape Madness」「Saitama☆Vocal Crisis」等虚构影片
- 对香港黑色电影也有兴趣，曾提及「红龙」
- 够买限制级（R-15）电影票时会让滨面帮忙伪造身份证

**爱好者行为：**
- 收集所有看过电影的电影票根
- 不仅喜欢电影本身，也支持播放这些电影的小众影院和放映设施
- 会向朋友（尤其是滨面）强行推荐/拉人一起看

**兴趣起源：**
- 在被关在研究设施中的时期，用来播放演示视频的屏幕是她为数不多的娱乐来源
- 由此发展出对活动影像的热爱，逐渐演变为对B级/C级电影的特殊喜好

### 4.2 身材焦虑

绢旗另一个有趣的个人特征是她的身材焦虑——她对胸部大小极其在意。

- 芙兰达身体突然发育后，绢旗对她产生了嫉妒心
- 在《超电磁炮》的Indian Poker事件中，与御坂美琴因一张据称能"让胸部变大"的卡牌发生冲突
- 在冲突中称美琴为「洗衣板浑蛋」
- 会购买学园都市内的「武藏野牛奶」——不是为了丰胸，而是因为身高曾被电影院拒绝入场——想长高

## 五、作者访谈与设定补遗

根据镰池和马的访谈及后记信息：

1. **创作灵感**：绢旗最爱的角色设计糅合了"年幼暗部杀手"和"电影宅少女"两个看似矛盾的标签。她的存在意义之一是展现学园都市暗部的"人性面"——从事黑暗工作的人也有正常的兴趣。

2. **名字由来**：「最愛」在日语中意为"最被爱的人"或"最喜爱的事物"，这个名字在故事中形成了黑色幽默——一个被当作实验品、从事暗杀工作的少女，名字却叫"最爱"。

3. **"超"口癖的设计**：镰池和马在访谈中说明，绢旗的"超"口癖是"黑暗的五月计划"的副作用之一——原本一方通行的思考模式中包含大量粗口，绢旗的脑内过滤器将这些转化为了"超"。

4. **角色形象的演变**：初登场时（旧约第15卷）更接近"酷美少女"（kuudere）形象，到旧约第19卷时已经转变为更具喜剧色彩和性感元素的角色。镰池和马表示这是写作过程中自然发展的结果。

5. **ITEM外传企划**：2023年开始连载的《とある暗部の少女共棲》（Toaru Anbu no ITEM）是专门聚焦ITEM的外传系列，由镰池和马亲自执笔，展示了ITEM早期的活动和成员间关系，绢旗在该系列中是核心角色之一。

6. **关于与一方通行的关联**：官方设定明确绢旗的能力是一方通行"反射"的劣化版本——都是自动防御，但绢旗的仅限物理攻击且可以被持续攻击突破。她是黑暗五月计划中唯一取得"半成功"的案例。
