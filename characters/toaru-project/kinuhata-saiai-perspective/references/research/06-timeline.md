# 06-timeline.md: 绢旗最爱 (Kinuhata Saiai) 时间线

> 涵盖：背景设定 → 旧约（15~22卷）→ 新约（1~22卷）→ 创约至今 → 外传作品的全主线出场记录

---

## 一、正式登场前——背景设定

| 时间 | 事件 | 备注 |
|------|------|------|
| 小说主线开始前约10年 | 绢旗最爱出生 | 作为"Child Error"（无监护人儿童）在学园都市生活 |
| 主线开始前 | 被选为「黑暗的五月计划」（Dark May Project）的受试者 | 该计划旨在通过移植一方通行的思考模式来复制其能力 |
| 实验期间 | 绢旗在设施中通过播放演示视频的屏幕发展出对电影的爱好 | 在封闭的设施中，屏幕上的影像成为了她的娱乐和慰藉 |
| 计划结束后 | 计划失败，绢旗被冷冻保存 | 她是少数没有在实验中报废的受试者 |
| ITEM招募前 | ITEM奉命销毁黑暗五月计划的研究资料 | 在设施中发现被保存的实验体 |

> 绢旗加入ITEM的确切时间点不明，但可以确定是在《魔禁》主线较早期的阶段。她在研究设施中度过的时间远超常人能想象的界限，这段经历塑造了她"害怕孤独、依赖同伴"的核心性格。

## 二、旧约篇（《魔法禁书目录》1~22卷）

### 2.1 旧约第15卷——ITEM全面登场（初登场卷）

**本卷定位：** 绢旗最爱的初登场卷。ITEM作为学园都市暗部组织全面登场。

| 剧情阶段 | 事件 | 备注 |
|----------|------|------|
| 开篇 | ITEM受命处理暗部组织"学校（SCHOOL）"的威胁 | 初登场 |
| 冲突 | 绢旗与砂皿筑麻（Sunazara Chimitsu）战斗，展示氮气装甲的防御力 | 第一次能力展示 |
| 战斗 | 以氮气装甲压制对手，将对手打穿建筑物 | 进攻性使用氮气装甲 |
| 转折 | 垣根帝督（Kakine Teitoku）——SCHOOL的首领、Level 5第二位出现 | 战力差距展示 |
| 败北 | 绢旗被垣根帝督的「未元物质」能力轻松击败 | Level 4 vs Level 5的明确差距 |
| 救援 | 滨面仕上安排救护车将受伤的绢旗救走 | 绢旗与滨面关系的起点 |
| 后方 | 芙兰达·赛维伦被麦野沉利杀死 | ITEM崩溃的开始 |

**战斗记录：**
- **对手：** 砂皿筑麻（SCHOOL成员）
- **对手：** 垣根帝督（SCHOOL首领，Level 5第二位）
- **战绩：** 击败砂皿筑麻，被垣根帝督击败
- **亮点：** 首次展示氮气装甲的自动防御能力和近战破坏力

> 旧约第15卷的彩页插图中，绢旗正在阅读的书是《魔禁》漫画本身——这是镰池和马的一个小彩蛋。

### 2.3 旧约第19卷——对抗Stephanie Gorgeouspalace

**本卷定位：** 绢旗在旧约中的重要战斗卷。此时的麦野已暂时退出，绢旗成为ITEM的代理首领。

| 剧情阶段 | 事件 | 备注 |
|----------|------|------|
| 背景 | 第三次世界大战（WWIII）前后的混乱局势 | 宏观背景 |
| 冲突 | 绢旗面对使用火焰能力的敌人Stephanie Gorgeouspalace | 能力相性极差的对手 |
| 危机 | 火焰消耗了空气中的氮气，绢旗的防御逐渐失效 | 能力弱点的第一次明确展示 |
| 逆转 | 绢旗使用随身携带的液氮罐补充氮气 | 战术补救 |
| 战后 | 绢旗从海市岛带（Kokusai Kaibi）处得知滨面回归的消息 | 情报交换 |
| 领导 | 绢旗作为ITEM的代理首领做出任务决策 | 短暂的领导期 |
| 结局 | 麦野回归，绢旗交还领导权 | ITEM重组前的过渡期 |

**战斗记录：**
- **对手：** Stephanie Gorgeouspalace（能力者，火焰系）
- **战绩：** 有惊无险——在氮气耗尽前成功补充
- **亮点：** 第一次明确展示绢旗能力的弱点和对策

### 3.2 新约第5卷——与誉望万化再会

**本卷定位：** 绢旗的出身设定进一步展开——与"黑暗的五月计划"的另一幸存者对峙。

| 剧情阶段 | 事件 | 备注 |
|----------|------|------|
| 相遇 | 学园都市出现新的暗部威胁「Freshman」（新生） | —— |
| 冲突 | 绢旗与誉望万化（Kuroyoru Umidori）——同为黑暗五月计划幸存者——相遇 | 决定性的遭遇 |
| 对峙 | 两人互相确认身份，气氛紧张但未直接开战 | 复杂的"同类"关系 |
| 对话 | 誉望万化称绢旗为"优等生"，绢旗表示如果她妨碍自己就亲手埋葬她 | 宿命的对峙 |
| 插曲 | 绢旗发现誉望万化牙痛，拖她去看牙医 | 典型的绢旗式"不坦率的关心" |
| 结局 | 两人各自离开——彼此承认但不同路 | —— |

**关系：**
- 誉望万化是黑暗五月计划中除了绢旗之外的另一个知名幸存者
- 与相对稳定的绢旗不同，誉望万化人格极不稳定且有暴力倾向
- 两人之间没有友情，只有"来自同一个地狱"的认同感和敌对感并存

### 3.5 新约后期其他出场

| 卷次 | 事件 | 备注 |
|------|------|------|

## 五、外传作品

### 5.1 《とある暗部の少女共棲》（Toaru Anbu no ITEM）

**作品性质：** 2023年开始连载的ITEM外传系列，镰池和马执笔，Nilitsu插画。

| 卷次 | 发售日期 | 绢旗相关内容 |
|------|----------|-------------|
| 第1卷 | 2023年3月10日 | ITEM的组建和早期任务，绢旗被招募的经过 |
| 第2卷 | 2024年6月7日 | ITEM内部关系和日常任务 |
| 第3卷 | 2024年6月7日 | 团队在暗部中的生存斗争 |
| 第4卷 | 2025年3月7日 | 深入展现ITEM成员的个人故事 |
| 第5卷 | 2025年9月10日 | 团队面临新威胁 |
| 第6卷 | **2026年2月10日** | **绢旗最爱焦点卷**——在一次银行抢劫事件后，她发现有无辜者被卷入；为了保护这个人，她冒着被麦野处分的风险逃跑 |

**第6卷的剧情焦点（已公布）：**
> 绢旗在银行抢劫事件后发现一名无辜者被强行牵扯进了事件中，她带着这个人逃亡——这使她与ITEM和麦野处于对立位置，让自己的生命也陷入危险之中。

> 这是绢旗全系列中第一次成为一卷故事的核心焦点角色。

## 七、登场时间线总览图

```
主线前：Child Error → 黑暗五月计划实验体 → 冷冻保存
                            ↓
旧约第15卷：初登场 · ITEM全面出击 …… 核心登场

                            ↓
新约第1卷：ITEM重组 · 接受麦野回归
新约第5卷：与誉望万化再会 · 黑暗五月计划幸存者的对峙

                            ↓

                            ↓
外传：《とある暗部の少女共棲》——ITEM早期故事（2023~）

```
