### 1.2 外观特征

**初登场造型（旧约15卷）：**
- 褐色头发，身高接近180cm
- 穿着高级上衣，但品味奇特——会将毛衣穿在外套里面
- 眼神阴沉，嘴角常带充满自信的微笑
- 整体给人以轻浮却危险的印象

**复活后造型（新约6卷）：**
- 全身内脏一半以上用「未元物质」取代
- 皮肤颜色异常发白，白得与衣服难以区分
- 头发与衣服的色素残留，但色彩层次无法区隔衣服与肌肤
- 嘴巴和眼窝中是与此淡薄色素极不搭调的深邃黑暗
- 给人强烈的不协调感和异质性的压迫

**独角仙-05形态（新约6卷以后）：**
- 主体形态为白色独角仙（约15米长）
- 可缩小为钥匙扣大小的白色独角仙挂件
- 也可变回人形，外观与原垣根帝督相同

## 二、能力设定：未元物质（Dark Matter）

### 2.1 能力本质

未元物质（Dark Matter）是垣根帝督的专属能力，其本质是——**产生这个世上不存在的物质**。

这是真正的核心设定：这里说的「不存在」不是指「尚未发现」或「理论上应该存在」，而是指**根本不可能存在于这个世界上的物质**。

这种物质最重要的特征在于：**它不适用既存的物理法则**。由于未元物质本身没有基本粒子结构（或者具有这个世界不存在的粒子结构），因此：

- 光波和电子波只要通过未元物质构成的白色翅膀上的细小空隙，就会改变性质
- 这些波会被**扭曲成不可能存在的矢量**
- 这些扭曲后的能量依照未元物质**独自的法则行动**
- 它可以同时拥有互相矛盾的物理性质

### 2.2 能力效果详解

**创造范围：**
- 可以创造任意形态、任意性质的物质
- 可以决定该物质的外观、硬度、弹性、温度、光学特性等
- 未元物质虽然没有血液，但含有类似生物电流的脉冲信号

**注意事项（限制）：**
- 不能创造反物质（反物质有基本粒子结构，只是电荷相反）
- 不能创造黑洞（广义相对论的常规产物）
- 不能创造正常物质（因为正常物质存在于此世界）

**未元物质与反物质的区别：**
这一概念在粉丝中常有误解。未元物质不是反物质——反物质虽然罕见，但它在物理上是「可能的」（有正电子、反质子等结构）。未元物质是真正不可能存在于这个世界的、完全「异质」的东西。

### 2.3 白翼——未元物质的标准形态

启动能力时，垣根帝督背部会自动张开**六片白色翅膀**，这六片翅膀由未元物质构成：

| 特性 | 说明 |
|------|------|
| 大小 | 可延伸至20米以上 |
| 数量 | 6片（天使翼结构） |
| 颜色 | 纯白（对比一方通行的黑翼） |
| 材质 | 纯未元物质 |
| 可塑性 | 可变形为无数羽毛散开 |

**战术应用：**
- **飞行**：以翅膀高速移动
- **防御**：用翅膀形成屏障
- **打击/斩击**：翅膀边缘可进行物理切割
- **烈风/冲击波**：翅膀挥动产生强力风压
- **光攻击**：从翅膀间隙通过的光波和电子波被扭曲后发射
- **能量类型**：可产生超过25,000种以上的能量类型

### 2.4 能力的内在工作机制

未元物质的核心运行逻辑：**垣根帝督不是控制物质的行为，而是直接定义物质的规则。**

更详细地说：
1. 通常的物理性能力（如一方通行的矢量操作）是在既有物理法则的框架内运作
2. 未元物质则是**创造新的物理法则**——垣根帝督创造的物质有自己的规则
3. 这意味着对手不能通过分析「物质」来反制，因为他们面对的可能是性质完全未知的东西
4. 推测该能力的实际原理是将「不存在」的粒子从虚数空间（或高维空间）中拉入现实

### 2.5 对战一方通行时的能力特殊性

垣根帝督能够在一定程度上绕过一方通行的「反射」能力，其原理如下：

一方通行的反射是在既有物理法则的框架内运作——他通过计算接触物的矢量来反射一切物理攻击。但未元物质由于其「不适用既有物理法则」的特性，可以在某种程度上不被一方通行的计算模型所捕获。

具体来说：
1. 垣根帝督让未元物质上的细小空隙过滤光波和电子波
2. 这些波被扭曲成不可能存在的矢量
3. 一方通行需要重新定义自己的计算模型才能应对
4. 这意味着垣根帝督是已知少数能对一方通行造成实质性威胁的对手

但在一方通行暴走（黑翼形态）时，他通过重新定义世界公式，反过来也能操控未元物质，从而击败了垣根帝督。

## 三、等级与排名

### 3.1 Level 5 第二位

学园都市仅有7人的Level 5超能力者中，垣根帝督排名第二。关于这个排名的具体含义：

| 对比对象 | 关系 |
|---------|------|
| 第一位：一方通行 | 垣根帝督的宿敌，他始终认为自己应与一方通行同等甚至更强 |
| 第三位：御坂美琴 | 垣根帝督完全未将其视为威胁 |
| 第四位：麦野沉利 | 暗部「道具（ITEM）」的首领，与SCHOOL在暗部大战中对立 |
| 第五位：食蜂操祈 | 心理系能力者，垣根帝督对其评价不高 |

**关于排名的争议：**
垣根帝督始终对自己屈居第二位而不满。他并非不能接受「有人比我强」，而是不能接受「仅凭排名就被定性为不如一方通行」。在他看来，排名的判定标准存在缺陷——未元物质的能力价值无法通过常规的能力测定来评估。

这种对排名制度的执念，与一方通行对于「最强」称号的虚无态度形成鲜明对比。

### 3.2 SYSTEM（Level 6）的潜在可能性

- 垣根帝督是除一方通行之外，唯一被认为有潜力达到Level 6（绝对能力者/SYSTEM）的超能力者
- 亚雷斯塔·克劳利将其视为「备用计划」——如果一方通行无法完成Level 6 Shift计划，垣根帝督是替代方案
- 这一定位对垣根帝督而言是一种隐晦的侮辱——他并非被选中的「正选」，而仅仅是「备胎」
- 垣根帝督对亚雷斯塔计划的内幕有一定程度的了解，这使他产生了一种特殊的偏执

### 3.3 能力测定难题

未元物质的能力测定面临一个根本性问题：**学园都市的能力测定系统无法测量「不存在于这个世界的物质」。**

这意味着垣根帝督的「第二位」排名本身就是一个近似评价——实际能力可能远超这个排名。这进一步加强了垣根帝督的傲慢，因为他知道自己的能力评价被低估了。

## 四、角色定位与主题意义

### 4.1 与一方通行的对照关系

垣根帝督在角色设计上明显是一方通行的对照镜：

| 维度 | 一方通行 | 垣根帝督 |
|------|---------|---------|
| 排名 | 第一位 | 第二位 |
| 能力本质 | 矢量操作（操控既有法则） | 未元物质（创造新法则） |
| 能力象征 | 黑翼（毁灭） | 白翼（创造？/未知） |
| 人格原型 | 从虚无走向守护 | 从傲慢走向被取代 |
| 亚雷斯塔眼中的角色 | 正选 | 备用计划 |
| 精神状态 | 虚无→觉醒 | 傲慢→执念 |
| 最终走向 | 成为统括理事长 | 人格被造物取代 |

黑白双翼的意象尤其值得注意——一方通行暴走时出现的是黑翼，而垣根帝督的未元物质始终是白翼。这种视觉对比强化了他们作为光与暗、创造与毁灭的对照关系。

### 4.2 「触及神之领域者」

在复活后，垣根帝督获得了「触及神之领域者」这一称号。这层含义在于：

- 创造「不存在于此世界的物质」相当于行使了神明的创造权能
- 用未元物质重建自己的身体，超越了人类的生命界限
- 以分布式意识网络的形式存在，获得了类似「全知」的信息收集能力
- 量产能力——将未元物质的力量分配给所有分身——这是单一个体不可能达到的领域

### 4.3 角色主题

垣根帝督的角色弧线围绕以下主题展开：

1. **执着与执念**：他对超越一方通行的执着近乎病态，但这执着背后是更深层的存在焦虑——他需要证明自己是「被选择的」，而非「备用计划」

2. **自我的分裂与取代**：他性格中的残忍与温柔最终分裂为两个独立的意识，温柔的部分（独角仙-05）反过来取代了残忍的部分（原垣根帝督）

3. **造物反噬造物主**：被他创造的工具获得了自我意识，并判定创造者本身是「应该被消除的危险因素」——这一命题在科幻作品中具有深刻的象征意义

4. **无法被救赎的恶**：与一方通行获得救赎不同，垣根帝督的「恶」从未获得赎罪的机会。取代他的独角仙-05是「善良的人格」，但原垣根帝督从未被原谅或救赎

## 五、组织设定：暗部「学校（SCHOOL）」

### 5.1 组织概况

SCHOOL是学园都市的暗部组织之一，垣根帝督是首领。该组织在暗部大战期间活跃。

| 项目 | 内容 |
|------|------|
| 名称 | SCHOOL（学校） |
| 首领 | 垣根帝督（第二位） |
| 主要活动 | 暗部对抗、装备争夺、情报活动 |
| 周期 | 活跃于10月9日前后 |
| 结局 | 垣根帝督战败后瓦解 |

### 5.2 组织成员

| 成员 | 代号/角色 | 备注 |
|------|----------|------|
| 垣根帝督 | 首领/未元物质 | Level 5第二位，组织核心战力 |
| 心理定规（ unnamed girl ） | 前线成员 | 能读取/调整人与人之间的心理距离，拒绝参与对抗一方通行的战斗 |
| 弓箭猎虎（Yumiya Rakko） | 狙击手 | 原成员，后被ITEM击杀 |
| 砂皿致密（Sunazara Chimitsu） | 狙击手（替补） | 弓箭猎虎死后加入，操作远程攻击武器 |
| 誉望万化（Yobou Banka） | 战斗员 | 前线战斗成员 |

### 5.3 暗部大战中的角色

在10月9日的暗部大冲突中，SCHOOL主要行动包括：
1. 攻击暗部「人员（MEMBER）」——抢夺「镊子」（特殊装备）
2. 攻击暗部「道具（ITEM）」——与麦野沉利等人交战
3. 追击最后之作（Last Order）——以此作为对付一方通行的筹码
4. 与一方通行决战——在垣根帝督战败后瓦解

## 六、能力进化路径

垣根帝督的能力进化体现了从「物质创造」到「存在方式本身变革」的飞跃：

### 第一阶段：标准未元物质（旧约15卷）
- 创造各种形态的未元物质
- 主要应用为白翼的各种战术运用
- 物质创造范围有限，需要集中意识控制

### 第二阶段：自我组织化（新约4~6卷）
- 学会用未元物质复制人体细胞
- 用未元物质替代已失去的内脏器官
- 实现身体的大规模修复

### 第三阶段：分布式意识（新约6卷）
- 将未元物质作为身体的一部分，切割后各部分独立运作
- 通过「喷发点」将能力平等分配给所有个体
- 将身体区块化，各部分独立传递情报但无直接线路连接
- 实现对一方通行的电子信号逆流攻击的免疫

### 第四阶段：独角仙形态（新约6卷末）
- 创造大量半自律型未元物质生命体（白色独角仙）
- 每个独角仙都是他意识的分身
- 即使主体被摧毁，任意分身都可以成为新的主体

### 第五阶段：独角仙-05独立（新约6卷末）
- 被创造的独角仙-05获得了自我意识
- 取代了原垣根帝督的主体意识
- 未元物质网络不因核心的更替而崩溃——与御坂网络不同

## 七、设定中的特殊注解

### 7.1 未元物质网络的特性

未元物质网络与御坂网络的关键差异：

| 特性 | 御坂网络 | 未元物质网络 |
|------|---------|------------|
| 结构 | 以御坂美琴为模板，有核心 | 不需要核心的稳定系统 |
| 故障影响 | 一部分出问题可能波及全体 | 分割成多少块就产生多少意志 |
| 意识整合 | 共享但统一 | 会合时意识整合为一 |
| 稳定性 | 存在被黑客攻击的风险 | 区块化难以被外部侵入 |

### 7.2 提取思念情报的能力

垣根帝督复活后获得的额外能力——通过未元物质读取物质表面的细微纹路，提取残留的思念情报（Psychometry）：

- 方法类似于用唱针读取黑胶唱片
- 未元物质如同录音磁头，读取环境的电磁记录
- 可以作为情报收集手段
- 这一能力在追踪最后之作和芙蕾米亚时被使用

### 7.3 未元物质的色彩意象

垣根帝督的未元物质始终呈现白色，这一色彩选择具有多重含义：

- **对立于黑色**：与一方通行的黑翼形成鲜明对照（白 vs 黑）
- **天使的拟像**：白翼的外观与天使翅膀高度相似，暗示垣根帝督「触及神之领域」
- **纯净中的异质**：纯白的外观掩盖了其作为「不存在物质」的怪异本质，一种视觉上的欺骗
- **独角仙-05的延续**：取代后的独角仙-05仍然使用白色，象征虽然人格改变，但本质未变

## 八、衍生作品设定

### 8.1 科学未元物质（Toaru Kagaku no Dark Matter）

以垣根帝督为主角的外传漫画，进一步拓展了其背景设定：

- 揭示了更多的SCHOOL活动细节
- 深入描绘了垣根帝督与木原一党的关系
- 展现了他不为人知的思考方式和行事逻辑
- 补充了他成为SCHOOL首领之前的部分经历

### 8.2 游戏作品中的设定

在各类《魔法禁书目录》游戏改编作品中，垣根帝督通常作为：
- 可操作角色（部分格斗游戏）
- BOSS角色
- 拥有独特技能树的高难度角色
- 其未元物质能力在游戏中常表现为：高伤害、多形态、攻击种类丰富
