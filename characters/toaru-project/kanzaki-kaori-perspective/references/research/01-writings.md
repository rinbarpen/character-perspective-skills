### 1.2 外观特征

神裂火织拥有十分显眼的外貌特征，在全系列角色中辨识度极高：

**基本造型：**
- 黑色长发，扎成高马尾，垂至腰际
- 身材高挑（175cm），匀称修长
- 面容冷峻端正，眼神锐利，符合武士/剑士形象

**标志性着装：**
- 白色T恤（下摆扎入牛仔裤中，强调腰线，突显身材线条）
- 蓝色牛仔裤
- **腰带上插着一柄长达2米的日本刀「令刀」——「七天七刀」（Nanasen / Seven Heavens Seven Swords）**
- 上臂绑有白色发圈（用来扎马尾）
- 脚穿运动鞋（便于战斗移动）

**官方衣着说明：**
- 作者镰池和马在初期设定中明确表示，神裂的着装设计是为了塑造"现代女剑士"的独特形象——将传统剑道元素与都市日常结合。黑色马尾和长刀是她的视觉核心符号。

> 外观在旧约、新约中基本保持一致，没有明显的阶段性变化（与一方通行、上条当麻不同）。

### 1.3 性格设定

神裂火织的性格呈现出「外在冷峻严肃 × 内在温柔守护」的明显张力：

| 核心矛盾 | 身为圣人拥有强大力量，却始终为"无法拯救所有人"而自责 |
| 社交定位 | 在必要之恶教会中与史提尔搭档，行动配合默契，但性格对比鲜明（冷vs热） |
| 对上条态度 | 旧约第4卷初登场时将上条视为敌人攻击，后因上条救了自己和天草式而欠下巨大的人情债 |

## 二、能力设定

### 2.1 圣人（Saint）体质

圣人是指在身体特征上与耶稣基督拥有相同「神子」印记的人类。全世界范围内仅存不到20人。

**圣人能力的核心机制：**
- 可以通过「偶像崇拜（Idol Theory）」理论，再现耶稣基督受难时所发挥的部分力量
- 身体能力获得飞跃性的强化——力量、速度、耐久力远超常人极限
- 能够短时间内获得等同于「天使」级别战斗力的加成
- 圣人的能力并非魔法术式，而是**身体本身**的特性

**圣人的限制：**
- 圣人化会给身体带来巨大负担，无法长时间维持
- 圣人体质对于特定类型的魔法攻击（尤其是针对神子性质的）反而存在克制弱点
- 在教会管理的世界中，圣人身份本身即是一种政治/军事威慑力

### 2.2 魔法术式——「七闪」与「唯闪」

神裂火织使用的剑术魔法以日本刀「七天七刀」为载体，融合了天草式十字凄教的独特技法。

#### 七闪（Nanana / Seven Glints）

| 项目 | 内容 |
|------|------|
| 类别 | 拔刀术 + 钢丝攻击的复合术式 |
| 原理 | 刀身不直接出鞘，而是利用**七根钢丝**进行高速切断攻击 |
| 速度 | 攻击速度超过音速，肉眼几乎无法捕捉 |
| 特征 | 钢丝的配置方式可以自由变换，攻击轨道极其复杂多变 |
| 威力 | 可以轻松切断钢筋、混凝土等坚硬物体 |
| 别名 | 被形容为"看不见的斩击" |

> 七闪是神裂在**不杀生**原则下使用的招式——钢丝切割可以造成压制而非致命伤。这也体现了她"不愿杀人"的信念。

#### 唯闪（Ittou / Only Flash / 唯闪）

| 项目 | 内容 |
|------|------|
| 类别 | 真正的拔刀术——圣人全力一击 |
| 原理 | 将圣人力量全部灌注于刀身，以一击决胜负 |
| 速度 | 远超七闪，被形容为"超越人类认知极限的一刀" |
| 威力 | 足以切断几乎一切物理存在（包括天使级别的防御） |
| 使用条件 | 必须在圣人化状态下全力施展 |
| 副作用 | 使用后会造成明显的身体负担和消耗 |

> 「唯闪」这个词本身暗示了"唯一的闪光"——神裂只有在真正需要一击决胜负时才会使用。

#### 令刀「七天七刀」
- 全长约2米，比普通日本刀更长
- 刀身纤细却异常坚固
- 并非普通的物理武器，而是经过术式加持的魔法武器
- 刀名「七天七刀」在原作中没有详细解释命名缘由，但从发音和文字来看可能关联佛教/神道概念

### 2.3 天草式十字凄教术式

作为天草式十字凄教的前女教皇，神裂对天草式的全部术式了如指掌：

- **融合术式**：天草式的最大特征是将佛教、神道教和基督教三种宗教的术式融合使用
- **隐形术式**：通过在日常物品（如手机、文具等）中隐藏符文来隐藏魔法阵营身份
- **协同术式**：多人协作发动的大型术式，神裂作为前女教皇精通指挥这一切

## 三、组织背景

### 3.1 天草式十字凄教（Amakusa Church）

天草式十字凄教是日本本土的一个隐秘基督教组织，在幕府禁教时期为躲避迫害而形成：

- 融合了**基督教**（教义核心）+ **佛教**（仪式形式）+ **神道教**（本地习俗）三种宗教元素
- 通过在外表上伪装成佛教寺院来逃避幕府的迫害
- 这种"多重宗教融合"的传统延续到了现代
- 神裂是第X代女教皇，领导着整个组织

根据官方设定，天草式在日本地下运作数百年，成员以伪装身份生活在社会中，在必要时集结行动。

### 3.2 英国清教·必要之恶教会（Necessarius）

必要之恶教会是英国清教麾下的**暗部组织**，专门负责处理魔法侧的危险事件和敌对势力：

- 与英国王室和英国清教最高主教萝拉·史都华保持直属关系
- 成员包括神裂火织、史提尔·马格努斯等专门战斗人员
- 组织代号的含义——"必要的恶"，意指为了更大的善而容忍并运用那些"邪恶的力量"

神裂离开天草式后加入必要之恶教会，目的之一是为了**从清教侧保护天草式**，避免天草式因为过去的历史被英国清教清算。

### 3.3 圣人身份的政治意义

神裂的圣人身份在全系列中有极其重要的政治意义：

- **战略威慑**：圣人相当于魔法侧的"核武器"，每一个圣人都是国家级的战略资产
- **英国清教的底牌**：英国清教拥有神裂这名圣人是其在魔法侧维持影响力的重要资本
- **国际均衡**：罗马正教拥有后方之水等多位圣人，英国清教的圣人力量是三教会谈中的重要谈判筹码

## 四、重要关联设定

### 4.1 圣人vs超能力者的战力标尺

神裂是全系列中"魔法侧"战力的重要参考系：

- **对上条当麻（旧约）**：没有幻想杀手时远强于上条，需要上条利用圣人机制弱点才能勉强抵抗
- **对后方之水（圣人之战）**：同为圣人，后方之水是完全成熟后的圣人形态，战力在神裂之上
- **对一方通行**：官方设定中未直接对比，但圣人与Level 5超能力者的战力比较是系列长期的讨论话题
- **对天使级敌人**：圣人化后的神裂有能力与天使级敌人短暂交锋

### 4.2 「裂神者」（God's Seat Breaker）

神裂的别称「裂神者」来源于她在战斗中的表现力——以凡人之身、以剑术切断神明级别的存在。这个称号并非官方能力等级，而是敌我双方对她战斗力的敬畏之词。

> 罗马正教相关人物曾评价：如果神裂完全发挥圣人力量，可能足以对抗神之右席（God's Right Seat）的某些成员。

## 五、作者访谈与设定补遗

根据镰池和马的访谈及后记信息：

1. **创作灵感**：神裂火织的角色设计融合了"女剑士"和"现代修女"两个反差元素。镰池和马曾表示，创作神裂的初衷是想刻画一个"既有强大力量又有内心柔软一面"的女性角色。

2. **名字由来**：「神裂」这个姓氏可能暗示"撕裂神明"的力量，与其称号「裂神者」呼应；「火织」与日本传统工艺有关，暗示天草式的日本背景。

3. **角色定位**：镰池和马在访谈中明确将神裂定位为"上条当麻理解魔法侧世界的第一扇窗户"——正是通过与神裂的冲突，上条第一次深入接触到了魔法侧的核心矛盾。

4. **圣人设定**：关于圣人只有不到20人的设定，镰池和马在访谈中表示这个数字是刻意控制的——圣人必须在故事中保持"极其稀有"的存在感，不能批量出现。
