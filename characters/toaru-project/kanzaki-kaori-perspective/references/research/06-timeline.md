# 06-timeline.md: 神裂火织 (Kanzaki Kaori) 时间线

> 涵盖：旧约（1~22卷）→ 新约（1~22卷）→ 创约至今的全主线出场记录

---

## 一、正式登场前——背景设定

| 时间 | 事件 | 备注 |
|------|------|------|
| 小说主线开始前多年 | 神裂火织出生于日本，成长过程中展现出特异的身体能力 | —— |
| 主线开始前 | 继承天草式十字凄教的女教皇之位 | 天草式是融合基督教+佛教+神道教的日本隐秘组织 |
| 主线开始前 | 被认定为「圣人」——全世界不到20人 | 因其身体中有与耶稣基督相似的神子印记 |
| 主线开始前 | 为了保护天草式不被英国清教清算，主动离开天草式，加入英国清教·必要之恶教会（Necessarius） | 这是她最重要的背景设定 |

> 神裂加入必要之恶教会的具体时间点不明，但可以肯定是在旧约第1卷之前数月到一年左右。

## 二、旧约篇（《魔法禁书目录》1~22卷）

### 2.1 旧约第4卷——天草式篇（核心登场卷）

| 剧情阶段 | 事件 | 备注 |
|----------|------|------|
| 开篇 | 神裂与史提尔一同出现在学园都市，目标是抹杀"使用禁书目录的魔法"的上条当麻 | 初登场 |
| 冲突 | 神裂对上条使用七闪，展示压倒性的战力差距 | 经典的"在路灯上跳跃"场景 |
| 对峙 | 上条在劣势中坚持不放弃，神裂开始动摇 | 对上条产生兴趣的起点 |
| 转折 | 天草式十字凄教被卷入阴谋，操纵天使术式事件爆发 | 事件核心 |
| 合作 | 神裂罕见地向上条求助：「我一个人做不到。救救天草式」 | 全系唯一的求助场景 |
| 高潮 | 神裂与上条联手对抗天使级威胁，神裂使用唯闪 | 圣人之力的全面展示 |
| 结局 | 天草式获救，神裂向上条道谢，确立"人情债"关系 | 「这份恩情我绝对不会忘记」 |

**本卷定位：** 神裂的核心女主角卷。旧约第4卷也是全系列公认的经典篇章之一，首次深入刻画了魔法侧的内部矛盾和世界观。

### 2.2 旧约第12~13卷——法国篇

| 事件 | 备注 |
|------|------|
| 神裂与天草式成员一同介入法国的魔法事件 | 显示天草式成员仍然追随前女教皇的行动 |
| 与上条当麻在法国再会 | 同盟关系的持续 |
| 保护上条免受罗马正教魔法师的袭击 | 履行承诺 |

### 2.3 旧约第16卷——后方之水登场

| 事件 | 备注 |
|------|------|
| 后方之水（William Orwell）——同时拥有圣人和圣母双重体质的最强圣人——登场 | 神裂面临同级别的强劲对手 |
| 神裂与后方之水的战斗虽然没有正面展开，但通过对比展现了圣人的战力层级 | 后方之水比神裂更接近"完全体圣人" |

### 2.4 旧约第19~22卷——第三次世界大战篇

| 剧情阶段 | 事件 |
|----------|------|
| 开战初期 | 神裂作为英国清教战力参与战争前线 |
| 战中 | 配合上条和茵蒂克丝的行动，保护重要人物 |
| 终局 | 在战争结束阶段参与善后工作 |
| 战后 | 与各方势力一起重建国际秩序 |

## 三、新约篇（《新约 魔法禁书目录》1~22卷）

### 3.1 新约第2~3卷——美国夏威夷篇

| 事件 | 备注 |
|------|------|
| 神裂与英国清教成员一同应对夏威夷发生的危机 | 作为必要之恶教会的标准出动 |
| 与上条当麻在当地会合 | 情报交换和战力协作 |

### 3.2 新约第4卷——魔术结社与学园都市

| 事件 | 备注 |
|------|------|
| 神裂参与对战多个魔术结社的干涉行动 | 展现其在国际魔法社会的活动范围 |
| 多次协助上条脱离危机 | 持续履行"保护当麻"的承诺 |

### 3.3 新约第9~10卷——奥帝努斯篇（魔神篇）

| 事件 | 备注 |
|------|------|
| 魔神奥帝努斯引发世界规模的大事件 | 神裂的层级已不足以直接对抗魔神 |
| 在上条当麻对抗奥帝努斯的过程中提供间接支援 | 见证上条对抗远超圣人级的敌人 |
| 面对魔神级力量时的无力感 | 标志着神裂"能完全保护重要的人"这一信念的挑战 |

### 3.4 新约第11卷——圣人会议

| 事件 | 备注 |
|------|------|
| 多位圣人（包括后方之水、布伦希德等）聚集的事件 | 神裂在圣人群体中的定位讨论 |
| 展现了圣人之间的合作与竞争关系 | 世界仅不到20人的特殊"俱乐部" |

### 3.5 新约第20~22卷——魔神与 Coronzon 篇

| 事件 | 备注 |
|------|------|
| 柯隆尊（Coronzon）大恶魔事件 | 魔法侧面临毁灭级别的危机 |
| 神裂参与对抗柯隆尊的势力 | 必要之恶教会全面投入 |
| 与众多协力者一同面对末日级威胁 | 跨阵营合作的体现 |

## 四、创约篇（《创约 魔法禁书目录》至今）

### 4.1 创约早期

神裂在创约中继续作为英国清教·必要之恶教会的战力活跃。截至目前（创约多个卷次），神裂的出场频率相较于旧约有所下降，但其作为圣人级战力和重要盟友的地位并未改变。

已知动向：
- 以英国清教所属魔法师身份处理魔法侧的各种事件
- 持续守护天草式成员
- 在必要时介入上条当麻面临的危机

> 创约仍在连载中，神裂的未来出场和角色发展有待后续内容补充。
