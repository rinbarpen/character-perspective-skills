> 本文基于《魔法禁书目录》系列官方作品及设定集整理，涵盖旧约、新约至创约的全部已知设定。

> 其能力代号与角色名均为「一方通行」，英文名 Accelerator（加速器），意指其拥有媲美粒子加速器的强大计算能力。

### 1.2 外观特征与变化

一方通行的外观经历了显著的演变过程：

**初期（旧约1-5卷）：**
- 黑发、黑底白条纹短袖T恤
- 皮肤苍白、身材瘦削
- 因常年开启「反射」能力，将紫外线等有害射线全部反射，导致身体缺乏色素合成

**中期（旧约6卷以后，脑损伤后）：**
- 头发因接受冥土追魂的治疗而变为白色
- 改穿白底灰条纹长袖T恤
- 右手拄拐杖辅助行走
- 脖子上戴有黑色项圈式电极（连接御坂网络用）
- 瞳孔因缺乏色素而透出血液颜色，呈现红色

**后期（新约/创约）：**
- 维持白发红瞳的经典形象
- 拐杖经过多次改造，内置干扰装置、小型马达和重量感应器
- 项圈电极持续升级

### 1.3 性格设定

一方通行的性格经历了从「自认恶党」到「赎罪者」再到「守护者」的深刻转变：

- **表层性格**：自称「无可救药的恶党」，态度冷漠、言语刻薄、作风张狂
- **内在本质**：内心深处对「善」有强烈的渴望，将自己贬低为恶，不愿承认内心柔软的自我防卫机制
- **对敌态度**：毫不留情、作风残忍大胆，但不会攻击不抵抗的小孩
- **核心转变**：遇到最后之作后，将其视为精神支柱，誓言守护御坂妹妹们
- **爱华斯评价**：「过去曾经犯下大错，被那些罪行折磨之下仍试图走上正途之人」
- **行为模式**：走的是「投身黑暗保护光明」的路线——身处暗部却守护着珍惜的人们

## 二、能力详解：矢量操作 (Vector Control)

### 2.1 基本原理

一方通行的能力核心是**操纵一切矢量的方向和大小**。只要经过皮肤接触（或在一定范围内的AIM力场覆盖区域），就可以自由操控动能、热能、电能等所有能量的方向。

**关键机制：**
- 操作前必须计算「变换前的方向」与「变换后的方向」
- 即使是默认的「反射」机能，也是在不自觉中进行着高度精密的演算
- 能力本质不仅是战斗用的矢量操控，更是一种**能够观测能量方向并逆向推算法则真相**的能力

### 2.2 默认防御：反射（常驻技能）

这是能力最基础的形态，也是他被称为「无敌」的根本原因：

- 能将接触皮肤外**数公分**处的任何能量方向进行反向操作
- **可反射的对象**：动能（子弹、冲击）、热能、电能、化学能、核能、磁能等一切已知能量
- **刻意不反射的对象**：
  - 重力（否则会被抛到大气圈外）
  - 可见光（否则全身会发光，无法视物）
- **特殊性质**：
  - 在**无意识状态下也能持续反射**（睡眠时也有效）
  - 身体**内部也能反射**（麻醉气体、生化武器无效）
  - 身上一尘不染、不需要洗澡
- **防御上限**：根据「树形图设计者」的运算，理论上连**核武器**也无法伤其分毫
- **唯一弱点**：核爆会消耗周围氧气，形成真空导致窒息

### 2.3 主动应用招式

| 招式名称 | 首次出现 | 说明 |
|---------|---------|------|
| **狂风之枪** | 旧约3 | 集中气流产生风速120m/s的大气团，威力如被汽车撞飞 |
| **等离子体（Plasma）** | 旧约3 | 压缩空气形成直径20公尺、超过1万摄氏度的超高温球体 |
| **龙卷风之翼** | 旧约8 | 背上形成四道龙卷风用于飞行，最高时速可达750公里以上（超越战斗机） |
| **血液操控** | 旧约3 | 通过伤口接触反转血液动量，可杀人也可止血治疗 |
| **生物电操控** | 旧约10 | 操控生物电流，可调整人格资料、反向解析他人大脑 |
| **伤ロ恢复** | 旧约8 | 改变自身组织电流信号促进愈合，也可促进毛发生长 |
| **地球自转炮** | 旧约5（提及） | 夺取地球自转约5分钟的能量作为攻击手段，使地球自转减慢 |
| **逆运算解析** | 多处 | 通过接触理解未知法则（如解析魔法原理、撕毁恶魔契约） |

### 2.4 能力限制

**脑部受伤前（全盛期）：**
- 拥有完全独立的演算能力，无时间限制
- 破坏力极强，曾被认为可以夺取地球自转能量

**脑部受伤后（8月31日事件后）：**
- 为了消除最后之作脑内的病毒，被天井亚雄枪击前额叶
- 失去了大部分演算能力和语言能力，无法正常使用矢量操作
- 在冥土追魂的帮助下装上**项圈型黑色电极**：
  - 通过电极连接「御坂网络」（约1万名御坂妹妹）进行代理演算
  - 恢复后的计算能力不到原来的一半
  - **能力使用时限**：最初15分钟 → 改良后30分钟
  - **通常模式续航**：48小时
  - 可通过专用充电装置补充电源
  - 为节省电源，平时会使用手枪作为武器
  - 受到干扰电波会影响使用，严重时无法站立或说话

### 2.5 能力的真正本质

一方通行的矢量操作不止是战斗用能力。其**更深层的本质**是：

> 能够观测能量方向并进行逆向推算，使产生能量的法则变得更加明确。通过这一能力，他可以解析完全未知的力量体系（如魔法），甚至能够逆向推算撕毁恶魔契约。

这一本质在他后续与魔法则的对抗中发挥了关键作用。

## 三、进化形态：翼

一方通行的「翼」是其能力觉醒（Awakening）的体现，本质是AIM扩散力场具象化形成的类天使力量。从黑翼到白翼再到铂金翼，代表了他在心境和力量上的层层突破。

### 3.1 黑翼

| 项目 | 内容 |
|------|------|
| **首次出现** | 旧约13卷（与木原数多战斗时不完全觉醒） |
| **完全觉醒** | 旧约15卷（与垣根帝督战斗时） |
| **外观** | 背后喷涌而出的黑色羽翼状有机物，全长可达100米以上 |
| **本质** | AIM扩散力场凝聚形成的类似「暗物质」的性质，与垣根帝督的能力有共通之处 |
| **特殊性质** | 翅膀能自主代替一方通行进行矢量演算，速度远超他的正常计算 |
| **使用代价** | 一方通行在此状态下会**失去理智**，无法自控，说出的语言变成**天使语** |
| **能力提升** | 可在不接触的情况下操纵远处的矢量，破坏力呈数量级增长 |
| **关联设定** | 根据Aiwass的说法，黑翼关联**欧西里斯世代（Aeon of Osiris）**；Index指出其能量性质与天使之力（Telesma）相似但本质不同 |

### 3.2 白翼

| 项目 | 内容 |
|------|------|
| **首次出现** | 旧约22卷（第三次世界大战篇，抵挡右方之火的天使之炮） |
| **外观** | 白色的羽翼，头顶出现**白色光环** |
| **本质** | 从「愤怒」向「守护」的心境转变后的稳定形态 |
| **特殊性质** | 能够操纵**虚构矢量（Imaginary Vector）**——即并非物理存在的抽象方向的力 |
| **能力提升** | 可将未知力量转化为浮力，瞬间飞上3000米高空 |
| **关键战绩** | 抵挡了足以**击沉大陆**的天使之力（Telesma炮击） |
| **心态对比** | 使用白翼时**不会失控**，可以保持清醒理智 |
| **关联设定** | 被称为「神にも等しい力の片鱗を振るう者」（挥舞近似神明之力的存在） |

### 3.3 铂金翼

| 项目 | 内容 |
|------|------|
| **首次出现** | 新约22卷（科隆尊篇高潮部分） |
| **外观** | 近乎透明的铂金色翅膀，散发蓝色光芒，形态与**爱华斯（Aiwass）** 的翅膀相似 |
| **触发条件** | 跨越深渊（Cross the Abyss），通过逆源质拼图545的协助与御坂网络的连接 |
| **核心机制** | 将**第三棵树「克洛诺斯（Clonoth）」** 嵌入世界 |
| **特殊性质** | 能够操纵**概念层面的矢量**——灵魂、神性、法则等原本没有「方向」的存在 |
| **关键战绩** | 反弹科隆尊的**「烈焰之剑」（Flaming Sword）**——一种卡巴拉概念攻击；将大恶魔科隆尊的灵魂与肉体分离 |
| **关联设定** | 克洛诺斯是区别于生命之树（Sephiroth）和邪恶之树（Qliphoth）的第三棵人造树，代表由人类科技文明开辟的可能性 |

### 3.4 天使化本质

一方通行的「翼」与魔法侧的「天使」有着本质上的关联和区别：

- **AIM扩散力场**：翼的本质是超能力者的AIM力场发生了类似于「天使」的质变
- **与天使之力的区别**：Index分析黑翼时指出，其能量的确与天使之力（Telesma）相似，但根源完全不同，可能关联到**恶魔学**领域
- **跨体系力量**：一方通行通过超能力体系达到了与魔法侧天使同等的存在层次，这是学园都市的「人工天使」设想的具体体现

## 四、御坂网络与最后之作

### 4.1 御坂网络

御坂网络是由约20000名御坂妹妹（御坂美琴的克隆体）通过脑部电信号构成的共有意识网络。

- **网络规模**：最初制造约20000名，实际投入约10031名用于绝对能力者进化计划
- **网络功能**：所有妹妹共享感官、记忆和计算资源
- **等级**：大多数妹妹能力为Level 2的「缺陷电力」；远程操控需特定权限
- **AIM扩散力场作用**：妹妹们被送往世界各地后，其AIM力场形成全球网络，成为亚雷斯塔·克劳利构建「人工天界」的基础

### 4.2 最后之作 (Last Order)

最后之作者是御坂妹妹中的**第20001号**，是所有妹妹的**司令塔和管理者**：

- **制造目的**：作为安全装置管理整个御坂网络，防止妹妹们反叛或失控
- **命令权限**：其命令位阶高于一般妹妹，可通过脑部电流信号从外部操纵整个御坂网络
- **外表特征**：约10岁左右的外表，与其他妹妹不同，表情丰富、性格活泼、有显著的呆毛，自称使用「御坂御坂」口癖
- **能力**：Level 3的「缺陷电力」（未完成体，原本不应离开培养器）

### 4.3 一方通行与最后之作的关系

这段关系是一方通行角色成长的核心驱动因素：

| 阶段 | 内容 |
|------|------|
| **相遇前** | 一方通行在绝对能力者进化计划中杀害了10031名御坂妹妹，是纯粹的「恶党」 |
| **8月31日事件** | 最后之作脑中被植入病毒后逃出研究所，主动找到一方通行求助；一方通行为拯救她而脑部中枪，从此命运绑定 |
| **共生关系** | 最后之作用御坂网络的管理权限弥补了一方通行的能力损失；一方通行则成为妹妹们最坚定的守护者 |
| **精神支柱** | 最后之佐成为一方通行「唯一的心灵支柱」和「最重要的守护对象」 |
| **关系本质** | 更接近父女般的亲情羁绊，他说的「我想永远和你在一起」是基于家人般的承诺 |

## 五、进化成长轨迹

### 5.1 三大阶段

**第一阶段 —— 不可一世的No.1（旧约1-3卷）**
- 被学园都市当作实验工具操控而不自知
- 狂妄自大，以「最强」自居
- 为追求Level 6而参与屠杀上万个御坂妹妹
- 被上条当麻击败后开始动摇

**第二阶段 —— 跌入谷底的赎罪者（旧约5-13卷）**
- 为救最后之作脑部中枪，失去大部分能力
- 被迫依赖御坂网络和项圈电极，能力使用受时间限制
- 加入暗部组织「集团（Group）」，在黑暗中挣扎生存
- 学会低头和动脑，不再仅凭力量碾压

**第三阶段 —— 跨越深渊的守护者（旧约14卷至今）**
- 觉醒黑翼、白翼、铂金翼，逐步突破自身极限
- 从被操控者成长为学园都市统括理事长
- 完成从「杀戮者」到「守护者」的终极蜕变

### 5.2 关键事件时间线

| 卷数 | 事件 | 意义 |
|------|------|------|
| 旧约3 | 与上条当麻初战，被击败 | 第一次被彻底打败，三观开始动摇 |
| 旧约5 | 拯救最后之作，脑部中枪 | 失去演算能力，装上电极，命运转折点 |
| 旧约8 | 加入暗部「集团」 | 开始在系统内部寻求改变 |
| 旧约10 | 击败木原数多 | 首次主动保护他人，心境转变加速 |
| 旧约13 | 木原数多来袭，黑翼初现 | 能力觉醒的起点 |
| 旧约15 | 与垣根帝督死战，黑翼完全觉醒 | 完整掌握黑翼，意识进入新层次 |
| 旧约22 | 俄罗斯篇，白翼觉醒 | 从愤怒的破坏到坚定的守护的升华 |
| 新约6 | 再次击败垣根帝督（第二战） | 确立守护者身份的坚决立场 |
| 新约22 | 科隆尊篇，铂金翼觉醒 | 达到与魔法则最高层级抗衡的高度 |
| 创约 | 亚雷斯塔临终委任为学园都市理事长 | 从棋子变为执棋者 |

### 5.3 最终定位：学园都市第二任统括理事长

在新约篇末尾，亚雷斯塔·克劳利在弥留之际将学园都市的统括理事长之位托付给了一方通行。这标志着：

- 一方通行从「被实验操控的小白鼠」变为了「管理整个系统的人」
- 他发动「手铐行动」，以极其果断的手段清除学园都市的暗部势力
- 为了肃清黑暗，不惜自愿入狱，以自身为诱饵引出所有敌人

## 六、角色定位与主题

### 6.1 魔法禁书目录中的结构性定位

一方通行是与上条当麻并列的**三主角体系**中的重要一柱：

- **上条当麻**：以「幻想杀手」抹消异常，维系日常与非日常的边界
- **一方通行**：以「矢量操作」在科学阵营内部与黑暗斗争，从内部改革系统
- **滨面仕上**：以「不入流的幸运」搅动局势，作为底层视角的代表

### 6.2 核心母题：救赎与守护

一方通行的角色弧贯穿了以下核心主题：

1. **罪恶与赎罪**：无法原谅杀害上万妹妹的自己，通过守护她们来寻求救赎
2. **暴力与守护**：走的是「以破坏来保护」的道路，用最残酷的手段达成最温柔的目的
3. **孤独与羁绊**：从因能力而被迫孤立的「白色怪物」，到为守护重要之人而战的人类
4. **善与恶的自省**：自居为「恶党」，却处处行保护之实；内心憧憬上条当麻式的英雄主义，却选择了一条截然不同的黑暗道路

### 6.3 与其他角色的关系

| 人物 | 关系性质 |
|------|---------|
| **上条当麻** | 憧憬对象、亦敌亦友的存在。一方通行将其视为「英雄」，笨拙地模仿着他的维护他人的决心 |
| **御坂美琴** | 因杀害妹妹们而背负罪孽的关系，两人有着痛苦的间接联系 |
| **最后之作** | 心灵支柱、类似于家人的存在，他一切守护行为的原点 |
| **番外个体** | 集合了所有御坂对一方通行的恨意而诞生的个体，与他形成「原谅与仇恨」的两极对照 |
| **黄泉川爱穗** | 母亲般的存在，少数像对待普通孩子一样对待他的人 |
| **垣根帝督** | 同为Level 5的宿敌，在理念和力量层面持续对抗 |
| **亚雷斯塔·克劳利** | 棋子与棋手的关系最终升格为委托与继承的关系 |

## 七、常见设定澄清

| 问题 | 解答 |
