# 佐天泪子 时间线

## 登场前背景

| 时间 | 事件 |
|------|------|
| 约13年前 | 出生于学园都市外的普通家庭 |
| 童年时期 | 告诉哥哥自己会获得超能力，立下进入学园都市的目标 |
| 入学前 | 母亲虽有顾虑但最终支持她的决定，赠予护身符。泪子将其挂在包上一直保留至今 |
| 入学时 | 进入栅川中学，与初春饰利成为同班同学兼好友。能力鉴定为 Level 0 |

## 《某科学的超电磁炮》第一季（2009-2010）

### 初登场 ~ 日常篇（第1-2集）

| 集数 | 事件 |
|------|------|
| 第1集 | 在可丽饼店与初春一起首次见到御坂美琴。起初以为 Level 5 会很高傲，但看到美琴阻止银行抢劫、救下孩子后改变看法。试图阻止抢劫犯时受轻伤 |
| 初期 | 成为四人组（美琴、黑子、初春、泪子）的一员。展现对都市传说的兴趣和掀初春裙子的标志性行为 |

### 幻想御手篇（第3-12集）

| 集数 | 事件 |
|------|------|
| 第6-8集 | 逐渐注意到身边好友各自拥有能力，内心滋生羡慕。尝试接触关于能力提升的传闻 |
| 第9集 | **从网络暗链获得幻想御手音频文件**。陷入"用还是不用"的挣扎。目睹黑子为保护他人受伤却无能为力，最终决定使用 |
| 第9-10集 | 使用幻想御手后获得"空力使"能力（能在手掌上吹动树叶）。又邀请好友真子、明美、小麦一起使用 |
| 第10集 | 好友明美陷入昏迷。泪子在电话中哭着向初春坦白一切，深深自责。初春赶到前，泪子也陷入意识不明状态 |
| 第11集 | 持续昏迷。木山春生被揭露为幻想御手事件的幕后黑手 |
| 第12集 | 美琴击败木山春生和 AIM 聚合体后，泪子苏醒。在医院天台上与初春重逢，道歉并说"差点失去比能力更重要的东西"。立即恢复掀初春裙子的日常行为，四人组重聚 |

### 补习与成长（第13-14集）

| 集数 | 事件 |
|------|------|
| 第13-14集 | 作为幻想御手使用者参加由月咏小萌老师主持的补充讲习。参加体力训练，面对高年级学生的嘲讽坦然接受。在小萌老师的教导下重新思考"个人现实"的意义。拿到依然显示 Level 0 的成绩单时露出释然的微笑。**决定继续努力，但不再执着于"获得能力"** |

### Big Spider 篇（第15-16集）

| 集数 | 事件 |
|------|------|
| 第15集 | 在咖啡厅与大家讨论 Big Spider 袭击事件。美琴说讨厌轻易放弃的人，泪子觉得像是在说自己。对初春承诺"再也不会放弃" |
| 第16集 | 对黑妻大绵和固法美伟的过去产生好奇。拜访固法美伟的公寓了解真相。对美琴说"不要只根据过去来评判一个人" |

### Poltergeist 篇（第19-24集）

| 集数 | 事件 |
|------|------|
| 第19集 | 与春上衿衣成为朋友，帮忙修补浴衣。在聚会中展现出色的手工能力 |
| 第20-22集 | 在 Poltergeist 事件中调和黑子和初春因意见不合产生的矛盾 |
| 第23集 | 发现泰瑞丝缇娜·木原·莱夫莱恩的真实身份。强迫黑子和初春互相道歉 |
| 第24集 | **关键转折**：装备棒球棒。在 Capacity Down 让所有能力者失效时，因自己是 Level 0 不受影响。初春通过耳机指示音响系统位置，泪子冲锋砸毁装置，让美琴能够释放能力。**黑子第一次真正认可泪子** |

## 时间线特点总结

1. **主要活跃平台**：动画《超电磁炮》才是泪子的主要舞台，在漫画和本传《魔禁》中出场较少
2. **角色发展曲线**：核心成长集中在第一季幻想御手篇，后续篇章更多地展示她"已经成长后"的状态——从渴望能力→接受自己→用自己的方式帮助他人
3. **两条叙事线**：
   - **明线**（动画呈现）：从日常→幻想御手→Poltergeist→Silent Party→大霸星祭→Dream Ranker
   - **暗线**（社交媒体与暗部的交集）：通过 FUKIDASHI 与芙兰达的友谊、印第安扑克事件——她总是在不经意间触碰到学园都市暗面的边界
4. **与芙兰达的悲剧**：从 Dream Ranker 篇开始的这条支线至今没有向泪子揭示真相，她在时间线的当下仍然在等待一个已经死去的人
5. **持续的主题**：从幻想御手到狱门脱出，泪子始终在证明"**普通人也能在关键时刻改变局势**"这一命题
