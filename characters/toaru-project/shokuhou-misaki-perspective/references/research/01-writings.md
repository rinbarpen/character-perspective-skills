> 来源：小说第2卷初登场描写、动画第2季第1集

### 1.3 性格特征

- **表面人设**：优雅、从容、装可爱的大小姐形象 —— 这是她刻意经营的面具
- **真实性格**：心思缜密、城府极深、极度执着、在关键时刻展现惊人的决断力和善意
- **标志性动作**：拨弄前发、用遥控器指着对方、发出「ふふん」的笑声
- **说话方式**：使用「ですわ」「ますわ」等优雅大小姐口吻，但在情绪激动时会崩坏
- **核心执念**：对上条当麻（她称其为「上条さん」）的病态执着，是所有行动的深层动机

> 来源：小说新约第4卷、第7卷；『とある科学の超電磁砲』漫画第69话以后

## 二、能力设定

### 2.1 能力总览

| 项目 | 数据 |
|------|------|
| **能力名** | 心理掌握（Mental Out / メンタルアウト） |
| **能力等级** | **Level 5（超能力者）** |
| **排名** | 学园都市七名Level 5中 **第5位** |
| **能力类型** | 精神系（念动能力分支） |
| **媒介** | 遥控器（通过电流操控脑内物质） |

> 来源：小说第2卷、『とある魔術の禁書目録』公式书

### 2.2 能力详解

心理掌握（Mental Out）被官方描述为「学园都市最强的精神系能力」。其本质是**通过电流信号控制大脑内的生物电信号，从而操控他人的精神**。遥控器是她为自己设计的聚焦媒介，不通过遥控器也能使用，但精度和范围会下降。

**已知应用分支（在小说中明确出现过）：**

| 能力分支 | 效果 | 出处 |
|----------|------|------|
| **精神感应（Telepathy）** | 读取他人思维（浅层/深层） | 小说SS2 |
| **记忆读取** | 强制读取目标记忆 | 小说第2卷 |
| **记忆篡改** | 删除、修改、植入记忆 | 小说第2卷、新约第10卷 |
| **记忆写入** | 将假记忆写入目标大脑 | 『超電磁砲』漫画第71话 |
| **思考诱导** | 暗示、催眠、影响判断 | 小说第2卷 |
| **感情操纵** | 改变目标情绪状态 | 小说第2卷 |
| **洗脑/完全控制** | 完全夺取身体控制权 | 小说第2卷 |
| **五感操控** | 改变目标所见所闻所感 | 小说第2卷 |
| **人格覆写** | 暂时或永久改写人格 | 小说第2卷 |
| **计算力外部化** | 将人脑作为外部计算设备使用 | 小说新约第10卷 |
| **远程精神侵入** | 跨越物理距离操控目标 | 『超電磁砲』漫画第105话 |
| **心理屏障（Mental Barrier）** | 建立防御性精神屏障 | 小说新约第10卷 |
| **精神攻击** | 直接通过精神力量进行攻击 | 小说第2卷 |

> 来源：小说第2卷、SS2、新约第4卷、新约第7卷、新约第10卷、『とある科学の超電磁砲』漫画

### 2.3 能力限制

- **对象限制**：能力无法对**大脑构造异于常人者**起效（例如幻想杀手——上条当麻并非其能力无效化，而是其意识/精神构造特殊；对同样拥有强大精神能力的对手也效果受限）
- **距离限制**：距离越远，操控精度越低
- **时间限制**：持续操控需要集中注意力
- **缺陷**：对自己无法使用能力，也无法对自己进行精神操作

> 来源：小说新约第4卷、新约第7卷

## 三、身份与地位

### 3.1 常盘台的女王

食蜂操祈在常盘台中学建立了自己的派阀（通称「食蜂派阀」），与御坂美琴为首的派阀形成微妙的对峙关系。她在学校中被称为「女王」（Queen），拥有极高的地位和影响力。

- 派阀成员多为常盘台名门女学生，对食蜂忠心耿耿
- 通过自己的能力（微妙地）维系派阀内部的团结
- 与「常盘台的王牌」御坂美琴表面保持礼貌，实质关系紧张
- 是常盘台学生中少数几个不以努力型能力者身份闻名，而是以绝对的精神控制力闻名的存在

### 3.2 学园都市高层关系

食蜂操祈与学园都市的暗面有着深层次的联系：
- 统括理事会中的某些成员与她有秘密联系
- 她知晓学园都市暗部的许多秘密
- 她的能力使其成为学园都市高层既想利用又恐惧的存在
- 参与了「Dolly」（多莉）克隆人计划相关事件

## 四、角色定位

食蜂操祈是《魔法禁书目录》系列中**最复杂的女性角色之一**。她的角色定位呈现多重性：

1. **表面反派**：初登场时作为能力者犯罪事件的幕后黑手，给人反派印象
2. **灰色角色**：随着剧情展开，她的行动逻辑逐渐展现——她不是纯粹的恶人，而是有自己的道德准则和执着目标
3. **悲剧性人物**：失去关键记忆（对上条当麻的记忆被消除后又重新靠自己找回）的经历赋予她厚重的人物弧光
4. **暗面的知情者**：她是学园都市中少数同时了解科学侧暗面和表面世界运作的角色之一

> 来源：小说新约第4卷、第7卷、『とある科学の超電磁砲』漫画「Astral Buddy」篇
