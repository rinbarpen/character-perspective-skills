# 食蜂操祈 经典台词与对话

> 本文整理自《魔法禁书目录》原作小说、动画及《科学超电磁炮》漫画/动画中的食蜂操祈重要台词。
> 原文为日文，附中文翻译。出处标注卷/话/集。

---

## 一、对上条当麻——「上条さん」系列

食蜂操祈对上条当麻的称呼「上条さん」是她最具辨识度的语言标志。在整个人物体系中，**只有她一个人用这个称呼方式**，这种独特性本身就是重要的角色符号。

### 1.1 初识/记忆尚存时期的称呼

**场景**：食蜂最初与上条相遇时，彼时的上条还记得她。

```
「上条さん、助けてくださいませんか？」
（上条先生，您能帮帮我吗？）
```
> 出处：小说新约第7卷 食蜂的回忆场景

### 1.2 跨越记忆的呼唤

**场景**：上条失去与食蜂相关的记忆后，食蜂仍然坚持用「上条さん」称呼他，即使对方已经不记得她。

```
「——上条さん」
```
> 出处：小说新约第4卷 初登场。仅一个称呼，承载了无数未说出口的情感

### 1.3 新约第7卷——食蜂视角的告白式独白

**场景**：新约第7卷中，食蜂独自面对自己的内心，揭示了她的感情真相：

```
「上条さんは、私のことを覚えていない。それでも構わないんです。
だって、私は上条さんのことを覚えているから。」
（上条先生不记得我。但那也没关系。
因为，我记得上条先生的事情。）
```
> 出处：小说新约第7卷 食蜂独白

```
「たとえ世界の誰もが忘れても、私は絶対に忘れない。
上条さんに助けられたあの日のことを。」
（就算世界上所有人都忘了，我也绝对不会忘记。
上条先生救了我的那一天的事情。）
```
> 出处：小说新约第7卷

### 1.4 对上条当麻的恳求

**场景**：在某个危急时刻，食蜂对上条发出请求：

```
「お願いです、上条さん。私を、思い出してください。」
（求求您了，上条先生。请想起我吧。）
```
> 出处：小说新约第10卷

## 二、与御坂美琴的对峙——「女王VS王牌」

### 2.1 初次正式见面（超电磁炮漫画）

**场景**：食蜂与美琴在常盘台走廊首次正面交锋：

```
食蜂：「あら、御坂さん。今日もご機嫌よう。」
美琴：「……食蜂。」
食蜂：「そんなに怖い顔をしないでくださいまし。同じ常盤台の——」
美琴：「同級生だからって、仲良くしなきゃいけない義務はないわ。」
```
> 出处：『とある科学の超電磁砲』漫画第69话

### 2.2 女王式挑衅

```
「同じLevel 5でも、御坂さんは"努力型"、私は"才能型"。
この差は、いくら努力しても埋められないんですのよ？」
（同样是Level 5，御坂小姐是"努力型"，而我呢，是"才能型"。
这个差距，可不是靠努力就能弥补的哦？）
```
> 出处：『とある科学の超電磁砲』漫画第70话（含 paraphrased 还原）

### 2.3 临时合作时的对话

**场景**：面对共同威胁，两人不得不合作：

```
美琴：「……信じないわよ。」
食蜂：「同じですわ。私も御坂さんのことは信じておりませんもの。
でも——今は、それどころではないのでしょう？」
```
> 出处：小说新约第10卷

### 2.4 「エア」の呼び方

食蜂有时会用轻蔑的口吻称呼美琴为「エア」（Ea，暗示"空气"或轻蔑的爱称），引发美琴的强烈不满：

```
食蜂：「あら、エア。あなたも来ていたの。」
美琴：「エアって呼ばないで！」
```
> 出处：动画『とある科学の超電磁砲T』第1集

## 三、操控他人时的宣言——「心理掌握」

### 3.1 招牌能力发动宣告

```
「——心理掌握（メンタルアウト）」
```
> 出处：小说第2卷初登场时。简短、冷静，不带任何多余修辞

### 3.2 操控前的从容

**场景**：在操控对手之前，食蜂展现出压倒性的从容：

```
「私の前で隠し事はできませんわよ。
だって——あなたの脳内は、私の掌の上ですもの。」
（在我面前是藏不了秘密的哦。
因为——你的大脑，就在我的掌心之中呢。）
```
> 出处：小说第2卷

### 3.3 对敌人的宣告

```
「抵抗しても無駄ですわ。あなたの思考は、もう私のものです。」
（抵抗也是没用的。你的思考，已经是我的东西了。）
```
> 出处：动画『とある魔術の禁書目録』第2季第2集

### 3.4 能力的自信

```
「この学園都市で、私の精神操作に耐えられる人間は———
指で数えるほどしかいませんわ。」
（在这个学园都市里，能抵抗我精神操纵的人——
用一只手就数得过来呢。）
```
> 出处：小说SS2

## 四、女王式发言——优雅中的锋芒

### 4.1 派阀领袖的威严

**场景**：面对挑战者或不受控的学妹：

```
「わたくし、面倒なことは大嫌いですの。
ですから——おとなしく従っていただけませんこと？」
（我呢，最讨厌麻烦的事情了。
所以——能不能请你乖乖听话呢？）
```
> 出处：『とある科学の超電磁砲』漫画第70话

### 4.2 对常识的蔑视

```
「常識？そんなもの、他人の頭の中にだけあれば十分ですわ。」
（常识？那种东西，只要存在于别人的脑子里就够了哦。）
```
> 出处：小说新约第7卷

### 4.3 优雅的威胁

```
「わたくしを怒らせると、どうなるか——
……想像してみてくださいまし。」
（把我惹怒了会怎么样呢——
……请试想看看吧。）
```
> 出处：『とある科学の超電磁砲』漫画第71话

## 五、情绪失控时的台词

### 5.1 对上条当麻的绝望呼喊

**场景**：当她意识到自己关于上条的珍贵记忆被动摇时：

```
「やめて……やめてください……
それだけは、それだけはやめて……！」
（不要……请不要……
只有那个，只有那个求求你不要……！）
```
> 出处：小说新约第10卷

### 5.2 语气崩坏的真实面

当食蜂失去从容时，她的大小姐口吻会崩坏，暴露出本音：

```
「うるさいうるさいうるさい！もうほっといて！」
（吵死了吵死了吵死了！别管我了啦！）
```
> 出处：『とある科学の超電磁砲』漫画第105话

### 5.3 哭泣时的独白

```
「……ずるいですよ、上条さん。
あなただけ、何も覚えていないなんて。」
（……太狡猾了，上条先生。
只有你一个人，什么都不记得了。）
```
> 出处：小说新约第7卷

## 六、与蜜蚁爱愉的对话

### 6.1 昔日友情的碎片

```
蜜蚁：「操祈、まだあの実験のこと、気にしてるの？」
食蜂：「……忘れたことにしてくださいまし。」
```
> 出处：『とある科学の超電磁砲』「Astral Buddy」篇

### 6.2 对过去的告别

```
「過去は変えられませんわ。でも——未来は、私の掌の上です。」
（过去是无法改变的。但是——未来，就在我的掌心之中。）
```
> 出处：小说新约第10卷

## 七、标志性语气词与口头禅

| 表现 | 含义 | 使用场景 |
|------|------|----------|
| 「ですわ」「ますわ」 | 优雅大小姐语尾 | 日常对话 |
| 「あら」 | "哎呀"——表示惊讶或轻蔑 | 遇到意外情况 |
| 「ふふん」 | 得意的轻笑 | 自信/嘲讽时 |
| 「〜ですの」 | 略带疑问的优雅反问 | 挑衅对手时 |
| 「上条さん」 | 独有的称呼方式 | 所有人中只有她这样叫 |
| 「エア」 | 对美琴的蔑称 | 挑衅美琴时 |

> 来源：综合小说、漫画、动画语料整理
