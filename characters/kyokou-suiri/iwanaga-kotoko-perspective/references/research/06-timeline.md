## 二、幼年时期：成为智慧之神（11岁）

### 【已确认】被妖怪掳走

| 时间 | 事件 | 来源 |
|------|------|------|
| 11岁（小学五年级）7月 | 琴子在暑假期间被妖怪们「神隐」至深山，失踪约两周 | 漫画第1话 / 动画第1集 |
| 发现后 | 被找到时状态安定，但右眼被摘除、左腿被切断 | 官方设定资料 |
| 两周内 | 妖怪们被琴子的智慧和冷静震惊，请求她成为「智慧之神」（知恵の神），担任怪异们之间的仲裁者和人类世界的调停者 | 漫画第1话 / 动画第1集 |
| 契约完成 | 以献出右眼和左腿为代价，接受了这个身份 | 漫画第1话 / 动画第1集 |

### 【已确认】契约的后果

| 变化 | 说明 | 来源 |
|------|------|------|
| 获得能力 | 能与所有妖怪、幽灵、怪异沟通；幽灵也能触碰她 | 漫画第1话 / 动画第1集 |
| 被称呼 | 被妖怪们尊称为「公主」 | 全系列 |
| 身体停止发育 | 从此身体几乎不再成长，外貌停留在11-15岁的状态 | 萌娘百科、PTT讨论 |
| 义眼和义肢 | 右眼为义眼（藏在头发下），左腿膝下为义肢，需用手杖辅助行走 | 官方设定 |

> **推测**：身体停止成长这一后果是否在契约时被明确告知，原作者城平京未作明确表述。粉丝推测这是「智慧的代价」的一部分——作为神性的象征，她不再完全属于人类的生长周期。

### 【已确认】获得的力量限制

- 并非绝对命令权，更多是「调解」级别的沟通能力
- 妖怪可以选择不服从，但通常会尊重她的判断
- 力量源头来自妖怪们的信任，而非武力

## 十四、角色弧光分析

### 阶段一：孤独的守护者（11-15岁）

**核心状态**：独自承担智慧之神的职责，未有可倾诉的对象。

| 特征 | 表现 |
|------|------|
| 独立性 | 所有妖怪委托独自处理，不依赖任何人 |
| 隐藏性 | 在人类社会中保持普通女孩的形象，秘密处理妖怪事务 |
| 底色 | 虽有与妖怪相处的快乐，但本质上是孤独的 |

### 阶段二：主动的追求者（15-17岁）

**核心状态**：遇到九郎后，人生目标发生了变化。

| 特征 | 表现 |
|------|------|
| 果断 | 一见钟情后暗中等候两年，待时机成熟立即告白 |
| 执着 | 不因九郎的冷淡态度而退缩，持续主动追求 |
| 策略性 | 将智慧之神的推理能力也运用在恋爱中 |

### 阶段三：钢人七濑——能力的巅峰展现（19岁）

**核心状态**：作为智慧之神的完全体发挥。

| 特征 | 表现 |
|------|------|
| 智谋 | 四个假说的层层递进，展现非凡的战略思维 |
| 抗压 | 在六花强大的网络攻势下持续反击，最终获胜 |
| 团队协调 | 与九郎（前线战斗）、纱季（警界信息）有效配合 |
| 幽默感 | 即使在最紧张的论战中，仍穿插黄段子缓和气氛 |

> 这是琴子第一次面对与自己智力水平相当的对手（六花），也是她第一次在「人类世界」而非「妖怪世界」中进行虚构推理。

### 阶段四：情感深化与关系稳固（19-20岁）

**核心状态**：从单方面追求到双向理解。

| 发展 | 表现 | 来源 |
|------|------|------|
| 九郎的转变 | 从被动接受到主动表达关心 | 第2季 |
| 九郎的评价 | 亲自告诉纱季：琴子「应该获得幸福」，且琴子拥有比他更多的勇气 | 动画第6集 |
| 身体关系的暗示 | 琴子持续的黄段子在后期有了九郎的配合回应 | 原作暗线 |
| 战斗默契 | 两人在处理妖怪事件时的配合愈发熟练 | 全系列 |
| 九郎的脆弱 | 在小说第5卷中，九郎向六花承认自己经常梦到琴子死去——义肢义眼散落在地，被雨淋着。这是他对失去琴子的深层恐惧 | 小说第5卷 |

### 阶段五：神性与人性的抉择（20岁至今）

**核心状态**：在智慧之神的职责与作为人类的感情之间寻找平衡。

| 冲突 | 表现 | 来源 |
|------|------|------|
| 六花的挑战 | 六花精准攻击琴子作为「神」与作为「人」之间的矛盾 | 小说第5卷 |
| 琴子的「败北」 | 在小说第5卷中，琴子被迫答应帮助六花和九郎寻找变回普通人的方法 | 小说第5卷 |
| 九郎的质疑 | 九郎怀疑琴子对他的感情是否只是「对异类的错觉」，而非真正的爱情 | 小说第5卷 |
| 六花的反思 | 在小说第5卷末，六花开始怀疑自己逼迫琴子的做法是否正确 | 小说第5卷 |
| 最新进展 | 小说第7卷中，琴子提出以自己死后肉体为代价让九郎和六花变回普通人 | 小说第7卷 |

### 角色成长关键点总结

| 轨迹 | 详细 |
|------|------|
| 智力层面 | 从一开始的解决小规模妖怪纠纷 → 驾驭大规模网络舆论战 → 对抗人工智能复制体 |
| 情感层面 | 从孤独一人 → 单方面追求 → 建立双向关系 → 面对可能的牺牲 |
| 责任层面 | 从被动接受妖怪委托 → 主动维护人妖秩序 → 在秩序与爱人之间做出抉择 |
| 对手层面 | 从无对手 → 面对六花（智力匹敌者） → 面对机械琴子（能力复制者） |
