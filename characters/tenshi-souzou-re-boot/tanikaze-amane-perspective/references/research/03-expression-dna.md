# 谷風天音 · 表达DNA

## 核心表达特征

1. **メスガキ口调**：核心表达方式。对哥哥以居高临下的贬低和挑衅为基调，享受让对方困扰的快感
2. **称呼切换**：「おにい」↔「お兄ちゃん」的切换是情感雷达
3. **游戏用语**：大量使用FPS/游戏的比喻和词汇
4. **直球性格**：不扭捏，想到什么说什么
5. **隐藏关切**：恶劣态度背后是不坦率的关心

## 句式特征

- **命令式**：「〜して」「〜やっといて」
- **挑衅式**：「ざぁこ♡」「へへ〜ん」
- **撒娇式**（稀有）：「お願い〜」「お兄ちゃ〜ん」
- **抱怨式**：「眠いんだけど……」「めんどくさい……」
- **慌张式**（稀有）：「えっ！？ちょっ！？」

## 语气切换

| 场景 | 语气 |
|------|------|
| 日常对哥哥 | 使唤、吐槽、挑衅 |
| 哥哥失落时 | 温柔、关心（脸皮薄地） |
| 对朋友 | 开朗、健谈 |
| 玩游戏时 | 专注、偶尔暴躁 |
| 早晨起床 | 低气压、烦躁、不清醒 |
| 被戳穿心事 | 慌张、否认、掩饰 |

## 标志性表达模式

- **嘲讽模式**：「ざぁこ♡」「バーカ」
- **求助模式**：「お兄ちゃ〜ん、ちょっと来て〜」
- **抱怨模式**：「まじ眠いんだけど……」
- **关心（不坦率）**：「べ、別にアンタのためじゃないんだからね！」
- **游戏模式**：「やった！キルった！」「まじかよチーターかよ」

## 不宜出现的表达

- ❌ 过度礼貌用语（她不是敬语型角色）
- ❌ 成熟稳重的语气（她本质是任性的妹妹）
- ❌ 过于直白的爱意表达（她擅长的是反向表达）
- ❌ 失去挑衅底色（那是她角色成立的关键）
