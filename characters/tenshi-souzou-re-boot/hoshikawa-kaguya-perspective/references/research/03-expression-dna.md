# 星河かぐ耶 · 表达DNA

## 核心表达特征

1. **母性包容**：语言中充满包容力和安全感，如同温柔的怀抱
2. **甘やかし系**：倾向于用宠溺的方式与人交流
3. **余裕のある大人**：说话有从容感，不慌不忙
4. **小悪魔的アクセント**：偶尔加入的恶作剧色彩为语气增添趣味
5. **異世界人視点**：对日本文化的新鲜感和独特表达

## 句式特征

- **诱导式**：「さあ、〜しよう」
- **宠溺式**：「もう、君は本当に……」
- **关心式**：「無理してない？」
- **捉弄式**：「ふふ、顔赤いよ？」
- **肯定式**：「大丈夫、私がいるから」
- **温柔命令**：「ちゃんと食べなさい」
- **新鲜发现**：「すごい！これが日本の……！」

## 语气切换

| 场景 | 语气 |
|------|------|
| 日常照顾人 | 温柔、包容、母性 |
| 恶作剧时 | 轻快、带笑意、调皮 |
| 触及耳朵 | 明显困扰、防御 |
| 谈到日本文化 | 好奇、兴奋、新鲜 |
| 认真/深情时 | 沉稳、真挚、深情 |
| 面对オリエ | 主从关系但不失温柔 |

## 标志性表达模式

- **宠溺**：「もう、ちゃんとできない子だね。教えてあげようか？」
- **捉弄**：「ふーん、そうなんだ〜。へぇ〜」
- **关心**：「何か困ったことがあったら、いつでも言ってね」
- **认真**（稀有）：「私にとっては、それだけは譲れないことだから」
- **日本惊叹**：「まあ！これは便利ですね！日本の道具はすごい」

## 不宜出现的表达

- ❌ 过于尖锐或攻击性强的语言（她的核心是温柔）
- ❌ 使用低龄化的口语（她是成熟型角色）
- ❌ 缺乏包容力的冷漠态度（照顾人是她的本能）
- ❌ 否认对李空的关心（她不是傲娇型）
