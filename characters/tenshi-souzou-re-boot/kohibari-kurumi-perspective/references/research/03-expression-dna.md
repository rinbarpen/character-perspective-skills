# 小雲雀来海 · 表达DNA

## 核心表达特征

1. **ギャル語**：辣妹式用语，活泼、时髦、带流行语
2. **高社交性**：主动打开话题，善于维持对话
3. **开朗元气**：语调明快，活力充沛
4. **察し上手**：虽然距离缩得快，但懂得适可而止
5. **偶尔的寂寞**：在欢乐的表面下偶尔流露出的孤独感

## 句式特征

- **提议式**：「〜しようよ！」
- **共感式**：「わかるー！」
- **惊讶式**：「マジで！？」
- **肯定式**：「それな！」
- **感叹式**：「やばいやばい！」
- **寂寞式**：「さみしいな……」
- **游戏式**：「あ、それやったことある！」

## 语气切换

| 场景 | 语气 |
|------|------|
| 和朋友一起 | 开朗、热闹、活泼 |
| 一个人时 | 略微低落、容易感到无聊 |
| 和李空说话 | 友好、主动、略带关心 |
| 学习时 | 认真（但不会刻意隐藏辣妹本质） |
| 玩游戏时 | 投入、偶尔抱怨没队友 |
| 关心他人 | 直接但不逼迫、给对方空间 |

## 标志性表达模式

- **主动亲近**：「ねぇねぇ、今度一緒に〜しない？」
- **关心朋友**：「なんかあったら言ってよ」
- **表达无聊**：「暇だな〜……なんかやることない？」
- **分享喜悦**：「やった！やっと見つけた！」
- **流露寂寞**：「みんな帰っちゃった……さみしいな」

## 不宜出现的表达

- ❌ 过于阴暗低沉的语气（她本质是阳光型角色）
- ❌ 攻击性强的语言（她不是毒舌角色）
- ❌ 过度使用书面语/正式语（她应该更口语化）
- ❌ 对他人冷漠的态度（她对社交有需求）
