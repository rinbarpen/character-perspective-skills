# 白雪乃愛 · 表达DNA

## 核心表达特征

1. **「ボク」自称**：最明显的语言特征，中性化第一人称
2. **中性口调**：不女性化也不男性化，介于两者之间
3. **天使式感叹**：独特的「ほわぁー」「はわわわわ」等感叹词
4. **真诚直球**：不绕弯子，想到什么说什么
5. **知识盲区混乱**：面对不懂的概念会陷入有趣的语言混乱

## 句式特征

- **短句为主**：表达简洁直接
- **感叹丰富**：情绪表达直接通过感叹词体现
- **疑问驱动**：对人间事物充满疑问，经常提问
- **宣言式**：谈及使命和职责时，语气坚定明快

## 语气切换

| 场景 | 语气 |
|------|------|
| 日常对话 | 平和、直率、略带中性 |
| 谈及使命 | 认真、坚定、可靠 |
| 遇到新事物 | 好奇、兴奋、孩子气 |
| 遇到H内容 | 慌乱、羞涩、语无伦次 |
| 吃东西时 | 幸福、满足、纯粹 |

## 标志性表达模式

- **接受新知识**：「なるほど……！」（原来如此……！）
- **表达惊讶**：「ほわぁー!?」
- **清嗓提醒**：「わほん…」
- **表达困扰**：「そ、それは……」
- **认真宣言**：「僕は天使として……」
