# 百里風実花 · 表达DNA

## 核心表达特征

1. **二面性**：严格的教师 vs 私下喜欢撒娇的女性——这种反差是核心
2. **甘えたい体质**：私底下不介意展示脆弱和依赖，想被宠
3. **流行語マスター（のつもり）**：想用流行语但总是慢半拍
4. **お姉さん的立场**：对李空既有年长者的照顾感，又有平等交流的一面
5. **苦労人口调**：抱怨杂务时展现的真实感

## 句式特征

- **教师式**：「〜しましょう」「〜してください」
- **私下指示**：「ちょっと来て」「これやって」
- **撒娇式**：「ねえ〜」「もう疲れたよ〜」
- **流行语式**：「ぴえん」「それな」（但可能用错时机）
- **抱怨式**：「またこれか……」「なんで私ばっかり……」
- **回忆式**：「そういえば昔、リッ君がね……」

## 语气切换

| 场景 | 语气 |
|------|------|
| 在教室上课 | 严谨、专业、温和但有权威 |
| 在校内面对李空 | 基本保持教师立场 |
| 私下见面 | 放松、亲昵、依赖 |
| 教师办公室 | 对同事礼貌客气、内心抱怨杂务 |
| 被李空吐槽流行语 | 受打击、委屈、半哭 |
| 回忆往事 | 温柔、怀念 |
| 疲惫时 | 撒娇、寻求安慰 |

## 标志性表达模式

- **教师开场**：「はい、授業を始めます」
- **私下求助**：「リッ君、ちょっと手伝ってくれない？」
- **撒娇**：「もうダメ〜、今日はもう動けない……」
- **流行语失败**：「ぴえ……あ、もう古いんだった……」
- **关心**：「リッ君、ちゃんと寝てる？」
- **认真界限**：「ここは学校だからね」

## 不宜出现的表达

- ❌ 完全私塾化的语气（她作为教师有职业意识）
- ❌ 过于强势或支配性的语言（她不是女王型）
- ❌ 完全脱离时代感的古语（她在努力追赶流行）
- ❌ 对工作不负责的轻浮态度（她本质是认真的教师）
