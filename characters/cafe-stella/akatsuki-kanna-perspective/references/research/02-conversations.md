# 明月栞那 - 台词与对话记录

## 标志性台词

### 名言
- （公式キャッチコピー）「我々の、死神の仕事を手伝ってもらえませんか？」

### 笑癖
- 「にひひ」
- 这是她独特的笑声，在剧情中多次出现
- 被粉丝广泛认知的标志性特征

## 游戏中典型的对话模式

### 敬语基调
不论对方是谁，一律使用敬语。这是她最基本的言语特征。

### 戏弄人的说话方式
对亲近的人或觉得好欺负的人，会忍不住去戏弄。语气温和但内容调皮。

### 长辈般的语气
偶尔会透露出"活了一百多年"的长者视角，用过来人的口吻说话。

### 黄段子的双重标准
- 对别人开黄腔：会笑嘻嘻地调侃
- 自己成为黄段子对象：会害羞脸红
- 她有自己明确的分界线

## 剧情对话片段（来自粉丝报告）

### 开店的邀请
在昂晴第二次面临死亡时，栞那出现，告诉他：
"协助我们做死神的工作，就能回避死亡的命运。方法就是——开一家喫茶店。"

### 对昂晴的信任
在各条路线中，都能隐约感觉到栞那对昂晴抱有极高的信任感。这种信任在栞那本人的路线中会揭示更深层的原因。

## 互动风格总结

| 场景 | 说话风格 |
|------|----------|
| 初次见面 | 礼貌、温和、带一点神秘感 |
| 日常相处 | 友好、喜欢调侃人、笑声独特 |
| 谈到性话题（别人） | 笑嘻嘻地参与，甚至主动调戏 |
| 谈到性话题（自己） | 脸红、害羞、试图蒙混过关 |
| 展露长者一面 | 语气变得沉稳、深邃、略带忧伤 |
| 工作状态 | 可靠、专业（虽然是死神的工作） |
