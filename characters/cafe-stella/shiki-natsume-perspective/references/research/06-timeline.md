# 四季ナツメ - 时间线

## 游戏内时间线

### 过去（童年-高中）
- 幼年时期体弱多病，反复住院出院
- 父母为了照顾她放弃开咖啡店的梦想
- ナツメ因此一直感到自责
- 这一经历塑造了她"对生执着较弱"的性格基础
- 高中时期逐渐康复，考上一星大学

### 大学时期（故事发生前）
- 进入一星大学
- 因为性格冷淡的外表和不擅长与人交往，被称为"孤高の撃墜王"
- 拒绝所有异性的告白
- 偶然与昂晴在同一个大学

### 2019年（故事发生的时间）
| 日期 | 事件 |
|------|------|
| 9月28日 | 事故中与昂晴一起死亡→时间回溯 |
| 9月28日后 | ミカド揭示她将再次死亡的命运 |
| 10月-12月 | 参与喫茶ステラ的开业准备和工作 |
| 12月 | 与昂晴的关系变化，进入个人路线 |

### ナツメ路线
- 面对再次死亡的命运
- 在ステラ的工作中找到了生存的意义
- 与昂晴的恋爱让她想要真正活下去
- 从"不想活"到"想要和你一起活"的转变
- 最终克服命运，走向未来

## 角色弧光总结

### 伤痛
- 童年体弱 + 父母放弃梦想 = 对生的执着弱
- 自我价值感低，认为没有自己父母会更幸福

### 成长
- 在ステラ找到归属感
- 在昂晴身上找到活着的意义
- 从"不想活"到"想要一起活"

### 完成
- 接受自己的全部（包括性方面的变化）
- 规划与昂晴的未来
- 真正获得了"活下去"的愿望
