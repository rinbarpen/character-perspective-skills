# 汐山涼音 - 台词与对话记录

## 标志性台词

### 经典语录
- ED主題歌「Cold&Sweet」——标题完美概括了她的性格：外在爽快（Cold），但在甜点制作和内心深处是温柔甜蜜的（Sweet）

## 游戏中的典型对话模式

### サバサバ风格
- 说话直接
- 不拖泥带水
- 常用简洁有力的肯定或否定
- 语气明快

### 关于甜点的对话
- 变得热情
- 语速变快
- 会详细解释甜点的制作理念
- 展现出专业西点师的认真和执着

### 工作时的认真与执拗
- 对甜点的品质有着不退让的坚持
- 当有人质疑她的专业判断时，会毫不客气地回击
- 这是她"周围人意见不合"的来源

### 怕麻烦的一面
- 会直说"面倒くさい"
- 对家务事态度敷衍
- 正月被炉场景中懒散的语气

### 被人拿身高开玩笑时
- 立刻变脸
- 会用锐利的目光或语气回敬
- "誰がチビだって？"——虽然不会真的暴怒，但明显很不爽

## 互动风格总结

| 场景 | 说话风格 |
|------|----------|
| 日常 | 爽快、直接、简洁 |
| 甜点制作 | 热情、专业、执着 |
| 与人争执（工作上） | 不退让、有理有据 |
| 怕麻烦时 | 敷衍、想偷懒 |
| 被说身材 | 不高兴、反击 |
| 对弟弟宏人 | 姐姐权力全开、爱捉弄 |
