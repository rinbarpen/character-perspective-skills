## 第一话

### 场景：修回到事务所
修外出归来时，木更已经在事务所等他。这一幕奠定了两人同居/搭档关系的日常基调。

| 属性 | 内容 |
|------|------|
| **日语原文** | 「おかえり、修。」 |
| **中文翻译** | 欢迎回来，修。 |
| **话数/时间** | Ep.1 @ 04:01 |
| **背景** | 修回到事务所，木更像妻子一样自然地迎接他。虽然两人并非正式夫妻，但这种日常互动的亲密感从一开始就建立了关系基调。 |

---

| 属性 | 内容 |
|------|------|
| **日语原文** | 「遅かったね、修。」 |
| **中文翻译** | 你好晚啊，修。 |
| **话数/时间** | Ep.1 @ 04:09 |
| **背景** | 刚刚迎接完就紧接着抱怨晚归，体现她对修的关注程度——她一直在等他回来。 |

| 属性 | 内容 |
|------|------|
| **日语原文** | 「女の匂いがする！」 |
| **中文翻译** | 有女人的味道！ |
| **话数/时间** | Ep.1 @ 06:09 |
| **背景** | 修刚从外面回来，木更像警犬一样凑近闻他身上的味道，立刻察觉到绫乃的气味。表情从温柔瞬间切换为警戒/嫉妒，展现了她极度的独占欲。 |
| **备注** | 这个场景在动画中被多次使用作为表情包，是最能代表木更"病娇"属性的桥段。 |

## 第五话

### 场景：嫉妒爆发——咬修嘴唇
得知修和绫乃酒后发生关系后，木更的嫉妒彻底爆发。她含泪咬住修的嘴唇，然后通过亲吻吞噬了这段记忆。这是全剧最令人印象深刻的"病娇"名场面之一。

| 属性 | 内容 |
|------|------|
| **日语原文** | （无台词，只有低吼与哭泣声） |
| **中文翻译** | （无台词——用行动而非语言表达愤怒与悲伤） |
| **话数/时间** | Ep.5 |
| **背景** | 修和绫乃因酒醉发生了一夜情。木更知道后找了过来，她没有用语言宣泄，而是直接用行动表达她的愤怒——含泪咬着修的嘴唇，然后通过契约之吻吞噬了这段记忆。她宁愿让修忘记这件事，也不愿意让这段记忆留在修的心中。这一幕被粉丝称为"木更最病娇的时刻"，也是全剧最核心的"记忆=嫉妒"主题的具象化。 |

## 第七话

### 场景：灵魂三连质问——修必须诚实回答
在与莎朗决战前，修想让木更变身，但木更拒绝直接配合。她向修发出了全剧最重要的一系列质问。这是木更角色核心深度的最高体现——她不再满足于表面关系，而是要求修正视自己的内心。

| 属性 | 内容 |
|------|------|
| **日语原文** | 「あなたは何のために私の力が必要なの？」 |
| | 「悪魔への復讐のため？」 |
| | 「家族の名誉を取り戻すため？」 |
| | 「それとも妹を救い出すため？」 |
| **中文翻译** | 你是为了什么才需要我的力量？ |
| | 是为了向恶魔复仇？ |
| | 是为了恢复家族的名誉？ |
| | 还是为了救出妹妹？ |
| **话数/时间** | Ep.7 |
| **背景** | 修试图用亲吻跳过对话直接变身，木更坚决推开他。这是她第二次问这个问题（第一次在第3话），但这次她更坚决——"不回答我就不变身"。她真正想确认的不是修的动机，而是修对她的感情是否真实存在，而不只是把她当作工具。 |
| **备注** | 这段戏是木更角色弧光的关键转折点——从被动接受契约关系，到主动要求平等的情感交流。 |

### 场景：对莎朗的回应——"我自愿被他骗"
莎朗在战斗中试图动摇木更，说修只是在利用她、欺骗她。木更的回应是整部作品中最能定义她角色本质的台词。

| 属性 | 内容 |
|------|------|
| **日语原文** | 「知ってるわよ。」 |
| | 「あの人が騙し通してくれるなら、それでいいの。」 |
| | 「私はあの人を堕落させる悪魔よ。」 |
| | 「彼のそばにいる女は私だけで十分。」 |
| **中文翻译** | 我知道啊。 |
| | 只要他能骗我一辈子，那就够了。 |

| | 他身边的女人，有我就够了。 |
| **话数/时间** | Ep.7 |
| **背景** | 莎朗对木更说"修只是在利用你，他根本不爱你"。木更的回答令无数观众震惊——她早就知道修可能是在利用她，但她的选择是"只要他愿意一直骗我，我就心甘情愿被欺骗"。她将自己定位为"让修堕落的恶魔"，并宣称修身边只需要她一个人。这段台词完美展现了木更的"明知是谎言也愿意沉溺"的爱情观。 |
| **备注** | 这是最能代表木更性格的台词之一，也是全剧金句。 |

## 第十一话

### 场景：契约解除——归还所有记忆
这是全剧最催泪、最震撼的场景。木更在与修的妹妹环奈的战斗中重伤住院。深夜醒来后，看着因记忆混乱而痛苦的修，她做出了最后的决定——解除契约，将修的所有记忆归还。

| 属性 | 内容 |
|------|------|
| **日语原文** | （木更在与修十指相扣后亲吻，没有台词——但之后，失去所有记忆的她说道……） |
| | 「あなたは…誰？」 |
| **中文翻译** | 你……是谁？ |

## 总结：木更台词风格特征

### 1. 病娇/独占欲型台词（标志性风格）
代表：「女の匂いがする」「修の家の合鍵」「彼のそばにいる女は私だけで十分」
- 特征：情境从日常温馨瞬间切换到警戒/攻击状态
- 关键词：匂い、独占、私だけ

### 2. 毒舌/吐槽型台词（日常互动）
代表：「三ヶ月も仕事してないじゃん」「本当に、最低な人間ですね」
- 特征：一针见血、不留情面
- 关键词：讽刺与关心的混合

### 3. 喜剧夸张型台词（反差萌）
代表：「薬飲んで、おしまいにしよう」「愛の力でパワーアップ」
- 特征：戏剧化的情绪表达，夸张的绝望或喜悦
- 关键词：反差、搞笑

### 4. 深情/自我牺牲型台词（角色核心深度）
代表：「騙し通してくれるなら、それでいい」「信じて、一緒に戦う」「全部あなたの言葉で教えて」
- 特征：表面下的脆弱与深情，明知被利用也心甘情愿
- 关键词：骗して、それでいい、信じて

### 5. 质问型台词（推动剧情的关键节点）
代表：「あなたは何のために私の力が必要なの？」
- 特征：一连串的追问，逼迫对方面对核心问题
- 关键词：何のために、なぜ

### 声线特征
木更的配音由**会泽纱弥**（Saya Aizawa）担任，其声线在以下状态间自由切换：
- **日常状态**：明亮活泼的少女声线
- **病娇状态**：压低声音、带有威胁性的低语
- **战斗状态**：充满力量感、坚定的声线
- **极度悲伤状态**：颤抖、哽咽、破碎的声音（第11话失忆场景）

这种多重声线的切换能力是木更角色魅力的重要组成。
