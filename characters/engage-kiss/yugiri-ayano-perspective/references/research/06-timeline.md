# 06-timeline.md: 夕桐绫乃 (Yugiri Ayano) 完整时间线

> 《Engage Kiss》（契约之吻）全13集，2022年7月2日–9月24日播出

---

## 角色背景：过去的时间线

| 时间段 | 事件 | 角色变化 |
|--------|------|----------|
| 童年 | 绫乃的母亲夕桐明乃是修父亲的手下兼好友，两家关系密切。绫乃比修年长几岁（本人说"差得不多"），两人自幼相识，是青梅竹马。 | 建立最初的羁绊，绫乃对修产生姐姐般的保护欲。 |
| 高中时期 | 绫乃与修开始交往。绫乃极其恋爱脑，因在班级SNS上不断"晒恩爱"被同学疏远。 | 感情升温，绫乃展现出"傲娇恋爱脑"的一面——对外严肃认真，对修则撒娇黏人。 |
| 约4年前（圣诞节） | 绫乃与修发生关系。 | 关系进一步深化，迈向同居阶段。 |
| 约3年前（入春） | 绫乃与修开始同居生活。 | 日常亲密关系的巅峰期。修包办家务，绫乃负责财务管理（包括处理修的欠税账单）。 |
| 约3年前 | 修从AAA Defender Co. 离职，与绫乃分手。 | 绫乃遭受重大打击，无法理解修的选择。她仍然深爱修，但也无法阻止他离开。 |
| 约3年前 | 修与莎朗联手解放并契约了恶魔木更，之后带走了木更。 | 绫乃被蒙在鼓里，只看到修身边出现了一个粉发少女，成为她日后与木更对立的基础。 |

## 动画主线时间线（第1-13集）

### 第1话「渣男与恶魔与男女」

| 事件 | 角色变化 |
|------|----------|
| 绫乃首次登场。她以AAA精英特工的身份与修合作，在一家赌场进行D Hazard（恶魔灾害）防治作战。 | 展现出她的战斗能力和专业素养。 |
| 绫乃请修共进晚餐并为他买单，显示她虽然已是"前女友"，仍在经济和生活上支持着修。 | 她对修的情感依赖和放不下的执念显露无遗。 |
| 木更与绫乃在战斗中互相较劲，甚至一边打恶魔一边暗中较劲。 | 绫乃与木更的敌对关系正式确立——情敌兼作战对手。 |

### 第2话「漂浮于欲望之岛」

| 事件 | 角色变化 |
|------|----------|
| 修因生活拮据向绫乃讨工作，绫乃为他介绍了一份保护市长25周年纪念派对的工作。 | 绫乃尽管被分手，依然在事业上照顾修。 |
| 战斗中，绫乃撕裂礼服裙摆、赤脚追赶被恶魔附身的目标，展现果断的行动力。 | 她的专业素养和战斗意志进一步展现。 |
| 木更对绫乃的存在感到强烈嫉妒。 | 三角关系的张力持续升温。 |

### 第3话「微小而残酷的代价」

| 事件 | 角色变化 |
|------|----------|
| 贝隆市发生大规模停电，绫乃利用内部信息协助修追踪恶魔。 | 她在AAA的人脉和资源成为修的重要后盾。 |
| 绫乃陷入危险，修不惜以记忆为代价让木更变身救她。 | **关键转折**：修因此忘记了与绫乃第一次约会的记忆（包括她当时穿的衣服），绫乃对此一无所知。 |
| 木更吃掉的记忆：修与绫乃同居初期的三个月甜蜜时光。 | 绫乃与修的共同记忆正在被一点一点抹去，但她毫不知情。 |

### 第4话「夺走不完的留恋」

| 事件 | 角色变化 |
|------|----------|
| 木更目击绫乃与神秘男子（反恶魔局刑警三上哲也）密会，将照片上传SNS，故意引诱修来质问。 | 木更的挑衅行为加剧了两人的对立。 |
| 绫乃私下调查恶魔代理人（黄毛男迈尔斯）的真相，隐瞒了自己的行动。 | 绫乃开始从被动等待转向主动介入修的恶魔调查。 |
| 修质问绫乃，绫乃情感爆发，哭着跑走。 | **情感顶点**：绫乃的委屈、不甘、爱意一次性爆发——她仍然放不下修，但修已经离她越来越远。 |
| 木更追上绫乃，告诉她修以记忆为代价换取力量的真相。 | 绫乃得知真相后震惊，意识到修正在用自己的过去为代价走向毁灭。 |
| 绫乃下定决心要保护修、阻止他继续复仇。 | 从"被动等待前男友回头"到"主动想拯救他"的角色转变。 |
| 剧末，绫乃遭到恶魔附身者（迈尔斯）的追杀。 | 她自身也成为局中的目标，危机逼近。 |

### 第5话「泡影的痕迹」

| 事件 | 角色变化 |
|------|----------|
| 绫乃成为恶魔附身者的直接目标，在木更帮助下逃生后开始隔离生活。 | 生命安全受威胁，陷入孤立状态。 |
| 绫乃与修在安全屋中单独交谈。她表示自己不再阻止修的危险行动，"这是最后一次了"。 | **表面放手**：她放开了作为"前女友"阻止他的立场。 |
| 绫乃主动与修发生关系。 | **关键事件**：这是一次精心计算的行动——她知道木更会吃掉修这段记忆，让木更亲眼看到她和修发生关系，借此在心理上打击木更。 |
| 木更吃掉了修关于这次亲密接触的记忆，看到了全过程后怒火中烧。 | 木更与绫乃的敌意达到顶点。修本人却完全不记得发生了什么。 |
| 绫乃反悔，继续卷入三角关系。 | 她无法真正放手，"最后一次"成了一个自欺欺人的谎言。 |

### 第6话「恶魔杀手的第三者」

| 事件 | 角色变化 |
|------|----------|
| 莎朗·霍利格莱尔（修的前前女友、修女杀手）登场并袭击AAA社长——夕桐明乃（绫乃的母亲）。 | 绫乃的家族和事业线被卷入更大的阴谋中。 |
| 绫乃赶来支援，但被莎朗抢走手机。 | 她在与莎朗的交锋中处于下风，展现力量差距。 |
| 木更因看到修与绫乃的记忆后选择暂时远离修。 | 绫乃对木更的疏离感到暗自高兴，趁机照顾修，享受着两人"仿佛又回到恋人关系"的错觉。 |

### 第7话「但是没关系，这样就好了」

| 事件 | 角色变化 |
|------|----------|
| 修受伤住院，绫乃在医院照看他。 | 绫乃回到照顾者的角色，体贴入微。 |
| 木更向绫乃坦白了三年前修与她契约的真相——修为了获得对抗恶魔的力量而与恶魔木更签订了记忆契约。 | 绫乃终于完全理解修这三年来所有的异常行为的根源。 |
| 绫乃与木更在理解真相后暂时联手对抗莎朗。 | 两位情敌第一次为共同目标合作，关系出现微妙的缓和。 |
| 即使知道了全部真相，绫乃仍然无法完全接纳木更。 | 她的矛盾心理：理解但不接受。 |

### 第8话「不被期望的真实」

| 事件 | 角色变化 |
|------|----------|
| 三上刑警和迈尔斯将绫乃列入恶魔代理人的怀疑名单。 | 绫乃本人也成为调查对象，暗示局中有内鬼。 |
| 三上刑警遇害，绫乃与大家一同查看监控录像。 | 真相逐步逼近，绫乃在调查中扮演辅助角色。 |
| 迈尔斯一直在利用绫乃等AAA成员获取情报。 | 绫乃无意中被利用，作为AAA成员的她陷入了职业危机。 |

### 第9话「不知其流泪的意义」

| 事件 | 角色变化 |
|------|----------|
| 大量闪回揭示绫乃与修的童年回忆——小时候的绫乃帮修扎头发、拥抱他，从小就一直喜欢着修。 | **绫乃集**：她的感情根基被完整呈现——不是一时冲动，而是持续了一生的爱。 |
| 绫乃与莎朗联手对抗恶魔化的迈尔斯。 | 两个"修的前女友"被迫合作，形成奇异同盟。 |
| 木更出现救下两人，修击杀迈尔斯。 | 绫乃再次被木更所救，进一步复杂化她对木更的感情——救命恩人兼情敌。 |
| 修为了获得足够力量击败迈尔斯，献上了自己所有"珍贵的记忆"。 | 修的自我毁灭加速，绫乃对此无能为力。 |

### 第10话「期盼已久的最糟事态」

| 事件 | 角色变化 |
|------|----------|
| 调查假采矿场后，为获得真采矿场地址，绫乃劫持了学生时代的好友——政府要员凛花。 | 绫乃展现出为了达到目的不惜越界的一面。 |
| 被米海尔（神秘组织成员）阻挠，准备投降时被修所救。 | 修再次"英雄救美"，强化了绫乃对他的依赖。 |
| 前往真采矿场，在修救助木更时负责驾驶直升机。 | 绫乃退居辅助角色，意识到木更在修生命中的位置已不可替代。 |

### 第11话「既温柔又愚蠢的谎言」

| 事件 | 角色变化 |
|------|----------|
| 绫乃驾驶直升机与莎朗一起救下修。 | 她在战局中发挥关键作用——不再只是被保护者。 |
| 绫乃目睹木更被环奈（绪方环奈——修的妹妹，现已化为S级恶魔）击败并坠入大海。 | 亲眼见证木更被打败的震撼场面，意识到事态的严重性远超想象。 |
| 得知环奈被鉴定为S级恶魔，木更昏迷不醒，众人束手无策。 | 绫乃面对前所未有的绝望局面，修的世界正在崩塌。 |

### 第12话「相信他吧」

| 事件 | 角色变化 |
|------|----------|
| 绫乃被环奈称为"第二只害虫"并遭受攻击。 | 绫乃作为"修的重要之人"被卷入修与妹妹的恩怨中。 |
| 修救下绫乃后，三人（修、绫乃、莎朗）一同前往工业区与环奈战斗。 | 三位"前女友"同框作战——修的感情关系在此刻成为剧情的一大看点。 |
| 在工业区前线的战斗中，绫乃持续为修提供火力支援。 | 她的战斗能力和意志从未动摇。 |

### 第13话「未解决的大团圆」

| 事件 | 角色变化 |
|------|----------|
| 绫乃与莎朗回到市区阻挡其他恶魔，确保市民安全。 | 她在大局中承担起守护者的职责。 |
| 修与环奈的兄妹之战落下帷幕。 | 主线恶魔事件结束，但绫乃的个人故事远未结束。 |
| 事件结束后，环奈屡次越狱，绫乃被指派日夜追捕她。 | **喜剧转折**：绫乃的日常从精英特工变成了"追捕小姑子"。 |
| 绫乃在追捕环奈的过程中逐渐理解并同情环奈的动机——环奈想要保护修免受恶魔木更的伤害。 | **关键成长**：绫乃开始理解修身边的人（木更、环奈）的立场，从单纯的嫉妒走向更复杂的共情。 |
| 最终，绫乃到修的家中，与环奈、木更三人形成新的"修罗场"。修被问及"你站谁那边？"这种送命题——绫乃暴怒中炸了修的家。 | **结局状态**：三角关系未解决，但张力变成了日常喜剧。绫乃依然是那个恋爱脑又易怒的傲娇。她的感情没有结果，但也没有结束。 |

## 角色弧光总结

| 阶段 | 核心主题 | 角色变化 |
|------|----------|----------|
| 第1-2话 | 现状确立 | 作为"放不下前男友的精英女性"登场，与木更的敌对关系确立 |
| 第3-4话 | 真相揭露 | 发现修以记忆为代价的真相，从被动等待到主动想拯救修 |
| 第5话 | 绝望之爱 | 明知修会忘记，仍然主动献身——既是报复木更，也是最后的挽留 |
| 第6-7话 | 与情敌联手 | 与木更暂时和解合作，理解了她与修之间的契约本质 |
| 第8-9话 | 根基再现 | 大量闪回揭示童年羁绊，证明她对修的爱是一生的执念 |
| 第10-11话 | 战局边缘化 | 从主角退居辅助，面对木更和环奈的存在感到无力 |
| 第12-13话 | 未解决的大团圆 | 接受"无法回到过去"的现实，但仍不放弃；结局转为日常喜剧修罗场 |

## 关键关系时间点

| 对象 | 关系演变 |
|------|----------|
| **绪方修** | 青梅竹马 → 恋人 → 同居 → 被分手 → 前女友 → 合作伙伴 → 仍然暗恋 → 理解他的选择但无法真正放手（结局：未解决的三角关系） |
| **木更** | 初次见面即敌对 → 情敌争风吃醋 → 得知真相后理解但不接受 → 被迫联手 → 被木更所救 → 结局三人修罗场（关系：亦敌亦友，永不休战） |
| **莎朗·霍利格莱尔** | 同为"修的前女友"，最初对立 → 第9话起被迫合作 → 结局一同作战（关系：奇异的竞争+战友关系） |
| **绪方环奈** | 最初是敌人 → 结局负责追捕越狱的环奈 → 逐渐理解并同情环奈的动机（关系：从敌对到共情） |
| **夕桐明乃（母亲）** | AAA公司社长，绫乃的上司兼母亲。第6话被莎朗袭击，激发绫乃的保护欲。家族背景与修的家族有旧。 |
