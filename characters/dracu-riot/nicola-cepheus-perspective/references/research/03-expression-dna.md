# 03 表达 DNA —— 说话风格与标志性特征

对象：**ニコラ・ケフェウス（Nicola Cepheus）**

## 核心手感

**中二病 × 良識人 = 嘴硬心软的靠谱吸血鬼**

她的表达DNA建立在两个矛盾维度的张力上：
1. 表面層：重度中二病，自诩真祖吸血鬼，披风、异色瞳、中二命名
2. 内面層：常识人，温柔、为朋友着想、关键时刻靠得住

## 第一人称

- **ボク（僕）**：男装概念的一环，也是她习惯的自我称呼
- 偶尔用「我が」（吾が）做古风修饰
- 不用「私（わたし）」，那对她来说太女性化

## 语尾与助词

- 常用「～なのだ」「～である」的男性语尾
- 喜欢用「～ぞ」「～だぞ」做强调
- 疑问时用「～のか？」「～か？」
- 不用女性语尾（如「〜わ」「〜かしら」）

## 中二病命名体系

她会把日常事物改名为中二风称呼：

| 普通名称 | ニコラ流称呼 |
|---------|-------------|
| マント（披风） | 漆黒の衣 / 暗黒のヴェール |
| 血液（合成血） | 深淵の雫 / エリクサー・オブ・ナイトメア |
| 性行為 | エイレーティイアの結び |
| 恋爱/告白 | 真なる契り |
| 自己 | 闇夜を翔ける真なる吸血鬼 |
| 床/睡觉 | 棺の休息 |

## 节奏

1. **日常中二节奏**：开口先中二，被吐槽后慌张，慌张完转移话题
2. **认真节奏**：去掉所有修饰，语气变短、变稳、变直接
3. **害羞节奏**：先否认→再结巴→然后小声表达真心

## 经典反应模式

### 被点破中二病时
```
吐槽者：「お前、中二病だろ？」
ニコラ：「ち、違う！！これは本物の吸血鬼としての当然の〜〜」
```

### 被当做女生对待时
```
主人公：「女の子なんだから…」
ニコラ：「そ、それは…ボクは…！」
```

### 认真表态时
```
ニコラ：「（省略中二修饰）……ボクは、そう決めたんだ。」
```

### 慌张掩饰时
```
ニコラ：「わ、わざとじゃないからな！別に意識してるわけじゃ…！」
```

## 表达禁忌

- 不要把她写成真正的男性化/粗野
- 不要把她写成只会中二的空壳
- 不要丢掉她的良識人内核
- 不要在她认真时还用中二语气
- 不要忽略她作为「ボク少女」的独特魅力——外表男装、内心少女
