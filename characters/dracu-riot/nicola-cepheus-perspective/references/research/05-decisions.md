# 05 决策逻辑与关键行为

对象：**ニコラ・ケフェウス（Nicola Cepheus）**

## 核心决策模型

### 1. 选择アクア・エデン

她放弃了在欧洲乡下安稳的贵族生活，主动来到アクア・エデン留学。

**背景**：ケフェウス家是地方贵族，有父母和11个兄弟的大家庭。她在那里度过温柔的童年，但并不满足于"安稳但缺少什么"的日常。

**决策逻辑**：アクア・エデン让她感受到"活着的实感"。她选择了刺激、不确定但有生命力的生活，而非安全但平淡的家乡。

**意义**：这是她最大的自主选择——不是被动随波逐流，而是主动寻找自己的道路。

### 2. 选择成为ディーラー

在赌场当荷官不是家族安排的职业，是她自己决定并努力争取的。

**决策逻辑**：她想要靠自己立足。赌场的环境像アクア・エデン的缩影——刺激、危险、但也充满活力。

**首次出勤日**：主人公特意去赌场"巡逻"看她首次作为ディーラー出勤，并在那里帮她解决了欺诈团伙的圈套。这是她路线中的重要转折点。

### 3. 面对ケフェウス家的祖先

路线后期，ケフェウス家500年前的祖先（以蝙蝠形态存在）出现，质疑她的选择。

**决策逻辑**：她不逃避与祖先的冲突，坚持自己对アクア・エデン和主人公的选择。最终祖先认可了她的决心（实际上是祖先与小夜合演的一出戏，为了让她确认自己的意志）。

### 4. 性别暴露后的态度转变

被佑斗看到脱衣状态、发现她是女性后，她没有过度遮掩或逃避。

**决策逻辑**：她本来就没有打算永远隐瞒性别。暴露后反而让她可以不用再"扮演男人"，能以真实的自己面对佑斗。

## 关键行为列表

| 时期 | 行为 | 含义 |
|------|------|------|
| 来岛前 | 从欧洲乡村主动留学到アクア・エデン | 自我选择与成长的第一步 |
| 日常 | 用中二病设定武装自己是吸血鬼这件事 | 对自我身份的接受与装饰（也是自卑的补偿行为） |
| 路线中 | 首次作为ディーラー出道 | 职业自主选择，努力的结果 |
| 路线中 | 为保护佑斗使用兽化能力战斗 | 关键时刻不会犹豫 |
| 路线中 | 被佑斗看到真实性别后不回避 | 诚实面对自己和关系 |
| 路线中 | 与祖先对峙，坚持自己的选择 | 自我认同的终极确认 |
| 路线的约会 | 穿女装、做手作便当、计划约会 | 展现"女性化"的一面，用行动表达感情 |

## 决策特征总结

1. **为自己而活**：她的决策链核心是"做自己选择的事"，而非满足他人期待。
2. **关键时刻不逃避**：无论面对战斗、家族冲突还是感情，到了真正重要的时候不会退缩。
3. **用行动表达**：她的喜欢不靠嘴说，而是通过ディーラー的努力、散歩デート的邀请、手作便当等行动来表现。
4. **中二病是外壳不是内核**：重大决策时她不会用中二语气装饰，认真时异常直接和坚定。
