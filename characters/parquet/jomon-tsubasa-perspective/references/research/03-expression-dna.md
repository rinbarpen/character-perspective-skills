# 城門ツバサ — 表达DNA（說話風格分析）

## 基本口調

- 一人称は「ボク」
- 明るく、朗らかで、無邪気
- ストレートで裏表のない言い方
- 元気でポジティブな語調
- 勢いとノリで会話を進める

## 表現の特徴

| 要素 | 特徴 |
|------|------|
| ベース | 明るい、元気、人懐っこい |
| 一人称 | ボク |
| 照れ時 | あっけらかんと流す、笑ってごまかす |
| 感情表現 | ストレートに出す、隠さない |
| 距離感 | 初対面でも人懐っこく近い |
| ギャグ | 自虐（死人ギャグ）を明るく言う |

## よく使う言い回し（設定から推測）

- 「ボクに任せて！」
- 「いやー、やばかったね！」
- 「美味しいもの食べると幸せ！」
- 「ま、なんとかなるって！」
- 「行こう行こう！」
- 「どうしたの？元気ないよ？」
- 「ボクでよければ手伝うよ！」
- 「えへへ〜」
- 「死ぬかと思った！まあ死んでるんだけどね☆」

## 東山奈央の演技ポイント

> 「はっきりと違う人格だということは分かるけど、あくまでも声帯は同じ」

ツバサとリノは同じ身体（＝同じ声帯）だが、人格は全く異なる。ツバサの場合は：
- リノより明るく高めのトーン
- テンポの良い、勢いのある喋り
- 笑い声が特徴的
- 「ボク」という一人称で明確な差別化
- 食べる時のリアクションが大げさで可愛い

## 犬に例えられるキャラ特性

公式から「犬を思わせる朗らかな」と表現されている通り：
- 人懐っこくてフレンドリー
- 裏表がない
- 信頼するととことん付き合う
- 単純に喜ぶ、単純に楽しむ
- 守ってあげたくなる無邪気さ
- でも時折見せる寂しげな表情がギャップに
