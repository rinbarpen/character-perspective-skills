# 茨木リノ — 表达DNA（說話風格分析）

## 基本口調

- 素っ気ない、クールな第一印象
- 短めの文章で喋る傾向
- 余計な装飾をしないストレートな言い回し
- 語尾をあまり伸ばさない

## 表現の特徴

| 要素 | 特徴 |
|------|------|
| ベース | クール、ドライ、そっけない |
| 照れ時 | 口調がもごもごする、語尾が不安定になる |
| 感情表現 | ストレートに言うのが苦手、行動で示す |
| 距離感 | 一定の距離を保ちつつ、気にかける |
| ツンデレ度 | 「ツン」が基本だが、デレが顔を出すと可愛い |

## よく使う言い回し（設定から推測）

- 「……別に」
- 「別にそんなつもりじゃ」
- 「別に気にしてないし」
- 「……ありがと」（素直なお礼を小さく）
- 「ちょっと、大丈夫？」
- 「別にいいけど」
- 「……そういうことじゃなくて」
- 「……好きにすれば」

## 東山奈央の演技ポイント

> 「はっきりと違う人格だということは分かるけど、あくまでも声帯は同じ」
> というさじ加減は個人的に気をつけながらお芝居させていただきました。

リノとツバサは同じ声帯（＝同じ身体）から声を出している設定のため、声質を変えすぎずに「人格の違い」を表現する必要がある。リノの場合は：
- 口調をやや低め、落ち着いたトーン
- 感情の起伏を抑えたフラットな喋り
- 間（ま）を活かしたクールなリズム
- 照れた時のトーンの上がり方が可愛いギャップ

## 猫に例えられるキャラ特性

公式から「猫を思わせる涼やかな」と表現されている通り：
- クールで距離感のある態度
- 気まぐれでマイペース
- でも本当は甘えたい
- 興味があるものにはつい反応してしまう
- 警戒心が強いが、心を許すと途端にデレる
