# 杜野凛世 · 时间线与角色弧光

## 基本时间线

### 出生〜15歳（鳥取時代）
- 鸟取県的呉服店（老舗）に生まれる
- 5人兄弟姐妹中的第4子（姉1人、兄2人、妹1人）
- 严格的家教：漫画禁止、芸道優先、外出制限
- 8岁时，年长的姐姐远嫁——凛世失去最重要的玩伴
- 在芸道全般（華道・茶道・書道・日本舞踊・武道等）修行中度过童年
- 通过折り紙和百人一首寻找心灵的慰藉
- 几乎没有朋友、不谙世事

### 15-16歳（283プロ入所以前）
- 为某事（具体未明）来东京
- 在283プロ事務所附近行走时木屐带断裂
- 与P相遇→被帮助→被スカウト
- 「プロデューサーさま」的称呼从此开始

### 2018年4月（游戏开始）
- 正式加入283プロダクション
- 加入放課後クライマックスガールズ
- 与樹里、恋鐘、咲耶、千雪、羽那共同开始寮生活
- 初期状态：几乎不说话、对一切都感到新鲜和困惑
- 藁人形を常に携帯（这时候就开始了吗？）

### W.I.N.G.編（2018年〜）
- 作为放クラの一員，参加W.I.N.G.
- 与父母的对立→以W.I.N.G.优胜为条件达成妥协
- 这一时期的凛世："没有自己的梦想"——一切为了P
- 从P那里获得前进的动力

### ファン感謝祭編
- 开始隐约意识到"自己想做的事"
- 通过阅读少女漫画学习"世间の事"
- 在寮生活中逐渐适应都市生活
- 与智代子建立起"漫画的借主-借出者"关系

### G.R.A.D.編
- 凛世的DA（Dance）属性被强化
- 因100時間回転配信梗被设定为DA属性（官方认证的梗）
- コミュ中表现出更强的自我意识和判断力
- 与P的互动更加自然，不再仅仅是"追随"

## 角色弧光的5阶段

### 第1段階：覚醒前——"箱入り娘"（初期）
**特征**：
- 没有自己的意见、欲望、梦想
- 完全以P为中心行动
- 对一切都感到新鲜，但缺乏主体性
- 言行像机器人/町内放送
**代表卡片**：R【白いツバサ】

### 第2段階：依存——"Pの夢が私の夢"（W.I.N.G.編）
**特征**：
- 开始产生"想为P做点什么"的愿望
- 但"想做"的理由仍是"为了让P高兴"
- 开始接触外部世界（少女漫画、寮生活）
- 情感开始萌芽，但仍不知如何表达
**代表卡片**：SR【想ひいろは】

### 第3段階：葛藤——"私のしたいこと"（ファン感謝祭〜G.R.A.D.）
**特征**：
- 在P的引导下，开始寻找"对自己来说重要的事"
- 发现"自己想做的事"的过程伴随着不安和迷茫
- 逐渐意识到"不能永远跟在P身后"
- 开始对自己的"重い愛"有所自觉
**代表卡片**：SSR【水色感情】

### 第4段階：自立——"隣に立つ"（Landing Point〜）
**特征**：
- 能够独立做出判断和行动
- 即使是P无法触及的地方也能自己走下去
- 从"被守护"到"守护"的转变完成
- 可以主动表达自己的意见（「プロデューサーさま……それは違います」）
- 与P从"主従"关系进化到"対等"关系
**代表卡片**：SSR【ロー・ポジション】

### 第5段階：統合——"大和撫子の完成形"（最新）
**特征**：
- 保留大和撫子的传统美德，但获得了自我主体性
- 可以同时做到"一歩引く"与"自己主張"
- 对P的爱从"依存"进化为"相互尊重"
- 拥有自己的梦想，并愿意与P分享
**代表卡片**：UR【絵空靴 星月夜を歩いて】、SSR【歩幅をそろえて】

## 关系性的时间变化

### 对P的关系变化
```
スカウト時：恩人——"この人が凛世を必要としている"
W.I.N.G.編：絶対的依存——"プロデューサーさまのため"
ファン感謝祭：信頼と感謝——"プロデューサーさまがいてくれる"
G.R.A.D.：相棒意識の萌芽——"凛世も頑張る"
Landing Point：対等の可能性——"一緒に歩く"
トワコレ【花は】：含蓄の成功——"伝わることもある"
```

### 对放クラメンバー的关系变化
```
初期：遠慮がちな新入り——"皆さまはすごい"
中期：距離の縮まり——"樹里さまと寮で一緒"
現在：かけがえのない仲間——"放クラの凛世"
```

## 未来展望

根据最新剧情（2026年「sand song」「歩幅をそろえて」等），凛世仍在持续成长：

1. **アイドルとして**：进一步确立自己的风格，在放クラ中发挥独特的存在感
2. **個人として**：进一步强化自我主体性，从"Pの凛世"到"凛世という一個人"
3. **Pとの関係性**：从"主従"到"対等"的转变将继续深化
4. **家族との関係**：已获得认可、但仍需持续证明
