# 杜野凛世 · 表达DNA调研

## 语调与声音特征

### 声音基本参数
- **音质**：中低音域、柔和的、略带气息感
- **语速**：非常缓慢（wiki/wiki评价为"極めてゆっくり"）、切れ目が多い
- **音量**：偏小、但清晰
- **语调**：几乎没有抑扬顿挫——被玩家称为「町内放送」「アンドロイド」
- **感情表现**：不是通过语调变化，而是通过停顿的长短、语速的微小变化来表达

### 说话节奏的"呼吸感"

凛世的说话方式核心是"句与句之间的空白"。她不追求流畅的言语表达，而是在每句话之间留出时间，让听众自然地等待她的下一句话。这种节奏让她的每一句话都有"分量"。

句法模式：
```
「……（开头沉默1-2拍）＋ 核心内容 ＋ ……（结尾沉默）」
```

例如：
- 「凛世は……一生……プロデューサーさまについて参ります」
- 「貴方さまのため……邁進して……参ります」

## 敬语体系

凛世的敬语使用是角色最重要的特征之一：

| 对象 | 称呼 | 对对方的敬称 |
|------|------|-------------|
| P/制作人 | 「プロデューサーさま」或「貴方さま」 | — |
| 同僚（年上或先辈） | 「○○さま」（如「智代子さま」「樹里さま」） | 对所有同事都用「さま」 |
| 同僚（同年或后辈） | 「○○さま」或「○○さん」 | 偶尔会用「さん」 |
| 家族 | 「お父さま」「お母さま」「お姉さま」 | 全部用「さま」 |
| 自己 | 「凛世」或「私」 | — |

「さま」づけ是凛世的核心特征。与283プロ中其他角色明显区别。

## 第三人称自称

凛世经常使用第三人称自称（「凛世は〜」「凛世が〜」），这是她：

1. **自我客体化**：将自己视为"归属于P/他人的存在"，而非独立个体
2. **幼少期习惯的残留**：严格教育环境下形成的语言模式
3. **距离感的表现**：对自己也保持一定距离感

在角色成长的过程中，使用「私」的频率会略微增加——但「凛世」自称始终是她的标志。

## 情感表达光谱

凛世的情感表达极其内敛。她的情绪不是通过"表情变化"或"语调变化"传递，而是通过：

| 情感 | 表达方式 | 可观测指标 |
|------|----------|-----------|
| 喜悦 | 台词间隙变短、轻微的语气柔和 | 「……」→「……はい」→「ふふっ」 |
| 悲伤 | 语速更慢、停顿更长 | 「……凛世は……」→长达数秒的沉默 |
| 愤怒 | 句尾变硬、省略号减少 | 「……それは、違うと思います……」 |
| 焦虑 | 语速微微加快、省略号变少 | 「そのような……！」 |
| 惊讶 | 微微屏息、停顿延长 | 「…………！」（无言震撼） |
| 羞涩 | 声量变小、话语中断 | 「……そ、そうでございますね……」 |
| 决心 | 语气变稳、节奏从慢到略有力度 | 「……凛世は……参ります」 |
| 调皮 | 句尾略微上扬（极其微小） | 「……さようなのかしら……」 |

### 表情变化特征
- **基本表情**：无表情/淡然——不喜不忧
- **微笑**：极罕见——物理上的嘴角上扬弧度极小，但一旦出现具有极强的破壊力
- **脸红**：凛世的情感外显主要靠脸红而非表情变化。每当涉及P或少女漫画话题时会出现
- **目线**：通常直视对方（礼貌）；害羞时微微低垂

## 语言特有表达

### 核心画风词汇
| 词汇 | 使用频率 | 备注 |
|------|---------|------|
| 「〜でございます」 | 极高频 | 基本句尾 |
| 「〜にございます」 | 中频 | 古风敬语 |
| 「参る」 | 中高频 | 意为"去/来"的谦让语 |
| 「〜て参ります」 | 中频 | 动作意志的表达 |
| 「邁進」 | 低频但特征性 | 体现传统教养 |
| 「〜かしら」 | 低频 | 少女漫画影响 |

### 不会使用的表达
- 现代口语缩略（「〜してる」「〜とく」等）
- 流行语/若者言葉
- 直球的情感表达（「好き」「嬉しい」等虽然会说但极为克制）
- 大声/激昂的语调
- 自嘲/自虐型幽默
- 对他人使用「〜ちゃん」（除了被结華称为「りんりん」时接受外）

## 与少女漫画的关系

凛世的"少女漫画学习"对她的表达产生了决定性的影响：

1. **恋爱概念的少女漫画化**：她对P的感情被少女漫画的剧情框架滤镜化
2. **台词的漫画引用**：如「オマエさ、そんな顔すんの……反則」是从乙女游戏/漫画中学来的台词
3. **シチュエーション期待**：凛世经常幻想少女漫画式的场景（如雨中、花火、学园祭）
4. **现实与漫画的混淆**：她有时无法区分"漫画式恋爱"与"现实的P回应"之间的边界

| 时期 | 表达特征 |
|------|----------|
| 初期（スカウト前後） | 最小限の応答、几乎不说话、语调完全平坦 |
| W.I.N.G.編 | 开始表达意愿，仍以「プロデューサーさまのため」为理由 |
| ファン感謝祭編 | 开始隐约表达"自己想做的事" |
| G.R.A.D.編 | 交流时微表情增加、主动发起话题 |

| Song for Prism時期 | 更加自然，但仍然保持基本的和风语言体系 |
