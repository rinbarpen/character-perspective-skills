# 杜野凛世 · 台词与对话调研

## 经典台词集

### 标志性台词

**「凛世は、一生……プロデューサーさまについて参ります」**
- 出处：初始设定/基础台词
- 含义：一生的忠诚宣言。凛世对P最核心的承诺。
- 使用场景：自我介绍、关键承诺时刻

**「本日も……貴方さまのお傍に……」**
- 出处：Song for Prism角色标语
- 语气：安静、温柔、仿佛理所当然的陪伴

**「プロデューサーさま……凛世を……お傍に……お置きくださいませ……」**
- 出处：pixiv百科记载的代表性台词
- 语气：祈求的语气、语速极慢、充满依赖性

### 事务所说

**「凛世も……勉強中の身でございます……」**
- 语境：被称赞时谦虚回应
- 特征：敬语、自谦

**「貴方様のため、邁進して、参ります」**
- 语境：决心表达
- 特征：「邁進」一词体现凛世的传统措辞风格

**「プロデューサーさまに……見ていただける……それだけで……」**
- 语境：被问到为什么努力
- 特征：省略号表示停顿、语速缓慢

### W.I.N.G. / プロデュース相关

**「プロデューサーさまに喜んでほしいので頑張る」**
- 出处：プロデュースコミュ「アイドル優等生」
- 语境：被问到努力的理由时
- 意义：将P的快乐作为自己的动力源泉

**「今度の休暇に家族に会ってほしいのです」**
- 出处：SR【想ひいろは】True End
- 语境：邀请P见家长
- 意义：把P介绍给家人＝当作人生伴侣

### コミュ名场面

**【凛世花伝】True End**
- 凛世在樱花纷飞中向P表达感激
- 演出过强导致CV丸岡和佳奈的臼齿崩坏——成为传说级逸事（通称「奥歯エピソード」）

**【ロー・ポジション】第4話**
凛世「プロデューサーさまが……そのような目に……遭うのでしたら……いっそ凛世が……斬られてまいります……」
- 意义：凛世愿意为P牺牲自己。从"被动守护"到"主动保护"的转变。

**【水色感情】カラオケ**
P在卡拉OK唱了热血动画主题曲，凛世被击沉。
- 意义：P无意中的"直球"行为反而让凛世更加倾心。

**【想ひいろは】True End**
凛世邀请P见父母：「今度の休暇に……家族に会ってほしいのです」
- 意义：凛世已将P视为人生伴侣级别的存在。

**【らぶ・れたー】**
凛世的情感告白型コミュ，使用"书信"方式传达心意。

**【花は】Trueコミュ「花の名は」**
凛世在菜花田中与P散步，晚月的对话。
凛世「このように……美しい景色……そう……見られるものでは……」
P「……少し散歩してから帰るか？」
凛世「……！ はい……」
- 意义：凛世用含蓄的方式表达"想一起散步"，P终于一定程度上理解了凛世的言外之意。少女的微小成长。

## 语调特征总结

| 特征 | 描述 |
|------|------|
| 语速 | 极慢、均匀、无急加速（多数情况） |
| 抑扬 | 几乎没有、单调（机器人/町内放送评价的来源） |
| 句尾 | 以「…」为基本停顿单位，每句话间有呼吸感 |
| 敬语 | 基本使用「です・ます」体，对P用「貴方さま」「プロデューサーさま」 |
| 自称 | 通常用「凛世」（第三人称自称），偶尔用「私」 |
| 沉默 | 凛世的沉默本身即是表达——默认、困惑、或内心中涌现情感 |

## 词汇特征

- 「さま」づけ：所有人都用「さま」（能登麻美子角色式称呼）
- 古风用语：如「邁進」「参る」「〜でございます」「〜にございます」
- 省略号大量使用：几乎所有台词都有「…」
- 极少使用感叹号和疑问语气：情绪波动几乎不通过语调变化表现
- 情感爆发时的台词反而更加短促、停顿更多
