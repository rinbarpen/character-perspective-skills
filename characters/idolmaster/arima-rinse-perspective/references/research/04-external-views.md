# 杜野凛世 · 外部视角与评价调研

## ファンコミュニティ評価

### 人气定位
- Shiny Colors人气角色，常年位于角色人气投票前列
- 拥有大量固定推し（担当P）
- 独特的存在感在283プロ中无人能替代

### 代表的なファン呼称
| 呼称 | 由来 |
|------|------|
| 「もりちゃん」 | 友人/同学对她的称呼 |
| 「りんりん」 | 三峰結華命名的爱称 |
| 「70」 | 胸围70cm → 被作为象征性数字使用 |
| 「04」 | 放課後クライマックスガールズ中4人目 |

### キャッチフレーズ系ファン評価
- 「シャニマスで一番重い愛」——全游戏中对P感情最重的角色
- 「無機質なのに一番情感がある」——矛盾的评价但获得多数共鸣
- 「すべての行為がPへの愛で説明できる」——凛世行为的唯一原理=对P的爱
- 「町内放送的な声量で『一生ついて参ります』と言われると本当に一生ものになる」
- 「シャニマスは凛世を登录したまま放置すると『プロデューサーさま……凛世を……お傍に……』と言って待ってるゲーム」

### 最も支持されるキャラの魅力
1. **对P的绝对忠誠**——凛世的"重い愛"被多数P玩家视为"理想的被爱方式"
2. **大和撫子のギャップ**——外表传统内里炽热
3. **成長弧の深さ**——从"没有自己的梦想"到找到自我，凛世的弧光被评价为シャニマス最高级别
4. **コミュの文学性**——凛世のコミュは「行間を読ませる」と高評価

### 批判的视角
- 部分新规P认为凛世"太重い"——对P的依存度让人不适
- 有意见认为凛世的成长弧过慢——但在角色设计上这是有意为之
- 存在性表现争议——"以P为中心的人生"是否健康
- 也有声音说凛世的コミュ"太难懂"——行间信息量过大

## 二次創作傾向

### ジャンル分布
- **P杜（P×凛世）**：占据最大份额。内容以日常/甘やかし/凛世の重い愛描写为主
- **放クラ内**：ユニットメンバー間の日常
- **シリアス**：凛世の内面描写、成長劇
- **ギャグ**：针对凛世的"天然/钝感/重い"特征的搞笑

### 二次創作でよく使われる設定
- バレンタインデー：凛世が「本命」と認識して手作りチョコを渡す
- デート：凛世が少女漫画から学んだ"デート像"と现実のギャップ
- 実家帰省：Pと共に鳥取へ帰省するif
- 結婚後：凛世が主婦になった姿のif
