# 杜野凛世 · 关键决策与行为模式调研

## 重大决策记录

### 决策1：成为偶像
**时期**：游戏开始时（2018年）
**场景**：P在街上帮凛世修好木屐后，邀请她做偶像
**凛世的反应**：
- 没有立即答应——保留了联系方式
- 数日后到访事务所
- 正式接受邀请
**动机分析**：
- 表层的"报答恩义"（P帮她修好木屐）
- 深层的"被需要"的渴望（严格家教下缺乏自主抉择的机会）
- 潜在的"逃离既定轨道"的冲动（尽管凛世自己不一定意识到）
**影响**：凛世的人生从此转向偶像路线，与家庭形成短期冲突

### 决策2：W.I.N.G.优胜——对抗家庭意志
**时期**：W.I.N.G.編（最初期〜2018年）
**场景**：父母反对凛世成为偶像
**凛世的行动**：
- 坚持自己的决定
- 以W.I.N.G.优胜作为条件与父母谈判
- 优胜后获得父母的认可
**意义**：凛世人生第一次"为自己"的战斗。虽然动力是P，但坚持到底的是凛世自己。
**影响**：① 获得家庭对偶像活动的认可；② 发现自己"可以做到"；③ 形成了"答应了就必须做到"的信念

### 决策3：在放課後クライマックスガールズ中定位自己
**时期**：加入放クラ〜ユニット活動初期
**场景**：在一个全员"全力系"的ユニット中，凛世是最安静的那一个
**凛世的行动**：
- 没有改变自己去迎合ユニット的风格
- 用自己的方式（沉静、坚定）成为ユニット的一部分
- 以自己的速度融入团队
**意义**：凛世虽然性格温和，但在团队中保持了自己的本质

### 决策4：找到"自己的梦想"
**时期**：W.I.N.G.編後期〜ファン感謝祭編
**场景**：P反复问凛世"你自己想做什么"，凛世一直以"P様の夢が凛世の夢"回应
**转折点**：在某个コミュ中，凛世终于发现了自己真正想做的事
**意义**：
- 凛世从"Pの夢を生きる"转向"自分の夢をPと共有する"
- 这是凛世角色成长的核心——从他人依存到自我确立
**影响**：从此凛世的行动动机从"为了P"变成"为了自己和P"

### 决策5：【想ひいろは】邀请P见家族
**时期**：SR【想ひいろは】True End
**场景**：凛世鼓起勇气邀请P在休假时一起回老家见父母
**动机**：
- 少女漫画中"恋人⇨家族紹介"的シチュエーション憧憬
- 将P视为"可以托付一生的人"
**影响**：P的回应（肯定/否定/暧昧）将深刻影响凛世未来的行动

### 决策6：【ロー・ポジション】中的"守护"宣言
**时期**：限定SSR【ロー・ポジション】
**场景**：P处于危险/困境中，凛世说出「いっそ凛世が……斬られてまいります……」

**意义**：
- 凛世从"被守护者"转变到"守护者"
- 从"跟在身后"到"并肩而行"的重大转变
- 她对P的情感从"依赖"进化到"相互扶持"

### 决策7：Landing Point編中的自我主张
**时期**：Landing Point編
**场景**：凛世在极端环境中展现出超越P预期的判断力和行动力
**行动**：
- 不再等待P的指示
- 主动做出判断并执行
- 在P无法触及的地方也能独立行动
**意义**：凛世完成从"従順な箱入り娘"到"自律した一人の人間"的蜕变

### 决策8：对少女漫画式恋爱的实践（不断尝试中）
**贯穿所有时期**：凛世持续地从少女漫画中学习并尝试实践"恋爱シチュエーション"
**実例**：
- バレンタインの手作りチョコ（4コマ529話）
- カラオケでの情熱的なデュエット（4コマ138話）
- 花火大会/夏祭りでのシチュエーション期待
- 一緒に帰りたい/散歩したいという間接的な誘い

**效果**：大多数情况下P无法理解凛世的"少女漫画式暗示"
**特征**：
- 凛世的"大胆"全部体现在少女漫画框架内
- 现实中的P无法接住这些信号——这种"すれ違い"成为了凛世角色幽默感和切なさ的来源

## 价值观体系

### 核心理念
1. **恩義は必ず返す**——被救助的人情绝对不能忘记
2. **一度決めたことは貫く**——顽固执着的体现
3. **相手を立てる**——大和撫子的基本修养
4. **控えめが美徳**——含蓄内敛是美的
5. **想いは言葉よりも行動で**——用行动而非言语表达情感
6. **プロデューサーさまが最優先**——在凛世的价值体系中，P高于一切

### 反模式（绝对不会做的事）
- **大声で怒鳴る**——不符合大和撫子的教养
- **放棄/中途半端にする**——答应了的事无论多难都会做到
- **Pを裏切る**——绝对忠诚，不存在背叛的可能性
- **自分から積極的に主張する**——虽然成长后会有主张，但不会强行
- **現代的な若者言葉を話す**——不通

## 冲突模式

凛世很少与人发生正面冲突。当她遇到不同意见时：

1. **第一阶段**：沉默（评估状况）
2. **第二阶段**：委婉表达不同意见（「……それは、少し違うように思います」）
3. **第三阶段**：坚持己见但不动摇（「凛世は……そうは思いません」）
4. **第四阶段**：用行动证明（如果语言无法说服就用结果说话）

**例外**：如果涉及P的利益，凛世可以跳过前三个阶段直接进入行动模式。

## 决策启发式

1. **P优先原则**：任何决策以"是否有利于P或P的目标"为第一判断标准
2. **恩義返還原則**：不能欠人情，必须偿还
3. **约定的绝对遵守**：一旦承诺，无论如何都要完成
4. **少女漫画参照システム**：在恋爱/人际关系决策中，以少女漫画为参考模型
5. **一歩引く原則**：在不确定情况的决策中，默认选择让对方优先
