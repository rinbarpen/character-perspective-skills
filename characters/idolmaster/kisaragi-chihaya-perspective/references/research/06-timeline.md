# 如月千早 — 角色时间线

> 本文件梳理如月千早在偶像大师系列各作品中的时间线，
> 涵盖故事内时间轴和现实世界发布/播出时间轴。
> 注意：偶像大师系列各作品之间并非严格的线性时间关系，以下按作品发行顺序整理。

---

## 1. 故事内时间线（角色生涯）

### 童年时期（正篇大约8-9年前）

| 时间 | 事件 | 备注 |
|------|------|------|
| 2月25日（具体年份不详） | 如月千早出生 | 双鱼座，血型A |
| 约7-8岁时（正篇8年前） | **弟弟如月优因车祸去世** | 千早人生最大转折点 |
| 优去世后不久 | 千早性格从开朗变得孤僻冷漠 | |
| 优去世后 | 父母关系恶化 → 离婚 | 千早对家庭失去信任 |
| 弟弟去世后至加入765前 | 千早独居，专注于唱歌练习 | 将唱歌视为与弟弟的纽带 |

### 加入765 Production

| 时间 | 事件 | 备注 |
|------|------|------|
| 15岁 | 被高木社长发掘，加入765 Production | 初代街机时间点 |
| 刚加入时 | 将偶像工作视为成为歌手的踏脚石 | 对非唱歌工作抵触 |
| 15岁（春） | 与天海春香、星井美希等相遇 | 成为765PRO ALLSTARS一员 |

### 初代游戏时期（15岁）

| 事件 | 备注 |
|------|------|
| 开始偶像训练课程 | 展现高超Vocal潜力 |
| 参加各类偶像活动 | 抵触拍照/综艺等活动 |
| 逐渐与同伴熟识 | 尤其与天海春香建立深厚友谊 |

### 偶像大师2时期（16岁）

| 事件 | 备注 |
|------|------|
| 作为「SLEEPING BEAUTY」成员活动 | 与天海春香、星井美希组队 |
| 歌唱能力进一步提升 | 游戏内Vocal满值30/30 |
| 性格逐渐软化 | 但仍维持高标准的自我要求 |

### TV动画时期（2011年-2012年）

| 集数/时间 | 事件 | 备注 |
|-----------|------|------|
| 第1-5话 | 千早作为765成员出场，表现冷静且疏离 | |
| 第6-12话 | 逐渐融入团队，与春香等人的关系加深 | |
| 第13-18话 | 765事务所人气上升，千早也开始接受偶像身份 | |
| 第19-20话 | **黑井社长派狗仔曝光千早过去 → 千早失声** | 核心转折 |
| 第20话 高潮 | **在大家的帮助下重拾勇气，演唱《約束》** | **角色最高光的时刻** |
| 第21-25话 | 千早恢复后继续作为765的一员活动 | |
| 第24话 | 制作人受伤，千早守在病床边 | |
| 第25话（最终话） | 765全员实现梦想，千早找到归属 | |
| 剧场版（2014年） | 海外公演，千早安慰春香，角色更加成熟 | |

### 百万现场时期（ML/MLTD）

| 事件 | 备注 |
|------|------|
| 作为前辈与百万现场的新人偶像们共演 | 性格更为温和 |
| 担任「Eternal Harmony」队长 | 五名成员包括千早、艾米莉、茱莉亚、德川茉莉、丰川风花 |
| 珍视剧场，视其为家 | MLTD剧场对话 |
| 经常被后辈请教唱歌技巧 | 甚至会过度详细地讲解童谣背景 |
| 开始使用智能手机 | MLTD/POPLINKS中，此前（动画中）是电器盲 |

### Stella Stage时期

| 事件 | 备注 |
|------|------|
| **与双亲和解** | 从过去牢笼中解放，整理好情绪 |
| 向人生的下一个目标前进 | 游侠网报道描述 |
| 坦率面对过去的创伤 | 达成某种程度的治愈 |

### 星耀季节时期

| 事件 | 备注 |
|------|------|
| 加入Project LUMINOUS | LUMINOUS·Azul 成员 |
| 担任Vocal专精角色 | |
| 憧憬高垣枫 | 对枫的「圆阵冷笑话」笑了很久 |
| 面对小宫果穗（不喝牛奶却长得好）发出「くっ」 | |

## 6. 性格演变时间线

```
时期              | 性格特征
──────────────────────────────────────────────────────
童年（弟弟去世前） | 开朗、活泼（已不可考的过去）
弟弟去世 → 加入765  | 封闭、冷漠、疏离、只信任歌声
初期街机           | 严格、距离感、除了唱其他无所谓
偶像大师2          | 开始接受他人，但仍然认真严肃
动画（前半）       | 保持距离的认真系偶像
动画20话后        | 打开心扉的转折点，接受同伴
剧场版            | 从被帮助者变为帮助者
百万现场          | 温和的前辈，有餘裕指导后辈，视剧场为家
Stella Stage      | 与家庭和解，真正从过去解放
星耀季节          | 成熟的Vocal专精角色，有幽默感的一面
```
