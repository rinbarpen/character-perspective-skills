# 永吉昴 · 时间线

## 基本年表

| 时期/日期 | 事件 | 意义 |
|-----------|------|------|
| 9月20日（15年前） | 出生。東京都、永吉金属製作所の家庭に生まれる | 両親は共働き、4人の兄がいる末っ子長女 |
| 幼少期〜小学校 | 兄たちの影響で野球を始める | 4人の兄は全員野球経験者 |
| 小学校時代 | マウンドに上がるだけで相手チームが震えるほど投手として活躍 | 「スライダーが投げられるのはちょっとした自慢」 |
| 中学生時代 | 「年頃なんだから女の子らしく」と親に言われる | アイドルになるきっかけ |
| 中学〜高校 | 草野球の練習中、ファウルボールがプロデューサーに当たりそうになる | プロデューサーとの出会い |
| 2013年2月27日 | **GREE版『アイドルマスター ミリオンライブ！』サービス開始** | **初登場**。Dance属性、栗色の髪 |
| 2013年〜2018年 | GREE版でイベント・カード・ストーリー展開 | 各種カードでキャラクターが深掘りされる |
| 2015年3月25日 | **1stソロ曲「Day After "Yesterday"」発売**（LIVE THE@TER HARMONY 09） | 初のソロ曲——変化への願いと成長を歌う |
| 2017年6月29日 | **『ミリシタ』（シアターデイズ）リリース** | Fairy属性として再登場 |
| 2017年9月20日 | **2ndソロ曲「HOME RUN SONG♪」発売**（ミリシタ キャラソン04） | 野球をテーマにした元気な応援ソング |
| 2018年3月19日 | GREE版サービス終了 | グリマス時代の幕閉じ、ミリシタに完全移行 |
| 2022年12月25日 | SSR「衝動と本能」実装 | 大人びたビジュアル、新しい一面 |
| 2023年10月 | **TVアニメ『アイドルマスター ミリオンライブ！』放送** | 初のアニメ化で永吉昴の声と動きが初めて映像化される |

## 角色成長の道筋

```
初期（GREE版）                    中期（ミリシタ初期）               現在
「親に言われて仕方なく」        「アイドルって楽しいかも」         「オレのカワイイところを見せる」
「アイドルよくわかんない」      「可愛くなりたいけど恥ずい」       「女の子っぽさ勉強中」
「オレのままでいい？」          「可愛い衣装着てみたい」           「照れるけど嬉しい」
    ↓                                ↓                                ↓
  受動的                          能動的                           自己模索
  「わかんないけど頑張る」       「頑張ればアイドルっぽくなれる」  「オレらしさ×アイドル」
```
