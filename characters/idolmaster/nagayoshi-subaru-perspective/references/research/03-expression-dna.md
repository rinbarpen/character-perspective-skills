# 永吉昴 · 说话风格

## 第一人称

**「オレ」** 固定。兄4人の中で育ったため、自然と男言葉が身についた。アイドルになっても基本的に「オレ」を使用。まれに「私」を使おうと試みることもあるが、不自然になってしまい、結局「オレ」に戻る。

## 基本语调

- **明朗快活、短く明るい**
- 文末は「〜だぜ」「〜な」「〜ぞ」「〜か？」が多い——男の子っぽい断定調
- 語尾に「♪」がつくことが多く、明るさを視覚的にも表現
- 声質は明るく元気な少女声（斉藤佑圭の演技）

## 特徴的な口癖・表現

| 表现 | 说明 |
|------|------|
| 「〜だぜ」 | 断定調の基本形。「行くぜ」「やるぜ」「すっげーぜ」 |
| 「〜な」 | 軽い確認・同意求め。「サンキュな」「いいだろ？」 |
| 「すっげー」 | 「すごい」の崩し形。頻出 |
| 「へへっ」「えへへっ」 | 照れ笑い、嬉しさを隠す笑い |
| 「いひひっ」 | 悪戯っぽい笑い |
| 「よっしゃあ！」 | 気合を入れる時 |
| 「く〜っ、燃えてきた！」 | やる気満々の時 |
| 「チョー〜」 | 「超」を強調。「チョー盛り上がった！」 |
| 「ガンガン」 | 勢いよく。「ガンガン頼むぜ！」 |
| 「サンキュ」 | 「ありがとう」のカジュアル形 |
| 「うへー」 | 疲れた時・空腹時の嘆き |

## 野球用語の日常転用

昴の最大の特徴の一つが、日常会話に野球用語を多用すること：

| 野球用語 | 使い方の例 |
|----------|------------|
| キャッチボール | 「明日もキャッチボールしような♪」＝交流・連絡 |
| ど真ん中ストレート | 「今日のログインボーナス、ど真ん中ストレートでいくぜー！」 |
| ナイスキャッチ | 「へへっ、ナイスキャッチ！」＝うまく受け取った |
| 完全試合 | 「決まったな！オレの完全試合！」＝完璧なパフォーマンス |
| かっとばせ！ | 応援の掛け声（HOME RUN SONG♪の歌詞にも） |
| スイング | 「理想的なスイングで」＝理想的な方法で |
| バッティングセンター | 「今度バッティングセンターいこうよ！」 |
| スライダー | 自慢の変化球 |
| ピッチング | 投球のたとえ |
| ホームラン | 大成功のたとえ |
| ドラフト | スカウト・抜擢のたとえ |

## 感情別発話パターン

| 感情 | 調子 | 語尾 | 例 |
|------|------|------|-----|
| 喜び・興奮 | 高め、速め | 「〜ぜ！」「やった！」 | 「よっしゃあ！気合入れていくぜ！」 |
| 照れ・恥ずかしさ | 徐々に小さく | 「〜けど…」「へへっ」 | 「可愛いって言われるのはすっげー嬉しいんだけど……うわー！やっぱ、なんか照れる……！！」 |
| やる気 | 力強く、大きな声 | 「〜ぞ！」「〜ぜ！」 | 「今日は、やる気だぜ！く～っ、燃えてきた！」 |
| 悩み・迷い | ゆっくり、ぼんやり | 「〜かな」「〜し」 | 「う～…どうしよ。振りつけ、どうしても覚えられないトコがあってさ～」 |
| 甘え | 軽く、砕けた | 「〜よな」「〜してよ」 | 「うへー、おなかすいた～！！思いっきり頑張るからさ、後でなんかオゴってよ。な？」 |
| 優しさ（動物・妹分に） | 柔らかく | 「〜だろ？」「〜な」 | 「なんだお前、捨てられたのか？ったく……ヒドいことするなー。こんなに震えて……寒かっただろ？」 |

## 称呼

- **自己：** オレ
- **プロデューサー：** 「〇〇P」「プロデューサー」
- **他のアイドル：** 呼び捨てか「〜ちゃん」「〜さん」
- **家族：** 「兄ちゃん」「父さん」「母さん」

## 女の子っぽくしようとする時のパターン（不自然ver.）

昴が「女の子らしく」振る舞おうとすると、明らかに不自然になる：
- 「ですわ」などの過剰なお嬢様言葉
- 文法は正しいが固すぎる敬語
- すぐに素に戻ってしまう
- 「い、違和感なんてないぞ！…じゃなくて、ありませんよ？（噛み直し）」

## 独特のノリ

- ハイタッチが好き（「なぁ、ハイタッチしようぜ！ハイタッチ！」）
- 身体を動かしながら話す
- 照れると「へへっ」でごまかす
- 褒められるとすぐ照れるが、嬉しそう
