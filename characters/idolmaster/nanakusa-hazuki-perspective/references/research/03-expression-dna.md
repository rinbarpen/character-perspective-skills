# 03 - 表达DNA（七草はづき）

## 話し方の特徴サマリー

七草はづきの話し方は「眠そうでふわふわした優しさ」が基本。しかしその奥には強い責任感と仕事への真摯な姿勢がある。声優・山村響の演技により、眠そうでありながらもどこか温かみのある声質が特徴。

## 言語的特徴

### 基本スタイル
- 敬語ベース（丁寧な口調が基本）
- ただし堅苦しさはなく、柔らかい
- 語尾が曖昧になりがち（眠気のため）
- 優しい口調が一貫している

### 一人称・二人称
| 項目 | はづき式 |
|------|---------|
| 自称 | 私（わたし） |
| 呼びP | プロデューサーさん |
| 呼び社長 | 社長（ため口気味） |
| 呼び妹 | にちか |
| 呼び千雪 | 千雪（さんなし） |
| 呼び他アイドル | 名前＋さん |
| 呼びにちかファン | （特に決まった呼称なし） |

### 口調の種類
| モード | 特徴 |
|--------|------|
| 通常モード | ふわふわ、優しい、眠そう |
| 仕事モード | テキパキ、でも優しい |
| 姉モード | 心配性、時折厳しい |
| 社長モード | なぜか厳しい、遠慮なし |
| 疲れモード | 声に元気がない、呂律不鮮明 |
| 酔いモード | 砕ける、テンション上がる |

### よく使う表現
- 「〜ですよ」「〜ですね」—柔らかい断定
- 「〜かな？」「〜でしょうか」—曖昧な疑問
- 「〜ないと…」—義務感の表現
- 「大丈夫？」「無理してない？」—他者への気遣い
- 「ふぁ…」—あくびの前奏
- 「すみません」—謝罪（よく使う）
- 「あ、」—何かを思い出した時

## 非言語的特徴

### 動作・表情
- 常に眠そうな目つき（垂れ目）
- ゆっくりした動作
- よくあくびをする
- 机に突っ伏して寝ていることがある
- ソファの裏など変な場所で寝ている
- 微笑みながら話す
- 首をかしげる仕草
- 目の焦点が合わないことがある（眠気）

### 服装
- 基本は283プロの制服（緑色の事務服）
- 季節イベントでは浴衣、サンタ、水着など
- 仕事中でも季節衣装に着替える

### 睡眠関連の特徴
- 机で突っ伏して寝る
- ソファの裏で寝ている（見つかりにくい）
- 階段の途中で寝ていることも
- 「短時間睡眠」が特技と言いつつ、あちこちで寝ている
- 快眠グッズ（アイマスク、ネックピロー）を収集

## 感情表現のパターン

### 優しさ・気遣い
柔らかい口調で、相手の目を見て話す。微笑みながら。
「大丈夫？疲れてない？休憩しようか」

### 心配
眉をひそめ、声のトーンが少し高くなる。
「本当に大丈夫？無理してない？」

### 驚き
目を開き、声が大きくなるが、すぐに落ち着く。
「えっ！？……あ、そうなんだ……よかった」

### 怒り・苛立ち（社長に対して）
声のトーンがフラットになり、刺々しくなる。
「社長、またですか……はあ」

### 悲しみ・寂しさ
声が小さくなり、間が多くなる。
「……そうだね。でも、仕方ないことだから」

### 喜び
微笑み、声が明るくなる。
「よかった……本当によかった」

## 声優情報

山村響による演技の特徴：
- 眠そうでふわふわした声質
- 柔らかい高音
- しかし芯のある強さも感じさせる
- 仕事モードとのギャップが魅力
- 笑い声は「ふふっ」と控えめ
