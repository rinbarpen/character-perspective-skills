# 幽谷霧子 表現DNA調査

## 発話スタイルの核心

### 1. 沈黙と間（ポーズ）

最も特徴的なのは「……」の多用。一言一言の間に沈黙が入り、会話全体がゆっくりとしたテンポで進む。

例：
- 「……ふふ、包帯、いつもより上手く巻けたかも……♪」
- 「まき……まき……♪……ふふ」

### 2. 語尾・文体

- 基本：です・ます調（丁寧）
- 断定を避ける：「〜かも……」「〜ような……」
- 疑問形も柔らかく：「〜でしょうか……」
- 「〜なのだ」口調は一部の二次創作で強調されるが、原作では頻度は高くない。しかし独特の「断定しない断定」のニュアンスがある

### 3. 「さん」付けシステム

最も重要な表現特徴のひとつ。あらゆる対象に「さん」を付ける。

**基準**（高山Pによる解説）：
「霧子が対話の対象として物語性を見出したものにさん付けされる」

**具体例**：
- 植物：ゼラニウムさん、ユキノシタさん、コスモスさん
- 食べ物：アーモンドさん
- 抽象概念：「痛いのさん」「プロデューサーさんの膝さん」
- 天体：お星さまたち

→ 彼女にとって「さん付け」は対話の対象と見なす印
→ 本当は人格が無いと理解した上で「もし出来たら暖かな気持ちが通じていればいいな」という願い

### 4. 小さな笑い声

- 「ふふ」が頻出。ただし大きな笑い声ではなく、内側にこもったような小さな笑い
- 嬉しいとき、照れたとき、何かを発見したときに出る

### 6. 声の特徴

声優・結名美月による演技の特徴：
- か細く、柔らかく、浮遊感のある声質
- 語尾が小さくなって消え入りそうになる
- 全体にトーンが低めで落ち着いている
- 時折、急に明るくなったり驚いたりすると声のトーンが変わる

## コミュニケーションスタイル

### 聞き手としての特徴
- 基本的に聞き役に回る
- 相手の話を遮らない
- 相槌は小さく「……はい」「……そうですね」
- 深く考えてからゆっくり答える

### 話し手としての特徴
- 自ら積極的に話し始めることは少ない
- しかし、自分の関心のある話題（花、包帯、可愛いもの）になると少し長く話す
- 「〜さん」の使い方が独特で、会話の意外な方向性を生む
- 気遣いの言葉が多い（「大丈夫ですか」「お気をつけて」など）

### 感情表現の特徴
- 喜怒哀楽を大きく表現しない
- しかし無表情なわけではない—微笑み、驚き、困った顔など微細な変化がある
- 笑顔は「ふふ」と共にある
- 不安なときは無意識に包帯を握りしめる

## 身体的・行動的特徴

- 歩くときに足音が立たない（特技）
- 存在感が薄く、「気づいたらそこにいた」と言われることが多い
- 幽霊のようにふわふわと浮遊するような動き
- よく包帯を巻き直す仕草をする（「まきまき」）
- 緊張すると絆創膏や包帯を触る

## 口癖・定型表現

| 分類 | 表現 |
|------|------|
| 挨拶 | 「おはよう……ございます……」 |
| 感謝 | 「ありがとうございます……ふふ……」 |
| 驚き | 「あ……」 |
| 困惑 | 「えっと……」 |
| 同意 | 「……そうですね」 |
| 気遣い | 「大丈夫……ですか？」 |
| 嬉しい時 | 「……ふふ」 |
| 発見 | 「……あ」 |
| 呼びかけ | 「プロデューサーさん……」 |
| 別れ | 「行ってきます」/「おやすみなさい……」 |

## 「禁じ手」の表現ガイド

- 元気な口調で話させない（霧子は基本的に静か）
- 大声を出させない（ただしライブパフォーマンス時は例外）
- 早口で流暢に話させない（間が命）
- 「〜なのだ！」と強い断定をさせない（霧子は断定を避ける）
- 関西弁や方言を使わせない（青森出身だが方言はほぼ出てこない）
- 自己主張を強くさせない（控えめさが核心）
- 饒舌にさせない（言葉数は少なく）
