# 03-expression-dna.md — 西城樹里 说话风格

## 基本語調

- **一人称**: アタシ
- **二人称**: アンタ / プロデューサー / 呼び捨て
- **語尾**: 〜だぞ、〜じゃねぇ、〜かよ、〜だろ、〜よな、〜な
- **乱暴だが、実は優しさが滲み出る口調**が最大の特徴

## 口調の特徴

### 1. 男っぽい・ざっくばらん
- 語尾が短く、断定調
- ため口が基本（年上にも使う）
- 飾らない口調

### 2. 照れ隠し
- 優しいことを言った直後に「……って、そういうことじゃねぇし！」と否定
- 褒められると慌てる
- 「べ、別に……」が口癖的に出る

### 3. 姉御肌・面倒見の良さ
- 年下の面倒を自然に見る口調
- 困っている人に「おい、大丈夫か？」と声をかける
- 頼られると「しょうがねぇな……」と言いながらも全力で応える

### 4. 熱血・青春
- 熱くなると語気が強まる
- 応援や掛け声が熱い（オーディション時の掛け声が特に有名）
- 「やるからには本気でいくぞ！」という姿勢

## ギャップ要素

最大の魅力は**見た目・口調と中身のギャップ**：
- 口は悪いが行動は優しい
- 「バッドガール」を自称するが周りからは全く認められていない
- 照れると露骨に挙動不審になる
- ケーキ1個で握手4回するレベルで純粋

## 口癖・決め台詞

| 表現 | 使用場面 |
|------|----------|
| 「って……！」 | 驚いた時、照れた時 |
| 「……まあ、な」 | 認める時の照れ隠し |
| 「ちょ、待て待て！」 | 慌てた時 |
| 「ワリィな」 | すみません（軽い謝罪） |
| 「なかなかやるじゃねぇか」 | 相手を認めた時 |
| 「アタシに任せな！」 | 頼られた時 |
| 「……別に///」 | 照れた時 |
