## 10. 话音特徴まとめ

| 特徴 | 詳細 |
|------|------|
| 自称 | 第三人称「ありさ」（几乎不用「私」） |
| 对制作人的称呼 | 「○○Pさん」（与其他ML角色相同的标准称呼，加上名字占位符） |
| 对其他偶像的称呼 | 「アイドルちゃん」——这是她的核心特征称呼方式 |
| 笑声 | 「ムフフ♪」「ムフフ…♪」——标志性满足笑声 |
| 其他感叹词 | 「ほあああ！」「ムホーッ！！」「ふおお」「やぁ～んっ！」 |
| 语尾倾向 | 以「〜ですっ！」「〜ますっ！」「〜ですよぉ〜！」「〜ますよっ！」等ハイテンション礼貌体为主 |
| 加速词 | 「超〜」「とか」「バッチリ」「激レア」「マジ」「エキサイティング」 |
| 典型台詞模式 | 先激动描述场景→突然后悔说太多→用「ムフフ…♪」收尾 |
| 与偶像宅视角相关的表达 | 「ご褒美」「激レア」「布教用」「家宝決定」「聖地巡礼」「保存用」等典型的宅用语 |
