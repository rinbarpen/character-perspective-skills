## 七、角色成长弧线总结

### 阶段一：发现自我（2013-2014）
**关键词**：模索（摸索）
- 发现唱歌的乐趣
- 确立ON/OFF模式
- 首次以角色身份站上舞台

### 阶段二：拓展关系（2015-2017）
**关键词**：関係（关系）
- 与百合子的深化
- 加入多个单元
- 开始在公众面前尝试ON模式生日致辞

### 阶段三：多元表现（2018-2020）
**关键词**：展開（拓展）
- ピコピコプラネッツ确立游戏偶像定位
- Chrono-Lexica展现黑暗面
- 3D实装实现ON/OFF视觉表现

### 阶段四：自我接受（2021-2023）
**关键词**：統合（统合）
- 动画化——面对更广大观众
- 「スポットライト・ミラーランド」对双重自我的思考
- ON模式不再只是「表演」——它成为自然的表达方式

### 阶段五：开花结果（2024-2026）
**关键词**：確立（确立）
- 个人SOLO专辑
- 商业化合作（电竞眼镜）
- 从「内向的游戏宅偶像」到「拥有稳定标签的御宅系代表偶像」

```
2013-2014         2015-2017         2018-2020        2021-2023        2024-2026
摸 索             关 系             拓 展             统 合             确 立
┌────────┐        ┌────────┐        ┌────────┐       ┌────────┐       ┌────────┐
│ON/OFF  │   →    │百合子   │   →    │ピコプラ  │  →   │动画·    │  →    │个人专   │
│模式确立  │        │友情的   │        │黑暗面   │       │双重自我  │        │商业合作  │
│首次solo │        │深化    │        │多元单元  │       │接受     │        │地位确立  │
│害羞     │        │更多单元  │        │         │       │         │        │         │
└────────┘        └────────┘        └────────┘       └────────┘       └────────┘
```

**贯穿始终的不变核心**：
- 喜欢歌（从最初到最后，从未改变）
- ゲーマー（游戏玩家身份永远是她的一部分）
- 使用名字「杏奈」自称
- 对亲近的人展现真实的自己
- ON/OFF的双重身份，但从对立走向统一

## 八、总结

望月杏奈在百万Live系列中走过了13年的旅程。她从2013年一个害羞的14岁游戏宅少女，成长为2026年拥有个人专辑的成熟偶像。她的身份从「内向者×表演者」的对立，逐渐走向「我就是这样的人」的自我接受。

她的成长不是把自己变成了另一个人——而是她学会了在不同的场合调用不同的自己，并理解到：**舞台上的她和屏幕前的她，都是真实的自己。**
