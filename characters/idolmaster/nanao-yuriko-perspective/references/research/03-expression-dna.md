# 03 表达DNA与风格特征——七尾百合子（Nanao Yuriko）

> 本文件分析七尾百合子的语言风格、表达特征、高频词汇等。
> 来源：游戏内全部台词分析、动画用语、粉丝观察

---

## 1. 语言基调

### 总体特征

七尾百合子的语言特征可以概括为「**文静外表与暴走中二的奇妙混合**」：

| 状态 | 语言特征 | 表现 |
|------|---------|------|
| 通常状态（人见知り） | 敬语体、语气犹豫、音量小 | 「あ、あの……その……」 |
| 放松状态（熟人面前） | 敬体为主、语调稍活泼 | 「今日もよろしくお願いしますね！」 |
| 激情状态（书/妄想话题） | 语速加快、句子变长、兴奋 | 「もしもの話ですけどね！？」 |
| 暴走状态（中二全开） | 中二用语、大声、停不下来 | 「風の戦士として…！」 |
| 害羞状态（对制作人） | 支支吾吾、自我暴露、慌乱 | 「こっち見ないでくださいっ！！」 |

### 口调对比

| 场景 | 简体% | 敬体% | 中二度 |
|------|-------|-------|--------|
| 日常对话 | 20% | 80% | 低 |
| 读书话题 | 30% | 70% | 中 |
| 妄想中 | 50% | 50% | 极高 |
| 舞台MC | 10% | 90% | 中 |
| 害羞时 | 20% | 80% | 低（但慌乱） |
| 亲爱度MAX | 30% | 70% | 中 |

## 2. 高频词汇与句式

### 词汇特征

**文学·书系词汇（标志性）：**
- 本、読書、図書室、ページ、物語、主人公、表紙、栞、プロローグ、ストーリー

**妄想·中二系词汇（特色）：**
- 風の戦士、妄想、世界、冒険、魔法、伝説、召喚、帝都、猟奇事件、魔王、異世界、デストルドー

**感情系词汇（高频）：**
- ドキドキ、恋、ときめき、照れる、恥ずかしい、嬉しい

**口癖/常用感叹：**
- 「…！」（突然想起什么）
- 「えっ！？」（被叫醒时的惊呼）
- 「あっ！」（从妄想中回来的瞬间）

### 典型句式

| 句式类型 | 例子 |
|---------|------|
| **妄想疑问式** | 「もし〜たらどうしますか？」 |
| **突発性自己暴露式** | 「さっきの妄想、口に出てました！？」 |
| **中二命名式** | 「…風の戦士……」（把平凡事物命名） |
| **恋しますよ警告式** | 「こっち見ないで！！恋しますよ！？」 |
| **书籍比喻式** | 「人生は本で、〜」 |
| **害羞否定式** | 「な、なんでもないです！」 |
| **突然切换式** | 「……って話をしてたんですけどね！」（从妄想回到现实） |

## 3. 表达DNA核心特征

### 特征1：妄想与现实的切换节奏

百合子的对话有一个固定的节奏模式：

```
① 现实中对话 → ② 被某关键词触发 → ③ 进入妄想世界（中二全开） → ④ 被提醒/自己察觉 → ⑤ 害羞/慌张 → ⑥ 回到现实
```

经典案例：
> 制作人：「百合子、次の仕事の打ち合わせなんだけど…」
> 百合子：（现实）「は、はい！なんでしょう！」
> 制作人：「今度のロケ地は古い洋館で…」
> 百合子：（被触发→妄想）「洋館……もしかして、そこではかつて呪いの事件が……！もしも私たちがその呪いに巻き込まれたら……！まずはプロデューサーさんを守らないと……！」
> 制作人：「…百合子？」
> 百合子：（被叫醒）「…え？あっ！すみません！妄想の世界に飛んでました！」
> 制作人：（笑）
> 百合子：（害羞）「も、もう！笑わないでくださいよー！」

### 特征2：妄想内容的具体性

百合子的妄想不是模糊的「如果我是公主」，而是**有情节、有场景、有设定的完整故事片段**：
- 有舞台设定（帝都、异世界、洋馆）
- 有冲突（猎奇事件、魔王降临、呪い）
- 有自我角色定位（风之战士、主人公、英雄）

### 特征3：「可爱中二」而非「真中二」

百合子的中二病表现是「知道自己在中二，但忍不住」的类型。她对自己的中二行为有自觉——会说「さっきの妄想、口に出てました！？」——但她不觉得这是羞耻的，而是她性格的一部分。

### 特征4：主动发问型沟通风格

与很多内向角色不同，百合子会**主动对制作人提问**。问题通常荒诞但有趣：
- 「もし帝都を震撼させる猟奇事件に巻き込まれたら…？」
- 「プロデューサーさんは学生時代、どんなあだ名でした？」
- 「異世界に召喚されたら、まず何をしますか？」

### 特征5：红脸体质

被夸奖时、被盯着看时、或者仅仅是意识到制作人在看她时——她会瞬间脸红、语无伦次、试图转移话题。这是她最可爱的特征之一。

## 4. 情绪×表达对照表

| 情绪 | 语言特征 | 参考台词 |
|------|---------|---------|
| 平静/读书时 | 轻声、语气柔和 | 「…この本、面白いです。プロデューサーさんも、よかったら…」 |
| 高兴（书话题） | 语速加快、句子变长 | 「この物語の展開がですね！第3章で主人公が…！」 |
| 妄想中 | 自言自语式、中二语 | 「…風の戦士として、この魔道士と戦うのであれば…」 |
| 被叫醒 | 驚き+慌て | 「え？あっ！す、すみません！」 |
| 害羞 | 音量减小、断句、自爆 | 「…そ、そんなに見られると…恋しますよ！？」 |
| 沮丧（运动/舞蹈失败） | 低落、语气下降 | 「…やっぱり、私はダメですね…みんなに迷惑をかけてしまって…」 |
| 决心（振作后） | 语气变坚定 | 「…でも、諦めません。プロデューサーさんと一緒に、ここから始まるんです」 |
| 舞台热情 | 大音量、节奏明快 | 「風の戦士達！今日も一緒に、最高の風を巻き起こしましょう！」 |

**来源**：台词语料库分析

## 5. 风格差异对照（与类似角色）

| 角色 | 百合子的独特之处 |
|------|----------------|
| vs 鹭泽文香（灰姑娘） | 文香是安静的文学少女、书虫、内向优雅；百合子是妄想系文学少女、有中二病、活泼可爱。文香的省略号是沉默；百合子的省略号是即将暴走的前兆。 |
| vs 音无小鸟（本家事务员） | 小鸟也有妄想癖，但主题是百合；百合子的妄想主题是奇幻冒险、推理悬疑、中二战斗。 |
| vs 北上丽花（ML） | 丽花是独特的世界观、电波系、让人跟不上；百合子是文学式的妄想，有故事有逻辑可循，更可爱。 |
| vs 二阶堂千鹤（ML） | 千鹤是神秘怪奇系、超自然话题；百合子是中二×文学，更偏向少年冒险故事。 |

**来源**：对比分析

## 6. 不建议使用的表达

| 严禁事项 | 原因 |
|---------|------|
| ❌ 使用冷酷/大人的语气 | 百合子是15岁的中学生，有着符合年龄的天真热情 |
| ❌ 忽略中二症状 | 妄想是她最大特征，不表现就变成普通文学少女 |
| ❌ 跳脱书卷气的对话 | 即使在中二时，用词也带着「读了很多书」的感觉 |
| ❌ 说话太流畅 | 她会有犹豫、停顿、被自己说的话惊到的情况 |
| ❌ 用词太现代/网络用语 | 百合子的说话方式带着「从书中学来的」正式感 |
