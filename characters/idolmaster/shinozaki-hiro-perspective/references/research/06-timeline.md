## 学園アイドルマスター 配信後

### STEP1（親愛度1〜10話）

| 話数 | 出来事 | 意味 |
|------|--------|------|
| 1話 | 入学、廊下で倒れる | プロデューサーとの出会い |
| 2話 | 逆スカウト | 「わたしをプロデュースしてほしい」 |
| 3話 | 「ままならない」哲学の開示 | 広の核心的な価値観が明らかに |
| 4-5話 | 補習組での生活 | 千奈・佑芽との交流開始 |
| 6話 | レッスンでの奮闘 | 体力0の現実と向き合う |
| 7話 | プロデューサーの悪夢 | 広が「成功したら辞めるのでは」という不安を逆に論破 |
| 8話 | 佑芽のライブ鑑賞→突貫ライブ | 「悔しい」の自覚と初めての自主的行動 |
| 9話 | ライブ成功 | アイドルとしての成長の証明 |
| 10話 | 本当の動機を告白 | 「かわいくなりたかった」——最大のターニングポイント |

### STEP2（親愛度11〜20話）

| 話数 | 出来事 | 意味 |
|------|--------|------|
| 11-12話 | プロデュース体制の本格化 | 仮契約から本契約へ |
| 13-15話 | さらなる成長と壁 | 体力面・技術面の向上 |
| 16-18話 | 補習組3人の絆が深まる | 千奈・佑芽との友情の本格化 |
| 19話 | プロデューサーへの信頼の変化 | 距離が縮まる |
| 20話 | 「広って、呼んで」 | 名前呼び許可＝最大の信頼の証 |

### STEP3（親愛度21〜27話）

| 話数 | 出来事 | 意味 |
|------|--------|------|
| 21-23話 | H.I.Fへの布石 | 更なる高みへの挑戦 |
| 24-26話 | プロデューサーとの関係深化 | 「あなた」呼びが混ざる |
| 27話 | STEP3完 | ファンディスク的性格の充実編 |

### N.I.A編

| フェーズ | 出来事 | 意味 |
|---------|--------|------|
| プロデュース開始 | プロデューサーと二人三脚で戦略 | 広の強み（知力・神秘性）を活かした戦い |
| 神格化 | ミステリアスな雰囲気がバズり「女神」に | 広の外見的特徴が武器になる |
| 千奈との友情 | ライバルとしての切磋琢磨 | 初めての「同等の相手」との関係 |
| 佑芽との競争 | 追う側から追われる側へ | 成長の証明 |
| 決勝 | 見事に勝利 | 「趣味で友達に勝つ」ことの大切さを証明 |
| その後 | 父親にプロデューサーが認められる | 家族の承認とプロデューサーの承認の一致 |
