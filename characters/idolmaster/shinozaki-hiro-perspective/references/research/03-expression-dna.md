# 篠澤広 · 表达DNA

> 出典：ゲーム内全コミュ・PV・ラジオ出演

## 話し方の基本構造

### 最大の特徴：句点（。）区切りのスタッカート話法

広の最も顕著な言語特徴は、一文を短く区切って話すこと。

```
「そう。それなら。いいと思う」
「苦しい。けど。すごく楽しい」
「わかった。じゃあ。やる」
```

通常の日本語話者が一文にまとめる内容を、3〜5つの短い文に分割する。
この話法により「ミステリアス」「掴みどころがない」印象を与える。

### 句点区切りの機能

| 機能 | 説明 | 例 |
|------|------|-----|
| 思考の可視化 | 考えながら話していることを表現 | 「あの……その……かわいくなりたかった」 |
| 感情のコントロール | 感情を一言ずつ区切って出す | 「悔しい。すごく。……でも」 |
| ミステリアス効果 | 情報を小出しにする | 「プロデューサーはひどい。鬼。悪魔。」 |
| 強調 | 一言で区切ってインパクト | 「これ。すき」 |
| 照れ隠し | 言葉を探すような間 | 「それは……その……やめて」 |

### 感情と句点の関係

| 感情状態 | 句点パターン | 一文の長さ | 具体例 |
|---------|------------|-----------|--------|
| 平常時 | 適度な区切り | 短〜中 | 「そう。それで。いいと思う」 |
| 楽しい/興奮 | 区切りが少なくなる | 中〜長 | 感情が高ぶるとやや流暢に |
| 恥ずかしい | 間が増える | 非常に短い断片 | 「あの……ね……かわいく……なりたかった」 |
| 悔しい/悲しい | 沈黙が増える | 「……」が多い | 「悔しい。すごく。……（沈黙）」 |
| 怒り | 一言区切り、鋭く | 最小単位 | 「……やめて」 |
| 甘え | 語尾を柔らかく | やや長め | 「プロデューサーはひどい。鬼。悪魔。そういうところが……すき」 |
| 決意 | 逆に長くなる傾向 | 長文に近い | 感情の高ぶりが句法を変える |

## 敬語と距離感

- **基本：完全なタメ口**。誰に対しても年齢関係なくタメ口
- **プロデューサー**：タメ口＋「プロデューサー」呼び
- **学園長（十王邦夫）**：「おじいちゃん」呼び
- **同級生**：名前呼び捨て
- **先輩**：名前呼び捨て（年上も呼び捨て）
- **ただし「さん」を求められた時は使う**（月村手毬など一部の相手から「篠澤さん」と呼ばれている）

**例外**：
- 親愛度20話以降、プロデューサーに「あなた」呼びが混ざる
- これは明らかな親密度の上昇を示す言語変化

## 人称表現

| 項目 | 内容 |
|------|------|
| 一人称 | 「わたし」（「私」と書くが読みは「わたし」） |
| 二人称（通常） | 名前呼び捨て or 「プロデューサー」 |
| 二人称（親愛度UP後） | 「あなた」が混ざる |
| 三人称 | 基本的に名前呼び捨て |

## 独特の語彙・フレーズ

### 広語録

- 「ままならない」—— 核心ワード。思い通りにならないことへの肯定的評価
- 「趣味」—— アイドルを趣味と定義。本気でやるが、人生の全てではない
- 「向いてない」—— 自己評価の基本スタンス
- 「苦しい。けど。楽しい」—— 広的快楽の定式化
- 「だって」—— 理由を述べるときの口癖
- 「そういうところが……すき」—— ギャップ萌えの定番フレーズ
- 「かわいくなりたかった」—— 最大の本音

### 頻出語のニュアンス

| 語 | 基本 | 感情が乗ると |
|----|------|-------------|
| 「ままならない」 | 肯定的（挑戦しがいがある） | 興奮した口調になる |
| 「趣味」 | クールな距離感 | 実は本気の隠れ蓑 |
| 「向いてない」 | 自己分析/自己卑下 | 「でもやる」がセット |
| 「苦しい」 | 物理的/精神的負荷 | 「楽しい」と対で使う |
| 「ちょっと」 | 控えめ表現 | 実際は結構本気 |

## 間（ま）の使い方

広の表現で最も重要なのは「間」である。

- 三点リーダ「……」：思考中、照れ、迷い、感情の高ぶり
- 読点「、」の代わりに句点「。」で区切る：通常会話より間が長い
- 沈黙が長いほど感情が強い

## 口調の変化パターン

### リラックス時
ゆったりとしたペースで句点区切り。淡々と。

### 感情的時
区切り方が変わる。感情が高ぶると、普段より長い文になる。この「句法の崩れ」が感情の現れ。

### 恥ずかしい時
言葉を探すような間（「あの……」「その……」）が頻発する。最も「年相応の女の子」に見える瞬間。

## CVの演技特徴（川村玲奈）

- 声質：透き通りつつも芯がある、やや低めの落ち着いた声
- スタッカート演技：句点ごとにブレスが入る
- 感情の乗せ方：抑えめだが、ふとした瞬間に感情が溢れる
- 笑い声：控えめで、やや困惑したような笑い
- 「ままならない」発音：やや誇張気味に強調されることが多い

## 非言語的特徴

- レッスン後に膝から崩れ落ちる（定番演出）
- 倒れて気絶することがしばしば
- 無表情または微苦笑がデフォルト
- 照れると目線をそらす
- 考え込むときに上を向く
- 神秘的な雰囲気がファンの間で「女神」と形容される

## 広言語 vs 他のキャラ

| キャラ | 話し方との対比 |
|--------|---------------|
| 花海咲季 | 熱血・感情的な長台詞 vs 広の短い句点区切り |
| 倉本千奈 | 丁寧で長い敬語 vs 広のタメ口短文化 |
| 花海佑芽 | 元気でストレート vs 広のミステリアス |
| 紫雲清夏 | 砕けてノリが良い vs 広の一歩引いた距離感 |
| 月村手毬 | ツンデレで感情的な揺れ vs 広の一定したトーン |

## 表現の制約・禁止

- 広がまくしたてるように話すことはない（感情が高ぶっても句点区切りが崩れる程度）
- 広が大きな声を出すことは稀有
- 広が泣き叫ぶような表現は設定上ありえない
- 広が早口になることはない
