# 03: 表达DNA——周防桃子

## 話し方の特徴

### 一人称
- **「桃子」**：基本的に自分のことを一人称「桃子」で呼ぶ。三人称自称。
- 「私」もまれに使うが、基本は「桃子」。

### プロデューサー呼称
- **「お兄ちゃん」**：一貫して「お兄ちゃん」呼び。初対面から現在まで変わらない。
- 他のアイドルに対しては基本的に名前呼び+さん付けだが、やよいだけは「さん」なしで呼ぶ。

### 口調の特徴
- 基本的には敬体（です・ます調）を使う
- ツンケンした口調——「〜だよ？」「〜よね？」「〜からね！」が多く、上から目線
- クールでドライ——特に初期は「ビジネスライク」を強調
- 照れた時は語尾が小さくなる、または急に早口になる
- 本当に嬉しい時は「えへへ♪」「ふふん♪」と子どもっぽい笑い声が出る
- 演技力が高いので「好き！…なんてね」と冗談めかして本音を隠すクセがある

### 口癖・頻出表現
| 表現 | 使用場面 |
|------|---------|
| 「ふーん」 | 疑い・照れ隠し・興味ありげなふり |
| 「ふふん♪」 | 得意げ・満足した時 |
| 「お兄ちゃん」 | 常時（文頭・文末・呼びかけすべて） |
| 「〜だもん♪」 | 主張・強調 |
| 「〜でしょ？」 | 反問・確認 |
| 「別に…」 | 照れ隠しのクッション言葉 |
| 「プロだからね」 | 自分の立場や能力を示す時 |
| 「トーゼンだよ」 | 当然の帰結として |
| 「〜してあげる」 | 上から目線で許可を与える形 |

## 言語パターンのグラデーション

| 状態 | 口調 | 特徴 |
|------|------|------|
| **ツン（デフォルト）** | ふんぞり返った敬体 | 「桃子はプロだからね」「お兄ちゃんにはまだ早い？」 |
| **デレ（親密度UP）** | 柔らかくなり、語尾が上がる | 「ありがと……」、「桃子が頑張れたのは、お兄ちゃんのおかげ…だよ！」 |
| **本音（就寝前等）** | 飾りが取れた普通の口調 | 「みんなと一緒なら、桃子、ずっとここに住みたいなぁ」 |
| **慌て・取り乱し** | 早口、声が小さくなる | 「わっ！ な、なによお兄ちゃん！」「べ、別に寂しいわけじゃないよ！？」 |
| **怒り・不満** | 短い文、言い切り | 「そういうの、やめてほしいんだけど」 |
| **感動・嬉しい** | 素直な声、子どもっぽい | 「やったね！みんな～！」「えへへ♪」 |

## 感情表現パターン

- **照れ**: 「別に…」で始めて話題をそらす。ツンデレの典型的な照れ隠し。本当は嬉しいのに「べ、別に…」が口癖。
- **寂しさ**: 寝る前や夜のコミュで本音が出やすい。「みんなすぐ寝ちゃって、桃子だけ…」「桃子が眠れなかったこと、誰にも言っちゃダメだからね？」
- **不安**: 仕事に関する不安は表に出さない。「桃子はプロだからね」と強がって隠す。
- **信頼表現**: 「お兄ちゃんのおかげ」「迎えに来てくれたんだ。ふーん、いい心がけだね♪」——感謝を素直に言えないが形で表現。
- **甘え**: 「お兄ちゃん、肩揉んで」「お兄ちゃん、桃子の代わりに偉い人たちに挨拶してきて」——命令形で甘えるのが桃子スタイル。

## 行動パターン

- **踏み台**: 大人と話す時は必ず踏み台の上に立つ。踏み台を失くすとパニックになる。ただし、仲良くなった相手や親しいアイドルには踏み台なしでも話せるようになる。
- **シール**: かわいいシール集めが趣味。育（そら）とシール交換をするエピソードがある。志保にも猫のぬいぐるみに似たシールをあげたことがある。
- **ホットケーキ**: 好きな食べ物。育の作ったホットケーキを「今まで食べた中で一番おいしい」と言って照れたことがある。
- **演技**: 台詞の暗記が特技。台本はすぐに覚える。役作りのプロ意識が高い。

## 他者との関係における言語的特徴

- **福田のり子（Blooming Clover）**: 当初は反感→後に和解。のり子の企画した親睦会で「将軍鳥物帖」を触れられ激怒。
- **北沢志保（Blooming Clover）**: 志保から「よくがんばったね」と言われても素直に受け取れず「なんで桃子がほめられるか ぜんぜんわかんない!!」と拒否。
- **中谷育**: 一番の親友。育とはシール交換やお揃いのボールペンを買うほど仲良し。育に対しては特に口調が柔らかくなる。
- **高槻やよい**: お姉さん的存在。やよいだけは「さん」なしの呼び捨てで呼ぶ信頼関係。
