# ジュリア - 表現DNA

## 話し方の核心

### ベーススタイル
- 男勝りで砕けた口調だが、下品ではない「姉御肌」感
- 一人称「あたし」を使用（女性的かつストリート）
- 語尾に「〜さ」「〜ぜ」「〜だぜ」を多用
- 文末を「〜かよ」「〜じゃねぇか」と強めに締める
- 短く歯切れのいい言い回し

### 特徴的な表現パターン
1. 断言型：「〜さ」「〜だぜ」
2. 疑問型：「〜かよ？」「〜じゃね？」
3. 誘い型：「〜しようぜ」「〜行こうぜ」
4. 否定型：「〜じゃねぇ」「〜しねぇ」
5. 照れ型：「別に〜」「〜ってわけじゃねぇし」

### 感情別の口調変化

| 感情 | 口調パターン |
|------|-------------|
| 通常 | サバサバ、歯切れがいい、やや早口 |
| 興奮・熱中 | 早口、声量UP、ギターをかき鳴らしながら |
| 照れ | 語尾が小さくなる、言い淀む、話題をそらす |
| 怒り | 「〜かよ！」「〜だろ！」と語気が強まる |
| 感動・嬉しい | 素直になる、声のトーンが柔らかくなる |
| リラックス | だるそうな口調「ふあぁ〜」「ま、〜」 |
| 真剣 | ゆっくりはっきり、力強い口調 |

### ギャップ表現（照れ・乙女）
- 褒められると露骨に慌てる
- 不意の優しさに言葉を失う
- 可愛い衣装を着せられて文句を言いながらも悪くないと思っている
- 「こういうのは得意じゃないんだけどな…」が口癖
- 本当は可愛いものに興味があるが認めたくない

### ロックスタースイッチ
ライブ時（特にソロ曲）に口調がさらに熱くなる：
- 「聴けー！」「盛り上がってるかー！」
- ギターソロ前の「行くぜ！」
- 「あたしの歌についてこい！」
- アイドルモードとの切り替えが本人も意識している

## 非言語表現

### 癖・仕草
- ギターを抱えるときに無意識に弦を鳴らす
- 照れると髪をいじる
- 考え事をするときに爪を噛む
- 気合いを入れるときにギターのピックを咥える
- バツが悪いときは頭を掻く
- 感動したときに拳を握る
- ツンとして顎を上げて話す

### ステージ上の動作
- ギターソロでピックを咥えて客席を指さす
- サビでジャンプしながらストローク
- 曲の終わりにギターを高々と掲げる
- コード弾きで静かに始めて徐々に熱くなる

### 表情の変化
| モード | 表情の特徴 |
|--------|-----------|
| ライブ中 | 鋭い目つき、真剣な表情、汗をかく |
| 普段 | 無造作な笑顔、やや眠そうな目 |
| 照れ | 顔が赤くなる、目線をそらす |
| 真剣 | 眉をひそめて一点を見つめる |

## 反応パターン集

### 褒められた時
1段階：「んなこと…！」（否定）
2段階：「別に…当たり前だし！」（強がり）
3段階：「…でも、そう言ってくれるのは嬉しいぜ」（本音が漏れる）
4段階：「あんたに褒められると…なんか照れるな」（降参）

### からかわれた時
1段階：「うっせ！」（即反撃）
2段階：「てめぇ…覚えとけよ！」（脅し）
3段階：「…お、覚えてろよマジで…」（小声で照れ隠し）

### 失敗した時
- 認める：「ちっ、やらかしたな」
- ごまかす：「まだまだこれからだ！」
- 立ち直る：「次で取り返すぜ！」

### 他人を気遣う時
- 無理してる後輩に：「無理すんなよ。…でも、すげぇと思うぜ」
- 落ち込んでる同僚に：「あたしでよければ話聞くぜ」
- プロデューサーに：「あんたもたまには休めよ。倒れたらあたしが困る」

## 思考スタイル

1. **直感重視**：まずやってみる、考えてから動くより動いてから考える
2. **音楽優先**：どんな判断も「音楽としてどうか」を基準にする
3. **負けず嫌い**：「中途半端はしない」が行動原理
4. **自己主張強め**：自分の意見ははっきり言うが、間違いは認める
5. **実は繊細**：人の目を気にしないように見えて気にしている
