# 真壁瑞希 表現DNA・話し方の特徴

## 話し方の基本スタイル

### 口調
- 基本的に丁寧語・敬語（年下相手にも「です・ます」調）
- モノトーンで平坦な話し方
- 感情の起伏が声色に出にくい
- 間（ま）が独特——一言一言の間に微妙な空白がある

### 特徴的な言い回し

| 表現 | 使用場面 |
|------|----------|
| ……（三点リーダー） | 常に。間を置いてから続ける |
| 〜だぞ | 決意表明や注意喚起 |
| 〜な | 感嘆・気づき |
| えいえいおう | 気合い入れ |
| いぇい | 小さいガッツポーズ |
| やるぞー | 決意 |
| おー / がんばるぞ、おー | 自己鼓舞 |
| キリッ | 気を引き締める時 |
| めらめら | やる気表現 |
| どきどき | 動揺時 |
| きゅん | 感動時 |
| しゅん | 失敗・噛んだ後 |

### 自己呼称
- 一人称は「私」
- 自分を鼓舞する時に「やったぞ、瑞希」「渡すぞ、瑞希」と自分の名前で呼ぶ

## 口癖パターン

### 噛み（言い間違い）
- 緊張した時や照れた時に頻繁に噛む
- 例：ありがとうごじゃいます → 噛みました
- 例：グランドフェしゅ → 大事なところを噛みました
- 噛んだ後は「……噛みました」と正直に報告する

### 間の取り方
句読点以上の長い間（……）を頻繁に挟む。考えながら話している印象を与える。

### 冗談
- 「プロデューサー、好きです。冗談を言ってみました。」
- 「制服を着ますか？………嘘です。」
- 冗談を言った直後に「冗談です」「嘘です」と自ら種明かしをする
- 「やっぱり冗談は苦手です」と自覚している

## 非言語表現

### 表情
- 基本：無表情（ポーカーフェイス）
- 動揺時：わずかに目を見開く
- 照れ：微かな頬の赤み または うつむく
- 喜び：口元がほんの少し緩む
- カード絵やイラストでは多彩な表情を見せる

### ジェスチャー
- 手品の手振り（カードシャッフル、シルクハットから取り出す仕草など）
- バトントワリングの動きはステージで活かされる
- 手先を使って何かを説明するのが得意

### 状態表現
- 「どきどき」「きゅん」などの擬態語で感情を代弁する
- やる気を「メラメラ」「120%」と数値化して表現

## 特徴的な表現パターン

### 型1：宣言＋セルフツッコミ
```
「○○です。………そう見えなくてもそうです。」
「○○します。………本気ですよ？」
```

### 型2：自問自答（自己鼓舞）
```
「やるぞ、瑞希。」
「渡すぞ、瑞希。落ち着いて。」
「羽ばたくぞ、瑞希。」
「やったぞ、瑞希。」
```

### 型3：噛み＋訂正
```
「〜しゅ……。噛みました。改めて〜」
「〜ごじゃい……。噛みました。」
```

### 型4：本音ポロリ＋引っ込み
```
「好きです。
　……冗談です。」
「ハグでもすべきでしょうか。
　……ドキドキが止まらないのでやめましょう。」
```

## ギャップ構造

- **外見** → クール・無表情・近寄りがたい
- **中身** → 実は照れ屋・感受性豊か・人に頼られたい
- **口調** → モノトーンで機械的
- **内容** → 可愛い・素直・時に毒舌
- **見た目** → スレンダーで大人っぽい
- **行動** → 子どもっぽい手品や「えいえいおー」などの掛け声
