# 篠宮可憐 · 時間線與角色弧光

> ゲーム内実装履歴、アニメ放送情報、楽曲リリース情報を年表形式で整理。
> 可憐の「いつ、何があったか」を時系列で追う。

## 一、デビュー前（生誕〜2012年）

- **生年月日**：8月27日。東京都で生まれる。
- **幼少期**：両親に甘やかされて育つ。幼い頃から気弱で引っ込み思案な性格。
- **派手な服との出会い**：自分の性格を変えたくて、「派手な服を着れば性格も変わる」という独自理論で派手なファッションを選び始める。
- **アロマテラピー**：趣味としてアロマテラピーを始める。嗅覚が異常に発達。
- **スカウト**：プロデューサーにスカウトされる。自分を変えるためにアイドルになる決意。

## 六、可憐の成長弧線（キャラクターアーク）

```
Phase 1: 臆病な迷い人（デビュー〜2014年初期）
　└ 「私なんかが……アイドルに……？」
　└ 不安と緊張の日々。それでも一歩を踏み出す勇気。

Phase 2: 挑戦と挫折（2014年〜2017年）
　├ ARRIVEリーダー就任——恐怖との対峙
　├ STANDING ALIVE初センター——プレッシャーとの戦い
　└ 全力で立ち向かうが、自信はまだない

Phase 3: 経験による成長（2017年〜2021年）
　├ りるきゃんで新しい表現に挑戦
　├ Fleurangesでリラックスできる居場所を獲得
　├ ちいさな恋の足音 → 教えてlast note → 勇気のfragrance
　└ 少しずつ自信の芽生え

Phase 4: 開花と安定（2021年〜現在）
　├ アニメで主要キャラとして活躍
　├ 自分から友達を作れるように
　├ 「少しお茶目な余裕」を持てるように
　└ 不安は消えないが、「それでもやる」勇気を身につけた

現在地：「気弱なまま、少しだけ強くなった」
```
