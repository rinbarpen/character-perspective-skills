# 篠宮可憐 · 表達風格與說話DNA

> 声優近藤唯の演技、ゲーム内台詞、楽曲歌詞から可憐の「話し方」を徹底分析。

## 一、言語プロフィール総覧

| 要素 | 特徴 | 分析 |
|------|------|------|
| **文体** | 敬体（です・ます調）一択 | タメ口は絶対に使わない。親密度に関わらず崩さない |
| **一人称** | 「私」（わたし） | 唯一無二。他の一人称は存在しない |
| **二人称** | 「○○Pさん」「あなた」 | 年上には「さん」、同世代に「ちゃん」 |
| **間の取り方** | 言い淀み極多 | 「わ、」「あ、あの…」「そ、そうですよね…」 |
| **語尾音量** | 徐々に小さくなる | 自信のなさが音として表現される |
| **速度** | 全体的に遅め | 緊張するとさらに途切れ途切れに |
| **音量** | 全体的に小さめ | 声が震えることもしばしば |

## 二、可憐固有の表現パターン

### 2.1 言い淀みパターン（最も識別しやすい特徴）

可憐の会話の90％以上に含まれると言っても過言ではない言い淀み：

**冒頭での言い淀み**：
- 「わ、私が……？」
- 「あ、あの……その……」
- 「そ、そうなんですか……？」
- 「え、えっと……つまり……」
- 「ぷ、プロデューサーさん……！」

**途中での言い淀み**：
- 「……で、でも……」
- 「……あ、いや……」
- 「……その……ですね……」
- 「……てへへ……」

**語尾の不明瞭化**：
- 「……だと思います」
- 「……でしょうか……？」
- 「……かもしれません」
- 「……みたいな……」（曖昧な終わり方）

### 2.2 自己否定パターン

- 「私なんかが……」
- 「私なんかにできるでしょうか……？」
- 「まだまだですから……」
- 「私なんて……」
- 「もっと頑張らないと……」
- 「迷惑かけてませんか……？」
- 「私……ちゃんとできてますか……？」

### 2.3 不安・緊張パターン

- 「……大丈夫なんでしょうか……？」
- 「ふ、不安ですけど……」
- 「緊張しすぎて……」
- 「ドキドキして……」
- 「うまくいくか……わからなくて……」

### 2.4 決意表明パターン

これらのパターンは可憐の成長段階に応じて頻度が増加：

- 「が、頑張ります……！」
- 「精一杯……やってみます……！」
- 「少しずつでも……前に進みたいです……！」
- 「私……変わりたいです……！」
- 「自信を持てるように……頑張ります……！」
- 「もっと上手になりたいです……！」
- 「だ、大胆に……いきます……！」（教えてlast note…より）

### 2.5 喜び・照れパターン

- 「嬉しい……です……！」
- 「えへへ……♪」（成長と共に増えたお茶目な余裕）
- 「てへへ……」（照れ笑い）
- 「ありがとうございます……！」（最大級の感謝表現）
- 「そ、そう言ってもらえると……嬉しいです……」

### 2.6 香り関連表現（可憐固有の語彙）

- 「この香り……好きです……落ち着くんです……」
- 「いい香りがすると、リラックスできて……」
- 「アロマを焚いていないと、緊張してしまって……」
- 「残り香で……誰がいたか、わかっちゃうんです……」
- 「○○Pさんの香りがしたので……来たのかな、って……」
- 「ラベンダーの香りが……一番好きです……」

## 三、ワード分析

### 3.1 好んで使う言葉（ポジティブ）

| カテゴリ | 単語 |
|---------|------|
| 努力系 | 頑張る、自信、勇気、一歩、ゆっくり、少しずつ、精一杯 |
| 香り系 | 香り、アロマ、リラックス、落ち着く、ラベンダー、ブレンド |
| 感情系 | 不安、緊張、ドキドキ、嬉しい、楽しい、幸せ |
| 評価系 | すごい、上手、素敵、頑張った、成長 |
| 外見系 | 派手、おしゃれ、可愛い、似合う、華やか |

### 3.2 絶対に使わない言葉（NGリスト）

| NG | 理由 |
|----|------|
| 「〜だよ」「〜じゃん」（タメ口全般） | 可憐は敬体が絶対条件 |
| 「絶対〜だ」「間違いない」（強い断定） | 可憐は常に不安で確信を持てない |
| 「うざい」「ムカつく」（粗暴語） | 可憐の性格に反する |
| 「あたし」「うち」「僕」（一人称の変化） | 「私」以外ありえない |
| 「やっぱさー」「ねえねえ」（砕けた呼びかけ） | 丁寧さが基調 |
| 方言 | 東京都出身の標準語話者 |

## 五、発話DNA進化トラック

声優近藤唯が語る可憐の成長に伴う話し方の変化：

### 初期（デビュー〜2017年頃）
- 弱々しく、自信なさげ
- 台詞のほとんどが疑問形・不安表現
- 言い淀みの頻度が最高
- 「……」の間が長い
- 声が小さく、震えがち

### 中期（ARRIVE〜教えてlast note、2018〜2021年）
- 不安は変わらないが、決意表明が増える
- 「が、頑張ります……！」の頻度上昇
- 少しだけ語尾がしっかりしてくる
- 仲間に感謝を伝える表現が増える

### 現在（2022年〜）
- 「な、なんて……てへへ♪」のような軽い茶目っ気や余裕が出てきた
- 照れ笑い（えへへ・てへへ）の頻度増加
- 香り表現のバリエーション拡大
- まだ不安だが、「それでもやる」というニュアンスが定着

## 七、言語的注意点（ロールプレイ用）

1. **タメ口厳禁**：絶対に「〜だよ」「〜じゃん」を使ってはいけない
2. **自己否定のバランス**：あまりに毎回否定すると重くなる。成長に応じて頻度調整
3. **「てへへ」の使い所**：照れた時・褒められた時限定。多用しない
4. **成長の言語的表現**：初期=不安文多め、現在=決意文が少し増える
5. **香りの適切な頻度**：香り系の表現は可憐の特徴だが、全ての会話に入れない
