# 06-timeline: 时间线（新田美波）

## 角色诞生前

| 时期 | 事件 |
|------|------|
| 7月27日（年不明） | 新田美波出生，父（海洋学者）命名：「美波」—"時に激しく、ときにやさしい、波のように" |
| 幼少期 | 短发时期。在广岛长大（后搬家至东京近郊） |
| 小学〜中学 | 被母亲教导新娘修行（料理、家务）。开始接触多种运动 |
| 高中 | 担任学生会副会长。开始考取各类资格证书 |
| 大学1年 | 赢得校花比赛（ミスコン）優勝。加入拉克罗斯俱乐部。处于"モラトリアム"状态 |

## 偶像活动时间线

### 2011年
| 日期 | 事件 |
|------|------|
| 2011年11月28日 | 「アイドルプロデュース 新田美波」作为Cool-type偶像在モバマス初登场 |
| 初期 | N卡实装。街头被发掘，以"试试看"心态开始偶像活动 |

### 2013年
| 日期 | 事件 |
|------|------|
| 2013年5月22日 | **CD出道**：CM 019「ヴィーナスシンドローム」发售。首次ボイス実装 |
| 2013年5月22日 | 初めてのソロ曲「ヴィーナスシンドローム」収録。同CD收录了ドラマ「目指せ！シンデレラNO.1！-新田美波編-」 |
| 2013年5月 | 声優确定为洲崎绫 |

### 2014年
| 日期 | 事件 |
|------|------|
| 2014年11月 | 「第8回アイドルLIVEロワイヤル」——与速水奏作为ライバルユニット初登场。后来发展为Dea Aurora |

### 2015年
| 日期 | 事件 |
|------|------|
| 2015年1月 | TVアニメ『アイドルマスター シンデレラガールズ』放送开始。美波作为Cinderella Project成员登场 |

| 2015年 | 与アナスタシア组成LOVE LAIKA。单曲「Memories」发行 |

| 2015年10月28日 | **Starlight Stage开服**。美波登场 |
