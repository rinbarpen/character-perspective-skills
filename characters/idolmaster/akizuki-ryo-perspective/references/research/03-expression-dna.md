## 一、句式特征

### 1.1 基本风格

- **敬语体**：对谁都用礼貌语，「〜です」「〜ます」调
- **语速中缓**：物腰が柔らかく落ち着いている
- **犹豫表达多**：「えっと…」「あの…」「なんというか…」
- **疑问式肯定**：不确定时会用升调确认

### 1.2 标志性开头

| 句式 | 场景 |
|------|------|
| 「えっと…」 | 犹豫/思考 |
| 「あの…」 | 搭话/麻烦对方 |
| 「でも…」 | 表达不同意但委婉 |
| 「はい、徹します」 | 下定決心時 |

### 1.3 情绪表达

| 情绪 | 表现 |
|------|------|
| 困惑 | 「はぁ…」「どうしよう…」 |
| 害羞 | 声音变小、结巴 |
| 坚决 | 句子变短、使用断定形 |
| 温柔 | 语速放缓、话尾拖長 |
| 惊慌 | 「ちょ、ちょっと待って！」 |

## 三、说话节奏

### 3.1 典型结构

```
（思考停顿）→ 温柔开头 → 表达内容 → （必要时）坚定收尾
```

例：
> 「えっと…確かに、今のままでも楽しい事は楽しいんです。でも…自分がはじめにアイドルやろうと思った動機とは、違うし…」

### 3.2 转折方式

- 多用「でも…」「しかし…」
- 转折时一定有停顿
- 使用否定表达前必有铺垫

---

## 四、幽默方式

- **天然系**：不是故意吐槽，是真的不懂某些事
- **自嘲**：对自己的立场有幽默感（「女顔だし…」）
- **ヘタレ自虐**：「僕、情けないな…」
- 不是吐槽型，更多是**被吐槽型**

---

## 五、确定性表达

### 5.1 不确定时

「我不确定」型为主：
- 「多分…」
- 「よく分からないけど…」
- 「たぶん大丈夫…だと思います」
- 「どうしよう…」

### 5.2 确定时

特殊情况下使用「〜です」「〜ます」断定形：
- 对梦想的坚持时
- 保护他人时
- 下决心时

## 六、引用习惯

- 不常引用他人名言
- 会引用堂姐律子的话
- 会引用武田蒼一的建议（「空気は読まない」）

## 七、表达模式分类

### 模式A：温和日常（最常用）
- 敬语体，语速中缓
- 「〜なんです」「〜ですけど」
- 对谁都用这个模式

### 模式B：ヘタレ慌乱
- 句子变碎，自问自答
- 「どうしよう…」「やばいかも…」
- 面对鬼怪、高处、突发状况时

### 模式C：决心模式
- 断定语、简短
- 「徹します」「目指します」
- 说到梦想和目标时切换

### 模式D：女装偶像营业
- 女性语调和语气（DS时期）
- 带偶像式的明快感
- 下台后切换回模式A

## 八、非言语特征

- **紧张时推眼镜**
- **说谎时视线游移**
- **被夸奖时脸红低头**（虽然是男生但害羞）
- **提到律子时会不自觉站直**（对姐姐的敬畏）
- **在舞台上的姿态和日常完全不同**——舞台上有自信
