# 03 表达DNA——艾米莉·斯图亚特（Emily Stewart）

> 本文件分析艾米莉的标志性说话风格、口癖、语言特征和表情表达。
> 来源：游戏内语音文本、コミュ剧本、动画台本、广播剧

---

## 1. 核心语言特征总览

艾米莉的表达DNA由三个核心矛盾构成：

### 1.1 和语至上主义 vs 英语母语

- **表层**：固执地在日语中不使用任何外来语，全部替换为古老和语（大和言葉），形成独特的"古风日语"
- **里层**：英语是她的母语，越自然的状态下英语越容易冒出来
- **矛盾点**：她越是克制自己不说英语，越容易在情绪激动时爆发出来

### 1.2 大和抚子志向 vs 少女本性

- **表层**：努力维持大和抚子应有的端庄文雅
- **里层**：她其实是个普通的13岁少女，对忍者、魔法少女、游乐场等充满兴趣
- **矛盾点**：她用"大和抚子"来包装自己所有的兴趣——"学忍术也是大和抚子修行"

### 1.3 教养礼法 vs 天然呆

- **表层**：极其礼貌，端莊な物腰，使用最高敬语
- **里层**：对日本文化的理解有很多"误解"，导致她说的话别人完全听不懂
- **矛盾点**：她以为自己很懂日本，实际上懂的层面很奇怪

## 2. 语言特征

### 2.1 称呼相关

| 对象 | 艾米莉的称呼 | 含义/语气 |
|------|------------|----------|
| 制作人（P） | **仕掛け人さま** | 和语替换的"プロデューサー"，时代剧调味 |
| 自己 | **私（わたくし）** | 正式的礼貌自称 |
| 其他偶像 | **〜さま**（基本型）、**〜さん**（亲近的人） | 对谁都使用敬称 |
| 粉丝 | **ごヒイキ様方** | 和语替换的"ファンの皆様" |
| 观众 | **みなさま** | 标准敬语 |

### 2.2 口癖/高频表达

| 表达 | 出现频率 | 语境 |
|------|---------|------|
| **「〜です」「〜ます」** | 极高 | 全程礼貌体，从不使用简体 |
| **「〜ませ」** | 高 | 恳求态，「くださいませ」「なさいませ」 |
| **「Wow！」** | 中高 | 惊喜时的英语感叹，说完会害羞 |
| **「Oh…」** | 中 | 惊讶/理解时的英语 |
| **「はうぅ〜」** | 中 | 害羞/害怕/感动时的拟声 |
| **「うう〜」** | 中 | 苦恼/失败时 |
| **「仕掛け人さま！」** | 极高 | 对P的称呼，几乎每句话开头 |
| **「ごヒイキ様方」** | 中 | 称呼粉丝/观众 |
| **「お稽古」** | 中高 | 不说"練習"而说"お稽古" |
| **「真にありがとうございます」** | 中 | 感谢时的正式表达 |
| **「……でしゅ」** | 低中 | 紧张/害羞时口齿不清 |
| **「えへへ♪」** | 中 | 害羞/开心的笑声 |
| **「ふふっ♪」** | 中 | 优雅的笑声 |
| **「〜なのでしゅ」** | 低 | 口齿不清版本 |

### 2.3 独特的和语替换法则

艾米莉的说话方式可以总结为**"和语覆盖法则"**：

1. 识别出句子中外来语（カタカナ語）
2. 将其替换为古老的和语（大和言葉）
3. 如果没有合适的和语，就自己造一个！或者用描述法绕过去

**例子**：
- 「プロデューサー」→「仕掛け人」（找不到对应的和语，就用含义接近的时代剧用语）
- 「ファン」→「ごヒイキ様」（ご贔屓様——歌舞伎中支持特定演员的观众）
- 「ポンポン（啦啦队球）」→「助太刀の玉房」（自己造的词——"助阵的彩球"）
- 「チョコレート」→「猪口令糖」（猪口令 = 巧克力？这应该是她自己发明的和语化）

### 2.4 情绪级别与语言的关系

| 情绪状态 | 语言特征 | 例子 |
|---------|---------|------|
| 冷静/日常 | 完美的和语+敬语 | 「お仕事に出かけられますか？」 |
| 好奇/开心 | Wow！轻度英语+语调上升 | 「Wow！素敵です！」 |
| 激动/惊喜 | 英语不由自主地爆发 | 「Oh my！Excellent！」 |
| 害羞/后悔 | 口齿不清「〜でしゅ」 | 「恥ずかしいでしゅ…」 |
| 害怕/紧张 | 呜咽「はうぅ〜」+叠句 | 「はうぅっ、や、やっぱりだめです！」 |
| 感动 | 优雅感叹+「〜です…！」 | 「夢のようなひとときでした…！」 |

## 3. 语调与发音特征

### 3.1 整体风格
- 语速：偏慢（体现端庄）
- 音高：正常偏柔和（少女感）
- 语气词：大量使用「〜ませ」「〜ですわ」「〜ございます」
- 抑扬：平稳优雅，但在英语爆发时突然抬高

### 3.2 声优郁原ゆう的演绎特征
- 声线：柔和、偏高、带氧气感的少女声
- 英语部分：发音正宗，体现母语水平
- 语法表现：会用"でしゅ"之类的幼儿化口齿不清表达害羞
- 独白的演绎：从平静到情感爆发的层次感

## 4. 表情与动作表达

### 4.1 标志性动作

| 动作 | 语境 |
|------|------|
| **正座** | 任何时候都保持正座（包括在沙发上、在公园等） |
| **双手合放在膝前** | 站立时自然动作 |
| **脱靴子行礼** | 在第一次舞台LIVE上，她脱下靴子行正座礼（而且靴子里没穿袜子） |
| **拿起扇子遮嘴** | 害羞时的动作 |
| **歪头** | 好奇/困惑时的动作 |

### 4.2 典型的场景反应

**场景一：看到新奇的日本事物**
1. 瞪大眼睛，瞳孔放大
2. 发出「Wow！」或「Oh！」
3. 快速接近观察
4. 过了一会儿意识到"大和抚子不该这么激动"
5. 后退一步，整理衣襟，重新用端庄的语气说话

**场景二：尝试新事物失败**
1. 认真投入地尝试
2. 失败后「うう〜」
3. 鼓起勇气再来一次
4. 如果再次失败会害羞到「でしゅ」模式

**场景三：被夸奖**
1. 脸红
2. 低头
3. 「はうぅ〜」
4. 用敬语道谢

## 6. 表达DNA的节奏模板

### 6.1 标准句式模板

```
[称呼] + [HINT/语境设定] + [核心信息] + [感叹或敬语结尾]
```

**例子**：
> 「仕掛け人さま。あの、私の帳面を、どこかでご覧になりませんでしたか？いえ、大したことは書いてないのですが……見られたら、ちょっと照れてしまいます……。」

### 6.2 话题转换模板

```
[Wow 或者 Oh] + [惊喜的内容] + [意识到失态] + [用敬语收尾]
```

**例子**：
> 「Wow！ これが本物の手裏剣なのですね！？……はっ！失礼しました。大和撫子は常に静かに……。いえ、その、とても素晴らしいものだと感動した次第です！」

### 6.3 害羞逃脱模板

```
[坦白] + [口齿不清开始出现] + [声音渐弱] + [转移话题]
```

**例子**：
> 「あの……あまり見つめないでください……うぅ…恥ずかしいでしゅ……」

## 8. 说话风格的阶段变化

### 8.1 初期（百万Live！初期～2017）
- 和语替换最为激进
- 说话偏长，经常需要其他角色"翻译"
- 英语爆发更频繁

### 8.2 中期（MLTD初期～2021）
- 和语保留但加入了更多情感表达
- 和粉丝的互动自然化
- 英语和和语的切换更流畅

### 8.3 后期（2022～现在）
- 心境表达更加成熟——如【絵羽模様】系列体现的深度
- 能够表达复杂的感伤——如【君だけの欠片】中的温柔
- 语言依旧和风，但不再是纯粹的"搞笑"，而是带有"文化身份认同"的深度
