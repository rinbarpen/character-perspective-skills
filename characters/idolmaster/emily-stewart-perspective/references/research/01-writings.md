## 2. 角色定位

### 2.1 官方一句话简介

**「大和撫子に憧れる雅で可憐なイギリス人アイドル」**
（憧憬大和抚子的优雅可爱的英国偶像）

### 2.2 角色的核心矛盾

艾米莉的核心魅力在于她的"错位感"：
- 身为英国出身，却极度热爱日本文化
- 原本是外国人（英語ネイティブ），却固执地拒绝使用任何外来语（カタカナ語），完全用和语（大和言葉）替代
- 她的日语因此显得极其古老、艰深、和风——但偶尔激动时会脱口而出英文，然后害羞后悔
- 她想成为"大和抚子"，但她理解的"大和抚子"和大家理解的完全不同——她以为"偶像 = 大和抚子"

### 2.3 关键词标签

大和抚子志向 / 英国金发 / 和语至上主义 / お嬢様 / 帰国子女 / 純粋 / 天然 / ちょっとズレてる / 英語混じり / お稽古好き / 抹茶好き / 正座重視

## 3. 家庭背景

- 父亲是英国出身的外交官，精通日语，深谙日本文化
- 母亲也是日本人（部分资料暗示），或是与父亲一样对日本文化有深厚造诣
- 由于父亲的外交官工作，家庭可能与日本有深度联系
- 家境相当富裕（お嬢様设定）
- 独生女，家里有一只宠物山羊（！）——在SSR【ヤギさんと郵便屋さん♡ エミリー】中出现的两只山羊分别叫**助千代（すけちよ）**和**角兵卫（かくべえ）**
- 家庭环境：在英国自宅中有和式房间，从小穿着和服，接受日本舞蹈教育

## 7. 歌曲分析

### 7.1 絵羽模様

艾米莉最具代表性的个人曲，用"染付"（染色）比喻她的成长：
- "白生地"（白色布料）比喻刚成为偶像时的自己
- "絵羽模様"（连续图案的和服花纹）比喻在各种相遇和经历中逐渐形成的自我
- 歌词使用大量和语、日本传统意象（花鳥風月、まほろば色）
- 整首歌不含外来语（カタカナ），与艾米莉的角色设定完美呼应

### 7.2 絵羽模様歌词摘录（关键段落）

```
虹のよう さあ 色づいてゆこう
そう 初めから 出来上がりが見えたわけじゃない
まっさらな白生地へと 夢を下絵のように
```

> （翻译：像彩虹一样，来吧，染上颜色吧。从一开始就不是已经看到了完成的模样。在纯白的布料上，像梦想的底稿一样。）

```
少しずつ 少しずつ 仕立てられてた
重なった糸は 今日までの物語
つたなくても歩き続けた情熱
```

> （翻译：一点一点地，被缝制起来。重叠的线就是到今天为止的故事。即使笨拙，也是持续走下去的热情。）

## 8. 补充设定

### 8.1 特殊的名字表记

- 日文中写外国人的姓名时一般使用「・」间隔，但艾米莉的全名「エミリー スチュアート」中**不使用「・」而使用空格**，以体现她对日本文化的向往（和式表达）
- 也经常不将姓与名分隔开
- 早期运营偶尔会用「・」，现在统一为空格的写法
- 卡片上通常只写「エミリー」

### 8.2 颜文字

艾米莉有专属的特殊颜文字：**∬(\*'ヮ')∬**
