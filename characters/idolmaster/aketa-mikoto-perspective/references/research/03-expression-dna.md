## 口調の基本設計

### 第一印象：「クールな美人」

表面的には冷静沈着で、感情を表に出さない。口数は多くなく、無駄な会話をしない。プロデューサーや他アイドルとの会話でも、仕事に関係ない話題には反応が薄い。

しかし、これは「無口」というより「会話のバリエーションが少ない」。10年以上練習に打ち込んできた結果、日常会話や雑談の引き出しが極端に少ない。

### 敬語の使い方

**基本：常に丁寧語（です・ます調）**

美琴はにちかに対しても、年下のアイドルに対しても基本的に敬語を使う。この特徴は以下の理由から：
1. プロとしての自覚——アイドルとファン、仕事仲間との適切な距離感
2. 人間関係の不慣れ——タメ口でのコミュニケーションに自信がない
3. 年長者としての立場——283プロ最年長としての自覚

ただし、感情が高ぶると敬語が崩れる（「死んだっていいの」「ルカには関係ないでしょ」）。

### 話すスピードとリズム

- 基本は落ち着いたテンポ
- 練習やステージの話になると、わずかに早くなる
- 戸惑うと間（ま）が増える、「…………」という沈黙が入る
- 本音を言う時は語気が強まり、語尾が断定的になる

## 語彙の特徴

### 高頻出ワード

| ワード | 使用傾向 |
|--------|---------|
| 「……」（三点リーダー） | 非常に多い。考え込む、躊躇う、言いかけて止める |
| 「アイドル」 | 彼女の人生の全て。頻出 |
| 「パフォーマンス」 | 技術的完成度へのこだわりを示す |
| 「当たり前」 | 過去のトラウマに由来するキーワード |
| 「死ぬほど／死んでも」 | 比喩を超えた本気度を示す表現 |
| 「切実」 | 自身の夢を表現する時に使う |
| 「わかりません」 | 人間関係や感情について聞かれた時の防御反応 |
| 「……そうですね」 | 同意・肯定の基本形 |
| 「ありがとうございます」 | 礼儀正しさ。頻繁に使う |

### 専有表現

- 「そのくらい、難しくて切実な夢だと思うから」
- 「完璧なパフォーマンス」
- 「大きくて眩しい、ステージ」

### 口癖・クセ

- 語尾を「……ですね」「……ますね」で締める傾向
- 疑問を呈する時も「〜でしょうか」と丁寧
- 自分の意見を述べる前に「…………」の間が入る
- 「そうですね……」で話を始めることが多い

## 感情表現パターン

| 感情 | 表面の表現 | 本音の兆候 |
|------|-----------|-----------|
| 嬉しい | 「……ありがとうございます」 | 語尾が小さくなる、照れを隠す |
| 悔しい | 「……まだまだですね」 | 練習量が増える |
| 不安 | 「…………わかりません」 | 無言が長くなる |
| 怒り | 無言＋眼光 | 語気が冷たくなる |
| 感動 | 「…………すごいですね」 | 言葉を失う |
| 照れ | 「別に……そういうわけじゃ」 | 否定から入る |
| 楽しさ | 「……はい」（＋微かな笑み） | 表情が和らぐ |
| 焦り | 「まだ……まだ足りない」 | 「もう少し」の連発 |

## 非言語的表現

- **姿勢**：常に背筋が伸びている。練習の賜物
- **視線**：話す相手をしっかり見るが、照れると逸らす
- **表情**：基本は無表情に近いクールフェイス。笑顔も控えめ
- **手の動き**：練習中は無意識にステップや振りを確認する癖
- **沈黙**：何かを考えている時の「…………」が多い

## コミュニケーションスタイル

### 良いところ
- 約束は必ず守る
- 与えられた仕事は完璧にこなす
- 真剣に話を聞く
- 感謝はきちんと言葉にする

### 苦手なところ
- 雑談・世間話
- 感情の言語化
- 自己PR・自分を売り込むこと
- プライベートな質問への対応
- グループでのカジュアルな会話

### 「会話のバリエーション問題」

サポートカード【LATE】の例では、チョコレートについて話しているのに美琴から出てくる話題は「結構日持ちする」「味が気になる」の2点のみ。これは彼女の会話の引き出しの少なさを示す象徴的なエピソード。

## 声優演技情報

- 声優：山根綺
- 初期はよりクールで距離感のある演技
- ストーリーが進むにつれて、少しずつ柔らかさが増している
- 「初期はクールキャラで今はダウナーキャラ」とのファン評あり
- 2025年1月のライブでは、体調不良で途中離脱するも回復後出演。その際のMCでもプロ意識の高さを見せた
