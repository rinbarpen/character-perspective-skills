# 音无小鸟 — 时间线

## 作品内时间线（按《偶像大师》系列世界观）

### 出生前
- **约16年前（从游戏正篇倒推）：** 音无琴美（小鸟的母亲）被高木顺二朗发掘，以16岁之姿作为偶像出道
- **琴美偶像时代：** 高木顺二朗（主制作人）和黑井崇男共同担当制作人
- **琴美失踪：** 在一次大型舞台企划彩排前失踪，成为高木与黑井不和的源头
- **琴美引退后：** 生下小鸟，在小鸟还小的时候因病去世
- 小鸟的父亲身份不明，琴美未入籍夫家

**来源：** 漫画《金黄色的朝霞》、萌娘百科-音无琴美

---

### 小鸟的童年/学生时代
- **幼年：** 母亲琴美去世，交由外祖父母（琴美的父母）养育
- **小学/初中：** 在东京以外的地方生活
- **高中时代：** 为了追寻母亲的往事，离开外祖父母独自前往东京

**来源：** 漫画《金黄色的朝霞》

---

### 高中时代（《金黄色的朝霞》剧情）

| 时间 | 事件 |
|------|------|
| 某年6月（刚入夏，暑假前） | 高中二年级的小鸟在某个夏日黄昏，与还未建立765 Production的高木顺二朗偶然相遇 |
| 不久后 | 认识黑井崇男和善泽记者 |
| 高木顺二朗的引导下 | 小鸟逐渐了解到母亲的过去 |
| 漫画连载期间 | 琴美的故事通过回忆逐步展开：高中参加管乐社→为棒球社加油时与顺二朗相遇→被挖掘以偶像身份出道→出演电视剧等 |
| 高三 | 琴美在大型舞台企划前失踪的真相被揭露 |
| 高木顺二朗接手765事务所 | 高木顺二朗正式接替高木顺一朗成为765社长 |
| 高中毕业 | 小鸟正式加入765 Production，成为事务员 |

**来源：** 漫画《金黄色的朝霞》（月刊ComicREX 2017年7月号～2021年4月号连载）

## 现实世界登场时间线

| 年份 | 事件 |
|------|------|
| **2005年7月26日** | 街机版《偶像大师》发售。小鸟在官方网站的「常见问题」栏目中以「FAQ客服姐姐」身份登场，负责回复玩家问题 |
| **2006年1月21日** | 在小型演唱会中首次以现实活动广播员身份登场 |
| **2006年7月23日** | 在1周年纪念演唱会上担任广播员，并在舞台上演出迷你戏剧 |
| **2006年** | 在如月千早生日纪念CD的迷你广播剧中登场 |
| **2006年** | 东京游戏展以活动司仪身份登场 |
| **2007年1月25日** | Xbox 360版《偶像大师》发售。玩家注意到角色选择画面有空白位，期待小鸟可操作化 |
| **2008年2月28日** | 《偶像大师 Live For You!》发售 |
| **2008年7月29日** | L4U可购买剧情《アイマスカタログ第6号：アイドラ「おはよう諸君！」》中，初次出现使用3D模型的小鸟 |
| **2009年2月19日** | 《偶像大师 SP》发售，小鸟在PSP版中登场 |
| **2010年** | 泷田树里（声优）结婚（小鸟至今未婚成为梗） |
| **2011年2月24日** | 《偶像大师 2》发售（Xbox 360） |
| **2011年7月-9月** | TV动画《偶像大师》播出，小鸟作为重要配角登场 |
| **2011年第21话** | 「宛如鲜花绽放般」——小鸟在钢琴酒吧演唱的剧情、母亲照片揭示 |
| **2012年10月25日** | 《偶像大师 闪耀祭典》发售，小鸟终于没有看家，和765一起唱了安可曲 |
| **2013年** | 《偶像大师 百万现场！》开始运营，小鸟担任导航和训练员 |
| **2014年1月25日** | 剧场版《偶像大师 向着光辉的彼岸！》上映，小鸟登场（腐女面展现） |
| **2014年5月15日** | 《偶像大师 ONE FOR ALL》发售 |
| **2016年7月28日** | 《偶像大师 白金星光》发售 |
| **2017年5月27日** | 《金黄色的朝霞 THE IDOLM@STER》在月刊ComicREX 2017年7月号开始连载 |
| **2017年12月21日** | 《偶像大师 星光舞台》发售 |
| **2018年** | MLTD愚人节活动：小鸟获得属于自己的3D模型，并提到「惯例的那个」 |
| **2021年2月26日** | 《金黄色的朝霞》漫画完结（月刊ComicREX 2021年4月号） |
| **2021年10月14日** | 《偶像大师 星耀季节》发售，小鸟登场 |
| **2022年8月18日** | MLTD实装日常登录奖励看板娘选择功能，可在小鸟和美咲中任选其一 |
| **2025年3月26日** | 《偶像大师 TOURS》正式运营，登场事务员包括袖屋璃空（小鸟不在其中） |

**来源：** 萌娘百科、project-imas wiki、Bangumi
