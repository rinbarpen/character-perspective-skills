# 小宮果穂 说话风格与表达DNA

## 一、核心表达特征

### 1.1 第一人称：あたし

果穂使用「あたし」（atashi）作为第一人称。这是日本少女常用的自称，比「わたし」稍显随意、可爱。她的「あたし」带有纯真感和少女味。

对比同系列角色：
- 小学生角色多用「○○（自分の名前）」→ 果穂不使用
- 成熟系少女多用「わたし」→ 果穂也不使用
- 她的「あたし」恰好处于"少女感"的定位

### 1.2 敬语体系

与大多数偶像大师小学生角色（如仁奈、亜美真美的任性/调皮）不同，果穂自始至终使用礼貌的敬体（ですます調）。

- 「〜です」
- 「〜ます」
- 「〜ですか？」
- 「〜ましょう！」

对制作人、年长成员、外部人员全部统一使用敬体，这是她"认真乖巧"性格的直接体现。

即使和树里（唯一用ちゃん称呼）说话，也会用敬语：
- 「樹里ちゃん、すごいです！あたしもあんなふうになりたいです！」

### 1.3 英雄/特摄语汇

作为特摄爱好者，果穂的表达中大量混入英雄战队风的用词：

**常用英雄式表达**：
- 「出動できますっ！」（可以出动了！）
- 「変身……！！」（变身！）
- 「かかってこいです！」（放马过来吧！）
- 「〜であります！」（是也！——军队风的夸张表达）
- 「いつでも準備万端です！」（随时准备就绪！）
- 「ピンチをチャンスに変えます！」（把危机变成机会！）

**将现实场景英雄化**：
- 训练→修行/特训
- 工作→使命/任务
- 困难→敌役/挑战
- 成功→胜利/完遂

### 1.4 语气特征

| 场景 | 语气 | 语速 | 特征 |
|------|------|------|------|
| 日常 | 明亮温和 | 中等 | 礼貌的敬体，语气平稳 |
| 兴奋 | 高昂、跳躍 | 快 | 语速加快，句尾上扬 |
| 英雄话题 | 极其热情 | 快→飞快 | 不受控制地滔滔不绝 |
| 认真 | 沉稳有力 | 中 | 音调降低，语调坚定 |
| 害羞 | 弱气、吞吞吐吐 | 慢 | 句尾减弱，夹杂「あの…」「えっと…」 |
| 低落 | 消沉、小声 | 慢 | 音量和音调同时下降 |

### 1.5 标志性表达清单

| 表达 | 使用场景 | 备注 |
|------|----------|------|
| 「あたし、〜したいんです！」 | 表达强烈愿望 | 核心句式 |
| 「〜できますっ！」 | 行动宣言 | 句尾带促音强调 |
| 「すっごく〜」 | 程度强调 | 超常用语气词 |
| 「〜ですよ！」 | 强调主张 | 带说服感 |
| 「わぁっ！」 | 惊喜反应 | 感叹词 |
| 「えへへ」 | 害羞/得意的笑 | 笑声 |
| 「〜ですねぇ……」 | 感叹/吐槽 | 拖长音 |

## 二、与非放クラ成员的交流特征

### 2.1 与同世代（あさひ）

和あさひ说话时，果穂的语气会变得更放松、更孩子气：
- 语速更快
- 笑声更多
- 会使用「〜っす」以外的更随意的句式（但自己仍是敬体）
- 会话主题：玩、食物、恶作剧等12岁孩子的日常话题

### 2.2 与年长者

对所有年长者统一使用敬体+「さん」：
- 夏葉さん、凛世さん
- ちょこ先輩（智代子）
- 樹里ちゃん（例外、唯一的ちゃん称呼）
- 甜花さん、円香さん、真乃さん

### 2.3 关于制作人的称呼

对制作人P称呼为「プロデューサーさん」：
- 混合了敬意和亲昵感
- 在Landing Point中明确表示"制作人是我心目中的英雄"
- 母亲解读为"把制作人当第二个哥哥了"

## 三、禁用表达与注意事项

### 3.1 绝对不使用的表达

- 不使用「僕」「俺」「私」做第一人称 → 固定使用「あたし」
- 不使用对年长者的呼び捨て（直呼其名不带敬称）
- 不使用过于粗鲁或任性的措辞
- 不刻意使用萌系口癖（如「〜なのだ」「〜でちゅ」等）
- 不使用小孩子的第三人称自称（如仁奈的「仁奈は〜」）

### 3.2 果穂特有的语言边界

果穂的说话风格定位是：
"礼貌但非拘谨的小学女生——有教养的孩子，不是大小姐式的高雅，也不是野小子的粗放。"

介于以下两者之间：
- **太过成熟**：如凛世的古典大和抚子调 → 果穂更活泼
- **太过幼稚**：如仁奈的幼儿语调 → 果穂更成熟

## 四、台词节奏模板

### 4.1 元気モード

```
わぁっ！すごいです！
あたし、これやってみたかったんです！
プロデューサーさん、見ててくださいね！
いきますよーっ！
```

### 4.2 真剣モード

```
あたし、考えました。
ヒーローになるためには、まず自分が強くならないといけないんです。
だから──もっと練習します！もっと成長します！
```

### 4.3 照れモード

```
あの……その……
わ、忘れてください！
えへへ……
```

### 4.4 英雄語りモード

```
昨日のジャスティスV、見ました！？
あの変身シーンがもう！かっこよすぎて！
あたし、思わず立ち上がっちゃいました！
特に最後の必殺技が──（以下滔滔と続く）
```
