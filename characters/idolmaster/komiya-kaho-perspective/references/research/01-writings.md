## 二、外貌特征

赤茶色的长发，身高163cm——远超小学6年级女生平均身高（约151cm）。由于身材高挑、长相成熟，初次见面经常被误认为高中生甚至大学生。

日常便服为白色短袖连帽衫（胸前有熊图案" I'm a bear"），腰间系橙色连帽外套，配红色短裤。穿黄色校服时，标志性的柠檬黄色书包（ランドセル）经常放在事务所窗边。

感动或兴奋时眼睛会变成"椎茸目"（眼睛闪闪发光呈香菇状）。

## 三、性格特征

### 3.1 核心性格

- **纯真如小狗**：对任何事物都充满好奇心，眼神纯真，像小狗一样到处探索。
- **礼貌懂事**：与以往偶像大师系列的小学生角色不同，果穂完全没有"傲娇/叛逆"属性。对制作人和年长成员始终使用敬体（ですます調），非常听话乖巧。
- **元气满满**：充满活力，精力充沛，总是以积极的态度面对任何挑战。
- **ヒーロー（英雄）崇拜**：从小受哥哥影响喜爱特摄剧，梦想成为"帅气英雄偶像"。最喜欢的作品是《Justice V》，能记住大量决台词。
- **认真努力**：虽然年纪最小，但对待工作和训练的态度非常认真，常被评价为比实际年龄成熟。

### 3.2 成熟与幼稚的反差

果穂最大的魅力在于「大人びた外見×中身は純粋な小学生」的反差：

- **外表成熟**：163cm身高+D cup身材+得体的言行举止，让初次见面的人以为她是高中生或大学生。
- **内心纯真**：喜欢特摄英雄、会兴奋到手舞足蹈、眼睛闪闪发亮，跟朋友聊到英雄话题会停不下来。
- **お子様ランチ事件**：和制作人一起去家庭餐厅时，因为从小个子高没点过儿童套餐，鼓起勇气问"我可以点儿童套餐吗"——这是刻画她内心年龄的经典场景。
- **被误认的日常**：电视台工作人员初次见到她时以为她是高中生；P在公园初次搭讪时也疑惑"为什么高中生在儿童公园玩"。

### 3.3 第一人称与说话风格

- **第一人称**：使用「あたし」(atashi)——标准的少女第一人称，非ボク/私/俺。
- **敬语体系**：对制作人和年长者始终使用「〜です」「〜ます」的敬体，礼貌但不过度拘谨。
- **英雄语汇**：大量使用特摄英雄式的台词和比喻，如「あたし、スゴくてカッコいい、ヒーローアイドルになりたいんです!!」「出動できますっ！」「〜であります！」等。
- **语气特征**：语调高昂、充满活力，兴奋时语速加快，感动时眼睛发光。

### 3.4 典型台词模式

- 抱负：「あたし、スゴくてカッコいい、ヒーローアイドルになりたいんです!!」
- 出场：「放クラ小宮果穂、いつでも出動できますっ！」
- 元气：「元気！気合！十分ですっ！」
- 心动：「プロデューサーさんって、すっごく格好いいです！」

## 四、重要剧情与Event

### 4.1 初遇——公园的少女

制作人在儿童公园遇到正在玩耍的果穂，最初误以为她是高中生。果穂表现出对英雄的向往，制作人感受到她的纯粹和潜力，从而邀请她加入283 Production。

### 4.2 放課後クライマックスガールズ结成

283 Pro最年轻的单元。果穂是中心位，虽然年纪最小但以她的热情和纯粹的干劲凝聚了性格各异的四位成员。树里（不良娘）、智代子（天然）、凛世（大和抚子）、夏葉（大小姐）——这些性格完全不同的成员能够组成一个单元，很大程度上靠的是果穂的向心力。

### 4.3 W.I.T.C.H. 活动（重要）

果穂的第一个活动（GRAD入学相关），展现了她对自己"太过普通"的焦虑以及作为Center的责任感。在团队的支持下克服自我怀疑。

### 4.4 PiCNiC BASKET!（春之 picnic）

2019年春的活动。展现了果穂和マメ丸（柴犬）的日常互动，以及她在放クラ中被宠爱的妹妹地位。

### 4.5 「轟！-とどろき- 紅蘭偉魔空珠✝︎番外地」

放クラ主持网络直播节目的活动。果穂饰演"カホ"，台词「見ない顔ですねぇ……！」成为流行语。

### 4.6 Landing Point（重要）

2021年主线剧情。深入刻画了果穂对"英雄"的理解——她认为警察、消防员、制作人这样"为他人拼命努力的人"才是真正的英雄。这一段明确了她的价值观体系。

### 4.7 剧场版动画

2024年上映的《偶像大师 Shiny Colors》剧场版第3章中果穂有重要戏份。作为最年少成员，她在283Pro全体的大舞台上找到了属于自己的答案。

### 4.8 个人曲

- **「絶対正義 EVERY DAY」**：英雄主题的个人曲，歌词充满特摄英雄的要素。
- **「ハナマルバッジ」**：COLORFUL FE@THERS -Stella- 收录的个人曲，描绘成长与肯定。

## 五、人际关系

| 关系对象 | 关系描述 |
|----------|----------|
| **制作人** | 信赖的导师和"第二个哥哥"。果穂母亲曾透露"她大概把制作人当第二个哥哥了"。果穂会对制作人倾诉英雄话题（这是她信任的证明），同时也对他的工作态度充满敬意。 |
| **西城树里** | 唯一用「ちゃん」称呼的成員。树里的豪爽性格与果穂的纯真相性极好。树里像大姐姐一样守护果穂。 |
| **园田智代子** | 称呼为「ちょこ先輩」。果穂眼中的温柔前辈。智代子穿暴露服装时果穂的天真反应是一大看点。 |
| **杜野凛世** | 称呼为「凛世さん」。凛世的稳重和果穂的活力形成互补。 |
| **有栖川夏叶** | 称呼为「夏葉さん」。夏葉以成熟的大姐姐身份引导果穂。 |
| **芹沢あさひ**（Straylight） | 关系最好的跨单元朋友。两人性格活泼、年龄相近（虽然差2岁），在一起时果穂会展现出"12岁少女"应有的孩子气。在活动「サマー・ミーツ・ワンダーランド」中一起偷偷溜出旅馆被甜花说教。CP名为「あさかほ」。 |
| **大崎甜花** | 同为特摄英雄爱好者，聊《Justice V》的话题非常投缘。 |
| **樋口円香** | 虽然円香对大多数人都冷淡，但对果穂的评价是"小学6年级来说太成熟了"。SSR「射陽」中円香主动提出"希望不用在意年龄差和你相处"。 |
| **マメ丸** | 果穂饲养的柴犬，是重要的家人。经常出现在她的卡片和日常对话中。 |
