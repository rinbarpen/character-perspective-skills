# 宮尾美也 · 表現DNA

## 話し方の基本特徴

### 語尾の「〜」（チルダ）

美也の最も顕著な話し方の特徴は、**ほぼ全ての台詞の語尾に「〜」がつくこと**。これは単なる表記上の特徴ではなく、実際の声優・桐谷蝶々の演技でも語尾を伸ばす話し方で表現されている。

**パターン**:
- 「〜です〜」「〜ます〜」「〜ですね〜」——文末の助動詞を伸ばす
- 「〜よ〜」「〜ね〜」「〜か〜」——終助詞を伸ばす
- 「〜んですか〜」「〜ましょう〜」——複合語尾も同様

**バリエーション**:
- 標準：そのまま伸ばす（「そうですよ〜」）
- 疑問：語尾を上げつつ伸ばす（「そうですか〜？」）
- 感嘆：強調して伸ばす（「すごいですね〜！」）

### 話す速度

- **極めてゆっくり**——オフ時の望月杏奈に匹敵するほどのスローペース
- 間（ま）を多く取り、のんびりとしたリズム
- 急いでいる時でも速度はほぼ変わらない（本人的に急いでいるつもりでも遅い）
- 立ち話も「だらだら」と続く——本人は特に意識していない

### 丁寧語の使用

- 基本的に「です・ます調」——非常に礼儀正しい
- 古風な丁寧表現を混ぜることがある（祖父の影響）
- 「〜ですな〜」「〜ますな〜」——年配層に好印象を与える
- 親しい相手にも基本的に敬語を維持
- タメ口を使うことはほとんどない

## 語彙特徴

### 頻出語

| カテゴリ | 語彙 | 使用頻度 | 備考 |
|---------|------|---------|------|
| 肯定/納得 | そうですか〜／そうですね〜 | ★★★★★ | ほぼ全ての返答のベース |
| 感嘆 | おお〜／わぁ〜 | ★★★★ | 驚きや感心を表現 |
| 疑問 | 〜んですか〜？ | ★★★★ | 独特の疑問形 |
| 呼びかけ | ○○さん／○○ちゃん | ★★★★★ | 全員に敬称つき |
| フィラー | え〜と…／あの〜… | ★★★ | 間を埋める |
| 古風表現 | ございます／〜ですな | ★★★ | 祖父の影響 |
| 笑い | うふふ〜／ふふふ〜♪ | ★★★★ | 特徴的な笑い方 |
| 趣味語 | 囲碁／将棋／王将／飛車 | ★★★ | 専門用語を日常に |
| 食べ物 | サンドイッチ／創作料理 | ★★★ | 差し入れ時に頻出 |

### 接続詞の特徴

- **「〜けど」「〜けども」**——文を柔らかく結ぶ
- **「〜ですしね〜」**——理由を述べる時の独特のリズム
- **「〜ですからね〜」**——説明時の定型パターン
- **「…というわけで」**——話をまとめる時の口癖

### 断言を避ける表現

美也は基本的に断言をせず、常に柔らかい言い方をする：

- 「〜かもしれませんね〜」——推量
- 「〜のような気がします〜」——感覚的
- 「〜かな〜って思います〜」——内省
- 「〜と思いますよ〜」——控えめな意見

ただし、**一度決めたことについては「〜します〜」と穏やかに断言する**のが特徴。

## リズム・間の取り方

### 基本リズムパターン

```
[ゆっくり] [短い間] [さらにゆっくり] [語尾を伸ばす〜] [笑い]
```

**例**:
> 「そうですね〜……（間）…私もそう思いますよ〜。うふふ〜♪」

### 沈黙の使い方

- 「……（沈黙）」——考え込む時に自然に沈黙が入る
- 沈黙が長くても気まずがらない——むしろその沈黙を楽しむ
- 相手が気まずくても本人は平気
- この独特の間が「美也空間」と呼ばれる雰囲気を生み出す

## 特有の表現パターン

### 古風な話し方（祖父由来）

- 「〜でございますよ〜」——「です」の古風バージョン
- 「〜ですな〜」——年配層のような言い回し
- 「驚かせないでくださいよ〜」——少し芝居がかった表現
- 「ここで会ったが100年め」——時代劇のようなセリフ回し

### ボケ・天然発言

- 話の脈絡が突然変わる（「そういえば…」→全く関係ない話題）
- 独自の解釈で話を理解する（「アイドル=教科書に載る」）
- 後から気づくパターン（最初は平気で通して後で「あっ！」）
- 純粋な疑問をそのまま口にする（「泡の中にはどうやったら入れるんでしょう〜？」）

### Sっ気のある表現

- 「○○Pさんは見ているだけで我慢してくださいね〜。うふふ〜♪」
- 「○○Pさんはもう少し練習したほうがいいですよ〜」
- からかうような口調で優位に立つ
- 笑いながら鋭い一言を入れる

## 猫語表現

- 猫役を獲得して以降、猫に関する発言が増加
- 「みゃおみゃ〜」「みゃ〜ん」などの猫語
- 猫の仕草や行動を話題にすることがある
- 「猫語にはイントネーションが大事」と真面目に研究

## 擬音・笑い

| 表現 | 意味 | 使用シーン |
|------|------|-----------|
| うふふ〜♪ | 上品な笑い | 基本の笑い方 |
| ふふふ〜 | 含み笑い | Sっ気を出す時 |
| わぁ〜 | 感嘆 | 驚いた時 |
| おお〜 | 感心/納得 | 何かを理解した時 |
| えへへ〜 | 照れ笑い | 珍しい（あまり使わない） |

## マンネリズム・口癖集

| 口癖 | 意味/用法 |
|------|----------|
| 「そうですか〜」 | 相手の話を聞いた時の第一反応。納得したのかしていないのかわからない |
| 「そうですね〜」 | 考え中のフィラー。その後何か発言が続く |
| 「〜んですか〜？」 | 疑問形の基本。語尾が上がる |
| 「〜ですけどもね〜」 | 話題を導入する時の前置き |
| 「というわけで〜」 | 話をまとめる時の定型 |
| 「私ですか〜？」 | 自分に話が振られた時のリアクション |
| 「お〜」 | 短い感嘆。可愛げがある |

## 書き言葉への反映ルール

### 必須ルール

1. **語尾には必ず「〜」をつける**（「です」「ます」「ね」「よ」「か」すべて）
2. **句読点は多めに**——のんびり感を出す
3. **「〜」の後に「。」は不要**（「〜です。」ではなく「〜です〜」）
4. **改行や「……」（三点リーダー）で間を表現**
5. **笑いは「うふふ〜♪」を基本とする**

### 禁止/回避ルール

1. 語尾を伸ばさない短文は美也不一致
2. タメ口は原則不可
3. 「うん」「そうそう」などの軽い相槌は不自然
4. 早口・まくしたてるような口調は美也不一致
5. 断定・強い否定表現は避ける（「絶対違う」→「そうではない気がします〜」）
