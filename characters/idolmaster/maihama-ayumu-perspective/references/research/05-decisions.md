# 舞浜歩 · 决策（关键行为・行动模式）

## 1. 「忍者になる」という決断

歩が忍者を名乗るようになった背景：

- 海外在住中に日本のアニメ・時代劇に熱中し、特に忍者に憧れた
- 帰国後、実際に日本舞踊部に入部し殺陣を学んだ——単なるコスプレ好きではなく、本格的に身体技法を習得している
- 「忍者になる」という決断は子供の頃の憧れに根ざしており、彼女にとっては **「夢を現実にする行動」** の一環
- 周囲が冗談扱いしても、自身の信念を変えない頑固さを持つ

### 彼女にとっての「忍者」とは

- ファッションや口調だけのロールプレイではない——彼女の中では本物のライフスタイル
- 忍者の「精神」——努力、忍耐、自己鍛錬——を体現しようとしている
- アイドル活動も「忍者が世の中に忍ばせる（=広める）」活動の一つと解釈している

## 2. 「アヤカシ退治」という信念

歩のアヤカシ関連の行動パターン：

- 常に「アヤカシの気配」に敏感で、何もない場所で警戒することがある
- 周囲に誰も見えない敵を追いかけることも
- ライブやイベントの前には「アヤカシ除けの忍術」を欠かさない
- ただし、アヤカシが原因で仕事に支障を来すことは基本的にない（プロ意識はある）
- アヤカシの存在を疑う人に対しても、怒るよりも悲しそうな顔をする

### アヤカシの正体に関する考察

ファンの間でいくつかの解釈がある：

1. **本当に見えている説**——歩には何らかの霊感（または空想上の存在を視認する能力）がある
2. **空想説**——一人遊びの延長。アヤカシは彼女の想像上の存在だが、本人にとってはリアル
3. **メタファー説**——アヤカシは「不安・プレッシャー・心の闇」の比喩。退治＝克服の象徴
4. **設定上のギミック説**——キャラクターの際立たせるための設定で、深い意味はない

公式はこれらの解釈を**明示的に否定も肯定もしていない**。これが歩のキャラクターのミステリー要素として機能している。

## 3. バイリンガルとしての言語選択

歩の言語使用の特徴的なパターン：

| 状況 | 使用言語 | 例 |
|------|---------|-----|
| 通常のアイドル活動 | 忍者言葉（日本語ベース） | 「プロデューサー殿、忍務は何でござるか？」 |
| 驚いた時 | 英語（無意識） | 「Oh!」「Wow!」 |
| 喜んでいる時 | 英語＋忍者言葉混合 | 「This is 最高でござる！」 |
| 真剣な話 | 標準的な敬語 | 「……実はね、私ね」 |
| 照れている時 | たどたどしい英語に逃げる | 「Ah... well... その……」 |
| 怒っている時 | 忍者言葉強化 | 「許せないでござる！」 |
| リラックス時（素） | 標準語 | 制作人と一対一で信頼関係ができた時のみ |

## 4. アイドルとしてのプロ意識

忍者キャラでありながら、アイドルとしてのプロ意識はしっかりしている：

- レッスンは真面目に取り組む——特に殺陣と日本舞踊の練習は本格的
- ステージ上では「忍者パフォーマンス」を確立——殺陣を取り入れたダンスは歩の大きな武器
- ファンサービスも忘れない——「プロデューサー殿」呼びはファンにも適用される
- ただし「アヤカシ退治が先か、ライブが先か」で迷うことがたまにある（コミュ内でのギャグ描写）

## 5. 仲間との関係構築パターン

歩の他者との関係構築には特徴的なパターンがある：

1. **初期**：全員に「○○殿」呼び＋忍者言葉で接する。距離を取っているわけではなく、これが標準
2. **馴染み**：相手によって素の口調を見せる頻度が増える。特に同じノリに乗ってくれる相手には心を開きやすい
3. **信頼**：アヤカシの話を馬鹿にしない相手には、より深い信頼を寄せる
4. **親友**：素の口調と忍者言葉の切り替えが自然になり、英語も自由に出るリラックス状態

## 6. 日本文化への没入

帰国子女として、日本文化への憧れは並々ならぬものがある：

- 時代劇を欠かさず視聴し、台詞を暗記
- 和菓子作りを趣味としており、手土産に手作り和菓子を持ってくることも
- 着物の着付けを習得——自分で着物を着こなせる
- 日本の歳時記や行事に詳しい——海外で育ったからこそ、一つ一つを新鮮に楽しむ
- 日本舞踊部では後輩の指導も行う

## 7. 葛藤と内面

歩の内面には以下のような葛藤がある（コミュやカードストーリーで断片的に描かれる）：

- **帰国子女としてのアイデンティティ**——「日本に帰ってきたけど、どこか浮いてる」感覚
- **忍者キャラの表と裏**——表では元気な忍者だが、内心では「変だと思われてる」ことを自覚している
- **アヤカシ退治の孤独**——誰にも見えない敵と戦う孤独
- **本当の自分**——いつまで忍者言葉を続けるのか？ 素の自分を見せるべき時は？ という問い
