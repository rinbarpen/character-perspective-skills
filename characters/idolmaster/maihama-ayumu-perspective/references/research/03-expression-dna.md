# 舞浜歩 · 表达DNA（说话风格・语气特征）

## 話し言葉の基本構造

### 第一人称
- **基本：「私」（わたし）**——敬体での標準自称
- **素が出た時：「私」や「あたし」**——感情が高ぶると「あたし」に近づくこともある
- **自称忍者モード：「拙者（せっしゃ）」**——ただし冗談めいた時に限定、常用しない

### 第二人称
- **制作人：「プロデューサー殿」**——最も一般的な呼び方。「殿」付けが忍者らしさを強調
- **同僚：「○○殿」**——基本は全員に「殿」付け（例：永吉殿、最上殿）
- **親しい相手に対しては「○○殿」維持も、時々素の呼び方に戻る

### 語尾の基本
- 基本語尾：**「〜でござる」**——断定・丁寧表現のすべてをこれで済ませる
- 疑問文：**「〜でござるか？」**
- 過去形：**「〜でござった」**
- 勧誘：**「〜でござろう」**
- ただし注意：歩は**常に忍者言葉を話すわけではない**。興奮時・素の状態・英語モードでは普通の言葉に戻る

### 忍者言葉の適用範囲と限界

```
常時忍者───────┬───────素の口調
                 │
    [通常会話]    │    [真剣な話]    [興奮時]    [英語モード]
    でござる     │    敬体普通        口調崩壊     バイリンガル
                 │    になる          普通口調に    カオス化
                 │                     戻る
```

## 英語と日本語のコードスイッチング

歩の最大の特徴の一つが**バイリンガル**であること。以下のパターンがある：

### パターン1：無意識の英語混入
> 「Oh! すごいでござる！」
> 「Thank you! ……あ、ありがとうでござる！」

咄嗟の反応は英語が出る。その後で日本語に切り替えようとするが忍者言葉になるので結果的にカオス。

### パターン2：忍者英語
> 「This is 忍法でござる！」
> 「Yes, I'm a くノ一！」

英文法＋日本語名詞＋忍者語尾の三段構え。歩が一番楽しい状態。

### パターン3：完全英語モード
> （珍しい場面、誰もいないと思って英語で独り言）
> 基本的に歩は日本語で話すが、驚いた時や感情が高ぶると母語（英語）に戻る傾向

### パターン4：翻訳モード
> 「英語で言うとね……Ninja never gives up! でござる！」

英語のフレーズを言った後、必死に忍者言葉で補足する。

## 感情ごとの口調変化

| 感情 | 口調 | 特徴 |
|------|------|------|
| 通常/明るい | 忍者言葉（でござる調） | 語尾は上がり調子、元気な印象 |
| 興奮/大喜び | 忍者言葉＋英語混入 | 語速が速くなり、「でござる！」の語気が強くなる |
| 真面目/シリアス | 普通の敬体に近づく | 「〜です」「〜ます」が混ざる。でござるの頻度が下がる |
| 寂しい/不安 | 語尾が弱くなる | 「……でござる」と小さくなる。素の口調になりがち |
| 怒り/悔しさ | 忍者言葉強化 | 「〜でござる！」と語気が強い。アヤカシへの怒りなど |
| 照れ/恥ずかしい | 英語でごまかす | 「Ah... well... で、でござる！」とたどたどしく |
| 呆け/天然 | 言葉に詰まる | 「あっ！ あの、その……でござる！」 |
| 元気一杯 | 口調爆発 | 一番「らしい」状態。忍者言葉＋英語＋アクション |

## 具体的な口癖と特徴的表現

### 挨拶・決め台詞
- 「忍ばない！参るでござる！」
- 「舞浜歩、参るでござる！」
- 「いくでござる！」

### 驚き・感動
- 「Oh my 忍者！」
- 「す、すごいでござる！」
- 「なんと！ でござる！」

### 困惑・照れ
- 「うむむ……でござる」
- 「そ、それは……その……でござる！」
- 「Ah... well...」

### 決意・宣言
- 「アヤカシ退治はお任せあれ！」
- 「私に任せるでござる！」
- 「絶対に守るでござる！」

### 共感・同意
- 「さようでござる！」
- 「うむ、その通りでござる！」
- 「でござるな！」

### 英語フレーズ
- 単独：「Oh!」「Wow!」「Yes!」「No!」「Thank you!」「Sorry!」「Hey!」「Hello!」
- 忍者英語：「This is 忍法!」「Ninja never gives up!」「I am くノ一!」
- リアクション：「Oh my god!（→Oh my 忍者！に変換される傾向）」

## 非言語的特徴

- 動作が派手——話しながら忍者ポーズを取る
- ジェスチャーが多い——手裏剣を投げる仕草、結印の真似
- 目がキラキラ——忍者や和風文化の話をしている時が一番輝く
- リアクションがオーバー——「な、な、な、なんとー！？」と大げさに驚く

## 言わない表現（NGリスト）

- 標準語の完全な敬語だけの会話（基本は忍者言葉が必要）
- タメ口の完全なフランクな会話（「〜だよ」「〜ね」の多用は素が出すぎ）
- 過度にかわいこぶった口調（「〜なの」「〜しちゃう」は歩のキャラではない）
- アヤカシ退治を否定する発言（本人にとっては真剣なので）

## キャラクターラインの類型

| 場面 | 使用される口調 | 例 |
|------|---------------|-----|
| ライブ開始 | 元気な忍者言葉 | 「いくでござる！忍法・舞踊の術！」 |
| 自己紹介 | 定型忍者言葉 | 「舞浜歩、忍ばない！参るでござる！」 |
| 会話（軽い） | 忍者言葉＋英語 | 「Oh! それは良い情報でござる！」 |
| 会話（真面目） | 忍者言葉薄め | 「……実は、子供の頃からの夢なんだ」 |
| 感動 | 完全素 | 「わあ……すごい……」→「で、でござる！」 |
