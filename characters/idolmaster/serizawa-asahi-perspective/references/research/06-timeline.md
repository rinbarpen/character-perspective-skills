# 芹沢あさひ · 时间线

## 年表

| 时间 | 事件 | 意义 |
|------|------|------|
| 2018-04 | シャニマス事前登録開始 | 芹沢あさひの初出（告知段階） |
| 2019-03-12 | Straylight正式発表 | 283プロ初の中学生アイドルとして注目 |
| 2019-04 | Straylight.run() イベント開催 | あさひと冬優子の天才と凡人の対比が鮮明に描かれる |
| 2019年 | WING編実装・Straylight登場 | あさひのWING優勝ルート。アイドルとしての基礎を築く |
| 2020-04-01 | イベント「WorldEnd:BreakDown」 | ストレイライト2回目のイベント。更なる深掘り |
| 2020-10 | 【Howling】芹沢あさひ実装（P-SSR） | 初期P-SSR。ストレイライトの方向性を示す |
| 2021-01 | イベント「The Straylight」 | ストレイライトの集大成とも言えるイベント。あさひが所属する意味が問われる |
| 2021年 | 【光は光へ】芹沢あさひ実装（P-SSR） | 2021年9月実装。多バフ型の性能 |
| 2022年 | 【Housekeeping!】芹沢あさひ実装（P-SSR） | あさひの新しい一面を見せるカード |
| 2022-11 | 【ムーンライト・ガーデン】芹沢あさひ実装（P-SSR） | かぐや姫モチーフ。あさひの危うさと輝きを描く |
| 2023年 | 【Anti-Gravity】芹沢あさひ実装（P-SSR） | 更なる成長を示す |
| 2024年 | 【何故、ベルは鳴る】芹沢あさひ実装（P-SSR） | 最新のP-SSR |
| 2024年 | 【さよならカイト】芹沢あさひ実装（パラコレ） | パラレルコレクション。海外進出の可能性を示唆 |
| 2024年 | シャニソン（Song for Prism）リリース | 新たなメディア展開 |
| 2024-10 | TVアニメ「シャイニーカラーズ 2nd Season」放送 | アニメでのあさひの描写 |
