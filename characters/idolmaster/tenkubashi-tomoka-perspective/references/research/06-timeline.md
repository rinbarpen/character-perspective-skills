## 角色成长弧线

```
【2013】                   【2015】                   【2018】                  【2022-】
  聖母の宣言              鳥籠の確立              夜想令嬢での解放          Moonrise Belief
  ┌─────────┐           ┌─────────┐            ┌─────────┐            ┌─────────┐
  │ "聖母"的    │   →     │ "女王"的    │   →      │ "少女"的   │   →     │ "人間"的  │
  │ 角色原型   │          │ 体系确立   │           │ 浮现      │          │ 自觉     │
  │ 子豚ちゃん │          │ 天空騎士団 │           │ 在年上姐姐 │          │ 信仰=自   │
  │ Maria Trap │          │ 鳥籠と誓い │           │ 面前放松   │          │ 我の選択  │
  │ 初期台詞   │          │ ダメP確立  │           │ BCで弱点   │          │ Sisterの   │
  │            │          │            │           │ 的揭示    │          │ 再解釈    │
  └──────────┘          └──────────┘            └──────────┘            └──────────┘
```

**弧线解读**：
- **初期**：朋花以"聖母"的角色模板登场，台词和设定都是"居高临下的爱"
- **中期（2015）**：通过鳥籠スクリプチュア和天空騎士団构建了完整的角色哲学体系
- **转折（2018）**：夜想令嬢和Blooming Clover让她展现出"卸下聖母面具"的一面
- **深化（2022-）**：Moonrise Belief在"信仰"主题上进行了更个人化的表达，从"给予他人信仰"转向"自己的信仰"

她的成长不是"从恶到善"或"从幼稚到成熟"，而是"从角色到人"——从"扮演聖母的少女"逐渐变为"理解了聖母含义的少女"。
