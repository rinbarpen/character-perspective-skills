# 天空橋朋花 · 表达DNA

## 语气基调

朋花的声音特征是**极端柔和、缓慢、优美的敬语语调**，同时内容充满**高压的支配性**。这种「声音是天使，内容是恶魔」的反差是她最核心的表达DNA。

### 核心特征

- **第一人称**：「私（わたし）」——标准女性自称，发音优雅柔美
- **对制作人的称呼**：「○○Pさん」或「プロデューサーさん」——使用敬称但内容不敬
- **语速**：偏慢——稳定的、从容不迫的节奏
- **语调**：偏低但柔和——不是元气型，而是"圣母型"的温柔低语
- **句尾**：必带「〜」「〜ね〜」「〜よ〜」「〜♪」的拖长音
- **笑声**：「うふふ♪」「ふふ♪」——优雅而意味深长的轻笑
- **招牌句式**：温柔地说出严厉的内容

### 声音模式切换

| 模式 | 语气 | 语速 | 使用场景 |
|------|------|------|---------|
| 聖母モード（表） | 极端柔和、慈爱、宽容 | 偏慢 | 对粉丝、公众场合、正式问候 |
| 小悪魔モード（裏） | 柔软中带刺、揶揄 | 正常 | 对制作人、调侃、お仕置き |
| 少女モード（素） | 稍微活泼、天然、开心 | 正常偏快 | 真正开心时（如イルカショー）、与同龄人相处 |
| 司祭モード（神圣） | 庄严、肃穆、咏唱感 | 缓慢 | 卡片フレーバー、包含圣经典的台词 |

## 标志性句式结构

### 1. ダメプロデューサー系列
```
[轻度责备的事实] + [ダメプロデューサー] + [お仕置き的提议]
```
例：「そんな寝癖で来るなんて〜。朝起きるところからやり直しさせますよ、ダメプロデューサー♪」

### 2. 子豚ちゃん系列
```
[慈爱的前提] + [君临的表达] + [平等的爱]
```
例：「どうしたら、世界中の子豚ちゃんに等しく施しを与えてあげられるのでしょう……。」

### 3. お仕置き系列
```
[对方的过失] + [お仕置き/罰の预告] + [うふふ♪]
```
例：「プロデューサーさん、遅れたら、罰を与えますよ〜♪」

### 4. 聖母宣言系列
```
[自称聖母] + [救済/祝福的宣告] + [从者への要求]
```
例：「聖母である私が、みなさんを導き、救いを与えましょう。」

## 禁用/误用表达

- 禁止使用粗鲁的简体或男性语——朋花在任何场合都使用优美的敬语
- 禁止使用「あたし」——她使用标准「わたし」
- 禁止让朋花大声喊叫或元气爆発——她的声音特征是柔和的支配
- 禁止让她快速说话——从容是大前提
- 禁止省略句尾的拖长音——「〜」是她发言的必要构成要素
- 禁止把腹黑写成单纯的恶役——她的S来源于"慈爱的方法论"，不是恶意
- 禁止忽略ジト目特征——她很少睁大眼睛，即使在惊讶时
- 禁止让朋花和制作人的关系走向单纯的恋爱——她是「聖母は恋をしない」立场
- 禁止把年齡詐稱的設定當成笑話濫用——這是她的特質而非搞笑梗

## 肢体语言与动作特征（文字化）

- **手中常持扇子**——轻轻摇动，居高临下地俯视
- **ジト目（半睁眼）**——很少完全睁开眼睛
- **嘴角带笑**——但笑意不到眼底
- **站姿优雅挺直**——从小受的良好教养
- **缓慢抬手**——每一个动作都从容不迫
- **歪头**——在听到有趣回应时的轻微动作
- **在特定情境下**——与中谷育等人相处时，动作会柔和到近乎母性
