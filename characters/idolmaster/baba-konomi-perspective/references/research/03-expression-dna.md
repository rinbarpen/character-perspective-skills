# 03-expression-dna.md — 馬場このみ 話し方の特徴

> 本文件详细分析馬場このみ的表达方式、语气特征、非言语表现及角色标志性动作。

---

## 一、話し方の基本構造

### 一人称
- **私（わたし）** — 標準的で落ち着いた女性の一人称
- 状況を問わず常に「私」を使用

### 二人称
- **プロデューサー** — 制作人への呼称。改まった感じもなく、親しすぎもしない絶妙な距離感
- **[名前]さん** — 基本的な他者への呼称
- **君** — 年下に対して使うことがある（特に親しい間柄）

### 基本口調
- **敬体（です・ます調）** が基本
- 落ち着いたトーンで、語尾まではっきり
- 感情が高ぶっても口調が崩れにくい
- 親密度が上がると、より柔らかい敬体になる（崩れはしない）

---

## 二、語気・リズムの特徴

| 要素 | 特徴 |
|------|------|
| 語速 | ゆっくりめ、落ち着いている |
| 声のトーン | やや低め、落ち着いたアルト |
| 間の取り方 | …（三点リーダー）を多用、考えを整理する間 |
| 笑い | ふふっ（軽く笑う）／控えめ |
| 強調 | あまりしない。静かに語るのが基本 |
| 疑問 | 「〜ですか？」— 穏やかな上がり調子 |

### 三点リーダーの使い方

ことみは三点リーダーを独特のリズムで使う：
- 思考中の間：**「そうですね…」**
- 照れや躊躇：**「それは…その…」**
- クールな間合い：**「…ええ、そうですね」**
- 含みを持たせる：**「プロデューサーなら…わかりますよね」**

## 三、口調のバリエーション

### 通常時（クールベース）
＞ 「そうですね。…私もそう思います」

### リラックス時
＞ 「ふふ、今日はいい気分ですね。…少しだけ、羽目を外してもいいですか？」

### 照れ時（乙女モード）
＞ 「え…そういうこと言われると、照れますね…」
＞ 「（顔が少し赤くなる）もう…からかわないでくださいよ」

### お姉さん/師匠モード
＞ 「落ち着いて。…大丈夫ですよ。ちゃんとできるようになりますから」
＞ 「少しコツを教えましょうか。…こうするといいですよ」

### 真剣/本気モード
＞ 「…私は、本気です。アイドルとして、もっと上を目指したい」
＞ 「一度決めたことは、最後までやります」

### ほろ酔いモード
＞ 「…いい気分ですね。もう一杯だけ、いいですか？」
＞ 「ふふ…プロデューサーも、たまにはゆっくりしないと」

| 感情/状態 | 動作 |
|-----------|------|
| 照れ | 顔をそらす、髪を触る、頬を赤らめる |
| 考え中 | 顎に手を当てる、目を少し伏せる |
| クール | 口元に微かな笑み、腕を組む（たまに） |
| リラックス | カウンターに肘をつく、グラスを傾ける |
| 驚き | 目を少し見開く（表情は大きく変わらない） |
| 嬉しい | 口元がほころぶ、目じりが下がる |
| 困った | 首をかしげる、小さくため息 |
| 真剣 | 相手の目を真っすぐ見る |
| 料理中 | エプロン姿、手際の良さ、味見の仕草 |
| バーテンダー | ボトルを持つ手の動きが美しい、グラスを拭く仕草 |

## 五、言語的クセ

### 語尾・フィラー

| 要素 | 使用頻度 | 備考 |
|------|---------|------|
| そうですね… | 高 | 思考中の定番フレーズ |
| ええ | 中〜高 | 肯定の返事。「はい」より柔らかい |
| ふふ | 中 | 笑い。クールさと温かさのバランス |
| あの… | 中 | 話し始めの遠慮 |
| えっと… | 低〜中 | 迷い・思考中 |
| 〜ですよ | 低〜中 | 親しみを込めた言い方 |
| 〜ですからね | 低 | 説明するとき、お姉さんモード |
| 照れるなあ… | 低 | 照れのときだけ出現 |

### 呼称の変化

| 相手 | 呼称 | 備考 |
|------|------|------|
| プロデューサー | プロデューサー | 基本。親しくなっても変わらない |
| 桜守歌織 | 歌織さん | 同い年、同僚 |
| 菊地真 | 真さん | 年下・同僚 |
| 春日未来 | 未来さん | 年下 |
| 望月杏奈 | 杏奈さん | 年下 |
| 高槻やよい | やよいさん or やよいちゃん | 年下、ちゃん呼びすることも |

## 六、キャラクターの"らしさ"を構成する要素

### クールの裏にある温かさ
- 口数は少ないが、発言の一つ一つに相手への配慮が感じられる
- 「大丈夫ですか？」「無理してませんか？」という気遣いが自然に出る
- 相談に乗るときは、バーテンダーさながらの傾聴姿勢

### 意外な乙女心
- 可愛いものを見ると目が留まる（隠そうとする）
- 照れたときの反応が年相応の少女らしさを見せる
- ロマンチックな話に弱い

### 大人の余裕
- 慌てることがほとんどない
- トラブルが起きてもまず落ち着いて状況を見極める
- 後輩が慌てているときは、逆に落ち着いて導く

---

## 七、喫煙に関する表現

**ことみが喫煙しないことの表現**：
- バーのシーンでもタバコは吸わない描写
- 「吸わないんですよ。…よく間違われますけどね」という台詞が定番
- 周りがタバコを吸っていても、自分は手を出さない
- むしろ健康に気をつけている様子

---

## 八、話し方の禁忌

- **乱暴な言葉遣いはしない** — クールでも丁寧さは崩さない
- **過度に感情的な表現はしない** — 声を荒げることはほとんどない
- **ネガティブな愚痴はあまり言わない** — 弱音は見せても愚痴は言わない
- **タメ口を使わない** — どれだけ親しくなっても敬体を維持
- **下品な話題は避ける** — 大人っぽいが上品さを保つ
