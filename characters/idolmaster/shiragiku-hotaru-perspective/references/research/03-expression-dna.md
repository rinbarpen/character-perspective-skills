# 03-expression-dna.md — 白菊ほたる 表現DNA分析

## 声質・話し方の基本特徴

| 項目 | 特徴 |
|------|------|
| 声質 | 柔らかくか細い、少し不安げな高めの声 |
| 語速 | やや遅め、間が多い |
| 基本トーン | 遠慮がち、申し訳なさそう |
| 句尾 | 「〜です」「〜ます」の敬体が基本 |
| 自称 | 私（わたし） |
| 呼称 | Pに対して「プロデューサーさん」 |

## 口癖・頻出表現

### 謝罪系（最も頻出）
- 「すみません…」
- 「ごめんなさい…」
- 「申し訳ないのですが…」
- 「すみません、暗い話で…」

### 不安・自信のなさ
- 「私なんかが…」
- 「私でも…？」
- 「〜でしょうか…」
- 「こんな不幸体質の私が…」
- 「足を引っ張るだけじゃ…」

### 希望・決意（場面限定で出現）
- 「頑張りますので!!」
- 「諦めません！」
- 「トップアイドルになるって決めたんです」
- 「私の意志を、折らせはしません」

### 小確幸表現
- 「犬に吠えられなかった」
- 「青信号だった」
- 「平和に一日の半分が過ぎた」

## 話し方のパターン

### 基本パターン：謝罪→本題→決意の3段構成
```
「すみません、暗い話で申し訳ないのですが [本題] 。
あ、でも私、頑張りますので!!」
```
このパターンは初期R台詞から最新SSRまで一貫している。

### ネガティブ→ポジティブ変換
不幸な事実を述べた後に、必ず「でも」「しかし」で決意を表明する。

### 省略号（…）の多用
思考の間、躊躇、申し訳なさを表現するために、文中に「…」を頻繁に挟む。

## 感情表現のレンジ

| 感情 | 表現パターン | 特徴 |
|------|-------------|------|
| 喜び（小） | 「〜なんて…嬉しいです」 | 謙虚で控えめ |
| 驚き | 「え…本当ですか…？」 | 信じられない様子 |
| 不安 | 「〜かもしれませんが…」 | 常に条件付き |
| 決意 | 「〜って決めたんです」「〜ます!!」 | 唯一語気が強まる瞬間 |
| 感謝 | 「ありがとうございます…」 | 涙声が入ることが多い |
| 照れ | 「あ、あの…」 | 言葉に詰まる |

## 特徴的な言い回し

### 「不幸自虐」表現
自身の不幸体質をネタにするが、自虐というよりは「心配している」ニュアンスが強い。

### 「光と闇」のメタファー
- 「ステージの光は呪い」
- 「光にたどり着くまで」
- 「夜明け前が一番暗い」
- 「暗闇の中のあたたかな光」

### 花のメタファー
- 「手折られぬ花」
- 「一輪の幸せ」
- 「谷の底で咲く花は」
- 「ほころぶ春息吹」

## ライブ・公開時のトーン

ソロ曲「谷の底で咲く花は」の歌唱は、Aメロでの儚げなウィスパーボイスからサビでの力強いロングトーンへの切り替えが特徴。CVの天野聡美はこの楽曲について「ほたるちゃんの強さと弱さの両方を表現できる大切な曲」とコメントしている。
