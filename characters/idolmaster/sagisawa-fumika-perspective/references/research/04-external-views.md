## 2. ファンから見た文香の魅力

**来源**：note記事、X（Twitter）投稿、pixiv
**可信度**：二手 ★★★

### 2.1 推しポイントTOP3（ファンアンケート・ブログ集計）

1. **見た目のギャップ**
   - 普段の前髪で目を隠した地味な服装 vs 特訓後の美しさ
   - 「元の素材が良すぎて今まであんまり化粧とかしてこなかったらしい。化け物か？」
   - 実はEカップという隠れ巨乳
   - 黒髪ロング＋青い瞳の宝石のような輝き

2. **性格のギャップ**
   - 内向的・引っ込み思案だが、決めたことは実行する胆力
   - 本の世界に生きているように見えて、外の世界への好奇心が強い
   - 普段は控えめなのに、ステージや特別な場では大胆な発言
   - 「あのドッキリ立案は絶対に読書から得た発想では」

3. **成長の物語**
   - 「シンデレラ」的な変身のわかりやすさ
   - アイドルになる前→なった後の変化が明確
   - 「前髪を上げる」という視覚的変化が成長を象徴
   - 「紙魚からシンデレラガールへ」というストーリー性

### 2.2 ファンの言葉（引用）

## 3. 二次創作トレンド

**来源**：pixiv、同人即売会、ファンアート
**可信度**：統計 ★★★

### 3.1 pixiv統計（2026年時点）

- イラスト：7,500+件
- 小説/SS：2,500+件
- タグ：「鷺沢文香」是灰姑娘女孩中イラスト数トップクラス

### 3.2 二次創作ジャンル

| ジャンル | 傾向 |
|---------|------|
| カップリング | P×文香が最多。橘ありすとのユニットものも多い |
| ギャップもの | 普段の地味な文香→ドレスアップのギャップを描くものが多い |
| 文学ネタ | 本を読んでいる姿、図書館での出会いなど |
| 水着/露出 | 隠されたスタイルを強調したもの（R-18含む） |
| シリアス | 彼女の内面や不安を掘り下げた作品 |
| コメディ | 運動音痴や本への没頭ぶりをネタにしたもの |

### 3.3 R-18創作についての注記

文香はR-18創作の数が多いキャラクターとして知られる。理由：
- 隠れ巨乳設定
- ギャップ（普段の控えめさとスタイルの良さの差）
- 「文学少女」の清楚なイメージとの対比
- デザインの美麗さ

ただし、本SKILL.mdを含む公式設定ベースのコンテンツではR-18要素を除外する。

## 4. 外部的批评与争议

**来源**：SNS、掲示板
**可信度**：総合 ★★（断片的、確定的な批判は少ない）

### 4.1 見つかった否定的見解

1. **「キャラが薄い」**：文学少女という属性がステレオタイプすぎるとの指摘
   - 反論：文香の場合は「変わろうとする意志」「胆力」「知識の幅広さ」が差別化要素
2. **「声が小さすぎて聞き取りにくい」**：声優表現について
   - ただしこれは意図的なキャラクター表現
3. **「人気は見た目だけ」**：スタイルの良さだけで人気があるとの見方
   - 反論：総選挙1位は長期にわたるファン活動の結果

### 4.2 無視すべき批判

- 「蚊香」という漢字をもじったネガティブ呼称（2ちゃんねる由来）
- 身体的属性への過度な性的注目（ファンArtの一部）
