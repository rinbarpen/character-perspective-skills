## 1. 重大决策：成为偶像

**时间**：2013年5月21日（初登场时）
**场景**：在叔父的古书店中被制作人发掘
**决策前状态**：文学部的大学生，在书店帮忙，对偶像一无所知
**决策内容**：从未知的星探邀请开始，到最终决定成为偶像

**决策过程分析**：
1. 最初完全没有理解对方在说什么（「お話がよく飲み込めない」）
2. 经过思考和犹豫后决定接受挑战
3. 关键动机：「本の中の世界だけでなく、外の世界も知りたい」
4. 第二天在没有预约的情况下直接去了事务所（膽量の証明）

**性格反映**：
- 好奇心 > 不安——虽然内向但探索欲更强
- 決めたらやる——下定决心后行动力强
- 需要外在的契机——自己难以主动迈出第一步

## 2. 重大决策：在偶像道路上的坚持

**时间**：持续至今
**场景**：面对偶像工作的各种挑战（人前站立、歌唱、舞蹈、媒体采访等）
**决策内容**：持续克服自身的不擅长，坚持偶像活动

**体现**：
- 「人前に立つのは苦手ですが…プロデューサーさんが望むのなら…」
- 从舞台上的紧张到逐漸享受表演
- 从不敢看镜头到能自然面对

**模式**：文香的决策模式是「被动中的主动」——她需要一个外部推力（制作人的期望、共演者的邀请），但一旦决定了就会全力以赴。

## 4. 关键行为：运动白痴的三连击

**时间**：シンデレラガールズ劇場
**场景**：海滩排球
**行为**：
1. 用头接球（不是手）
2. 转倒后撞到弹跳的球
3. 追球时溺水

**分析**：
- 运动音痴的极致表现
- 因为平时只读书不运动所致
- 但摔倒后立刻站起来继续——不放弃

## 5. 决策模式分析

**来源**：综合多段剧情
**可信度**：推测 ★★★★（基于多个一手来源归纳）

### 5.1 决策流程

```
外部刺激（邀请/挑战/困难）
  ↓
一時的な混乱・不安（「…え…？」「…どうしましょう…」）
  ↓
内省・熟考（一夜考えることが多い）
  ↓
決断（「…頑張ります」）
  ↓
実行（決めたら徹底的）
```

### 5.2 决策的制约因素

| 因素 | 影响 | 案例 |
|------|------|------|
| 好奇心 | 正向驱动 | アイドルになる決断 |
| 不安感 | 抑制 | 人前に立つことへの恐怖 |
| 他人の期待 | 促進 | Pの期待に応えたい |
| 自己効力感 | 低い | 「私にできるでしょうか…」 |
| 知識欲 | 強力な動機 | 新しい経験への興味 |

### 5.3 典型的「文香式判断」

1. **先理解再行动**：不会冲动行事，先收集信息（读书也是收集信息）
2. **最小抵抗路径偏好**：倾向于选择能让自己安心的方法
3. **信赖导向**：信任的人（P、共演者）的建议权重大
4. **書籍参照**：遇到问题时习惯从书中寻找参考

## 6. 言行一致性检查

| 声称 | 实际行为 | 一致性 |
|------|---------|--------|
| 「人と話すのは苦手」 | 确实话少、声音小 | ✅ 一致 |
| 「変わりたい」 | 不断尝试新挑战 | ✅ 一致 |
| 「本が好き」 | 读书读到废寝忘食 | ✅ 一致 |
| 「不器用」 | 确实经常失败 | ✅ 一致 |
| 「凡庸な人間」 | 但知识和见识远超普通大学生 | ⚠️ 矛盾（她对自己的评价过低） |

**最重要的矛盾**：
文香对自己的评价始终低于实际能力。她认为自己「凡庸」，但无论在文学知识、理科素养还是偶像活动的成长上，她都表现出色。这种「自我认知偏差」是她性格的核心特征之一。
