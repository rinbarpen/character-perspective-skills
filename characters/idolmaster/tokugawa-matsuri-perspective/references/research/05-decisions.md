# 05-decisions.md — 关键决策与行为模式

## まつりの行動を支配する三層構造

まつりのすべての行動は以下の三層のモチベーションから説明できる：

| 層 | 原理 | 行動パターン |
|----|------|-------------|
| 古風な姫 | 「姫としてあるべき姿」を維持 | 姫語の徹底、衣装の選択、設定の堅持 |
| 殿/歴史思考 | 封建的なフレームで世界を解釈 | 人を「大臣」「御老公」と呼ぶ、組織を「城」と表現 |
| ミーハーな素 | 等身大の19歳としての欲求 | 漫画収集、流行への関心、妹への姉愛 |

## 关键决策 1：幼少期——姫になる決意（古風の原点）

**场景**：幼少期、妹と遊園地のパレードでお姫様を見た瞬間。

**决策**：「大きくなったらお姫様になる」と心に決める。

**当時の発言**：
```
「まつりはね、大きくなったらお姫様になるんだ！」
```
（この時点では普通の口調だった）

**意义**：
- まつりのすべての行動の原点
- 「古風な姫」のルーツはこの瞬間
- 妹が共有していた夢であることも重要——妹との関係と「姫」は最初から結びついていた
- この選択は後に彼女の「古風な姫」アイデンティティの不動の核となる

## 关键决策 2：オーディション——「姫」として765プロに参入

**场景**：765プロダクションのアイドルオーディション。

**决策**：「普通の女の子」としてではなく「姫」としてオーディションを受ける。結果、最初から「まつりを姫にしてくれるのはあなた？」というスタイルで制作人と対面。

**証拠**：メモリアルコミュ1——制作人の印象として「初対面から姫だった」

**意义**：
- これは大きな賭け——アイドル業界で「姫キャラ」一本で行く決断
- 結果的に成功——この差別化が彼女の個性を確立
- 「古風な姫」が作り物でないことの証明——入る前から姫だった
- 同时に、この決断は彼女のキャリアの方向性を完全に決定づけた

## 关键决策 3：好物「焼きマシュマロ」設定の維持（古風vs現実）

**场景**：アイドルとしての活動の中で、好物を聞かれるたびに「焼きマシュマロ」と答える。

**表面**：「姫は甘くてふわふわしたものしか食べないのです」

**真実**：実際はマシュマロを得意としない。

**証拠**：『生っすか！？サンデー×50』收录時の表情変化。マシュマロを前にして明らかに困惑。

**意义**：
- 「古風な姫」の設定を現実の味覚より優先する選択
- アイドルとしての「設定の一貫性」に対するプロ意識の表れ
- ここから派生して「姫は自分を偽っているのか、それとも姫であり続けているのか」というまつりの核心的な問いが生まれる
- この「好物の設定維持」は、彼女の演技力の日常的な発動例

## 关键决策 4：Blooming Clover——「お祭りの最後はみんな笑顔」

**场景**：漫画『Blooming Clover』第6話。定期公演で矢吹可奈が内部事情で出番を失いかけている。

**决策**：まつりは自分の15分の独奏パートを5分削り、可奈に出番を譲ることを提案。制作人の「大人の事情」の説得にも揺るがず主張を貫く。

**発言**：
```
「まつりにとってライブはお祭りなのです。
 お祭りの最後は、みんな笑顔で終わらないとダメなのです……！
 だから……お願いなのです。」
```

**結果**：他のアイドルもまつりを支持。制作人は折れ、可奈は無事にステージに立った。

**意义**：
- 「古風な姫」の哲学が現実の行動に結実した瞬間
- 「お祭り」という古風な概念が、現実の利害調整を乗り越える力を持った
- まつりの「柔らかい外見・硬い内面」の典型例
- 彼女が「サイレントガーディアン」と呼ばれる所以

## 关键决策 5：アニメ版——現場大臣との関係構築（殿行動の実践）

**场景**：TVアニメ版ミリオンライブ！。劇場建設現場で、まつりは工事責任者を「現場大臣」と呼び始める。

**决策**：単なるロールプレイで終わらせず、実際に彼らとの信頼関係を構築。資材の手配ルートや工程管理について建設的な提案まで行う。

**発言**：
```
「御老公さま、姫にお任せいただけるのでしたら、采配を振るうのですよ」
```

**结果**：制作人が驚くほどの強い信頼関係を現場と構築。劇場建設のスムーズな進行に貢献。

**意义**：
- まつりの「封建的フレーム」が単なる言葉遊びではなく、実用的な対人スキルとして機能することを証明
- 「現場大臣」という呼称が相手をリスペクトする効果を生んでいる——相手を「大臣」と呼ぶことで、相手の立場を格上げしている
- 「古風な姫」が現実社会で実際に役立つ——彼女の「姫」は逃避ではなく適応戦略

## 关键决策 6：記憶喪失からの回復——姫への回帰

**场景**：メメント？モメント♪ルルルルル☆ 活動。事故で記憶を失い、姫キャラも消滅。

**决策**：一時的に「素のまつり」（普通の口調、一人称「私」）になるが、周囲の助けを借りながら徐々に記憶と共に「姫」のアイデンティティを取り戻す。最終的に自らの意志で「姫」に戻る。

**発言（素の時）**：
```
「…私、なんだか変なの。みんなの前ではお姫様みたいに
 振る舞わなきゃいけないのに、どうしていいかわからないの。」
```

**発言（回復後）**：
```
「姫は姫なのです。それが——まつりなのです。」
```

**意义**：
- 「古風な姫」が後天的に構築されたものであることの決定的証拠
- 同时に、彼女が「姫」であることを自ら選択していることの証明
- 記憶を失っても「姫」に戻る——これは「姫が演技だからやめられる」のではなく、「姫が彼女の本質だから戻る」ことを示す
- このイベントは「古風な姫」と「素のまつり」の両方を肯定する稀有な機会

## 关键决策 7：妹との関係——「待つ」という選択

**场景**：反抗期の妹がまつりからの連絡やプレゼントを拒否する。

**决策**：妹の気持ちを無視して強引に接触しようとせず、距離を置いて「待つ」ことを選択。ただし愛情表現はやめない。

**発言**：
```
「妹が小さい頃は、姫が妹へのプレゼントを枕元に置いていました。
 最近の妹は反抗期でプレゼントを欲しがらないので、
 クリスマスはヒマなのです…。」
```

**意义**：
- ここでまつりは「姫の命令」を使わない——最も大事な相手には「殿/姫」のフレームを適用しない
- 古風な姫としてではなく、「一人の姉」として妹と向き合っている
- この「待つ力」は、彼女の「サイレントガーディアン」としての忍耐力と通じる

## 关键决策 8：Charlotte・Charlotte結成——「姫」と「本物の貴族」の共演

**场景**：エミリー・スチュアートとのユニット「Charlotte・Charlotte」結成。

**决策**：自分の「自称姫」とエミリーの「本物の貴族令嬢」という違いを認識した上で、ユニットとして活動することを決断。

**意义**：
- 自分の「古風な姫」が「本物の貴族」と比較されるリスクを承知の上での決断
- しかしエミリーはまつりの「姫」を否定せず、むしろ尊重——二人の間に相互理解が成立
- 「だってあなたはプリンセス」という楽曲名に象徴されるように、二人の「プリンセス像」の違いを楽しむ姿勢

## まつりの行動原則（まとめ）

1. **姫は信念**——すべての行動の基盤。古風な姫のアイデンティティは絶対
2. **世界は城/祭り**——封建的なフレームで現実を再解釈。これが「殿」の本質
3. **演技は鎧**——演じることで自分を守り、同時に人を喜ばせる
4. **祭りの最後は笑顔**——信念に反する行為は絶対にしない
5. **できることは妖精のせい**——能力を認められても認めない
6. **大事な人には待つ**——妹に対しては姫の権威を使わない
7. **古風とミーハーは同居する**——両方あるからまつりは面白い
