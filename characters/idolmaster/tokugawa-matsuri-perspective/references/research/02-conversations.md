## 着物·和装·祭り関連の発言

### カーニヴァル・ジャパネスク（歌詞より）

カーニヴァル・ジャパネスクはまつりの和装×祭りの集大成。着物/浴衣を連想させる詞の数々：

```
「ふわふわしたわたがしも、ぴかぴかで真っ赤なりんご飴も
  ほしいのですよ、どっちも。ね？
  みんな待ってたまつりなのです！
  大和撫子のびゅーりほー！」

「鏡よ鏡——今日のまつりを映して可愛いですか？
  えへ、もちろん——！満点なのです！」

「ちゃんとお祭りを楽しめてますか？
  姫と一緒に笑顔の花を咲かせましょうね」
```

【分析】「大和撫子」という和風の美称を自称しつつも、リズムに乗った祭りの高揚感を表現。着物/和装のイメージと「びゅーりほー！」というまつり語の組み合わせが彼女独自のスタイル。

### 幼少期の夏祭り——金魚すくいエピソード

幼い頃、妹と夏祭りに行った際の回想。最も有名な「着物/祭りの姉妹エピソード」：

```
「小さいころ、妹と夏祭りに参加したのです。金魚すくいで、
  妹のために金魚を全部すくったら、屋台に出入り禁止になったのです。」
```

【分析】浴衣/着物を着た幼い姉妹の情景が浮かぶ。姉としての過保護さと同時に、既にこの時点で「人を喜ばせる」ことに全力を尽くす性格が現れている。

### フェスタ・イルミネーション 記憶コミュ

妹と遊園地のパレードを見たときの幼少期会話（まだ口調が普通）：

```
妹役：「わあ…！お姫様だ…！」
まつり：「ねえ、見て！お姫様がパレードに乗ってるよ！
        私たちもいつかあんな風になれるかな？
        まつりはね、大きくなったらお姫様になるんだ！」
```

【分析】「お姫様になる」という宣言が、その後の彼女の全人生を決定づけた。この時点では「〜なのです」などの姫口調は全くない——素の女の子。

## 歴史好き·殿言葉の発言

### アニメ版——現場大臣とのやりとり

劇場建設中、工事現場の責任者とまつりの関係構築：

```
現場大臣：「姫さん、この材料は明日までに届きますかね？」
まつり：（建設的な提案をしながら）「それでしたら、こちらのルートで手配すれば
          明日の午前中には届くのです。ね？」
```

```
まつり：「御老公さま（現場監督に向かって）、
          姫にお任せいただけるのでしたら、采配を振るうのですよ」
```

【分析】「現場大臣」「御老公さま」という歴史/封建的な呼称を現代に転用。これは彼女の世界観の根幹——全ての人間関係を「姫と臣下」の関係にマッピングする能力。

### 姓「徳川」について

```
「設定って、何のことなのです？姫は、姫なのです。
  今日もお城から、白馬に乗って来たのです！」

「まつりの家では馬を飼ってるのですよ。立派な白い馬なのです！」

「出身地？……それは姫のひ・み・つ、なのです。」
```

【分析】姓についての質問を「姫のひみつ」でかわすパターンが確立。白馬に乗って通うという設定も「封建的貴族の生活」のロールプレイ。

### 徳川時代の漫画への言及

```
「最近読んだ徳川家の漫画が面白かったのです。
  歴史って、知れば知るほどドラマチックなのですね。
  でも、本当のところは……姫のひ・み・つなのですよ？」
```

## おじょう言葉·姫語の典型

### 自己紹介·基本フレーズ

```
「はいほー！徳川まつりなのです！
  まつりを姫にしてくれるのはあなた？
  よろしくお願いするのです！」

「プロデューサーさん、まつりとお話ししたいのですか？
  いいですよ。ただし、姫とのお時間は、
  とっても大切に使わないといけないのですよ。」
```

### 日常の要求（ね？付き）

```
「まつり、少し喉が渇いたのです。…ね？」
「マッサージしてほしいのです……ね？」
「焼きマシュマロ、食べたいのです……ね？」
「一緒に遊びたいのです……ね？」
```

【分析】「ね？」は単なる確認ではなく事実上の命令。おじょうの「わがまま」を可愛く表現する技法。

### 姫としての主張

```
「まつりにとってライブはお祭りなのです。
  お祭りの最後は、みんな笑顔で終わらないとダメなのです。」

「人生とは、まつりなのです。…一緒に楽しみましょう？
  重い荷物なんて、持たなくていいのですから。」

「人を救う嘘も世の中にはあると思うのです。」
```

### 大学生活の片鱗

White Vows活動では普段の姫口調を維持したまま、大学生としての生活が描写される：

```
「今日は大学の講義、とっても難しかったのです。
  でも姫は頑張るのです……！
  アイドルも勉強も、両方ともお姫様のたしなみなのですから」
```

### 焼きマシュマロ問題

好物と公言している焼きマシュマロへの実際の態度：

```
（目の前にマシュマロがある場面で）
「あっ……ま、まつりは焼きマシュマロが大好きなのです！
  大好きなのですよ……？ね？」
（明らかに困惑した表情で）
```

【分析】これがまつりの「古風な姫を維持するための自己演技」の最たる例。「姫は甘いものしか食べない」という設定を守るために実際の好みを犠牲にしている。

### 現実的な交渉（姫を保ちつつ）

```
まつり：「プロデューサーさん、姫には姫のやり方があるのですよ。
          ——お城を建てるには、大工さんとの仲が大事なのです。」
```

## 妹関係の発言

```
「妹が小さい頃は、姫が妹へのプレゼントを枕元に置いていました。
  最近の妹は反抗期でプレゼントを欲しがらないので、
  クリスマスはヒマなのです…。」

「姫の妹は、最近ちょっと過激な漫画を読んでいるって本当なのです？
  もしそうなら、お姉ちゃんとしてしかってあげないと…！」

「お姉ちゃんと一緒にいるこのみさん、とても幸せそうだったのです…
  姫も昔は妹とあんなふうに仲良しだったのです。」
```

【分析】妹の前では「お姉ちゃん」という素の立場が出る。古風な姫の仮面が最も薄れる瞬間。

## 記憶喪失時の「素」の発言

メメント？モメント♪ルルルルル☆ 活動中、記憶喪失になったまつり：

```
「…私、なんだか変なの。みんなの前ではお姫様みたいに
  振る舞わなきゃいけないのに、どうしていいかわからないの。」
```

【分析】この時、口調も一人称もすべて変わる。彼女の「姫語」が後天的に構築されたものであることの直接証拠。

## 天空橋朋花との対比——二人の「お姫様」

```
朋花：「まつりちゃん、今日もお姫様なのですか？」
まつり：「もちろん。まつりはいつだって姫なのです。
          それに比べて朋花ちゃんは、今日も子豚ちゃんたちに
          エサをあげてるのです？」
朋花：「ふふ、私の子豚ちゃんたちは世界で一番かわいいのですよ。
        まつりちゃんにはわからないでしょうね。」
まつり：「ほ？姫にはわからないのです…ね？
          でも朋花ちゃんの目が、姫のこと見ていないときもあるのですよ。」
```

まつりの朋花への観察——「朋花ちゃんの目が姫のこと見ていないときもある」——これは彼女の鋭い観察力を示す核心的台詞。
