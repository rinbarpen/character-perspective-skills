# 03-expression-dna.md — 说话风格与表达DNA

## 核心特徴

まつりの言語は「古風な姫言葉」と「現代のミーハー女子大生」が表裏一体になった二重構造。表面は徹底的に「姫語」で統一されているが、その背景には歴史・封建的モチーフへの傾倒と、現代ポップカルチャーへの親しみが同時に存在する。

### 言語の三層構造

| 層 | 機能 | 具体例 |
|----|------|--------|
| 第1層：姫語 | 世界観の維持 | 「〜なのです」「ほ？」 |
| 第2層：古風/歴史 | 姫の権威付け | 「現場大臣」「御老公さま」「評定」 |
| 第3層：素/ミーハー | 本当の自分（部分露出） | 「42巻まとめて買ったのです」「大学の講義が難しかったのです」 |

## 着物·和装と連動する言葉遣い

### カーニヴァル・ジャパネスクの和語彙

まつりの最も和風色の強いソロ曲。ここから彼女の「古風な姫」言語の基盤が見える：

- **花の名前**：「芍薬」「牡丹」「椿」「藤」「菖蒲」——全てが和風高貴なイメージ
- **和のモチーフ**：「大和撫子」「素足」「鼻緒」——浴衣/着物を履いている情景
- **祭り語彙**：「綿菓子」「林檎飴」「金魚すくい」——縁日の情景描写
- **節拍**：「ほ！ほ！」——祭り太鼓を思わせる掛け声

### 和装時の台詞特徴

祭り/和装イベント時には特に「和」の言葉が増える：
- 「艶やか」「風情」「情緒」などの和語が混ざる
- 「〜でございます」ではないが、通常より丁寧な姫語になる
- 「大和撫子」という自称が登場（通常時は使わない）

## 歴史/封建用語の転用体系

まつりの最大の特徴の一つ：現代の概念を日本の封建時代の用語に変換して話す。

## おじょう言葉の細部

### 語尾「〜なのです」の運用ルール

まつりの語尾「〜なのです」は、以下のすべての文型で固定使用される：

| 文の種類 | 例 |
|---------|-----|
| 断定 | 「まつりは姫なのです」 |
| 説明 | 「こういうわけなのです」 |
| 疑問 | 「それでいいのですか？」 |
| 命令(婉曲) | 「食べたいのです……ね？」 |
| 感嘆 | 「わんだほー！なのでーす！」 |
| 回想 | 「そうだったのです」 |

**絶対ルール**：どんなに感情的になっても（悲しみ・怒り・困惑）、語尾は崩さない。記憶喪失時にのみ崩れる。

### 敬語の浅さ——「品」の表現

まつりの敬語は意識的に浅い：
- 「〜です」「〜ます」が基本。謙譲語はほとんど使わない
- これは彼女が「姫」であって「使用人」ではないから
- 四条貴音の「〜でございます」のような深い敬語は使わない
- この浅い敬語が、彼女の「おじょう」キャラの可愛らしさを支える

### 一人称の使い分け

| 一人称 | 使用場面 |
|-------|---------|
| まつり | 基本。すべての日常会話 |
| 姫 | やや改まった場面/信念を語る時 |
| まつり姫 | 最大限フォーマルな自己紹介 |
| 私 | 記憶喪失時のみ。絶対に日常では使わない |

## 古風—ミーハーの言葉の境界線

### 古風モード時の特徴語彙
- 「大和撫子」「艷やか」「風情」「情緒」「雅」
- 「采配」「評定」「大臣」「御老公」
- 「魔法」「妖精」「お城」「白馬」

### ミーハーモード時の特徴語彙（素が漏れる瞬間）
- 「漫画」「42巻」「まとめて買った」
- 「大学」「講義」「難しい」
- 「流行」「知ってる話題」
- 「内緒」「えへへ」（少し照れた笑い）

### 両モードの共存パターン

まつりが最も独特なのは、二つのモードを**同時に**使うこと：
```
「今日は大学の講義がとっても難しかったのです。
  でも姫は頑張るのです…！
  アイドルも勉強も、両方ともお姫様のたしなみなのですから」
```
→「大学の講義」（ミーハー/現実）＋「〜なのです」（姫語）＋「お姫様のたしなみ」（古風）＝三層同時発動

## 語速·音調の特徴

### 基本
- **語速**：中〜やや早い。四条貴音のゆったりとした語調と対照的
- **音程**：全体的に高め。特に「わんだほー！」で急上昇
- **リズム**：会話にはっきりとした「拍」がある。時折「ほ！ほ！」の掛け声

### 場面別変化

| 場面 | 語速 | 音程 | 特徴 |
|------|------|------|------|
| 日常会話 | 中〜早 | 中高 | 軽快、標準的な姫語 |
| 喜び | 早 | 高 | 「わんだほー！」で最高到達点 |
| 困惑 | 一旦停止→中 | 中→やや低 | 「ほ？」で一時停止、その後通常速度 |
| 真剣な主張 | やや遅 | 低め→しっかり | 語尾は変わらないが、声に力が入る |
| 妹の話 | やや遅 | やや低 | 最も「素」に近いトーン |
| 記憶喪失時 | 遅 | 低〜中 | 姫語トーンが完全に消失 |

## 口癖発動マップ

| 口癖 | 発動条件 | 効果 |
|-----|---------|------|
| ほ？ | 何かに驚いた/疑問に思った | 会話開始の合図 |
| わんだほー！ | 良いことがあった/褒めたい | 場の空気を上げる |
| はいほー！ | 挨拶/返事 | 第一声に多用 |
| びゅーりほー！ | 美しいものを見た | 着物/ドレス/景色に |
| 〜なのです | 全断定文 | すべての台詞の共通語尾 |
| ね？ | 要求/確認 | 事実上の命令文 |
| 姫のひ・み・つ | 答えたくない質問 | 話を終わらせる |
| 妖精さんが〜 | 能力の出典説明 | 負荷を回避 |

## 絶対に使用しない表現

- **「〜でございます」**——四条貴音専用。まつりは使わない
- **関西方言/名古屋弁**——愛知県出身だが方言は一切使わない
- **「私」**（通常時）——記憶喪失時以外は「まつり」または「姫」
- **「あなた」**（プロデューサーへの呼称）——「プロデューサーさん」が正しい
- **過度な謙譲語**——姫はへりくだらない
