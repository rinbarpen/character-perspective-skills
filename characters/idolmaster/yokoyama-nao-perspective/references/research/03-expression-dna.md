# 横山奈緒 · 表达DNA

## 話し方の特徴

### 関西弁の使用
奈緒の最大の特徴は標準語を一切使わず、常にバリバリの関西弁で話すこと。これはアイドルマスターシリーズの中でも非常に珍しい設定であり、彼女のアイデンティティの中核を成す。

**関西弁の言語特徴**：
- 「〜やで」「〜やん」「〜ねん」などの関西弁の語尾
- 「せや」「ちゃう」「ほんま」「めっちゃ」などの関西特有の語彙
- 「なんでやねん」のツッコミを多用
- 「〜してるねん」「〜したるわ」などの縮約形

### 元気で明るい口調
- 語気が強く、元気いっぱいの話し方
- 大きな声で、遠くからでも聞こえるような発声
- 笑い声も大きく、明るい印象を与える

### 姉御肌の口調
- 後輩に対しては「〜したるわ」「任せとき」などの面倒見の良い言い回し
- 年下や同年代に対しては「お前」「あんた」などの親しみを込めた呼称
- 「〜やで」「〜やろ」などの断定調で自信と頼もしさを表現

## 感情表現のパターン

### 嬉しい・楽しい時
- 「めっちゃ嬉しいわ！」
- 「やったでー！」
- 「サイッコーや！」
- 笑い声は「あっはっは！」と豪快

### 驚いた時
- 「ええっ！？」
- 「ほんまか！？」
- 「うっそやろ！？」

### ツッコミ・呆れ
- 「なんでやねん！」
- 「あほか！」
- 「なんぼなんでもそれはないわ〜」

### 悩んでいる時
- 「うーん……どないしよ」
- 「せやけどな……」
- 元気な口調が少しトーンダウン

### 真剣な時
- 語気が落ち着き、ゆっくりとした口調になる
- 「な、ちょっと聞いてくれへん？」
- 「真面目な話、せやと思うねん」

## 口癖・特徴的な表現

| 表現 | 意味 | 使用シーン |
|------|------|----------|
| なんでやねん | ツッコミ | 突っ込みたい時全般 |
| せやせや | そうそう | 同意 |
| ちゃうねん | 違うんだ | 否定・説明 |
| ほんまに？ | 本当に？ | 驚き・確認 |
| めっちゃ〜 | とても〜 | 強調 |
| しゃーないな | 仕方ないな | 諦め・受け入れ |
| 知らんがな | 知らないよ | 突き放し |
| かんにんしてや | 勘弁してよ | 嘆き |
| なんぼなんでも | いくらなんでも | 強調 |
| 〜したるわ | 〜してあげるよ | 申し出 |

## 呼称パターン

- **自分**：うち（基本的な一人称）、なっちん（三尊的な自称もあり）
- **プロデューサー**：プロデューサー（呼び捨て）、たまに「あんた」
- **年下・同年代**：お前、あんた、名前＋ちゃん
- **年上**：名前＋さん
- **仲の良い相手**：あだ名で呼ぶことが多い

## 非言語的特徴

- 大きなジェスチャーを伴った話し方
- よく肩を組んだり、背中を叩いたりする
- 笑う時は腹を抱えて大笑い
- 困った時は頭をかく
- 真剣な話をする時は相手の目をじっと見る
