# 水濑伊织 — 时间线

> 注：偶像大师系列存在多个平行世界设定。以下时间线主要依据**主流设定**（2nd Vision系列、TV动画、剧场版）。

---

## 早期背景（设定中的身世）

| 时间 | 事件 |
|------|------|
| 出生前 | 水濑财团创立，父亲成为董事长 |
| 5月5日 | 水濑伊织出生，为家中第三个孩子（两个哥哥） |
| 童年 | 物质条件优渥但缺乏家庭温暖；父亲控制欲强；两个哥哥在商业上极为优秀，给伊织带来持续的自卑感 |
| 童年～中学 | 养成随身携带兔子玩偶"夏露露18世"的习惯；发展出对海外旅行和美食的兴趣；开始学习多国语言 |

---

## 偶像生涯主线

### 第一阶段：出道与成长（Nintendo DS/街机时代）

| 时间 | 事件 |
|------|------|
| 2005年 | **街机版《偶像大师》发售**——伊织作为765PRO 9位初始偶像之一首次登场 |
| 2005-2006年 | 街机运营期，伊织以14岁设定活跃 |
| 2007年1月 | 《THE IDOLM@STER MASTER ARTIST 08 水瀬伊織》发售 |
| 2007年9月 | 角色歌《フタリの記憶》（两人的记忆）收录于精选专辑 |
| 2009年 | Xbox 360版《偶像大师》登陆家用机 |

### 第二阶段：转型与龙宫小町（偶像大师2时代）

| 时间 | 事件 |
|------|------|
| 2010年 | PSP《偶像大师 SP》推出，伊织以可制作角色登场 |
| 2011年2月24日 | **《偶像大师2》Xbox 360版发售**——重大转折：伊织成为NPC角色，由律子制作 |
| 2011年 | 龙宫小町（伊织、梓、亚美）成立，伊织担任**队长** |
| 2011年 | 龙宫小町因不可制作引发玩家大规模不满 |
| 2011年- | 龙宫小町在THE IDOL ACADEMY AWARDS和THE IDOL ULTIMATE中与961 Production的Jupiter展开激烈竞争 |
| PS3版 | **コープロデュース（共同制作）路线追加**——伊织与律子的"共同制作"游戏路线上线 |

### 第三阶段：动画化与全球认知（2011年）

| 时间 | 事件 |
|------|------|
| 2011年7月 | **TV动画《偶像大师》开播**（全25集+1集特别篇） |
| 第2集 | 伊织与弥生合唱《Positive!》 |
| 第6集 | **龙宫小町成立**，伊织正式成为该组合成员 |
| 第8集 | 伊织、梓、亚美/真美合唱《Hanikami! First Bite》 |
| 第13集 | 全员合唱《i》，伊织以765PRO ALLSTARS一员演唱 |
| 2011年12月 | TV动画完结 |

### 第四阶段：剧场版与成熟期（2014年）

| 时间 | 事件 |
|------|------|
| 2014年1月 | **《偶像大师 ONE FOR ALL》发售**——伊织作为可玩角色回归 |
| 2014年1月25日 | **剧场版《偶像大师 走向光辉的未来》上映** |
| 剧场版剧情 | 伊织在候补生面前展现前辈风范；片头受邀棒球场开球；展现出"765第二Leader"的担当 |
| 2014年5月15日 | 《偶像大师 ONE FOR ALL》正式版发售 |
| 2014年 | 在《偶像大师 闪耀祭典》中登场（OVA） |

### 第五阶段：百万现场与新篇章

| 时间 | 事件 |
|------|------|
| 2013年 | 《偶像大师 百万现场》手游配信开始——伊织以Vocal阵营加入 |
| 2015年 | 《偶像大师 百万现场 剧场时光》情报公开 |
| 2017年 | 《偶像大师 百万现场 剧场时光》配信开始——伊织为Fairy阵营 |
| 2017年 | 《偶像大师 白金星光》发售——聚焦伊织磨练料理技能 |
| 2018年 | 《偶像大师 星光舞台》发售 |

### 第六阶段：星耀季节与跨媒体展开

| 时间 | 事件 |
|------|------|
| 2020年 | 《偶像大师 星耀季节》情报公开——伊织为Visual专精型 |
| 2021年7月 | **《碧蓝航线》×《偶像大师》联动**——伊织作为战列舰娘实装 |
| 2021年10月 | 《偶像大师 星耀季节》发售 |
| 2022年 | 《偶像大师 百万现场 剧场时光》新活动剧情 |
| 2023年 | **《偶像大师 百万现场》TV动画播出**——伊织以765前辈身份登场 |
| 2023年-现在 | 手游活动与新曲不断追加 |

## 角色弧光总结

| 阶段 | 时间段 | 核心变化 |
|------|--------|---------|
| **初始** | 街机~初代 | 典型的傲娇大小姐，用尖锐外表保护内心脆弱 |
| **转折** | 偶像大师2 | 成为龙宫小町队长，被迫学会领导力和责任感 |
| **升华** | TV动画（2011） | 性格立体化，展现可靠的同伴形象；台风夜泪水展示成长 |
| **成熟** | 剧场版（2014） | 从任性大小姐到可靠前辈的蜕变完成 |
| **传承** | 百万Live~现在 | 作为765前辈指导后辈，完成偶像精神的传承 |
