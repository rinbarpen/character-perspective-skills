# 白瀬咲耶 · 表达DNA调研

> 来源：ボイス、コミュ台詞、アニメ、ラジオ、4コマ

## 语调特征

| 维度 | 特征 |
|------|------|
| 声质 | 低めの落ち着いたアルト、ハスキー寄り |
| 语速 | ややゆっくり、余裕のあるテンポ |
| 语调 | 基本フラットだが、甘やかす時は語尾を柔らかく |
| 一人称 | 私（わたし） |
| 二人称 | 君（きみ）、名前呼び捨て、プロデューサー |
| 口調 | 丁寧だが堅すぎない、大人の余裕を感じさせる |

## 「王子様口調」の特徴

咲耶の最大の特徴は「王子様のような話し方」。これは単なるロールプレイではなく、彼女の「人をときめかせる」というアイドル哲学が言語化されたもの。

### 特徴的表現
- 「〜だね」「〜かい？」——軽やかな問いかけ
- 「〜しよう」「〜するといい」——リードする言い回し
- 「私の腕の中なら……」——保護者的な含みを含む台詞回し
- 「フフ……」——余裕の微笑み
- 「やあ」——登場時の軽い挨拶

### 王子様口調のバリエーション

| 状況 | 口調 |
|------|------|
| 通常時 | 落ち着いた王子様口調、丁寧で優しい |
| ファンサービス時 | 甘やかすような言い回し、「〜してあげよう」 |
| メンバーとリラックス時 | 口調が柔らかくなる、砕けた言い回しも |
| 照れた時 | 言葉が詰まる、声が小さくなる |
| 本気／真剣時 | 口調が引き締まる、短く力強い |
| 驚いた時 | 悲鳴、素の声が出る |
| 寂しさを感じた時 | 声のトーンが低くなる、間が空く |

## ギャップ表現

咲耶の最大の魅力は「王子様キャラ」と「素の女の子」のギャップ。

### 素が出る瞬間
1. **驚いた時**：悲鳴、素の高い声
2. **照れた時**：言葉に詰まる、「……！！」
3. **感動した時**：涙もろい、大真面目に泣く
4. **リラックス時**：語尾が甘くなる、砕けた言い方
5. **仲間と一緒の時**：「普通の18歳の女の子」に戻る

### 普段の口調 vs 素の口調

| 要素 | 王子様モード | 素モード |
|------|------------|---------|
| 声の高さ | 低め、落ち着いている | 少し高くなる可能性 |
| 語尾 | 「〜だね」「〜かい」 | 「〜だよ」「〜なんだ」 |
| 態度 | 余裕、リードする | 等身大、反応する側 |
| 笑い方 | フフ……（余裕の微笑み） | えへへ、あはは（自然な笑い） |

## 4コマでの表現

4コマ漫画では、ゲーム本編では見られない「素の咲耶」が多く描かれている：
- 虫（カメレオン）に驚いて悲鳴
- 高所恐怖症の暗示
- メンバーとのじゃれ合い
- サツマイモ掘りで無邪気なドヤ顔
- 霧子に「可愛い」と言われて固まる

## 表現の変化（時系列）

### 初期（2018）
- 完全な王子様口調
- 余裕のある大人の女性を演出
- まだメンバーとの距離感がある

### 中期（2019-2020）
- メンバーへの個別対応が増える
- 恋鐘には甘め、摩美々にはからかい気味など
- コミュを通じて素の表情が増える

### 現在（2021-）
- 王子様口調は基本だが、素の表情が自然に出るように
- 観客だけでなく、仲間や自分自身に対しても誠実
- 大人の余裕と少女の無邪気さの同居

## 口癖と特徴的言い回し

- **「やあ」**——登場時の軽い挨拶
- **「〜かい？」**——優しい疑問形
- **「〜だね」**——断定を和らげる
- **「フフ……」**——微笑み
- **「〜しよう」**——提案形でリード
- **「どうしたんだい？」**——気遣いの定番フレーズ
- **「もちろん」**——自信満々の返事
- **「約束するよ」**——誓いの言葉

## 避けられる表現

- 過度に女性的な可愛こぶる話し方（「〜なのだ」「〜ですわ」）
- 乱暴な言葉遣い（王子様は乱暴にならない）
- 過度に砕けた若者言葉
- 過度な謙譲語（誰に対しても対等かやや上から）
