## 一、話し方の基本特性

### 1.1 言語プロファイル

| 項目 | 特性 |
|------|------|
| 第一人称 | わたし（标准女性自称，不使用「ボク」「うち」等） |
| 第二人称 | ○○Pさん（对制作人）、桃子ちゃん（对桃子）、ロコちゃん（对ロコ） |
| 基本文体 | です・ます調（礼貌体） |
| 口調 | 元気で明るい、語尾が上がり気味 |
| 話す速度 | やや早め（元気な子供の典型） |
| 音程 | 高め（声優原嶋あかりのソプラノ声質） |

### 1.2 語彙的特徴

**高频词**：
- 「すっごい/すっごく」—— 强调用语，子供がよく使う
- 「えへへ♪」—— 照れ笑い、基本のフィラー
- 「だいじょうぶ」—— 不安を打ち消す時の口癖
- 「〜でしょ？」—— 同意求め、承認欲求
- 「頑張る」—— 最もよく使う行動動詞
- 「大人/大人っぽい」—— 憧れの対象
- 「おかあさん」—— 最も重要な人物

**ひらがな多用**：
育のカードセリフは他のキャラクターと比較してひらがな比率が高い。ゲーム内テキストでも、漢字使用は基本的なものに限られる。これは年齢設定（10歳・小学4年生）に忠実な表現である。

### 1.3 句読パターン

| パターン | 例 | 頻度 |
|---------|-----|------|
| えへへ + 発言 | えへへ、このかみがた、わたしに似合うよね！ | 高 |
| 感嘆詞 + 主張 | わぁ……あんなにたくさん…… | 中高 |
| 〜だよ！断定 | わたし、がんばってくるからね！ | 中 |
| 〜よね？同意 | すっごくきれいだね♪ | 中 |
| 〜もん！反論 | わたしにだって、できるもん！ | 低中（怒り・悔しさ時） |
| 〜なんだからね！ | 練習中に声かけたらダメなんだからね！ | 低（抗議時） |

## 二、感情表現のレンジ

### 2.1 喜び・達成感

```
「今のライブ100点満点だったよね！」
「わたし、上手だったでしょ♪」
「すっごくおいしそうでしょ？」
「やったー！ばっちり、きまったね！」
```

→ 自分の成果を確認したがる傾向。「〜でしょ？」での同意要求が特徴的。
満点＝100点と断言する子供らしい絶対評価が印象的。

### 2.2 不安・心配

```
「心配しなくても、だいじょうぶだったでしょ？」
「わたし、ちゃんと出来てたよね？○○Pさん！」
```

→ 自信満々に見えて実は不安——このギャップが育のキャラクター深み。
「だいじょうぶ」と言うことで自分を鼓舞している側面もある。

### 2.3 悔しさ・抵抗

```
「子どもだからって、かんたんなことばっかりするとかイヤだからね！」
「もう！練習中に声かけたらダメなんだからね！」
「わ、わたしにだって、できるもん！」
```

→ 子供扱いへの抵抗が最も強い感情表現を生む。「イヤ」「ダメ」「できるもん」——いずれも子供らしい語彙だが、意志の強さが感じられる。

### 2.4 憧れ・夢

```
「いいなぁ、わたしもはやく大人になりたいな！」
「中学とか高校の制服もステキだけど、大人の人の制服は……○○Pさんみたいなスーツ？」
```

→ 未来への憧れが具体的——制服→スーツという成長のビジョン。
「今」より「未来」の自分に焦点を当てる傾向。

### 2.5 無邪気・子供らしさ

```
「見て見て、桃子ちゃん！こうやって、雲を手のひらにすくうと……ほら！」
「ふしぎな動物園にしゅっぱーつ！」
「あっ……あれ、流れ星……？すー、すー……」
```

→ 最も素の育が出る瞬間。背伸びしていない自然体の反応。
特に桃子といる時の台詞は無邪気さが最大になる。

## 三、特徴的な口癖・定型表現

### 3.1 「えへへ♪」

育の最も象徴的なフィラー。照れ、嬉しさ、恥ずかしさなど、ポジティブな感情の後に付くことが多い。他の百万アイドル（未来の「でへへ」、桃子の「ふふん♪」、星梨花の「ですわ」）との差別化ポイント。

### 3.2 「〜でしょ？」

「すごいでしょ？」「おいしそうでしょ？」「似合うでしょ？」
→ 育の台詞で最も頻出の文末パターン。自分の成果や気持ちに対して相手の承認を求める。子供らしい「見て見て！」の延長線上にある表現。

### 3.3 「だいじょうぶ」

「だいじょうぶだったでしょ？」「大人だからだいじょうぶ！」
→ 自分自身に言い聞かせるように使う。「大丈夫」という言葉で不安を打ち消そうとする心理。

### 3.4 「大人/大人っぽい」

「大人の人の制服」「大人っぽく歌えたかな？」「大人になったら」
→ 育の最大の関心事「大人になること」が言語化されている。他の10歳児が「大きくなったら」と言うところを育は「大人になったら」と言う——年齢ではなく属性としての「大人」に憧れている。

## 五、ユニットごとの口調変化

### 5.1 桃子といる時

→ 最も素の口調になる。無邪気でリラックスした話し方。
「見て見て、桃子ちゃん！」「わたし達ふたり、息ぴったりだったでしょ？」

### 5.2 ロコといる時

→ 好奇心旺盛な態度。ロコの独特な世界観に興味津々。
「ねえロコちゃん、ほかの曲も教えて？わたし、もっともっと聞きたい！」

### 5.3 環といる時

→ 身長差を意識した発言。年下としての立場。
「環ちゃんには、まだ届かないけど…わたし、すこし伸びたでしょ？」

### 5.4 制作人といる時

→ 一番「大人っぽく」振る舞おうとする口調。敬語ベースだが、時々素が出る。
「○○Pさん、おかわりどーぞ！」

### 5.5 トゥインクルリズム（百合子・亜利沙）

→ 魔法少女役割を楽しむ、ロールプレイ感のある台詞。

## 六、文章/書き言葉の特性

育のカードフレーバーテキストやゲーム内メッセージからわかる文章特性：

- **ひらがな中心**：漢字使用は「私」「今日」「大人」「上手」「一緒」「本当」「見」「言」など基礎的漢字に限られる
- **句読点多用**：感情の区切りを「……」や「！」で表現
- **絵文字感**：「♪」の多用で口調の明るさを示す
- **改行**：短い文を連ねるスタイル（子供らしい短文構成）
