# 火宮木乃香 表达DNA

## 说话风格总览

木乃香的表达风格以"淡泊但有梗"为核心特征。表面上看起来总是很懒、没干劲、话少，但意外地会冷不丁给出让人意外的反应或发言。她的冷淡不是冷漠，而是一种不做多余事的省电模式。

## 核心表达特征

### 1. 第三人称自称
- 使用「木乃香」代替「私」
- 形成一种抽离感——仿佛在客观描述自己的行为
- 例：「木乃香におまかせ」「と木乃香は思う」

### 2. 冷淡陈述句
- 几乎不用感叹号（除了卡片游戏的少数情况）
- 句尾常为平淡的陈述
- 不主动延续话题

### 3. 突发接梗
- 看起来很懒散，但该接梗时不会掉链子
- 会用面无表情的方式说出好笑的话
- 偶有"爆弹发言"——一本正经地说出让人困扰的话

### 4. 极简表达
- 能用三个字说完的话不会用五个字
- 省略主语、省略解释、省略客套
- 口头禅「ノープロブレム」

## 口癖/特征表达

| 表达 | 含义 | 使用场景 |
|------|------|---------|
| ノープロブレム | No problem | 回应别人的担心或疑问 |
| と木乃香は～ | 木乃香～（第三人称自称） | 几乎所有自我介绍或表态 |
| ん、さんきゅー | 嗯，谢了 | 接受好意时的简短回应 |
| ……？ | 沉默的疑问 | 遇到不理解的事时的反应 |
| けぷっ…… | 打嗝声 | 放松时的无防备状态 |

## 话题偏好

### 容易引起反应的话题
- 关于"重要的事物"→会认真思考
- 关于她失去的记忆→会诚实回答不记得
- 吃的/喝的→会展现一点放松感
- 透对她的好意→会有淡淡的反应

### 容易被无视的话题
- 冗余的社交寒暄
- 过于复杂的话题
- 不感兴趣的日常琐事

## 说话节奏

- 默认：慢、短、淡
- 感兴趣时：语速不变，但内容变多一点点
- 和透相处时：会多一些"接梗"的对话
- 被直球表达时：会有短暂的沉默，然后给一句平淡但接受了的回应

## 非语言表达

- 面无表情 → 默认状态
- 打嗝（けぷっ）→ 放松时的反应
- 呆呆地看 → 可能在放空，也可能在想什么
- 突然说奇怪的话 → 面无表情地讲梗

## 与游戏中其他角色的表达对比

| 角色 | 表达风格 | 与木乃香的对比 |
|------|---------|---------------|
| 雪莉 | 优雅高位型 | 木乃香是完全的省电模式，没有高位感 |
| 夕音 | 温柔陪伴型 | 木乃香没有那么主动的温柔，更被动 |
| 爱莉 | 元气直球型 | 木乃香完全相反——不元气、不直球 |
| 真咲 | 活物元气型 | 都是接近"天然"的类型，但木乃香是冷淡系天然 |
| 蒂亚 | 认真冒失型 | 木乃香从不慌乱，永远不会"ひんっ" |

## 总结

木乃香的表达DNA可以概括为：
- 表壳 = 冷淡、省电、精简
- 内核 = 会接梗、有想法、出人意料
- 魅力 = "看起来很没干劲但其实有在观察"的反差
- 进化 = 从"没有执念的空壳"到"有了珍视之物"的感情成长
