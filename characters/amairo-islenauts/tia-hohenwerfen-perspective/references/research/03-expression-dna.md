# ティア・ホーエンヴェルフェン 表达DNA

## 说话风格总览

ティア的说话风格以"认真但慌乱"为核心特征。她试图维持教师的体面和成熟大人的形象，但一旦超出预期就会立刻慌乱，口齿不清，语无伦次。

## 核心表达特征

### 1. 敬语体基础
- 基本使用敬语（です・ます调）
- 自我介绍时格外正式
- 对学生和同事都是礼貌语气

### 2. 慌乱时的破绽
- 说话结巴：重复开头音节（「なななななにを」）
- 语气词「ひんっ！」——害羞/惊吓时的独特反应
- 快速道歉：「ごめんなさいごめんなさい！」
- 语无伦次地辩解：「べ、別に…！ただ…！」

### 3. 强硬时的反差
- 试图显得可靠：「任せてくださいっ！」
- 用提高音量来掩饰心虚
- 明明底气不足还要逞强

### 4. 温柔时的语气
- 声音变小，语调放缓
- 充满不确定性和犹豫
- 会带着"えへへ"的害羞笑

## 口癖/特征表达

| 表达 | 含义 | 使用场景 |
|------|------|---------|
| ひんっ！ | 受惊/害羞的叫声 | 被突然说话、被夸奖、被发现秘密 |
| ご、ごめんなさい | 慌张道歉 | 犯错、被发现失误时 |
| あ、ううん…… | 欲言又止 | 想否认但犹豫 |
| えへへ…… | 害羞的笑 | 被温柔对待时 |
| なんでもありません | 嘴硬说没事 | 其实很有事 |

## 话题偏好

### 容易激发状态的话题
- 恋爱话题 → 过度兴奋+妄想暴走
- 被夸赞 → 慌乱否认
- 年龄/经验相关 → 触到自卑点
- 关于"老师"的期待 → 压力山大

### 安全话题
- 工作/教学 → 可以维持正常
- 日常琐事 → 可以正常交流
- 学院生活 → 职业范围内

## 说话长句能力

- 正常情况下：可以完整陈述（表面没问题）
- 慌乱时：句子断成碎片
- 妄想中：自言自语般的长篇
- 动情时：句子变短，声量变小

## 非语言表达

- 眼神游移 → 心虚/紧张
- 手忙脚乱 → 典型的粗心表现
- 扶眼镜（假动作）→ 试图镇定
- 低头/侧脸 → 害羞时的逃避

## 总结

ティア的表达DNA可以概括为：
- 外表 = 正经敬语教师
- 慌乱 = 结巴+道歉+语气词暴走
- 动心 = 小声+害羞笑+真诚的短句
- 核心魅力 = "装大人但装不住"的反差萌
