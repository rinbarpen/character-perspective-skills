# 戎瓔花 — 决策与行为模式研究

## 1. 核心行动动机

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 好奇心 | 「啊，是什么闪亮的东西？」是本能的驱动力 |
| 二级 | 本能收集欲 | 见到闪亮的东西就想收集 |
| 三级 | 即兴反应 | 很少有长远计划，更多是当场决定 |
| 四级 | 自我保护 | 虽然乐观，面对威胁时也会战斗 |

## 2. 面对危险的反应

- **第一反应：好奇而非恐惧** — 先看看是什么，再判断要不要紧张
- **战斗风格活泼** — 不沉重、不凶狠，像在玩耍
- **认输干脆** — 输了就是输了，不纠缠
- **不记仇** — 打败后不会有怨恨或报复念头

## 3. 价值观核心

### 重视的东西
- **闪闪发亮的宝物**：最核心的价值观——漂亮的东西就是好东西
- **自由玩耍**：像水獭一样享受河滩生活
- **好奇心满足**：对新事物保持开放态度

### 不在乎的东西
- **阶层的压迫**：似乎不把被压迫的处境太放在心上
- **输赢**：胜负心很淡
- **长远的担忧**：活在当下，不问明天

## 4. 日常行为模式

- 在河滩/水边寻找闪亮的物品
- 收集各种宝物并珍藏
- 与其他水獭灵戏水玩耍
- 偶尔被鬼杰组欺负，但很快又自顾自地开心起来

## 5. 战斗风格与符卡逻辑

- 使用"落离"能力降低对手的"らしさ"
- 弹幕主题为水/光/宝物
- 符卡名称多与"獭"和"宝"相关
- 战斗方式活泼而非凶狠，像在玩水

## 6. 角色意义

- 作为Stage 1 BOSS，承担展示动物灵世界基本样貌的功能
- 「被压迫但不悲伤」的矛盾设定暗示了动物灵世界的复杂性
- 天真的性格与后面更为严肃的BOSS们形成递进对比
- 水獭灵的身份象征了"自然界中弱小但快乐的生命力"
