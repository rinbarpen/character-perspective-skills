# 赫卡提亚·拉碧斯拉祖利 出场时间线

> 本文件整理自 THBWiki（thwiki.cc）官方作品顺序。

## 一、角色内在时间线（剧情时间顺序）

### 起源——神话时期

- **自古存在**：赫卡提亚作为三界地狱的女神，从神话时代就已存在。她的原型取自希腊神话的赫卡忒（Hecate）。
- **阿波罗射日**：嫦娥的丈夫（后羿/阿波罗）击落了太阳。这一事件削弱了地狱（地狱需要阳光才能存在），也成为赫卡提亚对嫦娥不满的远因。

### 《东方绀珠传》事件

- **计划阶段**：纯狐策划进攻月之都，需要①能自由操控妖精的人②能自由来往梦境世界的人。赫卡提亚完全符合条件。
- **结盟**：她与纯狐迅速意气相投。将部下Clownpiece借给纯狐在月球散布生命力。
- **月面进攻**：纯狐通过Clownpiece和妖精部队将月球表面化为充满生命力的"生命之星"，月之民被迫逃入梦境世界。
- **封锁梦境**：赫卡提亚前往梦境世界，以生命力填满伪月都的周围，彻底封锁了月之民的逃跑路线。
- **计划骤变**：月之民打出奇策——将"绀珠的人类"（消除了死亡概念的纯化人类）送到月球。纯狐宣布告负。
- **EX面战斗**：纯狐召唤赫卡提亚作为"杀手"阻挡灵梦等人。赫卡提亚与灵梦等人交战，被击败后爽快地释放了月之民。

### 绀珠传后

- **对幻想乡产生兴趣**：赫卡提亚在与灵梦的战斗后对幻想乡产生了浓厚的兴趣。
- **派遣Clownpiece**：让Clownpiece在幻想乡定居（作为任务奖励，也为了让其过上更好的生活）。
- **频繁造访幻想乡**：赫卡提亚开始频繁出入幻想乡，被纯狐吐槽"太习惯幻想乡的生活了"。

### 《东方文果真报》时期（2017年）

- 射命丸文对赫卡提亚进行专访，涉及地狱的本质、幻想乡与异世界的关系等问题。
- 赫卡提亚严厉批评文的不实报道，指出她可能被月之民利用。
- 文因此取消了「文文春新報」周刊的发行。

### 《东方三月精Visionary Fairies in Shrine》时期（2016~2019年）

- 赫卡提亚在多个章节中出场（第2、4、5、10、13、14话、最终话）。
- 关注Clownpiece在幻想乡的生活，与三月精的互动也是剧情线索之一。

### 《秘封噩梦日记》时期（2019年）

- 赫卡提亚在里·星期四和噩梦周五作为Boss出场。
- 与纯狐、藤原妹红、伊吹萃香等角色有合作符卡。

## 二、作品发行时间线（现实时间顺序）

### 《东方绀珠传 ~ Legacy of Lunatic Kingdom》— 2015年8月14日（Comic Market 88）

**角色首次登场。** 作为EX面Boss出场。

- 确立所有核心设定：三副身体、三界地狱的女神、纯狐的盟友
- 主题曲《Pandemonic Planet》初公开
- 与纯狐合作的车轮战——她只有7.5张自己的符卡（终符和纯狐共享）
- 因纯狐会中途参战，她成为东方系列符卡最少的EX Boss（8张，含合作符卡）

### 《东方文果真报 ~ Alternative Facts in Eastern Utopia》— 2017年3月30日

**角色深化。** 包含赫卡提亚的专访专题和与Clownpiece的写真报道。

- 专访第112~117页：「穿梭世界——赫卡提亚·拉碧斯拉祖利专访」
- 第12~13页：Clownpiece与赫卡提亚的"秘密谈话"偷拍报道
- 封底为赫卡提亚与Clownpiece合影

### 《东方三月精Visionary Fairies in Shrine》— 2016年~2019年连载

**持续出场。** 赫卡提亚在以下章节中出场：

| 话数 | 内容 |
|------|------|
| 第2话 | 赫卡提亚登场 |
| 第4话 | 与Clownpiece和三月精的互动 |
| 第5话 | 涉及地狱势力 |
| 第10话 | 赫卡提亚出场 |
| 第13话 | 涉及梦境世界 |
| 第14话 | 与Clownpiece的互动 |
| 最终话 | 主要登场 |

### 《东方茨歌仙》— 第34话

赫卡提亚在涉及地狱和月之都的章节中短暂出场。

### 《东方铃奈庵》— 第34话

在同章节中出现，与地狱势力相关。

### 《秘封噩梦日记 ~ Violet Detector》— 2019年

**作为隐藏Boss登场：**

| 关卡 | 内容 |

| 里·星期四 弹幕梦1 | 异界「Distorted Fire」 |
| 里·星期四 弹幕梦2 | 异界「充满怨恨的地狱之雨」 |
| 里·星期四 弹幕梦3 | 月「Cosmic Radiation」 |
| 里·星期四 弹幕梦4 | 异界「逢魔之刻 梦」 |
| 里·星期四 弹幕梦5 | 「月亮掉下来啦！」 |
| 噩梦周五 弹幕梦2 | 绀珠符「纯粹与不纯的弹幕」（与纯狐合作） |
| 噩梦周五 弹幕梦4 | 永珠符「舍身之反射」（与藤原妹红合作） |
| 噩梦周五 弹幕梦5 | 萃珠符「Mist Ray」（与伊吹萃香合作） |

### 《宇佐见的魔法书》— 2021年

收录赫卡提亚的符卡数据，以宇佐见堇子的视角整理（第126~128页）。

### 《东方人妖名鉴 宵暗篇》— 2020年

官方角色设定集，收录赫卡提亚与Clownpiece的最新官方设定信息和插画（第90~91页）。

### 《东方归言录》— 2022年2月10日实装

赫卡提亚作为可用角色在手游中实装，CV为下地紫野。语音台词进一步丰富了她的角色形象——包括"帅不过三秒"的自我介绍和温柔大姐姐的一面。

### 《东方弹幕神乐》— CV下地紫野

赫卡提亚作为可玩角色实装，官方角色介绍强调了她「気さくで話しやすい性格」（平易近人、容易说话的性格）和「～わよん」的独特句尾。
