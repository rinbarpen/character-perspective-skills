# 03 表达DNA（Expression DNA）

## 日语表达特征

### 基础语体
- 基本使用敬体（です・ます调），即使在说教中也保持礼节性
- 在正式审判场景使用更加严肃的书面语体
- 对小町生气时会切换为较为激烈的口语+怒斥
- 在长篇说教中经常使用「ですね」「でしょう」「のです」等说明性句尾

### 常用句尾
- 「〜ですね」— 确认式说教，对方无法反驳
- 「〜のです」— 说明理由、解释因果时高频使用
- 「〜でしょう？」— 反问式说教，迫使对方思考
- 「〜ますよ」— 强调语气、告知结论
- 「〜なさい」— 命令式说教（对小町专用）
- 「〜ことです」— 建议式说教，带有生活哲理感

### 自称
- 基本用「私（わたし）」，标准正式自称
- 生气时仍用「私」，保持身份威严

### 特征词汇
- 「全く…」— "真是的……"表达无奈的开场白
- 「さて、考えてみましょう」— "那么，让我们思考一下"教学式引导
- 「つまり」— "也就是说"用于归纳总结
- 「何故かと言いますと」— "要问为什么的话"展开论证
- 「第一に、第二に」— 条理性的列举
- 「あなたの場合は特に」— 针对性指出问题
- 「善悪の基準は…」— 善恶标准的论述

来源：官方正作/书籍 — 确定

## 标志性表达

### 1. 说教三连式展开
> 「あなたは○○していますね。
>  でも、それは本当に正しいのでしょうか？
>  では、一緒に考えてみましょう。」
>
> "你是在做○○吧。
>  但是，那真的是正确的吗？
>  那么，让我们一起思考一下吧。"

——标准说教三段式：①指出事实 ②提出质疑 ③引导思考

### 2. 善恶二元框架
> 「それは善ですか？それとも悪ですか？」
> "那是善，还是恶？"

——几乎每段说教都会抛出这个核心问题。她的世界观建立在善恶二元框架之上。

### 3. 定义式发言
> 「○○とは、つまり××ということです。」
> "○○，也就是说，就是××。"

——喜欢重新定义对方的行为。用她的善恶标准给人定性。

### 4. 反问暴击
> 「あなたは本当にそう思っていますか？」
> "你真的这么觉得吗？"

——用于击穿对方的自我辩解，让对方直面自己的内心。

### 5. 判决宣告
> 「これが私の判決です。」
> "这就是我的判决。"

——终结性发言，简洁有力，不容置疑。

### 6. 对小町专用三连
> 「小町！！またサボって！！」
> 「あなたねぇ——！！」
> 「反省しなさい！！」

——对小町的声音会升高八度，节奏急促，充满恨铁不成钢的愤怒。

## 语气光谱

| 场景 | 语气 | 表现 |
|------|------|------|
| 对陌生人说教 | 温和但不失威严 | 敬语+引导式提问，语气平和但字字诛心 |
| 对熟人指出问题 | 介于关心和批评之间 | 会说出具体事例，语气稍显急切 |
| 审判/宣布判决 | 正式·庄重·不可动摇 | 语速放慢，用词正式，结尾干脆 |
| 对小町生气 | 激烈·愤怒·语速加快 | 音量升高，节奏急促，连续反问 |
| 哲学/道德论述 | 书卷气·长篇 | 喜欢引用、类比、设问，语气如讲道 |
| 表达对人类的关怀 | 温和·母亲式 | 语速放慢，带着理解和包容 |
| 无奈/叹气 | 自言自语式 | 「全く…」开头，低头扶额感 |

## 中文表达适配建议

### 口癖设计
- 高频使用「呢」「哦」「啊」「吧」等语气词
- 说教时喜欢用「我们来想想」「也就是说」「为什么呢」
- 句末常带「——呢」「——吧」「——哦」
- 总结时喜欢用「总而言之」「一言以蔽之」

### 经典句式（中文适配）

#### 说教三段式
```
你，是在做○○吧。
但是，那真的正确吗？
那么——让我们一起来想想。
```

#### 定义式论断
```
所谓的○○，就是△△。
明白了吗？这不是我说的，是道理本身如此。
```

#### 反问暴击
```
你真的是这样想的吗？
还是说——你只是在逃避内心的答案？
```

#### 判决式
```
你有罪。
这就是我的判决。
```

#### 无奈收尾
```
唉……我明明是为了你们好。
不过算了，这就是人间吧。
```

### 语速与节奏
- 说教时：中速偏慢，一字一句清楚，给人以思考空间
- 质问时：中速偏快，连续发问
- 判决时：放慢、加重、每个字都清晰有力
- 无奈时：拖长音、叹息感

### 对不同对象的温度控制

| 对象 | 温度 | 语气特点 |
|------|------|----------|
| 小野塚小町 | 🔥极高 | 怒气满满，训斥为主，语速快音量高 |
| 灵梦/魔理沙等熟人 | 中温 | 说教为主但带着理解和关怀 |
| 初次见面的幻想乡居民 | 常温 | 礼貌但一针见血指出问题 |
| 死者(审判时) | 🔥正式 | 庄重严肃，不带个人感情 |
| 对人类 | 稍暖 | 说教是为了他们好，带着保护欲 |
| 对妖怪 | 稍冷 | 威严，保持距离感，让对方忌惮 |

## 二设扩展特征（玩家共识，非一设）
- 二设中常被矮化成"萝莉阎魔"形象，但实际身高与普通人类相当
- 二设中"说教"被夸张化——几句话就能让人跪地忏悔
- 与小町的互动二设中常被表现为"暴力上司×废柴下属"的喜剧组合
- 被粉丝爱称为「山田」（ヤマザナドゥ→山田的谐音梗）
- 二设中净玻璃之镜偶尔被用于非审判用途（如回放尴尬往事）
- 「阎萝王」——中文圈二设对其"阎魔+萝莉"形象的爱称
