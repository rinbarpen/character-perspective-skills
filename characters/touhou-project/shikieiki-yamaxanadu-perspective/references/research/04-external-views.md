# 04 他者视角（External Views）

## 粉丝昵称与认知

### 核心昵称
- **山田（やまだ）**：来自"ヤマザナドゥ"前两个音节的谐音梗，中文圈也广泛使用
- **阎萝王**：中文圈特有——"阎魔"+"萝莉"+"王"的合成词
- **映姬**：名字的简称，用于日常称呼
- **四季大人/映姬大人**：敬称
- **サナダン / 萨那炭**：日文圈爱称
- **シャバダバドゥ**：名字发音的搞笑变体
- **阎魔大人**：正式敬称

来源：玩家共识 — 确定

### 昵称起源
- "山田"是典型的姓氏谐音梗——"ヤマザナドゥ"听起来像"山田さん"
- "阎萝王"起源于中文圈对其萝莉外貌和阎魔身份的反差认知
- 花映塚封面上的剪影就是她——但当时大多数玩家不知道那是谁

来源：玩家共识 — 确定

## 人气与地位

### 人气投票
- 东方Project历次人气投票中排名一般在20-40名之间
- 虽不如灵梦、魔理沙、咲夜等一线角色，但作为"非主要角色"有稳定的人气基础
- 在小町/阎魔系角色中稳居第一

### 在东方世界观中的地位
- 她是少有的"神明级"角色之一——阎魔是管理死后世界的神
- 与同为神明的八坂神奈子、泄矢诹访子、永江衣玖等不同，她的神格是"冥界之神"
- 《六十年不见的紫香花》中，八云紫（公认最强级角色）感应到她的气息后主动撤退，认为"三个人也不是她的对手"——这是其力量在设定中最显著的旁证

来源：玩家共识/官方小说 — 确定

## 常见二设形象

### "说教魔人"
一设中她确实喜欢说教，二设将此极端化为"说教就是她的攻击方式"——
- 能用说教让对手不战而败
- 被说教的人会陷入自我怀疑、跪地忏悔
- 就连妖怪听到她的声音都会条件反射地反省自己有没有做错事
- 战斗方式本身就是说教弹幕

来源：玩家共识 — 二设

### "阎萝王"
- 一设中她的身高其实算是中等偏矮（设定上并未明确但立绘显示不高）
- 二设中大量将其描绘为萝莉体型，与小町形成"高高在上的大人 × 小只暴君"的反差
- 常见场景：小町低头看着矮小的她挨训，画面喜感

来源：玩家共识 — 二设

### "恐怖上司"
- 对小町的严厉在二设中被夸张为性格全开模式
- 常见梗：
  - 小町远远看到她就溜
  - 四季映姬的脚步声等于小町的厄运信号
  - 说教内容从工作态度到人生哲学无所不包
  - 小町背后模仿她说话被抓包

来源：玩家共识 — 二设

### "认真过头的道德洁癖者"
- 二设中她被描绘为：

  | 场景 | 二设反应 |
  |------|----------|
  | 有人闯红灯 | 长篇说教1小时 |
  | 乱扔垃圾 | 上升到人生哲学层面 |
  | 说谎（即使是善意的） | 被判处"道德重整课程" |
  | 工作时间摸鱼 | "你知道这浪费了多少社会资源吗？" |
  | 浪费食物 | 从农民辛苦讲到地球资源 |

- 这种二设虽然夸张但保留了一设核心：她确实相信每件小事都关乎善恶

来源：玩家共识 — 二设

### "映姬妈妈"
- 另一条二设路线将她描绘为慈母型角色
- 虽然嘴上说教严厉，但内心非常关心对方
- 会给犯错的人重来的机会
- 说教的底色是"为你好"而不是"我要惩罚你"
- 这个形象其实更接近一设——她确实是为了人类好才说教的

来源：玩家共识 — 二设

### 与非曲直厅相关的二设

| 要素 | 二设表现 |
|------|----------|
| 是非曲直厅 | 被描绘为巨大的法庭，四季映姬坐在最高处 |
| 悔悟之棒 | 敲一下就能让人痛改前非（物理意义+精神意义） |
| 净玻璃之镜 | 用于回放一切黑历史，本人会觉得无聊但别人觉得很恐怖 |
| 小町 | "上班摸鱼界传奇"，四季映姬的血压调节器 |

### 身高梗
- 一设身高存在争议：花映塚立绘和格斗作像素绘中她的身高比例差异明显
- 二设中"矮"成了固定标签
- 嫉妒比自己高的人（尤其是小町）
- 站在高处说教增加威严感

## 一设与二设辨析要点

| 内容 | 分类 | 说明 |
|------|------|------|
| 身高很矮 | 二设 | 一设未明确身高，立绘间有矛盾，二设固定为矮 |
| 说教是最强攻击手段 | 二设夸张 | 一设确实喜欢说教，但不是攻击技能 |
| 是暴力上司 | 二设夸张 | 一设中只是严厉，不会物理暴力 |
| 对所有人都说教 | 一设基础 | 确实对所有人都会说教 |
| 净玻璃之镜能看一切 | 一设确定 | 但阎魔不是随时都在用 |
| 四季映姬是神明 | 一设确定 | 冥界之神，阎魔 |
| 她非常强 | 一设+官方作品旁证 | 紫主动回避，说明等级很高 |
| 四季映姬×小町是CP | 二设 | 官方只有上下级关系，无恋爱暗示 |

来源：玩家共识 — 确定/二设
