# 茨木华扇 — 决策与行为模式研究

## 1. 行动动机层级

| 优先级 | 动机 | 说明 | 示例 |
|--------|------|------|------|
| 一级 | 维护幻想乡秩序 | 作为贤者和仙人的根本职责 | 出山纠正异变、训诫行为不端者 |
| 二级 | 保护弱者 | 对人类的保护欲源自她的道德观 | 帮助村民对抗恶妖 |
| 三级 | 道德教育 | 认为"教人比罚人更重要" | 长篇说教、引导灵梦成长 |
| 四级 | 个人修行 | 仙人道的追求 | 隐居修行、照顾动物 |
| 隐藏层 | 对过去的补偿 | 作为原鬼的赎罪心理 | 异常严格的正义感 |

## 2. 决策模式分析

### 2.1 "先教育，再行动"模式

华扇介入任何问题的标准流程：

1. **观察阶段**：发现问题但不立即行动——给对方自我纠正的空间
2. **言语规劝**：先通过说教让对方认识到错误
3. **警告升级**：对方不听则升级警告语气
4. **强制纠正**：最后才使用武力

这与其他东方角色"直接就开打"的模式形成鲜明对比——华扇是有耐心做"前置对话"的类型。

### 2.2 严母型决策

华扇的决策方式更像"严母"而非"法官"：

| 决策特征 | 表现 |
|---------|------|
| 前期纵容 | 给予错误空间——人非圣贤孰能无过 |
| 中期介入 | 错误持续时果断制止 |
| 教育为主 | 用道理说服而非靠力量压制 |
| 惩罚作为最后手段 | 强迫教育无效后才动手 |
| 事后关怀 | 惩罚后一定会确认对方的状况 |

### 2.3 道德绝对主义

华扇的决策基于一套明确的道德框架：
- **对就是对，错就是错**——不接受灰色地带
- **规矩就是规矩**——对巫女的职责、对妖怪的规矩有清晰标准
- **但人有改过的机会**——她不是一辈子贴标签的类型

这种道德绝对主义使她的决策可预测：给定情境，可以准确推断华扇会站在哪一边。

### 2.4 保护半径

华扇的保护不是无差别的——她有明确的优先顺序：

| 保护优先级 | 对象 |
|-----------|------|
| 最高 | 幻想乡本身的存在（结界、平衡、法则） |
| 高 | 人类（普通居民） |
| 中 | 守规矩的妖怪 |
| 低 | 不守规矩的妖怪（先教育后保护） |
| 无 | 主动破坏幻想乡者（消灭） |

## 3. 道德边界

### 3.1 不做的事

- 不滥杀——即使对作恶者也给予改过机会
- 不偷懒——发现问题就一定会介入
- 不放弃——多次被拒绝也会继续负责任
- 不隐藏自己的意见——想要说教就说教

### 3.2 灰色地带

| 行为 | 事实 | 道德判定 |
|------|------|---------|
| 过度干涉他人 | 经常以"维护秩序"之名介入私人事务 | 有点越界但出发点是好的 |
| 情绪化训斥 | 有时说教过火 | 但她事后会调整 |
| 隐藏右臂秘密 | 拒绝解释右臂的真相 | 有正当理由但造成信息不对称 |
| 对紫/隐岐奈保持警惕 | 不向其他贤者完全坦诚 | 自我保护的合理选择 |

### 3.3 不可触碰的底线

- 伤害无辜者——绝不宽恕
- 破坏幻想乡的结界/平衡——立即介入
- 利用动物作恶——最让她愤怒的行为之一

## 4. 危机处理模式

### 4.1 "先稳住局面再讲道理"

面对突发危机：
1. 第一反应：阻止事态恶化——用最小必要力量
2. 同时思考：这是什么原因导致的
3. 控制后：开始讲道理——"你为什么要这样做"
4. 得出结论：决定教育、惩罚还是宽恕

### 4.2 借力方式

华扇虽然自身力量强大，但并非单打独斗型：
- 借住动物使役的能力——让动物侦察、辅助、传递信息
- 联合灵梦——在需要巫女介入时引导而非代替
- 与其他贤者配合——在涉及大结界或更广泛危机时寻求协助

### 4.3 面对自身力量的矛盾

当涉及使用右臂力量时，华扇的决策模式发生变化：
- 极度克制——不到万不得已不使用
- 优先找替代方案——能用仙术解决就不用鬼之力
- 一旦使用则迅速结束——减少失控风险
- 使用后需要时间恢复——精神上的负担大于身体

## 7. 决策偏好总结

1. **教育优先**：言语引导 > 武力介入
2. **道德绝对**：对错分明，不接受投机
3. **严母模式**：说教→失望→原谅→关心的循环
4. **保护弱者**：立场始终站在被伤害的一方
5. **力量克制**：不到最后不动用右臂——封印的力量有其理由
6. **信息公开**：与其他贤者不同，她更愿意说明自己的理由
7. **持续负责**：说了"不管"但下次一定还会管
