# 茨木华扇 — 他者评价与角色分析

> 来源：东方人气投票、THBWiki、Pixiv百科、粉丝讨论、各大东方社区

## 1. 其他角色的评价

### 八云紫
（同为幻想乡贤者）
- 紫对华扇的态度：保持表面客气但处处试探的复杂关系
- 两人同为贤者但行事风格迥异——紫暗中布局，华扇公开说教
- 紫认为华扇"太过正经过头了"（茨歌仙暗示）
- 但两人在守护幻想乡的大方向上一致，只是方法不同

### 摩多罗隐岐奈
（同为幻想乡贤者）
- 隐岐奈认识华扇已久，"清楚她的性格和思想"
- 华扇对隐岐奈的行为模式也有准确的预判
- 茨歌仙42话中，华扇对隐岐奈的出现保持冷静——显示了两人之间的长期了解
- 两人之间似乎存在微妙的竞争关系

### 博丽灵梦
- 灵梦对华扇的态度：表面嫌烦实则敬重
- 虽然经常被华扇说教，但灵梦知道华扇的严厉背后是关心
- 灵梦在关键时刻会寻求华扇的建议和帮助
- 华扇称灵梦"有才能"，灵梦也认可华扇的智慧和经验

### 雾雨魔理沙
- 魔理沙对华扇的态度：保持距离的尊重
- 她认为华扇"说教太多了"（茨歌仙中戏言）
- 但在面对涉及魔法或幻想乡正义的问题时，会认真听取华扇的意见

### 伊吹萃香
（原鬼时代的同僚）
- 萃香是华扇在鬼之四天王时期的同僚
- 萃香对华扇的"从鬼变为仙人"持复杂态度——既不反对也不完全认同
- 两人见面时会流露出不同于与人类相处的气息——一种同族间的默契
- 萃香偶尔会以旧称呼叫华扇，触及她的鬼之记忆

### 星熊勇仪
（原鬼时代的同僚）
- 勇仪是旧地狱的四天王之一，与华扇同为山之四天王
- 两人之间有一种"失去的同类"的羁绊
- 勇仪理解华扇的选择，但也惋惜失去了一个同族

### 其他人类村落居民
- 对华扇的评价普遍积极——她虽然严格但不滥用力量
- 村民视她为可靠的保护者
- 但也有少数人认为她"太过干涉他人生活"

## 3. 粉丝共识

### 外冷内热的典范——共识度极高
| 表 | 里 |
|----|----|
| 严厉的说教 | 发自内心的关心 |
| 对秩序的苛刻 | 对幻想乡的深沉爱意 |
| 训斥灵梦不够努力 | 比谁都希望灵梦成长 |
| 看似不近人情 | 养着一大堆小动物 |

### 右臂的猜想——共识度中高
粉丝对华扇右臂的猜测多种多样：
| 理论 | 支持度 | 说明 |
|------|--------|------|
| 封印了鬼之王的力量 | 高 | 最主流解读——她的鬼之力太强必须封印 |
| 其实是鬼臂的替代品 | 中 | 原本的鬼臂被斩后用法力形成现在的右臂 |
| 绷带下有眼睛/刻印 | 中 | 牵扯到茨木童子传说的各种细节 |
| 本身就是另一只鬼 | 低 | 少数派理论——右臂有自己的意识 |

### 仙人 vs 鬼的内心矛盾——共识度高
- 华扇的"正义感"是否是一种对过去鬼族暴行的补偿心理？——粉丝深度讨论常见话题
- 她对秩序的坚持是否源于"我自己曾经是无序的"？——角色深度解读
- 她对灵梦的严格是否是在复制自己从鬼变仙时经历的约束？——人格分析视角

### 鬼之四天王的空白
粉丝们热衷猜测第四位四天王是谁——华扇曾是四天王之一，但不仅仅是"前成员"，而是"离开者"。

### 教师/大姐姐定位——共识度极高
华扇在粉丝心中稳定定位为"幻想乡的教师/严厉的大姐姐"：
- 对灵梦：严格母亲型
- 对魔理沙：头疼老师型
- 对一般角色：说教阿姨型
- 对动物：慈爱奶奶型（反差最大）

### 反差萌的重灾区
华扇是东方角色中反差萌的标志性代表：
- 训斥别人的手同时温柔地摸着怀里的兔子
- 前一秒在发火、后一秒无奈地叹气然后微笑
- 说"不管了"但实际上一定会管

## 4. 主要争议与二设偏差

| 二设标签 | 偏差程度 | 说明 |
|---------|---------|------|
| 永远在生气 | 中到大 | 一设中她不是无时无刻不在说教——有大量的温和时刻 |
| 讨厌人类 | 大 | 恰恰相反——她正是因为关心人类才选择守护她们 |
| 纯仙人/与鬼无关 | 极大 | 一设明确她是原鬼。这不是"可能是"而是"确定是" |
| 右臂随时会暴走 | 中 | 封印是为了控制，但一设并未描述为"随时要爆炸"的定时炸弹 |
| 和老贤者一样都是谜语人 | 中 | 华扇比其他贤者直率得多——她的说教就是她的表达方式 |
| 讨厌紫 | 中 | 一设中两人存在竞争/试探，但并未明确"讨厌" |
| 不喜欢人类村落的热闹 | 中 | 她虽然隐居但频繁出入村落，并未表现出厌恶 |

## 5. 人气数据

东方人气投票（第19回，2024年）：
- 角色排名：稳定中上位（通常在20-40位区间）
- 茨歌仙连载期间人气显著上升
- 心绮楼/深秘录/凭依华的格斗游戏出场也维持了曝光度
- 作为格斗游戏可操作角色有稳定的玩家基础

作为漫画主角，华扇拥有东方角色中极为罕见的"长篇连载主角"身份——这在东方系列中是极高的待遇。
