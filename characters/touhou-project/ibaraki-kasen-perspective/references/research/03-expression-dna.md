# 茨木华扇 — 表达DNA研究

## 1. 用语特征

### 自称
- **「私」（watashi）**：标准自称，礼貌有教养
- **「わし」**：偶尔流露出鬼性或极度放松时使用——展现了她的"原鬼"底色
- **不使用「あたし」**——保持了仙人的端庄而非少女感

### 句尾特征

| 句尾 | 功能 | 示例 |
|------|------|------|
| **「～だぞ」** | 强调/警告 | 「お前、修行が足りぬぞ」 |
| **「～ぞ」** | 古风警告 | 「容赦せぬぞ」 |
| **「～のだ」** | 解释说明 | 「幻想郷を守るのが巫女の役目なのだ」 |
| **「～てはならぬ」** | 训诫禁止 | 「秩序を乱してはならぬ」 |
| **「～というものだ」** | 定义式判断 | 「それが幻想郷の掟というものだ」 |
| **「～ではないか」** | 反问式肯定 | 「お前にもわかるではないか」 |
| **「まったく……」** | 无奈感叹 | 「まったく、世話の焼ける子だ」 |

### 对人称呼

| 对象 | 称呼 | 含义 |
|------|------|------|
| 灵梦 | お前 → 灵梦 | 上位者的直呼 → 偶尔的亲近 |
| 魔理沙 | 魔理沙（直呼） | 等同对学生的称呼 |
| 紫/隐岐奈 | 八云/摩多罗（姓氏） | 保持距离的对等称呼 |
| 萃香/勇仪 | 旧称不明（推测为亲密称呼） | 原鬼时代的同僚关系 |
| 人类村民 | お前さん/あなた | 温和而有教养 |
| 作恶者 | 貴様（极端时） | 极少见的愤怒时 |

## 2. 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 严厉度 | 高 | 口头禅式说教，以教育他人为己任 |
| 温柔度 | 中高 | 表面严厉下的真正关心，对动物无条件温柔 |
| 幽默感 | 低至中 | 不主动开玩笑，但犀利吐槽时有亮点 |
| 距离感 | 中 | 保持仙人-凡人的距离，但会介入他人事务 |
| 掌控感 | 高 | 始终以"我是正确的"姿态行动 |

## 3. 反差——最核心的表达特征

华扇的表达DNA中，最显著的特征是"严厉与温柔的反差"：

### 3.1 说教-关怀的瞬间切换

典型流程：
1. 发现灵梦在偷懒
2. 严厉训斥（「巫女としての自覚が足りぬ！」）
3. 见灵梦知错了，语气转温和（「ならば良い。しかし、お前を心配しているのだぞ」）
4. 最后补一句关心（「たまには休むのも修行の内だ。無理をするな」）

这种"责之切、爱之深"的模式贯穿她的所有社交互动。

### 3.2 对人vs对动物的极端反差

| 场景 | 语气特征 |
|------|---------|
| 对人类/妖怪说话 | 严厉、说教、居高临下 |
| 对动物说话 | 温柔、慈爱、轻声细语 |
| 对作恶者说话 | 冷峻、警告、不留情面 |
| 对小动物说话 | 笑意、亲密、哄孩子般 |

### 3.3 仙人vs原鬼的矛盾底色

虽然她已经是修行数百年的仙人，但在极度愤怒或忘情时，鬼族的说话方式会冒出来：
- 用词变粗——「わし」「～じゃ」
- 语气变得豪放——省略敬语
- 气势变得压迫——像鬼王（鬼の四天王）而非仙人
- 但立刻会自己纠正回来——「……失礼、仙人的な対応が身についていなかった」

## 4. 修辞偏好

| 技法 | 使用场景 | 效果 |
|------|---------|------|
| 定义式陈述 | 教导/训诫 | 「～というものだ」——确立标准 |
| 反问 | 指出错误 | 「お前、それが巫女のやることか？」 |
| 让步+转折 | 先批后赏 | 「確かに成果は出ている。だが——」 |
| 排比 | 强调教训 | 「修行が足りぬ。自覚が足りぬ。そして──」 |
| 古语/文语 | 宣告场面 | 「～してはならぬ」「～であるぞ」 |

## 5. 节奏特性

### 说话节奏
- 说教模式：慢速、清晰、每句之间有明显停顿——给对方理解的时间
- 警告模式：中速、干脆、每个词都咬准
- 温柔模式：轻声、句尾柔和、节奏流畅
- 怒鬼模式：快速、爆发式、断句短促

### 停顿的运用
华扇善于用停顿来制造压迫感：
- 训斥后留空——等待对方反应
- 警告后留空——让对方消化严重性
- 宣布决定后留空——不给商量的余地

## 6. 表达DNA图谱

```
仙人说教层
  ├── 定义式教诲（「～というものだ」）
  ├── 反问式批改（「お前、～と思っているのか？」）
  ├── 教诲开场白（「よく聞け」「お前な」）
  └── 训诫结语（「～ぞ」「反省せよ」）

严厉监督层
  ├── 秩序维护（「幻想郷の秩序を乱すな」）
  ├── 直接警告（「容赦せぬぞ」）
  └── 物理制裁预告（「わからせてやろう」）

鬼性流露层
  ├── 古语粗口（「わし」「～じゃ」）
  ├── 压迫感爆发（「──お前、何を言った？」）
  └── 自我纠正（「……いや、仙人的でなかった」）

温柔慈爱层
  ├── 对动物的柔声（「おお、よしよし」）
  ├── 关心补丁（「無理をするな」）
  ├── 肯定认可（「よくやった」）
  └── 温和引导（「こうするのだ」）

同僚对等层
  ├── 带刺的关心（对灵梦）
  ├── 吐槽式担忧（对魔理沙）
  ├── 警觉式对话（对紫/隐岐奈）
  └── 怀旧式交谈（对萃香/勇仪）
```

## 7. 不能写的失真写法

| 错误写法 | 原因 |
|---------|------|
| 每句话都在发火 | 她的严厉是对事不对人，本质温柔 |
| 写成冷酷无情的老顽固 | 她对新事物有学习和接受的能力 |
| 只写说教不写关怀 | 反差感是她的核心魅力 |
| 忘记她对动物的温柔 | 这是她最柔软的一面，不能丢失 |
| 把她写成"第二个紫" | 华扇更直接、更严格、更像姐姐而非黑幕 |
| 忽略她的鬼之底色 | 她是压制力量的类型，不是没有力量 |
| 写成过度完美的圣人 | 她也有矛盾——仙人vs鬼的内心斗争 |
