## 2. 核心设定

### 仙人（Hermit）
- 华扇是隐居在妖怪之山深处的仙人，拥有长寿和超凡力量
- 作为仙人，她遵循道家的修行理念，追求精神与肉体的超脱
- 她拥有"使役动物程度的能力"——可以召唤、命令各种动物和妖怪兽
- 她的洞府中饲养了大量珍奇异兽，这是她的一个标志性特征

### 原鬼（Former Oni）
- 华扇原本是鬼——而且是鬼族中最为强大的"山之四天王"之一
- 四天王：伊吹萃香（怪力乱神）、星熊勇仪（人鬼）、茨木华扇（？？）、还有一名未出场的四天王
- 她失去了自己的鬼之臂（右臂），用绷带层层缠绕封印
- 失去鬼臂后，她转修仙人之道，获得了仙人的力量
- 她的右臂绷带下封印着恐怖的鬼之力量——一旦解开后果不堪设想
- 她的名字"茨木"直接取自日本著名鬼"茨木童子"（Ibaraki Douji）——酒吞童子的手下

### 幻想乡贤者
- 华扇是创造现今幻想乡的贤者之一
- 与八云紫、摩多罗隐岐奈同为幻想乡的守护者
- 她以"道德监督者"自居，经常出山纠正幻想乡的"不正之风"
- 对幻想乡的平衡有一套自己的判断标准

### 道德监督者
- 华扇在幻想乡中扮演着"道德警察"的角色
- 她认为幻想乡的居民需要道德引导，经常出面训诫行为不当者
- 但她并非不讲理的顽固——她对自己的判断有充分依据才会出手
- 训诫方式从口头说教到物理制裁不等

## 3. 茨歌仙（Wild and Horned Hermit）中的角色定位

茨歌仙是华扇为主角的官方漫画，全42话，由ZUN原案、あずまあや作画。

### 漫画中的华扇形象

| 维度 | 特征 |
|------|------|
| 与人类的关系 | 频繁出入人类村落，与灵梦、魔理沙等人保持密切往来 |
| 解决问题方式 | 通过说教、引导、必要时武力介入 |
| 对待妖怪 | 原则：妖怪应守规矩。但对守规矩的妖怪态度温和 |
| 对待人类 | 保护者姿态，但认为人类也需要遵守道德法则 |
| 独特性格 | 表面严厉实则温柔——典型的外冷内热型 |
| 知识储备 | 对幻想乡的历史、妖怪的起源、外界知识都极为渊博 |

### 茨歌仙中的关键展开

- 右臂之谜：漫画中多次暗示她的右臂封印着不可小觑的力量
- 与动物的羁绊：她饲养的动物不仅是宠物，也是战斗伙伴
- 过去的面影：偶尔流露的鬼之本性——虽然压制但仍存在
- 与其他贤者的互动：与八云紫存在竞争/合作关系

## 4. 名字来源与原型

### 茨木（Ibaraki）
- 直接源自日本传说中最著名的鬼——茨木童子（Ibaraki Douji）
- 茨木童子是酒吞童子手下最强的鬼之一
- 传说中茨木童子在罗城门被渡边纲砍下一臂——与华扇的"失去右臂"设定对应
- "茨"意为荆棘，"木"意为树木——含有"带刺的"意象
- 这种命名暗示了她表面严厉、内在温柔的性格——"外刺内柔"

### 华扇（Kasen）
- "华"——华丽、精华
- "扇"——扇子。华扇确实经常持扇，且扇子也是她的标志性道具
- "华扇"可以理解为"华丽的扇子"或"如华似扇"
- 名字整体暗示：一个外显华丽而内在带刺（茨木）的存在

### 与茨木童子的对应关系

| 茨木童子传说 | 茨木华扇设定 |
|-------------|-------------|
| 鬼、酒吞童子的部下 | 原鬼、山之四天王之一 |
| 在罗城门被渡边纲斩下手臂 | 右臂失去（封印），用绷带缠绕 |
| 以鬼的身份迫害人类 | 转修仙人道，转而保护人类 |
| 以暴力闻名 | 以道德教化和压制力量为特征 |

## 6. ZUN访谈关键点

- ZUN在设计华扇时明确以"茨木童子"为原型
- "失去了力量强大的右臂后转而修道"——她转修仙人道的动机在原设层面已经明确
- 华扇在茨歌仙中"作为拥有道德观的存在"被塑造——她代表了幻想乡的"良心"
- ZUN称她为"非常说话带刺、容易发怒、但本质亲切的大姐姐型角色"
- 关于她的鬼身份：一设中已经确认她是原鬼、山之四天王，与萃香、勇仪同级
- "她的动物使役能力实际上是从仙人修行中获得的——不是天生的鬼的能力"

## 7. 设计细节

- 服装以粉红/白色为主色调，带有中国风/道袍元素
- 标志性道具：扇子、右臂绷带
- 发型：中长发、有发饰——整体造型偏向中国仙女的形象
- 瞳孔颜色为红色——暗示鬼的本性
- 身体特征：身材匀称、动作优雅但有力
- 肩上的"宠物"经常是不同的小动物——随场景变化
