## 一、正作STG登场

### 1.1 东方花映冢（TH09）— 2005年

**首次登场作品**

| 项目 | 内容 |
|------|------|
| 作品编号 | TH09 |
| 发行年份 | 2005年8月14日（C68） |
| 登场形式 | 可选自机角色（8名可选角色之一） |
| 定位 | 在花之异变中四处采访的角色 |
| 主题曲 | 风神少女 |

**剧情定位：**
花映冢中，文作为"正在寻找新闻素材的记者"参与花之异变。她的路线剧情中，她不是解决异变的主力，而是以"报道异变"的姿态出现。

**关键对话（来源：TH09 射命丸文路线）：**
> 「每年都有各种各样的异变，这次的花之异变正好可以当成特辑来写呢。」

**花映冢中的意义：**
这是文在东方Project的首次登场，奠定其"记者"的核心身份。花映冢本身的"对战弹幕"模式与文的"采访"主题形成了有趣的呼应——拍照、收集、报道。

（来源：THBWiki — 东方花映冢/射命丸文）

### 1.2 东方风神录（TH10）— 2007年

**作为Boss登场（重要）**

| 项目 | 内容 |
|------|------|
| 作品编号 | TH10 |
| 发行年份 | 2007年8月18日（C72） |
| 登场形式 | Stage 4 Boss + Stage 5 中Boss |
| 定位 | 妖怪之山的守卫、采访者 |
| Stage 4 主题曲 | 妖怪之山 ～ Mysterious Mountain |
| Stage 5 主题曲 | 少女所见的日本原风景 |

**Stage 4 剧情：**
文在妖怪之山阻拦前往守矢神社的自机。她的理由是"采访优先，守卫第二"——这与其他守卫角色形成了鲜明对比。

**战斗形式：**
- Stage 4：作为该Stage的Boss登场
- Stage 5：作为中Boss再次登场

**剧情意义：**
风神录中的文是玩家对她的"第一次Boss战"体验。她在这里展现的特征——礼节的语气+强硬的行动+新闻执着——成为此后文中所有作品的基础人格模板。

（来源：THBWiki — 东方风神录/Stage 4）

## 二、弹幕摄影游戏登场（文的专属系列）

### 2.1 东方文花帖（TH09.5）— 2005年

**以文为主角的摄影游戏（核心作品）**

| 项目 | 内容 |
|------|------|
| 作品编号 | TH09.5 |
| 发行年份 | 2005年12月30日（C69） |
| 登场形式 | 唯一操作角色 |
| 游戏类型 | 弹幕摄影 |

**游戏机制：**
玩家操纵文，在弹幕中躲避的同时拍摄目标角色的照片。拍摄效果取决于：
- 拍摄距离（越近越好）
- 画面中的目标比例
- 是否拍到弹幕的精彩瞬间

**游戏内独白：**
每关开头和结尾有文的旁白，展现她对各个角色的"采访评价"。这些旁白是理解文作为记者的核心材料。

**可拍摄角色：**
共9个Level，涵盖幻想乡的主要角色（灵梦、魔理沙、咲夜、蕾米莉亚、幽幽子等）

**剧情意义：**
这是东方Project唯一一款将"摄影"作为核心游戏机制的作品。文作为记者的身份与游戏机制完美融合，奠定了"文=相机+采访"的不可分割的设定关联。

（来源：THBWiki — 东方文花帖）

---

### 2.2 东方文果真报（TH16.5）— 2017年

**文与姬海棠极双主角（重要作品）**

| 项目 | 内容 |
|------|------|
| 作品编号 | TH16.5 |
| 发行年份 | 2017年8月11日（C92） |
| 登场形式 | 可选角色（文/极双主角） |
| 游戏类型 | 弹幕摄影（续作） |
| 主题曲 | 新闻的真实性 |

**游戏机制：**
在文花帖的基础上增加了"新闻的真伪判断"机制。玩家需要：
- 拍下弹幕照片
- 选择"真实"或"虚构"作为新闻标题
- 根据选择获得不同的评分

**双主角制：**
- 射命丸文：追求速度和戏剧性的报道风格
- 姬海棠极：追求准确性和深度的报道风格
- 游戏可以在两人之间切换

**关键剧情：**
- 两人首次在正式作品中同台登场
- 展现了两种新闻理念的碰撞
- 对话中极对文的报道风格进行了多方位的吐槽

**剧情意义：**
这是文在2005年文花帖之后的"正式续作"，展示了文在12年间的角色发展——从"单一的八卦记者"到"与同行交流的职业记者"。

（来源：THBWiki — 东方文果真报）

## 六、登场密度分析

### 高频期（2005-2009年）

这是文出场最密集的时期，原因在于：
- 角色刚被创作出来，ZUN在多个作品中对其进行尝试
- 风神录（妖怪之山主题）与文（天狗角色）的搭配是自然的
- 摄影游戏（文花帖）的成功促使了更多出场机会

**作品数：** 6部（花映冢、文花帖（游戏+书籍）、求闻史纪、风神录、绯想天、非想天则）

### 中频期（2010-2017年）

出场频率有所降低，但仍是重要角色：
- 在格斗作中持续登场（凭依华）
- 获得文果真报（第二款摄影游戏）
- 在官方漫画中保持存在感

**作品数：** 5部（茨歌仙、求闻口授、铃奈庵、文果真报、凭依华）

### 低频期（2018年至今）

出场减少，但仍保持存在感：
- 在格斗作（刚欲异闻）中登场
- 在官方漫画（智灵奇传）中活跃
- 在STG中从主要角色转为背景提及

**作品数：** 4部（智灵奇传、虹龙洞、刚欲异闻、兽王园）

## 七、时间线中的角色变化

| 时间段 | 角色定位 | 核心特征 | 变化 |
|--------|---------|---------|------|
| 2005-2006（初期） | 单纯的新闻记者 | 好奇心、速度、报道 | 角色设定定型期 |
| 2007-2009（风神录前后） | 双重身份（天狗守卫+记者） | 天狗的责任感得到表现 | 增加了"守卫妖怪之山"的维度 |
| 2008-2009（格斗作） | 战斗员+记者 | 在格斗中展现了更多性格面向 | 战斗中的"采访第一"特征成型 |
| 2010-2017（漫画期） | 事件参与者 | 从旁观者到轻度参与者 | 与其他角色的互动更加丰富 |
| 2017（文果真报） | 同行交流中的记者 | 展现了与极的新闻理念碰撞 | 角色深度增加——有了同行间的自我反思 |
| 2020-至今 | 幻想乡的情报枢纽 | 不再仅是"八卦记者" | 被定位为幻想乡信息生态的一部分 |
