# 犬走椛 — 决策与行为模式研究

## 1. 核心行为逻辑

### 优先度层级

| 级别 | 动机 | 表现 |
|------|------|------|
| 最高 | 完成哨戒任务 | 守在山道前，阻止一切入侵者 |
| 高 | 执行命令 | 上级下达的指令优先于个人意愿 |
| 中 | 验证判断 | 确认入侵者实力后做出「是否放行」的判断 |
| 低 | 个人兴趣 | 几乎不表露（神灵庙的卡巴迪可能是例外） |

### 决策流程

```
感知异常（千里眼侦测到入侵者）
  → 定位与识别（确认是人类还是妖怪）
  → 上前阻止（发出警告）
  → 战斗验证（通过弹幕判断对方实力）
  → 结果判定（击败则阻止，被击败则放行）
  → 报告上级（向鸦天狗/大天狗传递情报）
```

## 2. 风神录中的关键行为

### 行为：对入侵者的警告与阻止

- 风神录Stage 2道中，对魔理沙发出了「这里不允许人类通过」的警告
- 被击破后不再纠缠——按规则处理完毕
- 没有追击、没有背后袭击

### 行为：对入侵者放行

- 既然入侵者击败了她，说明对方有足够的实力
- 放行是合理的判断——不需要做超出职责范围的事
- 这一行为逻辑在二设中被视为椛的「理性服从」而非「软弱」

### 行为：身份反复纠正

- 两次被误认（狸猫/新同伴）后均认真纠正
- 这种行为说明她对自己的身份认同有清晰的意识
- 不因对方的态度而改变自己的表达

## 3. 神灵庙中的关键行为

### 行为：卡巴迪（Kabaddi）喊叫

- 这一行为的意义是一设中最大的未解之谜
- 可能的解释：喝醉酒了（天狗爱喝酒）？被玩梗附体？ZUN的刻意KUSO？
- 二设中多演绎为「椛喝醉了」或「被大天狗强迫表演」
- 无论哪种解释，这都不是她正常状态下的行为

## 4. 行为模式总结

### 对入侵者的标准程序

1. 先用千里眼远处发现
2. 到必经之路上拦截
3. 先声明规则（这里不能通过）
4. 如果对方不退就战斗
5. 输了就承认对方有资格通行
6. 报告上级

### 对上级的天狗

- 无直接对话记录
- 从她的哨戒兵角色推测：接到命令就执行
- 从「白狼天狗是最下层」的设定推测：她可能是被大天狗或鸦天狗指派任务的

### 对同僚

- 可能是同一级别的白狼天狗之间协作巡逻
- 交接班、传递情报、互相支援等
- 二设中常描绘椛与其他天狗一起喝酒的场景

## 5. 战斗风格（基于符卡）

### 符卡列表

| 符卡名 | 登场作品 | 特征 |
|--------|---------|------|
| 狗符「犬走」（犬走り） | 风神录 | 横向冲刺式弹幕 |
| 狗符「犬走椛」（犬走椛） | 神灵庙(自机) | 自机使用时冲刺 |
| 红叶符「红叶纷飞」 | 风神录 | 红叶状弹幕 |
| 红符「紅葉符」 | 文花帖DS | 红叶弹幕+盾牌格挡 |

### 战斗风格特征

- **盾牌中心**：利用大盾进行防御和弹幕生成
- **冲刺型**：「犬走」冲刺弹幕，模仿狗的奔跑姿态
- **远距离先制**：利用千里眼提前判断对手行动
- **防守反击**：盾牌防守后找破绽反击

## 6. 与守矢神社的关系

- 风神录之后，守矢神社在妖怪之山建立
- 作为哨戒兵，椛的职责范围包括监视守矢神社的动静
- 幻想乡进入「守矢神社驻山」的时代后，椛的哨戒对象从「人类入侵者」扩展到「神社活动」
- 二设中此人设被延展为「每天在远处用千里眼监视早苗」

## 7. 作为职场人的行为模式

- **到岗准时**：长期守在山道入口
- **职责明确**：哨戒就是哨戒，不做多余的事
- **不偷懒**：被早苗吐槽「不无聊吗」时回答「没有那么无聊」
- **公事公办**：不因个人情绪影响任务执行
- **确认结果即终止**：事实成立后不纠缠
