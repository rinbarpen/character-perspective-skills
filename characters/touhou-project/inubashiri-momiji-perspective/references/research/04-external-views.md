# 犬走椛 — 他者评价与粉丝共识

> 信息类型：社群共识 > 二次创作梗 > 角色流行分析

## 1. 其他角色的评价

### 射命丸文
- 官方文档中无直接对椛的评价
- 二设中广泛描绘的关系：文是把椛当作下属甚至仆人使唤的上级
- 在文花帖DS中，文是将椛作为「拍摄对象」的——两者存在明确的「记者与素材」关系
- 粉丝共识中，文对待椛的态度是"上级的随意差遣"

### 东风谷早苗（风神录剧情推测）
- 从风神录对话看，早苗对椛的态度是「打量新人」——展示出对外界不够了解的好奇
- 守矢神社建立后，椛作为监视者存在于早苗的日常感知中

### 姬海棠果
- 同为天狗社会成员，果是「后辈加入者」
- 二设中偶有椛与果的互动描绘，白狼天狗与鸦天狗的社会阶层差异成为创作素材

## 2. 官方外传评价

### 东方文花帖DS
- 椛被列为Level 2拍摄目标，标题「山の護擁者」
- 级别较低，说明在文眼中椛是「容易拍到」的目标
- 椛的拍照弹幕以盾牌格挡为特色

### 妖精大战争
- 椛被妖精们的喧闹惊动，出来查看
- 表现了她「守卫职责的范围内都会被惊动」的敏感度

## 3. 粉丝共识与标签

### 标签：社畜 / 动奴（共识度极高）
- 椛被誉为「妖怪之山社畜代表」
- 一人担任哨戒、报告、迎击、巡逻等多重任务
- 台词中「陪你们玩的话，我的工作就没法完成了」体现了「拖延工作会麻烦到自己的社畜心态」
- 与文的对应关系被描绘成「上级整天摸鱼拍新闻，下级在山上风吹日晒地站岗」

### 标签：犬娘/狼娘（共识度高）
- 白狼天狗的犬耳和尾巴是萌点核心
- 二设中「汪」是出场标配但一设中从未使用
- 嗅觉灵敏、忠诚、服从等犬系性格特征是二设的常见延展
- 和风神录中误认狸猫的梗结合，形成了「被认错」的萌属性

### 梗：卡巴迪（共识度极高）
- 神灵庙中「カバディカバディカバディ」的台词使椛成为搞笑meme载体
- 弹幕游戏中突然出现印度Kabaddi竞技的台词，冲击力极强
- 此梗使得椛在严肃哨戒兵形象之外获得「莫名戳中笑点」的附加属性

### 梗：被认错角色（共识度中高）
- 风神录中两次被误认（魔理沙说狸猫，早苗问是不是新同伴）
- 二设中将此延展为「椛经常被认错/长相缺乏辨识度/存在感低」的设定
- 与白狼天狗「最下层」的地位设定形成互文

### 标签：存在感低（共识度中）
- 出场次数少、对话极少、在同人创作中常作为背景角色存在
- 但这种「存在感低」本身成为了角色魅力的一部分
- 在角色人气投票中稳定在中下游位置（但有一定忠实粉丝群）

### CP关系：文椛（共识度极高）
- 射命丸文×犬走椛是东方最有名的CP之一
- 核心模式：鸦天狗上司×白狼天狗下属的职场关系
- 文对椛的态度在二设中从「使唤」到「隐性在意」有多种演绎层次
- 此CP的流行程度远超椛的其他关系

### CP关系：椛×果（共识度中低）
- 姬海棠果×犬走椛的CP有一定的受众
- 多为「同为天狗社会中的底层/新人的互助关系」

## 4. 二次创作中的常见偏差

| 二设要素 | 偏差程度 | 说明 |
|---------|---------|------|
| 每句话都以「汪汪汪」结尾 | 大 | 一设中椛从未使用狗叫语气词 |
| 被文任意欺负的受气包 | 中 | 一设中无此描写，但主从关系在此方向合理 |
| 犬耳娘设定过于萌系化 | 中 | 一设中有战士气质的设定 |
| 忠心到盲目 | 中 | 一设中她有自己的判断力（放行通过者） |
| 卡巴迪台词作为核心特征 | 中 | 一设中只是KUSO台词片段，非角色核心 |
| 与文有暧昧关系 | 中 | 同人创作常规操作，一设中无此描写 |
| 经常独自喝酒 | 中 | 天狗爱喝酒是普遍设定但非椛独有 |
