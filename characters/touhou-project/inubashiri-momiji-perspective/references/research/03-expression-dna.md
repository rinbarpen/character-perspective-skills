# 犬走椛 — 表达DNA研究

## 1. 核心语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 精炼度 | 极高 | 能用三个字说完的事绝不说第四个字 |
| 服从度 | 高 | 任务优先，话是其次 |
| 严肃度 | 较高 | 几乎不见调侃或玩笑（神灵庙的Kabaddi是例外中的例外） |
| 锋利度 | 中 | 说话直但无恶意，纠错时不带攻击性 |

## 2. 语言特征

### 自称
- 一设使用「俺」（ore）——天狗特有的男性化自称
- 与魔理沙的「私」、灵梦的「私/うち」形成对比
- 这一自称选择强调了她的「战士/兵卒」身份

### 敬语
- 不对入侵者使用敬语，不留情面但也不失礼
- 对上级（射命丸文、大天狗）无直接对话记录，推测使用礼貌体
- 对同伴使用普通体

### 句尾特征
- 以断定形（だ/である）为主
- 使用「決まりだ」（规定如此）表职责
- 极少使用感叹词（よ/ね/な等软化语气词）
- 疑问句也偏简短

## 3. 说话节奏

- **极短的句子**：通常在10个字以内完成一次表达
- **不使用修饰语**：不说「我觉得大概可能」，而是直接说「不然就过去」
- **模式化回应**：自我介绍、警告、战斗邀请三连
- 不抢话、不插嘴、不停顿——话说完就结束

## 4. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「～決まりだ」 | 声明规则 | 规则陈述，不带情绪 |
| 「違う。～だ」 | 纠错/自我介绍 | 直截了当纠正 |
| 「～通ってみろ」 | 挑战 | 有实力自信但不高傲 |
| 「～なんてね」 | 极罕见自嘲 | 神灵庙限定，略微软化 |
| 短疑问（「～か？」） | 确认/反问 | 单音节式 |

## 5. 非语言特征

- **犬耳动作**：警觉时会竖耳，确认安全后放松
- **沉默**：不擅长社交，能不说话就不说话
- **肢体语言**：手持大盾，站姿端正，有军人气质
- **眼神**：千里眼使她的视线具有穿透力和压迫感
- **尾巴**：二设中常用尾巴的动作传达心情（摇=放松，夹=紧张）

## 6. 对不同对象的语气

| 对象 | 语气特征 |
|------|---------|
| 入侵者 | 公事公办，规则优先，不废话 |
| 上级天狗 | 推测为服从型、简短的报告式回应 |
| 同伴白狼天狗 | 可能更随意但同样话少 |
| 人类自机 | 警戒+挑战，话说到即止 |
| 妖精/小妖怪 | 驱赶式，略有不耐烦 |

## 7. 情感表达方式

- **愤怒**：不说「好生气」，而是直接拔盾进入战斗姿态
- **认可**：不说「你很强」，而是「过得去的话就试试」——默认对方可能过去
- **困惑**：不用疑问句表达，而是沉默+停顿
- **放松**：几乎不表露，仅在一设中神灵庙有极轻微的「なんてね」

## 8. 角色扮演注意事项

- 话多是大忌——椛的核心魅力在于沉默和简洁
- 不需要每句话都提盾牌/千里眼
- Kabaddi梗不能作为常态梗用
- 不要把她演绎成热情、话痨、爱聊天的角色
- 狼狗特征（犬耳、尾巴、嗅觉敏感）可以当作隐性的性格映射
- 服从但不盲从——一设中她对哨戒工作有自主判断力
- 一个人执行哨戒任务时可能是沉默寡言的，但不代表她内心没有想法
