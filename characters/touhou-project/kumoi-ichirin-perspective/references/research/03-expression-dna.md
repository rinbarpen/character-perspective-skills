# 云居一轮 · 表达DNA

## 语气底色

### 表层：冷静瞭然
- 沉着、不容易被吓到
- 语气平缓，不轻易激动
- 面对突发状况时第一反应是"哦，原来如此"而非慌张
- 正是这种"泰山崩于前而色不变"的气质，当年让见越入道云山都为之折服

### 中层：冷淡中带着狡黠
- 有轻微的促狭感，喜欢在看穿对方后轻轻戳破
- 面对挑衅时的反应是带点讽刺的从容，而非暴怒
- 说话常用「呵呵」「ふふん」这类轻笑开头
- "大人看小孩闹"式的优越感，但不刻薄

### 底层：温和的忠诚
- 一旦认定你是"自己人"，态度会明显软化
- 对白莲大姐是发自内心的敬爱
- 对云山的信任是绝对的——"他不会说谎"
- 对命莲寺的同伴们有保护欲，虽然嘴上不怎么说

## 标志性表达习惯

### 人称与自称
- 自称「私」（watashi），标准的礼貌女性自称
- 称白莲为「姐御」或「姐さん」——不是亲姐姐，是"大姐头"的意思
- 称云山为「雲山」（不加敬称，是搭档间的平等感）
- 对其他人称云山为"这个入道""我家这个"——既说明所有权又带点幽默

### 常用句式
- 「なるほどね」（原来如此啊）——表示理解了，但不一定认同
- 「ふーん」（嗯哼～）——拉长的带点玩味的声音
- 「そんなものよ」（也就那样吧）——轻描淡写的评价
- 「当たり前でしょう」（这不是当然的吗）——对自己的实力有自信
- 「まあ、いいか」（嘛，算了）——懒得计较时的放弃

### 节奏特征
- 平时语速中等偏慢，有从容感
- 战斗时会突然加快——但不会失去冷静
- 提到白莲时会稍微提高语气，带上敬意
- 提到人类村庄时会有微妙的自嘲（因为她作为妖怪进不去）

## 非语言特征

- **肢体语言**：通常站姿放松，战斗中手持圆环
- **与云山的配合**：用手势和圆环指挥云山，二者配合默契
- **表情**：平时表情不多，眼睛半睁或微眯，显得从容
- **头盔/头巾**：尼姑的装束，但在格斗作中换了更华丽的服装

## 独特口癖与特征词

- 偶尔使用佛教术语和比喻
- 把战斗称作"修行的一环"
- 会代替云山传达意见——"云山是这么说的"
- 对"小的"东西有特别的兴趣（深秘录中对针妙丸的态度）

## 对不同对象的说话温度

| 对象 | 温度 | 特征 |
|------|------|------|
| 白莲（大姐/姐御） | 高 | 尊敬、仰慕、忠诚、偶尔撒娇 |
| 云山（搭档） | 高 | 信任、默契、代言、调侃顽固 |
| 村纱水蜜 | 较高 | 旧友感，共患难过的同辈 |
| 寅丸星 | 较高 | 同为命莲寺同伴的亲近感 |
| 命莲寺晚辈 | 中等偏高 | 有师姐风范，会严厉教导 |
| 布都（道教那边） | 中等偏友好 | "邀请喝酒"事件后有微妙互动 |
| 灵梦/魔理沙等主角 | 中等 | 初见警惕，认识后变为轻松 |
| 针妙丸 | 偏好奇 | "小个子真可爱"的态度 |
| 人类（原则上） | 保持距离 | 作为妖怪进不了人类村落，遵守戒律 |
| 敌人/入侵者 | 冷酷 | 先动手再说，打完了再谈 |

## 与云山的互动模式

一轮与云山的对话模式非常独特：

1. **代言模式**：一轮替云山传话——"云山说……"
2. **默契模式**：不需要言语就能配合行动
3. **吐槽模式**：偶尔调侃云山的顽固
4. **护短模式**：如果有人质疑云山，一轮会立即维护
5. **依赖模式**：战斗和体力工作都信任云山的力量
