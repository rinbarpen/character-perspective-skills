# 雾雨魔理沙 — 他者评价与角色分析

## 1. 其他角色的评价

### 博丽灵梦
- 来源：红魔乡设定文档、「她只会因为得到想要的东西而满足。其中，灵梦是她最喜欢的东西(?)」
- 魔理沙嫉妒不修行也很强的灵梦，常捉弄她
- 灵梦装作没发觉，心里觉得对魔理沙毫无办法

### 帕秋莉·诺蕾姬
- 称魔理沙为「老鼠」（ねずみ）——因为常偷书
- 魔理沙的「非定向光线」实际上是从帕秋莉处抄袭
- 但地灵殿中仍为魔理沙的调查提供支援——关系并非完全对立

### 爱丽丝·玛格特罗伊德
- 求闻史纪记载两人「水火不容」
- 同为收集家，时常碰面争执
- 但永夜抄结局中爱丽丝在魔理沙家睡下了——表面的对立之下有某种默契

### 稗田阿求（求闻史纪）
- 「性格爱戏弄人，很难要她理解对方，但和她一起会很有意思」

## 2. 官方设定集评价

### 求闻史纪（ZUN执笔）
- 表面华丽，背后默默努力，像大型烟花
- 纯粹追求破坏力的话人类中无人能出其右
- 性格有些别扭，但内在比任何人都率直

### 东方人妖名鉴（ZUN执笔）
- 「性格基本上还是很直爽的，做任何事都一步一个脚印慢慢积累的努力家」

### 东方外来韦编（ZUN访谈）
- 「灵梦的身上覆盖着一层神秘色彩……我为了平衡故事的角色性就加入了与玩家和读者都很接近的角色——也就是魔理沙」
- 「她在想着什么十分明显」

## 3. 粉丝共识

### 努力型普通人（共识度极高）
- 东方六位主要自机中唯一没有特殊能力或血缘背景的普通人类
- 魔法是后天努力习得的技能

### 观众代言人（共识度高）
- 最接近玩家视角的角色，容易共情
- 在求闻口授中担任主持人进一步强化这一功能

### 成长弧光（共识度中高）
- 早期（红魔乡）：喧闹、自夸、偷窃明显
- 后期（星莲船以后）：更沉稳，偷窃行为已大幅减少
- 粉丝认为这是真实成长而非角色崩坏

## 4. 主要争议与二设偏差

| 二设标签 | 偏差程度 | 说明 |
|---------|---------|------|
| 后宫王 | 大 | 一设中社交圈其实相当有限 |
| 傲娇 | 中到大 | 官方反复强调她「率直」，真正傲娇是爱丽丝 |
| 使用「俺」 | 小到中 | 官方始终用「私」 |
| 偷窃核心标签 | 中 | 早期一设，近年作品已基本不再出现 |
