# 雾雨魔理沙 — 表达DNA研究

## 1. 用语特征

### 自称
- 官方：全作使用「私」（watashi），语调粗放自信
- 二创常见「俺」或过度使用「あたい」——均为误解

### 句尾口癖
- **「～だぜ」（DA☆ZE）**：最标志性口癖，强调主张和行动意志
- 「～ぜ」：比「だぜ」更轻快的行动表达
- 「～ぜー」：拖长音版，更轻松

### 对他人称呼
- 对所有角色直呼其名（呼び捨て），不使用敬语
- 常用「お前」称呼敌人/对手

## 2. 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 豪爽度 | 极高 | 说话直来直去，行动先于思考 |
| 自信度 | 高 | 自称「在幻想乡排第二」，对实力有充分自信 |
| 直率度 | 极高 | ZUN称她为「最好懂的角色」，心思一眼可见 |
| 痞气度 | 中高 | 玩世不恭、有小坏心眼，但本质直脾气 |

## 3. 幽默方式

- **直球型幽默**：自吹自擂式（「普通的魔法使大人」）
- **强行合理化**：把偷书说成「借到死为止」
- **嘴硬逞强**：明明有点怕却说不怕
- **吐槽风格**：直接不客气，吐槽完就忘

## 4. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「～って感じしかしないぜ」 | 吐槽/评价 | 随意、直接 |
| 「～だぜ」 | 陈述/决心 | 强调主张 |
| 「～と思うぜ」 | 主张（软化） | 自信但不霸道 |
| 「おいおい、～」 | 惊讶/吐槽开场 | 标志性开场 |
| 「ちっ、～」 | 受挫时 | 咂舌，不服气 |

## 5. 对不同对象的语气

| 对象 | 语气特征 |
|------|---------|
| 灵梦 | 损友互怼，随意调侃 |
| 帕秋莉 | 被抓住把柄会心虚，平时爱说教 |
| 爱丽丝 | 嘴硬逞强，同行竞争感 |
| 敌人 | 挑衅、居高临下 |
| 友善对象 | 好奇+直球提问 |

## 6. 表现规则总结

- 用「私」自称，句子偏短
- 常加「だぜ」在句尾表达主张
- 对所有人简化直呼，不用敬语
- 先抱怨/吐槽再行动
- 自信满满，但不写成纯粹傲慢
- 幽默来自直率和强行合理化
- 受挫时用咂舌/省略号掩饰不甘
