# 圣白莲 — 时间线与角色进化

> 信息来源：官方游戏、设定集、漫画、ZUN访谈

## 1. 千年之前的时代（约1000年前）

| 时间 | 事件 | 备注 |
|------|------|------|
| ~1000年前 | 白莲出生，为一个普通人类女性 | 具体年代不明 |
| ~1000年前 | 弟弟命莲出生 | 身为佛教僧侣 |
| ~1000年前 | 命莲重病，白莲为救弟学习禁术 | 人生转折点 |
| ~1000年前 | 白莲成功治愈命莲，但从此成为魔法使 | 第一个重大选择 |
| ~1000年前 | 命莲去世 | 命运的重击 |
| ~1000年前 | 白莲独自修行，用魔法帮助他人 | 从救人到助人 |
| ~1000年前 | 白莲开始同时保护人类与妖怪 | 立场的独立 |
| ~1000年前 | 人类与妖怪联合将白莲封印在圣辇船中 | 背叛与封印 |
| ~1000年 | 封印持续，白莲在黑暗中度过漫长岁月 | 千年的沉思 |

## 2. 封印期间

| 时间 | 状态 |
|------|------|
| 封印初期 | 痛苦、挣扎、不理解为什么被背叛 |
| 封印中期 | 开始反思，建立自己的哲学体系 |
| 封印后期 | 完全接受，将封印视为修行，规划未来 |
| 最终阶段 | 慈悲与智慧达到成熟，准备面对新世界 |

## 3. TH12 星莲船时代（2009年）

| 事件 | 细节 |
|------|------|
| TH12 星莲船发布（2009年8月） | 圣白莲作为Stage 6最终Boss首次登场 |
| 圣辇船事件 | 灵梦/魔理沙/早苗追踪宝船传说 |
| Lv6 Boss战 | 白莲全力阻止来访者接近「那个人」 |
| 战败解放 | 白莲被击败后解释了千年的真相 |
| 命莲寺建立 | 利用圣辇船残骸建立命莲寺 |
| 收留村纱水蜜 | 同样被封印在船中的水蜜加入 |
| 命莲寺团体形成 | 寅丸星、云居一轮、娜兹玲、幽谷响子相继加入 |

### 角色定位
- 首次登场即为最终Boss
- 拥有完整且独立的背景故事
- 世界观重量级角色设定

## 4. TH13 神灵庙时代（2011年）

| 事件 | 细节 |
|------|------|
| 神灵庙发布（2011年8月） | 白莲作为可选自机第六位登场 |
| 前往神灵庙调查 | 调查圣德太子复活事件 |
| 与霍青娥对话 | 千年存在的平行对比 |
| 与物部布都对话 | 关于信仰与战争的观点冲突 |
| 事件解决 | 确认圣德太子势力不会破坏平衡后离开 |

### 角色进化
- 从Boss到自机的转变
- 展现了作为调查者的主动性和独立性
- 开始与幻想乡其他千年势力产生联系

## 5. TH14 辉针城时代（2013年）

| 事件 | 细节 |
|------|------|
| 辉针城发布（2013年8月） | 白莲再次作为自机登场 |
| 辉针城调查 | 参与解决矮人万宝槌引起的异变 |
| 命莲寺日常化 | 白莲在幻想乡中的存在逐渐日常化 |

## 6. 格斗作时代（2013年-2018年）

### 心绮楼（2013年）
- 以「魔界住持」的身份参战
- 为了了解外界动静参加祭典
- 展现了近战能力和格斗适应性

### 深秘录（2015年）
- 调查都市传说异变
- 对现代外界事物展现出好奇心
- 与其他势力保持合作态度

### 凭依华（2018年）
- 参与凭依异变
- 对灵魂凭依现象的哲学思考
- 展现了命莲寺团体的整体战斗力

### 角色进化（格斗作）
- 从纯粹法师型角色转变为近战魔法结合的战斗风格
- 性格上展现出更多灵活性和幽默感
- 与幻想乡其他势力的互动更加频繁

## 7. TH15-17 新时代（2015年-2019年）

| 作品 | 角色定位 |
|------|---------|
| TH15 绀珠传（2015年） | 支援角色 |
| TH16 天空璋（2017年） | 异变支援角色 |
| TH17 鬼形兽（2019年） | 未登场 |

### 存在感变化
- 在ZUN的主线故事中出场频率下降
- 但在格斗作和漫画中持续有存在感
- 命莲寺团体整体在后续作品中的角色戏份有所减少

- 与其他势力互动的日常

### 《东方铃奈庵》
- 少量涉及命莲寺和白莲

### 《东方醉蝶华》
- 命莲寺团体有登场
- 展现了更多白莲在日常中的一面

## 9. 角色进化总结

### 初登场（TH12）——被封印的魔法使
- 庄严、神秘、背负沉重过去
- 战斗态度：守护者、不留情面
- 语言风格：正式、古风、晦涩

### 自机时代（TH13-14）——守护兽的魔法使
- 开始显现温和与包容的一面
- 与其他角色的对话增多
- 战斗态度：自由选择、主动探索
- 语言风格：佛教典故减少，日常化

### 格斗作时代（心绮楼~凭依华）——魔界住持
- 性格更加丰富：慈悲中带有幽默
- 展现了战斗的热情
- 与幻想乡社会的融合程度加深

### 成熟形态（现在）——命莲寺的圣人
- 完成了从「被封印的受害者」到「庇护者」的身份转变
- 理想主义与务实的平衡
- 在和平的日常中维持着自己的理念

## 10. 未解明的时间线问题

- 白莲与命莲的具体生卒年份未明确设定
- 被封印的具体时长（「约千年」是模糊表述）
- 在封印之前，她与妖怪之山、博丽神社（当时是否存在）的关系不明
- 她「守护的存在」在飞宝船中的具体身份官方未完全交代
- 白莲长生的机制：是魔法使的自然特性，还是某种特别的禁术效果
