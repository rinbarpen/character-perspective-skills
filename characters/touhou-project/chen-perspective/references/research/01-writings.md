## 三、官方设定详解

### 3.1 妖妖梦角色设定（omake.txt 原文）

```
○凶兆的黑猫 橙 Chen

二面BOSS，凭依在栖息于山中的猫妖的式神。
主要拥有使用妖术程度的能力。
其真正的身份，是蓝的式神。也就是说，橙是妖怪的式神的式神。

虽说是式神，平时也不是咒符的样子，而是保持一样的姿态，
无法区分。

在没有被式神附身的平时，也会说人话，普通的跳来跳去，但顶多拥有
吓唬人类这种程度的能力。另外，外观和性格不会改变，但冬天会在被
炉里蜷成球状。

将鬼神凭依在猫妖身上从而获得较高的妖力，但因为使役橙的也是式神，
所以其能力稍低。

并且碰到水的话式就会脱离。没有被式凭依的猫妖状态的橙一般也怕水，
反正总是怕水。
```

```
○间隙妖怪的式神的式神 橙 Chen

EXTRA STAGE的中BOSS。
自之前的伤痊愈还没过多久，竟会再次战斗。
这次是在主人附近，拥有更加强大的妖力，这个力量，会对眼前的人类
奏效么？
```

### 3.2 东方求闻史纪 完整条目

**能力：** 使用妖术程度的能力

**危险度：** 中

**人类友好度：** 普通

**主要活动场所：** 兽道、妖怪之山

**特征：**
妖兽之中的标准，有一定危险性的就是妖怪猫。其中的橙，是八云蓝（后述）的式神（式神不是种族，是加载在妖怪身上的软件，这软件的运行这里写作依凭。）。

她有锐利的爪和牙，长着两条长尾巴，是典型的妖怪猫（肉球则是不明。）。

式神状态的时候，她会使用各种妖术，十分危险。但很怕水。被水泼中就会破了式神状态，而变成普通的妖怪猫。

妖怪猫状态下，她是完全的兽性化，利用身体又咬又抓，但还是怕水。被水泼中动作就会迟钝。无论是式神状态还是妖怪猫，外表都是一样的。

她会说人话，特别是式神状态时，会使用妖术，看来很聪明。事实上只有附近人类的小孩程度的智慧。

平时她住在妖怪之山，有事就会来人类村庄袭击人类。不过成为八云蓝式神的时候，则是和八云蓝、八云紫一起行动。

**目击者报告：**
- 首先是怕水。雨天她就不出来了。如果逃到河中的话，她就不会再追来了吧。
- 其次是木天蓼。携带木天蓼的话，只需撒向她就会失去战意。要在兽道行走时，常备一些木天蓼就行了。
- 这妖怪的妖术，是以高速巧妙的活动来迷惑人的。冲出视界左边，突然又从右边出来。这是种活用速度的妖术，有些像魔术了。

### 3.3 永夜抄相关设定

在《东方永夜抄》（TH08）中，橙出现在结局 No.1 和 Normal Ending No.5 中。她在永夜抄中以协助者的身份登场，帮助主角方调查虚假之月的异变。

### 3.4 东方文花帖（书籍）设定

射命丸文对橙进行过采访报道。报道中提到橙试图建立「猫之里」，在猫群中鹤立鸡群，但其他猫似乎都不太听她的指挥。

## 五、性格特征（基于官方设定）

### 5.1 孩子气的智慧

官方明确记载只有「附近人类的小孩程度的智慧」。这不是歧视性的描述，而是说明她的认知和判断处于儿童水平——天真、直接、不擅长复杂推理。

### 5.2 式神/非式神的双重性

- **式神状态**：被鬼神凭依，获得妖术能力，能说人话，看起来聪明，能使用符卡
- **非式神状态**：能力仅剩「吓唬人类」，更加兽性化，行动模式更像猫
- 但无论哪种状态，外表和性格不变——她始终是她

### 5.3 对主人的忠诚

作为蓝的式神，橙对蓝表现出高度的依赖和忠诚。蓝对橙也有保护欲——在妖妖梦EX剧情中，蓝为橙被击败一事向主角方寻仇，说「橙是我的式神。就算是式神，也不是想做就做的」。

### 5.4 怕水

这是她的核心弱点设定——作为式神怕水（水会解除凭依），作为猫也怕水（天性双重恐惧）。她在雨天会躲起来，被水泼中会动作迟钝。

### 5.5 冬天蜷缩

冬天会在被炉里蜷成球状——这是官方认证的猫系行为。

### 6.2 符卡命名风格

橙的符卡大量使用阴阳道/道教/密教术语：
- 凤凰、晴明（安倍晴明）、道满（芦屋道满）、天仙、韦驮天、护法天童、尸解、鬼门金神、奇门遁甲
- 命名风格与蓝的式神身份高度契合——阴阳道主题，暗示她作为式神的身份来源
- 同时也在用「凤凰」「天仙」等华丽命名来弥补实际实力的不足

## 七、主题曲《凋叶棕》解析

| 项目 | 内容 |
|------|------|
| **曲名** | 凋叶棕（withered leaf） |
| **译名** | 凋零的棕叶 / 枯叶 |
| **首发** | 东方妖妖梦 二面BOSS曲 |
| **再使用** | — |

ZUN官方吐槽：「虽然做成了非常适合关卡的感觉，但我不认为这适合角色。」（出自人妖名鉴霄暗篇·ZUN评语）

「凋叶棕」之「棕」读作まき，与迷途之家「マヨヒガ」的第一音节发音相近，形成双关。

曲风特点：带有神秘感和东方韵味，节奏略显诡异——契合迷途之家的幽灵村氛围和橙作为「式神的式神」的神秘身份。

## 八、关键设定归纳

1. **式神的式神**：这是橙最核心的设定标签。她是八云蓝的式神，而蓝是八云紫的式神。这使得她在妖怪层级中处于「被使役者的被使役者」位置，实力也因此打折扣。

2. **猫又本质**：即使不被式神凭依，她也是一只独立的妖怪猫（猫又）。她不需要变成符纸——她始终维持实体状态，这点与一般的式神不同。

3. **智力水平**：官方明确为「人类小孩程度」。这不是贬义——是她的角色核心特征之一。她说话、思考、判断都带着孩子的稚气。

4. **猫的特性与弱点**：怕水、喜欢木天蓼（猫薄荷）、冬天在被炉里蜷缩、试图建立猫的村落——这些猫系特征是她角色魅力的重要组成部分。

5. **与蓝的关系**：她是蓝的式神，蓝宠爱她（为她的败北报仇），她依赖蓝（叫蓝大人）。这是她最重要的关系锚点。

6. **居住地**：迷途之家（妖怪之山中的幽灵村落），她是那里的守护者和猫群的首领。
