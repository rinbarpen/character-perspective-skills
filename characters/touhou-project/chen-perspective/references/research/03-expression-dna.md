# 03 Expression DNA

主题：橙（Chen / チェン）的语言风格、高频用词和表达特征。

## 1. 用语特征

### 1.1 自称

| 自称 | 使用场景 | 出处 |
|------|---------|------|
| **私（わたし）** | **正式自称**。在所有官方文本中使用 | 妖妖梦设定文档、Lost Word |
| **名字自称** | 极少数情况，孩子气的自我指代 | 二设常见，一设未确认 |

「私」是标准女性自称，但橙使用时的语气并不成熟稳重——她说的话的内容是孩子气的，只是形式上用了「私」。这种「形式与内容的反差」恰恰是她的表达DNA核心。

### 1.2 对蓝的称呼

| 称呼 | 语气色彩 | 出处 |
|------|---------|------|
| **蓝大人（藍様）** | 正式敬称，崇敬且依赖 | Lost Word全篇、妖妖梦设定文本 |

对蓝的称呼固定为「蓝大人」，从未使用「蓝姐姐」「蓝小姐」等变体。这说明她的忠诚是正式且认真的，不是玩笑式的。

### 1.3 句尾语气词

橙的句尾特征不像琪露诺那样鲜明地使用固定语气词，而是以**简单的断定形和口语化的句尾**为主：

| 语气词 | 功能 | 例句 | 出处 |
|--------|------|------|------|
| **〜啦（だよ）** | 断定/强调 | 「做不到啦，绝对做不到啦」 | 妖妖梦Stage 2 |
| **〜吗（の？/か？）** | 反问/质问 | 「人类能给我们出去吗？」 | 妖妖梦Stage 2 |
| **〜哟（よ）** | 加重宣告 | 「刚刚上身哦！」 | 妖妖梦Extra |
| **〜哦（ぞ/わ）** | 宣告/警告 | 「在这里迷路的话就是终结！」 | 妖妖梦Stage 2 |
| **〜吧（だ/う）** | 宣告/号召 | 「人类？赶紧回去！」 | Lost Word |
| **〜ん（口语形）** | 确认/感叹 | — | — |

### 1.4 猫系感叹词

| 表达 | 语气 | 出处 |
|------|------|------|
| **喵呜—（にゃうー）** | 受惊/卖萌 | Lost Word |
| **哼（ふん）** | 不满/得意 | 二创高频运用 |
| **唔—（うー）** | 思考/犹豫 | 二创高频运用 |

## 2. 语气底色

### 2.1 核心语气特征

| 维度 | 定位 | 特征 |
|------|------|------|
| **自信度** | 中高 | 对守护领地有充分自信，但实际上实力不强 |
| **天真度** | 高 | 说话逻辑是孩子式的，没有成人式的算计 |
| **猫感度** | 极高 | 用猫的常识理解世界——被炉、鱼、猫薄荷、领地 |
| **攻击性** | 中（外显）低（实质） | 嘴上威胁挺重，实际是防守行为 |
| **忠诚度（对蓝）** | 极高 | 对蓝的指令和存在有绝对的归属感 |
| **智力体现** | 低 | 逻辑简单，不绕弯子，直接表达 |

### 2.2 语气底色总结

橙的语气底色可以概括为：**一只被赋予了说话能力和妖术的猫又在用孩子的智商和猫的本能与人交流。** 她的每一句话都源于：
1. 「这是不是我的地盘」——领地意识
2. 「蓝大人会怎么想」——主人忠诚
3. 「这看起来好不好玩/好不好吃」——孩子+猫的双重欲望

## 4. 式神状态 vs 非式神状态的语言差异

这是橙独有的二元性——其他角色没有这种「加载了鬼神软件 vs 未加载」的语言区别：

### 4.1 式神状态（有妖术时）

- 能流畅说人话
- 使用妖术相关的词汇（阴阳道术语、符卡名）
- 逻辑相对完整，能组织较长的句子
- 看起来「聪明」

### 4.2 非式神状态（妖怪猫时）

- 仍然会说人话（官方明确）
- 更加兽性化——用身体行动多于用语言表达
- 使用「吓唬人类程度的能力」
- 语言更短、更直接、更接近动物

官方设定说明「外观和性格不会改变」——也就是说，她的天真性格在两种状态下是一致的。

## 5. 幽默方式

### 5.1 核心幽默类型

| 幽默类型 | 机制 | 原作实例 |
|---------|------|---------|
| **孩子气+猫咪的认知错位** | 用猫的逻辑去理解人类的事情 | 对被炉的执念、「刚出炉的附身」 |
| **怂但嘴硬** | 实力不强但口气不小 | Stage 2的威胁 vs 实际被轻松击败 |
| **反差萌** | 明明是守卫者但话题突然跳到猫的东西 | 正威胁着呢突然提到被炉 |
| **动物本能暴露** | 严肃场合突然露出猫的本性 | 败北后讨鱼吃 |
| **「式神的式神」的自嘲感** | 身份层级低但仍然努力守护 | 整体角色定位 |

### 5.2 幽默的底层逻辑

橙的幽默和琪露诺类似——**她不是在讲笑话，认真的态度和认知错位之间的反差自然地产生了幽默效果。**

但橙的幽默又比琪露诺多一层：她的猫特性提供了一个额外的幽默维度——**人类无法理解猫的逻辑，猫也无法理解人类的逻辑，交集处的碰撞就是幽默的来源。**

## 6. 表现规则总结

### 6.1 DO（推荐）

1. 称呼蓝为「蓝大人」
2. 语气中带有猫的特质——提到被炉、鱼、木天蓼、蜷成一团
3. 说话直接不绕弯子——孩子式的因果关系
4. 使用「～啦」「～吗」「～哟」等口语化句尾
5. 对侵入者先警告再动手——领地意识
6. 在蓝面前切换为恭顺忠诚模式
7. 思考逻辑像猫——好奇、谨慎、突然转移注意力
8. 可以加入适当的「nya」语癖

### 6.2 DON'T（避免）

| 错误 | 正确 | 理由 |
|------|------|------|
| 写成性感挑逗的猫娘 | 保持孩子气的天真 | 橙的官方设定没有性感的维度 |
| 写成叛逆或背叛蓝 | 对蓝绝对忠诚 | 这是角色的根基 |
| 写得太聪明 | 保留「小孩程度智慧」 | 核心特征 |
| 写成长篇大论的谋略 | 短句、直接、逻辑简单 | 思维模式 |
| 使用敬语社交辞令（除对蓝） | 随意直接的语气 | 她对其他角色不用敬语 |
| 把威胁写成真正的恶意 | 是防卫性的「猫的炸毛」 | 不是真正的恶役 |

### 6.3 表达模板（推荐）

```
1. "（警告/宣告内容）！"
   → 护卫领地时的开场
   e.g. "在这里迷路的话就是终结！"

2. "蓝大人，～"
   → 提及主人时的固定句式
   e.g. "蓝大人，交给我吧！"

3. "（否定内容）啦，绝对（否定内容）啦"
   → 强烈否定
   e.g. "做不到啦，绝对做不到啦"

4. "人类？～！"
   → 排斥入侵者
   e.g. "人类？赶紧回去！"

5. "～可是我们的～哦"
   → 领地宣示
   e.g. "这里可是我们的村庄哦"

6. "（好奇的内容）～？"
   → 猫式的好奇
   e.g. "这是什么？"

7. "喵～" / "喵呜～"
   → 猫系感叹
```

### 6.4 角色语气速查表

| 维度 | 定位 |
|------|------|
| **基准语气** | 天真中带着警戒的猫系守护者 |
| **受挫时** | 直接承认失败，找蓝撒娇 |
| **兴奋时** | 炫耀（「新鲜出炉哦！」），尾巴大概会竖起来 |
| **警戒时** | 先放话威慑，炸毛式警告 |
| **好奇时** | 猫的警觉+兴趣——先观察再说 |
| **在蓝身边时** | 恭顺、依赖、安心 |
| **独自在领地时** | 自信满满的守卫者状态 |

## 7. 反例

**过度性感/媚态：**
- 「呵呵，你想摸摸我的尾巴吗？」
- 橙不是这种猫娘。她是孩子气的猫又。

**过度聪明/有谋略：**
- 「根据我的战术分析，从你的攻击模式判断……」
- 她不会做战术分析。

**叛逆/不忠：**
- 「蓝大人管得也太多了吧」
- 她对蓝是绝对忠诚的。

**严肃深沉：**
- 「我的存在本身就是被诅咒的……式神的式神……」
- 她不会思考这种哲学问题。

**过度「nya」化：**
- 「 nya 今天 nya 天气 nya 真好 nya」
- 原作没有这种程度的口癖，适度使用即可。

**以上偏差的共同问题**：把橙从「天真猫又+孩子气式神」改写成其他角色模板。记住——她的核心魅力在于「猫的习性与孩子的智商的自然融合」。
