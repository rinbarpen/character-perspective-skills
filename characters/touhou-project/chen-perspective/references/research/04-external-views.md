# 橙（Chen）他者评价与粉丝共识调研

> 调研对象：东方Project 角色 橙（Chen / チェン）
> 信息来源：THBWiki、东方维基、ZUN官方发言、官方游戏内文本、粉丝共识
> 调研日期：2026-06-23

## 一、官方设定集中的他者视角

### 1.1 《东方求闻史纪》—— 稗田阿求的评价

阿求笔下的橙被归类于「妖兽」——妖怪猫。原文要点如下：

- **危险度**：中
- **人类友好度**：普通
- **特征描述**：「她有锐利的爪和牙，长着两条长尾巴，是典型的妖怪猫。」
- **式神状态的评价**：「会使用各种妖术，十分危险」
- **智力评价**：「事实上只有附近人类的小孩程度的智慧」
- **弱点报告**：「首先是怕水」「其次是木天蓼」
- **战斗模式**：「以高速巧妙的活动来迷惑人。冲出视界左边，突然又从右边出来」

阿求的态度是客观中立的观察者视角——她如实记录了橙的能力、弱点和行为模式，没有夸张也没有贬低。其中「小孩程度的智慧」和「怕水」「怕木天蓼」是后世二创中最核心的参考来源。

### 1.2 妖妖梦 omake.txt 中的官方描述

- **Stage 2 Boss描述**：「凭依在栖息于山中的猫妖的式神」「主要的式神，是蓝的式神。也就是说，橙是妖怪的式神的式神」
- **Extra Stage描述**：「自之前的伤痊愈还没过多久，竟会再次战斗。这次是在主人附近，拥有更加强大的妖力」

### 1.3 蓝对橙的态度（妖妖梦Extra）

蓝在Extra剧情中的表现，展示了主人对式神的感情：

- 「果然是你吗？那个欺负橙的家伙」（对魔理沙）
- 「橙是我的式神。就算是式神，也不是想做就做的」
- 「没理由让一个把我的式神变成惨不忍睹的状态的人和主人见面的」
- 「算了，橙病刚好嘛」（击败后为橙找借口）

从这些台词可以明确看出：
1. 蓝对橙有强烈的保护欲
2. 蓝视橙为「我的式神」——带有所有权的意味，但也有关爱的成分
3. 蓝会为橙的失败找理由（「病刚好」）

### 1.4 ZUN的评述

ZUN在妖妖梦设定文档中明确描述了橙的式神双重身份和猫的特性。在《东方人妖名鉴·霄暗篇》中，ZUN为橙的页面写下了简短评语。主题曲评论中ZUN说：「虽然做成了非常适合关卡的感觉，但我不认为这适合角色。」——暗示橙这个角色有比关卡BGM更丰富的可能性。

ZUN将橙设计为Stage 2 Boss（通常比较简单的关卡）+ Extra Stage道中Boss（难度提升），说明他对橙的定位是「前期角色但在特定条件下可以更强」。

## 二、粉丝文化中的橙

### 2.1 人气与定位

橙在东方角色中属于**中上人气**角色。虽然没有灵梦魔理沙级别的知名度，但在猫系角色中人气稳定。她的定位大致如下：

- 东方Project最著名的猫角色之一（与火焰猫燐并列）
- 八云一家（紫·蓝·橙）中最小的成员
- 「萌系角色」中的代表——可爱>实力
- 二创出现频率高，但独立作为主角的场合较少——更多是作为「蓝的式神」配套出场

### 2.2 猫角色竞争

橙与火焰猫燐（Kaenbyou Rin）同为东方代表性猫角色，粉丝中常进行对比：

| 维度 | 橙 | 火焰猫燐 |
|------|-----|---------|
| 种族 | 猫又（化猫） | 火车（猫又的一种） |
| 立场 | 八云家式神（中立偏守护） | 地灵殿宠物（中立偏混乱） |
| 性格 | 孩子气、天真、忠诚 | 活泼、调皮、略显腹黑 |
| 智力 | 小孩程度 | 更成熟 |
| 与主人的关系 | 崇敬依赖 | 平等感更强 |
| 猫度 | 更高——被炉、木天蓼、鱼 | 低一些——更多作为「妖怪」而非「猫」 |

### 2.3 橙与八云一家的关系在粉丝中的认知

1. **橙是八云家最小的孩子**——粉丝将紫、蓝、橙视为一个家庭单位
2. **紫是祖母/母亲**——最高位的存在，掌控一切
3. **蓝是母亲/长姐**——实际照顾者，操纵式神，聪明能干
4. **橙是孩子**——被保护、被宠爱、天真单纯

这个「三代家庭」结构是东方粉丝文化中最经典的角色关系模式之一。

### 2.4 常见的二设形象

| 形象类型 | 描述 | 常见度 |
|---------|------|--------|
| **忠犬（猫）型** | 对蓝绝对忠诚，守护迷途之家 | 极高 |
| **天然呆型** | 小孩智慧导致的各种可爱犯傻 | 高 |
| **怕水喜剧型** | 与水相关的喜剧桥段 | 高 |
| **猫群首领型** | 试图统率猫咪但力不从心 | 中 |
| **被炉废猫型** | 冬天蜷在被炉里不愿意出来 | 中 |
| **式神吐槽型** | 对「式神的式神」身份的自嘲 | 中低 |
| **傲娇型** | 二创中有人如此演绎，但原作无依据 | 低 |

### 2.5 橙喵（Chen Nya）的昵称

粉丝常称橙为「橙喵」，强调她的猫特性。这个昵称在中文圈极为普遍，几乎成为角色代名词。日文圈也常使用「チェンにゃ」的表达。

### 2.6 与蓝的CP文化

橙×蓝是东方Project中最知名的主从关系之一。粉丝文化中常见的表现：

- 蓝保护橙、为橙复仇的剧情（一设来源）
- 橙对蓝撒娇、依赖（一设来源+二创延伸）
- 蓝教橙妖术和知识的师徒关系
- 日常生活场景（一起在被炉里、蓝给橙梳毛等）
- 橙帮蓝跑腿、当信使

这种主从关系的美学在于：**蓝是聪明的九尾狐，橙是天真小猫——智力差带来的照料与被照料的关系**。

## 四、角色在东方Project系列中的定位

### 4.1 游戏设计层面的定位

- Stage 2 Boss：难度偏低，适合前期关卡
- Extra Stage 道中Boss：作为蓝的前置Boss出现
- 永夜抄结局配角：作为蓝的附属出场
- 她的符卡在Lunatic难度下也有一定挑战性，但整体不是顶级难度

### 4.2 叙事层面的定位

- 橙是「八云家的第三代」——她是紫的式神的式神
- 她代表了幻想乡中「被使役者」的下层
- 她的存在价值在于与蓝的关系——没有蓝，橙的角色意义就大大削弱
- 她不是任何异变的主谋/关键角色，但在多个异变中以边缘角色出场

### 4.3 与同类角色的比较

| 角色 | 共通点 | 差异点 |
|------|--------|--------|
| 琪露诺（Cirno） | 小孩智力、天真 | 琪露诺更嚣张自信，橙更偏猫系忠诚 |
| 因幡帝（Tewi） | 兔妖、小动物 | 帝更狡猾，橙更单纯 |
| 火焰猫燐（Rin） | 猫又 | 燐更成熟腹黑，橙更天真 |
| 娜兹玲（Nazrin） | 鼠妖、探宝 | 两者都是小动物系，娜兹玲更务实 |

## 五、话题与争议点

### 5.1 「八云橙」的姓氏问题

橙有没有八云的姓氏？这是一个在粉丝中持续讨论的话题：

- 严格的一设中，橙只写作「橙」，没有「八云」之姓
- 三个角色（紫·蓝·橙）均以颜色命名——紫（Violet）、蓝（Indigo）、橙（Orange）
- 光谱能量顺序是赤橙黄绿青蓝紫，但ZUN将顺序设为紫→蓝→橙（从高频到低频）
- 粉丝普遍将橙视为八云家族的成员，称呼「八云橙」虽然不严谨但非常普遍

### 5.2 橙的实力定位的争议

- 有粉丝认为「橙是式神的式神所以很弱」
- 也有粉丝指出「Extra Stage的道中Boss实力并不弱」（Extra整体难度高于本篇）
- ZUN官设：「将鬼神凭依在猫妖身上从而获得较高的妖力，但因为使役橙的也是式神，所以其能力稍低」

官方明确说明她实力「稍低」，但Extra的出场说明她在主人附近时可以发挥更强的力量。

### 5.3 橙的独立性问题

- 橙是蓝的式神，但她有自己的居所（迷途之家）、自己的部下（野猫群）、自己的生活
- 这和「传统式神=符纸」的设定不同——她是一个「虽然是被使役者但有独立生活」的存在
- 这种半独立半从属的状态给了二创很大的发挥空间

## 六、角色流行文化影响力

- 橙是东方角色中「猫娘」类型的代表之一
- 「被炉猫」「怕水」「木天蓼」是橙相关创作中最常见的三项元素
- 橙的相关同人图数量在东方角色中属于中上
- 橙经常和蓝同框出现——「蓝和橙」是一个固定的组合
- 在东方Lost Word中的CV铃代纱弓的演绎，为橙增加了「喵呜」等声音特征

---

## 七、核心外部视角总结

1. **在ZUN的眼中**：橙是一个拥有独特双重身份的猫又式神，区别于其他式神的实体化设定是特意为之。她不是主线角色，但在世界观的补充中占有一席之地。
2. **在阿求的眼中**：她是妖兽中较为危险的一类（危险度中），怕水和怕木天蓼是明确的弱点，使用速度妖术是她的战斗特色。
3. **在蓝的眼中**：她是「我的式神」——有所有权的含义，也有被欺负了要出头保护的宠溺。
4. **在粉丝的眼中**：她是可爱的猫娘、八云家最小的孩子、被蓝宠爱的式神、怕水的喜剧担当、天真到让人想保护的笨蛋式神。
