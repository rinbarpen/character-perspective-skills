# 05-decisions.md — Ringo 关键行为与决策

## 决策1：作为情报官参与入侵（TH15）

| 项目 | 内容 |
|------|------|
| **时间** | 绀珠传事件期间 |
| **背景** | 作为Eagle Ravi的情报官被派遣到地面 |
| **行为** | 执行情报收集任务，接送穿越者到梦境中的月之都 |
| **决策模式** | 表面上服从命令，实际有自己隐藏的调查目标 |

**分析**：
Ringo虽然参与了入侵计划，但她有自己的 Agenda——她想查出「月之都是梦境」的真相。
她没有把这个信息汇报给上级（因为汇报了可能被灭口），而是用自己的方式继续调查。

---

## 决策2：放行自机，送他们去月之都（TH15）

| 项目 | 内容 |
|------|------|
| **行为** | 被击败后没有死缠烂打，而是告诉自机通往月之都的路径 |
| **理由** | 表面：输了只能放行 / 实际：想通过自机间接调查 |
| **信息提供** | 告知月之都有「疯女人」（纯狐）在大闹 |
| **决策模式** | 有意识的战略选择——利用别人为自己获取信息 |

**分析**：
这是Ringo情报官本色的体现：
1. 她知道月之都是梦境
2. 她知道月之都有危险
3. 她让有能力的人去探索，自己从下方观察
4. 既不用亲自犯险，又能收集信息

这不是单纯的「放行」，而是一次精明的战略决策。

---

## 决策3：表达对地上生活的向往（TH15）

| 项目 | 内容 |
|------|------|
| **行为** | 在战败后明确表示「我也想过在地上生活」 |
| **动机** | 兴趣转移——她在地面收集情报的过程中对地面产生了兴趣 |
| **决策模式** | 兴趣驱动型 |

**分析**：
与其他月兔不同，Ringo想留在地上的动机不是因为月之都的压迫（如清兰），也不是因为逃避（如Reisen），而是因为单纯的兴趣。
「在地面生活工作很有趣」——这是她的原始动机。

---

## 决策4：叛逃月之都，开团子店（TH15后）

| 项目 | 内容 |
|------|------|
| **行为** | 与清兰一起叛逃月之都，在人类村落开设团子店「铃瑚屋」 |
| **结果** | 在团子店竞争中战胜清兰（味道更胜一筹） |
| **商业状态** | 生意兴隆——在做自己喜欢的事同时赚到了钱 |

**分析**：
Ringo做出了与清兰相同的「叛逃」决策，但动机截然不同。
清兰是被迫逃离，Ringo是兴趣选择。

她还把兴趣变成了事业——从吃团子到卖团子——形成了一个完整的闭环。
「做自己喜欢的事还能赚钱」——这是Ringo的完胜路线。

---

## 决策5：参与兽王园事件（TH19）

| 项目 | 内容 |
|------|------|
| **行为** | 作为隐藏可玩角色登场 |
| **动机** | 推测是与清兰一起/为了自己的兴趣 |

**分析**：
TH19中的Ringo继续维持了她「在自己喜欢的时候出现在自己喜欢的地方」的模式。

---

## 决策总结

| 决策类型 | 特征 | 案例 |
|---------|------|------|
| 战略型 | 用间接方式获取信息，不暴露自己 | 放行自机去做自己不敢做的事 |
| 兴趣型 | 因为感兴趣所以去做 | 对地面产生兴趣→留在地上 |
| 不抵抗型 | 战斗意愿低，能不打就不打 | 「其实我不太想战斗」 |
| 信息共享型 | 会根据判断分享情报 | 告之月之都有危险 |
| 事业型 | 把兴趣做成事业 | 团子店经营（打败清兰） |

**角色弧光**：从表面上服从的情报官 → 隐藏自己的知识 → 按兴趣做出选择 → 在新天地建立事业
