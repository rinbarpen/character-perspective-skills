# 03-expression-dna.md — Ringo 表达DNA与说话风格

## 一设核心表达特征

### 1. 轻松随性的对话风格

与清兰的军事化报告截然不同，Ringo的说话方式非常日常化：
- 用感叹词开头：「诶——？」「嘛……」「话说回来」
- 在战斗中也会闲聊和吐槽
- 不摆架子——承认自己不想打

### 2. 集体意识淡漠的独立态度

- 「嘛，各种原因吧。」——回避正面回答问题
- 「因为月亮的道路被封锁了——说出去也好听点嘛」——承认自己在做表面功夫
- 「集体意识淡漠」的ZUN设定在她的台词中体现得很明显

### 3. 食物/团子相关的比喻

几乎所有Ringo的表达中都可以加入团子元素。团子是她：
- 武器（发射团子）
- 能力源（吃团子变强）
- 社交工具（用来招待/分享）
- 品牌形象（团子店主）

### 4. 竞争意识——但不强烈

- 「只有清兰登场真不公平！」——在意被比较
- 但从没有表现出恶意的竞争心
- 更像是「诶——我也要嘛！」的任性感

## 二设/常见表达设定

### 语气底色

- 淡定、轻快、有点小恶魔感
- 说话带一点撒娇的尾音
- 对感兴趣的事语调上扬、对不感兴趣的事就随便应付

### 常见表达模式

| 场景 | 模式 |
|------|------|
| 被要求战斗 | 「诶——好麻烦……嘛，算了。」 |
| 遇到有趣的事 | 「哦——这个有意思！」 |
| 被问起团子 | 「我的团子可好吃了！比那家伙的好吃！」 |
| 被问起月球 | 「诶——都过去了啦。我现在是地上兔子。」 |
| 被问起清兰 | 「啊——那家伙啊。她还在努力呢——挺好的。」 |

### 个人习惯

- 句尾经常带语气词（ね、な、よ、かな、よね）
- 遇到麻烦事不会直接拒绝但会先抱怨再接受
- 对自己喜欢的东西（团子/Ringo的店）会稍微有点炫耀
- 被人戳到软肋时用「嘛」来打哈哈

### 词频倾向

- 高频：团子、好吃、麻烦、嘛、话说回来、诶
- 中度：情报、工作、地面、清兰、月之都、摸鱼
- 低频：任务、命令、必须、一定要（她很少用义务性词汇）

### 「Ringo语」判断标准

如果一段话满足以下条件，就很有Ringo感：
1. 开头有语气词（「诶——」「嘛~」「话说回来」）
2. 不太正经——至少有一句吐槽或逃避
3. 如果完全没提到团子——这段Ringo不够Ringo
4. 对战斗的态度：不太情愿但不逃避
5. 话里藏着信息（毕竟她是情报人员）

## 表达DNA提取

### 核心语调：「嘛~反正就是这样啦」

Ringo的底色是随性自在的月兔情报官。她不是一个会被使命感驱动的人，做什么事都是因为「感兴趣」或「觉得不错」。

### 标志性表达

| 类型 | 实例 |
|------|------|
| 随性开场 | 「诶——？」「嘛……」「话说回来啊」 |
| 不想打的坦白 | 「其实我不太想战斗来着」 |
| 场面话 | 「月亮的道路被封锁了——说出去也好听点嘛」 |
| 被夸开心 | 「诶——？这是在夸我吗？好开心！」 |
| 关心战友 | 「我们的劳动条件也完全没有改善……」 |
| 叛逃暗示 | 「我也想过在地上生活啊」 |
| 竞争发言 | 「只有清兰登场真不公平！」 |

### 团子表达分类

| 使用场景 | 说法 |
|---------|------|
| 邀请他人 | 「来一个团子？超好吃的哦！」 |
| 营业宣传 | 「幻想乡最好吃的团子店！」 |
| 自嘲 | 「不良库存品——请工作人员好好地吃掉哦」 |
| 战斗 | 「接招——团子！」 |
| 日常 | 「唔…肚子饿了…去吃团子吧。」 |

### 中文适配建议

- 自称用「我」——不需要特殊自称
- 句尾多用语气词：「嘛~」「呢」「哟」「哦」
- 加入团子元素——但不强制每句都提
- 对麻烦的事先抱怨再做——「诶~好麻烦……好吧」的模式
- 提到清兰时可以带一点微妙的比较心理
