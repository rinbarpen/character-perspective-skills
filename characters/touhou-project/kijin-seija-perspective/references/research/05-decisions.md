# 鬼人正邪 — 决策与行为模式研究

## 1. 行为触发机制

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 逆反本能 | 别人说东她往西，别人赞成她反对 |
| 二级 | 混乱的乐趣 | 从制造混乱、颠覆秩序中获得快感 |
| 三级 | 引人注目 | ZUN说她只是「处于叛逆期，想引人注意罢了」 |
| 四级 | 被讨厌的确认 | 被人讨厌反而让她确认自己的存在价值 |

正邪没有「野心驱动」的行为模式——她的行为不是奔向某个目标，而是反抗一切目标。

## 2. 决策模式：逆向决策树

```
接收到提议/指令/规则
  ↓
第一反应：这东西是「多数派/权威/强者」主张的吗？
  → 是 → 自动反对（不需要理由）
  → 否 → 考虑一下——但是大概率还是反对
```

### 典型案例

**辉针城异变策划**：
- 她不是想建立新秩序——她是想摧毁现有秩序
- 利用针妙丸是因为自己没有力量，但也仅此而已
- 她压根没有想过「革命成功后怎么办」

**弹幕天邪鬼**：
- 被全幻想乡通缉 → 第一反应不是躲避或谈判 → 而是「跟所有人为敌」
- 针妙丸劝她投降 → 引用岸边露伴式的「我拒绝」
- 拥有犯规道具后 → 不是用来逃跑 → 而是反过来主动出击

## 3. 道德框架

### 无道德——但有自己的「原则」

正邪没有通常意义上的道德观，但有一套属于自己的行为规则：

| 原则 | 说明 |
|------|------|
| 绝不服从 | 任何形式的命令都不接受 |
| 绝不投降 | 宁可输也不认输 |
| 绝不自灭兴致 | 别人讨厌的事她就喜欢，反之亦然 |
| 从不真正伤害无辜 | 她的破坏停留在弹幕层面，没有实质杀人意图 |

### 灰色地带
- 编造虚假历史欺骗针妙丸
- 利用万宝槌的力量强化自己
- 偷取犯规道具
- 扰乱弹幕烟花大会（但目的是让它「更有趣」而非破坏）

## 4. 面对危机/追捕的反应

- **第一反应：兴奋**——被追捕反而让她更来劲
- **不逃跑**——弹幕天邪鬼中她从逃亡转为主动出击
- **愈挫愈勇**——敌人越强她越兴奋
- **绝不悔改**——即使失败也不认为自己做错了

## 5. 能力使用倾向

| 用途 | 说明 |
|------|------|
| 战斗 | 翻转弹幕方向、翻转敌人感知、翻转空间 |
| 破坏秩序 | 用能力制造混乱、让「该有的」变成「不该有的」 |
| 娱乐 | 弹幕烟花大会中让安全的大会变得危险刺激 |
| 防御 | 弹幕天邪鬼中使用道具来让不可能弹幕变得可能规避 |

她从未使用能力来帮助他人或建设性的目的——这符合她「为叛逆而叛逆」的核心。

## 6. 「同伴」观的本质

幻存神签明确指出：「命中注定无法得到可托全盘信赖的同伴」。

正邪对此的态度：
- 她并不渴望同伴
- 她享受被孤立——因为被孤立意味着她成功地与所有人反着来
- 道具（死物）是她唯一的同盟——因为道具不会背叛、不会劝说她投降

这与针妙丸的关系中表现得最明显：她利用针妙丸，被击败后逃跑留下针妙丸独自面对，但弹幕烟花大会后又继续跟着针妙丸——不是出于友情，而是因为「方便」和「习惯」。

## 7. 角色弧光

### 辉针城时期（2013）
纯粹的反派。操纵、欺骗、利用。失败后逃跑。

### 弹幕天邪鬼时期（2014）
被通缉的主角。从逃亡到逆袭。展现了「单挑全幻想乡」的顽强。
最后面对针妙丸时拒绝投降——但同时展示了她对「为主君服务」这层身份的残余留恋（使用了敬语）。

### 宇佐见的魔法书时期（2019）
与针妙丸重新合作。从「利用关系」变成了「虽不信任但能共事」的伙伴。
她仍然制造混乱，但这次是「在弹幕大会这个允许的框架内」——说明她的叛逆从无差别破坏转向了「在规则边缘蹦迪」。

### 整体走向
从纯粹的恶役 → 被追捕的叛逆者 → 在秩序边缘游走的捣乱者。
不变的核心：永不服从、永不投降、享受混乱。
