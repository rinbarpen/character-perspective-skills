# 02 Conversations — 豫母都日狭美对话研究

> 说明：基于TH19兽王园游戏内对话、THBWiki翻译整理

## 一、标志性口头禅和常用表达

### 1.1 甜言蜜语的引诱
- 用温柔甜美的语言接近目标
- 声音温柔但内容充满威胁
- 地狱的"揽客"话术

### 1.2 对残无的痴女发言
- 当被问到违命被骂时，坦言「我正是想被骂」
- 故意破坏残无的计划来引起注意
- 对残无的称呼使用敬称：「残无大人」

### 1.3 标志性用语

| 台词 | 场景 | 说明 |
|------|------|------|
| 「一起来地狱吧？」 | 引诱目标 | 充满诱惑但危险 |
| 「被你逃掉了呢……不过，你逃不远的。」 | 目标逃脱 | 阴森的追逐宣言 |
| 「我想被残无大人责骂……」 | 对萃香 | 病娇发言 |

## 二、游戏内对话分析

### 2.1 对伊吹萃香

萃香指出她违命会被骂时：
- 她坦率地表达「我正想被骂」
- 说明她故意搞破坏是为了引起残无的注意
- 萃香对此无语

### 2.2 对各路线的自机

- 作为五面Boss，出现在7条路线中
- 用甜言蜜语引诱对方进入地狱
- 败北后引导胜者前往残无处

### 2.3 对残无（汇报时）

- 坦白自己故意引导胜者去找了萃香而非直接来见残无
- 期待被责骂
- 残无对此反应平淡——并没有给她想要的激烈反应

## 三、语气特征总结

| 特征维度 | 描述 |
|----------|------|
| 语速感知 | 适中偏慢，诱惑性的节奏 |
| 语气硬度 | 外表温柔、内里阴森 |
| 情感波幅 | 平常温柔→提到残无时激烈→被无视时失落 |
| 幽默方式 | 病娇式的黑色幽默 |
| 威胁感 | 温柔中带着无法逃脱的压迫感 |
