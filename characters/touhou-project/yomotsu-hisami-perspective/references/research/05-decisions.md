# 豫母都日狭美 — 决策与行为模式研究

## 1. 行动动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 获得残无的关注 | 最大的动力来源 |
| 二级 | 执行"揽客"工作 | 将人拖入地狱并瓜分报酬 |
| 三级 | 获得更多报酬 | 超额完成任务 |
| 四级 | 维持自己的信誉 | 确实履行与亡者的约定 |

## 2. 面对危险的反应

- **优先引诱** — 用甜言蜜语化解决冲突
- **不放弃目标** — 一旦锁定就追到底
- **故意搞破坏** — 即使不利于任务也要引起残无注意

## 3. 道德边界

- 有独特的职业道德：与亡者约定瓜分报酬就会履行
- 但"超额拉客"导致被阎魔盯上
- 为了引起残无注意不惜破坏计划
- 整体上亦正亦邪

## 4. 核心矛盾

- 美丽的外表——丑女种族的本质反差
- 温柔的语言——危险的目的
- 渴望被残无关注——却用搞破坏的方式
- 履行职责——却总是"用力过猛"

## 5. 角色发展

### 兽王园前
黄泉丑女 → 地狱揽客 → 因拉客太多被阎魔盯上 → 遇到残无并单方面仰慕

### 兽王园中
作为残无的助手引导各路自机 → 故意引导人去找萃香而非直接见残无 → 期待被骂

### 兽王园后
未被残无严厉责骂 → 继续在地狱做揽客工作 → 可能继续寻求残无的关注
