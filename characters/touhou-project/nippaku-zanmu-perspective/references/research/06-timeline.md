# 日白残无 — 作品时间线

## 人类时期（战国时代）

| 事件 | 说明 |
|------|------|
| 作为僧侣「日白残梦」出生 | 战国时代的人类 |
| 经历战国乱世 | 目睹无情的人间地狱 |
| 质疑"杀生"的意义 | 违反僧侣的戒律 |
| 得出「杀生即救赎」的结论 | 扭曲的救赎观 |
| 开始吸收动物灵与幽灵 | 逐渐变成妖怪 |
| 主动堕入地狱 | 在堕入地狱前自行前往 |

## 地狱时期

| 事件 | 说明 |
|------|------|
| 到达地狱 | 以人类的身份进入鬼的世界 |
| 凭借才能和话术获得鬼的信任 | 智能型鬼的崛起 |
| 自身鬼化（人鬼） | 完全成为鬼的一员 |
| 提出新地狱提案 | 将旧地狱与新地狱分离 |
| 成为新地狱的统治者之一 | 以对话和智慧统治 |
| 旧地狱居民因被抛弃而怨恨她 | 产生了大量怨恨 |

## 兽王园事件前

| 事件 | 说明 |
|------|------|
| 察觉到畜生界的野心 | 动物灵想统治地面 |
| 地面所有权归零 | 畜生进军的契机 |
| 地狱的鬼想开战 | 鬼与畜生的战争将导致地面毁灭 |
| 主动请缨处理 | 独自承担阻止战争的责任 |
| 招募美天/血枪/慧之子 | 分散到三个家族当间谍 |
| 背后煽动三个动物灵领袖 | 让他们同时进军→三足鼎立→分割统治 |

## 东方兽王园（TH19，2023年5月）

| 事件 | 说明 |
|------|------|
| 作为最终Boss（六面）登场 | 全事件的幕后黑手 |
| 三颗棋子被逐一击败 | 灵梦在战斗中了解到真相 |
| 发现灵梦的存在 | 意识到幻想乡有能获得妖怪支持的人类 |
| 与灵梦对决 | 最终决战 |
| 将幻想乡委托给灵梦 | 「你是像我一样的人」 |
| 作为自机可用 | 通关后可选 |

## 兽王园之后

| 时间 | 事件 |
|------|------|
| 兽王园后 | 回到地狱继续统治 |

## 关键节点总结

- 战国时代：破戒僧→堕入地狱
- 地狱时期：成为鬼→新地狱提案者→智能型统治者
- 兽王园（2023）：最终Boss兼事件黑幕→发现灵梦→改变计划→委托幻想乡
