# 日白残无 — 表达DNA研究

## 1. 用语特征

### 自称
- 使用「私」（watashi）
- 对下属使用命令形，但不粗暴
- 使用敬语但不疏远——巧妙的距离控制

### 句尾口癖
- 无特定标志性助词
- 喜欢用「のだ」「のだろう」来给出"理所当然"的语气
- 断定形为主，不拖泥带水

### 对他人称呼
- 对不同对象使用不同的称呼策略
- 对棋子：接近"宠物"式的称呼
- 对对等者（萃香）：使用对等的称呼

## 2. 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 沉着度 | 极高 | 极少失去冷静 |
| 话术度 | 极高 | 语言是主要武器 |
| 亲和度 | 表面高、内里低 | 温和但疏离 |
| 控制欲 | 高 | 习惯一切在掌控中 |

## 3. 幽默方式

- **冷幽默**：带着距离感的调侃
- **上位者的玩味**：对下属的"宠爱"式幽默
- **淡然的讽刺**：不激烈但有分量

## 4. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「～のだ」 | 说明/断定 | 理所当然 |
| 「～してくれ」 | 命令 | 温和但不可违抗 |
| 「～と思う」 | 主张 | 自信但不霸道 |
| 「面白い」 | 评价 | 对意外发展的玩味 |

## 5. 对不同对象的语气

| 对象 | 语气特征 |
|------|---------|
| 灵梦 | 对等→认同→委托 |
| 萃香 | 旧识之间的对等对话 |
| 棋子们 | 温和的命令与引导 |
| 日狭美 | 无奈的容忍 |
| 地狱的鬼 | 安抚的智者语气 |

## 6. 表现规则总结

- 沉着冷静，语速适中偏慢
- 语言是主要武器——善于话术
- 温和但不失威严
- 极少显露真实情感
- 对意外事件也保持从容
