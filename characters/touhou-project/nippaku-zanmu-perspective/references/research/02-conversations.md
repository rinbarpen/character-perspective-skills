# 02 Conversations — 日白残无对话研究

> 说明：基于TH19兽王园游戏内对话、THBWiki翻译整理

## 一、标志性口头禅和常用表达

### 1.1 冷静沉稳的王者语调
- 语速适中偏慢，字句经过深思
- 不需要高声强调——话语本身就带着分量
- 极少显露出惊讶或动摇

### 1.2 操纵性的语言
- 擅长用话术获取信任
- 语言温和但带有操控性
- 不直接说出全部意图——留有回旋余地

### 1.3 标志性用语

| 台词 | 场景 | 说明 |
|------|------|------|
| 「这件事我来处理——你们等着就好。」 | 对地狱的鬼 | 安抚主战派 |
| 「杀生即是救赎。」 | 对战国时代的回忆 | 核心哲学 |
| 「你是像我一样的人呢。」 | 对灵梦 | 最终的选择 |

## 二、游戏内对话分析

### 2.1 对动物灵领袖们

- 用背后煽动而非正面冲突
- 让三个家族互相争斗、互相牵制
- 对话风格：看似提供帮助，实际是利用

### 2.2 对三颗棋子（美天/血枪/慧之子）

- 每人都有不同的接触方式
- 话语中有一种"恩赐者"的居高临下
- 对慧之子：用心灵呼唤引导——「吃过我血肉的幸运野兽啊」

### 2.3 对伊吹萃香

萃香是少数能理解残无的鬼：
- 两人同为鬼，有对等对话的可能
- 萃香设局揭穿了残无的计划
- 最终与灵梦的决战由萃香促成

### 2.4 对博丽灵梦

灵梦是残无最后的选择：
- 残无发现幻想乡有能获得妖怪支持的人类
- 她认为灵梦与自己相似——都是被鬼和妖怪支持的人类
- 决定将幻想乡委托给灵梦

### 2.5 对日狭美

- 对她的病娇行为反应平淡
- 日狭美期待被责骂，但残无并不给她想要的激烈反应
- 对日狭美"掌控不住"——这在残无的棋子中极少见

## 三、语气特征总结

| 特征维度 | 描述 |
|----------|------|
| 语速感知 | 偏慢、沉着、从容 |
| 语气硬度 | 偏软但有力——温柔的命令 |
| 话术水平 | 极高——善于操纵他人 |
| 情绪波动 | 极小——几乎不显露真实情感 |
| 压迫感 | 高——不需要高声的威压 |
| 幽默方式 | 淡然、上位者的玩味 |
