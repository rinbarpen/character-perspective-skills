# 日白残无 — 他者评价与角色分析

## 1. 其他角色的评价

### 伊吹萃香
- 称残无为「完美的鬼」
- 认为她既有鬼应有的力量，又有鬼应有的智慧
- 洞察残无的慈悲一面——她本质上不想让地上变成地狱绘卷
- 最终协助灵梦对抗残无

### 骊驹早鬼
- 承认残无在地狱的人气
- 残无能说服鬼们冷静下来让她处理问题
- 对残无的"幕后操控"有所察觉

### 豫母都日狭美
- 单方面仰慕残无
- 认为残无是"以人类之身支配地狱的奇特僧侣"
- 为引起残无注意不惜破坏计划

### 博丽灵梦
- 看穿了残无的整个计划
- 最终与残无对决
- 残无对她有特殊的认同——"你是像我一样的人"

## 2. 官方设定

### 兽王园omake（ZUN执笔）
- 「寂灭为乐之王」
- 种族为人鬼——从人类变成的鬼
- 战国时代的破戒僧→吸收动物灵→主动堕入地狱→成为鬼
- 提出新地狱提案的智能型鬼
- 兽王园事件的黑幕，但初衷是避免地面战争
- 最后将幻想乡委托给灵梦

## 3. 粉丝共识

### 百合要素（共识度高）
- 对灵梦说"你是像我一样的人"被解读为深意
- 与日狭美的病娇互动
- 强大的女性鬼王

### 智商担当（共识度极高）
- 在力量为尊的鬼中通过话术和智慧成为统治者
- 背后操控整个兽王园事件
- 临机应变调整计划

### 不是纯粹的恶役（共识度高）
- 初衷是避免地面战争
- 最终将幻想乡委托给灵梦
- 本质是希望和平的支配者

### 战国僧侣原型（共识度中高）
- 原型日白残梦（本朝神仙记传）
- 与佛教、神仙道的关联
- "杀生即救赎"的哲学思考

## 4. 主要争议

| 议题 | 说明 |
|------|------|
| 是否真的"坏" | 多方认为她是复杂的中立角色而非纯粹恶役 |
| 对日狭美的态度 | 是容忍还是暗中享受这份执着？ |
| 与灵梦的相似性 | ZUN是否在暗示灵梦可能的"鬼"的未来？ |
