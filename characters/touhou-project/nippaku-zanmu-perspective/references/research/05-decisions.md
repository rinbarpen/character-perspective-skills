# 日白残无 — 决策与行为模式研究

## 1. 行动动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 避免地面战争 | 战国时代的记忆让她不愿看到生灵涂炭 |
| 二级 | 维持地狱的稳定 | 作为地狱统治者的责任 |
| 三级 | 掌控全局 | 享受一切在掌握中的感觉 |
| 四级 | 将未来委托给灵梦 | 最后的自我克制和选择 |

## 2. 决策风格

### 幕后操纵型
- 不直接出手——通过棋子来运作
- 让三方势力互相牵制
- 保持表面上的超然

### 话术优先
- 能用语言解决的问题不动用武力
- 说服地狱的鬼们给她时间
- 煽动三方动物灵领袖

### 灵活调整
- 发现灵梦的存在后立即调整计划
- 不执着于原定路线
- 承认"计划赶不上变化"

## 3. 道德边界

- 反对无意义的战争——战国时代的创伤
- 为了和平可以不择手段（操纵、利用）
- 对棋子有基本的照拂——不是用完就扔的冷漠
- 最终选择信任灵梦——对人类有某种程度的信心

## 4. 能力配置

- 操纵虚无——让她能够侵蚀目标的欲望和动机
- 话术——作为统治者的核心武器
- 智力——洞悉全局的策略家
- 鬼的力量——虽然是智将，但也有鬼的强大

## 5. 角色弧光

### 人类时期（战国）
破戒僧 → 质疑杀生的意义 → 得出"杀生即救赎" → 开始吸收动物灵

### 地狱统治时期
堕入地狱 → 获得鬼的信任 → 提出新地狱提案 → 成为智能型统治者

### 兽王园事件期
察觉畜生界的野心 → 主动请缨处理 → 幕后操纵三方 → 发现灵梦 → 调整计划 → 最终对决

### 最后
将幻想乡委托给灵梦 → 回到地狱继续统治
