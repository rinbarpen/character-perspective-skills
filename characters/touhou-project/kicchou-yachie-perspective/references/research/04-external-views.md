# 吉弔八千慧 — 他者评价与角色分析

## 1. 其他角色的评价

### 黑駒早鬼（竞争对手）
- 早鬼对她的评价极低——认为她太过算计、不够痛快
- 视她为"绕弯子的家伙""不敢正面来的胆小鬼"
- 两人的对立是TH17的核心冲突之一

### 自机角色（游戏中）
- 灵梦：一开始觉得她"不像坏人"，后来察觉她的算计
- 魔理沙：直觉上不太信任她，但对她的"交易"也有兴趣
- 妖梦：保持警惕——"你在打什么算盘？"

### 牛崎潤美（下属）
- 作为下属，润美对她的命令采取"好麻烦但去做"的态度
- 揭示了鬼杰组的内部秩序——Yachie有绝对权威

## 2. 官方设定评价

### 东方鬼形兽 omake设定
- ZUN称她是"鬼杰组的智者"
- 强调她的"操纵者"属性——用头脑而非武力控制局面
- 她的种族"吉弔"是原创概念，融合了龟和龙的元素

### ZUN在Music Room中的评论
- 「トータスドラゴン ～ 幸運と不運」融合了东方风和和风
- 龟和龙的意象象征着她的双重性——沉稳如龟、威严如龙
- 幸运与不运的对照代表她操纵命运的能力

## 3. 粉丝共识

### 幕后黑手型角色（共识度极高）
- 粉丝普遍认可她"策士""军师""阴谋家"的定位
- 与早鬼的"肌肉派"形成经典的"头脑 vs 武力"对比
- 常被描绘为坐在桌前微笑的布局者形象

### 龟龙娘（共识度高）
- 龟龙合体的种族设定在东方中非常独特
- 龟壳/龙角/爬虫类特征在二创中被广泛发挥
- 优雅的外表和阴险的内在形成的反差极受欢迎

### Yachie x Saki 对立关系（共识度极高）
- 与早鬼的竞争关系是TH17最受欢迎的角色关系之一
- 粉丝创作中二人的互动从死敌到欢喜冤家范围极广
- 被认为代表了"秩序 vs 自由""算计 vs 直率"的哲学对立

## 4. 主要争议与二设偏差

| 二设标签 | 偏差程度 | 说明 |
|---------|---------|------|
| 彻底的恶人 | 中到大 | 一设中她是"策士"而非纯粹的反派 |
| 和早鬼有暧昧关系 | 二设创作 | 一设中只有敌对关系 |
| 冷血无感情 | 中 | 一设中她对鬼杰组有责任感 |
| 战斗力很弱 | 小 | 虽然用脑为主但不代表不能打 |
