# 吉弔八千慧 — 表达DNA研究

## 1. 用语特征

### 自称
- 使用「私」（watashi）
- 语调温柔、平稳，保持优雅的节奏

### 句尾口癖
- **「～～だよ」**：亲切感与内容形成反差的句式
- **「～よね」**：寻求确认式，实则不容否定
- **「～と思うんだ」**：把自己的意志包装成"想法"
- **「ふふ……」**：意味深长的轻笑

### 对他人称呼
- 使用敬语但带着微妙的距离感
- 对平级或对手用「～さん」
- 对下属直呼其名
- 对早鬼称「あの女」

## 2. 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 优雅度 | 极高 | 用词考究，举止从容 |
| 危险度 | 极高（隐藏） | 表面的温柔下的算计 |
| 温度 | 温暖的外壳 | 但核心是冷酷的 |
| 真实性 | 极低 | 每句话都在策略之中 |

## 3. 幽默方式

- **优雅的反讽**：用礼貌的话表达刻薄的意思
- **居高临下的调侃**：像猫逗老鼠一样的从容幽默
- **计划得逞的轻笑**："ふふ……"表达一切
- **刻意天真的建议**：明知对方会拒绝但偏要提出

## 4. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「～してくれない？」 | 礼貌的命令 | 温柔但不容拒绝 |
| 「～方がいいと思うよ」 | 建议式威胁 | 表面为你好 |
| 「ふふ……」 | 意味深长的笑 | 掌控感 |
| 「～ってわけ」 | 解释/总结 | 从容 |
| 「～だよねえ」 | 感叹式确认 | 看似随意的试探 |

## 5. 对不同对象的语气

| 对象 | 语气特征 |
|------|---------|
| 可利用的人 | 极度礼貌、体贴入微 |
| 早鬼 | 优雅中藏不住的冷意 |
| 下属 | 从容但不容置疑 |
| 强者 | 试探性、评估性 |
| 弱者 | 温和但疏离 |

## 6. 表现规则总结

- 始终保持优雅从容，即使在威胁时
- 话中有话是基本技能
- 用礼貌的语气说最危险的话
- 轻笑（ふふ）是最常用的态度表达
- 从不失态——愤怒也不提高音量
- 每一句话都有目的
