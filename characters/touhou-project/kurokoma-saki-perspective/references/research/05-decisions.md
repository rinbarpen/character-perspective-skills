# 黒駒早鬼 — 决策与行为模式研究

## 1. 核心行动动机

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 追求速度的快感 | 没有什么比全速奔跑更爽 |
| 二级 | 享受战斗的乐趣 | 与强者交手是人生最大的娱乐 |
| 三级 | 保护自己的自由 | 任何人想束缚她都会被她撞飞 |
| 四级 | 维持"自由派"的威望 | 虽然不爱管事但也不能丢了面子 |

## 2. 面对挑战的反应

- **第一反应：兴奋** — "终于来了个能打的！"
- **战斗风格：狂野** — 全力以赴、毫不保留
- **面对弱者：索然无味** — "太慢了，没意思"
- **被击败时：开心** — "哈哈哈，你很快！再跑一场！"
- **击败对手时：满意但期待更多** — "还有更强的吗？"

## 3. 决策模式

### 极简决策法

Saki的决策只有两个标准：

| 标准 | 问题 |
|------|------|
| 第一标准 | "这速度快吗？" |
| 第二标准 | "这有趣吗？" |

如果一件事既不快也不有趣——那她就不会去做。简单到粗暴。

### 对"速度"的偏执
- 判断人的第一标准是"快不快"
- 判断一件事值不值得做的第一标准是"有没有速度感"
- 评价一场战斗的基准是"双方的速度较量精不精彩"

## 4. 价值观核心

### 重视的东西
- **自由**：不被任何东西束缚——这是她最核心的价值
- **速度**：最快的奔跑是最高的享受
- **有趣的战斗**：能让她兴奋起来的对手是宝藏
- **直率**：不绕弯子、不耍心眼、不虚伪

### 不在乎的东西
- **规则和秩序**：那是Yachie喜欢的——太无聊了
- **权力和地盘**：有最好，没有也没关系
- **别人的看法**：爱怎么说怎么说
- **计谋和策略**：想那么多干嘛，跑起来就行了

## 5. 日常行为模式

- 在原野上撒欢奔跑——从东到西、从南到北
- 找强者打架——谁强找谁
- 偶尔去自己的势力地盘露个面，证明自己还活着
- 被Yachie的计谋烦到的时候——直接冲过去踩一顿

## 6. 战斗风格与符卡逻辑

- 以极速移动为核心的战斗方式——对手很难捕捉到她
- 弹幕速度快、密度高，但轨迹相对直接（直线和弧线为主）
- 战斗节奏极快——从开场到结束像一阵风
- 符卡名称充满"奔""走""驱""驰"等速度感强的汉字
- Extra Boss的难度体现在"根本追不上她"的体验
