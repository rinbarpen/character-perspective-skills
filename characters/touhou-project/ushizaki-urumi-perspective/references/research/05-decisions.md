# 牛崎潤美 — 决策与行为模式研究

## 1. 核心行动动机

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 节能主义 | "怎么做最不麻烦"是所有判断的第一标准 |
| 二级 | 避免被迫 | 为了避免更大的麻烦而做一些最小限度的行动 |
| 三级 | 被激怒时的反击 | 一旦真正被触怒，会爆发出惊人的战斗力 |
| 四级 | 最低限度的义务感 | 对上方的命令不会反抗但也绝不积极 |

## 2. 面对危险的反应

- **第一反应：嫌麻烦** — "唉，非要打吗"
- **战斗节奏：慢热** — 不会一上来就全力以赴
- **被激怒后的爆发** — 懒散态度的切换点
- **认输时：淡然** — "算了，输了也好，不用再打了"

## 3. 决策模式

### "麻烦度"评判体系

润美用"麻烦度"衡量一切决策：

| 选项 | 麻烦度 | 判断 |
|------|--------|------|
| 立即战斗 | 高 | 尽量回避 |
| 不战而逃 | 中高 | 但会丢面子，可能更麻烦 |
| 认输投降 | 中 | 可行但看情况 |
| 认真战斗 | 极高 | 只在被激怒时才考虑 |
| 懒散应付 | 低 | 常规选项 |

### 被激怒的条件
- 对方的轻蔑或挑衅超出她的容忍底线
- 自己的懒散伪装被彻底看穿
- 触及她真正在意的东西（可能性极低）

## 4. 价值观核心

### 重视的东西
- **自己的节奏**：维持慵懒放松的状态
- **不被过度打扰**：安安静静过日子最好
- **最低限度的尊严**：虽然懒但也有底线

### 不在乎的东西
- **地位和名声**：底层也无所谓
- **胜败**：打赢了也麻烦，打输了也省事
- **他人的期望**：按自己的步调活

## 5. 日常行为模式

- 大部分时间保持半睡半醒的状态
- 被安排任务时以最节能的方式完成
- 随身携带巨大的镰刀——虽然看起来沉重，但对她来说不费力
- 能坐着不站着，能躺着不坐着

## 6. 战斗风格与符卡逻辑

- 利用操纵重量的能力制造"缓慢但无法回避"的弹幕
- 镰刀攻击配合重量变化，平时很慢但一击致命
- 战斗中的节奏变化体现她的内在转变——从慵懒到认真
- 符卡名称多与"牛""重""惰"相关
