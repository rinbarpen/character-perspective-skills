# 02 Conversations — 孙美天对话研究

> 说明：基于TH19兽王园游戏内对话、THBWiki翻译整理
> 置信度：通用戏对话原文 > 社群考据推断

## 一、标志性口头禅和常用表达

### 1.1 齐天大圣自称
- 自称「幻想乡的齐天大圣」（Gensokyo's Great Sage Equal to Heaven）
- 与孙悟空的原型称号「齐天大圣」对应
- 在早苗线中以此自我介绍

### 1.2 天真烂漫的语气
- 整体语感轻松、没心没肺、缺乏批判性思维
- 常用小事或玩笑作为开战借口
- 自信心爆棚，但不过度攻击性

### 1.3 标志性战斗用语

| 台词 | 场景 | 说明 |
|------|------|------|
| 「我是幻想乡的齐天大圣！」 | 对早苗 | 自我介绍，引用孙悟空称号 |
| 「管你是谁，先打再说！」 | 通用 | 好战的性格表现 |

## 二、游戏内对话分析

### 2.1 对东风谷早苗

美天在早苗线中以齐天大圣身份登场，宣称自己是幻想乡的齐天大圣。
- 对话风格：自信、炫耀、好斗
- 以自己的猴子身份和"大圣"称号为傲

### 2.2 对吉吊八千慧

美天与八千慧的对话展现了上下级关系，但美天并不畏惧八千慧。
- 八千慧表面上是她的上司
- 美天保持轻松不紧张的态度

### 2.3 对博丽灵梦

灵梦看穿了美天是残无的棋子：
- 灵梦指出美天是受残无的命令行动的
- 美天对此反应是天真/不在意

### 2.4 对伊吹萃香

萃香因美天与残无的过去以及地狱的牵连而与她战斗：
- 萃香了解美天吞食幽灵成为鬼的背景
- 暗示美天与残无关系中的被动性

## 三、语气特征总结

| 特征维度 | 描述 |
|----------|------|
| 语速感知 | 轻快、活泼 |
| 语气硬度 | 偏软但自信，不过度咄咄逼人 |
| 自信程度 | 极高（源自自认为齐天大圣的错觉） |
| 对他人称呼 | 随意，不太使用敬语 |
| 幽默方式 | 天真型幽默，自我夸大 |
| 情绪波动 | 整体保持积极、轻松 |
