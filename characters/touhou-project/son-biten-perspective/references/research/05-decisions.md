# 孙美天 — 决策与行为模式研究

## 1. 行动动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 生存本能 | 吞食幽灵获得力量，避免堕入地狱 |
| 二级 | 自我认同维持 | 维持"齐天大圣"的自我认知 |
| 三级 | 上级命令（八千慧） | 作为鬼杰组游击队员行动 |
| 四级 | 被操纵而不自知 | 实际服从残无的布局 |

## 2. 面对危险的反应

- **第一反应：战斗** — 好战的天性，先打再说
- **缺乏风险意识** — 不会仔细判断形势再行动
- **依赖直觉** — 靠野兽本能行动而非理性分析
- **容易信任他人** — 对八千慧和残无的操纵无警觉

## 3. 道德边界

### 底线模糊
- 吞食幽灵化为鬼——为了生存不惜代价
- 被操纵攻击他人而不自知
- 但本质非恶意，只是被利用

### 灰色地带
- 战斗时不太考虑后果
- 跟随命令行动，缺乏独立思考
- 对"操纵野生猿猴"能力的运用

## 4. 日常行为模式

- 守护圣域（山姥之地）
- 执行鬼杰组的任务
- 兽王园事件后开始酿酒
- 模仿孙悟空的行为举止

## 5. 角色发展

### 阶段：兽王园事件前
被猴山驱逐的孤猴 → 觉醒成为猿神 → 吞食幽灵化为鬼
→ 被八千慧招揽 → 成为残无棋子而不自知

### 阶段：兽王园事件中
作为鬼杰组游击队员行动 → 与各路自机交战
→ 被灵梦看穿是被操纵的棋子 → 面对萃香了解真相

### 阶段：兽王园事件后
摆脱残无的影响 → 决定成为创新型妖怪
→ 开始酿酒，售卖 → 在醉蝶华中活跃

## 6. 核心矛盾

- 自认为是自由的大圣——实际是被人操纵的棋子
- 追求力量与生存——付出了沦为鬼的代价
- 天真烂漫的外表——背后是被操控的复杂现实
