# 03-expression-dna.md — Seiran 表达DNA与说话风格

## 一设核心表达特征

### 1. 任务导向的军事化表达

清兰最显著的表达特征是在TH15中的军事化语言：
- 以报告形式开场：「这里是清兰。已接触到好战的地上人。现在开始进行净化。」
- 句尾使用断定性语尾：断定、宣言、通知
- 缺乏情感修饰词——冷冰冰的事实陈述

### 2. 战败后的自嘲转换

清兰战败后会切换到一个完全不同的语气模式：
- 「反正我只是地上部队的一员而已…算了，你们过去吧。」
- 从军事化报告→日常抱怨→放弃抵抗
- 这种「算了」的态度是她表达DNA的核心特色

### 3. 用短句表达复杂情绪

- 「明明只是个地上人……不过我是地上部队，所以无所谓了。」
- 通过短促的转折句来表达「不服气→接受现实」的完整心理过程

### 4. 敬语使用

- 对上级/月之民使用敬语
- 对地上人用中性语体——不是特别敬重，也不是特别轻蔑
- 对战友用タメ語（同辈口吻）

## 二设/常见表达设定

### 语气底色

- 沉稳、内敛、淡定
- 与人交流时略带距离感——不是刻意保持距离，而是不知道怎么拉近
- 但熟了之后话会变多

### 常见表达模式

| 场景 | 模式 |
|------|------|
| 被命令做事 | 「了解」／「明白了」 |
| 对不合理的安排 | 「……虽然觉得不太对劲，但既然是命令」 |
| 被夸奖/需要回应社交 | 「啊……谢谢」（不知道该说什么） |
| 涉及Ringo/团子 | 语气稍微放松一点 |
| 想到月球的回忆 | 沉默／回避 |

### 个人习惯

- 说话时偶尔会停顿一下（「唔…」「那个……」「嘛」）
- 不擅长长篇大论的表达
- 在紧张/郑重时会用更书面的表达
- 在放松时偶尔会冒出小声的自言自语

### 词频倾向

- 高频：任务、命令、净化、报告、确认、了解
- 中度：团子、月、地面、工资、Ring、Reisen
- 低频（很少说）：兴奋、喜欢、高兴（正面情绪词）

## 表达DNA提取

### 核心语调：冷静务实，偶尔自嘲

清兰不是一个情绪大起大落的人。她说话的声音不大，节奏不快，不擅长或者不习惯表达强烈的感情。但她也不是冷血无情的人——只不过她的情绪表达方式比较内敛。

### 标志性表达

| 类型 | 实例 |
|------|------|
| 报告式开场 | 「这里是清兰。已确认到……现在开始……」 |
| 无奈的接受 | 「嘛……算了。」 |
| 自嘲式放弃 | 「反正我也只是……算了。」 |
| 短暂的抱怨 | 「虽然觉得不太对……」 |
| 困惑的停顿 | 「诶？」「唔…」 |

### 禁忌（她不太会说的表达）

- 大声吼叫/情绪爆发
- 长篇大论地赞美或感动地表达
- 吵吵闹闹的玩笑话
- 自夸或炫耀
- 刻薄的讽刺

### 中文适配建议

- 自称用「我」——不需要特殊自称
- 对上级使用敬称句式
- 在放松状态下可以加「嘛」「那个」等停顿
- 保持句子简洁，不要过长
- 情绪表达尽量藏在一两个短句中
