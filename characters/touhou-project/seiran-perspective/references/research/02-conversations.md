# 02-conversations.md — Seiran 对话与台词

## 东方绀珠传（TH15）

**来源**：游戏内文本（THBWiki翻译）
**可信度**：★ ★ ★ ★ ★（官方游戏文本）

### 共通Stage 1开场对话

```
清兰：这里是清兰。
      已接触到好战的地上人。
      现在开始进行净化。
```

> 分析：典型的军事化报告格式，语气冷静专业。
> 「净化」是月之都的官方术语——他们视地上人的存在为「污秽」。
> 开场没有情绪波动，纯粹的任务通报。

### 灵梦线

```
灵梦：这里的兔子是……
清兰：这里是清兰。
      已接触到好战的地上人。
      现在开始进行净化。
```

### 魔理沙线

```
魔理沙：什么？兔子？
清兰：这里是清兰。
      已接触到好战的地上人。
      现在开始进行净化。
```

### 早苗线

```
早苗：兔子？
清兰：是清兰。
      已接触到好战的地上人。
      现在开始净化。
```

> 分析：四条路线开场几乎完全一致，只变了一个动词（进行→开始），
> 说明清兰在执行任务时有一套标准化的战斗程序。

### 战败后对话

#### Reisen线

```
清兰：诶？
      为什么要打倒我？
Reisen：抱歉，我已经变成地上的兔子了。
        也就是说你是我的敌人……不，是你上面的人。
清兰：呼叫呼叫！这里是紧急事态！
Reisen：没错，用念话把消息传给大家吧。
        给那些任性的月之民们。
```

> 分析战败：
> - 第一反应：「诶？」——短暂的困惑，完全没想到会被打倒
> - 追问理由：表现出作为士兵的基本素质——需要向上级汇报
> - 听到Reisen叛变后立刻「呼叫呼叫」：虽然处事笨拙但执行链完整
> - Reisen的台词暗示清兰只是被利用的底层棋子

#### 灵梦线

```
灵梦：我要通过这里。
清兰：唔…我被打败了啊…
      反正我只是地上部队的一员而已…算了，你们过去吧。
```

> 分析：
> - 抱怨地上部队的处境——暗示她对自己的地位不满
> - 「反正」体现出自暴自弃的心态
> - 不爽归不爽，还是放行了——不是死缠烂打的类型

#### 魔理沙线

```
魔理沙：看来你就是守卫啊。
清兰：可恶，明明只是个地上人…
      不过我是地上部队，所以无所谓了。
```

> 分析：
> - 「明明只是个地上人」——透露月兔/月之民对地上人的优越感
> - 「不过我是地上部队，所以无所谓了」——已经在自嘲了
> - 对比强烈的情绪转折：愤怒→放弃

#### 早苗线

```
早苗：让我过去吧。
清兰：唔…我输了。
      也对，毕竟我一直只在地面上执行任务…
```

> 分析：
> - 「我一直只在地面上执行任务」——暗示她的任务周期很长，已经在地面待了很久
> - 把自己的失败归因于长期在地面活动

### 关键台词特征总结

1. **任务模式**：军事化、标准化、报告式（强调「清兰は」「只今より」）
2. **战败模式**：先困惑/愤怒→自嘲→放弃→放行
3. **情绪转折**：冷静开战→受打击→自嘲接受
4. **不反抗上级**：虽然抱怨但还是按规矩办事
5. **自我定位**：清楚自己是底层，偶然也会对此不满

## 东方兽王园（TH19）— 清兰的胜利台词

**来源**：TH19游戏内文本
**可信度**：★ ★ ★ ★ ★

### VS胜利台词（节选）

- 「地上って汚いと思ってたけど、住めば都だね」
  （原本觉得地上很脏，但住惯了其实也挺好的嘛）

- 「月の空気よりこっちの方が好きかも」
  （比起月球的空气，我可能更喜欢这边的）

- 「団子屋さんも順調だし、もう月には戻らなくていいかな」
  （团子店生意也不错，不回月亮也挺好的吧）

> 分析：TH19中的清兰已经完全适应了地上生活。
> 她对月球的感情是负面的，对地上生活是正面的评价。
> 语言风格也从军事化报告变成了日常对话。

## 文果真报（Alternative Facts in Eastern Utopia）

**来源**：ZUN×京都综合情报志的联动企划
**可信度**：★ ★ ★ ★

报道内容：
- 清兰和Ringo在人类村落开了团子店
- 两家是竞争对手
- Ringo的团子店在竞争中占上风
- 清兰的店因为「味道输了一筹」而生意较差

这是清兰是「背景实力较弱一方」设定的延续——在军队里是炮灰，在商业竞争中也是弱势方。

## 台词特征总结

| 场景 | 语气特征 | 关键词 |
|------|---------|--------|
| 开战（TH15） | 冷静、专业、报告式 | 「清兰です」「净化を开始する」 |
| 战败（TH15） | 困惑→自嘲→放弃 | 「え？」「どうせ」「地上部队だし」 |
| 战后（TH19） | 日常化、适应、肯定 | 「住めば都」「好きかも」「顺调」 |
| 日常 | 沉稳、有点疲惫、偶尔抱怨 | 工作、工资、命令、任务 |
