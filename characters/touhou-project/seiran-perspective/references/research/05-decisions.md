# 05-decisions.md — Seiran 关键行为与决策

## 决策1：执行入侵任务（TH15）

| 项目 | 内容 |
|------|------|
| **时间** | 绀珠传事件期间 |
| **背景** | 接到来自月之都的命令，作为Eagle Ravi的先遣部队潜入幻想乡 |
| **行为** | 设置基地、执行净化任务、接触并攻击地上人 |
| **决策模式** | 服从命令型——没有犹豫、没有质疑、直接执行 |
| **结果** | 被击败 |

**分析**：
清兰在这一阶段完全是「工具人」模式。
她没有表现出对任务正当性的质疑，也没有表现出对地上人的同情。
这既是士兵的本分，也可能是因为她的处境不允许她有更多选择。

---

## 决策2：战败后放行（TH15）

| 项目 | 内容 |
|------|------|
| **时间** | 被击败后立即 |
| **行为** | 虽然抱怨，但选择放行自机 |
| **理由** | 「反正我只是地上部队的一员」 |
| **决策模式** | 放弃抵抗→接受现实 |

**分析**：
这个决策揭示了两点：
1. 清兰本质上不是一个残暴的人——打到差不多就放了
2. 她对任务的责任感有限——不是「不胜利就不罢休」的类型

---

## 决策3：叛逃月之都，定居幻想乡（TH15后）

| 项目 | 内容 |
|------|------|
| **时间** | 绀珠传事件后、兽王园事件前 |
| **行为** | 未返回月之都，选择留在地面 |
| **后续** | 在人类村落开团子店 |
| **决策模式** | 选择自由→追求平静生活 |

**分析**：
这是清兰最关键的人生决策。
从军事角度来看，叛逃意味着她无法再回到月之都，放弃了作为月兔士兵的一切。
从个人角度来看，这是她从「被安排的棋子」变成「选择自己生活」的转折点。

TH19中的台词证实了她对自己的选择感到满意。

---

## 决策4：在兽王园事件中出动（TH19）

| 项目 | 内容 |
|------|------|
| **时间** | 兽王园事件期间 |
| **行为** | 携带月之宝玉出门，试图在幻想乡制造「月兔乐园」 |
| **动机** | 创造一个适合月兔居住的地上环境 |
| **决策模式** | 主动型——不是为了任务，而是为了自己和同族的未来 |

**分析**：
相比TH15阶段的「被命令才行动」，
TH19的清兰已经转变为「为理想而主动行动」。
这是一个重要的角色发展阶段。

---

## 决策5：与Ringo的团子店竞争

| 项目 | 内容 |
|------|------|
| **行为** | 在人类村落开设团子店「清兰屋」 |
| **竞争关系** | 与Ringo的团子店直接竞争 |
| **结果** | 味道不如Ringo，竞争处于劣势 |
| **决策模式** | 积极经营但能力有限 |

**分析**：
清兰是「努力了但结果不理想」的典型。
在军队中是炮灰、在商业竞争中也是弱势方——这种设定的一致性很强。

---

## 决策总结

| 决策类型 | 特征 | 案例 |
|---------|------|------|
| 服从型 | 不加质疑地执行命令 | TH15入侵任务 |
| 放弃型 | 意识到无力抵抗后放弃 | 战败后放行 |
| 选择型 | 主动选择自己的道路 | 叛逃月之都 |
| 理想型 | 为自己和同族的未来行动 | TH19携带月之宝玉 |
| 努力型 | 虽然能力有限但坚持尝试 | 经营团子店 |

**角色弧光**：从一个被动的棋子→逐渐找回主动性→选择自己的生活方式。
