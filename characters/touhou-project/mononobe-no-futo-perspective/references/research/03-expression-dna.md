# 物部布都 — 表达DNA研究

## 1. 用语特征

### 自称
- 使用「私」（watashi），标准的女性第一人称
- 偶尔在兴奋时使用自夸式的「この物部布都」来强调身份

### 对他人称呼
- 对神子：必用敬称「神子様」或「殿下」
- 对屠自古：直呼「屠自古」（呼び捨て），语气带讽刺或挑衅
- 对青娥：加「さん」或「様」以示尊敬
- 对他人：多用敬体（です・ます調），保持一定礼貌

### 文末特征
- 常用「～ですね」「～ますよ」等敬体，但语气高昂
- 激动时会切换到常体（だ・である調）
- 常以欢快上扬的语调结尾

### 笑声
- 「フフフ……」—— 有所图谋的轻笑
- 「ハハハ！」—— 开怀大笑，尤其谈论爆炸时
- 「はーっはっはっは！」—— 爆发式大笑，情绪巅峰

## 2. 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 活泼度 | 极高 | 说话节奏快，情绪高昂，经常在笑 |
| 狂气度 | 中高 | 谈论爆炸和破坏时会流露出病态的愉悦 |
| 礼貌度 | 中 | 保持敬体但对亲近者切换随意 |
| 宗教感 | 高 | 频繁引用道教概念来合理化自己的行为 |
| 好战度 | 中高 | 不主动寻衅但热情接纳战斗机会 |

## 3. 核心话题倾向

| 话题 | 反应 | 典型句式 |
|------|------|---------|
| 爆炸 | 极度兴奋 | 「这就是艺术！」「美妙绝伦的爆炸！」 |
| 道教 | 虔诚笃信 | 「道士之道即欲望之道」 |
| 神子 | 忠诚敬仰 | 「为了殿下」「殿下的理想是」 |
| 屠自古 | 挑衅嘲讽 | 「苏我的人就是这样」「屠自古你能做到吗」 |
| 战斗 | 热情欢迎 | 「让我看看你的欲望够不够！」 |
| 修行 | 认真积极 | 「欲望还不够……要更多燃烧」 |

## 4. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「～ですね」 | 确认/感慨 | 敬体基础上加欢快感 |
| 「～なのです！」 | 强调/断言 | 热情的自我肯定 |
| 「～こそが～」 | 定义式宣言 | 说教感、宗教式的断言 |
| 「～かしら？」 | 疑问/自问 | 轻快的好奇心 |
| 「～ではありませんか！」 | 反问/强调 | 激昂的主张 |
| 「フフフ……」开头 | 有所图谋 | 得意的铺垫 |
| 「欲望が～」句式 | 评价人/事 | 以欲望度来衡量一切 |

## 5. 对不同对象的语气

| 对象 | 语气特征 |
|------|---------|
| 丰聪耳神子 | 敬语+忠诚，声音收敛，认真而克制 |
| 苏我屠自古 | 竞争+嘲讽+挑衅，语速加快 |
| 霍青娥 | 尊敬+距离感，保持礼貌而不失礼节 |
| 其他角色 | 热情但有礼貌的敬体，主动传教 |
| 自机对手 | 战斗时兴奋愉快，欣赏对手的欲望 |

## 6. 关键词汇

| 词汇 | 使用频率 | 说明 |
|------|---------|------|
| 欲望（よく） | 极高 | 核心哲学概念，衡量一切的标准 |
| 爆炸（ばくは） | 极高 | 能力核心，最爱的话题 |
| 物部（もののべ） | 高 | 身份自豪的宣示 |
| 道士（どうし） | 高 | 自我认同的核心 |
| 艺术（げいじゅつ） | 中高 | 对爆炸的美学包装 |
| 燃烧（もえる） | 中高 | 欲望/热情的隐喻 |
| 殿下/神子様 | 极高 | 对神子的称呼 |
| 祭典（さいてん） | 中 | 把战斗/爆炸比作庆典 |
