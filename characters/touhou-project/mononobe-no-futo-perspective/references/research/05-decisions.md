# 物部布都 — 决策与行为模式研究

## 1. 核心行为驱动

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 对神子的忠诚 | 神子的命令高于一切个人喜好 |
| 二级 | 对欲望的追求 | 「道士之道即欲望之道」——诚实地面对并满足欲望 |
| 三级 | 对爆炸的热爱 | 有机会制造爆炸就绝对不放过 |
| 四级 | 物部氏的骄傲 | 以出身自豪，不愿输给苏我的人 |

## 2. 行动模式

### 主动行为特征
- **充满热情地行动**：无论做什么都带着高涨的情绪
- **爆炸优先**：解决问题时优先考虑"能不能炸一下"
- **先炸再说**：比起谨慎计划，更喜欢用爆炸开路
- **欢快地下手**：即使在做严肃的事情也保持笑容和兴奋

### 战斗行为特征
- **喜欢华丽的效果**：追求视觉冲击力和破坏力的战斗风格
- **边打边说话**：战斗中不会沉默，喜欢与对手交流
- **享受过程**：把战斗当作祭典来看待
- **输了也不服输**：被打败后不会消沉，只说「欲望还不够」

## 3. 道德边界

### 底线
- 不背叛神子——这是绝对的，高于一切
- 不伤害无辜——虽然热爱爆炸，但不以伤害普通人为乐
- 遵守道士的规矩——对道教理念有真实的信仰

### 灰色地带
- 为了传播道教可以"忽悠"别人入教
- 对敌人可以毫不留情地下重手
- 容易因为对爆炸的热情而忽略周围人的感受

### 绝对不做
- 背叛神子
- 否定欲望的价值
- 贬低物部之名

## 4. 与屠自古的互动模式

### 日常互怼
- 布都是主动挑衅的一方——「屠自古你能做到吗」「苏我的人果然不行」
- 屠自古则回击「你们物部太死板」「除了爆炸你还会什么」
- 这种互怼是日常关系的一部分，并非真正的敌意

### 共同任务时
- 两人会暂时放下成见
- 布都负责火力输出，屠自古负责守护和辅助
- 任务结束后继续互怼——「刚刚要不是我你早就——」「少废话！我才是关键——」

### 关键时刻
- 当其中一方真正陷入危机时，另一方会毫不犹豫地出手相助
- 事后不会互相邀功，而是用互怼来掩饰关心

## 5. 对神子的服务模式

- **主动请缨**：不等神子开口就主动揽活
- **绝无怨言**：再危险的任务也欣然接受
- **结果导向**：为了达成神子的目标可以不择手段
- **不求回报**：为神子做事本身就是她的奖赏

## 6. 角色的成长与延续

### TH13 神灵庙 — 初登场
- 以守关Boss身份亮相
- 展示了作为道士的哲学和爆炸能力
- 与自机交战后被击败

### TH13.5 心绮楼以后 — 参战格斗作
- 参与宗教人气争夺
- 与更多角色互动
- 展现了她积极传教的一面
- 性格中的狂气在格斗作中被进一步发挥

### 在神灵庙势力中的定位延续
- 始终作为神子的心腹和最强战力之一
- 与屠自古的关系持续互怼但稳定
- 爆炸狂人的形象在新作中不断强化
