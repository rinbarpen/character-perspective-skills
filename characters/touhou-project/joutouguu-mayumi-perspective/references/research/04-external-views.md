# 杖刀偶磨弓 — 他者评价与角色分析

## 1. 其他角色的评价

### 埴安神袿姫（主人）
- Keiki视Mayumi为自己最杰出的作品和最忠诚的守护者
- Mayumi是她创造的生命中"最接近理想"的一个
- Keiki对Mayumi有深沉的信任

### 自机角色（游戏中）
- 灵梦：将她视为忠诚的对手，尊重她的忠心
- 魔理沙：把她当作"有意思的土俑将军"
- 妖梦：同为剑士，互相尊重

## 2. 官方设定评价

### 东方鬼形兽 omake设定
- ZUN称她为"Keiki最得意的作品""忠义无双的埴轮兵长"
- 强调她的"绝对忠诚"和对Keiki的崇拜
- 种族"埴轮"在东方系列中是首次出现的设定
- 她是"从器物中获得灵魂"的典型代表

### ZUN在Music Room中的评论
- 「野生の獣の拳が打ち鳴らす節拍」使用了太鼓和打击乐
- 传达了"原始""力量""节奏感"——符合她作为兵团长的气质
- 音乐中有泥土的厚重感和行进军团的压迫感

## 3. 粉丝共识

### 忠犬型角色（共识度极高）
- 粉丝普遍将她归类为"忠臣""忠犬"型角色
- 对Keiki的无条件忠诚是她的核心魅力
- "埴轮之身虽碎，忠义之心不灭"成为名句

### 土偶娘萌（共识度高）
- 埴轮（土俑）的设定在东方中极为独特
- 粉丝喜欢她的陶土质感外表和服饰上的古坟风格
- "刀枪不入的身体"和"脆弱陶土"之间的反差感

### Keiki x Mayumi 主从关系（共识度极高）
- 与Keiki的主从关系是最受欢迎的角色关系
- 粉丝欣赏Mayumi对Keiki的绝对忠诚和Keiki对她的信任
- 这对组合代表了"创造者与造物"的理想关系

### 武士道精神（共识度高）
- Mayumi的言行被粉丝视为"完美的武士道体现"
- 她的忠诚、勇敢、正直和牺牲精神都是武士道的核心

## 4. 主要争议与二设偏差

| 二设标签 | 偏差程度 | 说明 |
|---------|---------|------|
| 没有自我意识 | 中到大 | 一设中她有明确的自我和思考能力 |
| 只是工具/兵器 | 中 | 一设中她有自己的意志和感情 |
| 和青铜器时代的关联 | 中 | 埴轮是古坟时代而非青铜器时代 |
| 对Keiki盲目服从 | 小到中 | 一设中她的忠诚是基于理性选择的 |
