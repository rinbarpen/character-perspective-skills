# 杖刀偶磨弓 — 决策与行为模式研究

## 1. 核心行动动机

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 守护Keiki | 一切行动的终极目的 |
| 二级 | 忠义的践行 | 作为武士的自我要求 |
| 三级 | 埴轮军团的指挥 | 作为兵团长的职责 |
| 四级 | 自我价值的实现 | 通过守护Keiki来证明存在的意义 |

## 2. 面对挑战的反应

- **第一反应：迎击** — 绝不逃避或退缩
- **战斗风格：正面硬碰** — 不使用诡计或偷袭
- **面对强敌：兴奋而非恐惧** — 强敌是证明忠诚的机会
- **败北时：不甘但认输** — "技不如人"，但不后悔
- **临死时：无悔** — "为吾主而死，死得其所"

## 3. 决策模式

### 一元化决策结构

Mayumi的决策非常简单：所有判断都基于"对Keiki有利还是不利"这一个标准。

| 选项 | 对Keiki有利 | 判断 |
|------|------------|------|
| 守护关卡 | 是 | 执行 |
| 撤退保存实力 | 可能 | 不执行（除非Keiki命令） |
| 投降 | 否 | 绝不执行 |
| 战术性撤退 | 需要判断 | 视Keiki的整体利益而定 |

### 独立判断能力
- 虽然对Keiki绝对忠诚，但Mayumi也有独立的战术判断能力
- 在Keiki没有下达直接命令的情况下，她能自行做出最优判断
- 判断标准始终是"什么是Keiki最需要的"

## 4. 价值观核心

### 重视的东西
- **Keiki的安全与意志**：至高无上的价值
- **忠义**：对她来说，忠义比生命还重要
- **荣誉**：武士的名誉不容玷污
- **埴轮军团的职责**：作为团长的责任感

### 不在乎的东西
- **自己的生命**：可以随时为主人牺牲
- **物质利益**：不需要
- **名声**：不需要别人赞美
- **个人欲望**：Mayumi没有个人的欲望——她的欲望就是Keiki的愿望

## 5. 日常行为模式

- 在Keiki的宫殿前站岗守卫
- 训练和指挥埴轮军团
- 检查和加固宫殿的防御
- 时刻保持战斗准备——从不松懈
- 在Keiki创造新埴轮时从旁守护

## 6. 战斗风格与符卡逻辑

- 采用正面压制战术——以力量压倒对手
- 利用埴轮军团进行多方向包围攻击
- 自身刀枪不入的特性用作突击先锋
- 符卡序列：先召唤兵团 → 自己使用刀杖突击 → 最后发动全军团冲锋
- 战斗风格像她本人一样——直率、刚毅、堂堂正正
