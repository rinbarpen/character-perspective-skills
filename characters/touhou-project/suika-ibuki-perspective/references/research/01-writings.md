## 2. 种族设定：鬼族

### 2.1 鬼族在东方Project中的定位

鬼是东方Project中最高等级的妖怪种族之一，位于妖怪金字塔的顶端。根据《东方求闻史纪》记载：

- 鬼族是「最强的人类掠食者」，以捕食人类为生
- 远古时期生活在妖怪之山，构筑了役使天狗的社会体系
- 鬼族会定期捕捉村落中的人类，这是「恐怖心」的象征
- 性格爽朗但凶猛，一言不合便大打出手
- 一旦认同对方为伙伴就仁至义尽，绝不做背信弃义之事
- **最厌恶之事：扯谎、卑鄙行为**——鬼族誓死恪守约定
- 嗜酒，有千杯不倒的酒力
- 精神经历过严苛磨练，精通禅学的鬼并不少见

**来源：** 东方求闻史纪「鬼」条目、东方萃梦想附带文档

### 2.2 鬼族离开幻想乡的原因

根据《东方求闻史纪》：
> 数百年前鬼族数量开始减少，大结界张开后近乎消失。它们对「能退治自己的人类匮乏、生活优哉游哉的幻想乡生活」很是不满。

根据《东方萃梦想》附带文档：
> 鬼喜欢和人类进行**堂堂正正的比赛**（格斗、酒量等），但人类却用卑鄙的计策捕捉鬼。信赖关系被人类单方面瓦解，鬼族抛弃了人类。

两种记述角度不同，但共同指向：鬼族因对人类的失望而离去。

**来源：** 东方求闻史纪、东方萃梦想附带文档

### 2.3 萃香作为「再次出现的鬼」

萃香是鬼族从幻想乡消失后，**第一个再次出现的鬼**。由于幻想乡驱鬼的方法已然失传，没有人能退治鬼，萃香因此得以在幻想乡定居。

**来源：** 东方求闻史纪

## 3. 鬼族四天王（山之四天王）

### 3.1 四天王的设定

伊吹萃香是**鬼族四天王（山之四天王）**之一。四天王的设定源自日本酒吞童子传说的四天王体系。

已知成员：

| 称号/对应 | 角色 | 能力 | 首次登场 | 状态 |
|-----------|------|------|---------|------|
| 疏密（怪） | 伊吹萃香 | 操纵密与疏程度的能力 | 东方萃梦想 | 地上活动 |
| 怪力（力） | 星熊勇仪 | 操纵怪力乱神程度的能力 | 东方地灵殿 | 旧地狱·旧都 |
| 腕（乱） | 茨木华扇 | ？？？（现为仙人） | 东方茨歌仙 | 妖怪之山修行 |
| 第四位（神） | 未登场 | 不明 | — | 未公开 |

四天王曾一同住在妖怪之山，如今已经各自分散。萃香曾对勇仪说：「有机会还想再和你们一起四个人去登山呢」。

**「怪力乱神」** 的对应关系在粉丝考据中已形成广泛共识，但官方仅确认了萃香与勇仪为四天王成员，华扇的身份为广泛推测（原典为茨木童子）。

**来源：** 东方地灵殿游戏对话、东方非想天则符卡「四天王奥义·三步坏废」/「四天王奥义·三步必杀」、东方求闻口授

### 3.2 与其他四天王成员的关系

**星熊勇仪：**
- 同为鬼族四天王，好友兼酒友
- 萃香评价：「勇仪虽然咒术系的力量比我弱，但身体使出来的力量大概比我强，不过酒量没法跟我比」
- 两人符卡名对称：萃香的「四天王奥义·三步坏废」vs 勇仪的「四天王奥义·三步必杀」
- 勇仪戴的是单角，萃香是双角，体现不同的鬼族类型

**茨木华扇：**
- 同为山之四天王，旧识
- 华扇现在是仙人，手臂已失，关于过去发生了什么，两人之间似乎有某种微妙的距离感
- 萃香的伊吹瓢和茨木华扇的锁链在视觉上有呼应之处

**来源：** 东方求闻口授、东方地灵殿对话

## 4. 能力详解：操纵密与疏程度的能力

### 4.1 能力的本质

萃香可以自由操纵万物的「疏密」和「浓淡」，这是一个兼具具象和抽象层面的强大能力。

### 4.2 具象应用

| 应用方向 | 具体效果 | 备注 |
|---------|---------|------|
| **让密度变疏（扩散）** | 事物化为雾状 | 可将自己化为妖雾，用于监视、偷窥或回避攻击 |
| **让密度变大（聚集）** | 事物产生高温甚至熔化 | 可以聚集沙尘为巨石、压缩大气为真空 |
| **自身分裂** | 拔下头发吹气即可化为分身 | 可制造无数小分身（「鬼之百鬼夜行」） |
| **自身巨大化** | 聚集自身密度变大 | 身高力量同步增长 |
| **聚集物体** | 将细小灰尘/庞大物体聚集 | 日常便利用途（灵梦评价） |
| **创造黑洞** | 高度压缩物质/空间 | 符卡「萃鬼」/「天手力男之命」 |
| **修复天空** | 聚集被击碎的夜空 | 曾击碎映月之夜幕又在一夜之间修复 |

### 4.3 抽象应用

ZUN在采访中明确说明，萃香的能力不仅是物理层面的密度操纵，还可以操纵：

- **「人们的心念」的密度**——聚集人心来开宴会
- **「思想的疏密」**——让人神志不清或高度专注
- **「存在的密度」**——让自身的存在感变弱（透明化）或变强

这也是她能够成为宴会核心的原因：并非单纯用力量强迫人聚集，而是让人（的心）自然而然地被吸引而来。

### 4.4 能力的限制

- 距离限制：对远距离目标的密度操纵精度下降
- 「密」与「疏」是对立统一——过于聚集会导致高温甚至燃烧，过于扩散则无法维持实体
- 醉了的时候能力精确度会下降（虽然她一直处于微醺状态）

**来源：** 东方求闻史纪、ZUN访谈、东方萃梦想游戏文本

## 5. 伊吹瓢（鬼宝）

### 5.1 基本信息

伊吹瓢是萃香随身携带的紫色酒葫芦，名字源自伊吹山和传说中的酒吞童子。

| 特性 | 说明 |
|------|------|
| 容量 | **无限涌出酒**——永远喝不完 |
| 机制 | 葫芦内有「酒虫」（一种能将水转化为美酒的灵虫） |
| 酒力 | 酒非常烈，能给鬼之外的种族造成幻觉甚至影响记忆 |
| 防倒 | 装有防止翻倒的制动装置，一次倒出的酒量有限（figma设定） |
| 战斗用途 | 伊吹瓢本身也可当武器使用（挥砸/喷酒） |
| 象征 | 葫芦形状在东方文化中本身就是鬼的象征物之一 |

### 5.2 命名由来

「伊吹」取自伊吹山——传说中酒吞童子的出生地。伊吹瓢的设计参考了日本民间传说中鬼随身携带酒葫芦的形象。

**来源：** THBWiki、东方求闻史纪、东方书谱

## 6. 角色外观设计

### 6.1 基本外貌

- **发色：** 浅茶色（淡绿色系）长直发，发梢处扎成一束
- **瞳色：** 深红色
- **体型：** 幼儿般的身材，非常矮小
- **角：** 头部左右生有一对与身高极不相称的既长又弯曲的角，左角上系着紫色丝带

### 6.2 服装

- **上衣：** 白色无袖上衣（部分作品中为改良型）
- **裙子：** 紫色长裙，双层设计，上层有半透明感
- **头饰：** 后脑扎着红色大蝴蝶结
- **锁链：** 腰间用锁链坠着三个金属几何体——三角锥(△)、球(○)、立方体(□)
- **葫芦：** 紫色伊吹瓢，始终提在手中或系在腰间

### 6.3 三个几何体的含义（ZUN原话）

> 「○是无，也就是扩散；△是调和，也就是萃集本身；□是不变，代表自身。」

- **三角锥（△）** = 调和 / 萃集（密）
- **球体（○）** = 无 / 扩散（疏）
- **立方体（□）** = 不变 / 自身

这三个几何体是萃香施展能力的触媒/象征物。

### 6.4 锁链的意义

ZUN在东方书谱中说明：「虽然看起来不像鬼，但是我觉得锁链是鬼的象征（笑）」。

**来源：** 东方书谱、THBWiki设定画页面

## 7. 性格特征

萃香的性格由以下几个核心特质构成：

### 7.1 开朗与嗜酒

- 无比喜欢喝酒，随身携带能无限涌出酒的伊吹瓢
- 几乎无时无刻不在灌酒
- 自称有千杯不倒的酒量，勇仪来了也比不过
- 会因宴会减少而感到不满（这是萃梦想事件的核心动机）

### 7.2 天真与单纯

- 非常孩子气，喜欢热闹
- 好奇心旺盛
- 不怎么说谎（鬼族的种族特性）
- 被评价为鬼中的「异端儿」——因为她不够诚实正直（反而说明她的言行有欺诈性，这在鬼中很奇怪）

### 7.3 好斗与爱比试

- 喜欢与人一较高下，不论格斗还是比酒量
- 享受战斗本身，而非胜负
- 面对弱者会感到没意思但不会欺辱

### 7.4 孤单与怀旧

- 在热闹背后，萃香实际上有种深层的孤独——她是唯一留在地上活动的鬼
- 怀念鬼族还在妖怪之山的时代（「还想再四个人一起去登山」）
- 通过不断地举办宴会来填补某种空虚

### 7.5 观察者倾向

- 比表面看起来更敏锐
- 在宴会上观察每个人的行为细节（从萃梦想各Stage开场白可见）
- 对人际关系有深刻的洞察——她对咲夜、爱丽丝、妖梦、帕秋莉、魔理沙的评价都十分精准

**来源：** 东方求闻史纪、东方萃梦想故事文本、东方地灵殿对话

## 8. 与西瓜（西瓜）的关联

「萃香」的日文读音「すいか」与「西瓜（水瓜）」完全相同。虽然汉字不同，但这一读音梗被广泛运用：

- 粉丝爱称萃香为「西瓜」或「瓜」
- 二创中西瓜元素极其常见（西瓜头饰、西瓜图案服装等）
- ZUN本人也承认这是一个有意的谐音设计，但并非角色核心设定
- 并非所有语言版本中该谐音都成立（但中文语境中「西瓜」称呼已固化为通用爱称）

## 9. 原型考据：酒吞童子

伊吹萃香的原型毫无疑问是日本传说中最著名的鬼——**酒吞童子（Shuten-douji）**。

### 对应关系

| 传说要素 | 东方Project对应 |
|---------|----------------|
| 酒吞童子的姓氏「伊吹」 | 伊吹萃香的姓氏 |
| 酒吞童子出生地伊吹山 | 名字由来 |
| 酒吞童子的大江山 | 萃香符卡「鬼气」系列 |
| 酒吞童子的四天王 | 鬼族四天王设定 |
| 酒吞童子被斩首 | 萃香可通过能力重新聚集身体（不死性） |
| 酒吞童子好酒 | 萃香嗜酒 |
| 酒吞童子的酒葫芦 | 伊吹瓢 |

ZUN在设计萃香时明确以酒吞童子为原型，但做了大量改编：
- 将男性鬼王女性化
- 保留了豪爽嗜酒的特性
- 赋予了操纵密度的独特能力（并非单纯的力量型）
- 加入了「回归幻想乡重建人鬼关系」的深层动机

**来源：** ZUN访谈、THBWiki考据、喵玉殿论坛考据帖
