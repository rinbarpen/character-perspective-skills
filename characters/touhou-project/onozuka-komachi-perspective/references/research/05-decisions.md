# 小野塚小町（Onozuka Komachi）关键行为与决策分析

> 基于TH09、TH13.5、TH14.5、TH15.5的剧情文本整理

---

## 一、持续决策模式：摸鱼

### 决策1：日常选择摸鱼——持续的"偷懒"生活方式

| 属性 | 内容 |
|------|------|
| **类型** | 持续性主动决策 |
| **作品** | 全作品 |
| **背景** | 小町是三途河的摆渡人，负责将死者运送到冥界。她平时应该在河岸待命。 |
| **决策内容** | 一有机会就偷懒——睡觉、逛花田、喝酒、聊天，总之就是不工作 |
| **动机** | 底层原因未明确说明，但呈现为"纯粹的惰性+认为没必要那么拼命" |
| **后果** | 导致亡灵积压，三途河两岸效率低下，映姬一次又一次地来抓人 |
| **主动/被动** | 主动决策——她清楚地知道自己应该工作，但选择不干 |

**解读**：小町的"摸鱼"不是被动进行的——她是有意识地选择了不努力工作。从TH09开始，她的所有行为几乎都由这个核心决策驱动。

---

### 决策2：面对映姬的说教——"认错但不改"策略

| 属性 | 内容 |
|------|------|
| **类型** | 持续性决策模式 |
| **作品** | TH09、TH13.5、TH14.5、TH15.5 |
| **背景** | 映姬发现小町在偷懒，前来追究责任 |
| **决策内容** | 小町采取了"先说借口→被拆穿→口头认错→下次继续"的循环策略 |
| **典型模式** | 1. 先否认偷懒（"我只是在休息/我正要去干活"） → 2. 被映姬识破后道歉（"是我错了"） → 3. 承诺改正（"下次一定"） → 4. 映姬走后继续摸鱼 |
| **动机** | 避免正面冲突——她知道映姬不吃硬的，所以用"软对抗"的方式应对 |
| **后果** | 与映姬建立了独特的"抓捕摸鱼→说教→和好→再抓"的关系循环 |

**解读**：小町的"认错但不改"是一种聪明人的做法——她知道硬刚不过上司，不如面上认怂、背后照旧。这也说明她并不惧怕映姬的惩罚，因为她知道自己的核心工作是必要的，映姬不会真的开除她。

## 二、关于工作职责的决策

### 决策3：在工作中与亡灵聊天——把工作节奏调慢

| 属性 | 内容 |
|------|------|
| **类型** | 主动决策 |
| **作品** | TH14.5对话提及 |
| **背景** | 作为摆渡人，她的职责是把亡灵从现世运送到冥界，越快越好以缓解"拥堵" |
| **决策内容** | 小町选择在摆渡过程中和亡灵聊天、不紧不慢地划船，而不是尽快完成工作 |
| **动机** | 她认为死亡已经发生了，急也没有用；和亡灵聊聊天更能让他们安心上路 |
| **后果** | 工作进度慢——但也让不少亡灵在渡河时得到了安慰 |
| **主动/被动** | 主动——这是她自己选择的"工作风格" |

**解读**：这个决策展示了小町的另一面——她虽然偷懒，但对"客户"的态度是认真（以她自己的方式）。她不是"不负责任"，而是"以最低能耗负责任"。

---

### 决策4：优先处理"自己觉得重要"的工作

| 属性 | 内容 |
|------|------|
| **类型** | 选择性决策 |
| **作品** | TH09、TH14.5 |
| **背景** | 幻想乡发生异变时，亡灵大量增加，工作压力陡增 |
| **决策内容** | 小町并没有因为亡灵增加而加班加点多干——她按照自己的节奏干，能运多少是多少 |
| **动机** | "反正也做不完，急有什么用？" |
| **后果** | 映姬不得不亲自介入处理拥堵的亡灵 |
| **主动/被动** | 主动——她做出了"不做额外工作"的清醒选择 |

## 三、关于异变的应对决策

### 决策5：花之异变中的行动——被卷入而非主动参与

| 属性 | 内容 |
|------|------|
| **类型** | 被动→主动转换 |
| **作品** | TH09 |
| **背景** | 幻想乡漫天飞花，亡灵异常增多，大量灵魂涌向三途河 |
| **决策内容** | 开始：小町没有主动调查原因，而是继续日常摸鱼。后来：映姬逼她去检查状况，她才慢悠悠地出发。结果她被卷入了的战斗中。 |
| **动机** | 如果不是映姬强迫，她根本不会去管 |
| **后果** | 在花田中遇到了幻想乡的各位，打了一架，然后——继续摸鱼 |
| **被动/主动** | 最初是被迫行动。中期主动参与（一旦动起来了倒也不会半途而废） |

**解读**：小町不是"不做事"，而是"没人催就不做"。一旦被逼到必须行动的境地，她也不会消极怠工到让别人陷入危险。她属于"推一把动一下"的类型。

## 四、人际关系相关决策

### 决策9：与幻想乡居民建立"轻松朋友关系"

| 属性 | 内容 |
|------|------|
| **类型** | 持续性主动决策 |
| **作品** | TH09 起持续 |
| **背景** | 小町的工作接触到的不仅是亡灵，还有幻想乡的居民（尤其是那些会灵力的人） |
| **决策内容** | 她选择用轻松、随和的方式与幻想乡居民相处——不摆死神架子，不保持距离 |
| **举例** | 和萃香以"摸鱼同伴"相称；对灵梦的态度是"嘛，就是这么回事~" |
| **动机** | 她不需要跟这些活人建立深交，但友善的关系让工作更轻松 |
| **后果** | 她在幻想乡的社交圈出奇地广——虽然她并不是刻意经营的 |

| 属性 | 内容 |
|------|------|
| **类型** | 持续性决策 |
| **作品** | 全作品 |

| **风险** | 如果映姬真的生气了（极少见情况），小町也会老实收敛 |
| **主动/被动** | 主动决策——她清醒地选择了这套"软抵抗"策略 |

## 五、行为模式总结

### 小町决策的核心规律

1. **最低能耗原则**：在所有可能的选择中，永远选最轻松的那个
2. **被动触发优先**：如果不被外力推动，她不会主动做任何事
3. **不彻底怠工**：一旦动起来，她也不会让人失望——她不是没能力，只是不想用
4. **软抵抗而非硬对抗**：面对映姬时，她用"认错但不改"策略而非正面冲突
5. **不越界**：她只在"自己可以偷懒的范围内"偷懒，关键职责不会真的放弃
6. **社会性良好**：她对所有人都友善——不是因为善良，而是因为"不友善太累了"

### 小町不是什么样的人

- **不是不负责任**：她对该做的"核心工作"（摆渡灵魂）是做的，偷懒的是"额外工作量"
- **不是冷漠**：她对亡灵友善，对朋友热情——她只是懒得做"不想做的事"
- **不是没能力**：ZUN从未设定她能力不足——她是"有能力但懒得用"
- **不是叛逆**：她不是想反抗映姬的权威——她只是单纯地不想干活
