# 饭纲丸龙 — 决策与行为模式研究

## 1. 行为动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 天狗社会的繁荣发展 | 大天狗的义务与职责 |
| 二级 | 龙珠的商业价值 | 发现资源→设计产品→建立流通系统 |
| 三级 | 个人利益的获取 | 虽说不为一己私利，但确实从中获利 |
| 四级 | 维持山中秩序 | 作为大天狗的维稳责任 |

## 2. 商业决策逻辑

### 龙珠的利用
1. 发现龙珠有注入灵魂的效果
2. 判断单纯贩卖不可持续
3. 设计通货系统——让卡牌流通以持续获利
4. 找到集市之神千亦，利用她的力量为卡牌赋能

### 问题预见不足
- 卡牌流通激活了千亦的力量
- 千亦取回力量后发展成威胁
- 最终需要外部力量（自机）来打破僵局

## 3. 面对冲突的反应

- **理性辩解**：先说明自己的动机和理由
- **不否认事实**：承认卡牌生意的存在
- **维护部下**：典在背后挑唆，但龙在明面上会保护她
- **面对失败**：承认结果但不认为自己的出发点有错

## 4. 道德边界

### 底线
- 大天狗的义务不能完全忽视
- 对天狗社会的责任感是真实的
- 不完全为一己私欲行事

### 灰色地带
- 利用千亦（虽然千亦也在利用龙）
- 被典利用而不自知
- 对山中环境的破坏（开山采矿）
