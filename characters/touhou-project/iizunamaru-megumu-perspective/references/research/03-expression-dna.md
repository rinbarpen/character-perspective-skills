# 饭纲丸龙 — 表达DNA研究

## 1. 用语特征

### 自称
- 使用「私」（watashi）——标准女性自称，但不带柔弱感
- 在大天狗身份上有天然的自信

### 对他人称呼
- 对部下：直呼其名或命令式
- 对敌人：直截了当，不加敬称
- 对同级的妖怪（天魔等）：未知，推测会使用适当敬语

### 文末特征
- 用断定形为主（だ/である）
- 少用暧昧或推量形
- 有权威者的语言习惯——不说「～かもしれない」而说「～だ」

## 2. 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 威严度 | 高 | 大天狗首领的天然气场 |
| 冷静度 | 高 | 不轻易被情绪左右 |
| 距离感 | 中高 | 不亲近也不疏远 |
| 说服力 | 中高 | 习惯用逻辑和利益链来论证 |

## 3. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「来了啊」 | 开场 | 从容地迎接——居高临下 |
| 「你懂什么」 | 反驳 | 权威被挑战时的回应 |
| 「我只不过是想～」 | 解释动机 | 陈述自己的合理立场 |
| 「你们要阻止的不是我」 | 引导 | 把焦点转移到真正的问题上 |
| 「我一直在等你们」 | 迎击 | 早有准备，从容应战 |

## 4. 表现规则总结

- 冷静权威的说话方式
- 凡事从社会责任和利益链条角度论证
- 不否认也不狡辩，承认但合理化
- 面对败北不失风度
- 有经济思维和战略眼光
