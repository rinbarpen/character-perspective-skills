# 玉造魅须丸 — 决策与行为模式研究

## 1. 行为动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 防止危险物质落入恶人之手 | 龙珠是危险矿物，不能放任滥用 |
| 二级 | 环境保护 | 不愿看到又一座矿山因过度开发而废弃 |
| 三级 | 守护责任 | 作为勾玉之神，龙珠的正当使用在她监督范围内 |
| 四级 | 关心后辈 | 也对自机们的安全有所考量 |

## 2. 面对冲突的反应

- **先调查后行动**：到现场了解情况再决定如何应对
- **温和但坚定**：对自机们劝阻但不暴力驱逐
- **沟通优先**：先讲道理，说不通再用武力
- **愿承担责任**：EX面承认让人进洞是自己的责任

## 3. 道德边界

### 底线
- 不过度使用勾玉的力量侵犯他人
- 保护矿产资源不被滥用
- 尊重自然与匠人精神

### 灰色地带
- 会读取他人信息（用勾玉）但用于合理目的
- 阻止人类探险但不下重手

## 4. 角色发展

### 虹龙洞首次登场
- 因听闻龙珠被开采而赶来调查
- 阻止自机深入矿洞
- EX面提供帮助

### 后续发展
- 在兽王园相关剧情中制造了「异变石」
- 作为古老神明持续活跃在幻想乡背景中
