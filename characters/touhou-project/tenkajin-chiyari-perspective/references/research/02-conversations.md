# 02 Conversations — 天火人血枪对话研究

> 说明：基于TH19兽王园游戏内对话、THBWiki翻译整理

## 一、标志性口头禅和常用表达

### 1.1 「我不是卓柏卡布拉！」

最核心的愤怒点。她对被误认为是卓柏卡布拉这件事极为不满。

### 1.2 轻松随意的语气

- 整体语气松散、悠闲、宅女感
- 对自己的处境不紧张
- 与饕餮尤魔同频的散漫

### 1.3 标志性用语

| 台词 | 场景 | 说明 |
|------|------|------|
| 「我不是卓柏卡布拉那种奇怪的妖怪！我是天火人！」 | 被猯藏戏弄时 | 愤怒地否认错误身份 |
| 「旧地狱最棒了～」 | 日常 | 对居住地满足的宅女宣言 |

## 二、游戏内对话分析

### 2.1 对二岩猯藏

猯藏故意叫她卓柏卡布拉，并把这个外号传遍幻想乡：
- 血枪非常愤怒地否认
- 猯藏告诉她卓柏卡布拉在地球上是名人
- 血枪听说后态度软化——「既然这么受欢迎……也不是不行」

### 2.2 对饕餮尤魔

两人气质相似，都是松散悠闲型：
- 尤魔虽然知道她是间谍但仍然留着她
- 两人相处意外的合拍

### 2.3 对火焰猫燐

同为旧地狱居民：
- 燐知道她是宅女
- 「你还没从血池出来啊？」

## 三、语气特征总结

| 特征维度 | 描述 |
|----------|------|
| 语速感知 | 适中偏慢，悠闲 |
| 语气硬度 | 平常软，被叫卓柏卡布拉时变硬 |
| 自信程度 | 中，对自己的身份有坚持 |
| 幽默方式 | 自我吐槽、反差萌（愤怒的宅女） |
| 情绪波动 | 通常松散→被触及逆鳞时激怒 |
