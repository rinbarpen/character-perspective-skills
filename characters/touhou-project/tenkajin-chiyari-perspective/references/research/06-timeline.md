# 天火人血枪 — 作品时间线

## 登场前

| 时间 | 事件 |
|------|------|
| 远古 | 天火人血枪出现在旧地狱，以吸血为生 |
| 旧地狱时期 | 与残无在旧地狱相识 |
| 旧地狱被抛弃后 | 残无提议建立新地狱，留下旧地狱的居民产生了大量怨恨 |
| 发现血池 | 在旧地狱发现血池，从此变成宅女 |
| 被残无招募 | 残无利用她对旧地狱居民"被抛弃"的共鸣来招募她 |
| 加入豪族联盟 | 成为残无在豪族联盟中的间谍 |

## 东方兽王园（TH19，2023年5月）

| 事件 | 说明 |
|------|------|
| 作为Stage 2 Boss登场 | 在血池/旧地狱区域出现 |
| 与猯藏互动 | 被猯藏戏称为卓柏卡布拉 |
| 与尤魔互动 | 表面同事关系，实际互知底细 |
| 作为自机可用 | 通关后可选 |

## 兽王园之后

| 时间 | 事件 |
|------|------|
| 2023后 | 短暂以"卓柏卡布拉"的身份出名 |
| 2023后 | 搬到幻想乡命莲寺墓地居住 |

## 关键节点总结

- 兽王园（2023）：首次登场，Stage 2 Boss兼自机
- 核心主题：身份认同、宅女性格、被误认的愤怒
