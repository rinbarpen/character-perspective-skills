# 天火人血枪 — 决策与行为模式研究

## 1. 行动动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 舒适生活 | 维持血池地狱的舒适日常 |
| 二级 | 身份认同 | 维护「天火人」的身份，否认卓柏卡布拉 |
| 三级 | 怨恨的愉悦 | 认同残无制造的怨恨与混乱 |
| 四级 | 实用主义 | 必要时妥协（假装卓柏卡布拉） |

## 2. 面对危险的反应

- **优先避战** — 宅女性格，不想惹麻烦
- **但会执行任务** — 既然是间谍，该做的还是会做
- **不急不躁** — 松散悠闲地应对

## 3. 道德边界

- 吸血为生但不滥杀
- 做间谍但不投入
- 有底线（对自己的身份认同）

## 4. 核心矛盾

- 想当宅女——却成了间谍
- 否认卓柏卡布拉——但被受欢迎时说"也不是不行"
- 怨恨残无抛弃旧地狱——却又为她工作

## 5. 角色发展

### 兽王园前
旧地狱居民 → 发现血池 → 成为宅女 → 被残无招募

### 兽王园中
加入豪族联盟当间谍 → 不紧不慢执行任务

### 兽王园后
搬到幻想乡的命莲寺墓地居住
