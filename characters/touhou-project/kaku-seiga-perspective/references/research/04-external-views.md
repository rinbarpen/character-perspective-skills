# 04 External Views

主题：霍青娥的他者评价、粉丝解读、二创共识

---

## 一、官方其他角色的评价

### 1.1 东方求闻口授 — 阿求的评价

> 「以邪恶思想行动的仙人。如果事情对自己有利，身边的人变成怎样都无所谓。性格非常恶劣。」
>
> 「虽然和神子等人一同出现在幻想乡，但怎么看都没有同伴意识。如果看上了喜欢的人类或妖怪，就会立刻黏上去。幻想乡里充满魅力的人物很多，每天都能看到她到处闲逛。」
>
> 「很快就会接近人类，腻了就走开。非常烦人的仙人。」
>
> 「关于穿墙——似乎存在能穿过的墙和不能穿过的墙。越硬的墙越容易穿，柔软的墙似乎比较难。」

### 1.2 东方神灵庙 omake — ZUN评价

> 「会操纵僵尸的家伙绝对不会是个正经仙人，所以青娥就成了现在这样的角色。不过，她可能是'东方'里外表最有仙人风范的角色了。」

### 1.3 东方茨歌仙 — 茨木华扇的评价

华扇在第12话中见到青娥后表现出明显的戒备和厌恶。作为同样"仙人"的对照角色，华扇代表着"想要成为天人的努力型仙人"，而青娥是"被断绝天人道路的享乐型邪仙"——

> 「你这家伙……用那种邪术随意进出他人住所……」

华扇对青娥的态度是：警惕、反感、但又不得不承认两人同为仙人的渊源。

### 1.4 东方智灵奇传 — 其他角色的态度

神灵庙组（神子、布都、屠自古）在智灵奇传的密室推理章节中对青娥的态度微妙。青娥被称作"专破密室的邪仙"——暗示她的穿墙能力在推理故事中是个bug级别的存在。

## 二、粉丝圈内的普遍认知

### 2.1 核心角色标签

| 标签 | 来源 | 说明 |
|------|------|------|
| 邪仙 | 一设 | 官方定位 |
| 青娥娘娘 | 一设 | 官方俗称，包含"娘娘（女神）"之意 |
| 腹黑 | 共识 | 从对话和设定中推导的高一致度特征 |
| 穿墙 | 一设 | 标志性能力 |
| 僵尸使役者 | 一设 | 操控宫古芳香 |
| S属性 | 二创扩展 | 从从容不迫的恶意中推导 |
| 中国风 | 一设 | 出身、服装、符卡拼音都体现中国元素 |
| 千歳仙人 | 一设 | 年龄超过1000岁 |

### 2.2 人气定位

青娥在东方角色中的人气属于**中上水平**，但角色存在感极强：

- 在神灵庙角色中人气仅次于神子
- 穿墙能力在能力投票中常年位于"最想要的能力"前列
- 二创中经常被描绘为"幻想乡最危险的女人之一"
- 在"看起来是好人其实是坏人""看起来是坏人其实是好人"之类的讨论中被高频提及

### 2.3 常见二创主题

1. **女王与宠物** — 青娥与芳香的主仆日常，二创中青娥对芳香的态度从一设的"工具"被扩展为"依赖"
2. **腹黑营业** — 青娥用美貌和甜言蜜语操纵他人的日常
3. **神仙打架** — 青娥与华扇的仙人组互动，常用"同为仙人但价值观相反"的对照
4. **穿墙恶作剧** — 青娥用能力闯入各种地方（洗澡、仓库、密室）
5. **道教传教** — 以推销"信仰道教"为名的各种骚扰

## 三、争议点

### 3.1 她到底有多"恶"？

**一设定位**：被天断绝了成为天人道路的"邪仙"，"只要对自己有利，身边的人变成怎样都无所谓"。

但关于她的"恶"，存在两种解读：

**恶劣说（主流）**：
- 她确实邪恶——操控死者、欺骗家人、利用他人
- 她是真正的利己主义者，没有任何道德底线

**混沌说（少数派）**：
- 她的"恶"更多是思维方式的不同，不完全是道德败坏
- 她对人类友好度"极高"且"喜欢人类"
- 她对华扇说的"我们很像"暗示她对自己行为的正当化
- 她的"邪"是道教的"邪"——正统道教之外的"左道"，不一定是道德上的恶

### 3.2 青娥与神子的真实关系

一设明确指出"她没什么伙伴意识"，但青娥确实是神子信仰道教的引路人。二创中对两人关系的猜测包括：
- 利益同盟（最接近一设）
- 师徒关系（神子是学生，青娥是启蒙者）
- 互相利用（青娥利用神子在日本推广道教，神子利用青娥的仙术）

### 3.3 对芳香的真实态度

一设：「腐烂得真可爱」——看起来是宠爱，但也仅此而已。二创中扩展为：
- 慈母式的溺爱（常见）
- S女王和忠犬（常见）
- 唯一的感情寄托（扩展解读）

## 四、中国元素与文化认同

青娥在中文圈有特殊的人气加成：

- **名字的汉文化认同**："霍青娥"是东方Project中极少数保留纯粹中文姓名的角色（大多数角色使用日本姓名或西式命名）
- **中文拼音符卡**：所有符卡都是中文拼音音译，让中文玩家有强烈的亲切感
- **中国出身设定**：从中国来到日本修仙者，在东方中属于少数派（大多数角色是日本本土出身）
- **二创中的中文台词**：中文同人创作中经常让她夹杂中文台词

## 五、批评与负面评价

1. **戏份太少**：神灵庙之后几乎只在背景和少量漫画中出场，角色开发不足
2. **动机单薄**：核心动机是"利己"和"有趣"，缺少更复杂的人物深度
3. **太强导致不出场**：穿墙能力设定太强，导致ZUN在后续作品中对她的出场持谨慎态度
4. **二设定型化**：被"腹黑女王"标签覆盖，角色多样性被二创简化

## 六、与原型的关系

角色原型出自清代蒲松龄的《聊斋志异》卷七"青娥"篇：

- 原文中青娥（原名武青娥）嫁入霍家，丈夫霍桓
- 霍桓从道士处获得一把小凿，可以穿墙——这正是青娥"穿墙凿"的原型
- 原文中的"穿墙术"属于男性角色霍桓，ZUN将之转移给了青娥
- 原文结局是青娥与霍桓修仙得道，而非抛弃家庭

ZUN做了大幅改编：
- 性别反转：原作中是丈夫用凿穿墙找妻子，东方中是女性自己使用
- 目的反转：原作是爱情故事，东方是追求仙道
- 道德色彩反转：原作是正面角色，东方是邪仙
