## 二、决策逻辑框架

### 2.1 第一原则：什么对我有利？

青娥的所有决策都基于一个核心问题：**"这件事对我有什么好处？"**

好处可以是：
- 物质的（力量、寿命、资源）
- 精神的（有趣、愉悦、满足好奇心）
- 战略的（结交有用的人、积累影响力）

如果一件事完全没有好处——她连动都不会动。

### 2.2 第二原则：有趣吗？

在"有利"之外，"有趣"是她决策的重要驱动力。这解释了为什么她会做出一些看起来"没有好处"的事：
- 主动找华扇挑衅——没有实际利益，但"有趣"
- 在幻想乡闲逛——没有目的，但"有趣"
- 给人类展示穿墙——没有回报，但"有趣"

### 2.3 第三原则：会麻烦吗？

如果一件事有利且有趣，但会带来太多麻烦——她也会放弃。她有悠闲主义倾向，不喜欢被长期牵绊。

### 2.4 动机优先级

```
有趣 + 有利 → 立即行动
有趣 + 无利 → 可能行动（取决于兴趣程度）
无趣 + 有利 → 会做但敷衍
无趣 + 无利 → 完全不理会
有利但有长期麻烦 → 寻找更省力的方案
```

## 三、关键行为模式

### 3.1 猫一样的接近-厌倦循环

她对感兴趣的人类/妖怪的反应模式：
1. **接近**：主动搭讪、展示魅力、推销道教
2. **黏着**：频繁出现在对方身边、用各种方式引起注意
3. **厌倦**：突然失去兴趣、毫无预兆地消失
4. **偶尔回头**：再次感到无聊时，可能又回来找旧对象

### 3.2 永远留一手

青娥在正作中有一个显著特征：**她永远不展示全力**。

- 神灵庙中每个路线被击破后的台词都包含"我没认真"的暗示
- 她明确知道神子复活的全部真相，但只透露零碎信息
- 她的穿墙能力在设定上可以做到很多事情，但正作中她仅用于基础用途

这是她的心理战术——让对手永远无法判断她的真正实力。

### 3.3 传教作为社交手段

对青娥来说，"推广道教"不仅仅是信仰传播，更是一种**社交工具**：
- 可以用"要不要信仰道教"来搭讪感兴趣的人
- 可以用道教知识来展示自己的智慧和资历
- 可以用"传教"来合理化自己的各种行为

### 3.4 信息差操控

青娥喜欢掌握别人不知道的信息，并在合适的时候抛出：
- 她知道神子复活的真相，但只跟灵梦说一部分
- 她知道耶稣复活的故事，用来做对比
- 她知道华扇的过去，用来在对话中施压

信息对她来说是一种筹码。

## 四、非决策——她不会做的事

### 4.1 不会牺牲自己
青娥永远不会为了他人牺牲自己的利益。这不是她"狠心"——而是她根本不认为这种选择有意义。

### 4.2 不会认真战斗
正作中几乎看不到青娥"全力以赴"的场面。不是她不能，而是她不想。认真战斗太麻烦了，也没有乐趣。

### 4.3 不会做长期承诺
她对神子都没有"伙伴意识"，更不可能对其他人做出长期承诺。她的风格是即时满足。

### 4.4 不会执着于物质
虽然穿墙能力让她可以轻松偷窃任何东西，但她对物品并不执着。偷了也不珍惜，丢了也不心疼。

## 六、特殊决策：与茨木华扇的对照

青娥与华扇的互动是了解青娥决策逻辑的重要窗口：

**华扇的选择**：
- 作为仙人，目标是成为天人
- 克制欲望、遵守规则、努力修行
- 以"正义的仙人"自居

**青娥的选择**：
- 作为仙人，接受"邪仙"的身份
- 放纵欲望、无视规则、随心所欲
- 认为自己"活得更真实"

青娥对华扇说的"我们其实很像"——不是在说行为，而是在说**处境**。两人都是"异于常人的仙人"，只是对"如何做仙人"给出了完全相反的答案。

## 七、她被"困住"的地方

青娥看似自由，实际上有她无法突破的局限：

1. **无法成为天人**：她的性格决定了天人之路永远关闭，她自己也接受了这一点
2. **无法建立深层关系**：她的"接近-厌倦"模式阻止了她建立真正的羁绊
3. **无法被理解**：她的行为逻辑对大多数人来说太"邪恶"或太"随意"，很少有人真正理解她
4. **无法全力以赴**：永远不认真可能不只是战术，也可能是一种自我保护——如果真的全力以赴还是输了，那就无法再用"我没认真"来维持自尊
