# 姬海棠果 他者评价与粉丝认知

## 一设中他人的评价

### ZUN（官方创作者）的评价

**来自东方求闻口授访谈：**

"毕竟这是个只有文才知道的角色……文又不会把果的事情写成新闻。但是果有着文所不具备的能力。念写是接近于千里眼的能力，但是可以影像化。想到关键词立刻就能浮现相关的事情，如同Google一般的能力。关于果我虽然真的是一直都忘了（还没在幻想乡缘起和新闻中提及过的事情），但由于这本书的设定，我也完全无法让她登场，难以进行说明。"

——ZUN坦言他经常"忘了"果这个角色的存在。

### 东风谷早苗的评价

**来自《求闻口授》：**
早苗认为果的"文章攻击性有点强"。

### 鬼人正邪的评价（弹幕天邪鬼）

通过符卡名"足不出户的狗仔队"和"最近的便携照相机"来调侃果的家里蹲属性和老旧手机相机。

### 射命丸文的评价（DS拍摄评论）

"相机的性能太差劲了。做工这么粗糙，会被村里的人笑话的哟。河童的活儿干得也太马马虎虎了。"
"哎呀呀，可以连拍呢。不不不，我才没有羡慕呢。换胶卷的时间真短呢，虽然我一点也不羡慕。"
"最后因为害怕我的摄影技术而使用了念写呢。好不容易决定靠自己的力量出门去取材了。"

——文表面上在损果，实际透露出对果的相机功能的羡慕（连拍、快换胶卷）。

## 粉丝认知关键词

### 正面评价（来自投票评论）

- "幻想乡第一的好孩子"（幻想郷一のいい子）
- "东方屈指的好孩子"——明亮、活泼、纯粹的可爱鸦天狗大小姐
- "妖怪之山第一的ぐう聖天狗"（究极圣天狗）
- "东方第一善人"
- "虽说是ギャル风但认真且为同伴着想"
- "原作就已经性格好得一塌糊涂了"
- "阳光积极的家里蹲"
- "东方里罕见的积极正向率直的好孩子"

### 角色类型认知

1. **另一个天狗（文中の"もう一つの天狗"）：**
   果被视为"缺少登场机会的第二天狗"。提到天狗时大家首先想到文，果是"那个另一个天狗"。

2. **现代感角色：**
   使用手机相机+念写能力（类似Google搜索），在幻想乡中带来了现代感。被形容为"更像是现世JK（女子高中生）"。

3. **家里蹲记者：**
   "家里蹲"是她最深入人心的二设标签之一，官方也确认了她不出门取材的属性。

4. **纯粹的好孩子：**
   绝大多数粉丝认可她的善良本质——对古明地觉没有偏见是最常被引用的证据。

## 文果关系（あやはた）的粉丝解读

**最常见的CP倾向：**
- 欢喜冤家——竞争关系下隐藏着彼此认可
- 前辈后辈模式——文是经验丰富的前辈，果是努力追赶的后辈
- 某些二设中发展成恋爱关系

**粉丝常用设定：**
- 果暗恋文但不敢表白
- 文调侃果但也暗中关照她
- 两人偶尔合作采访

## 二设常见形象偏离

1. **阴暗/病娇化：**
   早期二设中果常被描绘成阴暗、病娇、跟踪狂（はたたん形象——黑色长发、眼神空洞）。这与她一设的阳光积极性格有较大偏差。

2. **废柴宅女：**
   因官方确认"不出门取材"，二设中常过度强调她的家里蹲属性，甚至到"不修边幅"的程度。

3. **胖果（デブはたて）：**
   因DS拍摄评论中大量食物相关台词，衍生出吃货→发胖的梗。

4. **沉默寡言：**
   某些二设把她描绘成不爱说话的阴角色，但她一设中其实言辞活泼、敢于宣战。

## 与其他角色的关系认知

- **与古明地觉：** 果是少数对觉没有偏见的角色，这被视为她"善良"的铁证。二设中这层关系被深化为友情。
- **与犬走椛：** 同属天狗社会中的同事，二设中常作为三人组（文果椛/三天狗）的一部分。
- **与宇佐见堇子（Sumireko）：** 同为"现代科技系"角色，果甚至有文章论述外来人的科技能给幻想乡带来好处。

## 人气偏低的原因分析（粉丝讨论中归纳）

1. 登场作品少——仅DS作为主要角色，后续多为客串/背景
2. 没有个人主题曲——所有登场的音乐都是共用曲
3. 性格温和缺少戏剧冲突——不像文那样有强烈的存在感
4. 被文的光芒掩盖——"提到天狗记者还是文"
5. 官方遗忘——ZUN自己也说过总忘记她

## 总体粉丝印象

果在核心粉丝中评价极高（"东方第一好孩子"），但由于登场机会少，认知度有限。她在投票中稳定在70-100名区间，属于"小众高评价"型角色。粉丝希望她能在新作中再次登场。
