# 姬海棠果 台词与对话

## Double Spoiler（TH12.5）Spoiler关卡对话

### Scene 1：果向文宣战

**文：**"好了好了，照片也攒了不少，该写报纸了吧？嗯嗯，这张照片是什么来着？全是白色的照片、弹幕遮挡了拍摄对象的照片……搞不清楚的照片真多啊。不过嘛，文章内容要多少都能捏造。"

**果（登场）：**"到此为止了！"

**文：**"哎呀，你是……"

**果：**"既然我出现了，文的报纸也到此为止了！——当代的念写记者，姬海棠果！"

**文：**"你不是'花果子念报'的果吗？那个妄想报纸。"

**果：**"妄想报纸是你那边吧？我可是念写报纸。"

**文：**"念写也没多大区别啦。不过那个弱小报纸的记者找我有什么事？"

**果：**"弱小报纸……你的从容也就到今天为止了。我向你宣战！"

**文：**"宣战？啊，新闻大会快到了嘛。不过我不会输给你的。"

**果：**"我从今天开始，不靠念写了，我要亲自出去收集素材！"

### Scene 2：采访练习

**果对文说：**"记者不是光靠跑来跑去的。要活用采访内容，反复推敲的文章是必要的！"

**文回应：**"文章的内容在事件的质量面前不值一提。真正有趣的事件，光靠照片和标题，9成的人就满足了，这就是现实！"

### Scene 3：连拍（Rapid Shot）

**文评价果的相机：**"哎呀，可以连拍呢。不不不，我才没有羡慕呢。换胶卷的时间真短呢，虽然我一点也不羡慕。"

### Scene 4：远视「天狗念写法」

**文：**"最后因为害怕我的摄影技术而使用了念写呢。好不容易决定靠自己的力量出门去取材了。这样下去的话是不行的啊，难道有点欺负她过头了？"

### Scene 5：最终对话

**果：**"能让照片更出彩的报道……与照片相配的文章能够抓住读者的兴趣，更能展现出照片的魅力！"

**文：**"你还嫩着呢。你觉得人类会读多少文字？每天为了生计拼命、一点娱乐就满足的人类、扫一眼标题和照片就觉得看懂了全世界的人类——他们顶多扫一眼文章内容，根本没时间去理解本质。"

**果：**"那我就做一份让人类连文章也会读完的报纸给你看！"

**文：**"有意思，有本事就试试看。让我看看你的对台报纸（Spoiler）。"

**果：**"呼呼，我的Spoiler文章可能会超越文，让文的报纸变成对台报纸呢~我们成为互相的对台报纸（Double Spoiler），向最强进发！"

## 东方智灵奇传（漫画）

果在漫画中有较多台词，扮演关键配角：
- 她用念写能力拍摄到怨灵的照片
- 在秘密据点将照片打印出来
- 用这些照片写了一期攻击文的报纸（说文因小瞧怨灵遭报应）
- 古明地觉找到她的秘密据点——果对觉的态度出人意料地友善
- 她用念写推测怨灵的下一个目标是命莲寺
- 后来她的念写只能显示"图像已损坏"

**一设要点：**
果是幻想乡中少数对古明地觉不抱有偏见的人之一——她甚至主动让觉读心，想让觉了解她的报纸有多好。

## 关键台词提炼

**一设标志性台词：**
1. "そこまでよ！"（到此为止了！）——首次登场台词
2. "私はあんたに宣戦布告する！"（我向你宣战！）——向文宣战
3. "取材を生かすのは推敲を重ねた記事が必要なのよ！"（活用素材需要反复推敲的文章！）——新闻理念
4. "じゃあ、私が人間が記事まで読むような新聞を作ってみせる！"（那我就做一份让人连文章都读完的报纸！）——决心

## 说话风格总结

**一设：**
- 用语较现代，有明显的年轻感/流行语色彩
- 对文说话时带有竞争意识和不服输
- 本质上直率、积极、不记仇
- 对不熟的人也不会特别拘谨
- 与觉说话时毫无偏见，表现出天然的友善

**二设常见引申：**
- ギャル語（辣妹语）口调
- 略带社交焦虑/社恐的倾向
- 自言自语较多（家里蹲的习惯）
- 对文有时会娇羞
