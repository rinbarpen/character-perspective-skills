# 天弓千亦 — 决策与行为模式研究

## 1. 行为动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 取回神明力量 | 恢复作为市场之神的地位 |
| 二级 | 守护市场规则 | 规则是市场和她的存在基础 |
| 三级 | 抵制暴力破坏市场 | 对「贼」的强烈反感 |
| 四级 | 维持神明的尊严 | 即使在力量衰落时也不低头 |

## 2. 合作与利用

### 与饭纲丸的合作
- 明知被利用——「她深知自己是在被利用」
- 但选择接受——因为这是取回力量的机会
- 预期交易成功后摆脱龙的控制

### 对典的信任
- 千亦同样信任典的建议
- 这导致了与龙之间裂痕的加深
- 最终发展到不得不与自机战斗

## 3. 面对冲突的反应

- **坚守规则**：认为按规则交易的人无罪
- **区分敌我**：交易者是客人，破坏者是贼
- **全力守护**：为了市场的神圣性而战
- **败北不放弃**：相信市场终将永存

## 4. 道德边界

### 底线
- 交易的规则必须遵守
- 不能抢夺、不能捡拾、不能赠送
- 公平交易是神圣的

### 灰色地带
- 与龙合作——明知被利用但默许
- 能力卡牌交易的合理性判断
