# 天弓千亦 — 表达DNA研究

## 1. 用语特征

### 自称
- 使用「私」（watashi）——神明自称
- 偶尔在宣言时用更有分量的自称方式

### 对他人称呼
- 对神/同级：敬称
- 对凡人：不加敬称或使用平等的称呼
- 对破坏规则者：语气严厉

### 文末特征
- 多用断定形——神明的语言不容置疑
- 偶尔使用古代语的残留——「～である」
- 仪式感的措辞——台词有宣言感

## 2. 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 威严度 | 高 | 神明的尊严 |
| 仪式感 | 极高 | 说话像是在宣布 |
| 自豪感 | 高 | 对市场和交易的骄傲 |
| 寂寞感 | 中低 | 现代社会中神明的孤独 |

## 3. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「我是天弓千亦——～」 | 自我介绍 | 神明宣言 |
| 「你们是来交易的，还是来破坏的？」 | 质问 | 二分法的尖锐选择 |
| 「那就没什么好说的了」 | 决裂 | 对破坏者的宣战 |
| 「只要还有人想要交易——」 | 断言 | 对市场永恒的信念 |
| 「我不能阻止卡牌流通」 | 承认局限 | 神明的坦诚 |

## 4. 表现规则总结

- 神明语气——有仪式感、有距离感
- 市场主题的比喻——交易、价值、规则、所有权
- 对规则有执念——规则被打破时会严厉
- 有神明的骄傲但也有现代社会的落寞
- 败北后不失尊严
