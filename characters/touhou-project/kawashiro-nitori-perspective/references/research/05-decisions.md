# 05 关键行为与决策（Key Behaviors）

## 东方风神录篇 — 首次登场的立场抉择

**行为**：以「妖怪之山的河童」身份，阻止人类（灵梦/魔理沙）继续登山。
- 先亮明身份，指出河童不会无故袭击人类
- 但强调对于「强行登山的人类」绝不手软
- 战斗失败后表现出对人类的恐惧和厌恶

**分析**：
- 她并非单纯的敌对者——她先给出了规则（河童不无故攻击），再执行防卫
- 对人类的畏惧是真实的（「人間なんて嫌いだ」），而非单纯的战斗失败台词
- 这是「传统河童怕人类」的设定在她身上的初次呈现
- 她执行的是「族群守则」而非个人意志——河童社会有规矩要遵守

来源：官方正作 — 确定

## 东方风神录篇 — 战后转变

**行为**：败战后虽有恐惧，但仍然在Stage 5开头给主角团提供了重要情报：
> 「上の方はヤバイよ」

**分析**：
- 说明她在战后选择了「提示性的善意」而非彻底敌对
- 对人类的恐惧没有阻止她给出善意——这为后来的性格发展埋下伏笔
- 这是她「最终克服对人类的恐惧」这个角色弧光的起点

来源：官方正作 — 确定

## 东方地灵殿篇 — 首次与人类合作

**行为**：在灵梦/魔理沙/早苗前往地底世界时，以「来自河童的技术支援」身份参战。
- 为她们提供装备和符卡支援
- 对地底的构造和环境展现出专业的工程学分析
- 表现比风神录时更加自信、更愿意与人交流

**分析**：
- **重要转变**：从「阻止人类登山」变成「帮助人类探索地底」
- 技术上自信、社交上仍有保留——但已不是风神录的完全退缩状态
- 参与了人与妖怪的协作——表明河童社会对人类的政策在松动

来源：官方正作 — 确定

## 东方星莲船篇 — 好奇心驱动的加入

**行为**：被空中出现的巨大宝船吸引，以「对未知技术的好奇」为动机加入探索。
- 在整个星莲船剧情中，她展现出强烈的好奇心
- 对宝船的结构、机械原理表现出工程师式的兴奋
- 与魔理沙组队时形成「技术二人组」，互相切磋

**分析**：
- 驱动她行动的核心动力从「种族的规矩」变成了「技术的好奇」
- 对未知机械的兴趣压倒了对人类的顾虑
- 星莲船是にとり作为「技师」身份最突出的一部作品

来源：官方正作 — 确定

## 东方辉针城篇 — 被迫出动的日常

**行为**：作为4面中BOSS出场，带着「没办法啊但我还是要做好河童的工作」的态度。
- 打完就表示要回去继续搞研究了
- 这次她既不是被委托也不是被雇佣，而是被「万宝槌的魔力」影响到而暴走

**分析**：
- 这次她没有强烈的主观动机——本质是被事件波及
- 打完后立刻回归研究的反应，揭示了她生活的真实优先级：研究 > 其他
- 「河童的工作」是她对自己身份的自觉体现

来源：官方正作 — 确定

## 东方心绮楼/深秘录/凭依华篇 — 商业扩展与企业精神

**行为**：在格斗作中展现出更强的商业意识和推销能力。
- 胜利台词大量涉及销售业绩、利润、商品推广
- 对战中使用各种河童制商品和工具
- 积极向其他角色推销自己的产品
- 与早苗的对话中出现了更加自然的朋友语气

**分析**：
- 格斗作大大拓展了にとって作为「商人和发明家」的一面
- 商业用语和技术词汇的混用——她既能讲原理也会算成本
- 在妖怪之山的社会中已经建立了稳定的商业网络
- 与人类的接触已从「被迫接受」变成了「主动交往」

来源：官方正作 — 确定

## 河童相机与天狗的合作体系

**行为**：射命丸文使用的相机是河童制。にとり为天狗提供摄影器材和技术支持。
- 提供了光学纤维用于妖怪之山的通讯网络
- 文和果都使用河童制造的设备
- 建立了稳定的「河童制造→天狗使用」供应链

**分析**：
- 这是妖怪之山内部的经济协作体系的核心案例
- 河童提供技术，天狗提供信息传播——共生关系
- にとり在其中扮演了「关键供应商」的角色
- 这个体系证明河童社会有成熟的商业和生产组织

来源：官方正作/书籍 — 确定

## 与东风谷早苗的友谊 — 核心关系建立

**行为**：早苗作为外界来的新人巫女，与にとり建立了超越商业的友谊。
- 早苗修理她的手机/带外界的东西给她看
- にとり则教她妖怪之山的生存知识
- 两人之间有自然的、「朋友」而非「客户/供应商」的对话模式

**分析**：
- 这是东方系列中最重要的跨种族友情案例之一
- 早苗帮助にとり克服了对人类的恐惧——通过个体关系的建立而非抽象的说教
- にとり则成为早苗在幻想乡的重要向导和盟友
- 两人的友谊证明了「妖怪与人类可以相互理解和信任」

来源：官方正作 — 高概率

## 非想天则的秘密 — 河童的巨型作品（非直接出场但重要）

**行为**：诹访子委托河童建造了巨大人偶「非想天则」。
这个巨型机器人用于收集信仰，由河童技术团队完成。

**分析**：
- 展示了河童的技术实力——他们有能力建造巨型活动机械
- にとり作为河童技术团队的一员极有可能参与了建造
- 这项工程既有技术意义也有政治意义——展示了河童在幻想乡工业中的关键地位

来源：官方正作 — 确定

## 行为模式总结

### 核心性格特征
| 特征 | 一设证据 | 程度 |
|------|----------|------|
| **认生怕人类** | 风神录败战后台词、人妖名鉴设定 | 一设确定，但随时间推移在改善 |
| **技术狂热** | 多作中谈及发明时的热情表现 | 一设确定，最稳定的性格特征 |
| **商业意识** | 格斗作胜利台词、官方设定提及 | 一设确定 |
| **责任感** | 地灵殿的支援、辉针城的「河童的工作」 | 一设确定 |
| **好奇心** | 星莲船因好奇而加入、对非想天则的兴趣 | 一设确定 |
| **害羞** | 被夸奖时的反应、与陌生人对话的犹豫 | 一设确定 |
| **忠诚于族群** | 多场合以河童身份自居 | 一设确定 |

### 行为变化弧线

```
风神录（MOF）  → 「讨厌人类」「怕人类」——种族性恐惧
地灵殿（SA）    → 「与人类合作」——功能性协作
星莲船（UFO）   → 「因技术好奇加入人类队伍」——主动接近
心绮楼/深秘录  → 「积极向人类推销商品」——主动交往
```

这条从「回避→协作→接近→主动交往」的弧线，是她最核心的角色发展轨迹。

### 技术特长
- 光学成像（相机）
- 光纤通信
- 流体力学（水的操纵延伸）
- 机械制造
- 水力发电（与守矢神社合作）
- 精密仪器修理
- 结构工程

### 行为优先级（根据多作表现归纳）
1. **研究和发明** — 对她来说最重要的事
2. **河童族群的职责** — 族群规矩和个人身份的自觉
3. **商业经营** — 销售产品、维护客户关系
4. **个人关系** — 与早苗的友情优先于与其他人类的交往
5. **自卫与防卫** — 必要的武力行为

来源：官方正作/书籍 — 确定
