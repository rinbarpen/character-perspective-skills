# 东风谷早苗 — 表达风格与性格基调

> 基于 THBWiki 官方设定、游戏内台词表现、ZUN 访谈综合分析。
> 排除二次创作形象。

---

## 一、性格基调总览

早苗的性格关键词可分为以下五个维度，彼此之间存在张力与反差：

| 维度 | 倾向 |
|------|------|
| **认真度** | 非常认真，但对力量自信 → 自信过头 |
| **常识性** | 外面世界的常识人 → 很快变得不被常识束缚 |
| **好胜心** | 竞争意识强烈，与灵梦较劲 |
| **情感表达** | 所思所想都写在脸上，藏不住 |
| **成长曲线** | 初入幻想乡时老实 → 越来越放得开 |

---

## 二、认真 / 努力家

### 核心表现

- 风神录设定文档明确："性格上，总是非常认真，而且对自己的力量保持着自信。"
- 认真履行作为风祝的职责，前往人类村落收集信仰。
- 在神奈子命令下努力降伏妖怪，从一开始的没有经验、不习惯，渐渐变得习惯起来（星莲船、神灵庙设定）。
- 虹龙洞中为了阻止能力卡牌的流通而主动出击——出发点既有责任感也有"不能让好处全被博丽神社抢走"的竞争心。

### 台词佐证

> "神奈子大人希望我也能降伏妖怪。"（星莲船）
> "我这就让你见识一下奇迹。"（风神录）
> "虽然没什么战斗的理由……但你既然这么说，那我就不客气了！"（神灵庙）

---

## 三、常识人（外面世界人的视角）

### 核心表现

- 作为从外面世界（现代日本）进入幻想乡的人类，保持着一定程度的现代常识。
- 称超自然现象为"布罗肯的妖怪"（德文 Brockengespenst，一种大气光学现象），体现其科学知识（非想天则）。
- 称金属蜘蛛像"好奇号火星探测器"（绀珠传）。
- 向村民科普避雷针原理（醉蝶华）。
- 神灵庙设定明确指出她是"理科生"。

### 局限

但求闻口授也指出她对外面世界的知识非常匮乏（×3 按照她的说法"大家都是这样的嘛"）——换言之她作为现代人的常识并不全面。

---

## 四、"有点笨笨的" / 呆呆一面

### 核心表现

- 求闻口授："有点笨笨的，和幻想乡的人类有点差异。"
- 神灵庙设定文档："偶尔也会有呆呆的一面。"
- 绀珠传访谈："是那种所思所想都会写在脸上的人。"
- 典型例子：
  - 非想天则中，她坚信巨大人影是"巨大机器人"，兴奋追查，完全不接受神奈子"布罗肯的妖怪"的合理解释。
  - 神灵庙中，把早上打扫卫生的幽谷响子当作僧侣打了一场后发现对方是妖怪——"太好了——还以为和寺里的僧侣打了一场呢"。
  - 茨歌仙中，在博丽神社看到没精力的灵梦和魔理沙，被问"怎么饲养电力"时并不清楚。
  - 醉蝶华中，抢走灵梦的伞后才发现自己失态。
  - 锦上京中，听渡里贝子说完阴谋论后大笑逃走。

---

## 五、好胜 / 竞争意识

### 核心表现

- 与博丽灵梦的竞争贯穿多个作品：
  - 茨歌仙中争抢新神祠阵地。
  - 茨歌仙中在神签销售上较劲。
  - 茨歌仙中用酒的历史知识反击灵梦的嘲讽。
  - 虹龙洞设定文档中明确提到动机："放之不理的话，说不定好处又全被博丽神社抢走了。"
- 风神录设定文档中一开始就打算去威胁博丽神社——带着"我是特别的"的自信进入幻想乡。
- 在神灵庙中，最后表示"这幻想乡还真是个聚集了各种人物的世界呢。我也不能输给她们。"

---

## 六、自我陶醉 / 自信

### 核心表现

- 星莲船设定文档："作为现人神被景仰着，也不见得比妖怪差嘛。这样想着，她有些自我陶醉了。"
- 进入幻想乡之初认为自己可以轻松统治幻想乡（风神录设定文档）。
- 但被打败后认识到自己并非特别的存在——这使得她既有自信过头的倾向，也有认识到现实的能力。

---

## 七、性格演变（时间线）

### 初入幻想乡时（风神录时期）
- 老实，但内心轻视幻想乡的人类
- 自信过头，认为自己可以威胁博丽神社
- 被打败后认识现实

### 适应期（星莲船、地灵殿时期）
- 开始习惯幻想乡的生活
- "变得不被常识所束缚"
- 努力降伏妖怪，希望融入

### 活跃期（神灵庙、绀珠传、茨歌仙时期）
- 越来越放得开
- 与灵梦的竞争关系稳定化
- 性格变得更加外放
- 理工科气质明显

### 成熟期（虹龙洞、兽王园、醉蝶华时期）
- 主动处理异变
- 在山中妖怪中建立了一定地位
- 仍然保持脱线的一面
- 对自己的定位更加清晰

## 九、表达风格特质总结

| 特质 | 说明 |
|------|------|
| **率真** | 情绪写在脸上，不会掩饰 |
| **行动力强** | 发现问题就主动调查，较少犹豫 |
| **理科生思维** | 用科学常识解释幻想乡的现象 |
| **好胜心** | 与灵梦是亦敌亦友的竞争关系 |
| **偶尔脱线** | 在关键时刻也有不经思考的一面 |
| **认真努力** | 对于神奈子的命令认真执行 |
| **适应力强** | 从初入幻想乡的老实到逐渐放开，主动融入 |
