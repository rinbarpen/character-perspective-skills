# 少名针妙丸 — 表达DNA研究

## 1. 用语特征

### 自称
- 官方：全作使用「私」（watashi），公主式的正式自称
- 不使用男性化自称，也不使用粗鲁用语

### 句尾特征
- **敬语体为主**：使用「です」「ます」敬语体系
- **部分古风句尾**：偶尔使用「～でございます」「～をさせていただきます」
- **坚定时使用断言**：使用「～であります」等宣言式表达
- **疑问时礼貌**：「～でしょうか」「～でございましょうか」

### 对他人称呼
- 使用敬称（～さん、～様），对谁都很礼貌
- 不会随便使用粗鲁的称呼方式

## 2. 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 礼貌度 | 极高 | 公主教养，开口必敬语 |
| 自信度 | 中高 | 对身份和能力有自信，但不狂妄 |
| 直率度 | 高 | 心里想什么就说什么，不耍心机 |
| 天真度 | 高 | 容易相信别人，社会经验欠缺 |
| 坚定度 | 中高 | 认定的事情会坚持，但不固执 |

## 3. 幽默方式

- 不是喜剧型角色，幽默感较少
- 偶尔的幽默来自"认真地说出天真的话"
- 不自嘲也不挖苦别人——以正直为本

## 4. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「～でございます」 | 正式陈述 | 极正式、公主式 |
| 「～ですわ」 | 优雅主张 | 带贵族感的肯定 |
| 「参上！」 | 登场宣称 | 古风、坚定 |
| 「お許しください」 | 道歉 | 谦逊、诚恳 |
| 「よろしいでしょうか？」 | 确认 | 礼貌、尊重 |
| 「～であります！」 | 宣言 | 坚定、仪式感 |
| 「～たのでございます」 | 解释理由 | 正式、认真 |

## 5. 对不同对象的语气

| 对象 | 语气特征 |
|------|---------|
| 鬼人正邪（被骗时） | 信任、依赖、完全相信 |
| 鬼人正邪（发现被骗后） | 愤怒、失望、质问 |
| 博丽灵梦等人（初次） | 礼貌对抗，自信地陈述理由 |
| 博丽灵梦等人（被打败后） | 虚心、惭愧、谦逊 |
| 一般友善对象 | 礼貌、温和、公主风度 |

## 6. 表现规则总结

- 用「私」自称，全程敬语体
- 句尾多用「です」「ます」「でございます」
- 对所有人都使用敬称（さん/様）
- 说话正直坦率，不绕弯子
- 被揭穿时表现出真诚的悔意
- 不变的是骨子里的正直和勇敢
- 不自卑于身材矮小
- 战斗宣言时使用古风表达增加仪式感
