## 一、游戏作品时间线（按发售日期排列）

### 2002年

#### 8月11日（Comic Market 62）— 东方红魔乡（TH06）

| 项目 | 内容 |
|------|------|
| **登场定位** | 二面关底Boss |
| **首次登场** | 东方Project Windows版系列初登场 |
| **称号** | 湖上的冰精（湖上の氷精） |
| **主题曲** | 活泼的纯情小姑娘（おてんば恋娘） |
| **戏份描述** | 在雾之湖附近徘徊时遭遇入侵红魔馆的灵梦一行，作为二面Boss阻挡去路。设定文档（omake.txt）描述其「脑袋不太好使」「喜欢恶作剧」「平时玩冻青蛙游戏，每3回大概会有1回失败，青蛙就会悲惨地支离破碎」。 |
| **符卡** | 冰符「Icicle Fall」、冻符「Perfect Freeze」 |
| **意义** | 琪露诺的首次登场，奠定了冰之妖精、笨蛋、爱恶作剧三大核心设定。红魔乡亦为东方Project在Windows平台上的复兴起点。 |

---

### 2003年

#### 8月17日（Comic Market 64）— 东方妖妖梦（TH07）

| 项目 | 内容 |
|------|------|
| **登场定位** | 一面道中Boss |
| **称号** | 冰之妖怪（氷の妖怪） |
| **戏份描述** | 在妖妖梦故事开头作为一面中Boss出现。ZUN在设定档中称其「喜欢寒冷地方的冰之妖怪」「并没有什么特别的目的而去攻击灵梦她们，仅仅是因为在那里所以便攻击了」。战斗后即不再出现。 |
| **角色分量** | 最低限度的杂鱼级登场，ZUN明确表示其「和故事没有关系」。 |
| **符卡** | 冰符「Icicle Fall」（Easy） |

### 2005年

#### 8月14日（Comic Market 68）— 东方花映塚（TH09）

| 项目 | 内容 |
|------|------|
| **登场定位** | 自机 / 敌机（可选角色） |
| **称号** | 冰之小妖精（氷の小さな妖精） |
| **主题曲** | 活泼纯情小姑娘的冒险（おてんば恋娘の冒険）——红魔乡主题曲的编曲版 |
| **戏份描述** | 首次成为可选自机角色。花映塚故事背景为「春雪异变」——大量幽灵涌现，琪露诺作为被卷入异变的妖精之一参战。自机剧情中，她与各角色对话并最终面对四季映姬。 |
| **⑨标志事件** | ZUN在花映塚介绍图中将琪露诺置于编号⑨位置，标注文字为「バカ」（笨蛋）。此为「⑨」及「笨蛋」别名的唯一官方来源。 |
| **自机剧情概要** | 琪露诺注意到异变后出发调查，沿途遭遇铃仙、小町等角色。铃仙评价她「波长很短」，四季映姬告诉她「妖精超出自己领域太多也可能死亡」。琪露诺最终并未理解异变的本质，但依然享受了战斗过程。 |
| **符卡** | 冰符「Sword Freezer」（自机）、冻符「Perfect Freezer」（自机）、冰符「Icicle Fall」（敌机）、冻符「Perfect Freeze」（敌机） |

#### 12月30日（Comic Market 69）— 东方文花帖（TH09.5）

| 项目 | 内容 |
|------|------|
| **登场定位** | Level 2 Boss（拍照对象） |
| **称号** | 不自然的冷气（不自然な冷気） |
| **戏份描述** | 射命丸文拍照取材的对象之一。琪露诺在此作中释放冻符「Minus K」（零下K，即绝对零度），此为弹幕摄影游戏中的最小拍照关卡之一，对应其较低的难度定位。 |

### 2009年

#### 8月15日（Comic Market 76）— 东方非想天则（TH12.3）

| 项目 | 内容 |
|------|------|
| **登场定位** | 自机 / 敌机（可选角色） |
| **称号** | 冰之妖精（氷の妖精） |
| **戏份描述** | 在格斗游戏中作为可选角色参战。故事模式中，琪露诺将非想天则的巨影误认作「大太法师」，独自四处寻找。她的故事线相对独立，与主线「非想天则巨影事件」关联较浅。 |
| **剧情概要** | 琪露诺在雾之湖上空看到巨影后，认为那是传说中的「大太法师」，决定去寻找。一路上击败了早苗等人，最终在守矢神社发现了非想天则的本体，但对其原理一无所知。结局中她得意洋洋地宣布自己「找到了大太法师」，但实际上什么都没搞懂。 |
| **角色分量** | 格斗游戏中约处于中低难度层级，保留了笨蛋+过度自信的性格刻画。对话中她将冻青蛙称为「结冰的修行」，并多次自称「最强」。 |

### 2010年

#### 8月14日（Comic Market 78）— 妖精大战争 ～ 东方三月精（TH12.8）

| 项目 | 内容 |
|------|------|
| **登场定位** | **唯一自机 / 全程主角**（东方系列至今唯一单自机弹幕游戏） |
| **作品性质** | 东方Project第12.8作，衍生弹幕射击游戏，立绘由比良坂真琴（三月精漫画绘师）绘制而非ZUN |
| **故事起因** | 光之三妖精（桑尼米尔克、露娜切露德、斯塔萨菲雅）砸毁了琪露诺在雾之湖畔的冰屋。琪露诺写下宣战布告书，但「很快就忘记了」——直到樱花开时重新发现战旗，才出发复仇。 |
| **路线系统** | 游戏开始有三个独立关卡（对应三妖精）可自由选择顺序，不同顺序构成6条路线，各有独立剧情和结局。 |
| **剧情概要** | 第1话（共通）：琪露诺发现被砸毁的冰屋后愤然写下战书，但随即忘记。第2话起：琪露诺依次击败莉莉霍瓦特和大妖精，最终逐一战胜光之三妖精。结局中三妖精被迫帮琪露诺重建冰屋。 |
| **Extra篇** | 在射命丸文的指引下，琪露诺挑战赏花后的魔理沙（放水状态），并取得胜利。事后琪露诺吹嘘自己打败了「最强的魔法使」。 |
| **游戏系统** | 核心机制为「弹幕冻结系统」——琪露诺可释放冰之防护罩（Ice Barrier）冻结附近敌弹；冻结的弹幕会连锁冻结周围弹幕，冰块消散时具有杀伤力；Bomb「Perfect Freeze」可冻结全屏弹幕（含特殊炎弹），使用期间无敌。 |
| **意义** | 琪露诺作为东方Project历史上唯一一次**单自机无替换的主角作品**的主角，是其在官方作品中分量最重的一次出场。作品名直接冠以「妖精大战争」，与漫画《东方三月精》互相联动。 |

### 2017年

#### 8月11日（Comic Market 92）— 东方天空璋（TH16）

| 项目 | 内容 |
|------|------|
| **登场定位** | **自机**（夏之妖精线） |
| **称号** | 烈日下不融的冰精（溶けない炎天下の氷精） |
| **戏份描述** | 第二次成为正作自机角色。故事中，琪露诺背后被摩多罗隐岐奈打开了「门」，由此获得了不输给盛夏酷暑的力量。她精神上陷入「无敌状态」（设定文档原文），自称要「支配幻想乡」。 |
| **自机剧情** | 琪露诺带着暴走的力量一路击败一面Boss爱塔妮缇拉尔瓦、二面Boss坂田合欢等人，最后见到摩多罗隐岐奈。隐岐奈指出琪露诺的力量暴走会伤害自身，最终收回了她的力量。琪露诺拒绝成为隐岐奈的二童子继任者。 |
| **角色分量** | 正作第二次自机出场，获得了ZUN难得的角色成长性描写——「获得力量→膨胀→被回收力量」。但最终结局仍回归原状，体现了妖精「永不改变」的本质。 |
| **能力变化** | 能力表记从「操纵冷气程度的能力」改为「操纵冰程度的能力」（自辉针城起已变更）。 |

## 四、妖精大战争 专题：主角地位分析

### 4.1 东方Project史上的唯一性

妖精大战争（TH12.8）是东方Project至今为止**唯一一款单自机无替换的弹幕射击游戏**。琪露诺是全程唯一可操控角色，这使其在东方系列所有角色中拥有独特的历史地位——没有其他角色获得过以自己为唯一主角的官方STG作品。

### 4.2 与漫画的联动关系

妖精大战争的剧情直接承接《东方三月精S》番外篇，立绘由三月精漫画绘师比良坂真琴担当而非ZUN本人绘制，体现了游戏与漫画的深度联动。游戏中的角色表情和对话风格与漫画中一致。

### 4.3 故事结构

```
起因 → 冰屋被三妖精砸毁 → 琪露诺写宣战布告书 → 忘记此事
发展 → 春天发现战旗 → 出发复仇 → 击败莉莉霍瓦特、大妖精
高潮 → 逐一战胜光之三妖精（路线顺序可变，共6条路线）
结局 → 三妖精被迫重建冰屋
Extra → 在文指引下挑战魔理沙 → 战胜放水的魔理沙
```

### 4.4 游戏系统与角色能力的深度绑定

妖精大战争的**弹幕冻结系统**是琪露诺能力「操纵冷气程度的能力」在游戏机制层面的直接映射：
- Ice Barrier（冰之防护罩）：释放冷气冻结周围弹幕
- 连锁冻结：冻结弹幕的冷气会传播至周围弹幕
- Perfect Freeze（完美冻结）：全屏冻结，对应红魔乡以来的标志性符卡
- 冰块消散杀伤：冻结的冰块融化时具有攻击判定

这是东方系列中角色能力与游戏系统绑定最紧密的案例之一，琪露诺的能力被提升为核心游戏机制而非仅仅叙事设定。

### 4.5 在琪露诺角色历史中的定位

妖精大战争是琪露诺角色地位的**最高点**——在此之前她只是2面Boss和杂鱼角色，在此之后她又回归了杂鱼定位。这款作品固定了琪露诺「最强（自称）」的角色形象，也成为后来所有二次创作中「琪露诺是主角/强者」设定的官方来源。

## 五、戏份定位演变总结

### 5.1 按参战性质分类

| 类型 | 作品 | 次数 |
|------|------|------|
| **Boss**（敌方头目） | 红魔乡（2面）、妖妖梦（1面中Boss）、文花帖（Lv.2）、辉针城（1面中Boss）、弹幕天邪鬼（第1日）、弹幕狂们的黑市（2nd Market） | 6次 |
| **自机**（可选操控） | 花映塚、非想天则、天空璋 | 3次（另加妖精大战争1次为主角） |
| **主角**（唯一自机） | 妖精大战争 | 1次 |
| **客串** | 心绮楼（背景）、虹龙洞（卡牌） | 2次 |

### 5.2 按角色分量演变

```
2002 红魔乡     ★★   二面Boss，首次登场，中等分量
2003 妖妖梦     ★     一面中Boss，杂鱼级
2005 花映塚     ★★★  首次自机，⑨事件
2005 文花帖     ★     拍照对象，低级关卡
2009 非想天则   ★★★  格斗游戏自机
2010 妖精大战争 ★★★★★ 唯一主角，角色地位巅峰
2013 辉针城     ★     一面中Boss，ZUN明言「与杂鱼无异」
2014 弹幕天邪鬼 ★★    第一日Boss
2017 天空璋     ★★★   第二次正作自机
2022 弹幕狂们的黑市 ★★ 市场Boss
```

### 5.3 ZUN对角色定位的官方表态时间线

| 时间 | 出处 | ZUN表述 |
|------|------|---------|
| 2002年 | 红魔乡 omake.txt | 「脑袋不太好使。不过她本来就是个孩子而已。」 |
| 2003年 | 妖妖梦 omake.txt | 「并没有什么特别的目的」「仅仅因为在那里所以便攻击了」 |
| 2013年 | 辉针城 设定文档 | 「基本上和杂鱼没什么大区别」 |
| 2017年 | 天空璋 设定文档 | 「不知为何获得了不会输给盛夏酷暑的力量」 |
| 2020年 | 人妖名鉴 宵暗篇 | 「和主线剧情完全没有关系」「没有太深的设定」 |

这一系列表述呈现高度一致性：ZUN始终将琪露诺定位为脱离主线的、无深度设定的、类似杂鱼的「背景生态型」角色。**妖精大战争是唯一的例外**——在此作中她获得了主角级的待遇，且这一待遇至今未在其他作品中重现。
