## 一、Windows 版弹幕射击游戏（STG）

### 1.1 主要登场

| 时间 | 作品 | 定位 | 详细说明 |
|------|------|------|----------|
| 2003-08-17 | **东方妖妖梦** ~ Perfect Cherry Blossom（TH07） | **最终Boss**（Stage 6 Boss） | Windows版初登场。异变引发者——派遣妖梦收集幻想乡的"春之力"以使西行妖满开。作为STG正作初代6面Boss，地位重要。Extra Stage的结局揭示了关于西行妖封印的真相。 |
| 2004-08-15 | **东方永夜抄** ~ Imperishable Night（TH08） | **自机**（结界组——与八云紫组队） | 与八云紫组成"结界组"解决永夜异变。是她在STG正作中唯一一次作为自机登场。全程与紫搭档，台词量丰富但热情度适中，体现"慵懒的协助者"角色定位。Extra关卡中与藤原妹红有关于"死亡与不死"的哲学对话。 |
| 2005-08-14 | **东方文花帖** ~ Shoot the Bullet（TH09） | Level 拍摄目标（Level 10） | 射命丸文的摄影游戏，幽幽子作为被拍摄对象在Level 10登场。 |
| 2008-08-16 | **东方地灵殿** ~ Subterranean Animism（TH11） | **未直接登场** | 本作中未以角色身份登场。 |
| 2014-05-11 | **东方辉针城** ~ Double Dealing Character（TH14） | 登场于"魔理沙的魔法书"收录符卡 | 本作本体中未直接出场。 |
| 2021-11-28 | **东方虹龙洞** ~ Unconnected Marketeers（TH17） | **能力卡牌提供者** | 提供"幽明の境"相关能力卡牌。角色未直接出场，以卡牌形式关联。 |
| 2022-08-14 | **弹幕狂们的黑市** ~ 100th Black Market（TH17.5） | 能力卡牌提供者 | 继续以卡牌形式关联。 |

### 1.2 STG正作登场频率总结

- 幽幽子在STG正作中的直接登场共**2次**（妖妖梦作为Boss、永夜抄作为自机）
- 此后在STG正作中以本体形象直接登场较少，多通过卡牌、书籍等方式间接出场
- 她在STG中的角色地位集中于早期（TH07-TH08时期）

## 二、Windows 版格斗游戏（黄昏边境合作）

| 时间 | 作品 | 定位 | 详细说明 |
|------|------|------|----------|
| 2008-05-25 | **东方绯想天** ~ Scarlet Weather Rhapsody（TH10.5） | **自机 / 敌机** | 在格斗游戏中首次成为自机角色。称号"优雅的亡灵"。剧情：主动调查异常天象，最终教训了引发地震异变的比那名居天子。展示了独立调查者的主动性。主题曲《幽雅に咲かせ、墨染の桜 ～ Border of Life》。 |
| 2009-08-15 | **东方非想天则** ~ Hisoutensoku（TH12.3） | **自机 / 敌机** | 连续作为自机登场。称号延续"优雅的亡灵"。剧情中与巨大人偶事件间接相关。胜利台词体系全面展示了"死亡哲学"的表达风格。 |
| 2015-05-10 | **东方深秘录** ~ Urban Legend in Limbo（TH14.5） | **自机**（DLC追加角色） | 作为DLC追加角色登场。因对"都市传说"（虚实之间的存在）感兴趣而参与深秘异变。进一步确立了"境界管理者"的定位。 |
| 2017-12-29 | **东方凭依华** ~ Antinomy of Common Flowers（TH15.5） | **自机 / 敌机** | 作为"完全凭依异变"的参与者登场。称号升格为"幽明之境的分界者"。展现了作为生死境界管理者的自觉。 |
| 2021-10-24 | **东方刚欲异闻** ~ Touhou Gouyoku Ibun | 待确认 | 在动作游戏中以相关角色身份登场。 |

### 格斗游戏登场频率总结

- 幽幽子在格斗游戏中的登场频率高于STG正作
- 绯想天～凭依华连续4作均为自机角色，是格斗游戏系列的常驻成员
- 格斗作是她获得最多台词量和角色塑造的作品

## 七、关键事件时间线（角色内时间）

以下是幽幽子在东方Project世界观内的关键事件时间线。由于幻想乡内的时间流动较为模糊，以下时间顺序基于作品内逻辑推定：

```
【数百年前】
  ├─ 幽幽子作为西行寺家的女儿出生（人类）
  ├─ 西行妖因吸收过多死者魂魄而妖化
  ├─ 为了封印西行妖，幽幽子的身体被埋葬于树下
  ├─ 幽幽子死亡，化为亡灵
  ├─ 成为白玉楼的主人
  ├─ 与八云紫成为旧友
  └─ 魂魄家开始侍奉西行寺家

【数年前】
  └─ 前任庭师魂魄妖忌隐退，妖梦继任

【春雪异变（妖妖梦）】
  ├─ 幽幽子派妖梦收集春之力
  ├─ 幻想乡陷入异常寒冬
  ├─ 灵梦/魔理沙解决异变，击败幽幽子
  ├─ 西行妖未能满开
  └─ 幽幽子从紫口中得知真相（西行妖满开会让她消失）

【永夜异变（永夜抄）】
  └─ 幽幽子与紫组队前往永远亭

【绯想天异变（绯想天）】
  ├─ 幽幽子注意到异常天象
  ├─ 主动离开白玉楼调查
  └─ 击败比那名居天子，恢复天气平衡

【深秘异变（深秘录）】
  └─ 幽幽子因对都市传说感兴趣而参与

【完全凭依异变（凭依华）】
  └─ 幽幽子以"幽明之境的分界者"身份参与
```

## 八、角色定位演变总结

```
妖妖梦（2003）──── 新作初登场
  ├─ 种族：亡灵（幽灵公主）
  ├─ 定位：最终Boss（异变的引发者）
  ├─ 称号：幽冥楼阁的亡灵少女 / 华胥的亡灵
  └─ 核心主题：追寻过去的记忆与西行妖的封印
      │
      ▼
永夜抄（2004）──── 第一次作为自机
  ├─ 定位：自机（紫的同伴）
  ├─ 关系深化：与紫的旧友关系被进一步刻画
  └─ 角色更加日常化、明朗化
      │
      ▼
绯想天 / 非想天则（2008-2009）── 格斗游戏活跃期
  ├─ 定位：自机（独立的调查者）
  ├─ 角色定型：死亡哲学的表达、与妖梦的互动模式
  └─ 从"异变引发者"转变为"秩序维护者"
      │
      ▼
深秘录 / 凭依华（2015-2017）── 境界管理者
  ├─ 定位：自机（境界的守护者）
  ├─ 称号升格：幽明之境的分界者
  └─ 从个人驱动走向责任驱动
      │
      ▼
虹龙洞 / 黑市（2021-2022）── 间接登场期
  └─ 以卡牌形式间接关联，本体未直接出场
```

### 关键趋势

1. **从STG到格斗**：妖妖梦（最终Boss）→ 绯想天/非想天则/深秘录/凭依华（格斗作常驻自机）

2. **从反派到盟友**：妖妖梦（异变引发者）→ 永夜抄/绯想天（异变解决者）

3. **从个人到责任驱动**：妖妖梦（追寻自我记忆）→ 凭依华（作为境界管理者的职责）

4. **核心关系固定**：与妖梦的主从关系（全作品贯穿）+ 与紫的旧友关系（核心设定）

5. **外表与内核的统一过程**：称号从"幽冥楼阁的亡灵少女"到"幽明之境的分界者"，体现角色地位升格
