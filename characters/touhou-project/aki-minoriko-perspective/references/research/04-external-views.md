# 04 他者视角（External Views）

## 粉丝认知与定位

### 核心昵称
- **穰子**：中文圈名称
- **ににこ**：日文圈爱称
- **丰收之神**：中文圈常见称呼
- **秋妹**：中文圈简称
- **芋神様**：日文圈戏称——因其与红薯、芋头等秋季食物的关联

来源：玩家共识 — 确定

### 人气地位
- 在东方Project人气投票中，穰子人气属于中下游水平，但高于姐姐静叶
- 在秋之姐妹中，穰子的人气明显高于静叶
- 作为"食物神"的身份让她在某些美食相关的二次创作中较为活跃
- 在非想天则的支援角色登场增加了她的曝光度

来源：玩家共识 — 确定

## 常见二设形象

### "吃货神"
穰子二设中最著名的形象——她与食物、丰收密不可分：
- 总是带着各种秋季食物（红薯、栗子、南瓜、柿子等）
- 做菜好吃的设定在二次创作中非常普遍
- 被想象成喜欢开收获祭、邀请大家来吃的开朗神明
- 有时被画成"边走边吃"的可爱形象

来源：玩家共识 — 二设

### 活泼妹妹与沉静姐姐
姐妹对比是二设最经典的主题：
- 穰子外向、主动、话多——静叶内向、安静、话少
- 穰子代表秋天"生"的一面——静叶代表秋天"逝去"的一面
- 穰子在很多人设中是"照顾姐姐"的角色
- 静叶是"我妹妹很受欢迎我很欣慰"的一方

来源：玩家共识 — 二设

### 与人类亲近
因为掌管丰收——直接影响人类生活——穰子在二设中比很多神明都更亲近人类：
- 会去人类的丰收祭上帮忙
- 和人类分享食物
- 对农作物生长了如指掌
- 被画成穿着围裙在厨房忙碌的样子

来源：玩家共识 — 二设

### 与稻田姬的关系
主题曲"因为会被稻田姬殿下骂的"中出现的"稻田姬"在二设中被扩展为：
- 秋之姐妹的直属上司（掌管更大范围的丰收）
- 穰子对她有些敬畏但也有时调皮
- 穰子被骂后会找姐姐撒娇

来源：玩家共识 — 二设

## 与其他角色的关系（二设）

| 角色 | 二设形象 |
|------|----------|
| **秋静叶** | 最亲爱的姐姐，两人性格互补，齐心协力的好姐妹 |
| **稻田姬** | 上司/导师，有点害怕但也很尊敬 |
| **东风谷早苗** | 击败她的人类巫女，但二设中常被友好相处 |
| **比那名居天子** | 同为不强势的神明，有时一起玩 |
| **莉莉怀特** | 报春的妖精——春天的使者，和秋天之神偶尔有"交接"二设 |

## 一设与二设辨析要点

| 内容 | 分类 | 说明 |
|------|------|------|
| 穰子比静叶受欢迎 | 一设事实 | 求闻口授确认，更多登场作品 |
| 穰子很会做饭 | 一设基础+二设放大 | 丰收之神自然和食物相关 |
| 穰子经常去找人类玩 | 推测→二设主流 | 她对人类友好，但频率并未一设明确 |
| 穰子害怕稻田姬 | 推测 | 主题曲名暗示了这一点 |
| 穰子爱吃红薯 | 二设 | 源于"甘甜丰收之神"的称号和秋日意象 |
| 穰子性格冒失 | 二设 | 她只是活泼开朗，不是笨拙 |

来源：官方/玩家共识 — 确定/二设
