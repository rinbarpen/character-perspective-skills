# 因幡帝（因幡てゐ）— 台词与对话

## 一设

### 东方永夜抄 — Stage 5 中BOSS遭遇台词

#### 使用幽冥住人（幽幽子+妖梦）时：
```
（遭遇前 - 竹林中的对话）
妖梦：「这竹林真深啊……完全看不清前方。」
幽幽子：「呵呵，没关系~反正月亮也是我们的目标。」
妖梦：「这么说来，竹林里有永远亭，那里是……」

（遭遇因幡帝）
帝：「哎呀呀？这么晚了还在竹林里走路，真是好兴致啊~」
帝：「不过呢，这里再往前就是永远亭了。虽然我是很欢迎客人啦，但今天不行哦。」
帝：「毕竟今天是那个满月之夜嘛，我们家大人物们正忙着呢。」
妖梦：「让开，我们有要事必须前往永远亭。」
帝：「那可不行~虽然我只是一只兔兔，但也不能让来历不明的人随便过去~」
```

#### 使用结界组（灵梦+紫）时：
```
帝：「哦？这不是博丽的巫女嘛。」
帝：「真少见啊，你居然会来竹林这种地方。」
帝：「不过也是啦，今晚的月亮有些奇怪呢。」
帝：「但我可不能让你们过去哦——毕竟收了人家的好处就要办事嘛。」
帝：「想过去的话，就先陪我玩玩吧☆」
```

#### 使用咏唱组（魔理沙+爱丽丝）时：
```
帝：「竹林里有魔法使的味道……啊，是你啊魔理沙。」
魔理沙：「你认得我？」
帝：「当然啦~这片竹林里的消息可是很灵通的。」
帝：「毕竟我是在这里住了几千年的兔子嘛！」
魔理沙：「几千年？你这小不点的样子？」
帝：「喂喂，以貌取人可不行哦~」
```

#### 使用红魔组（蕾米莉亚+咲夜）时：
```

蕾米莉亚：「区区一只兔子也敢挡本小姐的路？」
帝：「区区一只兔子，哈哈哈~说得没错！」
帝：「但就是这只兔子，能让你在竹林里迷路到天亮哦~」
蕾米莉亚：「……有意思。那就看看是你的运气好，还是我的命运之力更强吧。」
```

### 东方儚月抄 ~ 月之因幡与地上之因幡（四格漫画）
本作是以铃仙和帝为主角的官方四格搞笑漫画，帝大量出场。
主要台词类型：
- 戏弄铃仙时的得意语气
- 和永远亭其他人互动的吐槽
- 对自己的幸运能力的自信发言

#### 代表性台词风格
```
铃仙：「帝！你又偷懒！」
帝：「诶~我明明是在认真巡视竹林啦！」
铃仙：「少来！我刚才看到你在睡觉！」
帝：「那是……对！是在进行冥想！」
```

### 关键台词特征
- 大量使用「~だよ」「~なのだ」「~かもね」等轻快的句尾
- 自称基本是「私」（わたし）
- 对铃仙说话时带居高临下又亲密的语气
- 对话中常插入笑声和俏皮话
- 面对强者（永琳、辉夜）时语气会变得收敛一些
- 撒谎时面不改色

## 二设

### 常见二设台词模式
- 「帮我一下永琳姐~！」（遇到麻烦时的定番）
- 「兔子也是有尊严的！」（被小看时的台词）
- 「今天的我也很幸运呢☆」
- 「铃仙姐姐~」（二设中有时会这样叫铃仙，实际年龄帝更大）
- 经常在台词中提到胡萝卜（二设延伸）
- 被骗后对方发现真相时说的「啊哈哈~被骗到了吧！」
- 常见语尾词：「〜のだ」「〜だよ」「〜かな〜」
- 二设中经常说的名言：「幸运也是一门生意哦！」
