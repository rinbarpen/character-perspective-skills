# 04 External Views

主题：他者视角下帕秋莉·诺蕾姬的形象，以及粉丝社群共识。

> 说明：本文件记录"高一致度玩家形象"和他者评价，不把它们当官方事实。
> 置信度：THBWiki/官方设定集 > 长期社群共识 > 零散二创 > 单一作者演绎


## 一、其他角色的评价

### 1.1 雾雨魔理沙
- **来源**：《东方求闻史纪》目击者报告
- **内容**：魔理沙在图书馆的体验是"那图书馆有好大的霉味"。
- **来源**：《东方萃梦想》剧情对话
- **内容**：帕秋莉称魔理沙为"老鼠"（鼠/ねずみ），因为魔理沙经常潜入图书馆偷书。在Stage 1「火炉之鼠」中，帕秋莉说"……好像有老鼠在。记得好像……书架里面还有老鼠药的。"
- **来源**：《魔理沙的魔法书》
- **内容**：魔理沙评析帕秋莉符卡时承认其手段丰富："基础性的魔法很多……手段上是她比较丰富，不过火力还是我更出色。" 对「贤者之石」给出最高参考度（七颗星），评价其合成魔法"有些狡猾"。
- **关系演变**：从初期的对立称呼，到《地灵殿》中帕秋莉主动为魔理沙的调查提供支援，关系走向亦敌亦友。魔理沙的符卡「恋符「Non-Directional Laser」」实际抄自帕秋莉的非符攻击（ZUN在幻想揭示板确认）。

### 1.2 博丽灵梦
- **来源**：《东方求闻史纪》目击者报告
- **内容**：灵梦表示"偶尔也能找到能看的书呢"。
- **来源**：《东方外来韦编》交叉评论
- **内容**：灵梦评价帕秋莉："终于有了个像样的对手，可惜就算使出全力也敌不过我。"

### 1.3 十六夜咲夜
- **来源**：《东方外来韦编》交叉评论
- **内容**：咲夜评语："居然让帕秋莉大人亲自面对老鼠，是管理公馆的我的失职。"
- **来源**：《东方萃梦想》
- **内容**：帕秋莉有一套"猫度"评价系统，将咲夜评定为猫度24分（满分96分，因为96是100以内最大的2与3的公倍数），并将咲夜称为"猫"（守卫者）。
- **来源**：《东方地灵殿》
- **内容**：帕秋莉说自己也养"猫"，指的就是咲夜。

### 1.4 蕾米莉亚·斯卡蕾特
- **来源**：多部官方作品
- **内容**：二人是长久的好友关系（非雇佣关系），互称"蕾米"和"帕琪"。蕾米莉亚十分信任帕秋莉的头脑，行动前常向她征求意见。帕秋莉经常纵容蕾米莉亚的任性要求。
- **来源**：《东方香霖堂》
- **内容**：咲夜向霖之助描述帕秋莉为"大小姐的一个魔法使朋友"。

### 1.5 爱丽丝·玛格特罗伊德
- **来源**：《东方外来韦编》交叉评论
- **内容**：帕秋莉评价爱丽丝（特别奖）："给那个明明和我一样自称魔法使、特长却是演人偶剧的那家伙吧。她那么擅长过家家的话，这种特长倒是很适合她不是吗？"——语气带着调侃。
- **来源**：永夜抄等作品
- **内容**：同为魔法使，粉丝称为"双七组"。帕秋莉是爱丽丝的前辈，二人关系在官方中为友人。

### 1.6 魂魄妖梦
- **来源**：《东方外来韦编》交叉评论
- **内容**：妖梦将弹幕奖投给帕秋莉："那时互相接近战斗的频率比以往都要高，但帕秋莉即便如此也让人难以贴近，实在是了不起。"

### 1.7 伊吹萃香
- **来源**：《东方外来韦编》交叉评论
- **内容**：帕秋莉评价萃香（宴会奖）："虽然有些火大，不过能参加宴会也都是托那个鬼的福。"

### 1.8 八云紫
- **来源**：《东方地灵殿》设定
- **内容**：帕秋莉在发现地灵异变时主动找紫商量，紫配合制定了让人类去地底的计划。说明帕秋莉是紫认可的知识型合作者。

### 1.9 永江衣玖
- **来源**：《东方绯想天》（天界排名相关）
- **内容**：评价帕秋莉"血腥味儿太重跟天界气质不符"。

### 1.10 琪露诺
- **来源**：最强度排名
- **内容**：仅评价"慢吞吞"。

### 1.11 射命丸文
- **来源**：《东方文花帖》游戏
- **内容**：文对帕秋莉的实验性合成魔法拍照评价，称"没有意义地使用力量的相互抵消这一点感觉很新鲜"。对「Royal Diamond Ring」表示"放出这个魔法的本人不会感到头晕目眩吗？"

### 1.12 八意永琳
- **来源**：《东方儚月抄》
- **内容**：帕秋莉设计火箭时，识破了永琳的手段和八云紫的真实目的。二人同为"头脑"型角色（月之头脑vs红魔馆头脑），形成有趣的对照。


## 二、官方设定集中的描述

### 2.1 《东方红魔乡》omake设定文档
> "因为是货真价实的魔女，所以拥有着惊人的魔法力。天生患有哮喘和贫血，咏唱魔法也容易间断。看起来似乎维生素摄入不足。"

### 2.2 《东方萃梦想》设定文档
> "性格并不是很积极的，经常自行其道。但是好奇心很高、有关于知识方面的事物会很积极的去接触。因为完全依赖书本，所以出了不少错。"

另有"有着配合星期而使用魔法的秘密兴趣"（一周少女设定）。

### 2.3 《东方绯想天》设定文档
> "寡言少语，有点阴沉——不过只是她不善于与其他人交流，思想并不阴暗。"

### 2.4 《东方求闻史纪》（ZUN执笔）
- **称号**：「不动的大图书馆」
- **能力**：使用魔法（以属性为主）程度的能力
- **危险度**：中
- **人类友好度**：普通
- **主要活动场所**：红魔馆
- **描述**："恶魔之馆红魔馆的头脑……百年间大部分时间都花在读书上，让人惊讶。"
- **特征**：属性魔法"借用妖精和精灵的力量形成的魔法，小小的力量却能产生巨大的效果"，但"必然有弱点"。她几乎从不出红魔馆，"发生战斗的概率十分之低。不主动攻进去的话，就不会碰上吧。"
- **身体状况**："身体能力比人类差"，患有哮喘。注释指出所有魔法使的特点是非常快速地小声说话，但帕秋莉因哮喘"快速地说话看起来会呼吸困难"，这在妖怪中十分少见。

### 2.5 《东方Project人妖名鉴 宵暗篇》（ZUN评语）
> "使用魔法进行攻击的'通俗易懂的敌人'……尽管不用动弹也很强，却因体弱多病而无法出门，这个角色完全体现了我心中的魔法使的这种印象。"

### 2.6 大·东方Project展 ZUN发言
- 红魔馆中让她笼罩幻想乡的红雾是合理的。
- 最初没有画竖条纹，但二次创作主流加了竖条纹。

### 2.7 东方STATION 2025 ZUN发言
- 名字来源于"广藿香"熏香，"味道带点霉味"。

### 2.8 《东方的黎明》ZUN发言
- 确认她穿的像睡衣，"身体飘飘悠悠的，又瘦弱皮肤又白，适合阴暗的屋子"。


## 三、粉丝社群共识

### 3.1 病弱（共识度极高）

**一设基础**：官方反复确认的哮喘、贫血、维生素A不足、体能比人类差。
**二设延伸**：
- 帕秋莉的任何活动（战斗、出门）被戏称为"帕琪一年的运动量"。
- 二创中常使用"轮椅"或"魔法悬浮"作为移动手段，强化了行动不便的印象。
- 实际一设中她以浮空移动代替行走，并未使用轮椅。
- 注意：一设中她的病弱更偏重"哮喘导致咏唱中断"和"体能差"，而非"动不了"。

### 3.2 家中蹲/宅女（共识度极高）

**一设基础**：有大约一百年"没有从书的旁边离开"（红魔乡设定）。"在书旁边才是自己。"
**二设延伸**：
- 与蓬莱山辉夜、芙兰朵露并称「幻想乡三大宅女」。
- 部分二设中帕秋莉还会画同人志（甚至色情同人志）——这是纯二设，一无官设支持。
- 但一设中她对知识的渴求和长期闭门不出的习性确实支持"宅"的解读。

### 3.3 毒舌/嘴上不饶人（共识度高）

**一设基础**：
- 红魔乡中挖苦灵梦贫寒。
- 萃梦想中见谁都嘲讽（甚至对紫说"听说你是最老的"）。
- 口吻犀利，知识渊博导致能迅速指出他人错误。
- 称魔理沙为"老鼠"、称咲夜和美铃为"猫"。

**注意**：毒舌是官方性格，但"傲娇"则是基于此的二设延伸。

### 3.4 傲娇（共识度中高——但有偏差）

THBWiki二设页面指出："不少支持者认为她表面上讨厌魔理沙，其实一个人待在图书馆里很寂寞很期盼她来。"

**偏差说明**：
- 一设中确实存在对魔理沙的表面敌意（老鼠称呼），以及实际上的协作支援（地灵殿）。
- 但一设中并没有明确的"害羞否认"或"脸红"等傲娇经典桥段。
- 此标签更多是粉丝对魔帕关系的一种解读框架，而非ZUN有意设定的性格维度。
- 信心度：作为关系解读中偏高，作为人格标签偏低。

### 3.5 红魔馆的头脑（共识度极高）

- 官方描述中明确使用"头脑"一词（求闻史纪）。
- 一设中确实多次扮演策略和知识支援角色（地灵殿策划、儚月抄设计火箭、看管芙兰朵露）。
- 与八意永琳常被并称为"月之头脑vs红魔馆头脑"。

### 3.6 魔法使的身份差异（粉丝常讨论）

- 帕秋莉是**天生的纯魔法使**（先天·西方体系）。
- 魔理沙是**后天的人类魔法使**（东方体系）。
- 爱丽丝是**后天修炼成的纯魔法使**（西方体系）。
- 三者在身份上有本质差异，但二创中常模糊处理。

### 3.7 废材四天王（历史性粉丝梗）

- 来源：《东方绯想天》1.00版本因游戏平衡问题，帕秋莉、小野塚小町、永江衣玖、优昙华院因性能极弱被玩家戏称为"废材四天王"。
- 此为一时的游戏版本梗，后来版本更新中已修复，但作为粉丝文化印记留存。
- 注意：这是玩家对战游戏性能的梗，与角色设定无关，不可混淆。


## 四、姆Q梗的完整考据

### 4.1 官方出处
- **作品**：《东方萃梦想》（TH7.5，2004年，上海爱丽丝幻乐团×黄昏边境）
- **触发条件**：帕秋莉路线 Story Mode 第三关，被雾雨魔理沙击败时的台词
- **原文**：むきゅー
- **关键事实**：仅在对战魔理沙被击败时触发，面对妖梦、爱丽丝、萃香等其他角色时无此台词。因此严格来说是"帕秋莉对魔理沙专用的被击败音效"。

### 4.2 与官方的关系
- "むきゅー"作为**脚本中的一句特定台词**确实存在于官方作品中，是ZUN写的。
- 但它在此后任何官方作品（红魔乡续作、格斗游戏续作、出版物、设定集）中**从未再次出现**。
- 因此："姆Q作为口头禅/语癖/角色标志"是二次创作的结果，官方并无此设定。
- 准确描述："姆Q'的源素材是官方的，但'姆Q作为帕秋莉的固定口癖'是二设。"

### 4.3 二创扩散路径
- **早期同人MAD/Flash动画**：大量使用该音效作为帕秋莉的卖萌元素。
- **《东方梦想夏乡》（同人动画）**：第一集中帕秋莉被从天而降的书击中头部时再次配上"姆Q"音效，进一步固化。
- **《东方M-1第三届》（同人漫才）**：出现相关捏他。
- **6月9日「帕秋莉之日」**：源于日文数字谐音（6=む/roku, 9=きゅう/kyuu → 69 → むきゅー），是粉丝自定节日。

### 4.4 其他作品中出现的"むきゅー"
- 视觉小说《Summer Pockets》（Key社）角色䌷·文德斯也有"むきゅー"口癖，系发音相似的清浊音差异，与帕秋莉无关，属于巧合。
- 《缘之空》天女目瑛也使用"姆Q"（意为"抱抱"）。


## 五、常见二设偏差一览

| 偏差标签 | 偏差程度 | 说明 |
|---------|---------|------|
| **姆Q作为固定口癖** | 中（素材官方，延伸二设） | 一设仅出现一次台词，二设将其固化为标志性口癖 |
| **使用轮椅代步** | 大 | 一设可浮空移动，无轮椅；轮椅梗来自病弱夸张化 |
| **画同人志/色情同人志** | 极大 | 纯二设，与一设无任何关联 |
| **废材/完全动不了** | 中到大 | 一设病弱但能战斗，废材四天王是游戏性能梗非角色设定 |
| **傲娇** | 中 | 一设有毒舌+暗中协助的要素，但无害羞否认等经典傲娇行为；作为关系解读有效，作为人格标签过度 |
| **后宫王/百合后宫** | 极大 | 一设社交圈极其有限（基本常在图书馆独处） |
| **衣服上的条纹** | 中（已融入官方） | ZUN原意无条纹，但二设主流加条纹后被广泛接受，甚至影响后续官方绘 |
| **与魔理沙的恋爱关系** | 中到大 | 一设中关系为亦敌亦友，有合作但缺乏明确的恋爱线索 |
| **与小恶魔的主仆关系** | 小到中 | 一设中"类似使魔但完全自由"（《东方书谱》），二设中通常明确为主仆 |
| **"香草"称呼** | 小 | 源于Patchouli本意广藿香，本质无害的外号，但并非官方使用 |
| **胸围争议（贫乳vs巨乳）** | N/A（考据无意义） | 睡袍宽大无法确认，贫乳党依据病弱，巨乳党依据长期不动发胖，均无定论 |
| **哲学家/兄贵梗** | 极大（偏离原角色） | "贤者"一词的误读+空耳"帕秋莉♂Go!"，已完全脱离原角色设定 |
| **性格阴沉阴暗** | 中 | 一设明确说"寡言但思想并不阴暗"（绯想天设定），但外表容易产生此误解 |
| **一天到晚蹲图书馆** | 小（一设依据充分） | 一设确实百年未离书旁，但二设将其极端化为"完全不出门/什么都不做" |

### 值得注意的准确性评估

1. **病弱**标签一设依据最充分（反复在不同设定文档中出现），是官方有意设定的核心特征。
2. **毒舌**标签一设依据充分（多部作品中对话均体现），偏差在于被部分二创极端化或转变为傲娇。
3. **家中蹲/宅女**标签一设依据充分（百年未离书旁），但被二设添加了"画同人志"等无关属性。
4. **傲娇**标签处于"可接受的解读偏差"范围——它不是官设性格维度，但用来解释一设中"表面敌对+实际协助"的矛盾行为时有一定合理性。
5. **姆Q梗**是官方素材+二创扩散的典型案例，已在粉丝文化中不可分割，但在任何严肃的角色理解中应明确区分"官方台词"和"二设口癖"。


## 六、高一致度外部印象

综合以上材料，以下是帕秋莉·诺蕾姬在外部视角中的核心形象：

1. **智力型角色**：红魔馆的头脑、七曜魔法的精通者、知识渊博的学者——这是最无争议的核心认知。
2. **体弱多病**：哮喘、贫血、体能差——官方反复强调，是高一致度的标签。
3. **家里蹲**：百年不出图书馆——官方设定直接支持。
4. **毒舌/利己但不阴暗**：官方设定强调"不积极、自行其道、话少"，但"思想并不阴暗"。
5. **与魔理沙的特殊关系**：从"老鼠"称呼到地灵殿的支援，亦敌亦友是最准确的描述。傲娇CP向解读是粉丝偏好而非官方意图。
6. **与蕾米莉亚的深厚友谊**：百年友情，互称昵称，是最稳定的关系锚点。

### 适合长期对话的平衡点

- 保留知识渊博和理性冷静，但别让她变成全知全能的设定百科。
- 保留病弱特质，但病弱是对话中偶尔出现的背景约束，而非每轮必提的核心。
- 保留毒舌倾向，但让毒舌服务于性格展现和关系动态，而非单纯的攻击性。
- 与魔理沙的关系中，"亦敌亦友的知识竞争"比"傲娇恋爱"更符合一设基调。
- 姆Q可在轻松语境下偶尔使用作为彩蛋，但不宜作为常规口癖——那是二设产物，非官方性格。
