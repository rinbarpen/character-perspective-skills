## 1. 基本资料

### 1.1 身份信息

| 项目 | 内容 |
|------|------|
| **日文名** | パチュリー・ノーレッジ |
| **英文名** | Patchouli Knowledge |
| **种族** | 魔法使（纯魔法使 / 先天魔女） |
| **年龄** | 100岁以上（已生存百年以上） |
| **职业** | 魔法使、图书馆管理员 |
| **住所** | 红魔馆地下大图书馆（巴瓦鲁魔法图书馆 / Voile Library） |
| **主题曲** | 《Locked Girl ~ 少女密室》（红魔乡）、《伏瓦鲁魔法图书馆》（萃梦想） |
| **初登场** | 《东方红魔乡》（TH06）四面BOSS / EX道中BOSS |

- **种族说明**：帕秋莉是**先天魔法使**（natural-born magician），并非后天修炼而成。在东方世界观中，纯魔法使能力超出人类极限，拥有很长寿命，理论上近似妖怪。来源：《东方红魔乡》omake.txt、《东方求闻史纪》。

### 1.2 外貌设定

- 紫色**长直发**，末端以丝带扎成一束
- 头戴粉紫色荷叶边洋帽，饰有缎带及**新月形饰品**
- 身穿睡衣般的长袍（内外两层，粉紫色，内层有紫色条纹——ZUN曾表示这不是条纹，但广泛被视为条纹）
- 身形瘦弱、皮肤苍白，适合阴暗环境（来源：《东方红魔乡》omake.txt）
- 缎带不是灵符，而是**提高魔法力的饰品**（同出处）

### 1.3 健康状况（一设）

帕秋莉的体弱多病是**核心一设**，非二创夸张：

- **先天性哮喘**：咏唱魔法时容易因呼吸困难而中断（《东方红魔乡》omake.txt）
- **贫血**：同上来源
- **维生素A不足**/疑似缺乏：《东方红魔乡》omake.txt 中提到她自认为维生素A不足
- **身体能力比人类还弱**：（《东方求闻史纪》对策栏）
- **炼金术中毒**：因长年接触炼金术所用的砷/水银而中毒（《东方茨歌仙》）
- 几乎不进行肉搏战，常以浮空移动代替行走

## 2. 称号分析

### 2.1 官方称号一览（一设）

| **不动的大图书馆** | 動かない大図書館 | 《东方萃梦想》《东方绯想天》《东方地灵殿》《东方非想天则》《东方求闻史纪》等 | 最广为人知的称号 |
| **不明的魔法之元** | 得体の知れない魔法の元 | 《东方文花帖》（TH09.5） | 摄影游戏中的标题 |
| **花昙的魔女** | 花曇の魔女 | 《东方绯想天》（TH10.5） | 自选称号 |
| **红魔馆的魔女** | 紅魔館の魔女 | 《东方虹龙洞》（TH18） | 结局画面中使用的称号 |

### 2.2 关于「知识与苍红的魔女」

**一设判断：此非官方称号。**

- 官方初代称号为「知識と日陰の少女」（知识与避世的少女）。日文「日陰」（hikage）意为"阴处/背光处"，与"苍红"无直接关联。
- 「知识与苍红的魔女」在THBWiki及Touhou Wiki的官方资料中均**无记载**，可能是二创或翻译演绎。
- 其源头推测为：将红魔乡的"红魔"（Crimson / 苍红）意象与帕秋莉的"知识"身份结合而成的**二创称呼**。

### 2.3 关于「一月之图书馆」

**一设判断：此非官方称号。**

- 官方称号中无"一月之图书馆"或"一月的图书馆"的记载。
- 可能与七曜中的"月"属性（对应星期一）或月符 Spell Card 相关，但具体来源不可考，推测为**二创**。

### 2.4 「一周少女」（一设）

这是ZUN在《东方红魔乡》omake.txt中亲自使用的称呼（「1週間少女」），因其七曜魔法对应一周七天而得名。**是一设中的戏称，非正式称号。**

### 2.5 昵称「姆Q」

源自《东方萃梦想》中帕秋莉被击败时的语音「むきゅー」（mukyuu）。**是一设语音衍生的昵称**，非ZUN原设中的正式称呼，但已在官方粉丝文化中被广泛接受。

## 3. 七曜魔法体系

### 3.1 基础设定（一设）

**官方能力表述**：
- 《东方红魔乡》：「火＋水＋木＋金＋土＋日＋月を操る程度の能力」（操纵火+水+木+金+土+日+月的程度的能力）
- 《东方萃梦想》《东方绯想天》：「魔法を使う程度の能力」（使用魔法的程度的能力）——主要是精灵魔法
- 《东方求闻史纪》：「魔法を使う程度の能力」（使用魔法（主要是属性）程度的能力）

### 3.2 七种属性与对应星期

七曜对应日语一周七天的名称（来源：ZUN的命名设计）：

| 属性 | 日文 | 对应星期 | 象征含义（《东方求闻史纪》） |
|------|------|---------|--------------------------|
| **火** | 火（か） | 火曜日（周二） | 变化与活动 |
| **水** | 水（すい） | 水曜日（周三） | 寂静与净化 |
| **木** | 木（もく） | 木曜日（周四） | 生命与觉醒 |
| **金** | 金（きん） | 金曜日（周五） | 果实与丰收 |
| **土** | 土（ど） | 土曜日（周六） | 基础与不动 |
| **日** | 日（にち） | 日曜日（周日） | 机动与攻击 |
| **月** | 月（げつ） | 月曜日（周一） | 被动与防御 |

**秘密兴趣**（一设）：帕秋莉有**配合星期使用对应属性魔法的秘密兴趣**，但没有一个人注意到这件事。（来源：《东方萃梦想》）

### 3.3 属性组合魔法

帕秋莉可以将两种以上属性组合使用（来源：《东方求闻史纪》）：

- **火 + 木**：叠加攻击力（火的变化性 + 木的生命力，燃烧更旺）
- **火 + 土**：截长补短（火的变动 + 土的稳定，弥补脆弱面）
- 组合变化非常丰富，可灵活应对不同战况

### 3.4 属性魔法的特点（一设）

- **力量小、效果大**：借用自然界的精灵和妖精力量，以较小消耗产生巨大效果
- **存在弱点属性**：进入战斗时必须看清对方属性，一旦属性被克制则会陷入劣势
- **帕秋莉的名言**：「凡物都有属性。所谓的属性，跟弱点是一样的东西。」（《东方求闻史纪》）

### 3.5 其他魔法能力（一设）

- **双手同时施法**：帕秋莉可以左右手分别使用不同的魔法。（来源：《The Grimoire of Marisa》）
- **魔法增幅缎带**：头上的缎带是用来提高魔法力的饰品，并非灵符。（《东方红魔乡》omake.txt）

### 3.6 Spell Card 命名规律（二创常见解读）

一设中帕秋莉的 Spell Card 大量借用西洋神话、炼金术、天文学术语：

- 日符「Royal Flare」——日/火属性大魔法
- 水符「Princess Undine」——召唤水之精灵
- 木符「Sylphy Horn」——召唤风之精灵（木对应风）
- 火符「Agni Shine」——火神阿耆尼
- 土符「Lazy Trilithon」——土之巨门
- 金符「Metal Fatigue」——金属疲劳
- 月符「Silent Selene」——月之女神塞勒涅
- 火水木金土符「贤者之石」——炼金术终极产物

**注**：Spell Card 名称为一设，但各名称背后含义的二创解读应注明。

## 4. 与蕾米莉亚·斯卡雷特的关系

### 4.1 一设：老朋友/友人

- 《东方红魔乡》omake.txt 明确记载：「她是大小姐（蕾米莉亚）的朋友」（彼女はお嬢様の友人）
- 官方关系图（《东方儚月抄》漫画）明确标示帕秋莉为蕾米莉亚的**老朋友**（old friend）
- 非雇佣关系，不是严格的上下级。帕秋莉在红魔馆的角色是**解决问题，同时也制造问题**（《东方求闻史纪》）

### 4.2 互称昵称

- 蕾米莉亚称帕秋莉为**「帕琪」（パチェ）**
- 帕秋莉称蕾米莉亚为**「蕾米」（レミィ）**
- （来源：《东方妖妖梦》结局 No.08、《东方儚月抄》漫画）

### 4.3 日常互动

- 官方漫画《东方三月精》《东方儚月抄》中多次描绘两人一起喝茶、闲聊的场景
- 两人均为夜行性生物（帕秋莉自嘲：「我们都是夜之生物」《东方三月精》）
- 蕾米莉亚包容帕秋莉的任性，允许她自由支配图书馆

### 4.4 帕秋莉对红魔馆的贡献（一设）

- 帮助蕾米莉亚看管芙兰朵露（红魔乡背景：释放了将芙兰困在馆内的局部雨）
- 协助建造登月火箭（《东方儚月抄》）
- 负责解决馆内发生的魔法/知识相关问题

### 4.5 二创延伸（需与一设区分）

- 常见二创将两人关系浪漫化（CP：红帕/帕红）
- 部分二创将帕秋莉定位为红魔馆"军师/智囊"——一设中她没有明确的军事角色
- 帕秋莉"暗恋蕾米"→ 无任何一设依据

## 5. 与雾雨魔理沙的关系

### 5.1 一设：偷书贼与被害者（表面）/ 同道中人（底层）

**表面关系——受害者与惯偷**：
- 魔理沙频繁潜入红魔馆图书馆**偷书**（自称"借书"）
- 帕秋莉称魔理沙为**「老鼠」**（《东方萃梦想》）
- 在《东方三月精》第15话中，帕秋莉严肃地告诉魔理沙**没有借书的打算**

**底层关系——同为魔法研究者的尊重**：
- 尽管存在偷书冲突，两人作为高水平魔法使**相互认可**
- 《东方地灵殿》（TH11）中，帕秋莉作为魔理沙路线的**支援角色（Sub）**出场。这是她唯一一次直接协助魔理沙
- **防水书**：帕秋莉知道魔理沙喜欢在洗澡时看书，专门准备了防水书（东方Wiki记载）
- 魔理沙抄袭（沿用）了帕秋莉的魔法弹幕，命名为恋符「Non-directional Laser」（非定向激光），帕秋莉知情

### 5.2 官方关系分类

- Touhou Wiki 在双方页面上均将对方列为**「Ally（盟友）」**

### 5.3 二创延伸（需与一设区分）

- 帕秋莉 x 魔理沙（帕理沙 CP）——大量二创将两人配对为欢喜冤家/傲娇图书馆员 x 元气小偷
- 爱丽丝加入的修罗场三角关系——纯二创，无一设依据
- "口嫌体正直"式的傲娇互动——基于一设的合理延伸，但程度远超一设范围

## 6. 性格特征

### 6.1 一设中的性格描述

**宅 / 不爱外出**（一设）：
- 「不动的大图书馆」称号直接反映她几乎不离开图书馆
- 《东方萃梦想》角色介绍：「性格不是很积极，经常自行其道」
- 百年如一日与书为伴，离开书便无法生活（《东方求闻史纪》）

**博学 / 知识渊博**（一设）：
- 名字「Knowledge」即知识
- 红魔馆地下图书馆藏书量堪称幻想乡第一，包含大量魔导书和外面世界的书籍（《东方求闻史纪》）
- 创造新魔法后立即写成魔导书，导致书不断增加

**阴沉 / 寡言**（一设）：
- 《东方绯想天》介绍：「寡言少语，有点阴沉——不过只是她不善于与他人交流，思想并不阴暗」
- 「喜欢太阳晒不到的地方」（ZUN对主题曲的评语）

**好奇心强**（一设）：
- 《东方萃梦想》：「好奇心高，对于知识方面的事物会很积极地去获得」
- 《东方虹龙洞》：对异变的卡牌感兴趣，委托咲夜调查

**精明但体力不足**（一设）：
- 《东方萃梦想》：「很精明但力气不足」

### 6.2 关于「傲娇」倾向

**一设判断：官方未使用"傲娇"来描述帕秋莉。**

- ZUN在访谈和设定中**从未**使用「ツンデレ」（傲娇）一词描述帕秋莉
- 她确实有不坦率的一面（对魔理沙的态度），但更准确的一设描述是：**「不善于交流」**（《东方绯想天》）
- 「傲娇」标签是二创的**主要演绎方向**，有一定一设基础（不善表达 + 嘴硬心软），但程度被二创大幅放大

### 6.3 关键台词（官方游戏内文本）

> 「凡物都有属性。所谓的属性，跟弱点是一样的东西。」
> ——《东方求闻史纪》

> 「我们是夜之生物呢。」
> ——《东方三月精》（对蕾米莉亚）

> （被击败时）「むきゅー……」
> ——《东方萃梦想》（"姆Q"的源头）

### 6.4 「幻想乡三大宅女」（二创分类）

- 与蓬莱山辉夜（NEET姬）、芙兰朵露·斯卡雷特并称"三大宅女"
- 此为**粉丝分类**，非官方设定

## 7. 与小恶魔（こあくま / Koakuma）的关系

### 7.1 一设：不明确→逐渐补完

**《东方红魔乡》时期**（设定最模糊）：
- 小恶魔作为《东方红魔乡》四面道中 BOSS 登场（仅有像素图，无立绘、无名、无符卡）
- ZUN 最初**未给予任何公式设定**
- 「小恶魔」本身不是角色本名，而是恶魔种族中实力较弱个体的**统称**（《东方书谱》）

**ZUN 后续补充**（《东方书谱》等访谈）：
- 种族：恶魔（恶魔族中较弱的个体）
- 性格：「与二面BOSS（琪露诺）有很多相似点，喜怒无常且喜欢搞恶作剧，行动起来不顾首尾」
- 住所：红魔馆内（但ZUN**未明确说**她住在图书馆）
- 工作：「杂鱼级角色，基本上有一群。没有被分配什么工作，是一群自由的角色」

### 7.2 官方漫画中的实锤证据

- **《东方三月精》第二部**：魔理沙在图书馆躲避台风时，画面显示**小恶魔站在一旁侍奉**——证实她担任仆从角色
- **《东方儚月抄》四格漫画**：蕾米莉亚在图书馆内建"幻想乡之海"时，小恶魔在"海滩"边摆摊卖御好烧、拉面和荞麦面——确认她常驻图书馆

### 7.3 《东方幻想eclipse》2025年活动剧情

**「现阶段我能做的事 走出洋馆的恶魔」**（2025年4-5月）：
- 小恶魔因误处理图书馆魔法书而自责离开
- 先后前往人类村庄、永远亭、妖怪之山、命莲寺等处寻找新工作，均告失败
- 最后找回魔法书并回归红魔馆
- **剧情明确显示**：她继续在红魔馆图书馆工作，与帕秋莉存在工作上的**上下级关系**

### 7.4 当前一设定位总结

| 维度 | 官方结论 |
|------|---------|
| **关系性质** | 上司与下属（帕秋莉是上司，小恶魔是从者） |
| **工作内容** | 图书馆管理与书籍整理 |
| **常驻地** | 红魔馆大图书馆 |
| **所属** | 帕秋莉个人下属 / 红魔馆成员 |

### 7.5 二创延伸（需区分）

- 「帕秋莉召唤的使魔」→ 无一设依据，小恶魔在红魔馆是自然存在的恶魔
- 「主仆契约 / 魔法契约」→ 无一设依据
- 「姐妹般的关系」或浪漫关系 → 均为二创

## 9. 其他补充设定

### 9.1 名字由来

- **帕秋莉（Patchouli）**：取自香草植物**广藿香**（Patchouli），与寺院土霉气味相关（来源：ZUN《博丽幻想书谱》）
- **诺蕾姬（Knowledge）**：英语"Knowledge"（知识）的片假名音译，直接指向她的身份

### 9.2 「红魔馆的头脑」

- 《东方求闻史纪》记载她是「红魔馆的头脑」，负责解决馆内发生的问题，同时也负责制造问题
- 女仆长咲夜和门卫红美铃在一定程度上听从帕秋莉的命令

### 9.3 与爱丽丝·玛格特罗伊德的关系

- 同为魔法使，且都是擅长使用人偶/精灵的类型（来源：《东方求闻史纪》）
- 一设中与爱丽丝的关系描写较浅，仅作为"同为魔法使"的薄关联
- **二创**中常将两人塑造为好友、竞争对手或帕理沙三角中的一角

### 9.4 与八意永琳的间接交锋

- 《东方儚月抄》中：帕秋莉识破了八意永琳在火箭上做的手脚（Lunar Veil 碎片）
- 两人互相看穿了对方的计划，未直接冲突但存在智斗（一设）

### 9.5 对芙兰朵露的态度

- 红魔乡故事中，帕秋莉曾释放局部雨使地面泥泞，阻止芙兰朵露外出（一设）
- 总体一设关于两人关系的直接描写较少，但在帮助维持芙兰的封印/束缚方面有所涉及

### 9.6 帕秋莉的猫度评价（一设趣味细节）

- 《东方萃梦想》中，帕秋莉用「猫度」评价红魔馆守卫
- 范围 1-96
- 十六夜咲夜的猫度是 24（一设）

### 9.7 名字拼写争议

- 日文官方拼写为「パチュリー・ノーレッジ」（Patchouli Knowledge）
- 有时被误拼为"Patche"或"Patchy"——这些并非官方拼写
