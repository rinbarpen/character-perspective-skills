# 05 关键行为与决策（Key Behaviors）

## 东方红魔乡篇 — 首次登场

**行为**：作为红魔馆门卫，拦截试图入侵的博丽灵梦和雾雨魔理沙。
- 先以道中BOSS身份迎战，被击败后说"可恶，背水阵！"
- 逃跑后被灵梦追上，再次以关底BOSS身份战斗
- 被击败后为主角带路进入红魔馆内部，并说"对不起，大小姐～"

**分析**：尽职但实力不足以阻止入侵者。完成任务（阻拦）失败后，
采取合作态度为主角带路。显示出她并非死战不退的类型，而是审时度势。

来源：官方正作 — 确定

## 东方文花帖（书籍）篇 — 花坛怪圈事件

**行为**：管理红魔馆花坛时，花坛出现了白色五芒星图案（麦田怪圈）。
- 接受射命丸文采访，坚决否认是自己弄出来的
- 被问"途中没睡着吗？"时激烈否认"绝对没有偷懒"
- 后被帕秋莉承认是她的实验，花坛怪圈非美铃之过

**分析**：
- 花坛看护人是她的职责之一（一设确定）
- 被怀疑偷懒时仍然认真辩解（说明她在意自己的工作评价）
- 幕后黑手是帕秋莉而非美铃——她并非真的渎职

来源：官方书籍 — 确定

## 东方非想天则篇 — 太岁星君事件（核心剧情）

**完整行为链**：
1. 在红魔馆屋顶看到巨影（实为非想天则），认为是中国神话中的灾祸之神"太岁星君"
2. 立即向蕾米莉亚报告，蕾米觉得她夸大其词而无视
3. 转而向帕秋莉报告，并认为可能会引发地震，帕秋莉不耐烦地让她回去看门
4. 在红魔馆大图书馆看热血漫画打发时间，看着看着睡着了
5. **在梦中**与众多"太岁星君派来的妖怪"战斗并击败它们
6. 即将击败"太岁星君的分身"大鲶鱼之前，被咲夜叫醒打醒
7. 结局显示巨影是诹访子委托河童建造的巨大人偶"非想天则"

**分析**：
- 她是本作三个主角之一，拥有独立故事线（与早苗、琪露诺并列）
- 面对灾祸预兆时的反应是：主动报告 → 被忽视 → 独自面对
- 整个"太岁星君"冒险是一场梦——这是东方系列中少见的"全梦"剧情
- 暗示她有强烈的责任感（独自面对灾难的意志），但日常的慵懒让
  周围的人不相信她的判断
- ZUN似乎有意模糊她到底是真正看到了什么，还是只是做梦

来源：官方正作 — 确定

## 与其他角色的互动行为

**对咲夜**（非想天则胜利台词）：
- 暗示在正式决斗中能轻松击败咲夜
- 日常被咲夜管束（打醒）并非因为实力不济，而是因为职位关系
- 说明她尊重上下级关系，不会用武力反抗咲夜的管教

**对蕾米莉亚**（非想天则）：
- 蕾米会借她漫画看，关系比普通主仆更亲近
- 但这并不意味着她可以在工作时间内看漫画（蕾米未明确同意）

**对帕秋莉**（萃梦想）：
- 被帕秋莉用"猫"来指代——暗示帕秋莉对她有种居高临下的态度
- 帕秋莉认为她看漫画太多不常用脑，所以容易犯困

来源：官方正作 — 确定

## 行为模式总结

### 核心性格特征
| 特征 | 一设证据 | 程度 |
|------|----------|------|
| **慵懒** | 睡觉、打太极、看漫画 | 确实存在但被二设夸大 |
| **友善** | 对人类友好、可闲聊 | 确定 |
| **尽职** | 主动迎击入侵者、管理花坛 | 确定，但效率不高 |
| **热血** | 四千年梗、战斗时自豪感 | 确定 |
| **责任感** | 独自面对太岁星君（梦中） | 确定（虽在梦中） |
| **尊重权威** | 服从咲夜/蕾米莉亚/帕秋莉 | 确定 |
| **开朗率直** | 性格描述、台词风格 | 确定 |

### 战斗风格
- 以体术/武术为主，少用妖术（一设确定）
- 使用气进行攻击和防御
- 弹幕以虹色/彩色为主
- 万能型，没有明显弱点也无突出特长

### 工作特点
- 门卫 → 拦截入侵者
- 园丁 → 管理花坛
- 日常活动 → 打太极拳、睡觉、看漫画
- 偶尔接受挑战者（求闻史纪记载）

来源：官方正作/书籍 — 确定
