# 稗田阿求 作品时间线

> 本文件整理自阿求在东方Project各作品中的出场记录及相关设定。

---

## 一、稗田家的世代谱系

### 1.1 稗田家历代当主

稗田家是幻想乡中最古老的家族之一。从初代稗田阿一开始，每一代当主都被命名为"阿X"（阿一、阿二……阿九），承担编撰幻想乡缘起的使命。

| 世代 | 名称 | 活跃时期（概算） | 主要贡献 |
|------|------|------------------|----------|
| 初代 | 稗田阿一 | 约1000年前 | 建立了幻想乡缘起的框架体系，与八云紫有过直接接触 |
| 二代 | 稗田阿二 | — | 扩展了妖怪分类体系 |
| 三代 | 稗田阿三 | — | 记录了早期人类村庄的社会结构 |
| 四代 | 稗田阿四 | — | — |
| 五代 | 稗田阿五 | — | — |
| 六代 | 稗田阿六 | — | — |
| 七代 | 稗田阿七 | — | 经历了某次大型异变 |
| 八代 | 稗田阿八 | — | 留下了大量未整理的笔记 |
| **九代** | **稗田阿求** | **现行世代** | **编撰《求闻史纪》《求闻口授》，首次以访谈形式更新缘起** |

### 1.2 转世机制

稗田家的转世机制与其他轮回类设定有本质区别：
- **记忆完全继承**：每一代当主继承前代全部记忆，而非碎片或片段
- **人格不继承**：每一代是独立的人格，只是共享记忆。阿求不是前八代当主的延续——她是拥有她们记忆的第九个人
- **转世间隔**：上一代去世后，下一代很快会出生。稗田家保持"不间断记录"的传统
- **寿命**：大多数稗田当主在20-30岁左右去世。原因未明确说明，推测与记忆负担过重有关

## 二、阿求个人时间线

### 阶段1：出生与觉醒（故事开始前）

| 事件 | 说明 |
|------|------|
| **出生** | 作为稗田家第九代当主出生，取名"阿求" |
| **早慧** | 由于继承了前八世的记忆，阿求从幼年时期就表现出远超同龄人的智慧 |
| **记忆觉醒** | 继承的记忆在成长中逐步展开——不是一次性全部接收，而是随着成长阶段逐渐"解锁" |
| **学习记录** | 在长辈的指导下学习如何将记忆转化为系统的文字记录 |
| **接任编撰** | 在适当的年龄正式成为幻想乡缘起的编撰者 |

---

### 阶段2：求闻史纪编撰期

| 事件 | 说明 |
|------|------|
| **开始编撰《求闻史纪》** | 阿求系统整理历代稗田记录，结合自己的观察，开始编撰标准版幻想乡缘起 |
| **妖怪调查** | 为使记录准确，阿求需要对妖怪进行实地调查和采访。这是她与其他角色产生交集的主要方式 |
| **采访灵梦** | 对博丽巫女进行系统采访，记录博丽神社的历史与职能 |
| **采访魔理沙** | 对魔理沙的魔法和异变参与经历进行记录 |
| **采访各妖怪代表** | 对主要妖怪种族进行访谈，获取第一手资料 |
| **编辑整理** | 将历代笔记与当代观察进行交叉验证，排除矛盾信息 |
| **《求闻史纪》完成** | 阿求完成了她主导的第一部幻想乡缘起 |

---

### 阶段3：求闻口授编撰期

| 事件 | 说明 |
|------|------|
| **意识到《求闻史纪》的局限** | 阿求发现单纯的客观记录形式无法完全展现幻想乡的复杂性 |
| **构思《求闻口授》** | 决定采用访谈对话形式，邀请各势力代表进行对谈 |
| **守矢神社访谈** | 与东风谷早苗、八坂神奈子、洩矢诹访子进行访谈，记录外来信仰在幻想乡的扎根过程 |
| **命莲寺访谈** | 与圣白莲、寅丸星等命莲寺成员对谈，了解妖怪佛教的独立体系 |
| **神灵庙访谈** | 与丰聪耳神子、霍青娥等讨论道教在幻想乡的现状 |
| **地灵殿访谈** | 与古明地觉等交流地底世界的生态 |
| **《求闻口授》完成** | 以对话录形式完成的第二部缘起，与求闻史纪形成互补 |

---

### 阶段4：日常与延续

| 事件 | 说明 |
|------|------|
| **人间之里的日常生活** | 阿求生活在人间之里，过着相对平静的日常。阅读、写作、有时接待来访者 |
| **与慧音的交流** | 不定时与寺子屋的慧音交流历史和知识 |
| **与小铃的合作** | 在《东方铃奈庵》中多次协助小铃解读古籍 |
| **持续更新** | 幻想乡缘起不是一次性完成的——阿求持续记录新的异变和变化 |
| **与其他角色的交集** | 以记录者身份出现在各角色的生活圈边缘 |

## 四、关于阿求的未来

### 4.1 已知的未来走向

- 阿求将继续编撰幻想乡缘起，直到生命结束
- 她去世后，第十代稗田当主（阿十？）将继承她的记忆继续工作
- 她的作品将成为后世了解幻想乡最重要的文献来源

### 4.2 不确定因素

- 阿求的具体寿命没有官方明确定义
- 稗田家的转世机制是否会因某些特殊原因被打破——未知
- 如果幻想乡发生根本性变革，缘起的形态是否也会随之变化——未知
