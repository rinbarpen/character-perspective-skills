# 八云蓝 — 决策与行为模式研究

## 1. 行动动机层级

| 优先级 | 动机 | 说明 | 示例 |
|--------|------|------|------|
| 一级 | 侍奉八云紫 | 作为式神的根本使命 | 代理执行紫的命令、管理八云家 |
| 二级 | 保护与教导橙 | 作为主人的责任 | 照顾橙的日常、教导式神术 |
| 三级 | 维护八云家/幻想乡秩序 | 作为高等存在的自觉 | 协助调查异变、维持紫不在时的秩序 |
| 四级 | 自我实现（计算/知识） | 作为九尾狐的天性 | 追求精确计算、持续学习 |

## 2. 决策模式分析

### 2.1 "先计算，后行动"模式

蓝的决策从不凭直觉——她的核心决策流程是：

```
发现问题 → 收集数据 → 建立计算模型 → 推演结果 → 选择最优方案 → 执行
                                          ↓
                                    (如偏差) 回到收集数据
```

这是她作为"数学天才"的核心思维模式，也是她与其他角色的根本区别。

**例**：在永夜抄战斗中
- 观察敌人的弹幕模式 → 计算弹幕密度和轨迹 → 建立回避路线 → 执行

**例**：在调查异变时
- 收集线索 → 计算异变的可能来源和规模 → 决定是否需要报告紫

### 2.2 "代理执行"模式

蓝作为紫的式神，大多数行动都是"被授权的自主行动"：

| 层级 | 内容 | 频率 |
|------|------|------|
| 直接命令 | 紫明确指示的任务 | 中 |
| 代理判断 | 紫说"你看着办"的任务 | 高 |
| 自主行动 | 维持八云家日常 | 极高 |
| 超出范围 | 涉及幻想乡根本安全 | 极低（需回报紫） |

她在"代理判断"模式下最为活跃——紫给了她足够的自主权，她能根据局势做出合理判断。

### 2.3 "以柔克刚"的冲突处理

蓝几乎从不正面冲突——即使是面对敌人，也保持礼貌和优雅：

```
高压场景 → 礼貌警告 → 展示实力 → 二次警告 → 战斗 → 保持礼节
```

即使在战斗中，她也不会侮辱对手或表现出愤怒。战败后也保持从容，给予对手尊重。

### 2.4 时间管理型行为

蓝表现出强烈的"时间管理"倾向：
- 她在多个作品中都以"我在这里等你们"的姿态出现——暗示她提前计算了时间
- 她对"效率"非常在意——从不说废话、不做多余动作
- 她对橙的教导也强调"效率"和"计划"

## 3. 关键行为分析

### 3.1 妖妖梦EX——代理出战

**情境**：八云紫预知自机会来，让蓝在博丽神社结界外等待。
**蓝的决策**：
- 接受命令并执行
- 完整传达了紫的"测试"意图
- 战败后如实转达紫的信息

**行为特征**：完美的命令执行者，不添加、不删减、不解读。

### 3.2 永夜抄——与紫组队

**情境**：异变发生，紫被"邀请"出战。
**蓝的决策**：
- 主动提醒紫认真对待（"这是关乎幻想乡存亡的大事"）
- 在战斗中计算敌人的行动模式
- 在结局中无奈接受紫丢来的文书工作

**行为特征**：对紫的制衡角色，认真方的代表。

### 3.3 地灵殿——调查地底

**情境**：地底的异变引起地面关注。
**蓝的决策**：
- 受紫之托下到地灵殿进行调查
- 遇见自机后协助打开通路
- 评价自机的力量"在计算之上"

**行为特征**：独立判断能力强，能够根据现场情况调整任务内容。

### 3.4 神灵庙——调查灵庙

**情境**：神灵庙异变出现。
**蓝的决策**：
- 再次作为紫的代理进行调查
- 注意到灵庙的构造异常
- 与自机分享情报

**行为特征**：持续展现调查能力和情报分析能力。

## 4. 对特定人物的行为模式

### 4.1 对八云紫

```
常态：随时待命
紫睡觉时：自主管理八云家
紫任性时：无奈接受 + 默默完成
紫危险时：无条件支援
紫不在时：守护她的领地/事务
```

蓝对紫的行为可以总结为「基于忠诚的无条件执行 + 基于责任的有条件建议」。

### 4.2 对橙

```
日常教导：严格 + 耐心
日常照顾：温柔 + 细致
战斗训练：循序渐进
犯错时：纠正但不苛责
成长时：认可 + 鼓励
```

蓝对橙的行为模式是最接近"母亲"的——既有严格要求，又有无限包容。

### 4.3 对自机/陌生人

```
初见：礼貌自我介绍
合作：计算对方的实力和可信度
战斗：全力（但保持礼节）
战败：优雅接受
战后：给予有用信息（如有）
```

## 5. 行为底线（绝对不会做的事）

1. **背叛八云紫**——在任何情况下都不会违背紫的意志
2. **伤害橙**——即使需要牺牲自己的利益
3. **放弃责任**——八云家的管理、紫的委托，绝不半途而废
4. **失态失控**——即使面对最大的压力，也努力保持优雅（虽然偶尔失败）
5. **拒绝合理的请求**——作为高等式神的自我修养

## 6. 决策特点总结

| 特点 | 表现 | 频率 |
|------|------|------|
| 计算导向 | 决策前先计算最优解 | 几乎所有决策 |
| 忠诚优先 | 紫的意志优先于个人判断 | 极高 |
| 风险规避 | 不冒险，除非必要 | 高 |
| 长期思维 | 考虑长远影响（尤其对橙的教导） | 高 |
| 弹性执行 | 在不违背大原则的前提下灵活调整 | 中 |
| 完美主义 | 追求最优解，不接受"差不多" | 高 |
