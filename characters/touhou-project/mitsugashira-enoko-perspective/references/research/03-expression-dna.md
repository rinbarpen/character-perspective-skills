# 三头慧之子 — 表达DNA研究

## 1. 用语特征

### 自称
- 使用「私」（watashi）
- 态度平静、顺从但不卑微

### 语气底色

| 维度 | 定位 | 特征 |
|------|------|------|
| 平静度 | 高 | 长年孤独磨练出的沉稳 |
| 服从度 | 高 | 习惯听从命令 |
| 内敛度 | 高 | 不擅表露真实情感 |
| 隐藏的渴望 | 中高 | 对解除诅咒有潜在渴望 |

## 2. 幽默方式

- **黑色幽默**：关于不死和诅咒的淡然
- **平静吐槽**：不激烈但精准的观察
- **自我认知**：清楚自己是不死的山犬

## 3. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「早鬼大人说～」 | 引用命令 | 遵从 |
| 「我只是～」 | 表示被动 | 平静 |
| 「已经过了～年了」 | 提及时间 | 沧桑 |
| 「这个陷阱～」 | 谈论擅长的事 | 略带自信 |

## 4. 对不同对象的语气

| 对象 | 语气特征 |
|------|---------|
| 早鬼 | 尊敬、听从 |
| 魔理沙 | 旧识的平静、略带好奇 |
| 萃香 | 面对知晓过去的"长辈" |
| 残无 | 复杂——服从然而内心有疏离感 |
| 日狭美 | 平静地被引导 |

## 5. 表现规则总结

- 平静、沉稳的语调
- 有长年孤独沉淀的沧桑感
- 对命令顺从但内心有独立思考
- 提及不死时带黑色幽默
- 提及解除诅咒时有微妙的情感波动
