# 三头慧之子 — 决策与行为模式研究

## 1. 行动动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 服从命令 | 被诅咒/命令驱动的长期习惯 |
| 二级 | 寻求认可 | 早鬼的称赞让她感到被需要 |
| 三级 | 生存本能 | 作为不死的山犬继续存在 |
| 四级 | 解除诅咒的渴望 | 兽王园事件后向魔理沙求助 |

## 2. 面对危险的反应

- **冷静应对** — 长年累月的生命让她不会轻易慌乱
- **优先服从指挥** — 按上级命令行动
- **寻找退路** — 在被"不需要"时选择新的道路

## 3. 道德边界

- 执行残无的命令吞噬动物灵——但不带有主观恶意
- 被早鬼怀柔后忠诚于劲牙组——直到发现真相
- 对诅咒的态度——渴望解脱

## 4. 核心矛盾

- 不老不死的躯体——内心渴望终结诅咒
- 被残无唤醒"重生"——却是为了被利用
- 表面服从——内心有疏离感

## 5. 角色发展

### 兽王园前（数百年）
普通山犬→吃残无的肉→获得不完全的诅咒→独自隐居魔法森林→等待死亡

### 觉醒期
听到残无的呼唤→吞下宝珠→完全觉醒→被命令吞噬动物灵→遇到早鬼

### 兽王园中
被早鬼招揽→指挥劲牙组地面部队→与魔理沙再会→被萃香告知真相

### 兽王园后
向魔理沙求助解除诅咒→寻找脱离诅咒的方法
