# 三头慧之子 — 作品时间线

## 数百年前

| 事件 | 说明 |
|------|------|
| 作为普通山犬生存 | 生活在幻想乡周边 |
| 遇见残无（人类时期） | 残无还是人类僧侣时 |
| 吃残无的肉 | 撕咬残无的躯体，吃下他的血肉 |
| 背负诅咒 | 因此获得了不完全的不老不死 |
| 隐居魔法森林 | 在魔法森林中独自度过漫长岁月 |

## 觉醒前

| 事件 | 说明 |
|------|------|
| 在魔法森林中衰老 | 不完全诅咒只延长了几百年寿命 |
| 即将自然死亡 | 作为垂老的野兽等待生命终结 |
| 目击幼年魔理沙 | 看到魔理沙哭着逃进魔法森林 |

## 觉醒——兽王园前

| 事件 | 说明 |
|------|------|
| 听到残无的心灵呼唤 | 残无通过心灵感应唤醒了她 |
| 找到宝珠并吞下 | 在魔法森林中找到宝珠，清除污秽 |
| 成为完整的不死之身 | 真正的永恒诅咒完成 |
| 残无下令吞噬动物灵 | 开始攻击地面上的动物灵 |
| 遇到骊驹早鬼 | 被早鬼称赞为"刻耳柏洛斯"并招揽 |
| 加入劲牙组 | 成为早鬼的部下 |

## 东方兽王园（TH19，2023年5月）

| 事件 | 说明 |
|------|------|
| 作为Stage 3 Boss登场 | 在魔法森林/地上区域作为劲牙组将领出现 |
| 与魔理沙交战 | 与旧识再会 |
| 与萃香交战 | 被萃香告知残无不再需要她→"松了口气" |
| 被日狭美引导 | 日狭美将她送往地狱见残无 |
| 作为自机可用 | 通关后可选 |

## 兽王园之后

| 事件 | 说明 |
|------|------|
| 向魔理沙求助 | 请魔理沙帮助解除诅咒 |

## 关键节点总结

- 兽王园（2023）：首次登场，Stage 3 Boss兼自机
- 核心主题：诅咒的不死、孤独、服从与解脱的渴望
