# 姬虫百百世 — 决策与行为模式研究

## 1. 行为动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 食欲/食物 | 龙珠、盗采者、龙——一切围绕食欲 |
| 二级 | 挖矿的满足感 | 每天辛辛苦苦挖龙珠 |
| 三级 | 与龙的约定 | 答应了的事情会做到 |
| 四级 | 战斗的快感 | 享受与强者对决 |

## 2. 决策逻辑

### 为什么不直接吃龙
- 龙珠出现了→龙珠比龙好吃→选择龙珠
- 简单直接的食物链比较——哪边更好吃就选哪边

### 为什么接受合作
- 挖龙珠本来就在做了
- 接受合作还能继续挖
- 报酬是龙珠——本来就是她想要的

### 对待入侵者的态度
- 龙珠吃腻了→期待新口味→盗采者是点心
- 战斗欲+食欲的双重满足

## 3. 面对冲突的反应

- **兴奋迎接**：等待盗采者出现已经很久了
- **全力以赴**：EX面Boss的实力不是开玩笑的
- **败北认可**：「有两下子嘛」——对强者的尊重
- **不改变行为**：被击败了也不会放弃挖龙珠

## 4. 道德边界

### 底线
- 答应了的事会做（和龙的约定）
- 认朋友（龙是亲友）
- 败北了不纠缠

### 灰色地带
- 吃人（想——但未必真的做过）
- 吃朋友（想过——但有更好的食物所以没做）
- 独占矿山资源（这是我的地盘）

## 5. 与其他角色对比

- 相比千亦的深思熟虑和典的精妙算计，百百世的决策模式异常简单
- 她不是笨蛋——而是不需要复杂的逻辑
- 强者的特权：足够强所以可以简单
