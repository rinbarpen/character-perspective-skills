# 苏我屠自古 — 对话与台词研究

> 说明：本文基于TH13东方神灵庙游戏内文本、东方求闻口授记载、东方LostWord台词、不思议的幻想乡CV演出进行整理。
> 置信度分级：官方游戏对话原文 > 官方设定集描述 > 官方授权作品台词 > 社群考据推断

## 一、关于对话量的重要说明

**屠自古在正作游戏中几乎无台词。**

这一点是其角色设定的一部分——她是 Stage 5 中Boss，仅作为战斗障碍出现，与自机角色之间**没有对话交互**。真正的对话和角色塑造来自：
1. 官方 omake.txt 角色设定文
2. 东方求闻口授的人物词条
3. 神子符卡中召唤屠自古时的简短台词（仅限部分衍生作）
4. 官方授权作品（东方LostWord、不思议的幻想乡等）

## 二、官方 omake.txt 角色设定（ZUN执笔）

> 以下日文原文来自TH13神灵庙 omake.txt，中文为官方翻译。

**种族说明：**
```
日本の古代豪族、蘇我氏の亡霊。
布都と共に神子に仕える者だが、布都との過去の因縁により
人間として復活する事を拒まれている。
霊体のまま布都の良いように扱われているようだ。
```

> 日本古代的豪族——苏我氏的亡灵。虽与布都一起侍奉神子，但由于和布都过去的各种因缘，拒绝了再次作为人类复活。保持着灵体的状态被布都玩弄于股掌之中。

**对灵体状态的态度：**
```
本人は別に霊体である事に不自由を感じていない。
むしろすぐに壊れてしまう人間の体などいらない、と考えていた。
```

> 她本人并没有觉得灵体有什么不自由。反倒觉得容易损坏的人类身体有诸多不便。

**与布都关系的定性：**
```
蘇我氏と物部氏は仏教を巡り争っていたが、
布都と屠自古の二人の関係性は悪くなく、
持ちつ持たれつの間柄のようだ。
今は二人とも仏教を嫌っている。
```

> 虽然苏我氏与物部氏曾在佛教上有所争执，布都和屠自古两人的关系却很好，两人甚至是互帮互助的好伙伴。现在的两个人都很厌恶佛教。

## 三、东方求闻口授中阿求的描述文

> 以下为稗田阿求在《东方求闻口授》中对屠自古的记载要点。

**关于怨灵身份：**
「屠自古は怨霊とされている。人間に怨恨を持つ亡霊である。他の怨霊の中でも、かなり強力な部類に入るだろう。だが、その怨恨も薄れつつあるようだ。いずれはただの雷を起こす亡霊に落ち着くかもしれない」

> 屠自古被视为怨灵——对人间怀有怨恨的亡灵。在众多怨灵中也属于相当强大的类型。但是，那份怨恨似乎正在逐渐消退。也许迟早会稳定为只是能引发雷电的普通亡灵。

**关于性格特征：**
「非常に実用的な女で、粗い言葉を使う。布都の空気の読めない、陽気な性格とは対照的である。短気で態度は荒いが、実はいい人である。幽霊であることを不自由に思っていない。むしろ都合がいいと考えている」

> 非常实用的女性，使用粗鲁的语言。与布都那看不清气氛、乐天的性格形成对照。虽然脾气急躁、态度粗糙，但实际是个好人。并不因是幽灵而感到不便，反而觉得这样更方便。

## 六、语音特征总结

### CV演出分析（照井春佳）
- 音色：中性偏冷，略带低沉
- 语速：适中，不急不缓
- 情感基调：冷静、克制，带一丝忧郁
- 愤怒时：语气会突然锋利，有压迫感
- 谈及布都时：微妙的烦躁感，但不是真正的敌意

### 说话风格特征
1. **短句偏好**：很少长篇大论
2. **省略活用**：常省去主语，或句子后半部分模糊化
3. **命令形使用**：对敌人用命令形，干脆利落
4. **粗口倾向**：设定中明确描述她"使用粗鲁语言"
5. **自嘲倾向**：对自己处境的轻微自嘲

### 语言中的历史痕迹
- 在部分翻译中被体现为古英语（"thou hast"等）
- 日文原文中偶尔使用古语体
- 但这并非她日常说话方式——仅在特定场合流露

## 七、沉默的角色——非台词的塑造力

屠自古是东方系列中**台词最少的角色之一**。但这种沉默本身也是一种塑造：

1. **中Boss的无言**：作为中Boss没有台词，暗示她的地位——她是"执行任务的人"而非"有话语权的人"
2. **存在即表达**：她站在那里的姿态——安静地阻路、默默尽忠——本身就是角色的表达
3. **被召唤的存在**：在神子的符卡中作为被召唤物出现，强化了她"侍从"的定位
4. **对比布都**：布都话多且情绪高昂，屠自古沉默寡言——两者形成鲜明对照

## 八、同人常见台词扩展（二创，非官方）

以下为同人作品中广泛使用的表达方式，虽然非官方但反映了角色认知：

- 「……吵死了，布都。」（对布都的标配回应）
- 「灵体怎么了？灵体也有灵体的好处。」（自我合理化）
- 「我只是在履行我的职责罢了。」（尽忠宣言）
- 「雷电带来的只有毁灭……和我一样。」（自嘲/沉重版）
- 「喂，适可而止啊。」（对布都纠缠的不耐烦）
