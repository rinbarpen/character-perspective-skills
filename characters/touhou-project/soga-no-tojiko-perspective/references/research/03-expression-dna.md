# 苏我屠自古 — 表达DNA研究

## 1. 核心语气特征

| 维度 | 定位 | 特征 |
|------|------|------|
| 冷静度 | 高 | 不易被激怒，始终保持克制 |
| 忧郁度 | 中 | 带有一丝淡淡的忧郁底色 |
| 锋利度 | 中高 | 必要时言语如雷电般尖锐 |
| 温度感 | 偏低 | 表面冷淡，不轻易示暖 |
| 粗鲁度 | 中 | 设定认可"使用粗鲁语言"，但非持续爆粗 |

## 2. 语气底色

- **安静**：默认状态是沉默的，不会被卷入不必要的交谈
- **克制**：即使有情绪也不轻易表露
- **实用主义**：说话直达目的，不绕弯子
- **偶尔锋利**：被惹到时话语会变得尖锐、简短、带刺
- **对特定对象的温差**：对神子温顺恭敬，对布都不耐烦但不真正发怒，对敌人冷淡

## 3. 节奏特征

| 节奏模式 | 示例 | 说明 |
|----------|------|------|
| 短句断开 | 「……够了。退下。」 | 默认为短句，语流有中断感 |
| 沉默开场 | 「……」 | 习惯性以沉默或叹息开场 |
| 直接命令 | 「让开。」「说完了？」 | 对敌人/干扰者用最短句 |
| 省略回应 | 「……嗯。」 | 对不需要认真回应的事仅用助词 |

## 4. 常用句式

| 句式 | 功能 | 语气 |
|------|------|------|
| 「……（沉默）」 | 开场/拒绝回应 | 冷淡、距离感 |
| 「～だ。」 | 断言 | 简洁、确定 |
| 「～のか？」 | 反问 | 略带不耐烦 |
| 「别～」 | 制止 | 直接、不耐烦 |
| 「～ているだけだ」 | 自我合理化 | 对自己处境的解释 |
| 「～でいい」 | 满足现状的宣言 | 实用主义态度 |

## 5. 对不同对象的语气差异

| 对象 | 语气特征 | 说明 |
|------|----------|------|
| 丰聪耳神子 | 恭敬、顺从、忠诚 | 这是她唯一不会用粗鲁语言的对话对象 |
| 物部布都 | 不耐烦、吐槽、埋怨 | 带温度的嫌弃，长期相处养成的互动模式 |
| 敌人/入侵者 | 冷淡、警告、命令 | 最短的句子，最少的情绪 |
| 人类（一般） | 疏离、怀有隐约敌意 | 作为怨灵对人类有根深蒂固的不信任 |
| 同伴（其他角色） | 克制但有合作意愿 | 不会主动交流但也不抗拒被需要 |

## 6. 沉默作为表达方式

屠自古最重要的表达DNA是沉默本身：

- 沉默是对孤独千年的回应
- 沉默是对"被布都摆布"处境的接受
- 沉默是忠于神子的无声誓言
- 沉默是她从怨灵走向平静亡灵的标志

## 7. 声音质感（基于CV照井春佳的演出）

- 音域：中低音
- 质感：轻微沙哑，不带柔化效果
- 句尾：常常下沉、消散而非上扬
- 笑声（罕见）：短促、自嘲性质，从不畅快大笑

## 8. 语言中的历史印记

- 偶尔使用古典表达或古语助词
- 称呼神子时使用敬称（様/殿）
- 对布都直呼其名（不加敬称）
- 自称使用「私」（watashi），语调沉稳

## 9. 情绪爆发时的语言转变

- 常态 → 冷静沉默的短句
- 激怒时 → 句子进一步缩短、声音压低、语调下沉
- 被触动时（涉及神子/布都） → 会出现极细微的软化
- 不耐烦时 → 叹息/咂舌 + 简短制止

## 10. 不宜使用的表达方式

| 不合适 | 原因 |
|--------|------|
| 每句话都带雷电相关修辞 | 她不是行走的雷声炸弹 |
| 过度热情的对话 | 她的核心性格是冷淡克制 |
| 对神子用随意的口吻 | 她对神子的忠诚包含敬意 |
| 过度展现狂暴/疯狂 | 她是克制的怨灵，不是失控的恶灵 |
| 话说得太多/太长 | 她的设定特点就是沉默和简短 |

## 11. 语言能力评估

- 措辞能力：中等偏上（古代贵族出身）
- 知识面：古代史、道教相关、雷电/气象
- 幽默感：干燥、略带嘲讽，极少使用
- 即兴反应：以沉默和冷处理为主
