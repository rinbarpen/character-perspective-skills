# 苏我屠自古 — 时间线与角色弧光

> 说明：屠自古是历史人物原型角色，年份标注基于真实日本史和东方Project设定的结合。

## 飞鸟时代（生前，约6-7世纪）

| 时间 | 事件 |
|------|------|
| 生年不详 | 出生于苏我氏家族。苏我氏是当时日本最有权势的豪族之一，掌握财政和外交实权 |
| 约587年 | 苏我氏与物部氏的宗教战争激化。物部守屋被讨伐，物部氏衰亡 |
| 约592-622年 | 苏我马子-圣德太子（丰聪耳神子的历史原型）执政时期。屠自古与布都共同侍奉神子 |
| 约622年 | 神子去世（历史中圣德太子去世）。但实际按东方设定进入尸解仙的沉睡 |
| 约645年 | 乙巳之变。中大兄皇子和中臣镰足推翻苏我氏，苏我入鹿被杀。苏我氏事实上灭亡 |
| 此后 | 传说中臣镰足后被雷击而死——被认为是苏我氏的诅咒 |

## 尸解仙计划与失败

| 时间 | 事件 |
|------|------|
| 神子生前 | 神子与青娥策划尸解仙复活方案 |
| 计划阶段 | 布都自愿成为实验品，先一步沉眠 |
| 屠自古的尝试 | 屠自古试图通过同样的方式获得不死身体 |
| 失败时刻 | 布都阻挠——具体手法不明，但屠自古未能获得人类身体 |
| 结果 | 屠自古以亡灵形态存在，继续守护 |

关键点：
- 布都的阻挠并非出于恶意（关系并没变坏）
- 更像是"你就以这个形态继续存在吧"的实用判断
- 屠自古本人也接受了这个结果

## 角色弧光分析

### 弧线：从怨灵到普通亡灵

屠自古的角色弧光极其微妙——**她的角色发展不是变得更强大或更显眼，而是变得更"普通"**。

1. **生前**：大氏族成员，政治斗争的参与者
2. **死亡后初期**：怨灵——对未能复活为人的结果充满怨恨
3. **漫长的千年**：怨恨在时间中逐渐消磨
4. **求闻口授记载时**：怨恨已显著消退，趋于平静
5. **如今**：正在变成一个"只是能引发雷电的亡灵"

这是一个**消解型的角色弧光**——角色的成长不是"变得更强"，而是"变得更加释然"。

### 弧线：沉默中的存在确立

- 首次登场（神灵庙）：没有台词的中Boss
- 求闻口授：有了人物设定
- 衍生作品（LostWord等）：有了台词、互动、性格展现
- 定位演变：从"无台词障碍"→"有设定的配角"→"可玩的角色"

### 与其他角色的关系变化

| 关系 | 变化 |
|------|------|
| 对神子 | 生前→至今：始终如一的忠诚，从未动摇 |
| 对布都 | 生前（政治对手的关系）→ 千年相处（互帮互助的搭档） |
| 对人类 | 生前（本族为人类统治集团）→ 死后怨灵（对人类的怨恨）→ 现在（怨恨消退，但保持距离） |

## 状态总结（截至最新设定）

- **种族**：亡灵（原怨灵，正过渡为普通亡灵）
- **归属**：丰聪耳神子麾下
- **主要活动**：守护梦殿大祀廟，侍奉神子
- **与布都的关系**：互损但可靠的长期搭档
- **对自身状态的态度**：接受，不觉得不便
- **感情状态**：怨恨正在消退，取而代之的是平静的忠诚
