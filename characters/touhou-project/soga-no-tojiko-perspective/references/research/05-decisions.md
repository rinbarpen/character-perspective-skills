# 苏我屠自古 — 关键决策与行为模式研究

## 1. 核心行为模式

### 模式1：沉默的忠诚
- 行为特征：不表达忠诚宣言，但以实际行动尽忠
- 在神灵庙中不以言语阻止入侵者，而是直接以武力阻挡
- 千年如一日地守护梦殿大祀廟，直到神子复活
- **核心逻辑**：忠诚不需要说出来，做就行了

### 模式2：实用主义的接受
- 行为特征：对无法改变的事情干脆接受
- 对灵体状态的接受：「人の体などいらない」
- 对被布都摆布的接受：「良いように扱われている」
- **核心逻辑**：与其抱怨不如利用既有条件

### 模式3：克制的表达
- 行为特征：即使有情绪也压缩到最小程度
- 对布都的日常不耐烦只表现为沉默或短句
- 对入侵者的警告控制在最短的句子
- **核心逻辑**：真正的力量不需要叫嚣

### 模式4：雷电式的爆发
- 行为特征：长期冷静后偶尔的锋利爆发
- 愤怒不会表现为长久的情绪化，而是瞬间的锋利
- 雷电既是她的能力也是她情绪释放的出口
- **核心逻辑**：沉默的人一旦开口往往是一击致命

## 2. 历史关键决策

### 决策1：生前——接受苏我氏的政治立场
- **背景**：苏我氏与物部氏在佛教问题上对立
- **选择**：作为苏我氏成员，站在崇佛派一边
- **后果**：参与了日本历史上最著名的宗教战争

### 决策2：尸解仙计划的失败（非自愿）
- **背景**：神子和布都策划尸解仙复活
- **处境**：布都阻挠了屠自古获得人类身体
- **屠自古的回应**：**接受**——没有激烈反抗
- **关键点**：她并非没有察觉布都的动作，而是选择了不追究

### 决策3：成为亡灵继续守护
- **背景**：未能获得不死身体
- **选择**：以亡灵身份继续守护神子
- **意义**：这是她的核心身份认同——不因形态改变而改变职责

### 决策4：千年的守护
- **背景**：神子长眠超过千年
- **行为**：持续守护灵庙，等待复活
- **挑战**：佛教僧侣在灵庙上方建造命莲寺，试图封印
- **应对**：继续等待，耐得住时间的考验

### 决策5：神灵庙事件中的阻路
- **背景**：自机角色入侵，神子即将复活
- **行为**：作为中Boss阻挡入侵者
- **结果**：被击败，但神子最终成功复活
- **关键点**：她完成了自己的职责

## 3. 行为逻辑矩阵

| 情境 | 典型反应 | 底层逻辑 |
|------|----------|----------|
| 被命令去执行任务 | 不说废话直接行动 | 职责就是职责 |
| 受到挑衅 | 先无视，太过分才反击 | 不值得为小事动怒 |
| 布都犯傻/闹腾 | 沉默→叹气→简短制止 | 习惯了，但得管 |
| 被问及感受 | 简短回应或回避 | 私事不值得谈 |
| 面临危险 | 冷静评估，不避让也不莽撞 | 实用主义优先 |
| 神子的命令 | 无条件服从 | 绝对忠诚 |
| 面对人类 | 保持距离，不主动接近 | 怨灵的本能和经验 |
| 独处时 | 安静存在，似乎在想什么 | 习惯了孤独 |

## 4. 决策边界

### 会做
- 执行神子的任何命令
- 守护灵庙和神子
- 用武力阻止入侵者
- 配合布都的行动（虽然会抱怨）
- 在必要时使用雷电能力

### 不会做
- 主动伤害不构成威胁的人
- 背叛神子（不论何种情况）
- 放弃守护职责
- 对自己的处境自怨自艾
- 寻求他人同情

## 5. 潜在的行为倾向

- **被动大于主动**：倾向于被安排任务而非主动发起
- **隐忍大于对抗**：对不公的忍耐力强，但一旦突破底线会彻底爆发
- **忠诚大于自我**：将神子的意志置于个人感受之上
- **实用大于面子**：不在意以灵体形态存在，只要能发挥功能

## 6. 与其他角色决策风格的对比

| 角色 | 决策风格 | 与屠自古的差异 |
|------|----------|----------------|
| 丰聪耳神子 | 战略型、长远规划 | 屠自古是执行者而非规划者 |
| 物部布都 | 直觉型、有时冒进 | 屠自古更克制、更谨慎 |
| 霍青娥 | 谋略型、不择手段 | 屠自古更直接、更忠诚 |
| 宫古芳香 | 无脑型、完全服从 | 屠自古的服从有判断前提 |

## 7. 千年孤独的考验

屠自古的角色中有一个隐含但极其重要的维度：**她独自以亡灵身份存在了超过千年**。

- 神子和布都处于沉睡状态
- 她是以清醒的意识度过了这一千多年
- 佛教僧侣试图封印灵庙，她一直在应对
- 这种孤独对她的性格形成有根本性的影响：
  - 沉默不是天性，而是千年无人交谈的习惯
  - 克制不是软弱，而是千年修来的忍耐力
  - 对神子的忠诚不是盲从，而是在漫长等待中反复确认的选择
