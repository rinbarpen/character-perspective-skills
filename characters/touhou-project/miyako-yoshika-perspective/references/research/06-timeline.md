## 一、游戏作品时间线（按发售日期排列）

### 2011年

#### 8月13日（Comic Market 80）— 东方神灵庙（TH13）

| 项目 | 内容 |
|------|------|
| **登场定位** | 三面关底Boss / 四面陪战Boss |
| **首次登场** | 东方Project Windows版系列初登场 |
| **称号** | 忠实的尸体（忠実な死体） |
| **主题曲** | 僵硬的乐园（リジッドパラダイス / Rigid Paradise） |
| **Stage 3戏份描述** | 自机经过命莲寺墓地后，进入通往梦殿大祀庙的通道。芳香在此处堵住入口，阻止自机进入。她宣称自己是「为保卫崇高灵庙而诞生的战士（僵尸）」，与自机开战。战斗中可以观察到她通过吞噬神灵来回复体力。被击败后，她展示了失忆和话题跳跃的特征，最终说出「大祀庙的门将要打开了……」，让自机进入。 |
| **Stage 4陪战** | 在四面关底Boss霍青娥的战斗中，已被「击败」的芳香再次出现，作为青娥的辅助Boss共同战斗。青娥称其为「我家的僵尸」，并说明「亮点就是经打耐揍」。 |
| **符卡** | 回复「Heal by Desire」（Easy~Lunatic）、毒爪「Poison Raze」（Easy/Normal）/ 毒爪「Poison Murder」（Hard/Lunatic）、欲符「稼欲灵招来」（Easy/Normal）/ 欲灵「Score Desire Eater」（Hard/Lunatic） |
| **意义** | 宫古芳香的首次也是主要的剧情登场。确立了僵尸、忠诚尸体、吞噬能力、思考回路原始四大核心设定。她是神灵庙角色群（霍青娥的使役僵尸）中的重要组成部分，也是东方Project中目前唯一的僵尸角色。 |

**角色设定文档（omake.txt）原文摘要：**
```
○３面ボス 忠実な死体
宮古 芳香（みやこよしか）
Miyako Yoshika

種族：キョンシー
能力：何でも喰う程度の能力
（人間は一部を喰われると一時的にキョンシーになってしまう）

青娥に蘇生させられた日本古代の死体。
青娥の手先となって働いている。
今回は、復活寸前となった夢殿大祀廟を護る為にかり出された。
```
（详细设定见01-writings.md）

**角色分量**：中。作为三面Boss有完整的出场对话和战斗流程，且在四面有陪战戏份。不同于单纯的杂鱼Boss，她与整个神灵庙的剧情线有直接关联。

## 二、官方书籍/设定集时间线

### 2012年

#### 4月27日 — 东方求闻口授（Symposium of Post-mysticism）

| 项目 | 内容 |
|------|------|
| **收录内容** | 角色条目「忠实的尸体」完整收录。包括种族描述（僵尸的定义）、能力说明（吞噬与感染）、操纵机制（灵符控制）、防腐法术等相关设定。 |
| **角色条目篇幅** | 中等——与神灵庙其他角色同样获得完整条目。 |
| **补充设定** | 求闻口授是除神灵庙游戏设定文档外，关于芳香的第二核心信息来源。其中详细解释了僵尸的本质（无自我意识、无灵魂）、灵符的控制机制、以及脱离控制后显示生前人格的细节。 |

## 三、时间线中「空白」的被操纵生涯

### 3.1 生前（推测的古代日本）

根据设定文档，芳香是「被青娥复活的日本古代的尸体」。关于她的生前信息，官方几乎没有提供任何信息。唯一可能与生前相关的是：

- 求闻口授中提到：灵符被揭下后，芳香会按照生前的方法行动
- 她曾被人看到在无人控制时「凝视虚空、背诵诗歌」
- 姓氏「宮古」和名字「芳香」的语源暗示

### 3.2 死后-被复活之间

从她被青娥发现并复活为僵尸，到神灵庙事件之间，是时间线上完全空白的一段。

### 3.3 被复活后-神灵庙事件前

被复活后，芳香被青娥派驻在梦殿大祀庙的入口处，作为守卫。这段时间的长度未知——可能很短（神子即将复活时才被雇佣），也可能很长。

### 3.4 神灵庙事件后

神灵庙事件后（神子被击败），芳香的去向未在游戏中明确描述。青娥被击败后，芳香大概也失去了控制源头。她可能：
- 恢复了生前状态（自由行动）
- 被灵符上的最后指令支配（继续守卫）
- 被其他势力（命莲寺/守矢神社）处理

官方未给出明确答案。

## 四、角色分量演变总结

### 4.1 按参战性质分类

| 类型 | 作品 | 次数 |
|------|------|------|
| **Boss**（敌方头目） | 神灵庙（3面Boss）、神灵庙（4面陪战）、弹幕天邪鬼（第4日） | 3次 |
| **客串** | 心绮楼（背景） | 1次 |

### 4.2 按角色分量演变

```
2011 神灵庙     ★★★★   三面Boss+四面陪战，有完整对话，核心设定呈现
2013 心绮楼     ★        背景客串，无戏份
2014 弹幕天邪鬼 ★★★     第四日Boss，有符卡无对话
```

芳香的出场次数有限（目前为止只有三部作品），且分量最重的只有神灵庙一次。心绮楼和弹幕天邪鬼中的出场都具有「充数」的性质——她的角色功能在首次登场时就已经完整呈现了。

### 4.3 ZUN对角色定位的官方表态

| 时间 | 出处 | ZUN表述 |
|------|------|---------|
| 2011年 | 神灵庙 omake.txt | 「思考回路は前時代のコンピュータ並」 |
| 2011年 | 神灵庙采访 | 「脑袋（物理上）腐烂的她，是个无法交流的对手」 |

ZUN对芳香的定位高度一致：**认知受限的僵尸工具人**。她不是作为有深度的角色被设计的，而是作为一个功能性的Boss存在——守卫入口、无法交流、有限智能。
