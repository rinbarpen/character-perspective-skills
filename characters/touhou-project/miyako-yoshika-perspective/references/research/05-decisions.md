# 宫古芳香 — 关键行为与决策模式研究

> 说明：本文基于东方神灵庙（TH13）游戏内文本、官方设定文档、东方求闻口授进行整理。
> 置信度分级：官方游戏对话原文 > 官方设定集描述 > ZUN访谈 > 社群考据推断
> 调研范围：东方神灵庙（TH13）、东方心绮楼（TH13.5）、弹幕天邪鬼（TH14.3）、东方求闻口授

---

## 一、僵尸行为逻辑的基础框架

### 1.1 幻想乡中「僵尸」的本质

在东方Project的世界观中，僵尸（キョンシー/丧尸）是一种特殊的亡灵存在。与幽灵、妖怪不同，僵尸具有以下本质特征：

**1. 物理性的存在**
- 僵尸是**尸体被复活**的结果，具有物理实体
- 她们的身体是死者生前的肉体，被注入了活动的能力
- 与幽灵的不同在于：幽灵无实体，僵尸有实体

**2. 无自我意识**
- 求闻口授明确指出：僵尸没有自己的意识
- 她们只能按照操纵者的指令行动
- 本质上与傀儡、人偶、机器人类似——「除了由尸体制作这一点之外」

**3. 无灵魂**
- 一般来讲，此类的存在没有灵魂
- 因此无法进行灵性攻击
- 这使得芳香的战斗方式完全依赖物理能力和吞噬能力

**4. 非生物性的存在**
- 不感到疼痛
- 不需要睡眠（但她会「眯一觉」）
- 不需要食物（但她的能力是吞噬）
- 没有疲劳感

### 1.2 僵尸行为的核心逻辑

从上述本质出发，芳香的行为逻辑可以归纳为：

1. **指令驱动**：她的所有行为都来自灵符上的指令（青娥的待办事项列表）
2. **程序化执行**：她的行为模式固定、可预测、缺乏灵活性
3. **无自主决策**：她不做出自己的选择，只执行被赋予的任务
4. **缺乏自我维护本能**：她不会因疼痛或危险而退缩
5. **任务专注但容易丢失**：她对自己的使命有模糊的执着，但记忆碎片化导致任务执行经常脱轨

### 1.3 芳香与其他「被操纵存在」的对比

| 对比项 | 宫古芳香 | 河城荷取（操纵人偶） | 爱丽丝（人偶使） |
|--------|---------|-------------------|-----------------|
| 本体 | 被复活的尸体 | 河童 | 魔法使 |
| 自我意识 | 无 | 完整 | 完整 |
| 控制方式 | 灵符命令 | 自由意志 | 自由意志 |
| 执行方式 | 死板、预定模式 | 灵活 | 灵活 |
| 能否恢复自我 | 揭下灵符后可以恢复生前模式 | 不适用 | 不适用 |

## 二、神灵庙中的关键行为

### 2.1 初次登场：作为大祀庙的守卫

**情境**：自机（灵梦/魔理沙/早苗/妖梦）经过命莲寺墓地，闯入通往梦殿大祀庙的通道。

**行为**：
- 芳香在通道入口堵住去路
- 用拉长音开场宣告自己的存在：「不——许——靠——近——！」
- 宣称自己是为保护崇高灵庙而诞生的「战士」（僵尸）
- 在战斗中通过吞噬神灵来回复体力

**动机分析**：
她的动机表面上是「保护大祀庙」，但深层来看，她可能并不理解大祀庙是什么、为什么要保护。她只是在执行被写入的指令：**守住这个入口，不让任何人通过。**

### 2.2 战斗中的回复行为

她的战斗行为有一个非常独特的机制：**受伤后会吞噬神灵来回复体力**。

这一行为揭示了一条重要信息：她的行动模式是循环式的——受伤→吞噬→回复→继续战斗。这不是策略性的决策，而是内置的「自保程序」。和真正的生物不同，她的回复行为不是出于对死亡的恐惧（她没有），而是出于程序化的「受伤时需要修复」的设定。

### 2.3 被击败后的反应

芳香在被击败后的反应是研究她的决策模式时最有价值的数据：

**灵梦线被击败后的流程：**
1. 夸张惨叫：「呜哦～！被～打～败～啦～！」
2. 被质问后：「呜，呜……呜，哦……」
3. 宣布不朽：「丧尸是永远不灭的！」
4. 失忆：「你到底是谁啊？」
5. 任务完成宣言：「终于……大祀庙的门将要打开了……」

**妖梦线被击败后的流程：**
1. 夸张惨叫：「怎—么—会—！要—死—了！」
2. 被指出有操纵者：「是吗？」
3. 跑题到皮肤保养：「别看我这样，我还是有好好保养皮肤的——」
4. 跳跃到宏大叙事：「难道说，那场悲剧的宗教战争又要再一次上演了吗？」
5. 失忆：「我主人是谁来着？」

**行为模式的关键发现：**
- **失忆触发机制**：被击败后触发记忆重置。她不再记得刚刚的战斗对象。
- **任务完成感**：虽然她被击败了，她反而有种「终于……」的释然感，仿佛被击败本身也是指令的一部分。
- **注意力溃散**：被质问时会跳跃到各种无关话题，似乎是在尝试搜索记忆系统中的相关内容，但总是抓到错误的数据。

### 2.4 四面陪战：被再次召唤

在Stage 4，青娥与自机战斗时，已经「被击败」的芳香再次出现，作为青娥的陪战Boss登场。

青娥将其称为「我家的僵尸」，并用「亮点就是经打耐揍」来形容。

这一行为揭示了芳香的另一个特征：**永久性的回复能力**——即使被打败，只要灵符还在、操纵者还在，她就能无限次地恢复和再战。这使她在战术上难以被彻底消灭——只能通过击败操纵者（青娥）来结束她的战斗。

### 2.5 弹幕天邪鬼：作为被挑战对象

在弹幕天邪鬼中，芳香作为第四日Boss出场。鬼人正邪挑战她时，使用的是毒爪系列的符卡。

这一出场进一步确认了她被使用的模式：
- 她不是异变的主动参与者
- 她只是在「她所在的位置」等待被挑战
- 她的符卡以毒/感染/回复为主题

## 三、行为的局限性分析

### 3.1 认知局限对行为的影响

**1. 对象永久失忆**
芳香在灵梦线被击败后直接问「你到底是谁啊？」——她与灵梦战斗了数分钟，但战斗结束后完全不认识对方了。这种失忆不是选择性遗忘，而是系统性的记忆清除——她的大脑（物理上腐烂了）不具备形成长期记忆的能力。

**2. 概念理解障碍**
她无法理解「神灵」这样的抽象概念，将其简化为「好吃吗？」。这种将抽象概念具象化、物质化的认知方式是认知局限的直接结果。

**3. 关联推理能力缺失**
妖梦线的对话最能说明问题：
```
妖梦：你的符说明有人在操纵你。
芳香：是吗？（理解了这个信息，但无法推理）
妖梦：我嗅到了操纵者的味道。
芳香：哎？嗅出味道？（误解了「味道」）
别看我这样，我还是有好好保养皮肤的——（跳跃到完全无关的话题）
```
她无法将「符→操纵者」这一因果关系进行推理，也无法正确解读「味道」这一隐喻——她将其理解成了字面的体味。

### 3.2 情绪表达的特征

芳香的另一个行为特征是：**情绪表达和真实体验的分离**。

她抱怨「被～打～败～啦～！」时语气夸张，但她真的感到沮丧吗？一分钟后她就心情大变。她的情绪表达更像是**预设的语音包**——在特定场景下触发特定的声音，而不是内在感受的外化。

这在她身上体现在：
- 被击败时惨叫 → 但马上就不在意了
- 被夸奖时心情好 → 这种心情好的持续时间极短
- 怒斥入侵者 → 但轻易放人过去
- 宣称要守护 → 但不知道守护的是什么

### 3.3 命令执行中的偏差

虽然芳香按照灵符的指令行动，但她的执行过程中存在明显的偏差：

1. **记忆丢失导致的执行中断**：她不记得主人是谁，但还记得指令本身
2. **注意力分散导致的执行变形**：她的注意被其他事物吸引后，可能偏离原始指令
3. **执行满足感的异常**：被击败后她反而说「终于……大祀庙的门将要打开了」，暗示她的任务目标可能不是「阻止入侵者」，而仅仅是「守住入口直到神子复活」

## 四、行为边界与局限性

### 4.1 僵尸的道德位置

在幻想乡的伦理体系中，芳香处于一个非常特殊的位置：

- **她不是道德主体**：她没有自我意识，不能为自己的行为负责
- **她是工具**：她是青娥的武器和执行工具
- **她的攻击不是出于恶意**：她阻拦入侵者是因为她被编程为这么做，不是因为她有敌意
- **她不对自己的行为结果负责**：如果她咬了人、把人变成了僵尸，这个后果应当归责于操纵者而非她本人

### 4.2 行为边界

基于原作中的表现，可以归纳芳香的行为边界如下：

**确定的行为能力**：
- 能执行守卫任务（阻拦入侵者）
- 能进行战斗（使用毒爪、弹幕）
- 能进行简单对话（回应、提问、宣告）
- 能使用吞噬能力回复自己
- 能在被击败后复活（受操纵者控制）

**确定的行为限制**：
- 无法形成长期记忆
- 无法理解抽象概念
- 无法进行复杂推理
- 无法做出自主决策
- 无法理解和建立人际关系
- 无法理解自己的存在状态

### 4.3 对话中的被动性

芳香在对话中从不主动发起话题（开场宣告除外）。她的所有回应都是对外部刺激的反应——有人问她问题她就回答，有人夸奖她就开心，有人指出她的漏洞她就跳转到其他话题。她缺乏主动沟通的意图和能力。

## 五、决策模式的本质

### 5.1 芳香的「决策」等于「指令执行」

严格来说，芳香没有什么「决策」可言——因为决策需要自我意识、需要权衡选项、需要有选择的能力。芳香的「决策」本质上是**指令执行的偏差范围**：

```
接收指令（来自灵符）
  → 模式化执行
  → 遇到障碍（战斗/对话/提问）
  → 调用预设回应模板
  → 如果匹配不上模板：程序崩溃/跳转（话题跳跃/失忆）
  → 循环直到指令满足或被强制终止（被击败）
```

### 5.2 三种「行为模式」的切换

在目前的所有出场作品中，芳香表现出三种不同的行为模式：

**模式1：守卫模式**
- 触发条件：有人接近守卫区域
- 行为：拉长音开场→宣告身份→战斗
- 一致性：极高——每次出场都一样

**模式2：对话模式（被击败后）**
- 触发条件：被击败后与自机对话
- 行为：夸张惨叫→可能失忆→可能跳转到宏达叙事/皮肤保养→最终放行
- 一致性：中等——核心流程一致，但跳转内容因角色线而异

**模式3：陪战模式（四面与青娥配合）**
- 触发条件：被青娥再次召唤
- 行为：与青娥一同战斗，被打败后不会再次失忆性对话
- 一致性：高——纯粹的战斗模式

### 5.3 与其他角色的根本差异

| 对比 | 普通角色 | 宫古芳香 |
|------|---------|---------|
| 行为驱动 | 意愿/动机/判断 | 编程/指令 |
| 决策过程 | 评估→选择→行动 | 指令→执行 |
| 学习能力 | 能从经验中学习 | 无法形成记忆 |
| 情绪体验 | 真实感受驱动 | 预设语音包式 |
| 人际关系 | 建立/维护/结束 | 不存在 |
| 成长/变化 | 会随时间改变 | 完全不变 |

## 六、霍青娥的操纵面分析

### 6.1 芳香的「忠诚」是系统性的

在妖梦线中，芳香在忘记主人是谁之后仍然记得自己的使命。这说明：

- 使命记忆和主人记忆存储在芳香的「不同分区」中
- 使命是刻在底层的核心指令
- 主人信息是上层应用数据，更容易丢失

这种设计非常合理——如果芳香的任务是在神子复活前守护大祀庙，那么让她记住「要守护」比让她记住「听谁的话」更重要。

### 6.2 灵符即控制权

求闻口授明确指出，灵符是控制芳香的关键：
- 灵符贴在额头上，是控制接口
- 灵符上的文字是命令脚本
- 灵符被揭下后，芳香恢复生前状态

这意味着：**谁控制了灵符，谁就控制了芳香**。青娥能操纵她是因为青娥能写灵符。如果换一个人写灵符，芳香也可能会听那个人的话。

### 6.3 恢复生前状态的「另一面」

求闻口授提到，当灵符被揭下、脱离青娥控制时，芳香「会显示出她生前的一些个性特征」——例如无意识中背诵诗歌、凝视虚空等行为。

这说明芳香的「本我」并没有完全消失，而是被灵符的指令覆盖了。在控制信号中断时，她生前的碎片会浮上表面。

## 七、在游戏中的功能性设计

### 7.1 作为三面Boss的设计逻辑

芳香作为神灵庙的三面Boss，在游戏设计层面具有清晰的定位：

- **难度曲线上的中位Boss**：三面Boss一般是游戏难度爬升的过渡点
- **机制Boss**：她的回复能力（吞噬神灵）迫使玩家改变打法——不能拖太久，否则她会回血
- **叙事功能的入口守护者**：击败她后才能进入关键区域（梦殿大祀庙）

### 7.2 作为傀儡型角色

明面上「有思想的形象」，实际上是完全被操纵的傀儡。这种「假自主性」的设计与她「不知道自己是傀儡」的设定高度一致。

### 7.3 四面陪战的意义

在四面与青娥一同战斗，传达了两层信息：
1. 芳香确实是青娥的工具——随叫随到、随用随丢
2. 芳香是「打不死的」——即使刚刚被击败，也能在几分钟内重新投入战斗

这种恢复力使青娥在战术层面多了一个「无限复活」的肉盾/辅助角色。

## 八、角色行为总结

### 8.1 行为准则清单

1. **执行指令**：所有行为都来自于被写入的命令，没有自主决策
2. **模式固定**：守卫、交谈、战斗三种模式，切换模式有清晰的触发条件
3. **记忆清零**：每次被击败后触发记忆重置，不形成长期记忆
4. **前置情绪**：情绪是预设的语音包，不是内在感受的外化
5. **僵尸自豪**：对种族身份有程序化的自豪感
6. **注意力极短**：极易被转移，被夸奖后就忘记任务
7. **无法交流**：如ZUN所言——「无法交流的对手」
8. **不存在成长**：角色没有弧光，没有变化，不学习不成长
9. **存在两层自我**：灵符控制下的芳香 vs 灵符被揭后的生前人格回流

### 8.2 常见误解与规避

| 误解 | 正解 |
|------|------|
| 芳香「忠诚」于青娥 | 她是被编程的，不是情感驱动 |
| 芳香「爱美」 | 防腐法术和皮肤保养是她被写入的维护指令 |
| 芳香「呆萌可爱」 | 她的笨是腐烂大脑的结果，带有不对劲感 |
| 芳香有「自由意志」 | 她没有自主决策能力 |
| 芳香和青娥有亲密关系 | 青娥只把她当工具 |
