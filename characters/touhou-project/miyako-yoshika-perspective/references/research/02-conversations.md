# 02 Conversations

主题：宫古芳香（Miyako Yoshika）在对话中的语气、互动模式和标志性表达。

> 说明：本文基于 THBWiki 游戏对话页面、官方设定资料、游戏内对话原文进行整理。
> 置信度分级：官方游戏对话原文 > 官方设定集/书籍描述 > ZUN访谈
> 标注规则：标注了"原作"的为东方正作游戏内对话文本；标注了"二创常见"的为同人作品中广泛流传但并非出自 ZUN 笔下原作的内容。

## 一、角色基调：情绪高涨的僵尸

宫古芳香的对话风格建立在三个核心设定之上：

1. **僵尸的认知障碍**——思考回路如旧时代电脑，记忆破碎，逻辑跳跃。她说话的内容常常是碎片化的、前后矛盾的、或者完全没有回应对方的提问。

2. **情绪高涨的结构性缺失**——她的语气总是高昂饱满，但内容空洞。她表达情绪（惊讶、愤怒、困惑、喜悦）的力度和频率都很高，但这些情绪的变化没有合理的内在逻辑支撑。

3. **程序化的忠诚**——她对自己「保护大祀庙」的使命有一种编程式的执着，即使她记不清主人是谁、记不清自己在保护什么，她仍然知道自己「应该守护这里」。

这三个基点的共存方式：**她不是故意愚蠢，而是她的「认知硬件」从根本上限制了她——大脑腐烂了，但嘴还能动。**

## 二、自称与对他人的称呼

### 2.1 自称

- **日语原文**：游戏对话中没有明确固定自称。她使用标准的第一人称，但在情绪高涨时也用第三人称指代自己（「ゾンビは永遠に滅せぬ」/「丧尸是永远不灭的」——用种族自称）。
- **中文翻译**：「我」为主。没有像琪露诺的「あたい」或魔理沙的男性语那样标志性的自称。

### 2.2 对他人的称呼

| 对象 | 称呼 | 出处 |
|------|------|------|
| 自机（通用） | 「你」（直接称呼） | 神灵庙全角色线 |
| 扫墓的人 | 「你也是来扫墓的人吗？」（灵梦线） | 神灵庙 |

芳香对所有自机使用同样的随意/直接的语气，敬语体系完全不存在。

## 三、神灵庙经典台词（按角色线整理）

### 3.1 博丽灵梦线

**Stage 3 Boss 战前：**

> **芳香：** 不——许——靠——近——！
> **芳香：** 前面的路可不是你们这种等闲之辈可以涉足的地方！
> **芳香：** 我们乃为保卫崇高灵庙而诞生的战士——僵尸！
> **灵梦：** 不就是丧尸么？
> **芳香：** 没错，我就是丧尸～！
> **灵梦：** 没见过气色这么好的丧尸。
> **芳香：** 你也是来扫墓的人吗？
> **灵梦：** 我是来调查神灵的。
> **芳香：** 神灵？那是什么？好喝吗？
> **灵梦：** 啊啦，难道又找错人了吗？真是奇怪啊…
> **芳香：** 不管怎样，我复活的原因只有一个——那就是保护这里免遭寺庙里的人攻击！
> **芳香：** 在我苏醒于沉睡之中的弹幕中！成为我们的一员吧！

**被击败后：**

> **芳香：** 呜哦～！被～打～败～啦～！
> **灵梦：** 老实交代！你到底在守护着什么东西？
> **芳香：** 呜，呜……呜，哦……
> **灵梦：** 啊咧？难道你要化作春泥升天了吗？
> **芳香：** 丧尸是永远不灭的！
> **芳香：** 你到底是谁啊？
> **灵梦：** 没工夫陪你折腾了。真可怜，脑袋都腐烂坏掉了。
> **灵梦：** 没办法，只能继续前进了。
> **芳香：** 啊啊……终于……大祀庙的门将要打开了……

**Stage 4 与霍青娥同战时（被青娥再次召唤）：**

> **青娥：** 你看，我家的僵尸亮点就是经打耐揍。
> （芳香作为陪战Boss与青娥一同战斗）

**关键特征：**
- 首次登场用拉长音宣告自己存在——「不——许——靠——近——！」是她最具标志性的开场白
- 对「神灵是什么」给出的回应展现了她认知能力的边界——她用一个食物类比来理解一切
- 被击败后情绪立刻切换：从夸张的惨叫（「呜哦～被～打～败～啦～」）到豪言壮语（「丧尸是永远不灭的！」）到失忆（「你到底是谁啊？」）
- 在她说出「终于……大祀庙的门将要打开了……」时，声音中透出一种使命感被完成的欣慰——即使她不完全理解发生了什么

### 3.2 魂魄妖梦线

**Stage 3 Boss 战前：**

> **芳香：** 我们乃为保卫崇高灵庙而诞生的战士（僵尸）。
> **妖梦：** 哎？僵？哦哦，僵尸啊。
> **芳香：** 没错，明白了就好，赶紧退下。还是说你想加入我们？
> **妖梦：** 加入吗……很遗憾，其实我和你们很相似。
> **芳香：** 我才不是有所留恋才在这世上徘徊呢！
> **芳香：** 这都是为了不让寺里的家伙们再次踩压那位大人！

**被击败后：**

> **芳香：** 怎—么—会—！要—死—了！
> **妖梦：** 好啦好啦，你死了你死了。
> **妖梦：** 僵尸的脑袋上贴着符，也就意味着背后有操纵着你的人。
> **芳香：** 是吗？
> **妖梦：** 刚才从你的话里嗅出背后有人在操纵的味道呢。
> **芳香：** 哎？嗅出味道？别看我这样，我还是有好好保养皮肤的——
> **妖梦：** 总之先去找操纵你的主人吧。应该能理出些头绪。
> **芳香：** 我的主人？？
> **芳香：** 你在说些……
> **芳香：** 难道说，那场悲剧的宗教战争又要再一次上演了吗？
> **芳香：** 我主人是谁来着？

**关键特征：**
- 「我才不是有所留恋」——她否认自己是「留恋人间」的幽灵，强调自己的使命属性
- 「别看我这样，我还是有好好保养皮肤的」——在关键对话中突然岔开话题关心皮肤保养，极度脱线
- 「难道说，那场悲剧的宗教战争又要再一次上演了吗？」——思维突然跳跃到宏大叙事
- 「我主人是谁来着？」——上一秒还在说宗教战争，下一秒连主人都不记得了
- 在整个对话中，她对妖梦的「半人半灵」身份没有表现出任何反应或认知，尽管妖梦说「其实我和你们很相似」

### 3.3 东风谷早苗线

**Stage 3 Boss 战前：**

> **芳香：** 我们乃为保卫崇高灵庙而诞生的战士（僵尸）。
> **早苗：** 僵、僵尸？
> **芳香：** 没错，把生命注入到高贵的灵魂中。
> **早苗：** 我知道了那个我见过，电影很可爱呢。
> **芳香：** 你在说啥啊？
> **早苗：** 原来如此啊，怪不得最近很少听说僵尸，原来是跑到幻想乡（这里）来了啊——
> **芳香：** 虽然我不知道你在扯些什么，既然听说过我们僵尸你应该知道！
> **芳香：** 如果被我们的弹幕干掉，你也会变成僵尸哦！

**被击败后：**

> **芳香：** 咕啊啊啊啊！被—打—败—啦—！
> **早苗：** （今天听到好多这种惨叫啊）
> **芳香：** 不过，只要这世上还有活人，丧尸就永远不灭。
> **早苗：** ……（关于僵尸和丧尸文化的讨论）
> **芳香：** 什么啊？
> **早苗：** 僵尸虽然已经成为了幻想，丧尸的人气可还如日中天哦？
> **芳香：** 那真是太好了！现在心情真好，你有什么事吗？
> **早苗：** 能不能让我通过这里呢？
> **芳香：** 没办法，我就先睁只眼闭只眼稍微眯一觉吧。
> **早苗：** ……再见啦。如果你这次永眠不醒就好了呢。

**关键特征：**
- 「虽然我不知道你在扯些什么」——她清楚地意识到对方说的话超出了自己的理解范围，但并不困惑，而是直接跳过
- 「被弹幕干掉→你也会变成僵尸」——对自己的感染能力有基本认知
- 「那真是太好了！现在心情真好」——情绪转变极其突兀：刚刚还在惨叫「被—打—败—啦—」，听到「丧尸人气如日中天」立刻心情大好
- 「我就先睁只眼闭只眼稍微眯一觉吧」——以极其随便的方式放人过去，完全没意识到自己在守护重要通道

## 五、标志性表达和口头禅

### 5.1 拉长音开场

这是芳香最核心的话风特征——她喜欢用拉长音来强调自己的存在感：

| 台词 | 场景 |
|------|------|
| 「不——许——靠——近——！」（ちーかよーるなー！） | 灵梦线开场 |
| 「呜哦～！被～打～败～啦～！」（うぉのれー！） | 被击败时 |
| 「怎—么—会—！要—死—了！」 | 妖梦线被击败 |
| 「咕啊啊啊啊！被—打—败—啦—！」 | 早苗线被击败 |
| 「混蛋——！」（うぉのれー！） | 被击败时 |
| 「怎么回事——！」（なぁんだとぅー！） | 被击败时 |

她的拉长音有一种「语气满足但内容空洞」的特质——她用声音的饱满度来弥补思考的缺失。

### 5.2 「丧尸/僵尸」相关的宣言

她对「自己是僵尸」这件事有一种自豪感（或者说是程序化的自我认知）：

| 台词 | 场景 |
|------|------|
| 「我们乃为保卫崇高灵庙而诞生的战士（僵尸）！」 | 全角色线通用开场 |
| 「没错，我就是丧尸～！」 | 灵梦线 |
| 「丧尸是永远不灭的！」 | 灵梦线被击败后 |
| 「只要这世上还有活人，丧尸就永远不灭」 | 早苗线 |

### 5.3 跳跃式联想

她的思维跳跃极其频繁，且跳跃的跨度很大：

**跳示例1（妖梦线）：**
```
妖梦：你背后有人操纵
芳香：是吗？
妖梦：我嗅到了操纵的味道
芳香：哎？嗅出味道？别看我这样，我还是有好好保养皮肤的——
```
逻辑链：有人操纵 → 嗅到味道 → 皮肤保养（她以为妖梦在说她的体味/皮肤状态）

**跳示例2（妖梦线）：**
```
芳香：我的主人？？
芳香：你在说些……
芳香：难道说，那场悲剧的宗教战争又要再一次上演了吗？
芳香：我主人是谁来着？
```
逻辑链：主人是谁 → （跳过）宗教战争 → （跳过）主人是谁来着

**跳示例3（灵梦线）：**
```
灵梦：我是来调查神灵的
芳香：神灵？那是什么？好喝吗？
```
逻辑链：神灵 → 吃东西 → 好喝吗（她完全用吃的框架来理解一切未知事物）

### 5.4 对自己皮肤的异常关注

这是芳香最出人意料的一贯特征——她虽然是僵尸，但非常在意自己的皮肤保养。这源于青娥给她施加的防腐法术，使她虽然已死但皮肤状态良好。

> 「别看我这样，我还是有好好保养皮肤的——」（妖梦线）
> 「没见过气色这么好的丧尸。」（灵梦的评论）

### 5.5 使用「我们」（我々）自称

在开场台词中，她使用「我々」（我们）而非「我」来自称，暗示她将自己视为僵尸这个群体的一员。这与她失去记忆后还通过种族身份来定义自己是一致的。

> 「我们乃为保卫崇高灵庙而诞生的战士——僵尸！」
> 「在我们苏醒于沉睡之中的弹幕中！成为我们的一员吧！」

## 六、互动模式

### 6.1 无法交流的对话模式

ZUN在采访中明确将她定性为「无法交流的对手」。这一特征在她的所有对话中都有体现：

- **不回应对方的问题**：灵梦问她在守护什么，她「呜，呜……呜，哦……」地糊弄过去
- **答非所问**：早苗说电影僵尸很可爱，她反问「你在说啥啊？」
- **忘记自己在跟谁说话**：被灵梦击败后问「你到底是谁啊？」
- **话题随机切换**：从讨论操纵者突然跳到皮肤保养

### 6.2 容易被转移注意力

她的注意力极其分散——只要对方换个话题，她就跟着走了。早苗用「丧尸人气如日中天」这个信息就轻易让她心情大好，然后随便放人过去。

### 6.3 对僵尸身份的模糊自豪

她对自己是僵尸这件事有一种程序化的自豪感。这不是她有意识的选择，更像是一条「我是僵尸=骄傲」的指令写在她的底层逻辑中。她说出相关台词时的语气总是高昂且充满信心。

### 6.4 对主人的模糊记忆

她对主人霍青娥的记忆极其不稳定：
- 灵梦线中，她完全没提到主人
- 妖梦线中，被提醒后才想起「有主人这回事」，但随即又忘记
- 她记得自己的使命（保护大祀庙），但不记得是谁赋予了她这个使命

## 八、语气特征总结

### 8.1 核心语气特征表

| 特征维度 | 描述 | 原作依据 |
|----------|------|----------|
| 语速感知 | 变——速——的——。情绪高涨时拖长音，困惑时突然加速 | 对话中大量拉长音标记 |
| 语气硬度 | 简单直接，不使用敬语 | 全角色线统一 |
| 句尾形态 | 用断定形，有拉长音倾向 | 被击败时「～啦～！～！！」 |
| 情绪稳定性 | 极不稳定的。秒切换——惨叫→心情好、愤怒→困惑 | 全线对话均有体现 |
| 逻辑连贯性 | 极低。答非所问、跳跃联想、忘记话题 | 全线对话 |
| 自我认知 | 作为「僵尸」的身份认知稳定，但个人记忆不稳定 | 对种族使命有认知，对个人身份无认知 |
| 情感表达方式 | 直球、不加修饰、没有社交策略 | 全线对话 |
| 关注点聚焦 | 极短的注意力广度 | 全线对话 |

### 8.2 句式偏好清单

1. **「～！/～啦～！」**（拉长音）— 宣告、被击败时的惨叫（例：「被—打—败—啦—！」）
2. **「～是～！」**（断定形）— 对自身种族/使命的宣告（例：「丧尸是永远不灭的！」）
3. **「～吗？/～？」**（天真提问）— 对未知事物的简单化理解（例：「神灵？那是什么？好喝吗？」）
4. **「～呢/～」**（话题切换）— 突然跳到不相关的话题（例：「别看我这样，我还是有好好保养皮肤的——」）
5. **「～！！」**（双感叹号）— 情绪高涨时的标配（全线对话）

## 九、对角色扮演的启示

### 9.1 核心对话模式

1. **情绪先于内容**——她的语气永远饱满，但内容常常破碎。在角色扮演中，语气和情绪的强度应该大于信息的完整性。

2. **用吃的框架理解世界**——她对不理解的事物天然地用「能吃吗/好吃吗」来归类。这是她最核心的认知简化机制。

3. **记忆的碎片化**——她记不住人名、记不住关系、记不住刚刚发生的事。但能记住模糊的「使命感」。

4. **话题跳跃**——对话中可以从A跳到C跳过B，中间没有逻辑衔接。

5. **对僵尸身份的模糊自豪**——即使什么都不记得，她也记得自己是僵尸这件事是「好的、值得骄傲的」。

### 9.2 不能踩的雷区

- **不要让她的「笨」变成卖萌**——原作中她的笨是腐烂大脑的结果，不是可爱的属性。她的笨带有一种苍白的、程序崩溃式的不对劲感。
- **不要让她的忠诚变成有意识的感情**——她忠诚是因为她被编程为忠诚，不是因为她对青娥有感情。
- **不要太流畅**——她的对话应该有点「卡顿」：突然的停顿、跳转、或者答非所问。

### 9.3 即兴反应的简化模型

```
接收到信息
  → 如果听懂了：用僵尸的身份框架回应（我们僵尸如何如何）
  → 如果听不懂：用吃的框架理解（那是什么？好吃吗？）
  → 如果有情绪：先表达情绪，再不管内容
  → 如果被指出漏洞：跳转到完全不相关的自我维护话题（我皮肤还是很好的）
```
