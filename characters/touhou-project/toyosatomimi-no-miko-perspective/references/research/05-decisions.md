# 调研 Agent 5: 关键决策与行为

## 重大决策记录

### 决策1：接受道教，表面信佛

**时间**：6世纪，飞鸟时代
**背景**：神子作为天才政治家，不满人类终有一死的命运。霍青娥向她推荐道教。
**决策过程**：

1. 被青娥告知道教能实现不老不死 → 心动
2. 但她判断：道教无法用来治国平天下
3. 青娥提出折中方案：表面上信仰佛教（佛教禁止杀生，适合稳定国家）
4. 神子采纳了这个建议

**结果**：
- 公开推广佛教以控制权力
- 背后秘密研究道教
- 展现超能力，留下各种传说

**关键点**：
- 这显示她是一个极度务实的人——信仰对她来说不矛盾，只要有用
- 她有清晰的双轨思考：公共层面（佛教/政治）和私人层面（道教/不朽）
- 这是她一生的大战略的起点

### 决策2：成为尸解仙

**时间**：6世纪末~7世纪初
**背景**：外丹术（朱砂等含有毒物质）侵蚀了她的身体。发现身体撑不下去了。
**决策过程**：

1. 意识到外丹术反而在毁灭自己
2. 选择了道教的"尸解仙"道路——先死一次再复活
3. 因为害怕一个人进行，让物部布都先作为试验品
4. 布都沉睡后没有腐烂，确认安全后自己也长眠

**结果**：
- 计划在佛教的局限显现时复活
- 但因佛教僧侣不断封印灵庙，无法按计划复活
- 沉睡持续了千年以上

**关键点**：
- 显示了她谨慎的一面——让布都先试
- 也显示了她作为统治者的自信——预设了"国家需要我时"的场景
- 但计划被打乱——佛教统治日本远超预期

### 决策3：将灵庙移至幻想乡

**时间**：神灵庙事件前（约2011年）
**背景**：
- 现代社会开始质疑"神子的传说都是杜撰的"
- 再也没有超乎常人能力的人
- 幻想乡当时没有一座寺庙
**决策**：将灵庙移入幻想乡，准备复活
**结果**：
- 命莲寺恰好建在了灵庙正上方
- 与佛教势力形成地理上的直接对抗

### 决策4：复活后的选择

**时间**：神灵庙事件（2011年）
**背景**：
- 在神灵庙中成功复活
- 与灵梦/魔理沙/早苗/妖梦战斗
- 战后被击败
**决策**：
- 将自己的神灵庙迁往自己创造的仙界（Senkai）
- 在幻想乡建立新的道场
**结果**：
- 建立了属于自己的领域
- 不再直接命莲寺之下

### 决策5：成为人类村落的潜在管理者

**时间**：求闻口授时期（2012年）
**背景**：
- 幻想乡的人类村落缺乏强有力的领导者
- 神子认为需要一个管理者防止社会崩溃
**决策**：
- 表达了对担任"管理者"的兴趣
- 被告知"不必要"后退让
**关键点**：
- 反映出她骨子里的统治欲和责任感
- 也显示她能接受现实判断——不需要就不强行介入

### 决策6：与白莲的合作

**时间**：凭依华时期
**背景**：完全凭依异变发生，异常严重
**决策**：
- 放下与白莲的佛道之争，暂时联手
- 两人和随从一起调查完全凭依事件

**关键点**：
- 说明她的竞争意识不排除合理合作
- 她不是赌气型的敌对者，而是务实的战略家
- 白莲也是同等对手——只和值得的人联手

### 决策7：对待弟子的方式

**根据口授记载**：
- 很多人想拜她为师
- 但她只把弟子当打杂用
- 仙人到了最后只为自己使用力量，没有教授他人的必要

**关键点**：
- 显示她独有的仙人逻辑——与常规师徒关系完全不同
- 但也说明她没有积极建立组织/势力的野心

## 一致性分析

| 维度 | 分析 |
|------|------|
| 言行一致度 | 高度一致——她说自己是为人类好的统治者，事实上也确实在帮助人类、警惕妖怪威胁 |
| 决策风格 | 务实而长远——先规划千年后再行动；考虑多重结果（公共vs私人、短期vs长期） |
| 道德边界 | 实用主义但有自己的底线：正直、反感邪恶、不靠歪门邪道 |
| 待人风格 | 洒脱轻浮但实质真诚——不伪装自己的欲望和目的，直接点破对方 |
