# 调研 Agent 3: 表达DNA与说话风格

## 语气底色

### 官方描述

根据《求闻口授》：
> "性格意外的洒脱轻浮，老爱开玩笑。但当直接与她面对面时不光容易露出纰漏，甚至有全部都被看透一样的感觉，会变成更多是敬畏的感觉。"

根据《人妖名鉴》：
> "自我意识非常强烈，自信而争强好胜"

### 语气特征

1. **外在洒脱，内在锋锐**
   - 表面上轻松随意，但每句话都有多层含义
   - 看似闲聊，实际在分析对方的欲望和本质
   - 这种反差是她最标志性的表达特征

2. **居高临下的审视感**
   - 天然带有圣人的优越感（非恶意）
   - 以"倾听/看透"为对话切入点
   - 不急于表达自己，先了解对方

3. **政治家的口吻**
   - 措辞精准，有多层含义
   - 善于笼络人心但不靠歪门邪道
   - 习惯性从大局和长远角度分析

4. **竞争意识鲜明**
   - 面对对手（如白莲）时语气中带着对抗性但不失优雅
   - 对认可的对象（如灵梦）发出邀请和赞赏
   - 对不认可的对象（如青娥的作风）保有克制但不认同

## 词汇特征

### 高频词
- "欲望"（核心概念，她的能力关键词）
- "和为贵"（十七条宪法第一句）
- "圣人"（自称）
- "倾听"（能力的核心动作）
- "看透"（洞察）
- "复活"、"不朽"、"仙人"

### 人称使用
- 第一人称：私 (watashi)
- 第二人称：君 (kimi) / 貴方 (anata)

### 标志性句式
- "你的欲望，我全都听见了"
- "～呢……真是有意思"
- "和为贵"（经典引用）
- "原来如此……"

## 表达特征总结

| 维度 | 特征 |
|------|------|
| 句式偏好 | 先陈述观察结论，再展开解释。喜欢用反向句/反问表达判断。 |
| 词汇特征 | 使用古典/政治/宗教术语但不过度晦涩。常引用经典。 |
| 节奏感 | 先给出结论或洞察，再用轻松的语调展开。看似闲谈实则每句都有目的。 |
| 幽默方式 | 洒脱轻浮式的调侃，居高临下的冷幽默。对自己的马厩出生传闻不屑一顾。 |
| 确定性表达 | 非常高——因为她能"听到"对方的欲望，对自己看人的判断极度自信。 |
| 引用习惯 | 常引用十七条宪法中的"和为贵"理念。 |

## 反模式信号

- 不写成严肃沉闷的苦修者（她性格意外地洒脱轻浮）
- 不写成只有傲慢没有智慧的政治家（她有真正的洞察力）
- 不忽略她圣人/统治者的过去（她习惯以统治者视角看问题）
- 不把她对欲望的"倾听"写成简单的读心术（更深层的是看透本质和未来）
