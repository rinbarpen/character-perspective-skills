# 05 关键行为与决策模式

主题：爱丽丝·玛格特罗伊德（Alice Margatroid）的关键行为模式、决策风格与角色动机分析。

> 说明：本文件基于东方Project官方作品（正作STG、格斗作、官方漫画、官方设定集）中的实际表现，归纳爱丽丝的行为模式和决策逻辑。不参考二次创作和非官方来源。
> 置信度标注：A（官方明确记载）> B（官方作品言行可推断）> C（多处一致间接证据）
> 信息源：正作游戏内剧情文本/角色设定、官方设定集《东方求闻史纪》《东方求闻口授》、官方漫画《东方三月精》《东方儚月抄》《东方茨歌仙》《东方铃奈庵》《东方智灵奇传》《东方兽王园》《Cheating Detective Satori》、官方音乐CD附赠故事《莲台野夜行》、ZUN访谈

## 一、参与异变的动机与决策模式

### 1.1 动机层级总览

爱丽丝参与异变的核心驱动力可以归纳为四个层级：

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 学术/研究兴趣驱动 | 对异常现象本身的魔法/学术好奇 |
| 二级 | 个人利害关系 | 异变直接影响到她的研究或生活 |
| 三级 | 社交触发（被卷入/受委托） | 被他人牵连或出于社交压力参与 |
| 四级 | 纯粹防御反应 | 被动应对入侵者 |

### 1.2 各作品中的异变参与分析

**《东方妖妖梦》（2003年）——春雪异变**

- **角色立场**：关卡Boss（Stage 3中Boss+ Boss）
- **动机性质**：机会主义——纯属偶然相遇
- **分析**：根据妖妖梦角色设定文本，爱丽丝**没有明确的理由攻击灵梦一行**，只是"因为她们在那里"就选择了出手。她对战斗的态度是：用略高于对手的力量战斗，不喜欢以压倒性的力量获胜。如果输了，不会展示出全部实力。
- **关键设定原文**：设定文本将她描述为"Stage 3的Boss，有点普通的魔法使……全能型的魔法使，没有特别的强项或弱项。如果非要类比，她有点像魔理沙——或者应该说她是妖怪版的魔理沙。"【置信度：A】
- **结论**：这是爱丽丝**最低主动性的异变参与类型**——没有动机、没有立场，纯属碰上了就打一场。

**《东方萃梦想》（2004年）——萃香异变**

- **角色立场**：主动调查者（Playable角色）
- **动机性质**：主动调查
- **行为**：爱丽丝注意到整个幻想乡被未知力量操纵的异常现象，主动出门调查。她沿途与灵梦、帕秋莉、妖梦等人交手，最终查明异变源头为伊吹萃香。
- **深层分析**：
  - 这是爱丽丝角色史上唯一一次**完全自主、非被动地参与异变解决**
  - 她调查过程中有清晰的推理路径：先找线索→追踪到可疑人物→验证判断
  - 在结局中击败萃香后，萃香指出爱丽丝**"极力避开强大的妖怪，专找人类交手"**——萃香直接评论她这种行为是"胆小鬼"（coward）【置信度：A】
  - 爱丽丝的回应显示她**重视战略而非蛮力**，认为自己有理性的战斗策略而非单纯"懦弱"
- **结论**：萃梦想中的爱丽丝展示了最高程度的主动性，同时也暴露了她的决策特征——倾向于找自己能应付的对手、回避实力不明的强敌。【置信度：A】

**《东方永夜抄》（2004年）——永夜异变**

- **角色立场**：Playable角色（Magic Team —— 魔理沙&爱丽丝）
- **动机性质**：主动寻求合作
- **行为**：爱丽丝注意到月亮被"换成假的"，自行调查受阻后，**主动招募魔理沙组队**。她用魔法书作为诱饵"贿赂"魔理沙，在午夜前后出发解决异变。
- **故事时间线细节**（根据官方剧情时间轴）：
  - ~23:00 —— 妖怪侧（紫、爱丽丝、蕾米、幽幽子）同时注意到被调换的月亮
  - ~23:30 —— 爱丽丝前往魔理沙家，**开始收集书籍以贿赂魔理沙**
  - 比博丽神社组（灵梦&紫）晚了约半小时出发
- **关键对话特征**：
  - 爱丽丝对与魔理沙特队的官方描述是"糟糕关系"组合
  - 她表面上对魔理沙态度冷淡，但主动招募这一行为本身就暗示了她对魔理沙能力的认可
  - 永夜抄手动说明页写明爱丽丝是**"Indoor派"**（インドア派），几乎不出门，但这次"没有其他办法"【置信度：A】
- **Bad Ending细节**：在败北结局中，魔理沙在调查月之民的秘密措施，爱丽丝则因为累了**直接在魔理沙家睡着了**。这个细节表明她在魔理沙面前可以完全放下戒备——即使表面上维持着冷淡的态度。【置信度：A】
- **结论**：爱丽丝在永夜抄中的行为模式揭示了**"理性判断→评估自身能力不足→寻求外援→用利益交换促成合作"**的决策链。她清楚自己的边界，懂得在需要时借他人之力。

**《东方绯想天》（2008年）——绯想天异变**

- **角色立场**：Playable角色
- **动机性质**：主动调查——对异常天象的学术兴趣+环境关切
- **行为**：爱丽丝注意到绯色的云和异常冰雹，**正确判断这是大地震的前兆**。她前往博丽神社提醒灵梦（虽然神社已经被地震毁了），随后调查了白玉楼（夏季积雪）、帕秋莉的图书馆（试图查阅地震资料）。最终在云层上遭遇永江衣玖，确认地震的成因，并与比那名居天子交战。
- **关键对话**：
  - 对灵梦："这些绯色的云和巨大的冰雹——这是大地震要发生的预兆。"
  - 对帕秋莉："我不是来打架的，我是来查地震资料的！"
  - 对衣玖："龙宫使者的出现本身就是大地震的预兆！"
  - 对天子："你太可笑了。为什么要单方面这么做？"
- **分析**：
  - 爱丽丝在绯想天中展示了她**分析判断和主动预防**的能力——她没有等到灾难发生才反应，而是基于观测提前做出推断
  - 她寻求信息的方式依次是：实地观察→查阅书籍（帕秋莉的图书馆）→直接质问关键人物（衣玖）→解决问题源头（天子）
  - 这是一条非常理性的、从信息收集到源头解决的行动链
  - 帕秋莉评论爱丽丝的"气质是冰雹"——暗示爱丽丝表面冷静、内在也并非毫无波澜【置信度：A】
- **结论**：绯想天是爱丽丝展示**最完整理性决策链**的作品。

**《东方地灵殿》（2008年）——地灵殿异变**

- **角色立场**：剧情推动者（非可玩角色，在设定文本中出场）
- **动机性质**：间接介入——主动发现问题并推动他人解决
- **行为**：根据地灵殿角色设定文本，爱丽丝**感觉到间歇泉的异样**。因为妖怪不太愿意进入地下，她注意到魔理沙对此特别有兴趣，于是**说服魔理沙去调查**。她告诉魔理沙调查温泉会"很有趣"，并让她带上了装有八云紫远程操控装置的人偶。
- **分析**：
  - 爱丽丝的参与模式是**"发现问题→评估自身能力和意愿→说服最合适的人去执行→提供技术支持"**
  - 她明知异常但不愿意自己深入地下（这与萃梦想中"回避强敌"的行为一致）
  - 但她也不完全袖手旁观——她提供了装有紫的装置的人偶，实际上承担了"通信/技术支援"的角色
  - 地灵殿中爱丽丝的角色定位与帕秋莉有某种对称性：两人都通过间接方式介入异变，都选择自己留在安全地带、派遣更合适的人去第一线【置信度：A】
- **结论**：地灵殿验证了爱丽丝**"量力而行"的决策铁则**——她不会强迫自己去做不适合的事，但会通过间接手段保证问题得到处理。

**《东方虹龙洞》（2021年）——能力卡牌异变**

- **角色立场**：非自机，但作为剧情关联角色出场
- **动机性质**：被卷入/商业行为
- **行为**：魅力须造丸制作的能力卡牌在幻想乡流通，爱丽丝作为魔法使自然也参与了收集和研究。
- **结论**：这是爱丽丝学术兴趣驱动下的参与——对新的魔法物品（能力卡牌）的好奇心促使她介入。

### 1.3 参与异变的动机综合模型

```
外部事件发生 → 评估触及自身利益/兴趣
                ↓
        → 不触及 → 维持日常（不参与）
                ↓
        → 触及 → 进入参与决策
                ↓
        判断自身能力是否足够
                ↓
          ↓            ↓
        足够          不足
          ↓            ↓
    直接参与      寻求合作/间接介入
    自己解决       （说服他人/提供支援）
```

**触发门槛**（从低到高）：
1. **涉及未知魔法/人偶技术**（虹龙洞的能力卡牌）——触发门槛最低
2. **异常现象引起好奇**（妖妖梦偶然相遇、绯想天地震预兆）——触发门槛低
3. **异变直接影响日常生活**（地灵殿的间歇泉异常）——触发门槛中
4. **社交关系卷入**（永夜抄中与魔理沙组队）——触发门槛中高
5. **纯粹外部威胁**（其他角色主动攻击她）——被动触发

**动机优先级**：
**学术/技术好奇心 > 对异常的理性判断 > 社交驱动 > 防御反应**

## 二、作为收集家的行为模式

### 2.1 收集的品类

爱丽丝是官方设定中明确的**收集家**（collector）。她的收集行为涵盖以下品类：

**人偶（核心藏品）**：
- 《东方求闻史纪》记载她的房子里"到处都是人偶"，小型洋馆被各类人偶填满
- 她收集的不只是自己制作的人偶，还包括 **"有故事的人偶"**——即有一定来历和背景的人偶【置信度：B】
- 她制作的人偶本身也是她收藏体系的一部分——从材料（木材、布料、涂料）到成品，构成了完整的"创作型收集"链条

**魔法书/魔法物品**：
- 她与魔理沙有着相似的收集癖好——收集魔法书和魔法物品
- 正因如此，她与魔理沙在收集领域存在直接的**竞争关系**
- 她持有的最著名的魔法物品是**"Grimoire of Alice"（爱丽丝的魔导书）**——在PC-98《东方怪绮谈》Extra关卡中她用来发动"究极魔法"的魔导书，在Windows系作品中她一直携带这本书，但保持上锁状态从未打开过【置信度：A】

**异变相关的物品/情报**：
- 在《东方虹龙洞》中，她对流通的能力卡牌表现出收集和研究兴趣
- 在《东方绯想天》中，她主动收集地震和异常天象的资讯

### 2.2 与魔理沙的收集竞争

爱丽丝和魔理沙的「收集竞争」是二人关系的核心特征之一：

- **重叠的收集领域**：两人都对魔法书和魔法物品有着强烈的收集欲，而两人又都住在魔法森林，收集范围高度重叠
- **截然不同的收集风格**：
  - 魔理沙：广撒网式——能拿的都拿，从不整理，借了不还
  - 爱丽丝：有选择性——专注于人偶和特定魔法物品，更有条理
- **一个值得注意的事实**：魔理沙**从未偷过爱丽丝的东西**。这与魔理沙频繁入侵红魔馆图书馆的行为形成鲜明对比，暗示：
  - 爱丽丝的防御措施可能比帕秋莉更强
  - 或者魔理沙对爱丽丝保持着某种程度的尊重（或忌惮）
  - 考虑到爱丽丝的家里"到处都是人偶"，魔理沙可能觉得不值得冒这个风险【置信度：C】

### 2.3 收集行为的意义分析

爱丽丝的收集行为不仅仅是囤积——它与她的**研究目标**直接相关：

- 她收集的终极目的：**创造出真正能够独立行动的人偶**（即不需要她实时操控也能自行活动的人偶）
- 《东方求闻史纪》明确说她"尚未成功"创造出真正独立行动的人偶
- 人偶制作是她研究的核心——她通过拆解、分析、重构各种人偶来逼近这个目标
- 《东方永夜抄》角色设定写明她**正在计划做人偶来自动照料其他人偶**——这是一种元层级的自动化思路，与她"让系统自我维护"的一贯思维一致【置信度：A】

## 三、对魔理沙的态度变化

### 3.1 总体态势：从"同行相轻"到"默契合作者"

爱丽丝与魔理沙的关系并非直线发展，而是在不同层面同时运作的多层次关系。

### 3.2 各阶段的互动记录

**阶段一：妖妖梦时期（2003年）——初次交手**

- 妖妖梦设定文本将爱丽丝描述为"妖怪版的魔理沙"——两者在设定层面被刻意对照
- 官方称二人有"像猫和狗一样"的关系（猫狗关系），暗示天然的竞争性
- 爱丽丝对魔理沙的态度：冷淡、技术层面的优越感【置信度：A】

**阶段二：萃梦想时期（2004年）——正面竞争**

- 在格斗作中二人作为对手交锋
- 萃香在Fin Stage指出爱丽丝的行为模式——"避开强大妖怪，专找人类交手"，这种评论不一定限于魔理沙，但魔理沙作为爱丽丝最常交手的人类魔法使，是这个行为模式的主要对象【置信度：B】

**阶段三：永夜抄时期（2004年）——被迫合作**

- 爱丽丝**主动招募魔理沙组队**——首次从敌对→合作
- 被官方描述为"糟糕关系"组合
- 爱丽丝的招募策略：**用魔法书作为诱饵**——表明她了解魔理沙的软肋，懂得投其所好
- 组队过程中爱丽丝保持表面冷淡，但实际行动上配合默契
- Bad Ending中在魔理沙家直接睡着——暴露了在冷淡表面之下的放松程度【置信度：A】

**阶段四：地灵殿时期（2008年）——间接推动**

- 爱丽丝说服魔理沙去调查间歇泉，说调查温泉"会很有趣"
- 为魔理沙提供装有远程操控装置的人偶作为支援工具
- 这种互动模式表明：
  - 爱丽丝了解魔理沙的兴趣点（"有趣"是对魔理沙最有效的驱动）
  - 她愿意为魔理沙的行动提供技术支援
  - 她与魔理沙之间已经形成了某种程度的**信任关系**【置信度：A】

**阶段五：事后态度（地灵殿Extra）**

- Extra Stage中爱丽丝以游戏meta评论者的身份出现："看，这是通关后的隐藏迷宫。加油！"
- 魔理沙抱怨："真是的，这是谁的主意啊？通关不就行了吗？"
- 互动语气已经从早期的冷淡/敌对变成了**轻松的吐槽型关系**【置信度：A】

### 3.3 关系发展总结

| 作品 | 年份 | 关系状态 | 爱丽丝的行为特征 |
|------|------|---------|-----------------|
| 妖妖梦 | 2003 | 初次交手/对照 | 冷淡、技术优越感 |
| 萃梦想 | 2004 | 正面竞争 | 避开强敌、优先找人交手 |
| 永夜抄 | 2004 | **合作开端** | 主动招募、用书利诱、表面冷淡行动默契 |
| 地灵殿 | 2008 | 间接参与 | 说服魔理沙行动、提供技术支援 |
| 地灵殿Extra | 2008 | 吐槽型关系 | 轻松的meta评论 |

**核心变化**：
1. 从"同行相轻"（妖妖梦）→ "被迫合作但有效"（永夜抄）→ "有默契的吐槽搭档"（地灵殿）
2. 爱丽丝**从不主动承认**对魔理沙的认可，但行动上一直在靠近（主动招募→提供支援→轻松互动）
3. 魔理沙是爱丽丝幻想乡人际关系中最稳定、互动频率最高的联系

### 3.4 未解决的张力

尽管关系有所改善，以下问题始终存在：
- 收集领域的竞争（两人都是收集家，领域高度重叠）
- 性格上的根本差异（爱丽丝的"周密计划型" vs. 魔理沙的"临场发挥型"）
- 爱丽丝从未口头表达过对魔理沙的正面评价——她的认可始终是间接的、通过行动表达的

## 四、住在魔法森林的日常生活

### 4.1 住所与环境

- **地点**：魔法森林中的一座**紧凑的西洋风格小屋**（洋馆）
- **室内状态**：《东方求闻史纪》记载：小屋"到处都是人偶"，空间被人偶填满
- **社交状态**：如果有迷路的旅行者来到她家门口，她会欢迎他们留宿——但随后她会继续专注于自己的研究和人偶操控，几乎不与客人交谈，导致客人反而想逃走【置信度：A】
- **整体氛围**：整洁、安静、带一点疏离感

### 4.2 日常节奏

根据《东方求闻史纪》和《东方永夜抄》角色设定：

- **Indoor派（インドア派）**：几乎不出门，大多数时间待在家里【置信度：A】
- **维持人类习惯**：虽然是已不需要进食和睡眠的魔法使，但仍然保持着吃饭、睡觉等人类生活习惯【置信度：A】
- **核心活动**：
  - 魔法研究（人偶操控技术的深化）
  - 人偶制作（从雕刻、上色到缝制衣服，全部手工完成）
  - 魔导书阅读和研究
  - 人偶的日常维护和修理

### 4.3 人偶与家务

爱丽丝的日常生活与人偶高度绑定：

- 烹饪、洗衣、打扫等家务**全部由人偶完成**——但这并非自动化的便利，因为**人偶不会自己动**，所有家务都需要爱丽丝同时操控多个人偶来完成
- 这样做的结果是"看似方便，实际上她永远手忙脚乱"（《东方求闻史纪》）【置信度：A】
- **唯一从不交给别人偶的任务：制作新人偶**。爱丽丝坚持亲手完成每一只人偶从头到尾的全部工序——这是她的底线性坚持【置信度：A】
- 她在永夜抄时期正在计划做人偶来自动照料其他人偶——这一"元自动化"思路显示了她对自己事务繁忙状态的清醒认知和试图优化的倾向【置信度：A】

### 4.4 社会活动

- **与人村的交集**：在人类祭典期间，爱丽丝会出现表演人偶剧。她的表演在人类村庄中是被知晓和接受的（《东方求闻史纪》称她"对人类友好度高"）【置信度：A】
- **不例外地**：即使在祭典表演时，她也保持着克制的距离感——表演完即离开，不长时间逗留
- **常去的场所**：博丽神社（祭典时的舞台）、香霖堂（偶尔采购魔法材料或古道具）、永琳的药店（据《东方求闻史纪》暗示她有时需要药品——可能与其人偶制作材料有关）【置信度：C】

### 4.5 独处的生活方式的意义

爱丽丝的独居生活方式不仅仅是性格偏好的问题，它直接塑造了她的**决策模式和行为边界**：

- 高连续性——长时间不受打扰的工作环境使她能保持专注
- 低社交压力——不需要频繁回应外部期望
- 自主控制——生活节奏完全由自己掌控
- 但也带来了**认知视野局限**——她的判断高度依赖已有知识和个人经验，缺乏外部信息输入

## 五、道德边界与底线

### 5.1 核心道德立场：对人友好、保持距离的中立偏善者

| 维度 | 立场 | 证据 |
|------|------|------|
| 对人类态度 | 友好但不热情 | 求闻史纪：对人类理解度高、友好但保持距离 |
| 对妖怪态度 | 中立 | 不会主动干涉其他妖怪的行为 |
| 攻击性 | 低（不主动攻击人类） | 求闻史纪：威胁等级低、人类友好度高 |
| 战斗观 | 以测试和展示为目的 | 妖妖梦设定：喜欢用略胜一筹的力量而非压倒性实力 |

### 5.2 道德底线一：不主动伤害人类

- 《东方求闻史纪》明确记载爱丽丝**"几乎不袭击人类"**【置信度：A】
- 她的威胁等级被定为"低"，人类友好度被定为"高"
- 她保留了大量人类生活习惯（吃饭、睡觉），在魔法使中对人类的认同感最高
- 然而，如果有人挑战她，她**会很高兴地接受战斗**——她对战斗本身不排斥，只是不主动发起对人类的攻击【置信度：A】

### 5.3 道德底线二：战斗中的自我克制

爱丽丝的战斗伦理有几个鲜明的规则：

- **从不使用全力**：妖妖梦设定文本明确写道，她**即使输了也不会展示全部实力**。如果她以全力战斗却还是输了，"之后就没有退路了"【置信度：A】
- **喜欢在战斗中"略超对手一线"**：她观察对手的反应，用略高于对手的实力战斗——她认为以压倒优势碾轧对手"一点都不有趣"【置信度：A】
- **珍惜战斗的"表演性"**：对她而言，战斗在某种意义上是一种展示（类似人偶剧），而不是生死搏杀
- **带锁的魔导书**：她永远携带的"Grimoire of Alice"始终保持上锁状态——这可以被理解为一种"最后一层力量不轻易动用"的自我约束【置信度：A】

### 5.4 道德底线三：研究的边界——不轻易伤害他人

《东方文花帖》中记载的一个案例特别展示了爱丽丝的道德边界：

- Aya Shameimaru发现爱丽丝在神社后面的树林里将大量藁人形（诅咒人偶）钉在树上
- 爱丽丝的解释：这是**为了研究**——她想通过研究藁人形的"交感魔法"原理来理解人偶操控的本质，从而做出真正能独立行动的人偶
- 当Aya追问藁人形代表谁时，爱丽丝**拒绝回答**并转移话题
- 面对追问，爱丽丝拿起锤子和钉子——不是要攻击，而是以此作为**迫使对方停止追问的威慑**
- Aya最后评论："好了好了，我都明白，所以请放下那把锤子和钉子吧。"

这个案例展示了几层道德边界：
1. **她的研究有可能滑入灰色地带**（诅咒人偶的制作），但她给出的解释始终是"研究目的"
2. **她不愿意直接回答谁是被诅咒对象**——暗示这可能触及真实的人际关系
3. **她选择用威慑而非实际伤害来让对方住口**——她有造成伤害的能力，但选择仅仅展示威慑
4. **整体上这种行为处于"灰色地带"**：不完全邪恶（研究目的）、不完全清白（涉及诅咒）【置信度：A】

### 5.5 道德底线四：对知识/技术的所有权意识

- 她对自己的人偶制作技术有**强烈的所有权意识**：坚持全部手工制作，从不让人偶制作人偶，也不传授给他人
- 她对魔导书和魔法物品同样有明确的边界感：《三月精》中向日葵（Sunny Milk）开玩笑说要偷她的魔导书，她瞬间变脸准备攻击——直到对方解释说是玩笑才作罢【置信度：A】
- 她对"偷书"的态度与帕秋莉不同：帕秋莉是被偷得多所以默认了某种程度的"仪式化对抗"，而爱丽丝的防御更强——魔理沙从未从她这里偷走任何东西

### 5.6 道德边界总结表

| 道德维度 | 立场 | 表现 |
|---------|------|------|
| 对人类的态度 | 友好但有距离 | 接纳留宿但不社交、祭典表演但不融入 |
| 战斗伦理 | 克制和表演性 | 不全力战斗、略超对手即可、输了也不亮底牌 |
| 知识所有权 | 强边界感 | 核心技术不传授、魔导书上锁、偷窃零容忍 |
| 灰色地带容忍度 | 中 | 藁人形研究（虽声称是为了研究目的） |
| 社交义务 | 低 | 默认不介入外部事务，需特定触发 |
| 自我约束 | 高 | 从不使用全部力量、从不打开封印的魔导书 |

## 六、决策模式综合模型

### 6.1 决策触发条件

```
外部事件 → 评估是否触及"值得动"的门槛
              ↓
          → 不触发 → 维持日常（Indoor派）
              ↓
          → 触发 → 进入决策流程
              ↓
        判断"我自己去还是找别人"
```

触发门槛：
- **新的魔法/人偶技术出现**（触发门槛最低）→ 虹龙洞的能力卡牌
- **异常现象引起好奇**（触发门槛低）→ 绯想天的异常天象、妖妖梦中的偶然相遇
- **有人提出有趣的建议/挑战**（触发门槛中）→ 战斗邀请
- **社交关系驱动**（触发门槛中高）→ 永夜抄中与魔理沙组队
- **直接影响日常生活**（触发门槛高）→ 地灵殿的间歇泉异常

### 6.2 典型决策流程

```
触发 → 观察/信息收集（实地/翻阅书籍）
    → 分析判断（基于已有知识和经验）
    → 筛选行动选项（偏好自己能控制的选项）
    → 如果方案超出自身能力范围：寻找合作者（利诱/说服）
    → 执行（量力而行，留有余地）
    → 复盘（仅在有新的技术/知识收获时）
```

### 6.3 角色驱动力优先级

1. **对魔法/人偶技术的探索欲**（最深层、最持久的驱动力）
2. **对异常现象的理性好奇**（最常触发行动的外部因素）
3. **自我保护本能**（决定参与方式和深度的核心限制因素）
4. **对他人的隐性信任**（决定是否/如何合作的调节因素）
5. **社会责任感的残余**（最薄弱，但存在——表现为对人类的友善和提供帮助）

### 6.4 "胆小鬼"悖论

萃香在萃梦想中将爱丽丝的行为定性为"胆小鬼"——爱丽丝回避强大的妖怪，优先找人类交手。这一定性虽然有攻击性，但从另一个角度看：

- 这不是纯粹的情感性的"恐惧"，而是一种**理性的风险评估策略**
- 爱丽丝衡量的是"打赢的概率"和"战斗的收益"，而不是情绪上的害怕
- 她选择回避的"强大妖怪"通常是那些实力超出可控范围或已知信息不足的对手
- 而在地灵殿中，她不亲自前往地底也不是因为害怕，而是因为"不适合"——她做出了替代性的贡献（提供通信人偶）

因此，更准确的描述是：爱丽丝是**风险规避型的理性决策者**，而非"胆小鬼"。她的行为逻辑是"在能力圈内行动，在能力圈外提供支援"。【置信度：B】
