# 驹草山如 — 决策与行为模式研究

## 1. 行为动机层级

| 优先级 | 动机 | 说明 |
|--------|------|------|
| 一级 | 赌场的稳定运营 | 维持绅士们的社交场所 |
| 二级 | 个人兴趣（烟草与赌博） | 乐在其中 |
| 三级 | 山中社交关系 | 与天狗、山童、河童保持良好关系 |
| 四级 | 对龙珠/卡牌的商业兴趣 | 从中看到赌博的新彩头 |

## 2. 面对冲突的反应

- **先礼后兵**：先以营业口吻劝说，谈不拢再动手
- **点到为止**：败北后爽快认输
- **提供信息**：即使被击败也告知对方山顶的方向
- **不记仇**：战斗归战斗，不延展到私人关系

## 3. 道德边界

### 底线
- 赌场是绅士们的社交场所——不是纯粹的地下黑市
- 用烟草维持秩序而非制造混乱
- 愿赌服输

### 灰色地带
- 用精神操纵的烟草来维持赌场秩序（虽然是为了防止暴力）
- 对老友有食欲但没有付诸行动
- 从能力卡牌中看到赌博价值而非危险性

## 4. 关系处理

### 对饭纲丸龙
- 表面：老友关系，龙对她信赖
- 内心：有食欲——想吃掉龙
- 但不影响正常交往——食欲和友情是两回事

### 对自机
- 拦路但不是敌人
- 败北后提供信息
- 自始至终保持营业态度
