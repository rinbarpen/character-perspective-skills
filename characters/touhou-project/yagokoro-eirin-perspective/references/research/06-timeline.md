# 八意永琳 作品时间线

> 本文件整理自 THBWiki（thwiki.cc），按剧情时间与实际发行时间两条线索梳理。

---

## 一、剧情时间线

以下事件在东方Project剧情框架中的先后顺序。由于月人时间感的特殊性（亿年级），部分年代为推断。

### 太古时期（年代不详）

| 事件 | 说明 |
|------|------|
| **永琳协助月夜见建立月之都** | 与月夜见尊从地球移居月球，成为月都的创建者之一 |
| **八意家的天才** | 永琳作为八意家（药师天才家族）的最杰出成员，成为月之贤者，号"月之头脑" |
| **担任绵月姐妹的老师** | 负责教育月之公主绵月丰姬和绵月依姬，传授知识、法术和处世之道 |
| **浦岛太郎事件** | 绵月姐妹藏匿误入月都的地上人水江浦岛子三年。永琳最初提议"杀了他"，后在学生恳求下给出"冬眠三百年后送回"的替代方案 |

### 蓬莱之药时期（约千余年前）

| 事件 | 说明 |
|------|------|
| **制作蓬莱之药** | 辉夜出于好奇想要不老不死之药，永琳利用辉夜操纵永远与须臾的能力，两人共同制成蓬莱之药 |
| **辉夜被流放** | 辉夜服药后被月都视为沾染污秽，被处以极刑流放至地上 |
| **永琳未受罚** | 因制药本身不是服用，永琳未被处罚，留在月都继续担任贤者和老师 |
| **嫦娥偷药** | 据传嫦娥偷服的不死药也源于永琳所制 |

### 永琳背叛月都（约千余年前）

| 事件 | 说明 |
|------|------|
| **被派往地面** | 辉夜刑期届满，月都派遣使者前往地球接回辉夜，永琳是使者之一 |
| **杀害同行使者** | 出于愧疚，永琳与辉夜合谋杀死了其他月之使者 |
| **封口行动** | 将蓬莱之药交给目击地上人以封口，该人后被灭口（未服药） |
| **逃亡隐居** | 两人逃至远离人烟的山野，最终定居于幻想乡迷途竹林中的**永远亭** |

### 月面战争时期（数百年前——确切年代不详）

| 事件 | 说明 |
|------|------|
| **第一次月面战争** | 八云紫率领妖怪进攻月都。永琳虽已叛离月都，但其布下的结界断了紫的退路，间接帮助月都击退入侵 |
| **月都结界** | 此后月都在月球背面张开结界，与地面彻底隔绝 |

### 铃仙收留事件

| 事件 | 说明 |
|------|------|
| **收留铃仙** | 从月都逃出的月兔铃仙被永琳收留。给她取名"优昙华院"，将其作为弟子的同时也通过她获取月都情报 |
| **永远亭的形成** | 永琳、辉夜、铃仙以及地上的兔妖因幡帝共同组成了永远亭的日常生活 |

### 永夜异变（东方永夜抄——当前时间线）

| 事件 | 说明 |
|------|------|
| **异变起因** | 月都发现永琳和辉夜隐居地的线索，可能派遣追兵。铃仙的行踪也可能已暴露 |
| **永琳的应对** | 制造虚假满月取代真正的满月（天文密葬法），切断月与地之间的通路。将地上世界封闭在壶中天地中 |
| **六面Boss战** | 幻想乡四组自机先后击破永琳，最终见到辉夜 |
| **异变解决** | 异变以和解告终。自机们理解了月亮的真相，永琳也认识到幻想乡居民并无恶意 |
| **结果** | 永琳正式在幻想乡开设药店，开始融入幻想乡社会 |

### 文花帖时期

| 事件 | 说明 |
|------|------|
| **接受采访** | 射命丸文采访永琳，讨论蝴蝶梦丸的销售。永琳大谈信息经济学，透露爱丽丝是常客 |

### 儚月抄时期

| 事件 | 说明 |
|------|------|
| **第二次月面战争** | 八云紫散布"永琳想反攻月都"的谣言，策动第二次月面战争 |
| **永琳的布局** | 暗中协助灵梦等人顺利到达月面。提前写信告知绵月姐妹做好准备，让她们撇清与己的嫌疑 |
| **紫的计谋** | 幽幽子成功偷走永琳留在月都的酒。紫的目的是看到永琳动摇的表情，报了第一次月面战争之仇 |
| **紫的警告** | 紫临走前说"千万要建立结界，否则月都早晚会有灾难" |

### 绀珠传时期

| 事件 | 说明 |
|------|------|
| **异变发生** | 纯狐与地狱女神联手入侵，幻想乡面临危机 |
| **绀珠之药** | 永琳将能预知/体验未来的绀珠之药交给灵梦、铃仙等自机 |
| **幕后支持** | 永琳未亲自参战，以提供工具的方式间接保护了幻想乡 |

### 茨歌仙·智灵奇传时期

| 事件 | 说明 |
|------|------|
| **日常诊疗** | 在幻想乡以药师身份活动，为人类和妖怪提供诊疗 |
| **设定调整** | 在智灵奇传中被称为"前·月之头脑"，暗示与月都的关系已告一段落 |

## 二、作品实际发行时间线

以下为八意永琳在官方作品中登场的实际发行顺序：

| 年份 | 作品 | 登场类型 | 备注 |
|------|------|----------|------|
| **2004年** | 《东方永夜抄》（TH08） | **6面Boss** | **首次登场**。Final A 道中/Boss，Final B 道中。主题曲《千年幻想乡 ~ History of the Moon》 |
| **2005年** | 《东方文花帖》（TH09.5） | Level 9 Boss | 称号"街中的药贩"，符卡"蝴蝶梦丸Nightmare" |

| **2007-2009年** | 《东方儚月抄》（小说） | **重要角色** | 第1、2话，最终话核心登场 |

## 三、参与事件摘要

```
古代 ┬─ 与月夜见共建月之都
     ├─ 担任绵月姐妹老师
     ├─ 制作蓬莱之药（与辉夜合作）
     ├─ 辉夜被流放
     └─ 杀使者/背叛月都/带辉夜逃亡
         ↓
近代 ┬─ 第一次月面战争（结界断八云紫退路）
     ├─ 收留铃仙
     └─ 永远亭隐居生活
         ↓
现代 ┬─ 东方永夜抄（永夜异变——假月事件）
     ├─ 东方文花帖（接受采访）
     ├─ 东方儚月抄（第二次月面战争——书信布局）
     ├─ 东方三月精（骗酒/月之民与妖精论）
     ├─ 东方茨歌仙（日常诊疗）
     ├─ 东方绀珠传（提供绀珠之药）
     └─ 东方智灵奇传（前·月之头脑）
```
