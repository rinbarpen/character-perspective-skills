# 本居小铃 作品时间线

> 本文件整理自《东方铃奈庵～ Forbidden Scrollery.》全14卷，按作品情节时间顺序梳理。

---

## 一、作品登场一览

| 作品 | 类型 | 角色定位 |
|------|------|----------|
| 《东方铃奈庵～ Forbidden Scrollery.》(2012-2017) | 官方漫画（全14卷） | **主角** |
| 《东方外来韦编》(2015-) | 杂志 | 客串/提及 |
| 《东方文果真报》(2017) | 官方书籍 | 客串/提及 |
| 《东方智灵奇传》(2021-) | 官方漫画 | 客串 |

注：小铃是少有的**非自机、非Boss**却拥有自己专属长篇漫画的东方角色。这反映了ZUN对"普通人类视角"的重视。

---

## 二、剧情时间线

### 阶段1：父亲去世前后（背景设定）

| 事件 | 说明 |
|------|------|
| **父亲经营铃奈庵** | 小铃的父亲是铃奈庵的创始人和前店主，收藏了大量普通书籍和妖怪书籍 |
| **父亲去世** | 小铃的父亲在故事开始前已去世，具体死因未明确提及 |
| **小铃继承书店** | 年幼的小铃独自继承了铃奈庵的经营权 |
| **发现妖怪书籍** | 在整理父亲遗物时发现了大量妖怪撰写的书籍 |
| **能力觉醒** | 第一次发现自己能读懂妖怪的文字——"读懂任何书籍程度的能力"自然觉醒 |

---

### 阶段2：铃奈庵故事主线（漫画核心剧情）

#### 第1-2卷：能力的发现与初步风险

| 事件 | 说明 |
|------|------|
| **普通日常经营** | 小铃作为铃奈庵书店主的日常生活，出租普通书籍给人间之里的居民 |
| **妖怪来访** | 开始有妖怪来到书店借阅妖怪书籍，小铃发现了书店的特殊性 |
| **灵梦的注意** | 灵梦察觉到小铃的异常，警告她不要过度接触妖怪书籍 |
| **第一次真正危险** | 阅读了过于强大的封印之书，导致精神受到冲击 |
| **灵梦出手相救** | 灵梦第一次真正介入，退治了被小铃无意中释放的妖怪 |

#### 第3-5卷：人妖交界的书店

| 事件 | 说明 |
|------|------|
| **铃奈庵成为地标** | 越来越多的妖怪来访铃奈庵，书店成为人妖之间的桥梁 |
| **与魔理沙的书友关系** | 魔理沙开始频繁光顾铃奈庵，两人成为书友 |
| **与阿求的合作** | 阿求与小铃建立了知识协作关系，共同解读古籍 |
| **禁书事件** | 某本禁书引起了一系列事件，小铃在其中扮演了关键角色 |
| **能力的代价显现** | 长期阅读妖怪书籍对身体的负担开始显现——头痛、噩梦、偶尔恍惚 |

#### 第6-10卷：危险升级

| 事件 | 说明 |
|------|------|
| **大型异变关联** | 铃奈庵藏书中的某些内容与幻想乡正在发生的异变产生了关联 |
| **被妖怪盯上** | 某些妖怪开始试图利用小铃的能力读取被封印的古代知识 |
| **大封印事件** | 小铃被一本强大的魔导书附身，意识险些被吞噬 |
| **灵梦·魔理沙总动员** | 最严重的一次事件，需要灵梦和魔理沙联手才能解决 |
| **小铃的决心** | 事件后小铃一度陷入自我怀疑，但最终决定继续经营书店并阅读妖怪的书 |

#### 第11-14卷：终章

| 事件 | 说明 |
|------|------|
| **父亲藏书的真相逐渐浮现** | 小铃发现父亲留下的藏书背后隐藏着更大的秘密 |
| **最强封印之书的威胁** | 铃奈庵最危险的一本书被激活，引发了全书最大的危机 |
| **全员集结** | 灵梦、魔理沙、阿求以及部分友善妖怪合力应对危机 |
| **小铃的成长** | 小铃在最终危机中展现出超越"受害者"角色的主动性——用她的知识帮助解决了问题 |
| **铃奈庵的延续** | 危机平息后，铃奈庵继续作为人间之里的书店经营。小铃仍然在读书——只不过这次，她学会了在好奇与安全之间找到平衡 |

---

### 阶段3：铃奈庵结局后的状态

小铃在《东方铃奈庵》结局后保持以下状态：
- 继续经营铃奈庵
- 保留了所有妖怪藏书，但学会了设置阅读界限
- 与灵梦、魔理沙、阿求保持友谊
- 成为幻想乡中特殊的存在——人类之身、妖怪之书、桥梁之人
- 在后续作品中偶尔客串出现，但不再是主角

## 三、与其他作品的交叉时间

### 与主系列游戏的时间关系

小铃的活动时间基本与东方主系列游戏的时间线平行，但她的故事更偏向"日常中的非日常"，而非主系列解决异变的"大事件模式"。

### 与其他漫画的共同时间

小铃的日常与《东方茨歌仙》（茨木华扇为主角的漫画）大致处于同一时间段。两部漫画展现了幻想乡人类生活的不同面向——华扇更多与修行和仙人世界相关，小铃更多与书和知识世界相关。

---

> 以上时间线基于《东方铃奈庵》全14卷内容整理。
