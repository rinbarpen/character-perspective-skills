# Beta（贝塔/ベータ）时间线

---

## 使用说明

本时间线以 Beta 的个人经历为主线，分为六个主要阶段：
1. 被暗影救出前（～故事开始约2年前）
2. 暗影庭园初期（故事开始约2年前～约1年前）
3. 暗影庭园成长期（约1年前～故事主线开始）
4. 故事主线：与暗影重逢后（小说第1卷～第2卷 / 动画第1季）
5. 故事中后期（小说第3卷～第4卷 / 动画第2季）
6. 后续发展（小说第5卷～第6卷 / 未来展望）

---

## 第一阶段：被暗影救出前

**时间**：故事开始约2年前

| 时间 | 事件 | 备注 |
|------|------|------|
| 约15年前 | Beta 出生 | 10月27日，精灵族贵族家庭 |
| 幼年时期 | 与 Alpha 在社交界相识 | 两人在成为魔附体者前就已认识 |
| 约2年前 | 出现魔附体症状 | 身体被魔力侵蚀，痛苦不堪 |
| 约2年前 | 被席德/暗影发现并救出 | 暗影在其面前以"真正的暗影实力者"形象登场，治愈魔附体 |
| 约2年前 | **加入暗影庭园，成为第二位成员** | 代号Beta（β），从此命运改变 |

**关键说明**：
- Beta 是最初的两名成员之一（与 Alpha 一起），七影中除她俩外其他五人都是后来加入的
- 暗影最初只是在"中二病发作"，但 Beta 和 Alpha 认真对待了一切
- Beta 的魔附体症状在她被救后完全消失，取而代之的是强大的魔法能力

*来源：原作小说第1卷、手游《七阴列传》*

## 第二阶段：暗影庭园初期

**时间**：故事开始约2年前～约1年前

| 时间 | 事件 | 备注 |
|------|------|------|
| 加入后不久 | 第一次执行杀人任务 | 虽成功但留下严重心理创伤 |
| 第一次任务后 | **经历严重的噩梦期——害怕红色、精神濒临崩溃** | 这是她最重要的心理转折点 |
| 噩梦期 | 暗影每晚讲故事帮助她 | 安徒生童话、格林童话、日本民间故事、世界文学经典…… |
| 噩梦期 | **开始记录暗影的故事——这是《暗影大人战记》的起源** | 最初是为了帮助自己记住美好的故事，替换噩梦 |
| 噩梦期后 | 心理创伤逐渐愈合，对暗影的忠诚变得更加坚定 | "第二次生命的给予者"的认知在此形成 |
| 约1年前 | 七影其他成员陆续加入 | Gamma、Delta、Epsilon、Zeta、Eta 先后加入 |
| 约1年前 | **暗影庭园组织化** | 七影体系确立，各成员分工明确 |
| 约1年前 | 暗影离开暗影庭园前往米德加学园 | Beta 等人留在暗影庭园经营组织 |

**关键说明**：
- 噩梦期是 Beta 最脆弱的时期，也是她性格成型的核心时期
- 暗影讲故事并非出于深谋远虑的关怀——他只是觉得"一个做噩梦的小女孩很可怜"，但这个单纯的善举彻底改变了 Beta 的人生
- Beta 在暗影离开后，将全部的思念和忠诚转化为对组织建设的投入

*来源：原作小说第1卷、手游《七阴列传》、动画第1季回忆片段*

## 第三阶段：暗影庭园成长期

**时间**：故事开始约1年前～主线开始

| 时间 | 事件 | 备注 |
|------|------|------|
| 暗影离开后 | Beta 等人以暗影的教导为基础自我修炼 | 各自发展出独特的战斗风格 |
| 暗影离开后 | **Beta 以"夏目·卡夫卡"之名出版第一本书** | 将《罗密欧与朱丽叶》改编后出版，大受欢迎 |
| 暗影离开后 | 夏目·卡夫卡迅速走红，成为畅销作家 | 作品涵盖多个类型 |
| 暗影离开后 | Beta 通过社交活动建立情报网络 | 利用作家身份渗透上流社会 |
| 暗影离开后 | **暗影庭园经济体系建立**（Gamma主导） | Mitsugoshi 商会的成立为组织提供资金支持 |
| 暗影离开后 | Beta 与 Gamma 合作，情报+经济双线推进 | 为暗影庭园的壮大奠定基础 |
| 这段时期 | 七影各有发展，组织逐渐成熟 | Beta 的情报网络覆盖整个王国 |

**关键说明**：
- 暗影离开后的这段时间，Beta 等人完全不知道暗影在米德加学园过着"路人高中生"的生活
- Beta 将地球文学改编出版是她的"原罪"——暗影发现后感到失望，但已无法挽回
- 这一段时期在正篇中着墨不多，但在手游《七阴列传》中有详细描写

*来源：原作小说第1卷、手游《七阴列传》*

## 第四阶段：故事主线——与暗影重逢后

### 第1卷 / 动画第1季前半

| 时间 | 事件 | 备注 |

| 故事开始 | 暗影（席德）在米德加学园上学的第二个学期 | 表面上过着路人角色的生活 |

| 时间 | 事件 | 备注 |
|------|------|------|
| 第2卷 | Beta 参与"武心祭"（Bushin Festival）相关的情报工作 | 暗影以"暗影"身份大闹武心祭 |
| 第2卷 | Beta 与伊普西龙的日常竞争加剧 | 包括战斗风格和暗影的关注度 |
| 第2卷 | Beta 在幕后为暗影的行动提供情报支持 | 情报官的本职工作 |

**关键说明**：
- 在第1-2卷期间，Beta 和暗影的"重逢"更多是暗影庭园集体的行动，两人没有太多的单独互动
- Beta 对暗影的崇拜在重逢后不降反升——亲眼看到暗影战斗后，她的狂热更加深了
- 亚蕾克西雅在第2卷中正式与暗影庭园产生交集，Beta 对她的敌意开始显现

*来源：原作小说第1-2卷、动画第1季*

| 第4卷 | 以暗影的"妹妹"身份在现实世界行动 | 首次以非暗影庭园成员身份与暗影近距离相处 |
| 第4卷 | **体验现代科技**（电视、智能手机等） | 文化冲击与迅速适应 |
| 第4卷 | 在现实世界中与西野茜相遇 | 扩展了暗影庭园的跨世界网络 |
| 第4卷 | **将西野茜带回异世界** | 她成为第712号成员 |
| 第4卷 | Beta 成为西野茜在异世界的主要照顾者和沟通桥梁 | 展现了 Beta 温柔和负责任的一面 |

**关键说明**：
- 第3卷中 Beta 的战斗表现是她正篇中重要的动作戏
- 第4卷的现实世界篇是 Beta 个人成长的高光时刻——她不再仅仅是"暗影的追随者"，而是一个能够独立在陌生环境中生存并完成任务的人
- 她在现实世界中称呼暗影为"にーさま"（哥哥大人）这一细节，展现了两人关系的微妙变化
- 与西野茜的相识和照顾她的决定，展现了 Beta 性格中温柔体贴的一面

*来源：原作小说第3-4卷、动画第2季*

## 附：Beta 成长轨迹的关键节点

### 心理成长曲线

```
高心理稳定性
    ↑
    |                          ⑤返回后适应期
    |                   ④现实世界独立生存
    |             ③组织建设期·创作期  
    |        ②噩梦·创伤·治愈
    |   ①被救·新生活开始
    | 弱气精灵少女
    +--------------------------------→ 时间
```

### 关键成长转折点

1. **被暗影救出**（约2年前）：生命轨迹的第一次根本性转折
2. **第一次杀人后的噩梦期**（约2年前）：心理上的巨大考验，也是最深刻的治愈经历
3. **开始写作《暗影大人战记》**（约2年前）：将创伤转化为创作动力的关键
4. **以夏目·卡夫卡身份出道**（约1年前）：从暗影庭园成员到独立个体的身份扩展
5. **穿越到现实世界并独立生存**（第4卷）：从追随者到独立行动者的质变
6. **照顾西野茜**（第4卷后）：从被救者到救助者的角色转变

*来源：综合角色分析*
